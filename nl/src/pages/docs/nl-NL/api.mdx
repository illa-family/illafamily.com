Verbind met een API
Verbind met elke REST, GraphQL API.
Je kunt ILLA verbinden met de meeste API's met behulp van onze native integraties. Als je verbinding moet maken met een interne API of als er geen ondersteunde integratie is je kunt verbinden via onze REST API integratie of GraphQL Integratie.

1.  Maak een API bron 
    Bronnen vertegenwoordigen verbindingen met data bronnen en zijn wat je zoekt binnen je ILLA apps. Log in om te ILLA en navigeer naar het tabblad Resources > klik op nieuwe aanmaken en selecteer de API waarmee je verbinding wilt maken, zoals REST API. Hiermee opent u de Configureren van een resource modal , waar u verbinding informatie toevoegt.

Minimaal moet je de basis-URL van de API waarmee je verbinding wilt maken. U kunt ook URL-parameters, kopteksten, etc. doorgeven. Als je API authenticatie vereist, kun je dat ook op deze pagina configureren. Bijvoorbeeld, OAuth2.0 vereist een Authorization URL, Access Token URL, etc. Klik op Opslaan nadat je je verbindingsgegevens hebt ingevoerd. 
2\. Vraag je API
aan in de App editor, Klik + Nieuw in Actielijst > selecteer REST API en kies een bestaande REST API of maak een nieuwe > klik op aanmaken en update de acties met extra URL parameters, headers, etc.

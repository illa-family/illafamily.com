## <Heading hidden>Aangepaste API authenticatie</Heading>

ILLA ondersteunt standaard veel authenticatienormen (zie hier voor meer informatie). 
Daarnaast heeft ILLA een aangepaste auth-authenticatiemodus waarmee je meerdere stappen kunt combineren die vrijwel elke API-authenticatie ondersteunen. 
Deze verificatiemethoden kunnen worden gebruikt voor zowel REST als GraphQL bronnen.

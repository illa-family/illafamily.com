## <Heading hidden>Julkaise esittely</Heading>

### Pilvi

Kokeile yhden klikkauksen käyttöönottoa ILLA Cloudissa helpoin käyttökokemus.

### Itsenäinen isäntä

Voit itse isännöidä paikallisesti ILLA CLI, Docker compose, k8s helm jne. Olemme toistuvasti optimoineet itsensä isäntäprosessin, jotta voit tehdä itsensä isäntä- ja ohjelmistokäyttöön vain muutamassa minuutissa. Verrattuna pilveen voit käyttää ILLA-tuotteita turvallisemmin ja luotettavammin ja voit käyttää ILLA Builderia yksinkertaisesti vierailemalla rakennustyömaalla isäntänä ja ottamalla käyttöön rakentajan paikallisesti.

Lisätietoja itse isännästä on ILA CLI, Docker compose, k8s helm boot sivuilla.
Jos sinulla on vaikeuksia itsensä lähettämisessä, ota yhteyttä meihin GitHubissa tai business@illasoft.com.

Tarkistaa: [ILLA CLI](https://www.illacloud.com/docs/illa-cli)

<Heading hidden>SQL-Generierung</Heading>

Wir unterstützen die SQL-Generierung in diesen Ressourcen: PostgreSQL, MySQL, Oracle DB, MariaDB, TIDB, SupabaseDB und ClickHouse. Sie können die natürliche Sprache eingeben und eine vorbereitete SQL-Anweisung erhalten, wenn Sie diese Ressourcen verwenden. 

**Betrete die App-Bearbeitungsseite und erstelle eine Aktion mit einer der oben genannten Ressourcen.**

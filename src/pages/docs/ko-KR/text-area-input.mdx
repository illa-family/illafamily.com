## <h2 hidden>What is Text Area Input?</h2>

The Text area input component is a user interface element that allows users to enter and edit the the text area in a form or input field.

### 속성

| 속성          | 설명                                                                                                                                                           |
| ----------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| 기본값         | The initial value of the component. You can dynamically change the initial value by typing JavaScript in {{}}.                                               |
| 자리 표시자      | 입력에 값이 없을 때 표시할 메시지입니다.                                                                                                                                      |
| 상표          | 사용자에게 표시되는 필드의 이름                                                                                                                                            |
| 힌트          | 필드를 자세히 설명하는 데 사용되는 캡션                                                                                                                                       |
| 숨겨진 라벨      | 레이블 표시 여부 설정                                                                                                                                                 |
| 위치          | 구성 요소를 기준으로 레이블 위치 설정                                                                                                                                        |
| 조정          | Set the alignment (align to left or right) of the label                                                                                                      |
| 너비          | When the label is on the left side of the component, set the width ratio of the label.                                                                       |
| 이벤트 핸들러     | 구성 요소 이벤트에 대한 응답으로 쿼리를 트리거하거나 구성 요소를 제어하거나 다른 API를 호출합니다.                                                                                                    |
| 장애가 있는      | Control the status of whether the component is disabled. The component cannot be modified or focused when it is disabled.                                    |
| Read Only   | Control the status of whether the component is read-only. A read-only component can be selected and focused but cannot be modified.                          |
| 지우기 버튼 표시   | control whether or not a clear button is displayed in the input field                                                                                        |
| 문자 수 표시     | control whether or not the character count of the input is displayed                                                                                         |
| 힌트          | Users can enter the component tooltip here. The tooltip will be shown when it is focused. Markdown format is supported.                                      |
| 필수 입력 사항    | 스위치를 켠 후 입력 상자에서 유효성 검사를 수행합니다.                                                                                                                              |
| 정규 표현식      | specify a regular expression pattern that the input value must match                                                                                         |
| Max length  | set the maximum number of characters that can be entered into the input field.                                                                               |
| 최소 길이       | specify the minimum number of characters that the user must enter into the input field in order for the input to be considered valid.                        |
| 맞춤 규칙       | Create your validation logic here. The rules should be made in JavaScript and covered by {{}}.                                                               |
| 확인 정보 숨기기   | You can hide the error message by switching the hidden status when the input value is incorrect. You can dynamically change the hidden status by JavaScript. |
| 양식 데이터 키    | 데이터 속성을 작성할 때 양식 구성요소를 감싸는 키를 지정하십시오.                                                                                                                        |
| 높은          | specify the vertical size or height of the input field                                                                                                       |
| 숨다          | Dynamically control whether the component is hidden. You can change the hidden status through dynamical boolean value.                                       |
| Theme Color | Allows users to specify the button's background color and opacity                                                                                            |

### 방법

You can use other components to control the component. We support the following two methods:

-   **setValue**

To set the text area input value, for example, {{'value1'}}

| 속성    | 설명          |
| ----- | ----------- |
| Value | Input value |

-   **clearValue**

To clear the value of the selected component 

-   **focus**

When the focus method is called, the text area input field will be highlighted and ready for the user to start typing without the need for the user to click on the number input field

### 이벤트 핸들러

| 이벤트   | 설명                                                                 |
| ----- | ------------------------------------------------------------------ |
| 내용 변경 | When a user changes the input value                                |
| 집중하다  | When a user moves the mouse cursor on the input component          |
| 부제    | When a user is done inputting value and quit focusing on component |

### 응답 데이터

The component has some commonly used data, which can be called via **`{{componentName.propertyName}}`** in the app.

| Property name   | 설명                                                                                                 |
| --------------- | -------------------------------------------------------------------------------------------------- |
| 값               | user input value                                                                                   |
| colorScheme     | background color of the button                                                                     |
| customRule      | user-defined rule or validation logic that can be applied to the number input component            |
| 장애가 있는          | a Boolean value indicate the disabled status                                                       |
| displayName     | the name of this component (ie button1)                                                            |
| dynamicHeight   | allows the height of the text area to adjust dynamically based on the content entered by the user. |
| events          | generated or triggered when certain actions or events occur                                        |
| formDataKey     | the Form Data Key of input                                                                         |
| 숨다              | hidden status (true or false)                                                                      |
| 상표              | label value                                                                                        |
| labelAlign      | alignment of input label (left or right)                                                           |
| labelPosition   | position of input label (left or right)                                                            |
| labelWidth      | the integer representing width of label.                                                           |
| maxLength       | the value of maximum length                                                                        |
| minLength       | the value of minimum length                                                                        |
| 자리 표시자          | placeholder value                                                                                  |
| readOnly        | a Boolean value indicate the readyOnly status of input.                                            |
| 필수              | a Boolean value indicate the required status of input.                                             |
| resizeDirection | allows you to control the resizing behavior of the text area                                       |
| 정규 표현식          | apply pattern-based validation to the input text in the text area                                  |
| tooltipText     | value of tooltip text                                                                              |

Example: {{textarea1.value}}

### Use case

Next, we will demonstrate how to map the data from the data source to the Text area input with a button. 

#### Step 1 Add an action

Let us create a table in Supabase called `Project`.  There are two attributes in Project: `id` and `name`. The `name` is the value we want to map to our text area input.

Then we can create a new action for Supabase from the action list, listing all the data in project and names as `supabasedb2.`

Click **`Save`** and **`Run`** to activate this action.

![text_area_data]\(https://cdn.illacloud.com/official-website/img/official-site/components/text_area_data.jpg)

#### Step 2 Add Components

Next, we can add a **`text area input`** component and a **`button`** component to the canvas. 

We set the default value of text area to be 'None' and labeled the button as 'Set value' as shown below

![text_area_look]\(https://cdn.illacloud.com/official-website/img/official-site/components/text_area_look.jpg)

#### Step 3 Configure the component

For the **`button`** component, we can configure it to set the value in **`text area input`** component to the data of the document we read from supabasedb2.

1.  In the Edit event handler, select **`Control component`** in action, and select the **`textarea1`** component that you want to update as the target of the event.
2.  Choose the **`setValue`** action and set the value as  `{{s31.data}}` 

![text_area_config]\(https://cdn.illacloud.com/official-website/img/official-site/components/text_area_config.jpg)

#### Step 4 Test

Now that when you click the 'Set value' button, the value should change to the names.

![text_area_test]\(https://cdn.illacloud.com/official-website/img/official-site/components/text_area_test.jpg)

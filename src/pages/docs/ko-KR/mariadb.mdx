## <h2 hidden>What is MariaDB?</h2>

MariaDB in Illa is a database integration that allows you to connect and interact with a MariaDB database. MariaDB is a popular open-source relational database management system that is derived from MySQL.

With the MariaDB integration, you can query, insert, update, and delete data from a MariaDB database using SQL commands. With the Appwrite integration in Illa, you can query, insert, update, and delete data from a Appwrite database using SQL commands. This makes it easy to build data-driven applications and dashboards that display real-time data from a MariaDB database.

### Create MariaDB

There are two ways to create a resource in Illa after signing into your Illa account. 

-   Create in Resources

Sign into your Illa account, select `**Resources**` on the top of the page, and click `**Create New**` button. 

<img src='https://cdn.illacloud.com/official-website/img/docs/external_resource.png' alt='' />

Select **`MariaDB`** from the database list.

<img src='https://cdn.illacloud.com/official-website/img/docs/action_list.png' alt='' />

Connect to the database with the required parameters described in `Connection Settings` below. 

Click `**Test Connection**` to see if we can successfully connect to the database. If yes, click **`Save Resources`**, else, double check the hostname, port, username, and password is correct.

After creating a resource, the ready MariaDB will display as shown.

<img src='https://cdn.illacloud.com/official-website/img/docs/maria_ex_ready.png' alt='' />

-   Create in Builder

Sign into your Illa account, create a project in Illa Builder in the `**App**` page, and navigate to the **`Action List`** at the bottom of the page. Click `**new**`, then select **`MariaDB`** from the database list. Then, connect to the database with required parameters described in `Connection Settings` below.

Click `**Test Connection**` to see if we can successfully connect to the database. If yes, click **`Save Resources`**, else, double check the hostname, port, username, and password is correct.

### Connection Settings

Here we need to provide information for connecting to MariaDB database.

<img src='https://cdn.illacloud.com/official-website/img/docs/maria_config.png' alt='' />

| 속성      | 필수    | 설명                                                                                                                 |
| ------- | ----- | ------------------------------------------------------------------------------------------------------------------ |
| 이름      | 필수    | ILLA에서 이 자원의 이름을 지정하십시오.                                                                                           |
| 호스트 이름  | 필수    | The URL or IP address for your database                                                                            |
| 포트      | 필수    | The server host's port number that you should use to connect. If you don't specify a port, default port is '3306'. |
| 데이터 베이스 | 필수    | The name of the database                                                                                           |
| 사용자 이름  | 필수    | the username you wish to use when logging in to the MariaDB server.                                                |
| 비밀번호    | 필수    | Use this password for authentication.                                                                              |
| SSL 옵션  | 선택 과목 | decides how high a secure SSL TCP/IP connection is prioritized while negotiating with the server.                  |

### Configure MariaDB

| 속성        | 설명                                                                        |
| --------- | ------------------------------------------------------------------------- |
| SQL query | sql commands to fetch (select), insert, update, delete data from database |
| 변신 로봇     | transforming data into the style you like using Javascript                |

Example usage:

-   Select

```sql
SELECT * FROM users ORDER BY lastname;
```

-   구성 요소

```sql
INSERT INTO users

	(firstname, lastname, email)

VALUES

	(

		{{ NewUser.data.firstNameInput }},

		{{ NewUser.data.lastNameInput }},

		{{ NewUser.data.emailInput }}

	);
```

-   업데이트

```sql
UPDATE users
  SET firstname = '{{form1.updatedProfile.first}}'
  WHERE id = {{ form1.updatedProfile.uid}};
```

-   삭제

```sql
DELETE FROM users WHERE id = {{ form1.updatedProfile.uid }};
```

Обработчики событий позволяют срабатывать запросы, управляйте компонентами и выполняйте другие действия в ответ на пользовательские взаимодействия,
блибликовые компоненты
Используйте библиотеку пользовательского интерфейса ILLA, чтобы создать пользовательский интерфейс вашего приложения.
Компоненты являются неотъемлемой частью ILLA, и они функционируют аналогично React компонентов, в которых они могут принимать свойства и иметь внутреннее состояние. Объединяя библиотеку компонентов ILA с вашими данными, вы можете создавать интерактивные приложения.
Некоторые общие компоненты включают:
Компонент таблицы используется для отображения данных в виде таблицы, и предлагает такие функции, как сортировка, фильтрация, пагинация, экспорт CSV. 
Компонент формы используется для группировки полей ввода и отправки их в одну единицу. 
Входной компонент Select - это комбинация выпадающего списка и поля ввода, которые позволяют пользователям либо выбрать значение из списка, либо ввести значение непосредственно в виде текста. 
Компонент диаграммы используется для визуализации данных с помощью графиков.

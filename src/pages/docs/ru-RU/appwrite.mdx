## <Heading hidden>Appwrite База данных</Heading>

В рамках различных сервисов Appwrite, вы можете найти систему аутентификации пользователей и управления учетными записями, пользовательские настройки, стойкость баз данных и хранения, облачные функции, локализация, управление изображениями и многое другое. 

Сочетание облака ILLA и Appwrite, то есть сочетание низко-бокового кода с внешним низким кодом, поистине реализует полнофункциональную разработку с низким кодом.

[Нажмите здесь](https://appwrite.io/), чтобы узнать больше о Appwrite.

### Настроить ресурс Appwrite в ILLA Cloud

#### Два способа создания ресурсов

Первый способ - войти во вкладку `Ресурсы` > нажмите `Создать новое`, чтобы выбрать Appwrite. 

<img src='https://cdn.illacloud.com/official-website/img/docs/assemble/appwrite1.png' alt="" />

Другой способ - войти на страницу `app editor` > нажмите `+ New`, чтобы выбрать Appwrite > нажмите `+ New Resources`, чтобы добавить новую конфигурацию. 

<img src='https://cdn.illacloud.com/official-website/img/docs/assemble/appwrite2.png' alt="" />

#### Настроить приложение, заполнив следующие поля

Перед подключением к ресурсу Appwrite убедитесь, что вы внедрили Appwrite на вашей локальной машине или в облаке провайдера. [Нажмите здесь](https://appwrite.io/docs/installation), чтобы узнать больше об установке Appwrite. 

| Свойства       | \_обязательный | Описание                                                                                          |
| -------------- | -------------- | ------------------------------------------------------------------------------------------------- |
| Наименование   | обязательный   | Название ресурса при создании действий в ILLA. Вы можете настроить имя для легкой идентификации.  |
| Хост           | обязательный   | НАЗВАНИЕ ГОСТА ИЛИ IP.                                                                            |
| ID базы данных | обязательный   | ID базы данных                                                                                    |
| ID проекта     | обязательный   | ID проекта                                                                                        |
| Api ключ       | обязательный   | ТОКЕН                                                                                             |

### Создание действий в редакторе приложений

#### Создание действий следующим образом

Нажмите `+ New` на странице редактора приложения > для выбора Appwrite > выберите существующий ресурс Appwrite или создайте новый ресурс > нажмите кнопку `Create Action` для создания нового действия. 

<img src='https://cdn.illacloud.com/official-website/img/docs/assemble/appwrite4.png' alt="" />

После создания действия выберите метод. Мы предоставляем следующие методы: Список документов, Создать документ, Обновить документ, Получить документ и удалить документ. Конфигурация каждого метода приводится ниже.

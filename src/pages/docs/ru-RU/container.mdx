## <Heading hidden>Контейнер</Heading>

Вы можете группировать связанные компоненты вместе в карточке с помощью компонента контейнера. Контейнеры также поддерживают несколько представлений, которые позволяют разделять содержимое на различные виды, которые пользователи могут переключаться.

Контейнеры функционируют как и любой другой компонент. После добавления вы можете сгруппировать компоненты внутри него, нажав D или K и перетащив их в контейнер.

### Несколько видов

Контейнеры поддерживают несколько просмотров для группировки контента. Вы можете установить виды на вкладке **Просмотр**. Только один вид видим одновременно, и пользователи переключаются между ними, изменив текущий вид контейнера.

### Использовать другие компоненты для изменения представлений

Вы можете использовать другие компоненты для управления контейнерами, добавляя обработчики событий в другие компоненты. Поддерживаются следующие методы:

1.  Установить текущий вид по ключу
2.  Установить текущий вид по индексу
3.  Показать следующий (видимый) вид
4.  Показать предыдущий (видимый) вид

### Подключить вкладку к контейнеру

Вы можете использовать компонент tab для управления контейнером, включив **Link в контейнер** на вкладке **Inspect**.

После привязки к контейнеру, метки вкладок будут изменены на подписи просмотра. Вы можете изменить представление нажатием вкладок без обработчиков событий.

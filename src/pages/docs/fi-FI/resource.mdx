## <Heading hidden>Resurssi</Heading>

Yhdistä tietolähteeseesi ja soita sitä ILLA Builderiin. Tarjoamme sinulle mukautettavia liitäntämoduuleja erityyppisiä tietolähteitä varten. Voit luoda vakaan yhteyden tietojesi ja ILLA:n välille lyhyessä ajassa ja yksinkertaistaa huomattavasti tietojen käyttöprosessia.

Kun olet onnistuneesti luonut resurssin, voit soittaa sille missä tahansa ILLA-sovelluksessa. Voit myös jakaa resurssejasi yhteistyössä kehittäjien kanssa, mikä parantaa sovellusten rakentamisen tehokkuutta.

### Tuettujen resurssien tyyppi

ILLA tukee useita merkittäviä tietokantoja, kuten MySQL ja Postage, sekä API-ohjelmia kuten Rest API ja GraphQL.

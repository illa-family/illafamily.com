## <h2 hidden>Mitä Amazon S3 on?</h2>

Amazon S3 on erittäin skaalautuva ja turvallinen pilvipalvelu, jota tarjoaa Amazon Web Services (AWS). Yhdessä ILLA Cloudin kanssa kehittäjät voivat integroida Amazon S3:n saumattomasti sovelluksiinsa hyödyntäen vahvoja tallennusvalmiuksiaan tietojen tallentamiseksi ja hakemiseksi tehokkaasti. ILLA Cloud yksinkertaistaa integraatio prosessi, mahdollistaa kehittäjät saumattomasti yhteyden Amazon S3 ja käyttää sen tehokkaita ominaisuuksia, kuten tietojen salaus, versio, kulunvalvonta ja saumaton skaalautuvuus. Integroinnin Amazon S3 ILLA Cloud, kehittäjät voivat parantaa sovelluksiaan luotettavilla ja skaalautuvilla tallennusratkaisuilla, annetaan niille mahdollisuus keskittyä innovatiivisten ja datavetoisten kokemusten kehittämiseen.

### Luo Amazon S3 API

On kaksi tapaa luoda resurssi Illa sen jälkeen, kun olet kirjautunut Illa tilillesi.

-   Luo resursseissa

Kirjaudu sisään Illa tilillesi, valitse `**Resurssit **` sivun yläreunassa ja napsauta `**Create New**` painiketta.

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

Valitse **`Amazon S3`** API-listalta.

![action_list](https://cdn.illacloud.com/official-website/img/docs/action_list.png)

Yhdistä tietokantaan vaaditut parametrit alla olevassa `Connection Settings`ssa.

Napsauta `**Testaa yhteyttä**` nähdäksesi, pystymmekö yhdistämään tietokantaan onnistuneesti. Jos kyllä, klikkaa **`Save Resources`**, muu, tarkista isäntänimi, portti, käyttäjätunnus ja salasana on oikea.

Kun olet luonut resurssin, valmis Amazon S3 näyttää esitetyllä tavalla.

![amazon_ex_ready](https://cdn.illacloud.com/official-website/img/docs/assemble/amazon_ex_ready.jpg)

-   Luo rakentajaan

Kirjaudu Illa tilillesi, luo projekti Illa Builderiin `**App**`-sivulle, ja siirry sivun alareunassa olevaan **`Action List`** -luetteloon. Valitse `**new**`, valitse **`Amazon S3`** tietokantaluettelosta. Yhdistä tietokantaan sitten alla olevassa kohdassa `Connection Settings` kuvatut parametrit.

Napsauta `**Testaa yhteyttä**` nähdäksesi, pystymmekö yhdistämään tietokantaan onnistuneesti. Jos kyllä, klikkaa **`Save Resources`**, muu, tarkista isäntänimi, portti, käyttäjätunnus ja salasana on oikea.

### Yhteyden Asetukset

Tässä meidän on annettava tietoja yhteyden Amazon S3 API.

![amazon_config](https://cdn.illacloud.com/official-website/img/docs/assemble/amazon_config.jpg)

| Ominaisuudet                       | Pakollinen  | Kuvaus                                                                                                                                                                                                                               |
| ---------------------------------- | ----------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Nimi                               | vaadittu    | Resurssin nimi kun luodaan toimia ILLA:ssa.                                                                                                                                                                                          |
| Ämpärin nimi                       | valinnainen | Amazon S3:ssa on yksilöllinen tunniste, joka edustaa säilytysastiaa. Se tarjoaa loogisen erottelun järjestää ja käyttää tiedostoja                                                                                                   |
| S3 ACL ladattaville tiedostoille   | valinnainen | avulla voit määritellä hienojakoiset käyttöoikeudet tiedostoille kauhan sisällä. Se määrittelee, kuka voi suorittaa tiettyjä toimintoja, kuten lukeminen, kirjoittaminen tai poistamalla tiedostoja.                                 |
| Alue                               | vaadittu    | maantieteellinen sijainti, jossa Amazon S3 ämpäri on tallennettu. Se määrittää datan fyysisen sijainnin ja voi vaikuttaa tekijöihin kuten latenssi ja tiedonsiirtokustannukset.                                                      |
| Mukautettu S3-päätepiste           | valinnainen | voit määrittää tietyn päätepisteen URL-osoitteen käyttääksesi Amazon S3 resursseja. Se voi olla hyödyllinen skenaarioissa, joissa haluat käyttää mukautettua verkkotunnusta tai integroida CDN (Content Delivery Network).           |
| Pääsy avain                        | vaadittu    | käyttäjätunnukset, jotka todentavat pyyntösi käyttää Amazon S3:a                                                                                                                                                                     |
| Salainen käyttöoikeusavain         | vaadittu    | generoidaan kun luot IAM (Identity and Access Management) käyttäjän tai AWS (Amazon Web Services) pääsy avain pari.                                                                                                                  |

#### Yleiskatsaus

| Menetelmän Nimi  | Kuvaus                                                                                                                                                 |
| ---------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Toiminnon Tyyppi | luetella kaikki objektit ämpärissä, lukea objektin, ladata objektin, poistaa objektin, poistaa useita objekteja, ladata tietoja, ladata useita tietoja |

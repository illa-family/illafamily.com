## <h2 hidden>Mitä Hydra?</h2>

Hydra on tehokas integrointi ILLA Cloud alustan, joka yhdistää ja yhdistää saumattomasti erilaisia tietolähteitä ja sovelluksia. Keskeisenä keskuksena, Hydra mahdollistaa tehokkaan tiedonkulun ja viestinnän eri järjestelmien välillä, mahdollistaen reaaliaikaisen datan synkronoinnin, automatisoidun työnkulun ja virtaviivaistetun prosessin. Hydralla on vankat valmiudet, ja se mahdollistaa yritysten toiminnan optimoinnin, yhteistyön tehostamisen, ja tehdä tietopohjaisia päätöksiä helposti, kaikki ILLA Cloudin integroidun ekosysteemin sisällä.

### Luo Hydra

On olemassa kaksi tapaa luoda resurssi ILLAan, kun olet kirjautunut tilillesi.

-   Luo resursseissa

Kirjaudu Illa-tilillesi, valitse `**Resurssit**` sivun yläreunassa ja napsauta `**Luo uusi **` -painiketta.

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

Valitse **`Hydra`** tietokantaluettelosta.

![action_list](https://cdn.illacloud.com/official-website/img/docs/action_list_3.jpeg)

Yhdistä tietokantaan vaaditut parametrit alla olevassa `Connection Settings`ssa.

Napsauta `**Testaa yhteyttä**` nähdäksesi, pystymmekö yhdistämään tietokantaan onnistuneesti. Jos kyllä, klikkaa **`Save Resources`**, muu, tarkista isäntänimi, portti, käyttäjätunnus ja salasana on oikea.

Kun resurssi on luotu, valmis Hydra näyttää kuvatulla tavalla.

![hydra_ex_ready](https://cdn.illacloud.com/official-website/img/official-site/intergration/hydra_ex_ready.jpeg)

-   Luo rakentajaan

Kirjaudu Illa tilillesi, luo projekti Illa Builderiin `**App**`-sivulle, ja siirry sivun alareunassa olevaan **`Action List`** -luetteloon. Valitse `**new**`, valitse **`Hydra`** tietokantaluettelosta. Yhdistä sitten tietokantaan vaaditut parametrit alla olevassa `Connection Settings`ssa.

Napsauta `**Testaa yhteyttä**` nähdäksesi, pystymmekö yhdistämään tietokantaan onnistuneesti. Jos kyllä, klikkaa **`Save Resources`**, muu, tarkista isäntänimi, portti, käyttäjätunnus ja salasana on oikea.

### Configure Hydra

| Ominaisuudet  | Kuvaus                                                                           |
| ------------- | -------------------------------------------------------------------------------- |
| SQL-kysely    | sql komentoja noutaaksesi (valita), lisää, päivitä, poista tietoja tietokannasta |
| Muuntaja      | muuttamalla dataa tyyliin, jonka pidät Javascriptistä                            |

Esimerkin käyttö:

-   Valitse

```sql
VALITSE * käyttäjistä ORDER BY sukunimi;
```

-   Insert

```sql
INSERT INTO users

	(firstname, lastname, email)

VALUES

	(

		{{ NewUser.data.firstNameInput }},

		{{ NewUser.data.lastNameInput }},

		{{ NewUser.data.emailInput }}

	);
```

-   Päivitä

```sql
UPDATE users
  SET firstname = '{{form1.updatedProfile.first}}'
  WHERE id = {{ form1.updatedProfile.uid}};
```

-   Poista

```sql
DELETE FROM users WHERE id = {{ form1.updatedProfile.uid }};
```

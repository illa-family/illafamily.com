## <h2 hidden>Mikä on Microsoft SQL?</h2>

Microsoft SQL in Illa on tietokannan integrointi, jonka avulla käyttäjät voivat muodostaa yhteyden ja vuorovaikutuksessa Microsoft SQL (MS SQL) tietokantaan. MS SQL on suosittu avoimen lähdekoodin relaatiotietokannan hallintajärjestelmä, joka on tunnettu sen nopeus, luotettavuus ja helppokäyttöisyys.

MS SQL integraatio Illa, voit kysyä, lisätä, päivittää ja poistaa tietoja MS SQL tietokannasta SQL komentoja. Voit myös luoda mukautettuja SQL kyselyitä ja suorittaa ne Illa. Tämän ansiosta on helppoa rakentaa datapohjaisia sovelluksia ja kojelautoja, jotka näyttävät reaaliaikaista dataa MS SQL -tietokannasta.

### Luo MS SQL

On kaksi tapaa luoda resurssi Illa sen jälkeen, kun olet kirjautunut Illa tilillesi.

-   Luo resursseissa

Kirjaudu Illa-tilillesi, valitse `**Resurssit**` sivun yläreunassa ja napsauta `**Luo uusi **` -painiketta.

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

Valitse **`Microsoft SQL`** tietokantaluettelosta.

![action_list](https://cdn.illacloud.com/official-website/img/docs/action_list.png)

Yhdistä tietokantaan vaaditut parametrit alla olevassa `Connection Settings`ssa.

Napsauta `**Testaa yhteyttä**` nähdäksesi, pystymmekö yhdistämään tietokantaan onnistuneesti. Jos kyllä, klikkaa **`Save Resources`**, muu, tarkista isäntänimi, portti, käyttäjätunnus ja salasana on oikea.

Kun olet luonut resurssin, valmis MS SQL näyttää esitetyllä tavalla.

![ms_ex_ready](https://cdn.illacloud.com/official-website/img/docs/ms_ex_ready.png)

-   Luo rakentajaan

Kirjaudu Illa tilillesi, luo projekti Illa Builderiin `**App**`-sivulle, ja siirry sivun alareunassa olevaan **`Action List`** -luetteloon. Valitse `**new**`, ja valitse **`Microsoft SQL`** tietokantaluettelosta. Yhdistä sitten tietokantaan vaaditut parametrit alla olevassa `Connection Settings`ssa.

Napsauta `**Testaa yhteyttä**` nähdäksesi, pystymmekö yhdistämään tietokantaan onnistuneesti. Jos kyllä, klikkaa **`Save Resources`**, muu, tarkista isäntänimi, portti, käyttäjätunnus ja salasana on oikea.

### Yhteyden Asetukset

Tässä meidän on annettava tietoja yhteyden MS SQL tietokantaan.

![ms_config](https://cdn.illacloud.com/official-website/img/docs/ms_config.png)

| Ominaisuudet            | Pakollinen  | Kuvaus                                                                                                                         |
| ----------------------- | ----------- | ------------------------------------------------------------------------------------------------------------------------------ |
| Nimi                    | vaadittu    | Resurssin nimi kun luodaan toimia ILLA:ssa.                                                                                    |
| Hostname                | vaadittu    | Tietokannan URL-osoite tai IP-osoite                                                                                           |
| Portti                  | vaadittu    | Palvelimen portti numero, jota sinun pitäisi käyttää yhdistämiseen. Jos et määritä porttia, oletusportti on '1433'.            |
| Tietokanta              | vaadittu    | The name of the database                                                                                                       |
| Käyttäjätunnus          | vaadittu    | käyttäjätunnus, jota haluat käyttää, kun kirjaudut sisään MS SQL palvelimelle.                                                 |
| Salasana                | vaadittu    | Käytä tätä salasanaa tunnistautumiseen.                                                                                        |
| Yhteyden valinta        | valinnainen | avaimen arvon pari, joka määrittää yhteysvaihtoehdon                                                                           |
| SSL asetukset           | valinnainen | päättää, kuinka korkea turvallinen SSL TCP / IP yhteys on priorisoitu neuvotellessaan palvelimen kanssa.                       |

### Määritä MS SQL

| Ominaisuudet     | Kuvaus                                                                                                                                                                                                                                                    |
| ---------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Asetusten Tyyppi | konfiguraation tyyppi, jota käytät muodostaa yhteyden MS SQL Server tietokantaan: SQL ja Bulk insert. SQL-tilaan voit käyttää SQL-kyselyä. Bulk-lisäystilassa voit lisätä tiedot valittuun tauluun, jossa on muoto {{ [{x:a, y:b}, … ] }}                 |
| SQL-kysely       | sql komentoja noutaaksesi (valita), lisää, päivitä, poista tietoja tietokannasta.                                                                                                                                                                         |
| Muuntaja         | muuttamalla dataa tyyliin, jonka pidät JavaScriptin avulla                                                                                                                                                                                                |

Esimerkin käyttö:

-   Valitse

```sql
SELECT * FROM users ORDER BY lastname;
```

-   Insert

```sql
INSERT INTO users

	(firstname, lastname, email)

VALUES

	(

		{{ NewUser.data.firstNameInput }},

		{{ NewUser.data.lastNameInput }},

		{{ NewUser.data.emailInput }}

	);
```

-   Päivitä

```sql
UPDATE users
  SET firstname = '{{form1.updatedProfile.first}}'
  WHERE id = {{ form1.updatedProfile.uid}};
```

-   Poista

```sql
DELETE FROM users WHERE id = {{ form1.updatedProfile.uid }};
```

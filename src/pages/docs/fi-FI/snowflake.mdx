## <h2 hidden>Mitä lumihiutale on?</h2>

Vuonna Illa voit käyttää Snowflake kyselyn editori suorittaa SQL kyselyt vastaan Snowflake tietovarastoa. Voit muodostaa yhteyden Snowflakeen syöttämällä Snowflake-tilitietosi, mukaan lukien tilin nimi, käyttäjänimi, salasana ja varaston nimi. Kun olet yhdistänyt Snowflake, voit käyttää kyselyn editoria kirjoittamaan ja suorittamaan SQL kyselyt noutaa, manipuloida, ja analysoida tietoja.

### Luo Lumihiutale

On kaksi tapaa luoda resurssi Illa sen jälkeen, kun olet kirjautunut Illa tilillesi.

-   Luo resursseissa

Kirjaudu Illa-tilillesi, valitse `**Resurssit**` sivun yläreunassa ja napsauta `**Luo uusi **` -painiketta.

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

Valitse **`Snowflake`** tietokantaluettelosta.

![action_list](https://cdn.illacloud.com/official-website/img/docs/action_list.png)

Yhdistä tietokantaan vaaditut parametrit alla olevassa `Connection Settings`ssa.

Napsauta `**Testaa yhteyttä**` nähdäksesi, pystymmekö yhdistämään tietokantaan onnistuneesti. Jos kyllä, klikkaa **`Save Resources`**, muu, tarkista isäntänimi, portti, käyttäjätunnus ja salasana on oikea.

Kun olet luonut resurssin, valmis lumihiutale näyttää kuten on esitetty.

![snow_ex_ready](https://cdn.illacloud.com/official-website/img/docs/snow_ex_Ready.png)

-   Luo rakentajaan

Kirjaudu Illa tilillesi, luo projekti Illa Builderiin `**App**`-sivulle, ja siirry sivun alareunassa olevaan **`Action List`** -luetteloon. Valitse `**new**`, valitse **`Snowflake`** tietokantaluettelosta. Yhdistä sitten tietokantaan vaaditut parametrit alla olevassa `Connection Settings`ssa.

Napsauta `**Testaa yhteyttä**` nähdäksesi, pystymmekö yhdistämään tietokantaan onnistuneesti. Jos kyllä, klikkaa **`Save Resources`**, muu, tarkista isäntänimi, portti, käyttäjätunnus ja salasana on oikea.

### Yhteyden Asetukset

Täällä meidän on annettava tietoa yhteyden Snowflake tietokantaan.

![snow_config](https://cdn.illacloud.com/official-website/img/docs/snow_config.png)

| Ominaisuudet         | Pakollinen  | Kuvaus                                                                                                                                                             |
| -------------------- | ----------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Nimi                 | vaadittu    | Resurssin nimi kun luodaan toimia ILLA:ssa.                                                                                                                        |
| Tilin nimi           | vaadittu    | yksilöllinen tunniste, joka on määritetty jokaiselle lumihiutaletilille                                                                                            |
| Varasto              | vaadittu    | klusteri, jolla lasketaan kyselyjen käsittelyyn ja analyyttisten työmäärien suorittamiseen käytettävät resurssit.                                                  |
| Tietokanta           | vaadittu    | The name of the database                                                                                                                                           |
| Skeema               | valinnainen | tietokannan esineitä kuten taulukoita, näkymiä ja menettelyjä sisältävä säiliö                                                                                     |
| Rooli                | valinnainen | nimetty oikeuksien joukko, joka voidaan antaa yhdelle tai useammalle käyttäjälle                                                                                   |
| Todennus             | vaadittu    | turvallinen pääsy tietoihin ja resursseihin Snowflaken tietovarastossa (Basic Auth tai Key Pari (vain salattuja yksityisiä avaimia tuetaan))                       |
| Käyttäjätunnus       | vaadittu    | käyttäjätunnus, jota haluat käyttää, kun kirjaudut sisään Snowflake palvelimelle.                                                                                  |
| Salasana             | vaadittu    | Käytä tätä salasanaa tunnistautumiseen.                                                                                                                            |

### Luo Toiminnot

Olemme luoneet lumihiutaleresurssin, voimme lisätä toiminnon valitsemalla Snowflake toimintalistalta ja valitsemalla **`Create action`** -painikkeen.

![snow_resource_list](https://cdn.illacloud.com/official-website/img/docs/snow_resource_list.png)

Nyt olemme lisänneet Snowflake palvelimen toiminnaksi meidän rakennuksen sivulle.

![snow](https://cdn.illacloud.com/official-website/img/docs/snow.png)

### Määritä Lumihiutale

| Ominaisuudet  | Kuvaus                                                                           |
| ------------- | -------------------------------------------------------------------------------- |
| SQL-kysely    | sql komentoja noutaaksesi (valita), lisää, päivitä, poista tietoja tietokannasta |
| Muuntaja      | muuttamalla dataa tyyliin, jonka pidät JavaScriptin avulla                       |

Esimerkin käyttö:

-   Valitse

```sql
VALITSE * käyttäjistä ORDER BY sukunimi;
```

-   Insert

```sql
INSERT INTO users

	(firstname, lastname, email)

VALUES

	(

		{{ NewUser.data.firstNameInput }},

		{{ NewUser.data.lastNameInput }},

		{{ NewUser.data.emailInput }}

	);
```

-   Päivitä

```sql
UPDATE users
  SET firstname = '{{form1.updatedProfile.first}}'
  WHERE id = {{ form1.updatedProfile.uid}};
```

-   Poista

```sql
DELETE FROM users WHERE id = {{ form1.updatedProfile.uid }};
```

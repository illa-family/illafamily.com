## <h2 hidden>Mitä Airtable?</h2>

Airtable on suosittu pilvipohjainen relaatiotietokanta ja taulukkolaskentajärjestelmä, joka yhdistää taulukkolaskentataulukon yksinkertaisuuden perinteisen tietokannan monimutkaisuuteen. Sen avulla käyttäjät voivat luoda ja järjestää jäsenneltyjä tietoja käyttäjäystävällisessä käyttöliittymässä, tehdä siitä hyödyllinen eri tehtäviä, kuten projektinhallinta, sisällönsuunnittelu ja tiedonkeruu.

### Määritä takaisin loppu ilmalevyllä

Luo tili `Airtable`-palveluun ja luo tukikohta tarpeen mukaan.

Jotta voit yhdistää tukikohdan ILLA Cloudiin, siirry 'kehittäjäkeskukseen\`.

![air_developer_hub](https://cdn.illacloud.com/official-website/img/official-site/intergration/air_developer_hub.png)

Paina `Personal Access Token` -sivulla `Luo uusi token`.

![air_create_new](https://cdn.illacloud.com/official-website/img/official-site/intergration/air_create_new.jpeg)

Tämä avaa PAT:n konfiguraatioasetukset, joita olet luomassa.

Nimeä ensin PAT, valitse sitten soveltamisalat, joista haluat tämän PAT: n päästä. Lopuksi valitse, millä tasolla PAT on käytettävissä.

![air_new_token](https://cdn.illacloud.com/official-website/img/official-site/intergration/air_new_token.jpeg)

Napsauta **`create token`** -painiketta, meillä on nyt PAT, joka on valmis yhdistämään.

### Luo Airtable ILLA Cloudiin

On kaksi tapaa luoda resurssi Illa sen jälkeen, kun olet kirjautunut Illa tilillesi.

-   Luo resursseissa

Kirjaudu sisään Illa tilillesi, valitse `**Resurssit**` sivun yläreunassa ja napsauta `**Create New**` painiketta.

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

Valitse **`Airtable`** tietokantaluettelosta.

![action_list_3](https://cdn.illacloud.com/official-website/img/official-site/intergration/action_list_3.jpeg)

Yhdistä tietokantaan vaaditut parametrit alla olevassa `Connection Settings`ssa. Täytä tässä tapauksessa **`Personal Access Token`** kopioitiin `Airtable`sta ja liitä se **`Access Token`** -konfiguraatiossa.

Valitse **`Tallenna resurssit`** \*\*\*\*\*Luo resurssi.

Valmis `Airtable` näkyy sellaisena kuin se on esitetty.

![air_ex_ready](https://cdn.illacloud.com/official-website/img/official-site/intergration/air_ex_ready.jpeg)

-   Luo rakentajaan

Kirjaudu sisään Illa tilillesi, luo projekti Illa Builderiin `**App**`-sivulle, ja siirry sivun alareunassa olevaan **`Action List`** -luetteloon. Valitse `**new**`, valitse **`Airtable`** tietokantaluettelosta. Yhdistä tietokantaan sitten alla olevassa kohdassa `Connection Settings` kuvatut parametrit.

Napsauta `**Testaa yhteyttä**` nähdäksesi, pystymmekö yhdistämään tietokantaan onnistuneesti. Jos kyllä, klikkaa **`Save Resources`**, muu, tarkista isäntänimi, portti, käyttäjätunnus ja salasana on oikea.

### Yhteyden Asetukset

Tässä yhteydessä meidän on annettava tietoja, joiden avulla voimme yhdistää Airtable APIin.

![air_config](https://cdn.illacloud.com/official-website/img/official-site/intergration/air_config.jpeg)

| Ominaisuudet        | Pakollinen | Kuvaus                                                                                                                                                  |
| ------------------- | ---------- | ------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Nimi                | vaadittu   | Resurssin nimi kun luodaan toimia ILLA:ssa.                                                                                                             |
| Käyttöoikeus tunnus | vaadittu   | pala tietoja, jotka antavat kolmannen osapuolen sovelluksen tai palvelun luvan käyttää Airtable tilin tietoja ja suorittaa toimia puolestasi            |

### Luo Toiminnot

Olemme luoneet Airtable resurssin, voimme lisätä toiminnon valitsemalla Airtable toimintoluettelosta ja valitsemalla **`Create action`** -painikkeen.

![air_resource_list](https://cdn.illacloud.com/official-website/img/official-site/intergration/air_resource_list.jpeg)

Nyt olemme lisänneet Airtable API toiminnaksi meidän rakennuksen sivulle.

![air](https://cdn.illacloud.com/official-website/img/official-site/intergration/air.jpeg)

### Määritä Airtable

| Ominaisuudet     | Kuvaus                                                                                                                                          |
| ---------------- | ----------------------------------------------------------------------------------------------------------------------------------------------- |
| Perus ID         | yksilöllinen tunniste Airtable tietokannastasi. Voit saada tukikohdan ID:n perusosoitteestasi. Se on yleensä merkkijono, joka alkaa "sovellus". |
| Taulun nimi      | määrittää tietyn taulukon valitun perustan, jonka kanssa haluat olla vuorovaikutuksessa                                                         |
| Toiminnon Tyyppi | minkä tyyppinen toiminta sinun integraatio suorittaa määritetyn taulukon                                                                        |
| Muuntaja         | muuttamalla dataa tyyliin, jonka pidät JavaScriptin avulla                                                                                      |

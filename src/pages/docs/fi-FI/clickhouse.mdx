## <h2 hidden>Mitä ClickHouse on?</h2>

ClickHouse on nopea, kolumni suuntautunut tietokannan hallintajärjestelmä, joka on suunniteltu online-analyyttinen käsittely (OLAP) työkuormat. Se on avoimen lähdekoodin hanke, jonka loi venäläinen hakukoneyritys Yandex. ClickHouse on tunnettu kyvystään käsitellä suuria määriä dataa nopeasti ja tehokkaasti, joten se on ihanteellinen ratkaisu big data analytiikkaan.

Illa Cloudin avulla voit ottaa käyttöön ja hallita ClickHouse klustereita pilvessä ilman, että sinun tarvitsee huolehtia taustalla olevasta infrastruktuurista ja ylläpidosta.

Illa Cloud tarjoaa verkkopohjaisen käyttöliittymän ClickHouse klustereiden hallintaan, voit seurata klusterin suorituskykyä, skaalata resursseja ylös tai alas ja määrittää varmuuskopiot ja turvallisuusasetukset. Voit myös käyttää Illa Cloudia tehdäksesi kyselyjä ClickHouse datassa käyttäen web-pohjaista kyselyn editoria.

### Luo ClickHouse

On kaksi tapaa luoda resurssi Illa sen jälkeen, kun olet kirjautunut Illa tilillesi.

-   Luo resursseissa

Kirjaudu Illa-tilillesi, valitse `**Resurssit**` sivun yläreunassa ja napsauta `**Luo uusi **` -painiketta.

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

Valitse **`ClickHouse`** tietokantaluettelosta.

![action_list](https://cdn.illacloud.com/official-website/img/docs/action_list.png)

Yhdistä tietokantaan vaaditut parametrit alla olevassa `Connection Settings`ssa.

Napsauta `**Testaa yhteyttä**` nähdäksesi, pystymmekö yhdistämään tietokantaan onnistuneesti. Jos kyllä, klikkaa **`Save Resources`**, muu, tarkista isäntänimi, portti, käyttäjätunnus ja salasana on oikea.

Kun luot resurssi, valmis ClickHouse näyttää kuten on esitetty.

![click_config](https://cdn.illacloud.com/official-website/img/docs/click_config.png)

-   Luo rakentajaan

Kirjaudu Illa tilillesi, luo projekti Illa Builderiin `**App**`-sivulle, ja siirry sivun alareunassa olevaan **`Action List`** -luetteloon. Valitse `**new**`, valitse **`ClickHouse`** tietokantaluettelosta. Yhdistä sitten tietokantaan vaaditut parametrit alla olevassa `Connection Settings`ssa.

Napsauta `**Testaa yhteyttä**` nähdäksesi, pystymmekö yhdistämään tietokantaan onnistuneesti. Jos kyllä, klikkaa **`Save Resources`**, muu, tarkista isäntänimi, portti, käyttäjätunnus ja salasana on oikea.

### Yhteyden Asetukset

Tässä meidän on annettava tietoja yhteyden ClickHouse tietokantaan.

![click_config](https://cdn.illacloud.com/official-website/img/docs/click_config.png)

| Ominaisuudet      | Pakollinen  | Kuvaus                                                                                                                         |
| ----------------- | ----------- | ------------------------------------------------------------------------------------------------------------------------------ |
| Nimi              | vaadittu    | Resurssin nimi kun luodaan toimia ILLA:ssa.                                                                                    |
| Hostname          | vaadittu    | Tietokannan URL-osoite tai IP-osoite                                                                                           |
| Portti            | vaadittu    | Palvelimen portti numero, jota sinun pitäisi käyttää yhdistämiseen. Jos et määritä porttia, oletusportti on '9440'.            |
| Tietokanta        | vaadittu    | The name of the database                                                                                                       |
| Käyttäjätunnus    | vaadittu    | käyttäjätunnus, jota haluat käyttää, kun kirjaudut sisään ClickHouse palvelimelle.                                             |
| Salasana          | vaadittu    | Käytä tätä salasanaa tunnistautumiseen.                                                                                        |
| SSL asetukset     | valinnainen | päättää, kuinka korkea turvallinen SSL TCP / IP yhteys on priorisoitu neuvotellessaan palvelimen kanssa.                       |

### Configure ClickHouse

| Ominaisuudet  | Kuvaus                                                                           |
| ------------- | -------------------------------------------------------------------------------- |
| SQL-kysely    | sql komentoja noutaaksesi (valita), lisää, päivitä, poista tietoja tietokannasta |
| Muuntaja      | muuttamalla dataa tyyliin, jonka pidät JavaScriptin avulla                       |

Esimerkin käyttö:

-   Valitse

```sql
SELECT * FROM users ORDER BY lastname;
```

-   Insert

```sql
INSERT INTO users

	(firstname, lastname, email)

VALUES

	(

		{{ NewUser.data.firstNameInput }},

		{{ NewUser.data.lastNameInput }},

		{{ NewUser.data.emailInput }}

	);
```

-   Päivitä

```sql
UPDATE users
  SET firstname = '{{form1.updatedProfile.first}}'
  WHERE id = {{ form1.updatedProfile.uid}};
```

-   Poista

```sql
DELETE FROM users WHERE id = {{ form1.updatedProfile.uid }};
```

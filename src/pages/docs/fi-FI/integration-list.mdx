## <Heading hidden>Integraatiolista</Heading>

ILLA tarjoaa valikoiman valmiiksi rakennettuja integroituja, joiden avulla voit helposti liittää tietolähteisiin minimaalisen kokoonpanon. Valitse vain sopiva integrointi käytettävissä olevien vaihtoehtojen luettelosta, syötä tarvittavat tiedot ja voit alkaa käyttää tietolähdettäsi ILLA:ssa. Tämä poistaa tarpeen perustaa monimutkaisia konfiguraatioita ja mahdollistaa voit nopeasti aloittaa työskentelyn tietojasi.

### Tietokannan integraatiot

| Nimi          | Kuvaus                                                                                                                                                                                                                                                 |
| ------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Mysql         | MySQL on avoimen lähdekoodin relaatiotietokannan hallintajärjestelmä (RDBMS), joka perustuu strukturoituun kyselyn kieleen (SQL).                                                                                                                      |
| PostgreSQL    | PostgreSQL on avoimen lähdekoodin objekti-suhteellinen tietokannan hallintajärjestelmä (ORDBMS), joka on tunnettu luotettavuudestaan ja ominaisuus-rikkaus.                                                                                            |
| MariaDB       | MariaDB on avoimen lähdekoodin tietokannan hallintajärjestelmä, joka on MySQL-tietokannan haara.                                                                                                                                                       |
| TiDB          | TiDB on avoimen lähdekoodin NewSQL-tietokanta, joka tukee Hybrid Transactional and Analytical Processing -työkuormia.                                                                                                                                  |
| Redis         | Redis on avoimen lähdekoodin muistinsisäinen tietojen rakennekauppa, jota käytetään usein tietopankkina, välimuistina ja viestivälittäjänä.                                                                                                            |
| MongoDB       | MongoDB on ilmainen ja avoimen lähdekoodin cross-platform dokumentointipainotteinen tietokantaohjelma.                                                                                                                                                 |
| Joustava Haku | Elasticsearch on Apache Lucenen päälle rakennettu avoin, hajautettu haku- ja analytiikkamoottori.                                                                                                                                                      |
| Firebase      | Firebase on Googlen kehittämä pilvipohjainen mobiili- ja verkkosovellusten kehitysalusta.                                                                                                                                                              |
| Supabase DB   | Supabase on avoimen lähdekoodin alusta reaaliaikaisten palvelimettomien sovellusten rakentamiseen. Se on rakennettu PostgreSQL:n päälle ja tarjoaa useita työkaluja ja palveluita palvelinttömien sovellusten nopeaan rakentamiseen ja käyttöönottoon. |
| Clickhouse    | ClickHouse on avoimen lähdekoodin sarakkeen suuntautunut tietokannan hallintajärjestelmä, joka on suunniteltu online-analyyttinen käsittely (OLAP) ja reaaliaikainen tietojen analysointi.                                                             |

### API integraatiot

| Nimi                  | Kuvaus                                                                                                                                        |
| --------------------- | --------------------------------------------------------------------------------------------------------------------------------------------- |
| REST API              | REST on arkkitehtoninen tyyli rakennusten verkkopalveluissa. RESTful API ovat sovellusliittymiä, jotka seuraavat REST-arkkitehtonista tyyliä. |
| Amazon S3(General S3) | Amazon S3 on AWS:n tarjoama objektin tallennuspalvelu. Se tarjoaa turvallisen, skaalautuvan ja edullisen tallennustilan internetissä.         |
| SMTP                  | SMTP (Simple Mail Transfer Protocol) on protokolla sähköpostien lähettämiseen ja vastaanottamiseen internetin kautta.                         |
| GraphQL               | GraphQL on kyselyn kieli API ja runtime täyttää nämä kyselyt kanssa olemassa olevia tietoja.                                                  |

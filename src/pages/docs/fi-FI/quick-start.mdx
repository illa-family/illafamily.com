## <Heading hidden>Yleiskatsaus</Heading>

### 1.  Julkaise ILLA Builder

Ennen kuin aloitat ILLA Builderin käytön, sinun tulee käyttää ILLA Cloudia tai Itsesatamaa ILLAa sovelluksien ylläpitämiseen. Tarjoamme sinulle ILLA CLI, Docker Compose, ja K8s itse isäntä ILLA alle 5 minuutissa. Tarjoamme sinulle ILLA Cloud myöhemmin, jotta voit yksinkertaisesti aloittaa ILLA Builder.

<img
  src='https://cdn.illacloud.com/official-website/img/docs/qs_e_1.png'
  alt=""
/>

### 2.  Hanki käyttäjien tiedot

Rekisteröidy ja kirjaudu ILLA Builderiin ja kirjoita sitten resurssivälilehti. Klikkaa "Luo uusi" painiketta ja voit valita resurssityypin täällä. Tarjoamme sinulle REST API ja MySQL yhdistää tiedot. Postgres, Redis, ja MongoDB ovat saatavilla tässä kuussa.

<img
  src='https://cdn.illacloud.com/official-website/img/docs/qs_e_2.png'
  alt=""
/>

### 3.  Vedä ja pudota lisäosia

Kun tietosi on valmis, voit palata sovellusten välilehteen ja luoda sovelluksen. The insert list is on the right side of the page. Voit valita yhden niistä ja vetää sen kehykseen. Napsauta sitten lisäosaa ja muokkaa sen parametreja Inspect-paneelissa. Tarjoamme sinulle markdown -tilan muokataksesi jokaisen lisäyksen etikettiä. Jos tunnet Markdown-syntaksin, voit säätää tekstin kokoa ja paksuutta nopeasti. Sillä välin voit käyttää {{double brackets}} käyttääksesi toiminnon tietoja tekstinsyöttöruudussa saadaksesi lisää tuloksia ILLA Builderissa.

### 4.  Lisää toiminto sovelluksissa

Toimi on tarpeen, jotta tiedot voidaan muuntaa resursseista lisättäviksi. Me kutsumme sitä toiminnaksi. Voit kirjoittaa kyselyn toiminnassa, jotta voit saalis tietoja resursseistasi. Tällä perusteella voit kirjoittaa JavaScriptin toimintoihin, joilla voit suorittaa tietojenkäsittelyohjeita, kuten suodattimen ja yhdistämisen. MySQL esimerkki on alla:

<img
  src='https://cdn.illacloud.com/official-website/img/docs/qs_e_3.png'
  alt=""
/>

1.  Luo MySQL toiminto App editorissa. Nimetty se toiminta1.

2.  Syötä kyselyn koodi paneelissa, esim. SELECT \* FROM Käyttäjät.

3.  Suorita tämä toiminto, kyselyn tulos näytetään alhaalla olevassa paneelissa.

### 5.  Näytä tietosi

Kun toiminto on onnistuneesti luotu, sinun täytyy sitoa toiminnon tulos komponenttiin. Sinun pitäisi korvata tarkastuspaneelissa oleva arvomuuttuja toimintojen {{action1.data}} tuloksena. Sitten komponentin tiedot sitoutuvat toiminnon tulokseen.

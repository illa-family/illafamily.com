## <h2 hidden>Mitä PostgreSQL on?</h2>

PostgreSQL Illa on tietokannan integraatio, jonka avulla voit yhdistää ja vuorovaikutuksessa PostgreSQL tietokantaan. PostgreSQL on suosittu avoimen lähdekoodin relaatiotietokannan hallintajärjestelmä, joka tunnetaan sen vakaudesta, luotettavuudesta ja tehokkaista ominaisuuksista.

PostgreSQL integraatio Illa, voit kysyä, lisätä, päivittää ja poistaa tietoja PostgreSQL tietokannasta käyttäen SQL komentoja. Voit myös luoda mukautettuja SQL-kyselyitä ja suorittaa ne Illa-toiminnossa. Tämän ansiosta on helppoa rakentaa datapohjaisia sovelluksia ja dashboardeja, jotka näyttävät reaaliaikaista dataa PostgreSQL-tietokannasta.

### Luo PostgreSQL

On kaksi tapaa luoda resurssi Illa sen jälkeen, kun olet kirjautunut Illa tilillesi.

-   Luo resursseissa

Kirjaudu Illa-tilillesi, valitse `**Resurssit**` sivun yläreunassa ja napsauta `**Luo uusi **` -painiketta.

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

Valitse **`PostgreSQL`** tietokantaluettelosta.

![action_list](https://cdn.illacloud.com/official-website/img/docs/action_list.png)

Yhdistä tietokantaan vaaditut parametrit alla olevassa `Connection Settings`ssa.

Napsauta `**Testaa yhteyttä**` nähdäksesi, pystymmekö yhdistämään tietokantaan onnistuneesti. Jos kyllä, klikkaa **`Save Resources`**, muu, tarkista isäntänimi, portti, käyttäjätunnus ja salasana on oikea.

Kun olet luonut resurssin, valmis PostgreSQL näyttää esitetyllä tavalla.

![ex_post_ready](https://cdn.illacloud.com/official-website/img/docs/ex_post_ready.png)

-   Luo rakentajaan

Kirjaudu Illa tilillesi, luo projekti Illa Builderiin `**App**`-sivulle, ja siirry sivun alareunassa olevaan **`Action List`** -luetteloon. Valitse `**new**`, valitse **`PostgreSQL`** tietokantaluettelosta. Yhdistä sitten tietokantaan vaaditut parametrit alla olevassa `Connection Settings`ssa.

Napsauta `**Testaa yhteyttä**` nähdäksesi, pystymmekö yhdistämään tietokantaan onnistuneesti. Jos kyllä, klikkaa **`Save Resources`**, muu, tarkista isäntänimi, portti, käyttäjätunnus ja salasana on oikea.

### Yhteyden Asetukset

Täällä meidän on annettava tietoa yhteyden PostgreSQL tietokantaan.

![post_config](https://cdn.illacloud.com/official-website/img/docs/post_config.png)

| Ominaisuudet      | Pakollinen  | Kuvaus                                                                                                                         |
| ----------------- | ----------- | ------------------------------------------------------------------------------------------------------------------------------ |
| Nimi              | vaadittu    | Resurssin nimi kun luodaan toimia ILLA:ssa.                                                                                    |
| Hostname          | vaadittu    | Tietokannan URL-osoite tai IP-osoite                                                                                           |
| Portti            | vaadittu    | Palvelimen portti numero, jota sinun pitäisi käyttää yhdistämiseen. Jos et määritä porttia, oletusportti on '5432'.            |
| Tietokanta        | vaadittu    | The name of the database                                                                                                       |
| Käyttäjätunnus    | vaadittu    | käyttäjätunnus, jota haluat käyttää, kun kirjaudut sisään PostgreSQL palvelimelle.                                             |
| Salasana          | vaadittu    | Käytä tätä salasanaa tunnistautumiseen.                                                                                        |
| SSL asetukset     | valinnainen | päättää, kuinka korkea turvallinen SSL TCP / IP yhteys on priorisoitu neuvotellessaan palvelimen kanssa.                       |

### Määritä PostgreSQL

| Ominaisuudet  | Kuvaus                                                                           |
| ------------- | -------------------------------------------------------------------------------- |
| SQL-kysely    | sql komentoja noutaaksesi (valita), lisää, päivitä, poista tietoja tietokannasta |
| Muuntaja      | muuttamalla dataa tyyliin, jonka pidät JavaScriptin avulla                       |

Esimerkin käyttö:

-   Valitse

```sql
SELECT * FROM users ORDER BY lastname;
```

-   Insert

```sql
INSERT INTO users

	(firstname, lastname, email)

VALUES

	(

		{{ NewUser.data.firstNameInput }},

		{{ NewUser.data.lastNameInput }},

		{{ NewUser.data.emailInput }}

	);
```

-   Päivitä

```sql
UPDATE users
  SET firstname = '{{form1.updatedProfile.first}}'
  WHERE id = {{ form1.updatedProfile.uid}};
```

-   Poista

```sql
DELETE FROM users WHERE id = {{ form1.updatedProfile.uid }};
```

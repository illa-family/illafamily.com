## <h2 hidden>Mitä tulipalo on?</h2>

Firebase on Googlen tarjoama kattava alusta, joka tarjoaa laajan valikoiman työkaluja ja palveluita verkko- ja mobiilisovellusten kehittämiseen ja hallintaan. Se tarjoaa kehittäjille skaalautuvan ja turvallisen taustainfrastruktuurin sekä valmiiksi rakennetut ominaisuudet ja toiminnot nopeuttamaan kehitysprosessia. Firebasen integroinnilla ILLA Cloudiin kehittäjät voivat parantaa sovelluksiaan luotettavilla ja skaalautuvilla tallennusratkaisuilla, annetaan niille mahdollisuus keskittyä innovatiivisten ja datavetoisten kokemusten kehittämiseen.

### Luo Firebase API

On kaksi tapaa luoda resurssi Illa sen jälkeen, kun olet kirjautunut Illa tilillesi.

-   Luo resursseissa

Kirjaudu sisään Illa tilillesi, valitse `**Resurssit **` sivun yläreunassa ja napsauta `**Create New**` painiketta.

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

Valitse **`Firebase`** API-listalta.

![action_list](https://cdn.illacloud.com/official-website/img/docs/action_list.png)

Yhdistä tietokantaan vaaditut parametrit alla olevassa `Connection Settings`ssa.

Napsauta `**Testaa yhteyttä**` nähdäksesi, pystymmekö yhdistämään tietokantaan onnistuneesti. Jos kyllä, klikkaa **`Save Resources`**, muu, tarkista isäntänimi, portti, käyttäjätunnus ja salasana on oikea.

Kun resurssi on luotu, valmis Firebase näyttää esitetyllä tavalla.

![firebase_ex_ready](https://cdn.illacloud.com/official-website/img/official-site/intergration/firebase_ex_ready.jpg)

-   Luo rakentajaan

Kirjaudu Illa tilillesi, luo projekti Illa Builderiin `**App**`-sivulle, ja siirry sivun alareunassa olevaan **`Action List`** -luetteloon. Valitse `**new**`, valitse **`Firebase`** tietokantaluettelosta. Yhdistä tietokantaan sitten alla olevassa kohdassa `Connection Settings` kuvatut parametrit.

Napsauta `**Testaa yhteyttä**` nähdäksesi, pystymmekö yhdistämään tietokantaan onnistuneesti. Jos kyllä, klikkaa **`Save Resources`**, muu, tarkista isäntänimi, portti, käyttäjätunnus ja salasana on oikea.

### Yhteyden Asetukset

Tältä osin meidän on annettava tietoja Firebase APIin liittämiseksi.

![firebase_config](https://cdn.illacloud.com/official-website/img/official-site/intergration/firebase_config.jpg)

| Ominaisuudet               | Pakollinen  | Kuvaus                                                                                        |
| -------------------------- | ----------- | --------------------------------------------------------------------------------------------- |
| Nimi                       | vaadittu    | Resurssin nimi kun luodaan toimia ILLA:ssa.                                                   |
| Firebase Database URL      | valinnainen | päätepiste tai URL, jossa Firebase Realtime Database on hosted                                |
| Fire-Projektin Tunnus      | valinnainen | yksilöllinen tunniste kullekin Firebase hankkeelle, joka käyttää Firestore tietokanta palvelu |
| Yksityinen Avain           | vaadittu    | turvatunnus, jota käytetään Firebase palvelujen todentamiseen ja käyttöön ohjelmoitavalla     |

#### Hanki käyttäjä UID:llä

hae käyttäjätiedot määrittämällä käyttäjän yksilöllinen tunniste (UID). Sen avulla voit hakea käyttäjän tietoja, kuten näytön nimi, sähköposti, puhelinnumero, ja mukautettuja attribuutteja käyttäjän kanssa.

** Input **

| Ominaisuudet | Pakollinen | Kuvaus                                                                                                                            |
| ------------ | ---------- | --------------------------------------------------------------------------------------------------------------------------------- |
| UID          | vaadittu   | "Käyttäjän Tunniste." Se on yksilöllinen tunniste, joka on osoitettu jokaiselle käyttäjälle Firebase autentikointijärjestelmässä. |

#### Hae käyttäjä sähköpostilla

hae käyttäjän tiedot määrittämällä käyttäjän sähköpostiosoite. Sen avulla voit hakea käyttäjätietoja kuten UID-pohjainen menetelmä, mutta käyttämällä sähköpostia tunnisteena.

** Input **

| Ominaisuudet    | Pakollinen | Kuvaus                                                                      |
| --------------- | ---------- | --------------------------------------------------------------------------- |
| sähköposti      | vaadittu   | rekisteröity sähköpostiosoite, joka liittyy käyttäjäön, jonka haluat noutaa |

#### Hanki käyttäjä puhelinnumerolla

hae käyttäjätiedot määrittämällä käyttäjän puhelinnumero. Sen avulla voit hakea käyttäjätietoja kuten UID-pohjainen menetelmä, mutta käyttämällä puhelinnumero tunnuksena.

** Input **

| Ominaisuudet   | Pakollinen | Kuvaus                                                                  |
| -------------- | ---------- | ----------------------------------------------------------------------- |

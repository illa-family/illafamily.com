## <h2 hidden>Co je to tabulka?</h2>

Komponenta tabulky v ILLA Cloud je výkonná funkce, která uživatelům umožňuje zobrazovat, organizovat a manipulovat s tabulkovými daty v aplikaci. Poskytuje strukturované rozložení s řádky a sloupci, které uživatelům umožňuje pohodlným a strukturovaným způsobem prohlížet a interagovat s nimi.

### Vlastnosti

| Vlastnosti                             | Popis                                                                                                                                |
| -------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------ |
| Zdroj údajů                            | zdroj údajů, který obsahuje tabulku                                                                                                  |
| Prázdný stav                           | obsah nebo zpráva zobrazené, když tabulka nemá žádné údaje k zobrazení                                                               |
| Načítání                               | zda je složka tabulky v současné době ve stavu zatížení                                                                              |
| Sloupce                                | struktura a konfigurace sloupců v tabulce. (Podrobnosti naleznete v sekci o sloupcích níže)                                          |
| Sloupec                                | počet sloupců                                                                                                                        |
| Výběr více řádků                       | Pro přístup k datům použijte {{table.selectedRow[n].columnName}}.                                                                    |
| Výběr buňky                            | Určuje, zda povolit uživatelům vybrat buňku                                                                                          |
| Klikněte vně pro zrušení výběru        | Podporováno v režimu jediného výběru. Po povolení klikněte na jinou oblast než tabulku pro zrušení výběru řádku.                     |
| Přechodný                              | jak tabulka zpracovává obsah přesahující dostupný prostor v kontejneru tabulky (stránkování nebo scroll)                             |
| Povolit stránkování na straně serveru  | Zda povolit stránkování serveru                                                                                                      |
| Velikost stránky                       | počet řádků, které mají být zobrazeny na stránce, když je povolena stránkování na straně serveru                                     |
| Aktualizovat                           | Zda zobrazit ikonu obnovení.                                                                                                         |
| Stáhnout                               | Zobrazit tlačítko stahování v panelu nástrojů.                                                                                       |
| Stáhnout nezpracovaná data             | Zobrazení ikony nezpracovaných dat ke stažení.                                                                                       |
| Filtr                                  | Zobrazit tlačítko filtru v nástrojové liště                                                                                          |
| Manažer událostí                       | odhalování konkrétních akcí nebo událostí uživatelů a reakce na ně, jako jsou kliknutí, klávesnice nebo podklady formulářů.          |
| Zakázáno                               | neinteraktivní a nemůže být uživatelem upraven nebo spuštěn.                                                                         |

#### **Přehled vlastností sloupce**

Pro každý sloupec můžeme nakonfigurovat jeho vlastnosti tak, aby upravovaly data zobrazená pro každý sloupec.

| Vlastnosti                                        | Popis                                                                                                                                                                                                     |
| ------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Název sloupce                                     | specifikuje název nebo štítek sloupce, který slouží jako záhlaví k identifikaci údajů, které zastupuje.                                                                                                   |
| Typ sloupce                                       | definuje typ dat, která sloupec obsahuje, včetně auta, textu, data, značky, času, datumu číslo, procento, odkaz, tlačítko, skupina tlačítek, boolean, obrázek, ikonová skupina, Hodnocení, Markdown, měna |
| Mapovaná hodnota                                  | Použijte {{ currentRow }} pro přístup k datům aktuálního řádku. Například, pro výpočet "cost per click" podle sloupců "cost" a "clickTimes":                                                              |
| {{ currentRow.cost / currentRow.clickTimes }}     |                                                                                                                                                                                                           |
| Povolit řazení                                    | zda sloupec umožňuje uživatelům seřadit údaje tabulky na základě hodnot v tomto konkrétním sloupci                                                                                                        |
| Pozadí                                            | Nastavte pozadí sloupce. Použij {{ napsat podmíněný příkaz k nastavení barvy zobrazované každou buňkou za různých podmínek.                                                                               |
| Zarovnání                                         | řídí vodorovné seřízení obsahu ve sloupcích buňky.                                                                                                                                                        |

#### Typy sloupců:

-   Automaticky

dynamický typ, který automaticky určuje datový typ na základě hodnot ve sloupci

-   Text

představuje sloupec, který obsahuje textová data

-   Datum:

představuje hodnoty údajů, které odpovídají konkrétním datům

| Vlastnosti | Popis                                                            |
| ---------- | ---------------------------------------------------------------- |
| Formát     | přijímá různé formátovací kódy nebo vzory, výchozí je YYYY-MM-DD |

-   Značka

specializovaný formát pro zobrazení štítků nebo štítků spojených se zápisy dat

| Vlastnosti      | Popis                                                             |
| --------------- | ----------------------------------------------------------------- |
| Popisek štítku  | představuje text nebo štítek přiřazený ke štítku ve sloupci "tag" |
| Barva štítku    | definuje barvu přiřazenou značce ve sloupci "tag"                 |

-   Čas

přesně zobrazovat a manipulovat časové hodnoty v tabulce

| Vlastnosti | Popis                                                                       |
| ---------- | --------------------------------------------------------------------------- |
| Formát     | vizuální znázornění časových dat ve sloupci, výchozí hodnota je HH:mm:ss    |

-   Datum a čas

Obchody a reprezentuje hodnoty data a času

| Vlastnosti | Popis                                                                                                      |
| ---------- | ---------------------------------------------------------------------------------------------------------- |
| Formát     | specifikuje, jak mají být ve sloupci zobrazeny hodnoty data a času, výchozí hodnota je YYYY-MM-DD HH:mm:ss |

-   Číslo

zpracovávat numerická data

| Vlastnosti                 | Popis                                                                                                                                 |
| -------------------------- | ------------------------------------------------------------------------------------------------------------------------------------- |
| Desetinná místa            | specifikovat počet desetinných míst, která se mají zobrazit pro číselné hodnoty ve sloupci                                            |
| Zobrazit tisíce oddělovače | zda má být sloupec zobrazen tisícovka oddělovače, aby se zlepšila čitelnost a snadnost interpretace velkého počtu                     |

-   Procento

manipulovat s procentními hodnotami stručným a vizuálně přitažlivým způsobem

| Vlastnosti                 | Popis                                                                                             |
| -------------------------- | ------------------------------------------------------------------------------------------------- |
| Desetinná místa            | specifikovat počet desetinných míst, která mají být zobrazena pro procentuální hodnoty ve sloupci |
| Zobrazit tisíce oddělovače | zda by měl být do zobrazených procentních hodnot zahrnut tisícový oddělovač                       |

-   Odkaz

zahrnout klikatelné odkazy do konkrétního sloupce

-   Tlačítko

zahrnout interaktivní tlačítka do určitého sloupce

| Vlastnosti       | Popis                                                                                  |
| ---------------- | -------------------------------------------------------------------------------------- |
| Manažer událostí | definuje akci nebo událost, ke které dojde po kliknutí na tlačítko ve sloupci          |
| Zakázáno         | zda je tlačítko ve sloupci povoleno nebo zakázáno                                      |
| Barva motivu     | vyberte barvu motivu pro tlačítko                                                      |
| Varianta         | vizuální styl nebo varianta tlačítka (vyplňte nebo obrys)                              |

-   skupina tlačítek

umožňuje uživatelům vytvářet interaktivní skupiny tlačítek ve sloupci

Pro každé tlačítko ve sloupcích můžeme nakonfigurovat jeho vlastnosti.

| Vlastnosti        | Popis                                                                                                                               |
| ----------------- | ----------------------------------------------------------------------------------------------------------------------------------- |
| Mapovaná hodnota  | přiřadit každé tlačítko do skupiny tlačítek určitou hodnotu nebo pole ze zdroje dat                                                 |
| Varianta          | vizuální styl nebo vzhled tlačítek ve skupině tlačítek                                                                              |
| Manažer událostí  | akce nebo funkce spuštěná po kliknutí na tlačítko ve skupině nebo po interakci                                                      |
| Zakázáno          | zda jsou tlačítka ve skupině tlačítek povolena nebo zakázána                                                                        |
| Barva motivu      | upravte barvu nebo styl tlačítek ve skupině tak, aby odpovídaly celkovému motivu nebo značce jejich aplikace nebo webové stránky    |

-   Boolean

představuje údaje, které mají dva možné stavy: pravdivé nebo nepravdivé

-   Obrázek

Vyplňte sloupec v tabulce s údaji o obrázku

| Vlastnosti  | Popis                                                 |
| ----------- | ----------------------------------------------------- |
| Typ měřítka | jak je obrázek zmenšen nebo upraven v tabulkové buňce |

-   Skupina ikon

specializovaný sloupec, který umožňuje uživatelům zobrazit ikony ve skupině nebo sbírce

Pro každé tlačítko ve sloupcích můžeme nakonfigurovat jeho vlastnosti.

| Vlastnosti       | Popis                                                                                                                 |
| ---------------- | --------------------------------------------------------------------------------------------------------------------- |
| Ikona            | umožňuje uživatelům vybrat nebo specifikovat ikonu zobrazenou ve sloupci skupiny ikon                                 |
| Barva motivu     | určuje barevné schéma nebo téma použité na ikonu ve sloupci IconGroup                                                 |
| Manažer událostí | umožňuje uživatelům definovat událost nebo akci spuštěnou při interakci ikony ve sloupci IconGroup s uživateli        |
| Zakázáno         | stav sloupce IconGroup, určující, zda je zakázán nebo povolen pro interakci uživatele                                 |

-   Hodnocení

představuje ratingy nebo zpětnou vazbu poskytnuté položkám nebo subjektům

-   Markdown

schopnost vykreslovat formátovaný text pomocí Markdown syntaxe ve sloupcích

-   Měna

zpracovávat a formátovat hodnoty měn.

| Vlastnosti                 | Popis                                                                        |
| -------------------------- | ---------------------------------------------------------------------------- |
| Desetinná místa            | počet desetinných míst, která mají být zobrazena pro hodnoty měny ve sloupci |
| Kód měny                   | specifikuje kód nebo zkratku reprezentující měnu použitou ve sloupci         |
| Zobrazit tisíce oddělovače | zda je zobrazen tisícový oddělovač pro velké hodnoty měn                     |

### Metoda

K ovládání komponenty můžete použít jiné komponenty. Podporujeme tyto dva metody:

-   **selectPage**

vyberte všechny řádky na konkrétní stránce v tabulce s stránkami

| Vlastnosti       | Popis                                                |
| ---------------- | ---------------------------------------------------- |
| Konečná hodnota  | číslo stránky nebo index stránky, která bude vybrána |

-   **selektor**

vyberte konkrétní řádek v tabulce

| Vlastnosti            | Popis                                                                                                                                                                 |
| --------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Výchozí vybraný řádek | výchozí vybranou hodnotou řádku může být index řádku, jedinečný identifikátor nebo jakákoli jiná vlastnost, která jednoznačně identifikuje řádek, který má být vybrán |

-   **clearSelection**

vymazat aktuální výběr v tabulce

-   **setFilters**

použít filtry na data z tabulky

| Vlastnosti   | Popis                                   |
| ------------ | --------------------------------------- |
| Filtry       | podmínky filtrování nebo kritéria       |
| Režim filtru | určuje způsob použití filtrů            |

-   **ClearFilters**

vymaže všechny použité filtry v tabulce

-   **setSort**

nastavit třídění pro tabulku

| Vlastnosti  | Popis                                                     |
| ----------- | --------------------------------------------------------- |
| Sloupec     | Sloupce, na které se má použít třídění                    |
| Směr        | směr druhu, který může být "vzestupný" nebo "potomstvo"   |

### Případ použití

Dále ukážeme, jak mapovat data ze složky `**Upstash**` na `**tabulku**`.

#### Krok 1 Přidat akci

Vytvořme tabulku v Upstash s názvem **`Zaměstnane_profile`,** včetně 4 sloupců `name`, `level`, `age`, a `manželství`.

Pak můžeme vytvořit novou akci pro Upstash ze seznamu akcí v ILLA Cloud a pojmenovaný upstash2.

Chcete-li zobrazit všechna data v profilu zaměstnance, vložte textový blok níže v sekci Redis dotaz.

```sql
GET zaměstnanec_profil
```

Vzhledem k tomu, že výstupní data jsou seznamem slovníků, chceme je přeměnit na pole. V transformátoru jej povolte a vložte níže uvedený kód.

```jsx
const jsonString = data[0].result;

// Parse the JSON string into a JavaScript object
const jsonData = JSON.parse(jsonString);

// Access the "employees" property and map it to a new array
const mappedArray = jsonData.employees.map((employee) => ({
  name: employee.name,
  level: employee.level,
  age: employee.age,
  married: employee.married,
}));

return mappedArray;
```

#### Krok 2 Přidat komponenty

Dále můžeme přidat komponentu **`tabulky`** do plátna.

Do pole **`Zdroj údajů`** na stránce Po kliknutí na komponentu vložte `{{upstash2.data}}` pro načtení dat z Upstash.

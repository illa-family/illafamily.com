## <h2 hidden>Co je to Appwrite?</h2>

Zápis aplikací v Illa je integrace databáze, která umožňuje připojení a interakci s databází zápisů aplikací. Appwrite je populární systém pro správu relativních databází s otevřeným zdrojovým kódem, který je znám jeho rychlostí, spolehlivostí a snadností použití.

Díky integraci aplikací v Illa můžete pomocí SQL příkazů dotázat, vkládat, aktualizovat a mazat data z databáze Appwrite . Můžete také vytvořit vlastní SQL dotazy a spustit je v Ille. Díky tomu lze snadno sestavovat aplikace založené na datech a dashboardy, které zobrazují data z databáze Appwrite v reálném čase.

Tento kurz nastiňuje proces vytváření Admin panelu pomocí ILLA Tvůrce a Appwrite v několika jednoduchých krocích. ILLA je nízkokódová platforma pro vývojáře, která umožňuje rychlý vývoj a zavádění interních nástrojů. Umožňuje vytvářet stránky přetažením a vypouštěním komponent uživatelského rozhraní, připojením k databázi nebo API a psaním JavaScript. Chcete-li se dozvědět více o Appwrite, navštivte jejich webové stránky na <https://Appwrite.io/> Pojďme na začátku!

### Nastavte si konec backu v Appwrite

Po přihlášení do vašeho účtu Appwrite na nástěnce Appwrite klikněte na `Vytvořit projekt`a nastavte jméno na `První projekt`.

![app_create_project](https://cdn.illacloud.com/official-website/img/docs/app_create_project.png)

Pak můžeme přejít na stránku projektu. Aplikace poskytuje více platforem pro vytváření vaší aplikace i integrace.

![app_platforms](https://cdn.illacloud.com/official-website/img/docs/app_platforms.png)

Zde si jako příklad zvolíme webovou aplikaci.

Po vytvoření našeho projektu webové aplikace můžeme přejít na databázovou stránku, protože zatím používáme Appwrite pouze jako integraci databáze.

![app_dashboard](https://cdn.illacloud.com/official-website/img/docs/app_dahsboard.png)

Pak můžeme vytvořit databázi a vytvořit kolekce v databázi

![app_create_database](https://cdn.illacloud.com/official-website/img/docs/app_create_database.png)
Odtud můžeme vytvářet atributy a data ve sbírce.

![app_create_data](https://cdn.illacloud.com/official-website/img/docs/app_create_data.png)

Abychom mohli získat dokumenty podle atributů, musíme vytvořit odpovídající indexy pro každý atribut.

![app_create_index](https://cdn.illacloud.com/official-website/img/docs/app_create_index.png)

Tato databáze je nyní připravena k integraci s Illa.

### Vytvořit zápis aplikací

Po přihlášení do vašeho účtu Illa existují dva způsoby, jak vytvořit zdroj v Illa.

-   Vytvořit ve zdrojích

Přihlaste se ke svému účtu Illa, vyberte `Zdroje` v horní části stránky a klikněte na tlačítko `Vytvořit nové`.

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

Vyberte "Appwrite" ze seznamu databází.

![action_list](https://cdn.illacloud.com/official-website/img/docs/action_list.png)

Připojte se k databázi s požadovanými parametry popsanými v části `Nastavení připojení` níže.

Klepněte na tlačítko `Test Connection` a zjistěte, zda se můžeme úspěšně připojit k databázi. Pokud ano, klikněte na tlačítko "Uložit zdroje", jiné, dvakrát zkontrolujte jméno hostitele, port, uživatelské jméno a heslo je správné.

Po vytvoření zdroje se bude zobrazovat připravený zápis aplikací.

![app_ex_ready](https://cdn.illacloud.com/official-website/img/docs/app_ex_ready.png)

-   Vytvořit ve staviteli

Přihlaste se ke svému účtu Illa, vytvořte projekt na stránce `App` a přejděte do `Seznam akcí` v dolní části stránky. Klikněte na `nové`, pak vyberte `Appwrite` ze seznamu databází. Poté se připojte k databázi s požadovanými parametry popsanými v části `Nastavení připojení` níže.

Klepněte na tlačítko `Test Connection` a zjistěte, zda se můžeme úspěšně připojit k databázi. Pokud ano, klikněte na tlačítko "Uložit zdroje", jiné, dvakrát zkontrolujte jméno hostitele, port, uživatelské jméno a heslo je správné.

### Nastavení připojení

Zde musíme poskytnout informace pro připojení k databázi Appwrite .

![app_config](https://cdn.illacloud.com/official-website/img/docs/app_config.png)

| Vlastnosti      | Požadováno | Popis                                                                                                         |
| --------------- | ---------- | ------------------------------------------------------------------------------------------------------------- |
| Název           | vyžadováno | Název zdroje při vytváření akcí v ILLA.                                                                       |
| Hostitel        | vyžadováno | URL nebo IP adresa pro vaši databázi                                                                          |
| ID databáze     | vyžadováno | jedinečný identifikátor přiřazený každé databázi vytvořené v Appwrite                                         |
| ID projektu     | vyžadováno | jedinečný identifikátor, který reprezentuje celý váš projekt Appwrite                                         |
| Klíč API        | vyžadováno | tajný token, který slouží jako forma autentizace při zadávání API požadavků na Appwrite server                |

Pro nalezení požadovaného `Host`, `Database ID`, `Project ID`, a `API klíč` v Appwrite, nejprve přejdeme na tlačítko `nastavení` v levém dolním rohu stránky projektu.

Kopírování `project ID` pro `project ID` a `API Endpoint` pro `Host` v konfiguraci Appwrite v Ille.

![app_setting](https://cdn.illacloud.com/official-website/img/docs/app_setting.png)

Pak můžeme kliknout na tlačítko "Zobrazit API klíč" na stejné stránce. Pokud jste nevytvořili API pro tento projekt, klikněte na tlačítko "Vytvořit API klíč" a vyberte pouze složku "Database", protože to je to, co budeme používat z appwrite.

![app_api_key](https://cdn.illacloud.com/official-website/img/docs/app_api_key.png)

Zkopírujte `API Key Secret` pro `API key` v konfiguraci Illa. Lze ji zkopírovat pro tyto dva body.

![app_key_secret](https://cdn.illacloud.com/official-website/img/docs/app_key_secret.png)

Zbývá nám `ID databáze`, které najdeme na stránce `Databáze`. Kopírování a vložení do konfigurace Appwrite v Illa.

![app_database](https://cdn.illacloud.com/official-website/img/docs/app_database.png)

Konečný produkt by měl vypadat podobně.

![app_final_config](https://cdn.illacloud.com/official-website/img/docs/app_final_config.png)

### Konfigurovat Appwrite

#### Přehled

| Název metody    | Popis                                                                                        |
| --------------- | -------------------------------------------------------------------------------------------- |
| Metoda          | Vytvořit dokument, získat dokument, aktualizovat dokument, smazat dokument, seznam dokumentů |
| ID kolekce      | jedinečný identifikátor přiřazený sbírce databáze                                            |
| Transformátor   | transformace dat do stylu, který se vám líbí pomocí JavaScriptu                              |

#### Vytvořit dokument

vytvořit nový dokument ve sbírce

 Input 

| Vlastnosti   | Požadováno | Popis                                                                         |
| ------------ | ---------- | ----------------------------------------------------------------------------- |
| ID dokumentu | vyžadováno | jedinečný identifikátor přiřazený dokumentu o shromažďování údajů v databázi  |
| Údaje        | volitelné  | objekt obsahující pole a hodnoty nového dokumentu.                            |

Například vytvořit nový dokument ve sbírce se dvěma poli - `name` a `age`, můžete použít 1 jako `document ID` a `{{ {"Name": "David", "Age": 18}}}` pro `data`.

 Výstup 

objekt odezvy, který obsahuje ID nově vytvořeného dokumentu, jakož i další metadata, jako je datum a čas vytvoření.

![app_create_code](https://cdn.illacloud.com/official-website/img/docs/app_create_code.png)

#### Získat dokument

načíst specifický dokument z kolekce v databázi

 Input 

| Vlastnosti   | Požadováno | Popis                                                                        |
| ------------ | ---------- | ---------------------------------------------------------------------------- |
| ID dokumentu | vyžadováno | jedinečný identifikátor přiřazený dokumentu o shromažďování údajů v databázi |

Například dokument s ID `1`.

 Výstup 

objekt odpovědi, který obsahuje údaje požadovaného dokumentu

#### Aktualizovat dokument

aktualizuje údaje o existujícím dokumentu ve sbírce.

 Input

#### Seznam dokumentů

načíst více dokumentů z kolekce v databázi

 Input 

| Vlastnosti    | Požadováno | Popis                                                                           |
| ------------- | ---------- | ------------------------------------------------------------------------------- |
| Filtr         | volitelné  | definuje podmínky pro filtrování dokumentů na základě konkrétních polí a hodnot |
| Řadit podle   | volitelné  | Upřesněte pole, do kterého by měly být výsledné dokumenty seřazeny              |
| Limit         | volitelné  | omezit počet dokumentů vrácených v souboru výsledků                             |

Například uvést všechny dokumenty s názvem = James.

 Výstup 

Soubor dokumentů, které vyhovují filtru.

![app_list_data](https://cdn.illacloud.com/official-website/img/official-site/intergration/app_list_data.png)

Použij `{{Appwrite1.data[0].documents}}` k získání vrácené hodnoty.

![app_list_code](https://cdn.illacloud.com/official-website/img/docs/app_list_code.png)

## <h2 hidden>Jaký je časový rozsah?</h2>

komponenta časového intervalu je komponenta uživatelského rozhraní, která uživatelům umožňuje vybrat si čas z 24hodinového rozhraní, typicky používané ve formulářích nebo rozhraních zadávání dat, kde uživatel potřebuje zadat počáteční a koncový časový rozsah.

Je-li součástí časového intervalu přidána do plátna, zobrazí se vedle sebe dvě rozhraní po 24 hodinách. Jedno rozhraní se používá k výběru počátečního času, zatímco druhé se používá k výběru koncového času. Uživatel může upravit čas pohybem kurzoru myší nebo použitím šipek nahoru a dolů.

-   **setStartTime**

Pro nastavení počáteční hodnoty data například {{"value1"}}

### Případ použití

Dále ukážeme, jak mapovat data ze zdroje dat do časového rozmezí a pomocí tlačítka nastavit koncovou hodnotu na aktuální čas. 

#### Krok 1 Přidat akci

Vytvořme tabulku v REST API s názvem **`promoteCodes`,** včetně 6 sloupců: **`code`**, **`id`**, **`createdAt`**, **`expiredAt`**, **`startedAt`, `updatedAt`**. Použijeme starttedIn a vypršela doba platnosti pro použití vzorku. 

Pak můžeme vytvořit novou akci pro REST API ze seznamu akcí, seznam všech dat v promoteCodes a jménem restapi1. 

Klikněte na **`Uložit`** a **`Spustit`** pro aktivaci této akce.

#### Krok 2 Přidat komponenty

Dále můžeme přidat komponentu **`časového pásma`** a komponentu **`tlačítka`**. Pro výchozí počáteční hodnotu složky rozsahu dat používáme začátkuV okamžiku prvního propagace kódu od ostatních api dat.

```sql
{{restapi1.data.promoteCodes[0].startedAt}}
```

Obdobně používáme pro výchozí časovou hodnotu vypršela doba platnosti první propagace kódu z ostatních api dat.

```sql
{{restapi1.data.promoteCodes[0].expiredAt}}
```

Také jsme označili tlačítko jako "nyní", jak je znázorněno níže.

![time_range_0](https://cdn.illacloud.com/official-website/img/docs/time_range_0.png)

#### Krok 3 Konfigurace komponenty

Pro komponentu **`button`** můžeme nakonfigurovat tak, aby byla nastavena konečná hodnota data v komponentě **`range`** k dnešnímu datu.

1.  V obsluze události Editace vyberte **`Control component`** v akci, vyberte komponentu **`časového rozsahu`**, kterou chcete aktualizovat jako cíl události.
2.  Vyberte akci **`setEndTime`** a nastavte **`Čas ukončení`** jako `{{currentUserInfo.updatedAt}}` 

![time_range_config](https://cdn.illacloud.com/official-website/img/docs/time_range_config.png)

#### Krok 4 Zkouška

Nyní, když klepnete na tlačítko "Nyní", čas ukončení by se měl změnit na dnešní datum. Pokud jde o tady, máme 20:03:05, ale může to být jiné, když to testujete.

![time_range_1](https://cdn.illacloud.com/official-website/img/docs/time_range_1.png)

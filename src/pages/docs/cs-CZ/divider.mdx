## <h2 hidden>Co je to děliče?</h2>

V ILLA Cloud je dělič součástí vizuálního prvku, který pomáhá oddělovat nebo dělit části obsahu v rámci webové aplikace. Jedná se o jednoduchou vodorovnou čáru nebo vertikální čáru, kterou lze vložit mezi různé oddíly nebo komponenty s cílem zlepšit vizuální organizaci a srozumitelnost uživatelského rozhraní.

### Vlastnosti

| Vlastnosti      | Popis                                                                          |
| --------------- | ------------------------------------------------------------------------------ |
| Text            | přidat další informace k dělič komponentě                                      |
| Text align      | určuje horizontální zarovnání textu v komponentě oddělovače                    |
| Hidden          | skrýt komponentu oddělovače z uživatelského rozhraní                           |
| Velikost textu  | upravit velikost písma textu v komponentě oddělovače                           |

### Údaje

Komponenta má některá běžně používaná data, která lze v aplikaci volat prostřednictvím **`{{componentName.propertyName}}`**.

| Název vlastnosti    | Popis                                                                   |
| ------------------- | ----------------------------------------------------------------------- |
| barevné schéma      | definuje barvu textu v komponentě                                       |
| zobrazované jméno   | představuje název nebo štítek přiřazený komponentě                      |
| fs                  | souborový systém (fs) spojený s oddělovačem                             |
| hidden              | logická hodnota, která určuje, zda je komponenta viditelná nebo skrytá  |
| text                | hodnota textu                                                           |

### Případ použití

Můžete ji umístit kdekoliv, kde chcete stránku rozdělit na dvě sekce. Zde ukážeme dělicí čáru se třemi různými doplňky textu.

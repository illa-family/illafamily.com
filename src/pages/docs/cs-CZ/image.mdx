## <h2 hidden>Co je to obrázek?</h2>

Složka obrázku v ILLA Cloud je univerzální nástroj, který uživatelům umožňuje přidávat a zobrazovat obrázky ve svých aplikacích. Poskytuje různé funkce pro zlepšení vizuálních zkušeností pro uživatele.

### Vlastnosti

| Vlastnosti             | L 343, 22.12.2009, s. 1).                                                                                                          |
| ---------------------- | ---------------------------------------------------------------------------------------------------------------------------------- |
| Zdroj obrázku          | specifikovat zdroj obrázku, který má být zobrazen                                                                                  |
| Alternativní text      | alternativní text pro obrázek. Zobrazuje se v případě, že obrázek nelze načíst nebo pro uživatele se zrakovým postižením           |
| Typ měřítka            | jak je obraz zmenšen v rozměrech komponenty, včetně zásobníku, krytu, plnění, není, měřítka.                                       |
| Zpracovatel událostí   | umožňuje uživatelům definovat akce spouštěné uživatelskými interakcemi s obrázkem                                                  |
| Šipka nástroje         | přidat popisný popisek nebo popisný text, který se zobrazí, když se uživatel převléká nad obrázkem                                 |
| Hidden                 | řízení viditelnosti součásti obrazu.                                                                                               |
| Styles                 | umožňuje uživatelům přizpůsobit jeho vzhled. Uživatelé mohou použít styly, jako je poloměr                                         |

### Metoda

K ovládání komponenty můžete použít jiné komponenty. Podporujeme následující metody:

-   **setImageUrl**

dynamicky změnit zdroj nebo URL obrázku.

| Vlastnosti   | Popis             |
| ------------ | ----------------- |
| URL obrázku  | URL zdroje obrazu |

### Manažer událostí

| Událost | Popis                                                                     |
| ------- | ------------------------------------------------------------------------- |
| Click   | provést konkrétní akce nebo provést kód po kliknutí na komponentu obrázku |

### Údaje

Komponenta má některá běžně používaná data, která lze v aplikaci volat prostřednictvím **`{{componentName.propertyName}}`**.

| Název vlastnosti    | Popis                                                                                                                                      |
| ------------------- | ------------------------------------------------------------------------------------------------------------------------------------------ |
| altText             | textová alternativa, která popisuje obsah nebo účel obrazu                                                                                 |
| zobrazované jméno   | Hodnota řetězce, která představuje název nebo název obrázku.                                                                               |
| události            | Pole nebo objekt, který definuje manipulátory události spojené s komponentou obrázku, například klikněte                                   |
| hidden              | Logická hodnota, která určuje, zda je komponenta obrázku skrytá v uživatelském rozhraní.                                                   |
| imageSrc            | zdroj nebo umístění obrázku, který má být zobrazen v komponentě.                                                                           |
| objektFit           | jak je obraz namontován nebo rozvržen v rozměrech komponenty                                                                               |
| radius              | aplikovat na součást obrázku zaoblené rohy.                                                                                                |
| tooltipText         | hodnota řetězce, která představuje nápovědu nebo se zobrazí text, který se má zobrazit, když se uživatel převrátí nad komponentou obrázku. |

Příklad: {{image1.displayName}}

### Případ použití

Dále budeme demonstrovat, jak ovládat zobrazení obrázku tlačítkem.

#### Krok 1 Přidat komponenty

Přidejte komponentu **`Image`** a komponentu **\`tlačítko** do plátna.

Můžete nastavit `zdroj obrázku` komponenty obrázku na libovolnou adresu obrázku, kterou chcete zobrazit. Zde použijeme výchozí obrázek jako počáteční příklad a označte tlačítko jako "Změnit", jak je zobrazeno níže

![image_layout](https://cdn.illacloud.com/official-website/img/official-site/components/image_layout.jpg)

#### Krok 2 Konfigurace komponenty

Pro komponentu **`button`** můžeme nakonfigurovat tak, aby byl zdroj obrázku komponenty **`image`** nastaven na novou adresu.

1.  Klikněte na `**tlačítko**` komponentu, abyste otevřeli její stránku pro prohlížení. Pod **`Ovladač událostí`**, klikněte `**+ Nový**`.
2.  V obsluze události Editace vyberte **`Control component`** v akci, a vyberte **`image1`** komponentu, kterou chcete změnit jako cíl události.
3.  Vyberte **`setImageUrl`** akci
4.  Vložte novou adresu obrázku do pole **`Value`**

![image_config](https://cdn.illacloud.com/official-website/img/official-site/components/image_config.jpg)

#### Krok 3 Zkouška

Teď, když klepnete na tlačítko "Přehrát", obrázek by měl přepnout

![image_test](https://cdn.illacloud.com/official-website/img/official-site/components/image_test.gif)

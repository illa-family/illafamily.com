## <h2 hidden>Vlastní API ověření</h2>

ILLA ve výchozím nastavení podporuje mnoho ověřovacích standardů (viz zde pro více informací). 
ILLA má navíc vlastní autentizační režim, který umožňuje kombinovat více kroků podporujících prakticky jakoukoliv API autentizaci. 
Tyto metody ověřování mohou být použity jak pro REST, tak pro GrafQL.

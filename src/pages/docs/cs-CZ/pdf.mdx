## <h2 hidden>Co je to PDF?</h2>

PDF komponenta je funkce nebo prvek, který umožňuje uživatelům prohlížet a interagovat s PDF dokumenty v rámci aplikace. Poskytuje funkce pro vykreslování, navigaci a manipulaci s PDF soubory a umožňuje uživatelům bezproblémový přístup a práci s PDF.

### Vlastnosti

| Vlastnosti              | Popis                                                                                         |
| ----------------------- | --------------------------------------------------------------------------------------------- |
| Zdroj PDF               | specifikujte zdroj dokumentu pdf, který má být zobrazen                                       |
| Zobrazit panel nástrojů | ovládá viditelnost panelu nástrojů v rámci PDF součásti                                       |
| Šipka nástroje          | přidat popisný popisek nebo popisný text, který se zobrazí, když se uživatel převrátí nad pdf |
| Měřítko stránky na      | určuje chování škálování PDF stránek v rámci komponenty                                       |
| Hidden                  | určuje viditelnost komponenty PDF                                                             |

### Metoda

K ovládání komponenty můžete použít jiné komponenty. Podporujeme následující metody:

-   **setFileUrl**

dynamicky změnit zdroj nebo URL souboru pdf.

| Vlastnosti | Popis          |
| ---------- | -------------- |
| PDF zdroj  | pdf URL zdroje |

### Údaje

Komponenta má některá běžně používaná data, která lze v aplikaci volat prostřednictvím **`{{componentName.propertyName}}`**.

| Název vlastnosti    | Popis                                                                                                                                    |
| ------------------- | ---------------------------------------------------------------------------------------------------------------------------------------- |
| zobrazované jméno   | Hodnota řetězce, která představuje název nebo název pdf.                                                                                 |
| Výška               | vertikální rozměr nebo velikost složky PDF                                                                                               |
| hidden              | Logická hodnota, která určuje, zda je komponenta pdf skrytá v uživatelském rozhraní.                                                     |
| mód měřítka         | specifikuje, jak je dokument PDF zmenšen nebo uzpůsoben v dostupném rozsahu zobrazení komponenty                                         |
| showToolBar         | zda je zobrazena nástrojová lišta nebo ovládací panel spolu s PDF komponentou                                                            |
| tooltipText         | hodnota řetězce, která představuje nápovědu nebo se zobrazí text, který má být zobrazen, když se uživatel převlékne nad komponentou pdf. |
| URL                 | zdroj PDF souboru, který se zobrazí v komponentě                                                                                         |
| width               | horizontální rozměr nebo velikost složky PDF                                                                                             |

Příklad: {{pdf1.displayName}}

### Případ použití

Dále budeme demonstrovat, jak ovládat displej pdf tlačítkem.

#### Krok 1 Přidat komponenty

Přidejte komponentu **`PDF`** a komponentu **\`tlačítko** do plátna.

![pdf_layout](https://cdn.illacloud.com/official-website/img/docs/pdf_layout.jpg)

#### Krok 2 Konfigurace komponenty

Pro komponentu **`button`** můžeme nastavit zdroj obrazu komponenty **`PDF`** na pdf adresu.

1.  Klikněte na `**tlačítko**` komponentu, abyste otevřeli její stránku pro prohlížení. Pod **`Ovladač událostí`**, klikněte `**+ Nový**`.
2.  V obsluze události Editace vyberte **`Control component`** v akci, a vyberte komponentu **`PDF`**, kterou chcete změnit jako cíl události.
3.  Vyberte **`setFileUrl`** akci
4.  Vložte novou adresu obrázku do pole **`Value`**

![pdf_config](https://cdn.illacloud.com/official-website/img/docs/pdf_config.jpg)

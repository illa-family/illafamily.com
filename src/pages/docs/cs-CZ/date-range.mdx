## <h2 hidden>Jaký je rozsah dat?</h2>

Komponenta datumu je komponenta UI, která umožňuje uživatelům vybrat rozsah dat z kalendáře Obvykle se používá ve formulářích nebo rozhraních zadávání dat, kde uživatel potřebuje zadat rozsah počátečního a koncového data.

Když je komponenta datového rozsahu přidána do plátna, zobrazuje vedle sebe dvě rozhraní kalendáře. Jedno rozhraní kalendáře se používá k výběru data zahájení, zatímco druhé se používá k výběru data ukončení. Uživatel se může pohybovat přes kalendáře kliknutím na předchozí nebo další šipky, aby se přesunul na jiný měsíc, nebo kliknutím na konkrétní datum pro jeho výběr. Vybraná data budou zobrazena ve vstupních polích.

### Případ použití

Dále ukážeme, jak mapovat data ze zdroje dat do aktuálního rozsahu a pomocí tlačítka nastavit koncovou hodnotu na dnešní datum. 

#### Krok 1 Přidat akci

Vytvořme tabulku v REST API s názvem **`promoteCodes`,** včetně 6 sloupců: **`code`**, **`id`**, **`createdAt`**, **`expiredAt`**, **`startedAt`, `updatedAt`**. Použijeme starttedIn a vypršela doba platnosti pro použití vzorku. 

Pak můžeme vytvořit novou akci pro REST API ze seznamu akcí, seznam všech dat v promoteCodes a jménem restapi1. 

Klikněte na **`Uložit`** a **`Spustit`** pro aktivaci této akce.

#### Krok 2 Přidat komponenty

Dále můžeme přidat komponentu **`rozsah datumů`** a komponentu **`tlačítka`**. Pro výchozí počáteční hodnotu datumu složky použijeme začátekV době první propagace kódu od ostatních api dat.

```sql
{{restapi1.data.promoteCodes[0].startedAt}}
```

Podobně, pro výchozí koncovou hodnotu používáme vypršení doby první propagace kódu z ostatních api dat.

```sql
{{restapi1.data.promoteCodes[0].expiredAt}}
```

Také jsme označili tlačítko jako "Aktualizovat", jak je uvedeno níže.

![date_range_0](https://cdn.illacloud.com/official-website/img/docs/date_range_0.png)

#### Krok 3 Konfigurace komponenty

Pro komponentu **`button`** můžeme nakonfigurovat tak, aby byla nastavena konečná hodnota data v komponentě **`range`** k dnešnímu datu.

1.  V obsluze události Editace vyberte **`Control component`** v akci, vyberte komponentu **`rozsah datumů`**, kterou chcete aktualizovat jako cíl události.
2.  Vyberte akci **`setEndValue`** a nastavte **`Datum ukončení`** jako `{{currentUserInfo.updatedAt}}` 

![date_range_config](https://cdn.illacloud.com/official-website/img/docs/date_range_config.png)

#### Krok 4 Zkouška

Nyní, když klepnete na tlačítko 'Aktualizovat', datum ukončení by se mělo změnit na dnešní datum. Pokud jde o tuto otázku, máme 2023-5-6, ale může to být jiné, když ji testujete.

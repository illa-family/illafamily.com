## <h2 hidden>Co je timeline?</h2>

Časová složka je grafické znázornění událostí nebo datových bodů uspořádaných chronologicky podél vodorovné nebo svislé osy. Umožňuje uživatelům lineárně vizualizovat a procházet řadou událostí nebo milníků.

### Vlastnosti

Dostupné vlastnosti Timelinu. JavaScript může být napsán pro čtení nebo změnu informací o komponentách.

| Vlastnosti    | Popis                                                                                      |
| ------------- | ------------------------------------------------------------------------------------------ |
| Položky       | shromažďování jednotlivých událostí nebo datových bodů, které jsou zobrazeny na časové ose |
| Směr          | orientace nebo rozložení časové složky (horizontální nebo vertikální)                      |
| Nevyřízeno    | stav položky v časové ose                                                                  |
| Výška         | vertikální velikost nebo výška časové složky                                               |
| Hidden        | zda je časová složka viditelná nebo skrytá                                                 |

### Metody

K ovládání komponenty můžete použít jiné komponenty. Podporujeme tyto dva metody:

-   **setValue**

Pro nastavení vstupní hodnoty textové oblasti například {{"value1"}}

| Vlastnosti   | Popis       |
| ------------ | ----------- |
| Hodnota      | Input value |

-   **clearValue**

Možnost vymazat hodnotu vybrané komponenty

### Údaje

Komponenta má některá běžně používaná data, která lze v aplikaci volat prostřednictvím **`{{componentName.propertyName}}`**.

| Název vlastnosti      | Popis                                                                                                   |
| --------------------- | ------------------------------------------------------------------------------------------------------- |
| směr                  | orientace nebo rozložení časové osy                                                                     |
| dynamická Výška       | schopnost přizpůsobit výšku časové osy dynamicky na základě jejího obsahu nebo jiných faktorů.          |
| zobrazované jméno     | název této komponenty (tj. timeline1)                                                                   |
| hidden                | zda je časová složka viditelná nebo skrytá                                                              |
| položky               | shromažďování jednotlivých událostí nebo datových bodů zobrazených na časové ose                        |
| čeká                  | stav položky v časové ose                                                                               |
| směr změny velikosti  | směr, ve kterém může uživatel změnit nastavení časové osy                                               |

### Případ použití

Dále budeme demonstrovat, jak mapovat data ze zdroje dat do složky časového rámce.

#### Krok 1 Přidat akci

Vytvořme tabulku v Supabase s názvem **`timeline`,** obsahující pouze 1 sloupce: milníky. Existují 4 milníky, které chceme mapovat do složky časové osy.

Pak můžeme vytvořit novou akci pro Supabase ze seznamu akcí a pojmenovaný supabasedb1.

Chcete-li zobrazit všechna data v milníku, vložte textový blok níže v sekci SQL dotaz.

```sql
zvolte * z časového plánu;
```

Vzhledem k tomu, že výstupní data jsou seznamem slovníků, chceme je přeměnit na pole. V transformátoru jej povolte a vložte níže uvedený kód.

```jsx
const transformedData = data.map(obj) => Object.values(obj)[0]);
return transformedData;
```

Klikněte na **`Uložit`** a **`Spustit`** pro aktivaci této akce.

![timeline_data](https://cdn.illacloud.com/official-website/img/official-site/components/timeline_data.jpg)

#### Krok 2 Přidat komponenty

Dále můžeme přidat **`timeline`** složku do plátna.

Do pole `**Položky**` na stránce Po kliknutí na komponentu, vložte `{{supabasedb1.data}}` pro načtení dat z supabasy.

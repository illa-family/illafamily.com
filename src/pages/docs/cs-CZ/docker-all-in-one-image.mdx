## <h2 hidden>Docker vně jednoho obrázku</h2>

### L 343, 22.12.2009, s. 1).

Build illa utils all-in-one image and run it on your machine. Please checkout this repo for more info [build-all-in-one-image](https://github.com/illacloud/build-all-in-one-image).

Pozn.:

Důrazně jsme doporučili nasazení našich nástrojů pro automatické nasazení[illa-cli](https://github.com/illacloud/illa).

### Run an Official Image

Just execute:

```jsx
docker run illasoft/illa-builder:latest
```

this command will pull illasoft official image and run it on your docker environment.

And log in with the default username **`root`** and password **`password`**.

### Build Image Manually and Run

Install GNU make (or execute shell scripts in the scripts folder manually).

For build illa all-in-one image and run, type:

```jsx
make build
```

or execute command:

```jsx
docker build ./ -f ./Dockerfile -t illa-builder:local
```

Then run your local build image by:

```jsx
make run -test
```

or 

```jsx
docker run -d -p 80:2022 --name illa_builder_local -v ~/illa-database:/opt/illa/database -v ~/illa-drive:/opt/illa/drive illa-builder:local
```

### Zastavit a odebrat kontejner

```jsx
docker stop illa_builder_local
```

## <h2 hidden>Umístit úvod</h2>

### Cloud

Zkuste jedno kliknutí použít v ILLA Cloud pro nejjednodušší zážitek.

### Samohostitelský

Můžete hostovat lokálně s ILLA CLI, Docker compose, k8s helm atd. Opakovaně jsme optimalizovali vlastní hostitelský proces, který vám během několika minut umožní vytvořit vlastní hostitele a softwarové instalace. Ve srovnání s mrakem můžete používat produkty ILLA bezpečněji a spolehlivěji, a můžete použít ILLA Builder jednoduše navštívit místo nasazení po hostování a nasazení Stavitele lokálně.

Více informací o vlastním hostiteli naleznete v Docker compose, k8 helm zaváděcích stránkách.
Pokud máte problémy s vlastním nasazením, kontaktujte nás prosím na GitHub nebo business@illasoft.com.

## <Heading hidden>API-Authentifizierung</Heading>

ILLA kann fast jede Art von Authentifizierung unterstützen. Wir haben mit verschiedenen Unternehmen zusammengearbeitet, um alle Standardauthentifizierungen zu unterstützen und haben auch Methoden entwickelt, um kundenspezifische Authentifizierungsmethoden zu unterstützen, die wir nicht nativ unterstützen.

Sie können statische Authentifizierungs-Tokens verwenden oder ein Authentifizierungssystem erstellen, das den Endbenutzer jedes Mal nach seinen Zugangsdaten fragt, wenn er auf die Anwendung zugreift.

Diese Seite gibt Ihnen Anweisungen, wie Sie Ihre API authentifizieren können. Wenn Sie Fragen haben oder Hilfe benötigen, Sie können sich mit unserem Support-Team in Verbindung setzen, indem Sie die Intercom-Blble auf dieser Seite oder das Hilfe-Icon im Editor verwenden.

## <Heading hidden>Ressource</Heading>

Verbinden Sie sich mit Ihrer Datenquelle und rufen Sie sie im ILLA Builder auf. Wir stellen Ihnen anpassbare Verbindungsmodule für verschiedene Datenquellen zur Verfügung. Sie können in kurzer Zeit eine stabile Verbindung zwischen Ihren Daten und ILLA herstellen und so Ihren Datenzugriffsprozess erheblich vereinfachen.

Nachdem Sie eine Ressource erfolgreich erstellt haben, können Sie sie bei jeder App in ILLA anrufen. Sie können Ihre Ressourcen auch mit Ihren Mitentwicklern teilen und die Effizienz des Entwickelns von Apps verbessern.

### Unterstützter Ressourcentyp

ILLA unterstützt eine Reihe wichtiger Datenbanken, wie MySQL und Posta, sowie APIs wie die Rest API und GraphQL.

### Ressource erstellen

Ressourcen stellen Verbindungen zwischen ILLA Builder und Ihren Datenquellen dar. Erstellen Sie eine Ressource durch folgende Schritte:

#### 1.  Navigieren Sie zum Reiter "Ressourcen" in der Navigationsleiste der Startseite und klicken Sie auf "Neu erstellen"

<img
  src='https://cdn.illacloud.com/official-website/img/docs/re_e_1.png'
  alt=""
/>

#### 2.  Wählen Sie hier Ihren Datentyp

<img
  src='https://cdn.illacloud.com/official-website/img/docs/qs_e_2.png'
  alt=""
/>

#### 3.  Verfeinern Sie Ihre Datenquellinformationen und klicken Sie auf "Ressource erstellen" und beenden Sie den Prozess

<img
  src='https://cdn.illacloud.com/official-website/img/docs/re_e_2.png'
  alt=""
/>

Zusätzlich können Sie Ressourcen in der Aktionsliste unter App-Editor erstellen

> ### Verbindung testen
>
> Wenn Sie Ressourcen erstellen oder anzeigen Sie können den Test-Button im Ressourcenfeld anklicken, um zu überprüfen, ob die Datenquelle erfolgreich verbunden ist.

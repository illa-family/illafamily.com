## <h2 hidden>Was ist Upstash?</h2>

Upstash ist ein Cloud-native serverless Service mit Redis kompatibel, der eine einfache Möglichkeit bietet, Redis Funktionalität in Anwendungen zu integrieren. Im Kontext von ILLA Cloud ermöglicht die Upstash-Integration die Nutzung der Funktionen und Funktionen von Upstash innerhalb der ILLA Cloud.

Dieses Tutorial beschreibt den Prozess der Erstellung eines Admin-Panels mit Hilfe von ILLA Builder und Upstash in wenigen einfachen Schritten. ILLA Cloud ist eine Low-Code-Plattform für Entwickler, die die schnelle Entwicklung und den Einsatz von internen Tools ermöglicht. Es ermöglicht die Erstellung von Seiten durch Ziehen und Ablegen von UI-Komponenten, Verbindung zu jeder Datenbank oder API, und das Schreiben von JavaScript. Um mehr über Upstash, besuchen Sie ihre Website unter https://upstash.com/(https://upstash.com/) Lass uns beginnen!

### Backend auf Upstash einrichten

Erstellen Sie ein Konto auf Upstash. Sie können sich alternativ mit Ihren Amazon, Github oder Google-Konten anmelden.

Nachdem du dich bei `Upstash` in dein Konto eingeloggt hast, klicke auf die Schaltfläche `Datenbank erstellen` in Redis.

![up_home](https://cdn.illacloud.com/official-website/img/docs/up_home.jpg)

Klicken Sie auf dem `Upstash` Dashboard auf `Datenbank erstellen`.

![up_create_db](https://cdn.illacloud.com/official-website/img/docs/up_create_db.jpg)

Hinweis: Sie können nur eine einzige Datenbank im freien Bereich haben.

Für die Datenbankkonfiguration können Sie Ihre Datenbank so benennen, wie Sie wollen. Hier nennen wir es `db1`. Wähle die `primäre Region` und `read region` basierend auf deiner Schreib- und Lese-Position aus. Aktiviere `TSL/SSL`. Es ist optional, um die Räumung zu aktivieren.

![up_db_config](https://cdn.illacloud.com/official-website/img/docs/up_db_config.jpg)

Klicke auf den **`Erstellen`** Button, wir haben jetzt eine Datenbank bereit zu benutzen.

### Upstash in der ILLA Cloud erstellen

Es gibt zwei Möglichkeiten, nach der Anmeldung in Ihrem Illa-Konto eine Ressource in Illa zu erstellen.

-   In Ressourcen erstellen

Melde dich mit deinem Illa-Konto an, wähle `**Ressourcen**` oben auf der Seite und klicke auf `**Neues erstellen**` Button.

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

Wähle **`Upstash`** aus der Datenbankliste.

![action_list_0](https://cdn.illacloud.com/official-website/img/docs/action_list_0.jpg)

Verbinden Sie sich mit den erforderlichen Parametern, die in `Verbindungseinstellungen` unten beschrieben sind.

Klicke auf `**Verbindung testen**` um zu sehen, ob wir erfolgreich mit der Datenbank verbinden können. Wenn ja, klicke auf **`Ressourcen speichern`**, ansonsten überprüfe doppelt, ob der Hostname, der Port, der Benutzername und das Passwort korrekt sind.

Nach dem Erstellen einer Ressource wird der bereitgestellte `Upstash` wie angezeigt angezeigt.

![up_ex_ready](https://cdn.illacloud.com/official-website/img/docs/up_ex_ready.jpg)

-   Im Builder erstellen

Melde dich mit deinem Illa Account an, erstelle ein Projekt in Illa Builder auf der `**App**` Seite und navigieren Sie zur **`Action List`** am unteren Ende der Seite. Klicke auf `**new**`, dann wähle **`Upstash`** aus der Datenbankliste. Dann verbinden Sie sich mit der Datenbank mit den erforderlichen Parametern, die in `Verbindungseinstellungen` unten beschrieben sind.

Klicke auf `**Verbindung testen**` um zu sehen, ob wir erfolgreich mit der Datenbank verbinden können. Wenn ja, klicke auf **`Ressourcen speichern`**, ansonsten überprüfe doppelt, ob der Hostname, der Port, der Benutzername und das Passwort korrekt sind.

### Verbindungseinstellungen

Hier müssen wir Informationen zur Verbindung mit der Upstash-Datenbank bereitstellen.

![up_config](https://cdn.illacloud.com/official-website/img/docs/up_config_rev.jpg)

| Eigenschaften | Benötigt | Beschreibung                                                                                                                            |
| ------------- | -------- | --------------------------------------------------------------------------------------------------------------------------------------- |
| Name          | benötigt | Der Name für die Ressource beim Erstellen von Aktionen in der ILLA.                                                                     |
| Hostname      | benötigt | Die URL oder IP-Adresse für Ihre Datenbank                                                                                              |
| Port          | benötigt | Die Portnummer des Servers, die Sie verwenden sollten, um sich zu verbinden. Wenn Sie keinen Port angeben, ist der Standardport '6379'. |
| Passwort      | benötigt | Dieses Passwort für die Authentifizierung verwenden.                                                                                    |
| SSL-Option    | benötigt | ob SSL aktiviert werden soll (empfohlen zu aktivieren)                                                                                  |

Um die benötigten Informationen auszufüllen gehen Sie auf die Datenbank-Dashboard-Seite und klicken Sie auf die Datenbank, die wir gerade in Ihrem Upstash erstellt haben.

![up_info](https://cdn.illacloud.com/official-website/img/docs/up_info.jpg)

Sie können es kopieren, indem Sie mit der Maus über den Endpunkt, das Passwort und den Port fahren und auf das Kopiertextsymbol klicken.

In der Upstash-Integrationskonfiguration in der ILLA Cloud den Namen für `Name` angeben, `Endpoint` für `hostname`, gegeben `Port` für `Port` und kopieren Sie das verschlüsselte `Password` für `Password`.

### Upstash konfigurieren

| Eigenschaften  | Beschreibung                                                                           |
| -------------- | -------------------------------------------------------------------------------------- |
| Redis-Abfrage  | verwenden Sie Abfrage, um Daten in Ihrer Redis Datenbank abzurufen und zu manipulieren |
| Transformator  | Umwandlung von Daten in den Stil, den Sie mögen, mit JavaScript                        |

Beispiel Verwendung:

1.  Werte erhalten:
    -   Holen Sie sich einen Wert: Benutzen Sie den Befehl **`GET`** um den Wert eines bestimmten Schlüssels abzurufen. Zum Beispiel: **`GET key`**.
2.  Werte festlegen:
    -   Legen Sie einen Wert fest: Benutzen Sie den Befehl **`SET`** um den Wert eines Schlüssels festzulegen. Zum Beispiel **`SET key value`**.
3.  Hash-Operationen:
    -   Setze Feldwert-Paare in einem Hash: Benutze den Befehl **`HSET`** um ein Feldwert-Paar in einem Hash zu setzen. Zum Beispiel: **`HSET HashKey Feldwert`**.
    -   Holen Sie sich einen Wert von einem Hash: Benutzen Sie den **`HGET`** Befehl, um den Wert eines bestimmten Feldes in einem Hash abzurufen. Zum Beispiel: **`HGET HashKey Feld`**.
4.  Listenoperationen:
    -   Füge ein Element zu einer Liste hinzu: Benutze den Befehl **`LPUSH`** oder **`RPUSH`** um ein Element zum Anfang bzw. Ende einer Liste hinzuzufügen. Zum Beispiel: **`LPUSH listKey value`**.
    -   Elemente aus einer Liste abrufen: Benutzen Sie den Befehl **`LRANGE`** um einen Bereich von Elementen aus einer Liste zu holen. Zum Beispiel: **`LRANGE listKey start end`**.
5.  Set-Operationen:
    -   Füge Elemente zu einem Set hinzu: Benutze den Befehl **`SADD`** um ein oder mehrere Elemente zu einem Set hinzuzufügen. Zum Beispiel: **`SADD setKey value1 value2`**.
    -   Rufe Elemente aus einem Set ab: Benutze den Befehl **`SMEMBERS`** um alle Mitglieder eines Sets zu erhalten. Zum Beispiel: **`SMEMBERS setKey`**.
6.  Sortierte Set-Operationen:
    -   Fügen Sie Elemente zu einem sortierten Satz hinzu: Benutzen Sie den Befehl **`ZADD`** um Elemente zu einem sortierten Satz zusammen mit deren Punktzahl hinzuzufügen. Zum Beispiel: **`ZADD sortedSetKey score1 member1 score2 member2`**.
    -   Abrufen von Elementen aus einem sortierten Satz: Benutzen Sie den Befehl **`ZRANGE`** um einen Bereich von Elementen aus einem sortierten Satz basierend auf deren Punktzahlen abzurufen. Zum Beispiel: **`ZRANGE sortedSetKey start end`**.

Hinweis: Sie können die Redis-Dokumentation für eine vollständige Liste der Befehle und deren Verwendung verwenden.

### Gehäuse verwenden

Als nächstes zeigen wir, wie die Daten von `**Upstash**` auf die `table` Komponente zugeordnet werden können.

#### Schritt 1 Eine Aktion hinzufügen

Lassen Sie uns eine Tabelle in Upstash mit dem Namen **`employee_profile`, ** mit 4 Spalten `name`, `level`, `age` und `verheiratet` erstellen.

Dann können wir eine neue Aktion für Upstash aus der Aktionsliste in ILLA Cloud und Name upstash2 erstellen.

Um alle Daten im Mitarbeiter_Profil aufzulisten, setzen Sie den Code-Snippet unten in der Redis-Abfrage-Abfrage.

```sql
Mitarbeiterprofil erhalten
```

Da die Ausgabedaten eine Liste von Wörterbuchpaaren sind, möchten wir sie in ein Array verwandeln. In Transformer aktivieren Sie es und legen Sie den Code unten.

```jsx
const jsonString = data[0].result;

// Parse the JSON string into a JavaScript object
const jsonData = JSON.parse(jsonString);

// Access the "employees" property and map it to a new array
const mappedArray = jsonData.employees.map((employee) => ({
  name: employee.name,
  level: employee.level,
  age: employee.age,
  married: employee.married,
}));

return mappedArray;
```

![up_code](https://cdn.illacloud.com/official-website/img/docs/up_code.jpg)

Klicke **`Speichern`** und **`Run`** um diese Aktion zu aktivieren.

![up_data](https://cdn.illacloud.com/official-website/img/docs/up_data.jpg)

#### Schritt 2 Komponenten hinzufügen

Als nächstes können wir eine **`table`** Komponente zur Leinwand hinzufügen.

Im Feld **`Datenquelle`** auf der Inspektseite nach einem Klick auf die Komponente setze `{{upstash2.data}}` um die Daten von upstash.

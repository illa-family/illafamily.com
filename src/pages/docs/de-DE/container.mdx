## <Heading hidden>Container</Heading>

Sie können verwandte Komponenten mit Hilfe der Container-Komponente in eine Karte gruppieren. Container unterstützen auch mehrere Ansichten, die es Ihnen ermöglichen, Inhalte in unterschiedliche Ansichten zu unterteilen, zwischen denen Benutzer wechseln können.

Container funktionieren wie jede andere Komponente. Nach dem Hinzufügen können Sie Komponenten gruppieren, indem Sie D oder K drücken und sie in den Container ziehen.

### Mehrere Ansichten

Container unterstützen mehrere Ansichten zur Gruppierung von Inhalten. Du kannst Ansichten im **Views** Bereich des Tabs **Inspec** setzen. Nur eine Ansicht ist gleichzeitig sichtbar und die Benutzer wechseln zwischen ihnen, indem sie die aktuelle Ansicht der Container-Komponente ändern.

### Andere Komponenten verwenden, um die Ansichten zu ändern

Mit anderen Komponenten können Sie Container steuern, indem Sie Ereignishandler zu anderen Komponenten hinzufügen. Die folgenden Methoden werden unterstützt:

1.  Aktuelle Ansicht nach Taste festlegen
2.  Aktuelle Ansicht nach Index festlegen
3.  Zeige die nächste (sichtbare) Ansicht
4.  Vorherige (sichtbare) Ansicht anzeigen

### Einen Tab mit einem Container verbinden

Du kannst eine Tab-Komponente verwenden, um einen Container zu steuern, indem du **Link zu einem Container** des Tabs **Inspec** aktivierst.

Nach der Verlinkung mit einem Container, werden Tabs auf die Etiketten der Ansichten geändert. Sie können die Ansichten ändern, indem Sie auf Tabs ohne Eventhandler klicken.

## <h2 hidden>Was ist Airtable?</h2>

Airtable ist eine populäre relationale Datenbank- und Tabellenkalkulationsplattform, die die Einfachheit einer Tabellenkalkulation mit der Komplexität einer traditionellen Datenbank verbindet. Benutzer können strukturierte Daten in einer benutzerfreundlichen Oberfläche erstellen und organisieren Dies macht es nützlich für verschiedene Aufgaben wie Projektmanagement, Contentplanung und Datenerfassung.

### Backend auf Airtable einrichten

Erstelle ein Konto bei `Airtable` und erstelle deine Basis entsprechend deiner Bedürfnisse.

Um Ihre Basis mit ILLA Cloud zu verbinden, navigieren Sie zum `Developer Hub`.

![air_developer_hub](https://cdn.illacloud.com/official-website/img/official-site/intergration/air_developer_hub.png)

Klicken Sie auf der `Persönlichen Zugangs-Token` Seite auf `Neues Token`.

![air_create_new](https://cdn.illacloud.com/official-website/img/official-site/intergration/air_create_new.jpeg)

Dies öffnet die Konfigurationsoptionen für den PAT, den Sie erstellen.

Vorname des PAT, dann wählen Sie den Bereich aus, auf den dieser PAT Zugriff haben soll. Wählen Sie schließlich den Grad des Zugriffs für dieses PAT aus.

![air_new_token](https://cdn.illacloud.com/official-website/img/official-site/intergration/air_new_token.jpeg)

Klicke auf den **`Erstellen-Token`** Button, wir haben jetzt ein PAT bereit zur Verbindung.

### Airtable in ILLA Cloud erstellen

Es gibt zwei Möglichkeiten, nach der Anmeldung in Ihrem Illa-Konto eine Ressource in Illa zu erstellen.

-   In Ressourcen erstellen

Melde dich mit deinem Illa-Konto an, wähle `**Ressourcen**` oben auf der Seite und klicke auf `**Neues erstellen**` Button.

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

Wähle **`Airtable`** aus der Datenbankliste.

![action_list_3](https://cdn.illacloud.com/official-website/img/official-site/intergration/action_list_3.jpeg)

Verbinden Sie sich mit den erforderlichen Parametern, die in `Verbindungseinstellungen` unten beschrieben sind. In diesem Fall füllen Sie das **`Persönliche Zugangs-Token`** aus `Airtable` aus und fügen es in die **`Access Token`** in der Konfiguration ein.

Klicke **`Ressourcen speichern`** um \*\*\*\*\*\*ein Werk zu erstellen.

Die fertige `Airtable` wird wie angezeigt angezeigt.

![air_ex_ready](https://cdn.illacloud.com/official-website/img/official-site/intergration/air_ex_ready.jpeg)

-   Im Builder erstellen

Melde dich mit deinem Illa Account an, erstelle ein Projekt in Illa Builder auf der `**App**` Seite und navigieren Sie zur **`Action List`** am unteren Ende der Seite. Klicke auf `**new**`, dann wähle **`Airtable`** aus der Datenbankliste. Dann verbinden Sie sich mit der Datenbank mit den erforderlichen Parametern, die in `Verbindungseinstellungen` unten beschrieben sind.

Klicke auf `**Verbindung testen**` um zu sehen, ob wir erfolgreich mit der Datenbank verbinden können. Wenn ja, klicke auf **`Ressourcen speichern`**, ansonsten überprüfe doppelt, ob der Hostname, der Port, der Benutzername und das Passwort korrekt sind.

### Verbindungseinstellungen

Hier müssen wir Informationen zur Verbindung mit der Airtable API bereitstellen.

![air_config](https://cdn.illacloud.com/official-website/img/official-site/intergration/air_config.jpeg)

| Eigenschaften   | Benötigt | Beschreibung                                                                                                                                                                        |
| --------------- | -------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Name            | benötigt | Der Name für die Ressource beim Erstellen von Aktionen in der ILLA.                                                                                                                 |
| Zugangs-Token   | benötigt | eine Information, die einer Drittanbieteranwendung oder -dienstleistung die Erlaubnis gibt, auf die Daten Ihres Airtable Kontos zuzugreifen und Aktionen in Ihrem Namen auszuführen |

### Aktionen erstellen

Wir haben eine Airtable Ressource erstellt. Wir können die Aktion hinzufügen, indem wir Airtable aus der Aktionsliste auswählen und die **`Aktion erstellen`** Schaltfläche auswählen.

![air_resource_list](https://cdn.illacloud.com/official-website/img/official-site/intergration/air_resource_list.jpeg)

Jetzt haben wir die Airtable API als Aktion auf unserer Bauseite hinzugefügt.

![air](https://cdn.illacloud.com/official-website/img/official-site/intergration/air.jpeg)

### Airtable konfigurieren

| Eigenschaften  | Beschreibung                                                                                                                                                                    |
| -------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Basis-ID       | ein eindeutiger Identifikator für Ihre Airtable Datenbank. Sie können die Basis-ID von Ihrer Basis-URL erhalten. Es ist normalerweise eine Zeichenkette, die mit "app" beginnt. |
| Tabellenname   | spezifiziert die bestimmte Tabelle innerhalb der gewählten Basis, mit der Sie interagieren möchten                                                                              |
| Aktionstyp     | welche Art von Operation Ihre Integration in der angegebenen Tabelle durchführen wird                                                                                           |
| Transformator  | Umwandlung von Daten in den Stil, den Sie mögen, mit JavaScript                                                                                                                 |

#### Datensätze auflisten

ruft eine Liste von Datensätzen aus einer angegebenen Tabelle in Airtable ab

** Input **

| Eigenschaften          | Benötigt | Beschreibung                                                                                                         |
| ---------------------- | -------- | -------------------------------------------------------------------------------------------------------------------- |
| Ansicht                | optional | organisieren und anzeigen von Datensätzen nach bestimmten Kriterien                                                  |
| Felder                 | optional | Felder beziehen sich auf die Spalten in Ihrer Airtable                                                               |
| Nach Formel filtern    | optional | definiere eine Formel, die die Datensätze filtert, mit denen du interagieren möchtest                                |
| Sortieren              | optional | die Reihenfolge, in der Datensätze angezeigt oder abgerufen werden                                                   |
| Max. Datensätze        | optional | begrenzt die Anzahl der Datensätze, mit denen deine Integration in einer einzigen Anfrage abgerufen oder interagiert |
| Seitengröße            | optional | häufig in Szenarien verwendet, in denen Daten in mehreren Stadien abgerufen werden müssen (Seiten)                   |
| Versatz                | optional | gibt den Startpunkt für das Abrufen von Datensätzen auf einer bestimmten Seite an                                    |
| Zellenformat           | optional | wie Daten in einem bestimmten Feld angezeigt werden                                                                  |
| Zeitzone               | optional | stellt sicher, dass Datums- und Zeitberechnungen und Präsentationen korrekt und konsistent sind                      |
| Benutzerstandort       | optional | die Integrationsausgabe je nach geographischer Lage oder Präferenzen des Benutzers anpassen                          |

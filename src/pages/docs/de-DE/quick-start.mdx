## <Heading hidden>Übersicht</Heading>

### 1.  ILLA-Builder einrichten

Bevor Sie den ILLA Builder verwenden, sollten Sie die ILLA Cloud oder Self-Host-ILLA zum Hosten Ihrer Apps verwenden. Wir stellen Ihnen ILLA CLI, Docker Compose und K8s zur Verfügung, um in weniger als 5 Minuten die ILLA selbst zu beherbergen. Wir werden dir später ILLA Cloud zur Verfügung stellen, damit du einfach mit dem ILLA Builder loslegen kannst.

<img
  src='https://cdn.illacloud.com/official-website/img/docs/qs_e_1.png'
  alt=""
/>

### 2.  Benutzerdaten abrufen

Registrieren und beim ILLA Builder anmelden und dann den Ressourcen-Tab eingeben. Klicken Sie auf die Schaltfläche "Neu erstellen" und dann können Sie hier den Ressourcen-Typ auswählen. Wir stellen Ihnen REST API und MySQL zur Verfügung, um Ihre Daten zu verbinden. Postgres, Redis und MongoDB werden diesen Monat verfügbar sein.

<img
  src='https://cdn.illacloud.com/official-website/img/docs/qs_e_2.png'
  alt=""
/>

### 3.  Drag and Drop Einfügungen

Wenn Ihre Daten vorbereitet sind, können Sie auf die Registerkarte Apps zurück gehen und eine App erstellen. Die Insert-Liste befindet sich auf der rechten Seite der Seite. Sie können einen von ihnen auswählen und in den Rahmen ziehen. Klicken Sie dann auf die Einfügefunktion und ändern Sie die Parameter in der Kontrollleiste. Wir stellen Ihnen den Markdown-Modus zur Verfügung, um das Label jeder Einfügung zu bearbeiten. Wenn Sie mit der Markdown-Syntax vertraut sind, können Sie die Textgröße und -dicke schnell anpassen. In der Zwischenzeit kannst du mit {{double brackets}} Aktionsdaten in der Texteingabebox aufrufen, um mehr Ergebnisse im ILLA Builder zu erzielen.

### 4.  Aktion in Apps hinzufügen

Eine Aktion wird benötigt, um die Daten von Ressourcen in Einfügen zu transformieren. Wir nennen es Taten. Sie können eine Abfrage in Aktion schreiben, um Daten aus Ihrer Ressource zu fangen. Auf dieser Grundlage können Sie JavaScript in Aktionen schreiben, um Datenverarbeitungsanweisungen wie Filter und Zusammenführung auszuführen. Ein MySQL-Beispiel ist unten angegeben:

<img
  src='https://cdn.illacloud.com/official-website/img/docs/qs_e_3.png'
  alt=""
/>

1.  Erstellen Sie eine MySQL-Aktion im App-Editor. Aktion wurde benannt1.

2.  Geben Sie den Abfragecode in das Panel ein, z.B. SELECT \* FROM Benutzer.

3.  Führen Sie diese Aktion aus, das Abfrageergebnis wird im unteren Fenster angezeigt.

### 5.  Zeige deine Daten

Wenn die Aktion erfolgreich erstellt wurde, müssen Sie das Ergebnis der Aktion an die Komponente binden. Sie sollten die Wertevariable in der Kontrollleiste durch das Ergebnis der Aktionen {{action1.data}} ersetzen. Dann werden die Komponentendaten mit dem Actionresultat gebunden.

## <h2 hidden>Supabase DB</h2>

Supabase ist eine Open-Source Firebase-Alternative, mit der Entwickler Echtzeit-Web-und mobile Anwendungen erstellen können. Es bietet viele der gleichen Funktionen wie Firebase, wie Echtzeit-Datenbanken, Benutzerauthentifizierung, und Hosting, aber mit den zusätzlichen Vorteilen von Open-Source-und Bereitstellung einer SQL-API. Dies ermöglicht Entwicklern die Nutzung ihrer vorhandenen SQL Kenntnisse und Tools, was die Integration in andere Systeme erleichtert. Zusätzlich bietet Supabase eine flexiblere und anpassbare Plattform als Firebase, die im Hinblick auf Datenmodellierung und -nutzung restriktiver sein kann.

Dieses Tutorial beschreibt den Prozess der Erstellung eines Admin-Panels mit Hilfe von ILLA Builder und Supabase in wenigen einfachen Schritten. ILLA ist eine Low-Code-Plattform für Entwickler, die die schnelle Entwicklung und den Einsatz von internen Tools ermöglicht. Es ermöglicht die Erstellung von Seiten durch Ziehen und Ablegen von UI-Komponenten, Verbindung zu jeder Datenbank oder API, und das Schreiben von JavaScript. Um mehr über Supabase zu erfahren, besuche ihre Website unter <https://supabase.com/>. Los geht's!

Klicken Sie auf dem [Supabase Dashboard](https://app.supabase.com/projects), und setzen Sie den Namen auf adminPanel.

### Schritt 3: Mit Supabase und Konfigurations-CRUD verbinden

Beachte die Datenbankverbindungsinformationen unter [Projekteinstellungen](https://app.supabase.com/project/hdcfnsxpwwgboqomdrhp/settings/database) in Supabase.

<img
  src='https://cdn.illacloud.com/official-website/img/docs/assemble/supabase6.png'
  alt=""
/>

Klicken Sie in der Aktionsliste auf `+ Neu` und wählen Sie Supabase DB.

<img
  src='https://cdn.illacloud.com/official-website/img/docs/assemble/supabase7.png'
  alt=""
/>

Füllen Sie das Formular aus, um sich mit Ihrer Supabase Instanz zu verbinden. Teste die Verbindung und speichere die Ressource.

<img
  src='https://cdn.illacloud.com/official-website/img/docs/assemble/supabase8.png'
  alt=""
/>

Klicke auf `Create Action`, um eine Aktion mit der Supabase Ressource und der Konfiguration deines CRUD zu erstellen.

<img
  src='https://cdn.illacloud.com/official-website/img/docs/assemble/supabase9.png'
  alt=""
/>

Verwenden Sie {{ um die Eingabedaten des Front-Ends zu erhalten. Das folgende ist ein Beispiel für die Benutzerverwaltung im Admin-Bereich.

Suche nach einem Benutzer mit dem in Input1 eingegebenen Namen

```
SELECT *
AUS Benutzer
WHERE name = "{{input1.value}}"
;
```

Benutzerdaten aktualisieren. Benutzerinformationen bei Id-Treffer aktualisieren

```
UPDATE Benutzer
SET name = "{{input3.value}}"
, email = "{{input4.value}}"
WHERE id="{{input2.value}}"
;
```

Benutzerdaten einfügen

```
INSERT INTO Benutzerwerte("{{input5.value}}","{{input6.value}}","{{input7.value}}");
```

Benutzer mit Id löschen

```
VON Benutzer LÖSCHEN WIE Id = "{{input2.value}}";
```

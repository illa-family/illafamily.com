## <h2 hidden>Integrationsliste</h2>

ILLA bietet eine Reihe vorgefertigter Integrationen an, mit denen Sie sich mit minimaler Konfiguration leicht mit Ihren Datenquellen verbinden können. Wählen Sie einfach die passende Integration aus der Liste der verfügbaren Optionen aus, geben Sie die erforderlichen Anmeldedaten ein und Sie können Ihre Datenquelle innerhalb der ILLA nutzen. Dadurch entfällt die Notwendigkeit, komplexe Konfigurationen einzurichten und ermöglicht es Ihnen, schnell mit Ihren Daten zu arbeiten.

### Datenbankintegrationen

| Name             | Beschreibung                                                                                                                                                                                                                            |
| ---------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Mysql            | MySQL ist ein relationales Open-Source-Datenbankverwaltungssystem (RDBMS), das auf der Structured Query Language (SQL) basiert.                                                                                                         |
| PostgreSQL       | PostgreSQL ist ein quelloffenes objekt-relationales Datenbankmanagementsystem (ORDBMS), das für seine Zuverlässigkeit und seinen Funktionsumfang bekannt ist.                                                                           |
| MariaDB          | MariaDB ist ein relationales Open-Source-Datenbank-Management-System, das ein Abzweig der MySQL-Datenbank ist.                                                                                                                          |
| TiDB             | TiDB ist eine Open-Source NewSQL-Datenbank, die Hybrid Transactional und Analytical Processing Arbeitsbelastungen unterstützt.                                                                                                          |
| Redis            | Redis ist ein Open-Source-Datenstrukturspeicher, der oft als Datenbank-, Cache- und Message-Broker verwendet wird.                                                                                                                      |
| MongoDB          | MongoDB ist ein freies und Open-Source-Cross-Platform-basiertes Datenbank-Programm.                                                                                                                                                     |
| Elastische Suche | Elasticsearch ist eine Open-Source-Such- und Analysemaschine, die auf Apache Lucene aufbaut.                                                                                                                                            |
| Firebase         | Firebase ist eine Cloud-basierte Plattform für mobile und Web-Anwendungen, die von Google entwickelt wurde.                                                                                                                             |
| Supabase DB      | Supabase ist eine Open-Source-Plattform zur Erstellung serverloser Echtzeit-Anwendungen. Es basiert auf PostgreSQL und bietet eine Reihe von Tools und Services für die schnelle Erstellung und Bereitstellung serverloser Anwendungen. |
| Clickhouse       | ClickHouse ist ein quelloffenes Datenbank-Management-System, das für die Online-analytische Verarbeitung (OLAP) und Echtzeit-Datenanalyse konzipiert ist.                                                                               |

### API-Integrationen

| Name                  | Beschreibung                                                                                                                                           |
| --------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------ |
| REST-API              | REST ist ein architektonischer Stil für den Aufbau von Web-Dienstleistungen. RESTful APIs sind APIs, die dem REST-Architekturstil folgen.              |
| Amazon S3(General S3) | Amazon S3 ist ein von AWS angebotener Objekt-Speicherdienst. Es bietet eine sichere, skalierbare und kostengünstige Speicherung für Daten im Internet. |
| SMTP                  | SMTP (Simple Mail Transfer Protocol) ist ein Protokoll zum Senden und Empfangen von E-Mail-Nachrichten über das Internet.                              |
| GraphQL               | GraphQL ist eine Abfragesprache für APIs und eine Laufzeit für die Ausführung dieser Abfragen mit Ihren vorhandenen Daten.                             |

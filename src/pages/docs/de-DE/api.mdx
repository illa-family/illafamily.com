Verbinden Sie sich mit einer API
mit einer beliebigen REST, GraphQL API.
Sie können ILLA mit den meisten APIs über unsere native Integration verbinden. Wenn du mit einer internen API verbinden musst, oder wenn es keine unterstützte Integration gibt Sie können sich mit unserer REST API Integration oder GraphQL Integration verbinden.

1.  Erstellen Sie eine API-Ressource 
    Ressourcen repräsentieren Verbindungen zu Datenquellen und sind das, was Sie innerhalb Ihrer ILLA-Apps abfragen. Melden Sie sich bei ILLA an und navigieren Sie zum Ressourcen-Tab > Klicken Sie auf Neu erstellen und wählen Sie die API, mit der Sie sich verbinden möchten wie REST API. Dies öffnet den Modus Ressource einrichten, in dem Sie Ihre Verbindungsinformationen hinzufügen.

Sie benötigen mindestens die Basis-URL der API, mit der Sie sich verbinden möchten. Sie können auch URL-Parameter, Header, etc. übergeben. Wenn Ihre API eine Authentifizierung erfordert, können Sie dies auch auf dieser Seite konfigurieren. Zum Beispiel benötigt OAuth2.0 eine Autorisierungs-URL, Access Token URL, etc. Nachdem Sie Ihre Verbindungsdaten eingegeben haben, klicken Sie auf Ressource speichern. 
2. Abfragen Ihrer API
im App-Editor, Klicken Sie + Neu in der Aktionsliste > wählen Sie REST API und wählen Sie eine vorhandene REST API oder erstellen Sie eine neue > Klicken Sie auf Aktion erstellen und aktualisieren Sie die Aktionen mit zusätzlichen URL-Parametern, kopf, usw.

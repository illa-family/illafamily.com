Verwenden Sie die Bibliothek der ILLA der UI Komponenten, um die Benutzeroberfläche Ihrer App zu erstellen.
Komponenten sind ein wesentlicher Bestandteil der ILLA, und sie funktionieren ähnlich zu React Komponenten, indem sie Eigenschaften akzeptieren und einen internen Zustand haben. Indem Sie die ILLA-Bibliothek von Komponenten mit Ihren Daten kombinieren, können Sie interaktive Apps erstellen.
Gewöhnliche Komponenten sind:
Die Tabellenkomponente wird verwendet, um Daten in einem Tabellenformat anzuzeigen, und bietet Funktionen wie Sortieren, Filtern, Seitenaufbau und CSV-Export. 
Die Form Komponente wird verwendet, um Eingabefelder zu gruppieren und sie als eine Einheit einzureichen. 
Die Eingabekomponente Selektieren ist eine Kombination aus einer Dropdown-Liste und einem Eingabefeld, das es Benutzern erlaubt, entweder einen Wert aus einer Liste auszuwählen oder einen Wert direkt als Text einzugeben. 
Die Diagrammkomponente wird zur Visualisierung von Daten mit Diagrammen und Diagrammen verwendet.

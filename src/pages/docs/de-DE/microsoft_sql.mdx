## <h2 hidden>Was ist Microsoft SQL?</h2>

Microsoft SQL in Illa ist eine Datenbank-Integration, die es Benutzern ermöglicht, sich mit einer Microsoft SQL (MS SQL) Datenbank zu verbinden und zu interagieren. MS SQL ist ein beliebtes Open-Source-relationales Datenbank-Management-System, das für seine Geschwindigkeit, Zuverlässigkeit und Benutzerfreundlichkeit bekannt ist.

Mit der MS SQL Integration in Illa können Sie Daten aus einer MS SQL Datenbank mittels SQL-Befehlen abfragen, einfügen, aktualisieren und löschen. Sie können auch benutzerdefinierte SQL-Abfragen erstellen und innerhalb von Illa ausführen. Dies macht es einfach, datengesteuerte Anwendungen und Dashboards zu erstellen, die Echtzeitdaten aus einer MS SQL-Datenbank anzeigen.

### MS SQL erstellen

Es gibt zwei Möglichkeiten, nach der Anmeldung in Ihrem Illa-Konto eine Ressource in Illa zu erstellen.

-   In Ressourcen erstellen

Melde dich in dein Illa Konto an, wähle `**Ressourcen**` oben auf der Seite und klicke auf `**Neu erstellen**` Button.

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

Wähle **`Microsoft SQL`** aus der Datenbankliste.

![action_list](https://cdn.illacloud.com/official-website/img/docs/action_list.png)

Verbinden Sie sich mit den erforderlichen Parametern, die in `Verbindungseinstellungen` unten beschrieben sind.

Klicke auf `**Verbindung testen**` um zu sehen, ob wir erfolgreich mit der Datenbank verbinden können. Wenn ja, klicke auf **`Ressourcen speichern`**, ansonsten überprüfe doppelt, ob der Hostname, der Port, der Benutzername und das Passwort korrekt sind.

Nach der Erstellung einer Ressource wird das fertige MS SQL wie angezeigt angezeigt.

![ms_ex_ready](https://cdn.illacloud.com/official-website/img/docs/ms_ex_ready.png)

-   Im Builder erstellen

Melde dich bei deinem Illa Account an, erstelle ein Projekt in Illa Builder auf der `**App**` Seite und navigieren Sie zur **`Action List`** am unteren Ende der Seite. Klicken Sie auf `**new**`, dann wählen Sie **`Microsoft SQL`** aus der Datenbankliste. Verbinden Sie sich dann mit der Datenbank mit den erforderlichen Parametern, die in `Verbindungseinstellungen` unten beschrieben sind.

Klicke auf `**Verbindung testen**` um zu sehen, ob wir erfolgreich mit der Datenbank verbinden können. Wenn ja, klicke auf **`Ressourcen speichern`**, ansonsten überprüfe doppelt, ob der Hostname, der Port, der Benutzername und das Passwort korrekt sind.

### Verbindungseinstellungen

Hier müssen wir Informationen zur Verbindung mit MS SQL Datenbank zur Verfügung stellen.

![ms_config](https://cdn.illacloud.com/official-website/img/docs/ms_config.png)

| Eigenschaften         | Benötigt | Beschreibung                                                                                                                            |
| --------------------- | -------- | --------------------------------------------------------------------------------------------------------------------------------------- |
| Name                  | benötigt | Der Name für die Ressource beim Erstellen von Aktionen in der ILLA.                                                                     |
| Hostname              | benötigt | Die URL oder IP-Adresse für Ihre Datenbank                                                                                              |
| Port                  | benötigt | Die Portnummer des Servers, die Sie verwenden sollten, um sich zu verbinden. Wenn Sie keinen Port angeben, ist der Standardport '1433'. |
| Datenbank             | benötigt | Der Name der Datenbank                                                                                                                  |
| Benutzername          | benötigt | den Benutzernamen, den Sie beim Einloggen in den MS SQL Server verwenden möchten.                                                       |
| Passwort              | benötigt | Dieses Passwort für die Authentifizierung verwenden.                                                                                    |
| Verbindungsoption     | optional | key-value Paar, um die Verbindungsoption anzugeben                                                                                      |
| SSL-Optionen          | optional | entscheidet, wie hoch eine sichere SSL-TCP/IP-Verbindung bei der Verhandlung mit dem Server priorisiert wird.                           |

### Aktionen erstellen

Wir haben eine MS SQL-Ressource erstellt; wir können die Aktion hinzufügen, indem wir MS SQL aus der Aktionsliste auswählen und die **`Aktion erstellen`** Schaltfläche auswählen.

![ms_resource_list](https://cdn.illacloud.com/official-website/img/docs/ms_resource_list.png)

Jetzt haben wir den MS SQL Server als eine Aktion zu unserer Bauseite hinzugefügt.

![ms](https://cdn.illacloud.com/official-website/img/docs/ms.png)

### MS SQL konfigurieren

| Eigenschaften     | Beschreibung                                                                                                                                                                                                                                                                                   |
| ----------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Konfigurationstyp | die Art der Konfiguration, die Sie zur Verbindung mit Ihrer MS SQL Server Datenbank verwenden: SQL und Bulk Einfügen. Für den SQL-Modus können Sie SQL-Abfrage verwenden. Für Masseneinfügen-Modus können Sie Daten in eine ausgewählte Tabelle mit dem Format {{ [{x:a, y:b}, … einfügen ] }} |
| SQL-Abfrage       | sql Befehle zum Abrufen (Auswählen), Einfügen, Aktualisieren, Löschen von Daten aus der Datenbank.                                                                                                                                                                                             |
| Transformator     | Umwandlung von Daten in den Stil, den Sie mögen, mit JavaScript                                                                                                                                                                                                                                |

Beispiel Verwendung:

-   Auswählen

```sql
SELECT * FROM users ORDER BY lastname;
```

-   Insert

```sql
INSERT INTO users

	(firstname, lastname, email)

VALUES

	(

		{{ NewUser.data.firstNameInput }},

		{{ NewUser.data.lastNameInput }},

		{{ NewUser.data.emailInput }}

	);
```

-   Aktualisieren

```sql
UPDATE users
  SET firstname = '{{form1.updatedProfile.first}}'
  WHERE id = {{ form1.updatedProfile.uid}};
```

-   Löschen

```sql
DELETE FROM users WHERE id = {{ form1.updatedProfile.uid }};
```

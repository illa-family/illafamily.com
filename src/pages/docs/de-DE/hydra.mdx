## <h2 hidden>Was ist Hydra?</h2>

Hydra ist eine leistungsstarke Integration innerhalb der ILLA Cloud Plattform, die nahtlos verschiedene Datenquellen und Anwendungen verbindet und konsolidiert. Als zentraler Knotenpunkt ermöglicht Hydra einen effizienten Datenfluss und Kommunikation zwischen verschiedenen Systemen, was eine Echtzeit-Datensynchronisierung, automatisierte Workflows und rationalisierte Prozesse ermöglicht. Mit seinen robusten Fähigkeiten ermöglicht Hydra Unternehmen ihre Aktivitäten zu optimieren, die Zusammenarbeit zu verbessern, und datengestützte Entscheidungen mit Leichtigkeit, alle innerhalb des integrierten Ökosystems der ILLA Cloud.

### Hydra erstellen

Es gibt zwei Möglichkeiten, eine Ressource in ILLA nach der Anmeldung in Ihrem Konto zu erstellen.

-   In Ressourcen erstellen

Melde dich in dein Illa Konto an, wähle `**Ressourcen**` oben auf der Seite und klicke auf `**Neu erstellen**` Button.

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

Wähle **`Hydra`** aus der Datenbankliste.

![action_list](https://cdn.illacloud.com/official-website/img/docs/action_list.png)

Verbinden Sie sich mit den erforderlichen Parametern, die in `Verbindungseinstellungen` unten beschrieben sind.

Klicke auf `**Verbindung testen**` um zu sehen, ob wir erfolgreich mit der Datenbank verbinden können. Wenn ja, klicke auf **`Ressourcen speichern`**, ansonsten überprüfe doppelt, ob der Hostname, der Port, der Benutzername und das Passwort korrekt sind.

Nach dem Erstellen einer Ressource wird die fertige Hydra wie gezeigt angezeigt.

![hydra_ex_ready](https://cdn.illacloud.com/official-website/img/official-site/intergration/hydra_ex_ready.jpeg)

-   Im Builder erstellen

Melde dich bei deinem Illa Account an, erstelle ein Projekt in Illa Builder auf der `**App**` Seite und navigieren Sie zur **`Action List`** am unteren Ende der Seite. Klicke `**new**`, dann wähle **`Hydra`** aus der Datenbankliste. Verbinden Sie sich dann mit der Datenbank mit den erforderlichen Parametern, die in `Verbindungseinstellungen` unten beschrieben sind.

Klicke auf `**Verbindung testen**` um zu sehen, ob wir erfolgreich mit der Datenbank verbinden können. Wenn ja, klicke auf **`Ressourcen speichern`**, ansonsten überprüfe doppelt, ob der Hostname, der Port, der Benutzername und das Passwort korrekt sind.

### Verbindungseinstellungen

Hier müssen wir Informationen zur Verbindung mit der Hydra-Datenbank zur Verfügung stellen.

![hydra_config](https://cdn.illacloud.com/official-website/img/official-site/intergration/hydra_config.jpeg)

![hydra_config_1](https://cdn.illacloud.com/official-website/img/official-site/intergration/hydra_config_1.jpeg)

| Eigenschaften           | Benötigt | Beschreibung                                                                                                                                                                                                                                                                                   |
| ----------------------- | -------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Name                    | benötigt | Der Name für die Ressource beim Erstellen von Aktionen in der ILLA.                                                                                                                                                                                                                            |
| Hostname                | benötigt | Die URL oder IP-Adresse für Ihre Datenbank                                                                                                                                                                                                                                                     |
| Port                    | benötigt | Die Portnummer des Servers, die Sie verwenden sollten, um sich zu verbinden. Wenn Sie keinen Port angeben, ist der Standardport '5432'.                                                                                                                                                        |
| Datenbank               | benötigt | Der Name der Datenbank                                                                                                                                                                                                                                                                         |
| Benutzername            | benötigt | den Benutzernamen den Sie beim Einloggen auf dem Hydra Server verwenden möchten.                                                                                                                                                                                                               |
| Passwort                | benötigt | Dieses Passwort für die Authentifizierung verwenden.                                                                                                                                                                                                                                           |
| SSL-Optionen            | optional | entscheidet, wie hoch eine sichere SSL-TCP/IP-Verbindung bei der Verhandlung mit dem Server priorisiert wird.                                                                                                                                                                                  |
| CA-Zertifikat           | benötigt | ein digitales Zertifikat, das verwendet wird, um die Identität eines Servers zu verifizieren und einen sicheren Server zu etablieren, verschlüsselte Verbindung zwischen einem Client und einem Hydra Cluster. Wenn Sie es nicht angeben, haben wir ein Standardzertifikat für Sie ausgefüllt. |
| Kundenschlüssel         | optional | ein Parameter, der verwendet wird, um eine sichere Verbindung zwischen dem Hydra-Client und dem Hydra-Server herzustellen.                                                                                                                                                                     |
| Client-Zertifikat       | optional | eine Sicherheitsfunktion, die es einem Client erlaubt, sich bei einem Hydra-Server zu authentifizieren                                                                                                                                                                                         |

### Aktionen erstellen

Wir haben eine Hydra Ressource erstellt. Wir können die Aktion hinzufügen, indem wir Hyd aus der Aktionsliste auswählen und die **`Aktion erstellen`** Schaltfläche auswählen.

![hydra_resource_list](https://cdn.illacloud.com/official-website/img/official-site/intergration/hydra_resource_list.jpeg)

Jetzt haben wir den Hydra Server als Aktion auf unserer Bauseite hinzugefügt.

![hydra](https://cdn.illacloud.com/official-website/img/official-site/intergration/hydra.jpeg)

### Configure Hydra

| Eigenschaften  | Beschreibung                                                                                  |
| -------------- | --------------------------------------------------------------------------------------------- |
| SQL-Abfrage    | sql Befehle zum Abrufen (auswählen), einfügen, aktualisieren, Daten aus der Datenbank löschen |
| Transformator  | Umwandlung von Daten in den Stil, den Sie möchten, mit Javascript                             |

Beispiel Verwendung:

-   Auswählen

```sql
AUSWÄHLEN * AUS Benutzer VON Nachname;
```

-   Insert

```sql
INSERT INTO users

	(firstname, lastname, email)

VALUES

	(

		{{ NewUser.data.firstNameInput }},

		{{ NewUser.data.lastNameInput }},

		{{ NewUser.data.emailInput }}

	);
```

-   Aktualisieren

```sql
UPDATE users
  SET firstname = '{{form1.updatedProfile.first}}'
  WHERE id = {{ form1.updatedProfile.uid}};
```

-   Löschen

```sql
DELETE FROM users WHERE id = {{ form1.updatedProfile.uid }};
```

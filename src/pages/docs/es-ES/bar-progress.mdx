## <h2 hidden>¿Qué es el progreso de la Barra?</h2>

El componente Bar Progress es un elemento visual que representa el progreso o la finalización de una tarea o proceso. Muestra una barra horizontal que se llena o se recorta basándose en el valor de progreso especificado.

### Propiedades

| Propiedades               | Descripción                                                                                                                                        |
| ------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------- |
| Valor                     | el valor porcentual representa el progreso                                                                                                         |
| Ocultar etiqueta de valor | si la etiqueta de valor está oculta                                                                                                                |
| Etiqueta                  | El nombre del campo mostrado al usuario                                                                                                            |
| Etiqueta oculta           | Establecer si mostrar o no la etiqueta                                                                                                             |
| Sugerencia                | Un título usado para describir el campo en detalle                                                                                                 |
| Posición                  | Define la posición de la etiqueta relativa al componente                                                                                           |
| Alineación                | Establecer el alineamiento (alinear a la izquierda o a la derecha) de la etiqueta                                                                  |
| Width                     | Cuando la etiqueta esté en el lado izquierdo del componente, establezca la proporción de ancho de la etiqueta.                                     |
| Sugerencia                | Los usuarios pueden ingresar la descripción del componente aquí. La descripción se mostrará cuando esté enfocada. Formato Markdown soportado.      |
| Hidden                    | Controlar dinámicamente si el componente está oculto. Puede cambiar el estado oculto a través de un valor booleano dinámico.                       |
| Color del tema            | Permite a los usuarios especificar el color de fondo y opacidad del botón                                                                          |
| Styles                    | el ancho del trazo de la barra representa el progreso                                                                                              |

### Método

Puede usar otros componentes para controlar el componente. Apoyamos los dos métodos siguientes:

-   **setValue**

Para establecer el valor de entrada del área de texto, por ejemplo, {{"value1"}}

| Propiedades | Descripción |
| ----------- | ----------- |
| Valor       | Input value |

-   **limpiarValor**

Limpiar el valor del componente seleccionado

### Datos

El componente tiene algunos datos comúnmente utilizados, que pueden llamarse a través de **`{{componentName.propertyName}}`** en la aplicación.

| Nombre de la propiedad    | Descripción                                                |
| ------------------------- | ---------------------------------------------------------- |
| color                     | color de la barra                                          |
| nombre de pantalla        | el nombre de este componente (es decir, botón1)            |
| hidden                    | estado oculto (verdadero o falso)                          |
| etiqueta                  | valor de etiqueta                                          |
| alineación de etiqueta    | alineación de la etiqueta de entrada (izquierda o derecha) |
| posición de etiqueta      | posición de la etiqueta de entrada (izquierda o derecha)   |
| ancho de etiqueta         | el entero que representa el ancho de la etiqueta.          |
| Ancho de trazo            | ancho del trazo de barra                                   |
| tooltipText               | valor del texto del tooltip                                |
| valor                     | valor porcentual del progreso                              |

Ejemplo: {{barProgress1.value}}

### Caso de uso

Demostraremos cómo hacer un formulario con el progreso de barra en la parte inferior mostrando el progreso de terminación del formulario.

#### Paso 1 Añadir componentes

Añade un componente **`form`** con el texto predeterminado "Forma" y un botón "Enviar" a lienzo.

Añade un componente **`input`**, un componente **`number input`** y un componente **`bar progreso`** de `**Insert**` en el formulario.

Etiqueta el componente de entrada a ser "Nombre", el componente de entrada de número a ser "Edad" y el componente de progreso de la barra a ser "Progresar".

Establezca el marcador de posición del componente de entrada a "Por favor ponga su nombre aquí" y el valor por defecto de la entrada de número a 0 como se muestra a continuación.

![bar_layout](https://cdn.illacloud.com/official-website/img/official-site/components/bar_layout.jpg)

#### Paso 2 Configurar el componente

Para los componentes **`input`** y **`number input`**, podemos configurarlos para que establezcan el valor en el componente **`bar progress`** para reflejar el progreso de la terminación.

1.  Haz clic en el componente **`input`** y **`number input`** para abrir su página de inspección. Bajo **`Controlador de eventos`**, haga clic en `**+ Nuevo**`.
2.  En el controlador de edición de eventos, seleccione **`componente de control`** en acción, y seleccione el componente **`barra de progreso`** que desea actualizar como objetivo del evento.
3.  Elige la acción **`setValue`** y establece el valor como

```jsx
{
  {
    input1.value && numberInput1.value
      ? "100"
      : input1.value || numberInput1.value
      ? "50"
      : "0";
  }
}
```

![bar_input_num_config](https://cdn.illacloud.com/official-website/img/official-site/components/bar_input_num_config.jpg)

Nota: necesita establecer el manejador de eventos como se describe anteriormente a la entrada BOTH y a los componentes de entrada de números.

Para el componente **`botón`** podemos configurarlo para que envíe el formulario si el formulario está completado (es decir, el progreso de la barra 100%)

1.  Haga clic en el componente **`button`** para abrir su página de inspección. Bajo **`Controlador de eventos`**, haga clic en `**+ Nuevo**`.
2.  En el controlador de edición de eventos, seleccione **`componente de control`** en acción, y seleccione el componente **`form`** que desea actualizar como objetivo del evento.
3.  Elige el método **`submit`** y pon `{{barProgress1.value == 100}}` en **`Sólo ejecutar cuando`**

![bar_button_submit_config](https://cdn.illacloud.com/official-website/img/official-site/components/bar_button_submit_config.jpg)

Del mismo modo, podemos configurar el botón para mostrar advertencia cuando el usuario está intentando enviar un formulario incompleto.

1.  Haga clic en el componente **`button`** para abrir su página de inspección. Bajo **`Controlador de eventos`**, haga clic en `**+ Nuevo**`.
2.  En el controlador de edición de eventos, seleccione **`Mostrar notificación`** en acción, **`title`** "¡Error!" y **`description`** "Por favor, rellene el formulario antes de enviar". Elige `Error` para **`type`**, pon `{{2000}}` para la **`duración`** (2 segundos), y pon `{{barProgress1.value<100}}` para **`Sólo ejecutar cuando`.**

![bar_button_show_config](https://cdn.illacloud.com/official-website/img/official-site/components/bar_button_show_config.jpg)

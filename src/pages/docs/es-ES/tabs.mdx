## <h2 hidden>¿Qué es la pestaña?</h2>

El componente de Pestañas en la nube de ILLA es una característica versátil que permite a los usuarios organizar y presentar contenido en un diseño con pestañas. Permite la creación de múltiples pestañas dentro de un solo contenedor, permitiendo a los usuarios cambiar entre diferentes secciones de contenido con facilidad. El componente Tabs proporciona una interfaz fácil de usar para navegar a través de información relacionada, mejorando la experiencia general del usuario. Con opciones de estilo personalizables y la capacidad de añadir varios tipos de contenido, como texto, imágenes o componentes incrustados, el componente Tabs de ILLA Cloud ofrece flexibilidad y mejora la organización y la presentación de datos.

### Propiedades

| Propiedades                  | Descripción                                                                                                                                                                               |
| ---------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Enlazar a un contenedor      | enlazar el componente Tabs a un contenedor específico, permitiendo una navegación sin problemas entre las pestañas y el contenido asociado.                                               |
| Controlador de eventos       | detectar y responder a acciones o eventos específicos del usuario, tales como clics, pulsaciones de teclas o cambios de valor.                                                            |
| Deshabilitado                | desactivar pestañas específicas, evitando que los usuarios interactúen con ellas                                                                                                          |
| Sugerencia                   | información adicional o contexto sobre cada pestaña                                                                                                                                       |
| Diseño                       | ofrece diferentes opciones de diseño, permitiéndole organizar las pestañas de forma horizontal o vertical basándose en sus preferencias de diseño o en el espacio disponible.             |
| Align                        | alinear el componente Tabs a diferentes posiciones dentro de su contenedor padre, como la izquierda, el centro, o el alineamiento derecho                                                 |
| Hidden                       | ocultar pestañas específicas programáticamente.                                                                                                                                           |
| Colores de texto             | colores de texto personalizables, permitiéndole diseñar las etiquetas de la pestaña con diferentes colores para que coincidan con el diseño o la marca de su aplicación.                  |

### Datos

El componente tiene algunos datos comúnmente utilizados, que pueden llamarse a través de **`{{componentName.propertyName}}`** en la aplicación.

| Nombre de la propiedad          | Descripción                                                                                                                             |
| ------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------- |
| align                           | alineación de las pestañas dentro del componente, como la izquierda, el centro, o el alineamiento derecho                               |
| esquema de color                | define el color del texto en el componente                                                                                              |
| currentIndex                    | representa el índice de la pestaña seleccionada actualmente dentro de la lista de pestañas                                              |
| clave actual                    | identificador único o clave de la pestaña seleccionada                                                                                  |
| nombre de pantalla              | representa el nombre o la etiqueta asignada al componente                                                                               |
| hidden                          | valor booleano que determina si el componente de pestañas es visible o oculto                                                           |
| enlazar el nombre de la ventana | mostrar nombre o etiqueta del widget de enlace asociado con una pestaña particular                                                      |
| navigateContainer               | contenedor o destino donde el contenido de la pestaña debe ser mostrado o navegado a                                                    |
| tabLista                        | lista de pestañas, incluyendo sus nombres, teclas y otra información relevante                                                          |
| tabPosition                     | posición del clip de la pestaña, como la posición superior, inferior, izquierda o derecha                                               |
| tooltipText                     | texto mostrado como una descripción cuando el usuario pasa sobre una pestaña específica                                                 |
| ver lista                       | vistas o contenido asociado con cada pestaña, habilitando la representación de diferentes contenidos basados en la pestaña seleccionada |

Ejemplo: {{tabs1.displayName}}

### Caso de uso

Vamos a demostrar cómo usar el componente tabs combinado con un componente de contenedor para mostrar las tres vistas.

#### Paso 1 Añadir y configurar componentes

Añade un componente **`container`** y un componente **`tabs`**. En la página `inspect` del componente de pestañas, gira **`Link to a Container`** y elige el contenedor con el que quieres enlazar para `**Contenedor**`.

-   Ver 1:

Esta es la página de inicio, así colocamos un componente **`text`** con "Inicio" en la esquina superior izquierda del contenedor.

Muestra un componente **`text`** "Hola! Bienvenido a mi sitio web personal" en el medio y un componente **`botón`** etiquetado como "Aprende más sobre mí" debajo de él.

Para el componente **`button`** podemos configurarlo para que vaya a la siguiente vista (view2) si se hace clic.

1.  Haga clic en el componente **`button`** para abrir su página de inspección. Bajo **`Controlador de eventos`**, haga clic en `**+ Nuevo**`.
2.  En el controlador de edición de eventos, seleccione **`componente de control`** en acción, y seleccione el **`contenedor`** componente que desea actualizar como objetivo del evento.
3.  Elija el método **`showNextView`**.

![tab_view1](https://cdn.illacloud.com/official-website/img/docs/tab_view1.jpg)

-   Ver 2:

Este es el tema de la página, así que colocamos un componente **`text`** con "Acerca de mí" en la esquina superior izquierda del contenedor.

Mostrar el "nombre", "edad", y información de "género" con componentes de **`text`** y una imagen de perfil de ti o de cualquier persona que quieras usar con el componente `**image**`.

Añade un componente de **`button`** etiquetado como "Haz amigos conmigo".

Para el componente **`button`** podemos configurarlo para que vaya a la siguiente vista (vista3) si se hace clic.

1.  Haga clic en el componente **`button`** para abrir su página de inspección. Bajo **`Controlador de eventos`**, haga clic en `**+ Nuevo**`.
2.  En el controlador de edición de eventos, seleccione **`componente de control`** en acción, y seleccione el **`contenedor`** componente que desea actualizar como objetivo del evento.
3.  Elija el método **`showNextView`**.

![tab_view2](https://cdn.illacloud.com/official-website/img/docs/tab_view2.jpg)

-   Ver 3

Esta es la página del formulario, el visitante puede rellenar el formulario con sus datos para enviar una solicitud de amistad al propietario.

Hay un componente de formulario en el centro del contenedor con componentes de entrada etiquetados como "Tu nombre", "Tu edad" y "Género".

Cambie el texto del botón "Enviado" en la esquina inferior derecha a "Invitar".

![tab_view3](https://cdn.illacloud.com/official-website/img/docs/tab_view3.jpg)

Para el componente "Invitar" **`button`** podemos configurarlo para que envíe la información en el formulario si se hace clic.

1.  Haga clic en el componente **`button`** para abrir su página de inspección. Bajo **`Controlador de eventos`**, haga clic en `**+ Nuevo**`.
2.  En el controlador de edición de eventos, seleccione **`componente de control`** en acción, y seleccione el componente **`form`** que desea actualizar como objetivo del evento.
3.  Elija el método **`submit`**.

Para el componente **`form`** podemos configurarlo para que muestre la notificación si la enviamos.

1.  Haga clic en el componente **`form`** para abrir su página de inspección. Bajo **`Controlador de eventos`**, haga clic en `**+ Nuevo**`.
2.  En el controlador de edición de eventos, seleccione **`envio`** en el evento, seleccione **`Mostrar notificación`** en **`acción`**, Pon "Enviar con éxito" para el Título, "Solicitud de Amigo enviada" para Descripción, y selecciona `Éxito` para **`Tipo`**. Queremos que la notificación muestre por 2s así que pon `{{2000}}` para la duración.

![tab_form_event](https://cdn.illacloud.com/official-website/img/docs/tab_form_event.jpg)

## <h2 hidden>¿Qué es Amazon S3?</h2>

Amazon S3 es un servicio de almacenamiento en la nube altamente escalable y seguro ofrecido por Amazon Web Services (AWS). Cuando se combina con ILLA Cloud, los desarrolladores pueden integrar perfectamente Amazon S3 en sus aplicaciones, aprovechando sus robustas capacidades de almacenamiento para almacenar y recuperar datos de forma eficiente. ILLA Cloud simplifica el proceso de integración, permitiendo a los desarrolladores conectarse sin problemas a Amazon S3 y utilizar sus potentes características, como cifrado de datos, versionado, controles de acceso y escalabilidad perfecta. Con la integración de Amazon S3 en ILLA Cloud, los desarrolladores pueden mejorar sus aplicaciones con soluciones de almacenamiento confiables y escalables, para que se concentren en la creación de experiencias innovadoras y basadas en datos.

### Crear API de Amazon S3

Hay dos maneras de crear un recurso en Illa después de iniciar sesión en tu cuenta de Imax.

-   Crear en Recursos

Inicia sesión en tu cuenta de Imax, selecciona `**Recursos**` en la parte superior de la página y haz clic en el botón `**Crear nuevo**`.

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

Selecciona **`Amazon S3`** de la lista de API.

![action_list](https://cdn.illacloud.com/official-website/img/docs/action_list.png)

Conecte a la base de datos con los parámetros requeridos descritos en `Configuración de Conexión` a continuación.

Haga clic en `**Probar conexión**` para ver si podemos conectar con éxito a la base de datos. En caso afirmativo, haga clic en **`Guardar Recursos`**, lo demás, compruebe dos veces el nombre de host, el puerto, el nombre de usuario y la contraseña son correctos.

Después de crear un recurso, el Amazon S3 se mostrará como mostrado.

![amazon_ex_ready](https://cdn.illacloud.com/official-website/img/docs/assemble/amazon_ex_ready.jpg)

-   Crear en el Constructor

Inicia sesión en tu cuenta de Imax, crea un proyecto en Illa Builder en la página `**App**`, y navegar a la **`Lista de Acciones`** en la parte inferior de la página. Haz clic en `**nuevo**`, luego selecciona **`Amazon S3`** de la lista de bases de datos. Luego, conecte a la base de datos con los parámetros requeridos descritos en `Configuración de Conexión` a continuación.

Haga clic en `**Probar conexión**` para ver si podemos conectar con éxito a la base de datos. En caso afirmativo, haga clic en **`Guardar Recursos`**, lo demás, compruebe dos veces el nombre de host, el puerto, el nombre de usuario y la contraseña son correctos.

### Configuración de conexión

Aquí tenemos que proporcionar información para conectarse a la API de Amazon S3.

![amazon_config](https://cdn.illacloud.com/official-website/img/docs/assemble/amazon_config.jpg)

| Propiedades                      | Requerido | Descripción                                                                                                                                                                                                                           |
| -------------------------------- | --------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Nombre                           | requerido | El nombre del recurso al crear acciones en el ILLA.                                                                                                                                                                                   |
| Nombre del cubo                  | opcional  | un identificador único que representa un contenedor de almacenamiento en Amazon S3. Proporciona una separación lógica para organizar y acceder a tus archivos                                                                         |
| S3 ACL para archivos subidos     | opcional  | le permite definir los permisos de acceso finos para archivos dentro de su bucket. Especifica quién puede realizar operaciones específicas, como leer, escribir o borrar archivos.                                                    |
| Región                           | requerido | ubicación geográfica donde se almacena su cubo Amazon S3. Determina la ubicación física de sus datos y puede afectar factores como latencia y costos de transferencia de datos.                                                       |
| endpoint S3 personalizado        | opcional  | le permite configurar una URL de endpoints específica para acceder a sus recursos de Amazon S3. Puede ser útil para escenarios en los que desee utilizar un dominio personalizado o integrarse con un CDN (Content Delivery Network). |
| Clave de acceso                  | requerido | credenciales que autentifican sus solicitudes para acceder a Amazon S3                                                                                                                                                                |
| Clave de acceso secreta          | requerido | generado al crear un usuario de IAM (Identity and Access Management) o un par de claves de acceso de AWS (Amazon Web Services).                                                                                                       |

### Crear acciones

Hemos creado un recurso Amazon S3, podemos añadir la acción seleccionando Amazon S3 de la lista de acciones y eligiendo el botón **`Crear acción`** .

![amazon_resource_list](https://cdn.illacloud.com/official-website/img/docs/assemble/amazon_resource_list.jpg)

Ahora hemos añadido el servidor Amazon como una acción a nuestra página de construcción.

![amazon](https://cdn.illacloud.com/official-website/img/docs/assemble/amazon.jpg)

### Configurar Amazon S3

#### Resumen

| Nombre del método | Descripción                                                                                                                                                  |
| ----------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Tipo de acción    | listar todos los objetos en un cubeta, leer un objeto, descargar un objeto, eliminar un objeto, borrar varios objetos, subir datos, subir varios datos       |
| Nombre del cubo   | un identificador único que representa un contenedor de almacenamiento en Amazon S3.                                                                          |
| Transformador     | transforma los datos en el estilo que te gusta usando JavaScript                                                                                             |

#### Listar todos los objetos en un cubo

recuperar una lista de todos los objetos almacenados dentro de un cubo específico.

** Input **

| Propiedades                     | Requerido | Descripción                                                                                              |
| ------------------------------- | --------- | -------------------------------------------------------------------------------------------------------- |
| Prefijo para filtrar resultados | opcional  | filtrar los resultados para incluir sólo objetos cuya clave (nombre) comienza con un prefijo específico  |
| Delimitador                     | opcional  | especifica un carácter que separa las jerarquías de claves de objeto                                     |
| Generar URL firmada             | opcional  | si desea generar una URL de tiempo limitado que proporcione acceso temporal a un objeto específico       |
| Máximas claves                  | opcional  | limitar el número máximo de objetos devueltos en la respuesta                                            |

Por ejemplo, para obtener todos los objetos en la base de datos.

** Salida **

devuelve una matriz de objetos, cada uno incluyendo al menos una propiedad `objectKey`.

![amazon_list_all_data](https://cdn.illacloud.com/official-website/img/docs/assemble/amazon_list_all_data1.jpg)

Usa `{{s31.data}}` para obtener todos los datos devueltos.

![amazon_list_all_code](https://cdn.illacloud.com/official-website/img/docs/assemble/amazon_list_all_code1.jpg)

#### Leer un objeto

recuperar el contenido de un objeto específico almacenado en un cubo

** Input **

| Propiedades               | Requerido | Descripción                                                                               |
| ------------------------- | --------- | ----------------------------------------------------------------------------------------- |
| Clave del objeto          | requerido | el nombre o la ruta del objeto dentro del cubo                                            |
| Generar URL firmada       | opcional  | generar una URL de tiempo limitado que proporciona acceso temporal a un objeto específico |

Por ejemplo, para obtener el contenido del documento '1111.txt', pon `1111.txt` para **`Object key`**.

** Salida **

devolver el contenido y los metadatos de un documento

![amazon_read_data1](https://cdn.illacloud.com/official-website/img/docs/assemble/amazon_read_data1.jpg)

![amazon_read_data2](https://cdn.illacloud.com/official-website/img/docs/assemble/amazon_read_data2.jpg)

Usa `{{s31.data}}` para obtener los datos devueltos.

![amazon_read_code](https://cdn.illacloud.com/official-website/img/docs/assemble/amazon_read_code.jpg)

#### Eliminar múltiples objetos

elimina varios objetos simultáneamente de un cubo S3

** Input **

| Propiedades               | Requerido | Descripción                                                                               |
| ------------------------- | --------- | ----------------------------------------------------------------------------------------- |
| Clave del objeto          | requerido | el nombre o la ruta del objeto dentro del cubo                                            |
| Generar URL firmada       | opcional  | generar una URL de tiempo limitado que proporciona acceso temporal a un objeto específico |

Por ejemplo, para eliminar múltiples documentos, pon `{{["images.jpeg", "images-1.png"]}}` para **`clave de objeto`**.

** Salida **

#### Subir datos

subiendo un objeto en un cubo

** Input **

| Propiedades                                      | Requerido | Descripción                                                                                                   |
| ------------------------------------------------ | --------- | ------------------------------------------------------------------------------------------------------------- |
| Tipo de contenido                                | requerido | Tipo MIME de objeto que está subiendo.                                                                        |
| Subir nombre del objeto                          | requerido | el nombre del objeto que está subiendo.                                                                       |
| Subir datos                                      | requerido | los datos que desea subir a Amazon S3.                                                                        |
| Tiempo de espera para cargar un archivo (Minuto) | opcional  | el tiempo de subida en minutos antes de que el proceso de carga sea detenido por un error de tiempo de espera |

Por ejemplo, para eliminar múltiples documentos, pon `text` para **`content type`**, `text.txt` para **`upload object name`**, y `text` para **`upload data`**.

** Salida **

devolver el mensaje de si el documento se ha subido correctamente

![amazon_upload_data1](https://cdn.illacloud.com/official-website/img/docs/assemble/amazon_upload_data1.jpg)

![amazon_upload_data2](https://cdn.illacloud.com/official-website/img/docs/assemble/amazon_upload_data2.jpg)

#### Subir múltiples datos

subir una lista de objetos en un cubo

** Input **

| Propiedades                                      | Requerido | Descripción                                                                                                   |
| ------------------------------------------------ | --------- | ------------------------------------------------------------------------------------------------------------- |
| Tipo de contenido                                | requerido | Tipo MIME de objeto que está subiendo.                                                                        |
| Subir lista de nombres de objeto                 | requerido | una lista de nombres de los objetos que está subiendo.                                                        |
| Subir lista de datos                             | requerido | una lista de los datos que desea subir a Amazon S3.                                                           |
| Tiempo de espera para cargar un archivo (Minuto) | opcional  | el tiempo de subida en minutos antes de que el proceso de carga sea detenido por un error de tiempo de espera |

Por ejemplo, para borrar múltiples documentos, pon `text` para **`content type`**, `{{['text.txt', 'text2.txt']}}` for **`upload object name list`**, and `{{['text', 'text2']}}` for **`upload data list`**.

** Salida **

devolver el mensaje de si los documentos se han subido correctamente

![amazon_upload_data1](https://cdn.illacloud.com/official-website/img/docs/assemble/amazon_upload_data1.jpg)

![amazon_upload_data2](https://cdn.illacloud.com/official-website/img/docs/assemble/amazon_upload_data2.jpg)

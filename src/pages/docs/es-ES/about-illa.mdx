## <h2 hidden>Acerca de ILLA</h2>

ILLA le permite construir e implementar aplicaciones internas rápidamente. Conéctate a tus bases de datos y APIs, ensamble UIs con bloques de construcción de arrastrar y soltar como tablas y formularios, y escribir consultas para interactuar con datos usando SQL y JavaScript.

### ¿Qué es ILLA?

ILLA es una plataforma de código bajo abierto para que los desarrolladores construyan e implementen herramientas internas de manera notablemente rápida.

<img
  src='https://cdn.illacloud.com/official-website/img/docs/introduction/about-illa-cloud.png'
  alt=""
/>

Con ILLA usted puede conectar fácilmente a varias fuentes de datos como bases de datos, puntos finales API/GraphQL, herramientas SaaS y servicios de almacenamiento en la nube. Estas incluyen opciones populares como PostgreSQL, MongoDB, MySQL, Stripe, Google Sheets y AWS S3. Una vez conectado, ILLA puede ejecutar consultas para obtener y actualizar datos de estas fuentes. La interfaz de usuario (UI) proporciona componentes convenientes para visualizar y modificar los datos, tales como tablas, gráficos y formularios.

Con ILLA, usted puede construir rápida y fácilmente una interfaz de usuario (UI) usando widgets preconstruidos que usted puede arrastrar y soltar sobre un lienzo estilo cuadrícula. Esto simplifica la integración del front-end y back-end de su aplicación y optimiza el proceso de construcción. Además, ILLA soporta JavaScript dentro de widgets, consultas y otros componentes, permitiéndole agregar lógica, transformar datos y definir complejos flujos de trabajo. Esto hace que sea fácil personalizar su aplicación a sus necesidades específicas.

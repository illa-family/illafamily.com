## <h2 hidden>¿Qué es ClicHouse?</h2>

ClickHouse es un rápido sistema de gestión de bases de datos orientado a columnas que está diseñado para la carga de trabajo de procesamiento analítico (OLAP). Es un proyecto de código abierto creado por Yandex, la compañía rusa de motores de búsqueda. ClickHouse es conocida por su capacidad de procesar grandes cantidades de datos de forma rápida y eficiente, convirtiéndola en una solución ideal para grandes análisis de datos.

Con Illa Cloud, usted puede desplegar y administrar clústeres en la nube, sin tener que preocuparse por la infraestructura y el mantenimiento subyacentes.

Illa Cloud proporciona una interfaz web para administrar clústeres de ClickHouse, permitiéndole monitorizar el rendimiento del clúster, escalar recursos hacia arriba o hacia abajo y configurar las copias de seguridad y los ajustes de seguridad. También puede utilizar Illa Cloud para ejecutar consultas en los datos de ClickHouse usando un editor de consultas basado en la web.

### Crear ClickHouse

Hay dos maneras de crear un recurso en Illa después de iniciar sesión en tu cuenta de Imax.

-   Crear en Recursos

Inicia sesión en tu cuenta de Imax, selecciona `**Recursos**` en la parte superior de la página y haz clic en el botón `**Crear nuevo**`.

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

Seleccione **`ClickHouse`** de la lista de bases de datos.

![action_list](https://cdn.illacloud.com/official-website/img/docs/action_list.png)

Conecte a la base de datos con los parámetros requeridos descritos en `Configuración de Conexión` a continuación.

Haga clic en `**Probar conexión**` para ver si podemos conectar con éxito a la base de datos. En caso afirmativo, haga clic en **`Guardar Recursos`**, lo demás, compruebe dos veces el nombre de host, el puerto, el nombre de usuario y la contraseña son correctos.

Después de crear un recurso, el ClickHouse listo se mostrará como se muestra.

![click_config](https://cdn.illacloud.com/official-website/img/docs/click_config.png)

-   Crear en el Constructor

Inicia sesión en tu cuenta de Imax, crea un proyecto en Illa Builder en la página `**App**`, y navegar a la **`Lista de Acciones`** en la parte inferior de la página. Haga clic en `**nuevo**`, luego seleccione **`ClickHouse`** de la lista de bases de datos. Luego, conecte a la base de datos con los parámetros requeridos descritos en `Configuración de Conexión` a continuación.

Haga clic en `**Probar conexión**` para ver si podemos conectar con éxito a la base de datos. En caso afirmativo, haga clic en **`Guardar Recursos`**, lo demás, compruebe dos veces el nombre de host, el puerto, el nombre de usuario y la contraseña son correctos.

### Configuración de conexión

Aquí tenemos que proporcionar información para conectarse a la base de datos ClickHouse.

![click_config](https://cdn.illacloud.com/official-website/img/docs/click_config.png)

| Propiedades      | Requerido | Descripción                                                                                                                      |
| ---------------- | --------- | -------------------------------------------------------------------------------------------------------------------------------- |
| Nombre           | requerido | El nombre del recurso al crear acciones en el ILLA.                                                                              |
| Hostname         | requerido | La URL o dirección IP de su base de datos                                                                                        |
| Puerto           | requerido | El número de puerto del servidor que debe utilizar para conectarse. Si no especifica un puerto, el puerto por defecto es '9440'. |
| Base de datos    | requerido | El nombre de la base de datos                                                                                                    |
| Usuario          | requerido | el nombre de usuario que desea utilizar al iniciar sesión en el servidor ClickHouse.                                             |
| Contraseña       | requerido | Utilice esta contraseña para la autenticación.                                                                                   |
| Opciones SSL     | opcional  | decide hasta qué punto se prioriza una conexión segura TCP/IP SSL mientras se negocia con el servidor.                           |

### Crear acciones

Hemos creado un recurso ClickHouse, podemos añadir la acción seleccionando ClickHouse de la lista de acciones y eligiendo el botón **`Crear acción`**.

![click_resource_list](https://cdn.illacloud.com/official-website/img/docs/clik_resource_list.png)

Ahora hemos añadido el servidor ClickHouse como una acción a nuestra página de construcción.

![click](https://cdn.illacloud.com/official-website/img/docs/click.png)

### Configure ClickHouse

| Propiedades    | Descripción                                                                                    |
| -------------- | ---------------------------------------------------------------------------------------------- |
| Consulta SQL   | comandos sql para buscar (seleccionar), insertar, actualizar, borrar datos de la base de datos |
| Transformador  | transforma los datos en el estilo que te gusta usando JavaScript                               |

Ejemplo:

-   Seleccionar

```sql
SELECT * DE usuarios ORDER POR apellido;
```

-   Insert

```sql
INSERT INTO users

	(firstname, lastname, email)

VALUES

	(

		{{ NewUser.data.firstNameInput }},

		{{ NewUser.data.lastNameInput }},

		{{ NewUser.data.emailInput }}

	);
```

-   Actualizar

```sql
UPDATE users
  SET firstname = '{{form1.updatedProfile.first}}'
  WHERE id = {{ form1.updatedProfile.uid}};
```

-   Eliminar

```sql
DELETE FROM users WHERE id = {{ form1.updatedProfile.uid }};
```

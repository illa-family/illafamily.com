## <h2 hidden>¿Qué es la línea de tiempo?</h2>

Un componente de línea de tiempo es una representación gráfica de eventos o puntos de datos ordenados cronológicamente a lo largo de un eje horizontal o vertical. Permite a los usuarios visualizar y navegar por una serie de eventos o hitos de forma lineal.

### Propiedades

Propiedades disponibles de la línea de tiempo. JavaScript puede ser escrito para leer o alterar información sobre componentes.

| Propiedades    | Descripción                                                                                     |
| -------------- | ----------------------------------------------------------------------------------------------- |
| Elementos      | la recopilación de eventos individuales o puntos de datos que se muestran en la línea de tiempo |
| Dirección      | la orientación o disposición del componente de línea de tiempo (horizontal o vertical)          |
| Pendiente      | el estado de un elemento en la línea de tiempo                                                  |
| Altura         | tamaño o altura vertical del componente de línea de tiempo                                      |
| Hidden         | si el componente de línea de tiempo es visible u oculto                                         |

### Métodos

Puede usar otros componentes para controlar el componente. Apoyamos los dos métodos siguientes:

-   **setValue**

Para establecer el valor de entrada del área de texto, por ejemplo, {{"value1"}}

| Propiedades | Descripción |
| ----------- | ----------- |
| Valor       | Input value |

-   **limpiarValor**

Limpiar el valor del componente seleccionado

### Datos

El componente tiene algunos datos comúnmente utilizados, que pueden llamarse a través de **`{{componentName.propertyName}}`** en la aplicación.

| Nombre de la propiedad      | Descripción                                                                                                                |
| --------------------------- | -------------------------------------------------------------------------------------------------------------------------- |
| dirección                   | orientación o diseño de la línea de tiempo                                                                                 |
| altura dinámica             | la capacidad de ajustar la altura de la línea de tiempo dinámicamente basándose en su contenido u otros factores.          |
| nombre de pantalla          | el nombre de este componente (es decir, timeline1)                                                                         |
| hidden                      | si el componente de línea de tiempo es visible u oculto                                                                    |
| objetos                     | la recopilación de eventos individuales o puntos de datos mostrados en la línea de tiempo                                  |
| pendiente                   | el estado de un elemento en la línea de tiempo                                                                             |
| cambiar tamaño de dirección | la dirección en la que el componente de línea de tiempo puede ser redimensionado por el usuario                            |

Ejemplo: {{timeline1.items}}

### Caso de uso

A continuación, demostraremos cómo mapear los datos desde la fuente de datos al componente cronológico.

#### Paso 1 Añadir una acción

Vamos a crear una tabla en Supabase llamada **`timeline`,** incluyendo sólo una columna: hitos. Hay 4 hitos que queremos asignar al componente timeline.

Luego podemos crear una nueva acción para Supabase a partir de la lista de acciones y nombrar supabasedb1.

Para listar todos los datos en el hito, ponga el fragmento de código abajo en la sección de consultas SQL.

```sql
seleccionar * de la línea de tiempo;
```

Dado que los datos de salida son una lista de pares de diccionarios, queremos transformarlos en una matriz. En Transformer, actívalo y ponga el código abajo.

```jsx
const transformedData = data.map((obj) => Object.values(obj)[0]);
return transformedData;
```

Haz clic en **`Guardar`** y **`Ejecutar`** para activar esta acción.

![timeline_data](https://cdn.illacloud.com/official-website/img/official-site/components/timeline_data.jpg)

#### Paso 2 Añadir componentes

A continuación, podemos añadir un componente **`timeline`** al lienzo.

En el campo `**Artículos**` en la página de inspección después de hacer clic en el componente, ponga `{{supabasedb1.data}}` para recuperar los datos de la sumisión.

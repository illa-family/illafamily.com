## <h2 hidden>¿Qué es el Formulario?</h2>

El componente de formulario es un elemento visual que representa el progreso o la finalización de una tarea o proceso. Muestra una barra horizontal que se llena o se recorta basándose en el valor de progreso especificado.

### Propiedades

| Propiedades                                 | Descripción                                                                                                                                                          |
| ------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Controlador de eventos                      | detectar y responder a acciones o eventos específicos del usuario, tales como clics, pulsaciones de teclas o envíos de formularios.                                  |
| Deshabilitado                               | no interactivo y no puede ser modificado o activado por el usuario.                                                                                                  |
| Desactivar envío                            | desactivar la funcionalidad de envío de formularios.                                                                                                                 |
| Validar entradas al enviar                  | las entradas del formulario deben ser validadas para ser correctas o completas cuando el formulario es enviado.                                                      |
| Restablecer después de enviar correctamente | reinicia automáticamente sus valores y estado a sus valores iniciales o predeterminados después de un envío exitoso                                                  |
| Sugerencia                                  | un pequeño mensaje emergente o informativo que aparece cuando el usuario se desplaza o interactúa con un elemento específico, como un componente de formulario.      |
| Mostrar cabecera                            | si un componente de formulario debe mostrar una cabecera o sección de título                                                                                         |
| Mostrar pie de página                       | si un componente de formulario debe mostrar una sección de pie de página                                                                                             |
| Hidden                                      | oculta el componente del formulario de la vista del usuario                                                                                                          |
| Color del borde                             | color del borde que rodea un componente del formulario                                                                                                               |
| Radio de borde                              | curvatura de las esquinas del borde de un componente                                                                                                                 |
| Border Width                                | grosor o ancho del borde que rodea un componente del formulario                                                                                                      |
| Fondo                                       | define el color o imagen que rellena el área de contenido de un componente de formulario                                                                             |
| Sombra                                      | efecto visual de añadir un sutil desplazamiento "sombra" a un componente de formulario, haciéndolo aparecer levemente elevado o flotante sobre el fondo              |

### Método

Puede usar otros componentes para controlar el componente. Apoyamos los dos métodos siguientes:

-   **Enviado**

Para enviar la información rellenada en el campo del formulario

-   **reset**

reiniciar todos los campos de entrada en el componente del formulario

-   **setValue**

Para establecer el valor de entrada del área de texto, por ejemplo, {{"value1"}}

| Propiedades | Descripción |
| ----------- | ----------- |
| Valor       | Input value |

-   validate

Para verificar que la información de entrada es legal

### Datos

El componente tiene algunos datos comúnmente utilizados, que pueden llamarse a través de **`{{componentName.propertyName}}`** en la aplicación.

| Nombre de la propiedad         | Descripción                                                                                                                                                              |
| ------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Color de fondo                 | especifica el color de fondo de un componente de formulario                                                                                                              |
| bordeColor                     | define el color del borde que rodea un componente del formulario                                                                                                         |
| borderWidth                    | determina el grosor o ancho del borde que rodea un componente del formulario                                                                                             |
| deshabilitado                  | si un componente de formulario está deshabilitado o no                                                                                                                   |
| nombre de pantalla             | representa el nombre o la etiqueta asignada a un componente del formulario                                                                                               |
| Altura del pie                 | especifica la altura o tamaño de la sección del pie de página de un componente del formulario                                                                            |
| formData                       | contiene los datos o valores introducidos por el usuario en un componente del formulario                                                                                 |
| altura de cabecera             | determina la altura o tamaño de la sección de cabecera de un componente del formulario                                                                                   |
| no válido                      | si la entrada o el valor de un componente de formulario se considera inválido o no.                                                                                      |
| radius                         | define el radio del borde o la curvatura de las esquinas de un componente de forma                                                                                       |
| resetAfterSuccessful           | indica que un componente de formulario debe restablecer automáticamente sus valores y estado a sus valores iniciales o predeterminados después de un envío exitoso       |
| sombra                         | controla la presencia y apariencia de un efecto de sombra sobre un componente del formulario                                                                             |
| mostrar pie                    | determina si la sección de pie de página de un componente de formulario debe mostrarse o no                                                                              |
| mostrar cabecera               | visibilidad de la sección de cabecera de un componente de formulario                                                                                                     |
| tooltipText                    | contiene el texto o el contenido de una descripción asociada con un componente de formulario                                                                             |
| validar un mensaje enviado     | especifica que los valores de entrada de un componente de formulario deben ser validados para ser correctos o completos cuando el formulario es enviado                  |

Ejemplo: {{form1.disabled}}

### Caso de uso

Demostraremos cómo hacer un formulario con nombre y edad como entrada, así como el progreso de la barra para mostrar el progreso de terminación del formulario.

#### Paso 1 Añadir componentes

Añade un componente **`form`** con el texto predeterminado "Forma" y un botón "Enviar" a lienzo.

Añade un componente **`input`**, un componente **`number input`** y un componente **`bar progreso`** de `**Insert**` en el formulario.

Etiqueta el componente de entrada a ser "Nombre", el componente de entrada de número a ser "Edad" y el componente de progreso de la barra a ser "Progresar".

Establezca el marcador de posición del componente de entrada a "Por favor ponga su nombre aquí" y el valor por defecto de la entrada de número a 0 como se muestra a continuación.

![bar_layout](https://cdn.illacloud.com/official-website/img/official-site/components/bar_layout.jpg)

#### Paso 2 Configurar el componente

Para los componentes **`input`** y **`number input`**, podemos configurarlos para que establezcan el valor en el componente **`bar progress`** para reflejar el progreso de la terminación.

1.  Haz clic en el componente **`input`** y **`number input`** para abrir su página de inspección. Bajo **`Controlador de eventos`**, haga clic en `**+ Nuevo**`.
2.  En el controlador de edición de eventos, seleccione **`componente de control`** en acción, y seleccione el componente **`barra de progreso`** que desea actualizar como objetivo del evento.
3.  Elige la acción **`setValue`** y establece el valor como

```jsx
{
  {
    input1.value && numberInput1.value
      ? "100"
      : input1.value || numberInput1.value
      ? "50"
      : "0";
  }
}
```

![bar_input_num_config](https://cdn.illacloud.com/official-website/img/official-site/components/bar_input_num_config.jpg)

Nota: necesita establecer el manejador de eventos como se describe anteriormente a la entrada BOTH y a los componentes de entrada de números.

Para el componente **`botón`** podemos configurarlo para que envíe el formulario si el formulario está completado (es decir, el progreso de la barra 100%)

1.  Haga clic en el componente **`button`** para abrir su página de inspección. Bajo **`Controlador de eventos`**, haga clic en `**+ Nuevo**`.
2.  En el controlador de edición de eventos, seleccione **`componente de control`** en acción, y seleccione el componente **`form`** que desea actualizar como objetivo del evento.
3.  Elige el método **`submit`** y pon `{{barProgress1.value == 100}}` en **`Sólo ejecutar cuando`**

![bar_button_submit_config](https://cdn.illacloud.com/official-website/img/official-site/components/bar_button_submit_config.jpg)

Del mismo modo, podemos configurar el botón para mostrar advertencia cuando el usuario está intentando enviar un formulario incompleto.

1.  Haga clic en el componente **`button`** para abrir su página de inspección. Bajo **`Controlador de eventos`**, haga clic en `**+ Nuevo**`.
2.  En el controlador de edición de eventos, seleccione **`Mostrar notificación`** en acción, **`title`** "¡Error!" y **`description`** "Por favor, rellene el formulario antes de enviar". Elige `Error` para **`type`**, pon `{{2000}}` para la **`duración`** (2 segundos), y pon `{{barProgress1.value<100}}` para **`Sólo ejecutar cuando`.**

![bar_button_show_config](https://cdn.illacloud.com/official-website/img/official-site/components/bar_button_show_config.jpg)

## <h2 hidden>¿Qué es Dividente?</h2>

En ILLA Cloud, el componente divisor es un elemento visual que ayuda a separar o dividir secciones de contenido dentro de una aplicación web. Se trata de una línea horizontal o vertical simple que puede ser insertada entre diferentes secciones o componentes para mejorar la organización visual y la claridad de la interfaz de usuario.

### Propiedades

| Propiedades       | Descripción                                                                        |
| ----------------- | ---------------------------------------------------------------------------------- |
| Texto             | añadir información adicional al componente divisor                                 |
| Text align        | determina la alineación horizontal del texto dentro del componente divisor         |
| Hidden            | ocultar el componente divisor de la interfaz de usuario                            |
| Tamaño del texto  | ajustar el tamaño de fuente del texto dentro del componente divisor                |

### Datos

El componente tiene algunos datos comúnmente utilizados, que pueden llamarse a través de **`{{componentName.propertyName}}`** en la aplicación.

| Nombre de la propiedad | Descripción                                                                  |
| ---------------------- | ---------------------------------------------------------------------------- |
| esquema de color       | define el color del texto en el componente                                   |
| nombre de pantalla     | representa el nombre o la etiqueta asignada al componente                    |
| fs                     | sistema de archivos (fs) asociado con el divisor                             |
| hidden                 | valor booleano que determina si el componente es visible o oculto            |
| texto                  | el valor del texto                                                           |

Ejemplo: {{divider1.text}}

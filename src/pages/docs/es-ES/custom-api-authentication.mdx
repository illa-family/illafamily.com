## <Heading hidden>Autenticación API personalizada</Heading>

ILLA soporta muchos estándares de autenticación por defecto (ver aquí para más información). 
Adicionalmente, ILLA tiene un modo de autenticación de autenticación personalizada que le permite combinar varios pasos que soportan virtualmente cualquier autenticación de API. 
Estos métodos de autenticación pueden utilizarse tanto para los recursos REST como para GraphQL.

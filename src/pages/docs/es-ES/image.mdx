## <h2 hidden>¿Qué es Imagen?</h2>

El componente de imagen de ILLA Cloud es una herramienta versátil que permite a los usuarios añadir y mostrar imágenes dentro de sus aplicaciones. Proporciona varias funcionalidades para mejorar la experiencia visual de los usuarios.

### Propiedades

| Propiedades          | Descripción                                                                                                                                |
| -------------------- | ------------------------------------------------------------------------------------------------------------------------------------------ |
| Fuente de la imagen  | especificar la fuente de la imagen a mostrar                                                                                               |
| Texto Alt            | texto alternativo para la imagen. Se muestra cuando la imagen no puede ser cargada o para usuarios con discapacidades visuales             |
| Tipo de escala       | cómo la imagen es escalada dentro de las dimensiones del componente, incluyendo contenedor, cubierta, relleno, ninguna, escala.            |
| Manejador del Evento | permitiendo a los usuarios definir acciones activadas por interacciones del usuario con la imagen                                          |
| Sugerencia           | añadir una descripción o un texto descriptivo que aparece cuando el usuario pasa el cursor sobre la imagen                                 |
| Hidden               | controlar la visibilidad del componente de la imagen.                                                                                      |
| Styles               | permite a los usuarios personalizar su apariencia. Los usuarios pueden aplicar estilos como radio                                          |

### Método

Puede usar otros componentes para controlar el componente. Apoyamos los siguientes métodos:

-   **setImageUrl**

cambiar dinámicamente la fuente o URL de la imagen.

| Propiedades    | Descripción          |
| -------------- | -------------------- |
| URL de imagen  | url fuente de imagen |

### Controlador de eventos

| Evento | Descripción                                                                                    |
| ------ | ---------------------------------------------------------------------------------------------- |
| Click  | realizar acciones específicas o ejecutar código cuando se haga clic en el componente de imagen |

### Datos

El componente tiene algunos datos comúnmente utilizados, que pueden llamarse a través de **`{{componentName.propertyName}}`** en la aplicación.

| Nombre de la propiedad | Descripción                                                                                                                     |
| ---------------------- | ------------------------------------------------------------------------------------------------------------------------------- |
| altText                | una alternativa basada en texto que describe el contenido o propósito de la imagen                                              |
| nombre de pantalla     | Un valor de cadena que representa el nombre o el título de la imagen.                                                           |
| eventos                | Un array u objeto que define los controladores de eventos asociados con el componente de la imagen, como hacer clic             |
| hidden                 | Un valor booleano que determina si el componente de imagen está oculto en la interfaz de usuario.                               |
| imagen c               | la fuente o ubicación de la imagen a mostrar en el componente                                                                   |
| ajuste de objeto       | cómo se ajusta o escala la imagen dentro de las dimensiones del componente                                                      |
| radius                 | aplicar esquinas redondeadas al componente de la imagen.                                                                        |
| tooltipText            | que representa la descripción o pasa el texto a mostrar cuando el usuario pasa por encima del componente de la imagen.          |

Ejemplo: {{image1.displayName}}

### Caso de uso

A continuación, demostraremos cómo controlar la visualización de la imagen con un botón.

#### Paso 1 Añadir componentes

Añade un componente **`Image`** y un componente **`button`** al lienzo.

Puede establecer la `fuente de la imagen` del componente de la imagen a cualquier dirección de imagen que desee mostrar. Aquí usaremos la imagen predeterminada como ejemplo de inicio y etiquetaremos el botón como "Cambiar" como se muestra a continuación

![image_layout](https://cdn.illacloud.com/official-website/img/official-site/components/image_layout.jpg)

#### Paso 2 Configurar el componente

Para el componente **`button`** podemos configurarlo para establecer la fuente de la imagen del componente **`image`** a una nueva dirección de imagen.

1.  Haga clic en el componente `**botón**` para abrir su página de inspección. Bajo **`Controlador de eventos`**, haga clic en `**+ Nuevo**`.
2.  En el controlador de edición de eventos, seleccione **`componente de control`** en acción, y seleccione el componente **`image1`** que desea cambiar como objetivo del evento.
3.  Elige la acción **`setImageUrl`**
4.  Pon la nueva dirección de la imagen en el campo **`Valor`**

![image_config](https://cdn.illacloud.com/official-website/img/official-site/components/image_config.jpg)

#### Paso 3 Prueba

Ahora que al hacer clic en el botón "Reproducir", la imagen debe cambiar

## <h2 hidden>¿Qué son las estadísticas?</h2>

El componente de estadísticas en ILLA Cloud es una potente herramienta para mostrar y analizar datos numéricos. Proporciona una forma visualmente atractiva y personalizable de exhibir mediciones y estadísticas clave.

### Propiedades

| Propiedades                        | Descripción                                                                                                                                                          |
| ---------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Etiqueta                           | El nombre del campo mostrado al usuario                                                                                                                              |
| Valor primario                     | el valor principal o la calificación mostrada por el componente.                                                                                                     |
| Valor secundario                   | una calificación o valor adicional asociado con el valor primario, proporcionando información adicional o contexto.                                                  |
| Principal                          |                                                                                                                                                                      |
| Posición decimal                   | especificar el número de decimales que se pueden introducir en el componente                                                                                         |
| Mostrar signo de tendencia         | Habilitar esta propiedad muestra una flecha o signo indicando la tendencia o cambio en el valor de calificación.                                                     |
| Mostrar mil separadores            | Cuando se activa, esta propiedad añade mil separadores (coma o periodo) a los valores primarios y secundarios para una mejor legibilidad.                            |
| Activar color de tendencia         | cambiar el color basado en la tendencia o cambiar el valor de calificación.                                                                                          |
| Texto de prefijo                   | un fragmento corto de texto que aparece a la izquierda del campo de entrada, usado a menudo para proporcionar contexto o instrucciones adicionales al usuario        |
| Texto sufijo                       | un fragmento corto de texto que aparece a la derecha del campo de entrada, a menudo utilizado para proporcionar información adicional o retroalimentación al usuario |
| Secundario                         |                                                                                                                                                                      |
| Posición decimal                   | especificar el número de decimales que se pueden introducir en el componente                                                                                         |
| Mostrar signo de tendencia         | Habilitar esta propiedad muestra una flecha o signo indicando la tendencia o cambio en el valor de calificación.                                                     |
| Mostrar mil separadores            | Cuando se activa, esta propiedad añade mil separadores (coma o periodo) a los valores primarios y secundarios para una mejor legibilidad.                            |
| Activar color de tendencia         | cambiar el color basado en la tendencia o cambiar el valor de calificación.                                                                                          |
| Texto de prefijo                   | un fragmento corto de texto que aparece a la izquierda del campo de entrada, usado a menudo para proporcionar contexto o instrucciones adicionales al usuario        |
| Texto sufijo                       | un fragmento corto de texto que aparece a la derecha del campo de entrada, a menudo utilizado para proporcionar información adicional o retroalimentación al usuario |
| Controlador de eventos             | definir acciones o lógica personalizadas que se activan cuando el componente se interactúa o se cambia la calificación.                                              |
| Sugerencia                         | Los usuarios pueden ingresar la descripción del componente aquí. La descripción se mostrará cuando esté enfocada. Formato Markdown soportado.                        |

| Alineación                         | alineación del componente, como la izquierda, el centro o la derecha.                                                                                                |
| Color del tema                     | Permite a los usuarios especificar el color de fondo y opacidad del botón                                                                                            |
| Color positivo                     | determina el color usado para valoraciones positivas o altas                                                                                                         |
| Color negativo                     | define el color usado para calificaciones negativas o bajas.                                                                                                         |

### Método

Puede usar otros componentes para controlar el componente. Apoyamos los dos métodos siguientes:

-   **setPrimaryValue**

Para establecer el valor primario, por ejemplo, {{"value1"}}

| Propiedades     | Descripción                |
| --------------- | -------------------------- |
| Valor Principal | Valor principal de entrada |

-   **resetear valor primario**

Para restablecer el valor al valor por defecto del componente seleccionado. Si no se especifica el valor por defecto, limpia el valor.

### Controlador de eventos

| Evento | Descripción                                                                                              |
| ------ | -------------------------------------------------------------------------------------------------------- |
| Click  | Cuando un usuario hace clic en este botón, realiza una acción específica personalizada por los usuarios. |

| hidden                                 | estado oculto (verdadero o falso)                                                                                                       |

| valor primario                         | el valor numérico principal que se mostrará en el componente de estadísticas.                                                           |

| signo negativo                         | define el símbolo que se mostrará para valores negativos.                                                                               |

| valor primario                         | le permite configurar o actualizar programáticamente el valor principal del componente de estadísticas.                                 |

### Caso de uso

A continuación, demostraremos cómo mapear los datos de la fuente de datos a las estadísticas con un botón.

#### Paso 1 Añadir una acción

Creemos una tabla en Amazon S3. Uno de los documentos se llama

'1.txt' con el decimal '1.1111' como sus datos. Este es el valor que queremos mostrar en la entrada de números.

![s3data](https://cdn.illacloud.com/official-website/img/official-site/components/s3data.jpg)

Luego podemos crear una nueva acción para Amazon S3 a partir de la lista de acciones, listando todos los datos en el cubo y llamado s31.

Selecciona `Leer un objeto` para `**Tipo de acción**`. Pon `1.txt` para `**Clave de Objeto`.\*\*

Haz clic en **`Guardar`** y **`Ejecutar`** para activar esta acción.

![stat_data](https://cdn.illacloud.com/official-website/img/official-site/components/stat_data.jpg)

#### Paso 2 Añadir componentes

A continuación, podemos añadir un componente **`estadísticas`** y un componente **`botón`** al lienzo.

Establecemos el valor principal por defecto de las estadísticas a 0 y marcamos el botón como "Establecer valor" como se muestra a continuación

![stat_layout](https://cdn.illacloud.com/official-website/img/official-site/components/stat_layout.jpg)

#### Paso 3 Configurar el componente

Para el componente **`button`**, podemos configurarlo para establecer el valor en el componente **`estadísticas`** de los datos del documento que leemos de s31 api.

1.  En el controlador de edición de eventos, seleccione **`componente de control`** en acción, y seleccione el componente **`estadísticas`** que desea actualizar como objetivo del evento.
2.  Elige la acción **`setValue`** y establece el valor como `{{s31.data}}`

![stat_button_config](https://cdn.illacloud.com/official-website/img/official-site/components/stat_config.jpg)

#### Prueba de paso 4

Ahora que al hacer clic en el botón "Establecer valor", el valor debe cambiar a 1.1111.

![stat_test](https://cdn.illacloud.com/official-website/img/official-site/components/stat_test.gif)

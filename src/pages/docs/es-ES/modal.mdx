## <h2 hidden>¿Qué es Modal?</h2>

En ILLA Cloud, el componente modal es un elemento de interfaz de usuario que se utiliza para mostrar el contenido, , o acciones en una ventana enfocada y superpuesta. El componente modal aparece normalmente en la parte superior de la pantalla principal de la aplicación, interrumpiendo temporalmente el flujo de trabajo del usuario y llamando su atención sobre el contenido modal.

### Propiedades

| Propiedades                            | Descripción                                                                                                                  |
| -------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------- |
| Controlador de eventos                 | detectar y responder a acciones o eventos específicos del usuario, tales como clics, pulsaciones de teclas, cambios de valor |
| Haga clic en máscara para cerrar modal | si hace clic en la superposición de fondo o en la máscara de la modal cerrará la ventana modal                               |
| Mostrar cabecera                       | si el componente debe mostrar una cabecera o sección de título                                                               |
| Mostrar pie de página                  | si el componente debe mostrar una sección de pie de página                                                                   |
| Color del borde                        | color del borde que rodea un componente modal                                                                                |
| Radio de borde                         | curvatura de las esquinas del borde de un componente modal                                                                   |
| Border Width                           | grosor o ancho del borde que rodea un componente modal                                                                       |
| Fondo                                  | define el color o imagen que rellena el área de contenido de un componente modal                                             |

### Método

Puede usar otros componentes para controlar el componente. Apoyamos los dos métodos siguientes:

-   openModal

abrir el componente modal

-   cerrar Modal

cerrar el componente modal

### Caso de uso

Demostraremos cómo hacer un formulario con nombre y edad como entrada, así como el progreso de la barra para mostrar el progreso de terminación del formulario.

#### Paso 1 Añadir componentes

Agrega un componente **`button`** etiquetado como "Abrir Modal"

Agrega un componente modal al lienzo.

#### Paso 2 Configurar el componente

Para los componentes "Abrir Modal" **`button`** podemos configurarlo para abrir el componente **`modal`** si se hace clic.

1.  Haga clic en el componente **`button`** para abrir su página de inspección. Bajo **`Controlador de eventos`**, haga clic en `**+ Nuevo**`.
2.  En el controlador de edición de eventos, seleccione **`componente de control`** en acción, y selecciona el componente **`modal`** que quieres actualizar como objetivo del evento.
3.  Elige el método **`openModal`**

![open_modal](https://cdn.illacloud.com/official-website/img/docs/open_modal.jpg)

Haga clic en este botón para abrir el componente modal y puede poner lo que desee en el área central de la modal.

Para el componente "Cancelar" **`button`** podemos configurarlo para cerrar la modal si se hace clic

1.  Haga clic en el componente **`button`** para abrir su página de inspección. Bajo **`Controlador de eventos`**, haga clic en `**+ Nuevo**`.
2.  En el controlador de edición de eventos, seleccione **`componente de control`** en acción, y selecciona el componente **`modal`** que quieres actualizar como objetivo del evento.
3.  Elige el método **`closeModal`**

![close_modal](https://cdn.illacloud.com/official-website/img/docs/close_modal.jpg)

Del mismo modo, podemos configurar el botón de "Confirmar" a "Confirmar"

1.  Haga clic en el componente **`button`** para abrir su página de inspección. Bajo **`Controlador de eventos`**, haga clic en `**+ Nuevo**`.
2.  En el manejador de eventos Edición, selecciona **`Mostrar notificación`** en acción, **`title`** "¡Felicidades!" y **`description`** "¡Felicidades!". Elige `Éxito` para **`type`**, pon `{{2000}}` para la **`duración`** (2 segundos)

![modal_confirm](https://cdn.illacloud.com/official-website/img/docs/modal_confirm.jpg)

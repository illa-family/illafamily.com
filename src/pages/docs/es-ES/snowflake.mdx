## <h2 hidden>¿Qué es el Snowflake?</h2>

En Illa, puede utilizar el editor de consultas Snowflake para ejecutar consultas SQL contra su almacén de datos Snowflake. Puede conectarse a Snowflake ingresando las credenciales de su cuenta de Snowflake, incluyendo el nombre de la cuenta, el nombre de usuario, la contraseña y el nombre del almacén. Una vez que se haya conectado a Snowflake, puede utilizar el editor de consultas para escribir y ejecutar consultas SQL para recuperar, manipular y analizar sus datos.

### Crear Copo de Nieve

Hay dos maneras de crear un recurso en Illa después de iniciar sesión en tu cuenta de Imax.

-   Crear en Recursos

Inicia sesión en tu cuenta de Imax, selecciona `**Recursos**` en la parte superior de la página y haz clic en el botón `**Crear nuevo**`.

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

Selecciona **`Snowflake`** de la lista de bases de datos.

![action_list](https://cdn.illacloud.com/official-website/img/docs/action_list.png)

Conecte a la base de datos con los parámetros requeridos descritos en `Configuración de Conexión` a continuación.

Haga clic en `**Probar conexión**` para ver si podemos conectar con éxito a la base de datos. En caso afirmativo, haga clic en **`Guardar Recursos`**, lo demás, compruebe dos veces el nombre de host, el puerto, el nombre de usuario y la contraseña son correctos.

Después de crear un recurso, el Copo de Nieve listo se mostrará como mostrado.

![snow_ex_ready](https://cdn.illacloud.com/official-website/img/docs/snow_ex_Ready.png)

-   Crear en el Constructor

Inicia sesión en tu cuenta de Imax, crea un proyecto en Illa Builder en la página `**App**`, y navegar a la **`Lista de Acciones`** en la parte inferior de la página. Haz clic en `**nuevo**`, luego selecciona **`Copo de nieve`** de la lista de bases de datos. Luego, conecte a la base de datos con los parámetros requeridos descritos en `Configuración de Conexión` a continuación.

Haga clic en `**Probar conexión**` para ver si podemos conectar con éxito a la base de datos. En caso afirmativo, haga clic en **`Guardar Recursos`**, lo demás, compruebe dos veces el nombre de host, el puerto, el nombre de usuario y la contraseña son correctos.

### Configuración de conexión

Aquí tenemos que proporcionar información para conectarse a la base de datos de Snowflake.

![snow_config](https://cdn.illacloud.com/official-website/img/docs/snow_config.png)

| Propiedades         | Requerido | Descripción                                                                                                                                                        |
| ------------------- | --------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Nombre              | requerido | El nombre del recurso al crear acciones en el ILLA.                                                                                                                |
| Nombre de cuenta    | requerido | identificador único que está asignado a cada cuenta de Copo de Nieve                                                                                               |
| Almacén             | requerido | un clúster de recursos de cómputo que se utilizan para procesar consultas y ejecutar cargas de trabajo analíticas                                                  |
| Base de datos       | requerido | El nombre de la base de datos                                                                                                                                      |
| Esquema             | opcional  | un contenedor que contiene objetos de base de datos como tablas, vistas y procedimientos                                                                           |
| Rol                 | opcional  | un conjunto de privilegios nombrados que pueden ser asignados a uno o más usuarios                                                                                 |
| Autenticación       | requerido | acceso seguro a sus datos y recursos en el almacén de datos de Copo de Nieve (Auth Básica o par de claves (sólo se admiten claves privadas cifradas))              |
| Usuario             | requerido | el nombre de usuario que deseas usar al iniciar sesión en el servidor Snowflake.                                                                                   |
| Contraseña          | requerido | Utilice esta contraseña para la autenticación.                                                                                                                     |

### Crear acciones

Hemos creado un recurso de Copo de Nieve y podemos añadir la acción seleccionando Copo de Nieve de la lista de acciones y eligiendo el botón **`Crear acción`** .

![snow_resource_list](https://cdn.illacloud.com/official-website/img/docs/snow_resource_list.png)

Ahora hemos añadido el servidor Snowflake como una acción a nuestra página de construcción.

![snow](https://cdn.illacloud.com/official-website/img/docs/snow.png)

### Configurar Copo de Nieve

| Propiedades    | Descripción                                                                                    |
| -------------- | ---------------------------------------------------------------------------------------------- |
| Consulta SQL   | comandos sql para buscar (seleccionar), insertar, actualizar, borrar datos de la base de datos |
| Transformador  | transforma los datos en el estilo que te gusta usando JavaScript                               |

Ejemplo:

-   Seleccionar

```sql
SELECT * DE usuarios ORDER POR apellido;
```

-   Insert

```sql
INSERT INTO users

	(firstname, lastname, email)

VALUES

	(

		{{ NewUser.data.firstNameInput }},

		{{ NewUser.data.lastNameInput }},

		{{ NewUser.data.emailInput }}

	);
```

-   Actualizar

```sql
UPDATE users
  SET first name = '{{form1.updatedProfile.first}}'
  WHERE id = {{ form1.updatedProfile.uid}};
```

-   Eliminar

```sql
DELETE FROM users WHERE id = {{ form1.updatedProfile.uid }};
```

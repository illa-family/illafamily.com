## <h2 hidden>¿Qué es Airtable?</h2>

Airtable es una base de datos relacional y una plataforma de hojas de cálculo basada en la nube que combina la simplicidad de una hoja de cálculo con la complejidad de una base de datos tradicional. Permite a los usuarios crear y organizar datos estructurados en una interfaz amigable, haciéndola útil para diversas tareas como gestión de proyectos, planificación de contenido y recopilación de datos.

### Configura tu Back end en Airtable

Crea una cuenta en 'Airtable' y crea tu base según tu necesidad.

Para conectar su base a ILLA Cloud, vaya al `hub de desarrollador`.

![air_developer_hub](https://cdn.illacloud.com/official-website/img/official-site/intergration/air_developer_hub.png)

En la página `Token de Acceso Personal`, haga clic en `Crear nuevo token`.

![air_create_new](https://cdn.illacloud.com/official-website/img/official-site/intergration/air_create_new.jpeg)

Esto abrirá las opciones de configuración para el PAT que está creando.

Primero nombra el PAT, luego elija los ámbitos a los que le gustaría tener acceso este PAT. Por último, elija qué nivel de acceso tendrá este PAT.

![air_new_token](https://cdn.illacloud.com/official-website/img/official-site/intergration/air_new_token.jpeg)

Haz clic en el botón **`crear token`** , ahora tenemos un PAT listo para conectarse.

### Crear Airtable en la nube de ILLA

Hay dos maneras de crear un recurso en Illa después de iniciar sesión en tu cuenta de Imax.

-   Crear en Recursos

Inicia sesión en tu cuenta de Imax, selecciona `**Recursos**` en la parte superior de la página y haz clic en el botón `**Crear nuevo**`.

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

Selecciona **`Airtable`** de la lista de bases de datos.

![action_list_3](https://cdn.illacloud.com/official-website/img/official-site/intergration/action_list_3.jpeg)

Conecte a la base de datos con los parámetros requeridos descritos en `Configuración de Conexión` a continuación. En este caso, rellene el **`Token de Acceso Personal`** que copiamos de `Airtable` y péguelo en **`Token de Acceso`** en la configuración.

Haga clic en **`Guardar Recursos`** para \*\*\*\*\*crear un recurso.

El listo `Airtable` se mostrará como se muestra.

![air_ex_ready](https://cdn.illacloud.com/official-website/img/official-site/intergration/air_ex_ready.jpeg)

-   Crear en el Constructor

Inicia sesión en tu cuenta de Imax, crea un proyecto en Illa Builder en la página `**App**` y navegar a la **`Lista de Acciones`** en la parte inferior de la página. Haz clic en `**nuevo**`, luego selecciona **`Airtable`** de la lista de bases de datos. Luego, conecte a la base de datos con los parámetros requeridos descritos en `Configuración de Conexión` a continuación.

Haga clic en `**Probar conexión**` para ver si podemos conectar con éxito a la base de datos. En caso afirmativo, haga clic en **`Guardar Recursos`**, lo demás, compruebe dos veces el nombre de host, el puerto, el nombre de usuario y la contraseña son correctos.

### Configuración de conexión

Aquí tenemos que proporcionar información para conectarse a la API Airtable.

![air_config](https://cdn.illacloud.com/official-website/img/official-site/intergration/air_config.jpeg)

| Propiedades     | Requerido | Descripción                                                                                                                                                         |
| --------------- | --------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Nombre          | requerido | El nombre del recurso al crear acciones en el ILLA.                                                                                                                 |
| Token de acceso | requerido | una pieza de información que otorga permiso a una aplicación o servicio de terceros para acceder a los datos de su cuenta Airtable y realizar acciones en su nombre |

### Crear acciones

Hemos creado un recurso Airtable, podemos añadir la acción seleccionando Airtable de la lista de acciones y eligiendo el botón **`Crear acción`**.

![air_resource_list](https://cdn.illacloud.com/official-website/img/official-site/intergration/air_resource_list.jpeg)

Ahora hemos añadido la API Airtable como acción a nuestra página de construcción.

![air](https://cdn.illacloud.com/official-website/img/official-site/intergration/air.jpeg)

### Configurar Airtable

| Propiedades      | Descripción                                                                                                                                       |
| ---------------- | ------------------------------------------------------------------------------------------------------------------------------------------------- |
| ID base          | un identificador único para su base de datos Airtable. Puede obtener el ID base de su URL base. Normalmente es una cadena que comienza con "app". |
| Nombre de tabla  | especifica la tabla específica dentro de la base elegida con la que desea interactuar                                                             |
| Tipo de acción   | qué tipo de operación realizará su integración en la tabla especificada                                                                           |
| Transformador    | transforma los datos en el estilo que te gusta usando JavaScript                                                                                  |

#### Registros de Lista

recupera una lista de registros de una tabla especificada en Airtable

** Input **

| Propiedades               | Requerido | Descripción                                                                                               |
| ------------------------- | --------- | --------------------------------------------------------------------------------------------------------- |
| Ver                       | opcional  | organizar y mostrar registros basados en criterios específicos                                            |
| Campos                    | opcional  | Los campos se refieren a las columnas dentro de su tabla de Airtable                                      |
| Filtrar por fórmula       | opcional  | definir una fórmula que filtre los registros que desea recuperar o interactuar con                        |
| Ordenar                   | opcional  | el orden en el que se presentan o recuperan los registros                                                 |
| Máximos registros         | opcional  | limita el número de registros con los que tu integración obtiene o interactúa en una sola solicitud       |
| Tamaño de página          | opcional  | usado a menudo en escenarios donde los datos necesitan ser recuperados en varios lotes (páginas)          |
| Desplazamiento            | opcional  | especifica el punto de partida para recuperar registros en una página específica                          |
| Formato de celda          | opcional  | cómo se muestran los datos dentro de un campo específico                                                  |
| Zona horaria              | opcional  | asegura que los cálculos y presentaciones de fecha y hora sean precisos y coherentes                      |
| Ubicación del usuario     | opcional  | adaptar la salida de integración basada en la ubicación geográfica o preferencias del usuario             |

## <h2 hidden>Construye tus aplicaciones</h2>

Con nuestra extensa biblioteca de componentes UI, usted puede crear de forma rápida y sencilla interfaces muy pulidas. Puede escribir consultas para interactuar con datos y utilizar JavaScript en muchas partes diferentes de su aplicación.

### Editor

Construir, desplegar y ejecutar aplicaciones con el entorno de desarrollo web de ILLA

### Diseño y usabilidad (Páginas, Transformadores)

Personalizar aplicaciones con estilos y formato personalizados

### Manejadores de Eventos

Los manejadores de eventos permiten activar consultas, componentes de control y realizar otras acciones en respuesta a las interacciones del usuario

## Componentes de ensamblado

Utilice la biblioteca de componentes de interfaz de usuario de ILAL para construir la interfaz de usuario de su aplicación.

Los componentes son una parte esencial de ILA, y funcionan de forma similar a los componentes React en que pueden aceptar propiedades y tener estado interno. Al combinar la biblioteca de componentes de ILLA con sus datos, puede construir aplicaciones interactivas.

Algunos componentes comunes incluyen:

-   El componente Tabla se utiliza para mostrar datos en un formato tabular, y ofrece características tales como ordenar, filtrar, paginación y exportar CSV.
-   El componente Form se utiliza para agrupar los campos de entrada y enviarlos como una sola unidad.
-   El componente Select input es una combinación de un desplegable y un campo de entrada que permite a los usuarios seleccionar un valor de una lista o introducir un valor directamente como texto.
-   El componente Gráfico se utiliza para crear visualizaciones de datos usando gráficos y cartas.

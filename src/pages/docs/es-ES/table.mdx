## <h2 hidden>¿Qué es la tabla?</h2>

El componente de tabla en ILLA Cloud es una poderosa característica que permite a los usuarios mostrar, organizar y manipular datos tabulares dentro de la aplicación. Proporciona un diseño estructurado con filas y columnas, permitiendo a los usuarios ver e interactuar con datos de una manera conveniente y estructurada.

### Propiedades

| Propiedades                                  | Descripción                                                                                                                                   |
| -------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------- |
| Fuente de datos                              | fuente de datos que rellena la tabla                                                                                                          |
| Estado vacío                                 | el contenido o mensaje mostrado cuando la tabla no tiene datos que mostrar                                                                    |
| Cargando                                     | si el componente de tabla está actualmente en estado de carga                                                                                 |
| Columnas                                     | y configuración de las columnas en la tabla. (Ver la sección sobre Columnas abajo para más detalles)                                          |
| Columna                                      | el número de columnas                                                                                                                         |
| Selección de múltiples filas                 | Usar {{table.selectedRow[n].columnName}} para acceder a los datos.                                                                            |
| Selección de celda                           | Permitir a los usuarios seleccionar una celda                                                                                                 |
| Haga clic fuera para anular la selección     | Soportado en el modo de selección única de filas. Después de activarlo, haga clic en un área distinta a la tabla para deseleccionar la fila.  |
| Desbordamiento                               | cómo la tabla maneja el contenido que excede el espacio disponible dentro del contenedor de la tabla (paginación o desplazamiento)            |
| Habilitar paginación del lado del servidor   | Activar la paginación del servidor                                                                                                            |
| Tamaño de página                             | el número de filas que se mostrarán por página cuando la paginación del lado del servidor esté habilitada                                     |
| Refrescar                                    | Mostrar o no el icono de actualización.                                                                                                       |
| Descargar                                    | Mostrar botón de descarga en la barra de herramientas.                                                                                        |
| Descargar datos brutos                       | Mostrar o no el icono de descarga de datos crudos.                                                                                            |
| Filtro                                       | Mostrar botón de filtro en la barra de herramientas                                                                                           |
| Controlador de eventos                       | detectar y responder a acciones o eventos específicos del usuario, tales como clics, pulsaciones de teclas o envíos de formularios.           |
| Deshabilitado                                | no interactivo y no puede ser modificado o activado por el usuario.                                                                           |

#### **Resumen de propiedades de Column**

Para cada columna, podemos configurar sus propiedades para personalizar los datos mostrados para cada columna.

| Propiedades                                         | Descripción                                                                                                                                                                                                                 |
| --------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Título de columna                                   | especifica el nombre o etiqueta de la columna, que sirve como cabecera para identificar los datos que representa                                                                                                            |
| Tipo de columna                                     | define el tipo de datos que contiene la columna, incluyendo auto, texto, fecha, etiqueta, hora, dateTime, número, porcentaje, enlace, botón, grupo de botones, booleano, imagen, iconGroup, Clasificación, Markdown, moneda |
| Valor Asignado                                      | Usa {{ currentRow }} para acceder a los datos de la fila actual. Por ejemplo, para calcular "coste por clic" basado en las columnas "cost" y "clickTimes":                                                                  |
| {{ currentRow.cost / currentRow.clickTimes }}       |                                                                                                                                                                                                                             |
| Habilitar ordenación                                | si la columna permite a los usuarios ordenar los datos de la tabla basándose en los valores de esa columna específica                                                                                                       |
| Fondo                                               | Establecer el fondo de la columna. Use {{ write a conditional statement to set the color displayed by each cell under different conditions.                                                                                 |
| Alineación                                          | controla la alineación horizontal del contenido dentro de las celdas de columna                                                                                                                                             |

#### Tipos de columna:

-   Auto

un tipo dinámico que determina automáticamente el tipo de datos basado en los valores dentro de la columna

-   Texto

representa una columna que contiene datos textuales

-   Fecha

representa valores de datos que corresponden a fechas específicas

| Propiedades   | Descripción                                                           |
| ------------- | --------------------------------------------------------------------- |
| Formatear     | acepta varios códigos de formato o patrones, por defecto es YYY-MM-DD |

-   Etiqueta

un formato especializado para mostrar etiquetas o etiquetas asociadas con entradas de datos

| Propiedades           | Descripción                                                                        |
| --------------------- | ---------------------------------------------------------------------------------- |
| Etiqueta de etiqueta  | representa el texto o la etiqueta asignada a una etiqueta en la columna "etiqueta" |
| Color de etiqueta     | define el color asignado a la etiqueta en la columna "etiqueta"                    |

-   Hora

representar y manipular con precisión los valores de tiempo dentro de la tabla

| Propiedades   | Descripción                                                                                   |
| ------------- | --------------------------------------------------------------------------------------------- |
| Formatear     | la representación visual de los datos de tiempo dentro de la columna, por defecto es HH:mm:ss |

-   Fecha y hora

almacena y representa valores de fecha y hora

| Propiedades   | Descripción                                                                                                   |
| ------------- | ------------------------------------------------------------------------------------------------------------- |
| Formatear     | especifica cómo deben mostrarse los valores de fecha y hora en la columna, por defecto es AAAA-MM-DD HH:mm:ss |

-   Número

manejar datos numéricos

| Propiedades                  | Descripción                                                                                                                      |
| ---------------------------- | -------------------------------------------------------------------------------------------------------------------------------- |
| Posición decimal             | especificar el número de decimales a mostrar para los valores numéricos en la columna                                            |
| Mostrar miles de separadores | si la columna debe mostrar un separador de miles para mejorar la legibilidad y facilidad de interpretación para números grandes  |

-   Porcentaje

manejar los valores porcentuales de una manera concisa y visualmente atractiva

| Propiedades                  | Descripción                                                                              |
| ---------------------------- | ---------------------------------------------------------------------------------------- |
| Posición decimal             | especificar el número de decimales a mostrar para los valores porcentuales en la columna |
| Mostrar miles de separadores | si un separador de miles debe incluirse en los valores porcentuales mostrados            |

-   Enlace

incluir enlaces clicables dentro de una columna específica

-   Botón

incluir botones interactivos dentro de una columna específica

| Propiedades            | Descripción                                                                               |
| ---------------------- | ----------------------------------------------------------------------------------------- |
| Controlador de eventos | define la acción o evento que ocurre cuando se hace clic en el botón dentro de la columna |
| Deshabilitado          | si el botón de la columna está habilitado o desactivado                                   |
| Color del tema         | elige un color del tema para el botón                                                     |
| Variante               | estilo visual o variante del botón (relleno o contorno)                                   |

-   botónGrupo

permite a los usuarios crear grupos de botones interactivos dentro de una columna

Para cada botón en las columnas, podemos configurar sus propiedades.

| Propiedades            | Descripción                                                                                                                                 |
| ---------------------- | ------------------------------------------------------------------------------------------------------------------------------------------- |
| Valor Asignado         | asociar cada botón en el grupo de botones con un valor o campo específico de la fuente de datos                                             |
| Variante               | estilo visual o apariencia de los botones en el grupo de botones                                                                            |
| Controlador de eventos | la acción o función activada cuando se hace clic en un botón del grupo o se interactúa con                                                  |
| Deshabilitado          | si los botones del grupo de botones están habilitados o desactivados                                                                        |
| Color del tema         | personalizar el color o el estilo de los botones en el grupo para alinearse con el tema general o la marca de su aplicación o sitio web     |

-   Boolean

representa datos que tienen dos estados posibles: verdadero o falso

-   Imagen

rellenar una columna en la tabla con datos de imagen

| Propiedades    | Descripción                                                                |
| -------------- | -------------------------------------------------------------------------- |
| Tipo de escala | cómo la imagen es escalada o redimensionada dentro de la celda de la tabla |

-   IconGroup

una columna especializada que permite a los usuarios mostrar iconos dentro de un grupo o colección

Para cada botón en las columnas, podemos configurar sus propiedades.

| Propiedades            | Descripción                                                                                                                          |
| ---------------------- | ------------------------------------------------------------------------------------------------------------------------------------ |
| Icono                  | permite a los usuarios seleccionar o especificar el icono que se mostrará en la columna de IconGroup                                 |
| Color del tema         | determina el esquema de color o tema aplicado al icono dentro de la columna de IconGroup                                             |
| Controlador de eventos | permite a los usuarios definir un evento o acción cuando el icono en la columna de IconGroup es interactuado por los usuarios        |
| Deshabilitado          | estado de la columna IconGroup, determinando si está desactivada o habilitada para la interacción del usuario                        |

-   Valoración

representa valoraciones o comentarios dados a elementos o entidades

-   Markdown

capacidad para representar texto formateado usando la sintaxis Markdown dentro de las celdas de columna

-   Moneda

manejar y formatear valores de moneda.

| Propiedades                  | Descripción                                                                           |
| ---------------------------- | ------------------------------------------------------------------------------------- |
| Posición decimal             | el número de decimales a mostrar para los valores de la moneda dentro de la columna   |
| Código de moneda             | especifica el código o abreviatura que representa la moneda utilizada en la columna   |
| Mostrar miles de separadores | si se muestra un separador de miles para grandes valores de divisas                   |

### Método

Puede usar otros componentes para controlar el componente. Apoyamos los dos métodos siguientes:

-   **selectPage**

seleccione todas las filas de una página específica en una tabla paginada

| Propiedades  | Descripción                                                |
| ------------ | ---------------------------------------------------------- |
| Valor final  | el número de página o el índice de la página a seleccionar |

-   **selectRow**

seleccione una fila específica en la tabla

| Propiedades                     | Descripción                                                                                                                                                           |
| ------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Filas seleccionadas por defecto | valor de registro seleccionado por defecto, puede ser el índice de registro, un identificador único, o cualquier otra propiedad que identifique la fila a seleccionar |

-   **borrarSelección**

borrar la selección actual en la tabla

-   **setFilters**

aplicar filtros a los datos de la tabla

| Propiedades    | Descripción                               |
| -------------- | ----------------------------------------- |
| Filtros        | condiciones de filtrado o criterios       |
| Modo de filtro | determina cómo se aplican los filtros     |

-   **ClearFilters**

borra cualquier filtro aplicado en la tabla

-   **setSort**

establecer el orden de la tabla

| Propiedades | Descripción                                                        |
| ----------- | ------------------------------------------------------------------ |
| Columna     | la columna en la que se debe aplicar el ordenamiento               |
| Dirección   | dirección del tipo, que puede ser "ascendente" o "descendente"     |

### Caso de uso

A continuación, demostraremos cómo mapear los datos de `**Upstash**` al componente `**tabla**`.

#### Paso 1 Añadir una acción

Vamos a crear una tabla en Upstash llamada **`employee_profile`,** incluyendo 4 columnas `name`, `level`, `age` y `married`.

Entonces podemos crear una nueva acción para Upstash desde la lista de acciones en ILLA Cloud y nombrar upstash2.

Para listar todos los datos en employee_profile ponga el fragmento de código de abajo en la sección de consultas de Redis.

```sql
Obtener perfil de empleado
```

Dado que los datos de salida son una lista de pares de diccionarios, queremos transformarlos en una matriz. En Transformer, actívalo y ponga el código abajo.

```jsx
const jsonString = data[0].result;

// Parse the JSON string into a JavaScript object
const jsonData = JSON.parse(jsonString);

// Access the "employees" property and map it to a new array
const mappedArray = jsonData.employees.map((employee) => ({
  name: employee.name,
  level: employee.level,
  age: employee.age,
  married: employee.married,
}));

return mappedArray;
```

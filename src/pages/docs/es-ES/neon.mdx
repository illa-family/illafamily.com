## <h2 hidden>¿Qué es Neon?</h2>

Neon in Illa es una integración de base de datos que te permite conectar e interactuar con una base de datos Neon. Neon es un sistema de gestión de bases de datos relacionales de código abierto conocido por su velocidad, fiabilidad y facilidad de uso.

Con la integración de Neon en Illa, puede consultar, insertar, actualizar y eliminar datos de una base de datos de Neon usando comandos SQL. También puede crear consultas SQL personalizadas y ejecutarlas dentro de Illa. Esto hace que sea fácil construir aplicaciones basadas en datos y paneles que muestren datos en tiempo real desde una base de datos de Neon.

Este tutorial describe el proceso de creación de un Panel de Administración usando ILLA Builder y Neon en unos pocos pasos. ILLA es una plataforma de código bajo para desarrolladores que permite el rápido desarrollo y despliegue de herramientas internas. Permite crear páginas arrastrando y soltando componentes de la interfaz de usuario, conectando a cualquier base de datos o API, y escribiendo JavaScript. Para aprender más sobre Neon, visite su sitio web en https://neon.tech/ ¡Empecemos!

![neon_create_project_1](https://cdn.illacloud.com/official-website/img/docs/neon_create_project_1.png)

Importando datos a través de consultas SQL de ejemplo.

![neon_create_sql](https://cdn.illacloud.com/official-website/img/docs/neon_create_sql.png)

### Crear neón

Hay dos maneras de crear un recurso en Illa después de iniciar sesión en tu cuenta de Imax. 

-   Crear en Recursos

Inicia sesión en tu cuenta de Imax, selecciona `**Recursos**` en la parte superior de la página y haz clic en el botón `**Crear nuevo**`. 

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

Selecciona **`Neon`** de la lista de bases de datos.

![action_list](https://cdn.illacloud.com/official-website/img/docs/action_list.png)

Conecte a la base de datos con los parámetros requeridos descritos en `Configuración de Conexión` a continuación. 

Haga clic en `**Probar conexión**` para ver si podemos conectar con éxito a la base de datos. En caso afirmativo, haga clic en **`Guardar Recursos`**, lo demás, compruebe dos veces el nombre de host, el puerto, el nombre de usuario y la contraseña son correctos.

Después de crear un recurso, el Neon listo se mostrará como mostrado.

![neon_ex_ready](https://cdn.illacloud.com/official-website/img/docs/neon_ex_ready.png)

-   Crear en el Constructor

Inicia sesión en tu cuenta de Imax, crea un proyecto en Illa Builder en la página `**App**`, y navegar a la **`Lista de Acciones`** en la parte inferior de la página. Haz clic en `**nuevo**`, luego selecciona **`Neon`** de la lista de bases de datos. Luego, conecte a la base de datos con los parámetros requeridos descritos en `Configuración de Conexión` a continuación.

Haga clic en `**Probar conexión**` para ver si podemos conectar con éxito a la base de datos. En caso afirmativo, haga clic en **`Guardar Recursos`**, lo demás, compruebe dos veces el nombre de host, el puerto, el nombre de usuario y la contraseña son correctos.

### Configuración de conexión

Aquí tenemos que proporcionar información para conectarse a la base de datos Neon.

![neon_config](https://cdn.illacloud.com/official-website/img/docs/neon_config.png)

| Propiedades        | Requerido  | Descripción                                                                                                                                 |
| ------------------ | ---------- | ------------------------------------------------------------------------------------------------------------------------------------------- |
| Nombre             | requerido  | El nombre del recurso al crear acciones en el ILLA.                                                                                         |
| Cadena de conexión | opcional   | una configuración que se utiliza para especificar los detalles de una conexión a una fuente de datos externa, como una base de datos o API. |
| Hostname           | requerido  | La URL o dirección IP de su base de datos                                                                                                   |
| Puerto             | requerido  | El número de puerto del servidor que debe utilizar para conectarse. Si no especifica un puerto, el puerto por defecto es '5432'.            |
| Base de datos      | requerido  | El nombre de la base de datos                                                                                                               |
| Usuario            | requerido  | el nombre de usuario que desea utilizar al iniciar sesión en el servidor de Neon.                                                           |
| Contraseña         | requerido  | Utilice esta contraseña para la autenticación.                                                                                              |

Hay dos maneras de rellenar la información para configurar Neon.

-   Analizando cadena de conexión dada

En la página del proyecto Neon, haciendo desplazamientos, tenemos una determinada cadena de conexión en **`Conexión directa`**.

![neon_connection](https://cdn.illacloud.com/official-website/img/docs/neon_connection.png)

Nota: Pulsar el desenfoque muestra los números ocultos, luego podemos copiarlo.

Después de pegar la cadena en el área de texto de la cadena de conexión en la configuración, haga clic en **`Parse`**.

![neon_parsing](https://cdn.illacloud.com/official-website/img/docs/neon_parsing.png)

Tenemos toda la información rellenada automáticamente para nosotros.

![neon_parse_done](https://cdn.illacloud.com/official-website/img/docs/neon_parse_done.png)

-   Rellena manualmente la información.

También podemos encontrar información requerida en la página del proyecto y rellenarlos manualmente en la configuración de Illa. 

### Crear acciones

Hemos creado un recurso de Neón, podemos añadir la acción seleccionando Neón de la lista de acciones y eligiendo el botón **`Crear acción`**.

![neon_resource_list](https://cdn.illacloud.com/official-website/img/docs/neon_resource_list.png)

 Ahora hemos añadido el servidor Neon como una acción a nuestra página de construcción.

![neon](https://cdn.illacloud.com/official-website/img/docs/neon.png)

### Configurar Neon

| Propiedades   | Descripción                                                                                    |
| ------------- | ---------------------------------------------------------------------------------------------- |
| Consulta SQL  | comandos sql para buscar (seleccionar), insertar, actualizar, borrar datos de la base de datos |
| Transformador | transforma los datos en el estilo que te gusta usando JavaScript                               |

Ejemplo:

-   Seleccionar

```sql
SELECT * DE usuarios ORDER POR apellido;
```

-   Insert

```sql
INSERT INTO users

	(firstname, lastname, email)

VALUES

	(

		{{ NewUser.data.firstNameInput }},

		{{ NewUser.data.lastNameInput }},

		{{ NewUser.data.emailInput }}

	);
```

-   Actualizar

```sql
UPDATE users
  SET first name = '{{form1.updatedProfile.first }}'
  WHERE id = {{ form1.updatedProfile.uid }};
```

-   Eliminar

```sql
DELETE FROM users WHERE id = {{ form1.updatedProfile.uid }};
```

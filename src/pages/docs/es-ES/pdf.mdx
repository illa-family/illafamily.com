## <h2 hidden>¿Qué es PDF?</h2>

El componente PDF es una característica o elemento que permite a los usuarios ver e interactuar con documentos PDF dentro de la aplicación. Proporciona funcionalidad para renderizar, navegar y manipular archivos PDF, permitiendo a los usuarios acceder y trabajar con contenido PDF sin problemas.

### Propiedades

| Propiedades                   | Descripción                                                                                             |
| ----------------------------- | ------------------------------------------------------------------------------------------------------- |
| Fuente PDF                    | especificar la fuente del pdf que se mostrará                                                           |
| Mostrar barra de herramientas | controla la visibilidad de la barra de herramientas dentro del componente PDF                           |
| Sugerencia                    | añadir una descripción o un texto descriptivo que aparece cuando el usuario pasa el cursor sobre el pdf |
| Escalar página a              | determina el comportamiento de escalado de las páginas PDF dentro del componente                        |
| Hidden                        | determina la visibilidad del componente PDF                                                             |

### Método

Puede usar otros componentes para controlar el componente. Apoyamos los siguientes métodos:

-   **setFileUrl**

cambiar dinámicamente la fuente o URL del archivo pdf.

| Propiedades | Descripción       |
| ----------- | ----------------- |
| Fuente PDF  | url fuente de pdf |

### Datos

El componente tiene algunos datos comúnmente utilizados, que pueden llamarse a través de **`{{componentName.propertyName}}`** en la aplicación.

| Nombre de la propiedad | Descripción                                                                                                                  |
| ---------------------- | ---------------------------------------------------------------------------------------------------------------------------- |
| nombre de pantalla     | Un valor de cadena que representa el nombre o el título del pdf.                                                             |
| altura                 | la dimensión vertical o el tamaño del componente PDF                                                                         |
| hidden                 | Un valor booleano que determina si el componente pdf está oculto en la interfaz de usuario.                                  |
| modo de escala         | especifica cómo el documento PDF es escalado o encajado en el área de visualización disponible del componente                |
| showToolBar            | si se muestra una barra de herramientas o un panel de control junto con el componente PDF                                    |
| tooltipText            | que representa la descripción o pasa el texto a mostrar cuando el usuario pasa por encima del componente pdf.                |
| url                    | fuente del archivo PDF que se mostrará en el componente                                                                      |
| width                  | la dimensión horizontal o el tamaño del componente PDF                                                                       |

Ejemplo: {{pdf1.displayName}}

### Caso de uso

A continuación, demostraremos cómo controlar la pantalla de pdf con un botón.

#### Paso 1 Añadir componentes

Añade un componente **`PDF`** y un componente **`button`** al lienzo.

![pdf_layout](https://cdn.illacloud.com/official-website/img/docs/pdf_layout.jpg)

#### Paso 2 Configurar el componente

Para el componente **`button`** podemos configurarlo para establecer la fuente de la imagen del componente **`PDF`** a una dirección pdf.

1.  Haga clic en el componente `**botón**` para abrir su página de inspección. Bajo **`Controlador de eventos`**, haga clic en `**+ Nuevo**`.
2.  En el controlador de edición de eventos, seleccione **`componente de control`** en acción, y seleccione el componente **`PDF`** que desea cambiar como objetivo del evento.
3.  Elige la acción **`setFileUrl`**
4.  Pon la nueva dirección de la imagen en el campo **`Valor`**

![pdf_config](https://cdn.illacloud.com/official-website/img/docs/pdf_config.jpg)

#### Paso 3 Prueba

Ahora que al hacer clic en el botón "Reproducir", la imagen debe cambiar

![pdf_test](https://cdn.illacloud.com/official-website/img/docs/pdf_test.jpg)

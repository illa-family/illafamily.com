## <h2 hidden>¿Qué es la Appwrite?</h2>

Appwrite in Illa es una integración de base de datos que te permite conectar e interactuar con una base de datos Appwrite. Appwrite es un sistema de gestión de bases de datos relacionales de código abierto conocido por su velocidad, fiabilidad y facilidad de uso.

Con la integración de Appwrite en Illa, puede consultar, insertar, actualizar y eliminar datos de una base de datos de Appwrite usando comandos SQL. También puede crear consultas SQL personalizadas y ejecutarlas dentro de Illa. Esto hace que sea fácil construir aplicaciones basadas en datos y paneles que muestren datos en tiempo real desde una base de datos Appwrite.

Este tutorial describe el proceso de creación de un Panel de Administración usando ILLA Builder y Appwrite en unos pocos pasos. ILLA es una plataforma de código bajo para desarrolladores que permite el rápido desarrollo y despliegue de herramientas internas. Permite crear páginas arrastrando y soltando componentes de la interfaz de usuario, conectando a cualquier base de datos o API, y escribiendo JavaScript. Para obtener más información sobre Appwrite, visite su sitio web en https://Appwrite.io/(https://Appwrite.io/) ¡Empecemos!

### Configura tu Back end en Appwrite

Después de iniciar sesión en tu cuenta de Appwrite, en el panel de control de Appwrite, haz clic en `**Crear proyecto**` y establece el nombre a `Primer proyecto`.

![app_create_project](https://cdn.illacloud.com/official-website/img/docs/app_create_project.png)

Luego podemos navegar a la página del proyecto. Appwrite proporciona múltiples plataformas para construir tu aplicación, así como integraciones.

![app_platforms](https://cdn.illacloud.com/official-website/img/docs/app_platforms.png)

Aquí elegimos Web App como ejemplo.

Después de crear nuestro proyecto de aplicación Web, podemos ir a la página de la base de datos ya que sólo utilizamos Appwrite como integración de base de datos hasta ahora.

![app_dashboard](https://cdn.illacloud.com/official-website/img/docs/app_dahsboard.png)

Luego podemos crear la base de datos y crear colecciones dentro de la base de datos

![app_create_database](https://cdn.illacloud.com/official-website/img/docs/app_create_database.png)
Desde aquí podemos crear atributos y datos en la colección.

![app_create_data](https://cdn.illacloud.com/official-website/img/docs/app_create_data.png)

Para recuperar documentos por atributos, necesitamos hacer los índices correspondientes para cada atributo.

![app_create_index](https://cdn.illacloud.com/official-website/img/docs/app_create_index.png)

Esta base de datos ya está lista para que usted se integre con Illa.

### Crear Appwrite

Hay dos maneras de crear un recurso en Illa después de iniciar sesión en tu cuenta de Imax.

-   Crear en Recursos

Inicia sesión en tu cuenta de Imax, selecciona `**Recursos**` en la parte superior de la página y haz clic en el botón `**Crear nuevo**`.

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

Selecciona **`Appwrite`** de la lista de bases de datos.

![action_list](https://cdn.illacloud.com/official-website/img/docs/action_list.png)

Conecte a la base de datos con los parámetros requeridos descritos en `Configuración de Conexión` a continuación.

Haga clic en `**Probar conexión**` para ver si podemos conectar con éxito a la base de datos. En caso afirmativo, haga clic en **`Guardar Recursos`**, lo demás, compruebe dos veces el nombre de host, el puerto, el nombre de usuario y la contraseña son correctos.

Después de crear un recurso, la aplicación lista se mostrará como mostrada.

![app_ex_ready](https://cdn.illacloud.com/official-website/img/docs/app_ex_ready.png)

-   Crear en el Constructor

Inicia sesión en tu cuenta de Imax, crea un proyecto en Illa Builder en la página `**App**`, y navegar a la **`Lista de Acciones`** en la parte inferior de la página. Haz clic en `**nuevo**`, luego selecciona **`Appwrite`** de la lista de bases de datos. Luego, conecte a la base de datos con los parámetros requeridos descritos en `Configuración de Conexión` a continuación.

Haga clic en `**Probar conexión**` para ver si podemos conectar con éxito a la base de datos. En caso afirmativo, haga clic en **`Guardar Recursos`**, lo demás, compruebe dos veces el nombre de host, el puerto, el nombre de usuario y la contraseña son correctos.

### Configuración de conexión

Aquí necesitamos proporcionar información para conectarse a la base de datos de Appwrite.

![app_config](https://cdn.illacloud.com/official-website/img/docs/app_config.png)

| Propiedades         | Requerido | Descripción                                                                                                |
| ------------------- | --------- | ---------------------------------------------------------------------------------------------------------- |
| Nombre              | requerido | El nombre del recurso al crear acciones en el ILLA.                                                        |
| Anfitrión           | requerido | La URL o dirección IP de su base de datos                                                                  |
| ID de base de datos | requerido | identificador único asignado a cada base de datos creada en Appwrite                                       |
| ID del proyecto     | requerido | identificador único que representa todo tu proyecto Appwrite                                               |
| Clave API           | requerido | un token secreto que sirve como una forma de autenticación al hacer peticiones API al servidor de Appwrite |

Para encontrar el **`Host`**, **`Database ID`**, **`Project ID`**, y **`API key`** en Appwrite, primero vamos al botón `**configuración**` en la esquina inferior izquierda de la página del proyecto.

Copiando el `project ID` para **`project ID`** y `API Endpoint` para **`Host`** en la configuración de Appwrite en Illa.

![app_setting](https://cdn.illacloud.com/official-website/img/docs/app_setting.png)

Luego podemos hacer clic en el botón **`Ver claves API`** en la misma página. Si no has creado una API para este proyecto, haz clic en el botón **`Crear clave API`** y selecciona sólo la carpeta `base de datos` ya que eso es lo que usaremos desde Appwrite.

![app_api_key](https://cdn.illacloud.com/official-website/img/docs/app_api_key.png)

Copia el `API Key Text` para **`API key`** en la configuración de Iller. Se puede copiar para estos dos puntos.

![app_key_secret](https://cdn.illacloud.com/official-website/img/docs/app_key_secret.png)

Lo que nos queda es el **`ID de base de datos`** que podemos encontrar en la página `Base de datos`. Copiarlo y pegarlo en la configuración de Appwrite en Illa.

![app_database](https://cdn.illacloud.com/official-website/img/docs/app_database.png)

El producto final debería parecerse a esto.

![app_final_config](https://cdn.illacloud.com/official-website/img/docs/app_final_config.png)

![app_resource_list](https://cdn.illacloud.com/official-website/img/docs/app_resource_list.png)

### Configurar Appwrite

#### Resumen

| Nombre del método   | Descripción                                                                                                 |
| ------------------- | ----------------------------------------------------------------------------------------------------------- |
| Método              | Crear un documento, Obtener un documento, Actualizar un documento, Eliminar un documento, Listar documentos |
| ID de colección     | un identificador único asignado a una colección de base de datos                                            |
| Transformador       | transforma los datos en el estilo que te gusta usando JavaScript                                            |

#### Crear un documento

crear un nuevo documento en una colección

** Input **

| Propiedades      | Requerido | Descripción                                                                   |
| ---------------- | --------- | ----------------------------------------------------------------------------- |
| ID del documento | requerido | un identificador único asignado a un documento de colección de base de datos  |
| Datos            | opcional  | un objeto que contiene los campos y valores del nuevo documento.              |

Por ejemplo, para crear un nuevo documento en una colección con dos campos - **`name`** y **`age`**, puedes usar 1 como **`document ID`** y `{{ {"Name": "David", "Age": 18}}}` para **`data`**.

** Salida **

un objeto de respuesta que incluye el ID del documento recién creado, así como metadatos adicionales como la fecha y hora de creación.

![app_create_code](https://cdn.illacloud.com/official-website/img/docs/app_create_code.png)

#### Obtener un documento

recuperar un documento específico de una colección en la base de datos

** Input **

| Propiedades      | Requerido | Descripción                                                                  |
| ---------------- | --------- | ---------------------------------------------------------------------------- |
| ID del documento | requerido | un identificador único asignado a un documento de colección de base de datos |

Por ejemplo, podemos obtener el documento con id **`1`**.

** Salida **

un objeto de respuesta que incluye los datos del documento solicitado

![app_get_data](https://cdn.illacloud.com/official-website/img/docs/app_get_data.png)

Usa `{{Appwrite1.data}}` para obtener todo el valor devuelto.

![app_get_code](https://cdn.illacloud.com/official-website/img/docs/app_get_code.png)

#### Actualizar un documento

actualizar los datos de un documento existente en una colección.

** Input **

| Propiedades      | Requerido | Descripción                                                                                               |
| ---------------- | --------- | --------------------------------------------------------------------------------------------------------- |
| ID del documento | requerido | un identificador único asignado a un documento de colección de base de datos                              |
| Datos            | opcional  | Un objeto que contiene los campos y valores actualizados que desea asignar al documento.                  |

Por ejemplo, para actualizar un documento en una colección utilizando el método **`updateDocument`**, puedes poner **`1`** para el id de documento y `{{{"Name":"Fred"}}}` para datos.

** Salida **

un objeto de respuesta que incluye los datos del documento solicitado

![app_update)data](https://cdn.illacloud.com/official-website/img/docs/app_update_data.png)

#### Eliminar un documento

borrar un documento específico de una colección en la base de datos

** Input **

| Propiedades      | Requerido | Descripción                                                                  |
| ---------------- | --------- | ---------------------------------------------------------------------------- |
| ID del documento | requerido | un identificador único asignado a un documento de colección de base de datos |

Por ejemplo, podemos eliminar el documento con id **`1`**.

** Salida **

un objeto de respuesta que confirma la eliminación exitosa del documento.

![app_delete_data](https://cdn.illacloud.com/official-website/img/docs/app_delete_data.png)

#### Lista de documentos

recuperar varios documentos de una colección en la base de datos

** Input **

| Propiedades   | Requerido | Descripción                                                                                   |
| ------------- | --------- | --------------------------------------------------------------------------------------------- |
| Filtro        | opcional  | definir las condiciones para filtrar los documentos basándose en campos y valores específicos |
| Ordenar por   | opcional  | especificar el campo por el cual los documentos resultantes deben ser ordenados               |
| Límite        | opcional  | restringir el número de documentos devueltos en el conjunto de resultados                     |

Por ejemplo, para listar todos los documentos con Nombre = James.

** Salida **

Una matriz de documentos que satisfacen al filtro.

![app_list_data](https://cdn.illacloud.com/official-website/img/official-site/intergration/app_list_data.png)

Usa `{{Appwrite1.data[0].documents}}` para obtener el valor devuelto.

![app_list_code](https://cdn.illacloud.com/official-website/img/docs/app_list_code.png)

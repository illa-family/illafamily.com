## <h2 hidden>Base de datos Appwrite</h2>

Dentro de los diferentes servicios de Appwrite, usted puede encontrar autenticación de usuario y gestión de cuentas, preferencias de usuario, persistencia en la base de datos y almacenamiento, funciones en la nube, localización, manipulación de imágenes y más. 

La combinación de ILLA Cloud y Appwrite, es decir, la combinación de código bajo del lado del servidor y código bajo del front-end, realmente realizarán desarrollo completo de código bajo.

[Haz clic aquí](https://appwrite.io/) para aprender más sobre Appwrite.

### Configurar recurso Appwrite en ILLA Cloud

#### Dos maneras de crear recursos

La primera forma es entrar en la pestaña `Recursos` > haga clic en `Crear nuevo` para seleccionar Appwrite. 

<img src='https://cdn.illacloud.com/official-website/img/docs/assemble/appwrite1.png' alt="" />

La otra forma es entrar en la página `editor de aplicaciones` > haga clic en `+ Nuevo` para seleccionar Appwrite > haga clic en `+ Nuevos recursos` para agregar la nueva configuración. 

<img src='https://cdn.illacloud.com/official-website/img/docs/assemble/appwrite2.png' alt="" />

#### Configurar Appwrite rellenando los siguientes campos

Antes de conectar un recurso Appwrite, asegúrese de que ha desplegado Appwrite en su máquina local o un proveedor de nube. [Haz clic aquí](https://appwrite.io/docs/installation) para obtener más información sobre la instalación de Appwrite. 

| Propiedad           | es requerida | Descripción                                                                                                      |
| ------------------- | ------------ | ---------------------------------------------------------------------------------------------------------------- |
| Nombre              | requerido    | El nombre del recurso al crear acciones en el ILLA. Puede personalizar un nombre para una fácil identificación.  |
| Anfitrión           | requerido    | NOMBRE DE ALBERGUE O IP.                                                                                         |
| ID de base de datos | requerido    | ID de base de datos                                                                                              |
| ID del proyecto     | requerido    | ID del proyecto                                                                                                  |
| Clave Api           | requerido    | TOKEN                                                                                                            |

<img src='https://cdn.illacloud.com/official-website/img/docs/assemble/appwrite3.png' alt="" />

### Crear acciones en el editor de aplicaciones

#### Crear acciones de la siguiente manera

Haga clic en `+ Nuevo` en la página del editor de aplicaciones > para seleccionar Appwrite > seleccione un recurso de Appwrite existente o cree un nuevo recurso > haga clic en `Crear Acción` para crear una nueva acción. 

<img src='https://cdn.illacloud.com/official-website/img/docs/assemble/appwrite4.png' alt="" />

Después de crear una acción, seleccione un método. Proporcionamos los siguientes métodos: Listar documentos, Crear un documento, Actualizar un documento, Obtener un documento y Eliminar un documento. La configuración de cada método es la siguiente.

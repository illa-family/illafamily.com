## <Heading hidden>Editor de aplicativos</Heading>

O editor de APP é a interface para desenvolver o aplicativo e consiste das seguintes partes principais:

-   Barra
-   Frame
-   Editor de ação
-   Projeto de dados
-   Insert
-   Visualizar

<img alt="editor" src={'https://cdn.illacloud.com/official-website/img/official-site/content.png'} />

### Barra

A barra de ferramentas está localizada na parte superior do APP Editor, que é usada principalmente para configurar o aplicativo e pode ser usada para as seguintes tarefas：

-   Clique no ícone ILLA do lado esquerdo para retornar à página inicial.
-   Veja as informações do aplicativo e salve o status.
-   Mostrar/ocultar itens à esquerda, à direita, painel inferior.
-   Ajustando o dimensionamento dos itens.
-   Mostra todos os desenvolvedores do APP.
-   Alertar o aplicativo para o status do bug atual.
-   Compartilhe o aplicativo com outros desenvolvedores.
-   Executando o APP.

### Frame

Frame é a área do aplicativo onde você pode editar a interface gráfica do aplicativo. Você pode arrastar e soltar inserções no quadro a partir do painel de inserções na mão direita; você também pode selecionar inserções que foram colocadas no quadro e usar o mouse para arrastar e soltar para modificar a posição e o tamanho das inserções.

Quando você estiver arrastando e soltando insere ou modifica o seu tamanho, um quadrado tracejado aparecerá no fundo do quadro para ajudá-lo a organizar as inserções dentro da moldura. Quando você encontrar inserções sobrepostas enquanto arrasta e solta, O Construtor ILLA dará prioridade à inserção que você está se movendo e automaticamente moverá o bloqueio inserido para fora do caminho para você.

#### Pré-visualizar

O botão de pré-visualização está localizado no canto superior direito do quadro. Quando clicado, o editor entrará no modo de pré-visualização, os painéis inferiores de esquerda e direita serão ocultados e as inserções no quadro não serão modificadas, todas as inserções serão apresentadas na sua frente no tamanho 1:1.

> ### Navegadores suportados
>
> Recomendamos que você abra o ILLA Builder no Google Chrome.

### Editor de Ação

O Editor de Ação está localizado abaixo do quadro e é a área onde você escreve código para interagir com dados de recursos. Nesta área, você pode encontrar operações criadas, selecionar o recurso operacional atual, escrever instruções de consulta e configurar transformadores. ou poderá editar a declaração de consultas no editor de ações e quando atualizar a declaração de consulta, o editor de apps atualizará automaticamente seus resultados de ação.

Ao inserir instruções SQL de ação, você pode optar por usar o modo SQL para editá-las através de código simples, ou você pode escolher o modo GUI para modificá-los no framework de ação que lhe damos para concluir rapidamente a edição da ação. ou tutoriais detalhados, veja a documentação da operação.

### Lista de ações

A lista de ações está localizada no lado esquerdo do painel de configuração onde você pode encontrar a ação que você precisa usar e modificá-la no editor de edição. ou também pode adicionar uma nova ação aqui, ver detalhes do processo de criação.

### Workspace de dados

O espaço de trabalho dos dados está localizado no painel à esquerda do quadro, onde você pode visualizar os inseridos, ações e status global do editor, tudo exibido no formato Json. Você também pode editar os dados, selecionando-os no espaço de trabalho ou excluindo-os clicando no botão Mais Configurações.

### Insert

A inserção está localizada no painel à direita do quadro, e você pode mover a inserção para o quadro arrastando e soltando com o mouse. Você pode encontrar a inserção que você precisa por inserir um tipo, ou você pode procurar por ele e encontrá-lo rapidamente.

### Visualizar

A inspeção está localizada no painel à direita do quadro. Ao selecionar uma inserção no frame ou espaço de trabalho dos dados, o painel à direita mudará automaticamente para a página inserir. A partir desta página você pode definir variáveis, atualize o nome de inserção, crie manipuladores de eventos ou realize uma operação de inserção de eliminação. A barra de ferramentas (toolbar) também será definida de forma diferente para diferentes inserções para que você possa alterar o conteúdo das diferentes inserções.

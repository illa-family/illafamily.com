## <Heading hidden>Recurso</Heading>

Conecte-se à fonte de dados e chame-a no seu Construtor ILLA. Nós lhe fornecemos módulos de conexão adaptáveis para diferentes tipos de fontes de dados. Você pode criar uma conexão estável entre seus dados e ILLA em pouco tempo, simplificando muito seu processo de acesso aos dados.

Depois de criar com sucesso um recurso, você pode chamá-lo em qualquer app no ILLA. Você também pode compartilhar seus recursos com os co-desenvolvedores, melhorando a eficiência dos aplicativos de construção.

### Tipo de recursos suportados

ILLA suporta uma série de principais bases de dados, como MySQL e Postage, bem como APIs como a API Rest e GraphQL.

### Criar um recurso

Os recursos representam conexões entre o Construtor ILLA e suas fontes de dados. Crie um recurso seguindo as seguintes etapas:

#### 1.  Navegue até a aba Repositório na barra de navegação da página inicial e clique no botão "Criar novo"

<img
  src='https://cdn.illacloud.com/official-website/img/docs/re_e_1.png'
  alt=""
/>

#### 2.  Escolha o seu tipo de dado aqui

<img
  src='https://cdn.illacloud.com/official-website/img/docs/qs_e_2.png'
  alt=""
/>

#### 3.  Refinar suas informações de fonte de dados e clique no botão "Criar Recursos" e depois finalizar o processo

<img
  src='https://cdn.illacloud.com/official-website/img/docs/re_e_2.png'
  alt=""
/>

Além disso, você pode criar recursos na lista de ações sob o editor de App

> ### Testar conexão
>
> Quando estiver criando ou visualizando recursos, você pode clicar no botão de teste sob o painel de recursos para verificar se a fonte de dados está conectada com sucesso.

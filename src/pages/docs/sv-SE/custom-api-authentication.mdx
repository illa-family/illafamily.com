## <h2 hidden>Anpassad API-autentisering</h2>

ILLA stöder många autentiseringsstandarder som standard (se här för mer information). 
Dessutom har ILLA ett Custom Auth autentiseringsläge som låter dig kombinera flera steg som stöder praktiskt taget alla API-autentisering. 
Dessa autentiseringsmetoder kan användas för både REST och GraphQL-resurser.

## <Heading hidden>API-autentisering</Heading>

ILLA kan stödja nästan alla typer av autentisering. Vi har samarbetat med olika företag för att stödja alla standardtyper av autentisering och har även utvecklat metoder för att stödja eventuella anpassade autentiseringsmetoder som vi inte stöder direkt.

Du kan använda statiska autentiseringstoken eller skapa ett autentiseringssystem som uppmanar slutanvändaren till deras uppgifter varje gång de får tillgång till programmet.

Denna sida kommer att ge dig instruktioner om hur du autentiserar ditt API. Om du har frågor eller behöver hjälp, du kan kontakta vårt supportteam med Intercom bubblan på denna sida eller hjälp-ikonen inne i editorn.

## <h2 hidden>Wat is Upstash?</h2>

Upstash is een cloud-native server less service met Redis compatible dat een eenvoudige manier biedt om Redis functionaliteit te integreren in applicaties. In de context van ILLA Cloud stelt de Upstash integratie gebruikers in staat om gebruik te maken van de functies en mogelijkheden van Upstash binnen de ILLA Cloud toepassing.

Deze handleiding schetst het proces van het maken van een Admin Panel met behulp van ILLA Builder en Upstash in een paar eenvoudige stappen. ILLA Cloud is een laag codeerplatform voor ontwikkelaars waarmee interne hulpmiddelen snel kunnen worden ontwikkeld en ingezet. Het maakt het mogelijk om pagina's te maken door UI-componenten te slepen en te laten vallen, door verbinding te maken met elke database of API en door JavaScript te schrijven. Voor meer informatie over Upstash, ga naar hun website op https://upstash.com/(https://upstash.com/) Laten we beginnen!

### Stel je Back end in op Upstash

Maak een account aan op Upstash. Je kunt als alternatief inloggen met je Amazon, Github of Google accounts.

Na het inloggen op je account op `Upstash`, klik op de `Create database` knop in Redis.

![up_home](https://cdn.illacloud.com/official-website/img/docs/up_home.jpg)

Klik op het `Upstash` dashboard op `Create database`.

![up_create_db](https://cdn.illacloud.com/official-website/img/docs/up_create_db.jpg)

Opmerking: je kunt alleen EEN database hebben in de vrije rang.

Voor de database configuratie kunt u uw database zo noemen als u wilt. Hier noemen we het `db1`. Kies de `primaire regio` en `lees regio` gebaseerd op je schrijf en lees gebruikerslocaties. Schakel `TSL/SSL` in. Het is optioneel om Eviction in te schakelen.

![up_db_config](https://cdn.illacloud.com/official-website/img/docs/up_db_config.jpg)

Klik op de **`create`** knop, we hebben nu een database klaar om te gebruiken.

### Upstash in ILLA Cloud aanmaken

Er zijn twee manieren om een bron in Illa te maken nadat je bent ingelogd op je Illa account.

-   Maak in Bronnen

Log in op je Illa account, selecteer `**Resources**` bovenaan de pagina, en klik op de `**Maak Nieuw**` knop.

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

Selecteer **`Upstash`** uit de database lijst.

![action_list_0](https://cdn.illacloud.com/official-website/img/docs/action_list_0.jpg)

Maak verbinding met de database met de vereiste parameters beschreven in `Connectie Instellingen` hieronder.

Klik op `**Test Connection**` om te zien of we met succes verbinding kunnen maken met de database. Zo ja, klik **`Resources`**, anders controleer je de hostnaam, port, gebruikersnaam en wachtwoord correct.

Na het aanmaken van een bron, wordt de `Upstash` weergegeven zoals getoond.

![up_ex_ready](https://cdn.illacloud.com/official-website/img/docs/up_ex_ready.jpg)

-   Maak in Bouwer

Meld u aan bij uw Illa account, maak een project aan in Illa Builder op de `**App**` pagina en navigeer naar de **`Action List`** onderaan de pagina. Klik `**nieuw**` en selecteer dan **`Upstash`** uit de database lijst. Maak vervolgens verbinding met de database met de vereiste parameters beschreven in `Connectie Instellingen` hieronder.

Klik op `**Test Connection**` om te zien of we met succes verbinding kunnen maken met de database. Zo ja, klik **`Resources`**, anders controleer je de hostnaam, port, gebruikersnaam en wachtwoord correct.

### Upstash configureren

| Eigenschappen  | Beschrijving                                                                                    |
| -------------- | ----------------------------------------------------------------------------------------------- |
| Redis query    | query gebruiken om gegevens op te halen en te bewerken die zijn opgeslagen in de Redis-database |
| Transformator  | data omzetten naar de stijl die je leuk vindt met behulp van JavaScript                         |

Voorbeeld gebruik:

1.  Waarden ophalen:
    -   Krijg een waarde: gebruik het **`GET`** commando om de waarde die gekoppeld is aan een specifieke sleutel op te halen. Bijvoorbeeld: **`GET key`**.
2.  Stel waarden in:
    -   Stel een waarde in: gebruik het **`SET`** commando om de waarde van een toets in te stellen. Bijvoorbeeld **`SET key value`**.
3.  Hash operaties:
    -   Stel field-value paren in in een hash: Gebruik het **`HSET`** commando om een field-value paar in een hash in te stellen. Bijvoorbeeld: **`HSET hashKey field value`**.
    -   Verkrijg een waarde van een hash: gebruik het **`HGET`** commando om de waarde van een specifiek veld in een hash op te halen. Bijvoorbeeld: **`HGET hashKey field`**.
4.  Lijst acties:
    -   Voeg een element toe aan een lijst: gebruik de **`LPUSH`** of **`RPUSH`** commando om respectievelijk een element aan het begin of einde van een lijst toe te voegen. Bijvoorbeeld: **`LPUSH listKey value`**.
    -   Haal elementen op uit een lijst: gebruik het **`LRANGE`** commando om een reeks elementen uit een lijst op te halen. Bijvoorbeeld: **`LRANGE listKey start einde`**.
5.  Activiteiten instellen:
    -   Voeg elementen toe aan een set: gebruik het **`SAD`** commando om een of meer elementen aan een set toe te voegen. Bijvoorbeeld: **`SADD setKey value1 value2`**.
    -   Haal elementen op uit een set: Gebruik het **`SMEMBERS`** commando om alle leden van een set te krijgen. Bijvoorbeeld: **`SMEMBERS setKey`**.
6.  Operaties gesorteerd:
    -   Voeg elementen toe aan een gesorteerde set: gebruik het **`ZADD`** commando om elementen toe te voegen aan een gesorteerde set samen met hun scores. Bijvoorbeeld: **`ZADD gesorteerdeSetKey score1 score2 member2`**.
    -   Haal elementen op uit een gesorteerde set: gebruik het **`ZRANGE`** commando om een reeks elementen op te halen uit een gesorteerde set op basis van hun scores. Bijvoorbeeld: **`ZRANGE gesorteerdeSetKey start einde`**.

Opmerking: Je kunt verwijzen naar de Redis-documentatie voor een volledig uitgebreide lijst van commando's en hun gebruik.

### Gebruik geval

Vervolgens zullen we demonstreren hoe we de gegevens van `**Upstash**` naar de `table` component kunnen toewijzen.

#### Stap 1 Voeg een actie toe

Laat ons een tabel maken in Upstash genaamd **`employee_profile`,** inclusief 4 kolommen `naam`, `level`, `age`, `married`.

Dan kunnen we een nieuwe actie maken voor Upstash van de actielijst in ILLA Cloud en de naam upstash2.

Om alle gegevens in employee_profile weer te geven, zet u de code snippet hieronder in de query sectie Redis.

```sql
GET werknemer_profiel
```

Aangezien de uitvoergegevens een woordenboek zijn, willen we het veranderen in een array. Activeer het in Transformatie en zet de code hieronder in.

```jsx
const jsonString = data[0].result;

// Parse the JSON string into a JavaScript object
const jsonData = JSON.parse(jsonString);

// Access the "employees" property and map it to a new array
const mappedArray = jsonData.employees.map((employee) => ({
  name: employee.name,
  level: employee.level,
  age: employee.age,
  married: employee.married,
}));

return mappedArray;
```

![up_code](https://cdn.illacloud.com/official-website/img/docs/up_code.jpg)

Klik **`Opslaan`** en **`Run`** om deze actie te activeren.

![up_data](https://cdn.illacloud.com/official-website/img/docs/up_data.jpg)

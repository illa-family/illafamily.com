## <h2 hidden>Quickly Deploy</h2>

Je kunt ILLA op een opmerkelijk snelle manier implementeren, volg de instructies hieronder:

Download eerst dit [docker bestand](https://github.com/illacloud/deploy-illa-manually/tree/main/docker) naar je computer, voer daarna de volgende code uit.

```jsx
cd docker;
/bin/bash ./scripts/run-official-image.sh;
```

En log in met de standaard gebruikersnaam en wachtwoord:

Gebruikersnaam: **`root`**

Wachtwoord: **`password`** (alleen de zelf host modus).

## <h2 hidden>Wat is Microsoft SQL?</h2>

Microsoft SQL in Illa is een database integratie waarmee gebruikers verbinding en interactie kunnen maken met een Microsoft SQL (MS SQL) database. MS SQL is een populair open-source database beheer systeem dat bekend staat om de snelheid, betrouwbaarheid en gebruiksgemak.

Met de MS SQL-integratie in Illa kunt u gegevens opzoeken, toevoegen, bijwerken en verwijderen uit een MS SQL database met behulp van SQL commando's. U kunt ook aangepaste SQL-query's maken en uitvoeren binnen Illa. Dit maakt het eenvoudig om data-driven applicaties en dashboards te bouwen die real-time gegevens weergeven uit een MS SQL database.

### Maak MS SQL

Er zijn twee manieren om een bron in Illa te maken nadat je bent ingelogd op je Illa account.

-   Maak in Bronnen

Log in op je Illa account, selecteer `**Resources**` bovenaan de pagina, en klik op de `**Maak Nieuw**` knop.

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

Selecteer **`Microsoft SQL`** uit de database lijst.

![action_list](https://cdn.illacloud.com/official-website/img/docs/action_list.png)

Maak verbinding met de database met de vereiste parameters beschreven in `Connectie Instellingen` hieronder.

Klik op `**Test Connection**` om te zien of we met succes verbinding kunnen maken met de database. Zo ja, klik **`Resources`**, anders controleer je de hostnaam, port, gebruikersnaam en wachtwoord correct.

Na het aanmaken van een bron zal de aangepaste MS SQL worden weergegeven zoals getoond.

![ms_ex_ready](https://cdn.illacloud.com/official-website/img/docs/ms_ex_ready.png)

-   Maak in Bouwer

Log in op je Illa account, maak een project aan in Illa Builder in de `**App**` pagina en navigeer naar de **`Action List`** onderaan de pagina. Klik `**nieuw**` en selecteer dan **`Microsoft SQL`** uit de database lijst. Maak vervolgens verbinding met de database met de vereiste parameters beschreven in `Connectie Instellingen` hieronder.

Klik op `**Test Connection**` om te zien of we met succes verbinding kunnen maken met de database. Zo ja, klik **`Resources`**, anders controleer je de hostnaam, port, gebruikersnaam en wachtwoord correct.

### Verbindingsinstellingen

Hier moeten we informatie verstrekken voor het verbinden met MS SQL database.

![ms_config](https://cdn.illacloud.com/official-website/img/docs/ms_config.png)

| Eigenschappen           | vereist   | Beschrijving                                                                                                                   |
| ----------------------- | --------- | ------------------------------------------------------------------------------------------------------------------------------ |
| naam                    | verplicht | De naam voor resource bij het maken van acties in de ILLA.                                                                     |
| Hostname                | verplicht | De URL of het IP-adres van uw database                                                                                         |
| Poort                   | verplicht | Het poortnummer van de server host dat u moet gebruiken om te verbinden. Als je geen poort opgeeft, is standaard poort '1433'. |
| Database                | verplicht | De naam van de database                                                                                                        |
| Gebruikersnaam          | verplicht | de gebruikersnaam die u wilt gebruiken bij het inloggen op de MS SQL server.                                                   |
| Wachtwoord              | verplicht | Gebruik dit wachtwoord voor verificatie.                                                                                       |
| Verbindingsoptie        | optioneel | Sleutelwaarde paar om de verbindingsoptie op te geven                                                                          |
| SSL opties              | optioneel | Bepaal hoe hoog een veilige SSL TCP/IP verbinding prioriteit heeft tijdens het onderhandelen met de server.                    |

### Acties aanmaken

We hebben een MS SQL bron gemaakt; we kunnen de actie toevoegen door MS SQL uit actie lijst te selecteren en te kiezen voor de **`Create action`** knop.

![ms_resource_list](https://cdn.illacloud.com/official-website/img/docs/ms_resource_list.png)

Nu hebben we de MS SQL server toegevoegd als een actie aan onze bouwpagina.

![ms](https://cdn.illacloud.com/official-website/img/docs/ms.png)

### MS SQL configureren

| Eigenschappen    | Beschrijving                                                                                                                                                                                                                                                               |
| ---------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Configuratietype | het type configuratie dat u gebruikt om te verbinden met uw MS SQL Server database: SQL en Bulk ingevoegd. Voor SQL-modus, kunt u SQL query gebruiken. Voor Bulk invoegen modus kun je gegevens invoegen in een geselecteerde tabel met het formaat {{ [{x:a, y:b}, â€¦ ] }} |
| SQL query        | sql commando's om op te halen (select), invoegen, bijwerken, gegevens uit de database verwijderen.                                                                                                                                                                         |
| Transformator    | data omzetten naar de stijl die je leuk vindt met behulp van JavaScript                                                                                                                                                                                                    |

Voorbeeld gebruik:

-   Selecteren

```sql
SELECT * FROM users ORDER BY lastname;
```

-   Insert

```sql
INSERT INTO users

	(firstname, lastname, email)

VALUES

	(

		{{ NewUser.data.firstNameInput }},

		{{ NewUser.data.lastNameInput }},

		{{ NewUser.data.emailInput }}

	);
```

-   Vernieuwen

```sql
UPDATE users
  SET firstname = '{{form1.updatedProfile.first}}'
  WHERE id = {{ form1.updatedProfile.uid}};
```

-   Verwijderen

```sql
DELETE FROM users WHERE id = {{ form1.updatedProfile.uid }};
```

## <h2 hidden>Wat is een tijdsbereik?</h2>

Tijd Bereik component is een UI component waarmee gebruikers meerdere keren kunnen selecteren van een 24-uurs interface, Doorgaans gebruikt in formulieren of gegevensinvoerinterfaces waar de gebruiker een start- en eindtijdbereik moet opgeven.

Wanneer een tijdbereik onderdeel wordt toegevoegd aan het canvas, wordt er 24 uur naast elkaar twee interfaces weergegeven. EÃ©n interface wordt gebruikt om de starttijd te selecteren, de andere wordt gebruikt om de eindtijd te selecteren. De gebruiker kan de tijd aanpassen door de muiscursor te verplaatsen of de pijlen omhoog en omlaag te gebruiken.

### Gebruik geval

Vervolgens zullen we laten zien hoe we de data van de gegevensbron kunnen koppelen aan tijdbereik en de eindwaarde kunnen instellen aan de huidige tijd met een knop. 

#### Stap 1 Voeg een actie toe

Laat ons een tabel aanmaken in REST API genaamd **`promoteCodes`,** inclusief 6 kolommen: **`code`**, **`id`**, **`createdAt`**, **`expiredAt`**, **`startedAt`, `updatedAt`**. We zullen het startedAt en verlopen voor voorbeeldgebruik. 

Dan kunnen we een nieuwe actie voor REST API maken uit de actielijst, waarin alle gegevens in promoteCodes en restapi1 worden weergegeven. 

Klik **`Opslaan`** en **`Run`** om deze actie te activeren.

#### Stap 2 Componenten toevoegen

Vervolgens kunnen we een **`time range`** component en een **`button`** component toevoegen aan het canvas. Voor de standaard begintijdwaarde van het datumbereik gebruiken we de startOp tijd van de eerste promotiecode van de rest van de api gegevens.

```sql
{{restapi1.data.promoteCodes[0].startedAt}}
```

Evenzo gebruiken we voor de standaard eindtijdwaarde VervaldatumOp het moment van de eerste promotiecode van de rest van de api data.

```sql
{{restapi1.data.promoteCodes[0].expiredAt}}
```

We noemden de knop ook als "Nu" zoals hieronder weergegeven.

![time_range_0](https://cdn.illacloud.com/official-website/img/docs/time_range_0.png)

#### Stap 3 Configureer het component

Voor het **`button`** component kunnen we het configureren om de einddatum waarde in **`time range`** op de datum van vandaag in te stellen.

1.  In de Event Handler Bewerken Selecteer **`Control component`** in actie, selecteer de **`time range`** component die u wilt bijwerken als doel van het event.
2.  Kies de **`setEndTime`** actie en zet de **`End time`** als `{{currentUserInfo.updatedAt}}` 

![time_range_config](https://cdn.illacloud.com/official-website/img/docs/time_range_config.png)

Nu wanneer je op de knop "Nu" klikt, moet de eindtijd veranderen naar de datum van vandaag. Wat dit betreft hebben we 20:03:05 maar het kan anders zijn als je het testen.

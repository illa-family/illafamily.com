## <h2 hidden>Wat is tabel?</h2>

De tabelcomponent in ILLA Cloud is een krachtige functie waarmee gebruikers tabelgegevens binnen de applicatie kunnen weergeven, organiseren en manipuleren. Het biedt een gestructureerde indeling met rijen en kolommen, zodat gebruikers op een handige en gestructureerde manier gegevens kunnen bekijken en uitwisselen.

### Eigenschappen

| Eigenschappen                      | Beschrijving                                                                                                                                |
| ---------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------- |
| Data bron                          | bron van gegevens die de tabel populeert                                                                                                    |
| Lege staat                         | de inhoud of het bericht dat wordt weergegeven als de tabel geen gegevens te tonen heeft                                                    |
| Laden...                           | of de tabelcomponent zich momenteel in een staat van laden bevindt                                                                          |
| Kolommen                           | structuur en configuratie van de kolommen in de tabel. (Zie het gedeelte over de kolommen hieronder voor meer details)                      |

## Gebruik geval

Vervolgens zullen we demonstreren hoe we de gegevens van `**Upstash**` kunnen toewijzen aan het `**table**` component.

#### Stap 1 Voeg een actie toe

Laat ons een tabel maken in Upstash genaamd **`employee_profile`,** inclusief 4 kolommen `naam`, `level`, `age`, `married`.

Dan kunnen we een nieuwe actie voor Upstash van de actielijst in ILLA Cloud maken met de naam upstash2.

Om alle gegevens in employee_profile weer te geven, zet u de code snippet hieronder in de query sectie Redis.

```sql
GET werknemer_profiel
```

Aangezien de uitvoergegevens een woordenboek zijn, willen we het veranderen in een array. Activeer het in Transformatie en zet de code hieronder in.

```jsx
const jsonString = data[0].result;

// Parse the JSON string into a JavaScript object
const jsonData = JSON.parse(jsonString);

// Access the "employees" property and map it to a new array
const mappedArray = jsonData.employees.map((employee) => ({
  name: employee.name,
  level: employee.level,
  age: employee.age,
  married: employee.married,
}));

return mappedArray;
```

Klik **`Opslaan`** en **`Run`** om deze actie te activeren.

![up_data](https://cdn.illacloud.com/official-website/img/docs/up_data.jpg)

Vervolgens kunnen we een **`table`** component toevoegen aan het canvas.

In het **`Data Source`** veld in het Inspect pagina nadat je op het component klikt, zet `{{upstash2.data}}` om de gegevens van Upstash op te halen.

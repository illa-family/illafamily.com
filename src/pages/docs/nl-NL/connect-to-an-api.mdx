## <h2 hidden>Verbind met een API</h2>

Verbind met elke REST, GraphQL API.

Je kunt ILLA verbinden met de meeste API's met behulp van onze native integraties. Als je verbinding moet maken met een interne API of als er geen ondersteunde integratie is je kunt verbinden via onze REST API integratie of GraphQL Integratie.

### 1.  Maak een API bron aan

Bronnen vertegenwoordigen verbindingen met gegevensbronnen en zijn wat u via uw ILLA-apps vraagt.Bronnen vertegenwoordigen verbindingen met gegevensbronnen en zijn wat u via uw ILLA-apps vraagt.Bronnen vertegenwoordigen verbindingen met gegevensbronnen en zijn wat u via uw ILLA-apps vraagt.Log in bij ILLA en navigeer naar het tabblad **Bronnen** > klik **Nieuwe maken** en selecteer de API waarmee u wilt verbinden zoals **REST API**. Hiermee opent u de **Configureer een resource** modal , waar u uw verbindingsinformatie toevoegt.

<img
  src='https://cdn.illacloud.com/official-website/img/docs/connect/connect-api/config.png'
  alt=""
/>

Minimaal moet je de basis-URL van de API waarmee je verbinding wilt maken. U kunt ook URL-parameters, kopteksten, etc. doorgeven. Als je API authenticatie vereist, kun je dat ook op deze pagina configureren. Bijvoorbeeld, OAuth2.0 vereist een Authorization URL, Access Token URL, etc. Nadat je je connectie gegevens hebt opgegeven, klik je op **Opslaan resource.**

### 2.  Vraag je API aan

In de app-editor, klik **+ Nieuw** in **Actie-lijst** > selecteer **REST API** en kies een bestaande REST API of maak een nieuwe > klik **maak actie** en werk de acties bij met extra URL parameters, headers, etc.

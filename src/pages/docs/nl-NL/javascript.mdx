## <h2 hidden>JavaScript Guider</h2>

JavaScript is de belangrijkste component van de ILLA Bouwer, Dit is het hulpmiddel dat u gebruikt om de toegang tot gegevens en logica-constructie van uw API op te bouwen. Masteren hoe JavaScript op te roepen, je hebt het gebruik beheerst van ILLA Builder.

When building your APP in ILLA Builder, you can use `{{template syntax}}` to write custom JavaScript statements for calling and manipulating data, or write JS code directly in the transformer.

### Operatie gegevens

#### Toegang tot gegevens op objecten

In ILLA Builder kunt u JavaScript-expressies in elk tekstinvoervak invoeren met behulp van de sjabloonsyntaxis. Data-objecten zijn refereerbaar, en je kunt de gegevens van alle invoegingen, acties en transformers bewerken via JavaScript. U kunt bijvoorbeeld `{{textInput.data}}` gebruiken voor toegang tot de inhoud van een tekstinvoer ingevoegd, `{{action.data}}` om toegang te krijgen tot de gegevens uit een actie-database en `{{transformer.value}}` om toegang te krijgen tot de gegevens van een transformator.

#### Toegang tot nummers in arrays

Wanneer je toegang nodig hebt tot een waarde in een specifieke array, u kunt de waarde krijgen door de positie of het abonnement van de gegevens die u nodig heeft te gebruiken.

```javascript
const list = ['a', 'b', 'c']
return list[0] // Retourneer de eerste waarde in lijst 'a'
```

### Gegevens weergeven

Er zijn vier locaties om gegevens in ILLA Builder te bekijken: de dataseruimte in het linkerpaneel, de data prompt onder het tekstvak na het invoeren van een JavaScript expressie, en de data display via de data invoegen om de gegevens weer te geven.

#### Data werkruimte presentatie

Je vindt de gegevenswerkruimte door het linkerpaneel in de app-editor te openen, dus de gegevens worden gepresenteerd als gestructureerde JSON waarin elke eigenschap en lijst kan worden uitgebreid en alle gegevens visueel worden weergegeven in de werkruimte.

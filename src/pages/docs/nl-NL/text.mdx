## <h2 hidden>Text</h2>

<img src='https://cdn.illacloud.com/official-website/img/docs/assemble/text1.png' alt="" />

### Eigenschappen

| Eigenschap             | Beschrijving                                                                                                                                                                                                                                         |
| ---------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Inhoud                 | Stel de tekst in die getoond wordt op het tekstcomponent. Markdown en platte tekst worden ondersteund.                                                                                                                                               |
| Knopinfo               | Een tooltip weergegeven op het onderdeel wanneer gebruikers over het onderdeel bewegen.                                                                                                                                                              |
| Horizontale uitlijning | Om de horizontale positie van de tekstinhoud in de component in te stellen                                                                                                                                                                           |
| Verticale uitlijning   | Om de verticale positie van de tekstinhoud in het component in te stellen                                                                                                                                                                            |
| Hidden                 | Om in te stellen of deze component weer te geven onder welke voorwaarden het moet worden weergegeven. Bijvoorbeeld, als de huidige gebruiker geen Gebruiker A is, moet dit component verborgen zijn: {{ currentUserInfo.nickname != 'Gebruiker A' }} |
| Kleur                  | Om de tekstkleur in tekentekstformaat in te stellen                                                                                                                                                                                                  |
| Tekst grootte          | Om de tekstgrootte in tekentekst in te stellen                                                                                                                                                                                                       |

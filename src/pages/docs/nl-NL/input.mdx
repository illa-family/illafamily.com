## <h2 hidden>Wat is Input?</h2>

Tekstinvoercomponent is een gebruikersinterface element waarmee gebruikers tekst in een formulier of invoerveld kunnen invoeren en bewerken.

### Methode

U kunt andere componenten gebruiken om het onderdeel te beheren. Wij steunen de volgende twee methoden:

-   **setvalue**

Om de invoerwaarde in te stellen, bijvoorbeeld {{'value1'}}

| Eigenschappen | Beschrijving |
| ------------- | ------------ |
| Waarde        | Input value  |

-   **clearvalue**

Om de waarde van het geselecteerde component te wissen 

-   **focus**

Wanneer de focus methode wordt gebeld, het invoerveld zal gemarkeerd worden en klaar voor de gebruiker om te beginnen met typen zonder dat de gebruiker hoeft te klikken op het invoerveld

Voorbeeld gebruik:

Input component support luisteren naar de `onChange` gebeurtenis van andere componenten met behulp van het ingebouwde event systeem. Instellen door de volgende voorbeeld stappen te volgen:

1.  Voeg een gebeurtenistrigger toe aan de component waarnaar u wilt luisteren. Bijvoorbeeld, als u wilt luisteren naar het **`onChange`** evenement van een Radio Group component, je zou een event-handler aan dat component van Radio Group toevoegen.
2.  In de Event Handler bewerken, selecteer **`Control component`** in actie, selecteer het **`Input`** onderdeel dat u wilt bijwerken als het doel van het event.
3.  Kies de **`SetValue`** actie en selecteer de juiste waarde voor de input component. Dit zal de waarde zijn die de invoercomponent heeft en zal bijwerken wanneer de gebeurtenis wordt geactiveerd.
4.  Sla de instellingen voor de trigger van de gebeurtenis op en herhaal het proces voor alle andere componenten waarnaar u wilt luisteren.

Zodra u de event triggers hebt ingesteld, wordt het invoercomponent automatisch bijgewerkt wanneer het **`onChange`** evenement wordt geactiveerd op de andere componenten. Dit stelt u in staat om dynamische interfaces te maken die reageren op gebruikersinvoer in realtime het makkelijker maken voor gebruikers om te navigeren en te communiceren met uw applicatie.

### Event handler

| Gebeurtenis | Beschrijving                                                                                             |
| ----------- | -------------------------------------------------------------------------------------------------------- |
| Veranderen  | Wanneer een gebruiker de geselecteerde invoerwaarde wijzigt                                              |
| Focus       | Wanneer een gebruiker de muiscursor beweegt op het invoercomponent                                       |
| Blur        | Wanneer een gebruiker wordt gedaan om waarde in te voeren en stopt met focussen op de cascader component |

### Gegevens

Het component heeft enkele vaak gebruikte gegevens, die kunnen worden opgeroepen via **`{{componentName.propertyName}}`** in de app.

| Naam van eigenschap | Beschrijving                                                                         |
| ------------------- | ------------------------------------------------------------------------------------ |
| waarde              | gebruiker invoerwaarde                                                               |
| kleurenschema       | achtergrondkleur van de knop                                                         |
| uitgeschakeld       | een booleaanse waarde geeft de uitgeschakelde status aan                             |
| weergavenaam        | de naam van dit onderdeel (bijv. knop1)                                              |
| formDataKey         | de formuliergegevenssleutel van invoer                                               |
| hidden              | verborgen status (waar of onwaar)                                                    |
| Label               | label waarde                                                                         |
| uitlijnen           | uitlijning van invoerlabel (links of rechts)                                         |
| labeling            | positie van invoerlabel (links of rechts)                                            |
| labelBreedte        | het geheel getal waarin de breedte van het label wordt weergegeven.                  |
| maxLengte           | de waarde van de maximale lengte                                                     |
| minnLengte          | de waarde van de minimale lengte                                                     |
| placeholder         | placeholder waarde                                                                   |
| prefixText          | waarde van het tekstvoorvoegsel                                                      |
| achtervoegseltekst  | waarde van het tekstachtervoegsel                                                    |
| alleen lezen        | een Booleaanse waarde geeft de readyOnly status van invoer aan.                      |
| verplicht           | een Booleaanse waarde geeft de vereiste status van invoer aan.                       |
| regex               | reguliere expressie van input                                                        |
| toon aantal         | een Booleaanse waarde geeft aan of het aantal tekens van de invoer wordt weergegeven |
| tooltipText         | waarde van tekst tooltip                                                             |

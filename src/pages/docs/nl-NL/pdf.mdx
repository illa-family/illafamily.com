## <h2 hidden>Wat is PDF?</h2>

Het PDF-component is een functie of element waarmee gebruikers de applicatie kunnen bekijken en communiceren met PDF-documenten. Het biedt functionaliteit voor het renderen, navigeren en manipuleren van PDF-bestanden, waardoor gebruikers naadloos toegang krijgen tot en werken met PDF-inhoud.

### Eigenschappen

| Eigenschappen       | Beschrijving                                                                                                   |
| ------------------- | -------------------------------------------------------------------------------------------------------------- |
| PDF bron            | Geef de bron van de te tonen pdf op                                                                            |
| Toon werkbalk       | controleert de zichtbaarheid van de werkbalk binnen het PDF-component                                          |
| Knopinfo            | voeg een tooltip of een beschrijvende tekst toe die wordt weergegeven wanneer de gebruiker over de pdf beweegt |
| Pagina schalen naar | bepaalt het schaalgedrag van de PDF-pagina's binnen het onderdeel                                              |
| Hidden              | bepaalt de zichtbaarheid van het PDF-component                                                                 |

### Methode

U kunt andere componenten gebruiken om het onderdeel te beheren. Wij steunen de volgende methoden:

-   **setFileUrl**

De bron of URL van het PDF-bestand dynamisch wijzigen.

| Eigenschappen | Beschrijving    |
| ------------- | --------------- |
| PDF Bron      | pdf bron url    |

### Gegevens

Het component heeft enkele vaak gebruikte gegevens, die kunnen worden opgeroepen via **`{{componentName.propertyName}}`** in de app.

| Naam van eigenschap | Beschrijving                                                                                                                   |
| ------------------- | ------------------------------------------------------------------------------------------------------------------------------ |
| weergavenaam        | Een tekenreeks die de naam of titel van de PDF weergeeft.                                                                      |
| hoogte              | de verticale dimensie of grootte van het PDF-component                                                                         |
| hidden              | Een booleaanse waarde die bepaalt of de PDF-component verborgen is in de gebruikersinterface.                                  |
| schaalmodus         | Specificeert hoe het PDF-document wordt geschaald of past binnen het beschikbare weergavegebied van het component              |
| showToolBar         | of er een werkbalk of controlepaneel wordt weergegeven samen met het PDF-component                                             |
| tooltipText         | tekenreekswaarde die de tooltip of de zweeft over tekst om weer te geven wanneer de gebruiker over het onderdeel pdf beweegt.  |
| URL                 | bron van het PDF-bestand dat in het component wordt weergegeven                                                                |
| width               | de horizontale dimensie of grootte van het PDF-component                                                                       |

Voorbeeld: {{pdf1.displayName}}

### Gebruik geval

Vervolgens laten we zien hoe we de pdf weergave met een knop kunnen bedienen.

#### Stap 1 Componenten toevoegen

Voeg een **`PDF`** component en een **`button`** toe aan het canvas.

![pdf_layout](https://cdn.illacloud.com/official-website/img/docs/pdf_layout.jpg)

#### Stap 2 Configureer het component

Voor het **`button`** component kunnen we het configureren om de bron van de afbeelding van het **`PDF`** component op een pdf adres in te stellen.

1.  Klik op de `**button**` component om de inspectie pagina te openen. Onder **`Event handler`**, klik `**+ Nieuw**`.
2.  In de Event Handler Bewerken Selecteer **`Control component`** in actie, en selecteer het onderdeel **`PDF`** dat u wilt veranderen als doel van het event.
3.  Kies de **`setFileUrl`** actie
4.  Plaats het nieuwe adres van de afbeelding in het veld **`Value`**

![pdf_config](https://cdn.illacloud.com/official-website/img/docs/pdf_config.jpg)

#### Stap 3 test

Wanneer je nu op de knop "Afspelen" klikt, moet de afbeelding wisselen

![pdf_test](https://cdn.illacloud.com/official-website/img/docs/pdf_test.jpg)

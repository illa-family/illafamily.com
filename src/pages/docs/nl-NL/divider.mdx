## <h2 hidden>Wat is verdeler?</h2>

In ILLA Cloud is de verdelingscomponent een visueel element dat helpt om secties van inhoud binnen een webapplicatie te scheiden of te delen. Het is een eenvoudige horizontale of verticale lijn die tussen verschillende secties of onderdelen kan worden ingevoegd om de visuele organisatie en de helderheid van de gebruikersinterface te verbeteren.

### Eigenschappen

| Eigenschappen   | Beschrijving                                                                            |
| --------------- | --------------------------------------------------------------------------------------- |
| Tekstveld       | voeg aanvullende informatie toe aan de verdeler component                               |
| Text align      | bepaalt de horizontale uitlijning van de tekst in de verdeler                           |
| Hidden          | verberg het component van de deler van de gebruikersinterface                           |
| Tekst grootte   | de lettergrootte van de tekst in de component van de deler aanpassen                    |

### Gegevens

Het component heeft enkele vaak gebruikte gegevens, die kunnen worden opgeroepen via **`{{componentName.propertyName}}`** in de app.

| Naam van eigenschap | Beschrijving                                                                                |
| ------------------- | ------------------------------------------------------------------------------------------- |
| kleurenschema       | definieert de kleur van de tekst in het component                                           |
| weergavenaam        | vertegenwoordigt de naam of het label die is toegewezen aan het component                   |
| fs                  | bestandssysteem (fs) geassocieerd met de delvider                                           |
| hidden              | booleaanse waarde die bepaalt of de component zichtbaar of verborgen is                     |
| Tekst               | de tekstwaarde                                                                              |

Voorbeeld: {{divider1.text}}

### Gebruik geval

Je kunt het overal plaatsen waar je de pagina in twee secties wilt verdelen. Hier laten we de verdeler zien met drie verschillende tekstlijnen.

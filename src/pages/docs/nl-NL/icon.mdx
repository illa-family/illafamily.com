## <h2 hidden>Wat is icoon?</h2>

Het Icon component in ILLA Cloud is een krachtig kenmerk waarmee gebruikers de visuele aantrekkingskracht en functionaliteit van hun toepassingen kunnen verbeteren. Met het onderdeel Icoon kunnen gebruikers eenvoudig iconen van populaire icoonbibliotheken toevoegen aan hun gebruikersinterfaces, bieden hun een meer intuÃ¯tieve en boeiende ervaring aan hun gebruikers.

### Eigenschappen

| Eigenschappen      | Beschrijving                                                                                                                                                   |
| ------------------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Pictogram          | stelt gebruikers in staat om gemakkelijk een pictogram te selecteren uit een vooraf gedefinieerde set opties                                                   |
| Event Handler      | Trigger query's, beheer componenten of bel andere API's in reactie op gebeurtenissen in component.                                                             |
| Knopinfo           | Gebruikers kunnen hier de tooltip van de component invoeren. De tooltip wordt getoond wanneer de focus is gefocuerd. Markdown formaat wordt ondersteund.       |
| Hidden             | Dynamisch beheren of het onderdeel verborgen is. Je kunt de verborgen status wijzigen via een dynamische booleaanse waarde.                                    |
| Thema kleur        | Sta gebruikers toe om de achtergrondkleur en ondoorzichtigheid van de knop te specificeren                                                                     |

### Event handler

| Gebeurtenis | Beschrijving                                                                                              |
| ----------- | --------------------------------------------------------------------------------------------------------- |
| Click       | Wanneer een gebruiker op dit pictogram klikt, voer specifieke actie uit die is aangepast door gebruikers. |

### Gegevens

Het component heeft enkele vaak gebruikte gegevens, die kunnen worden opgeroepen via **`{{componentName.propertyName}}`** in de app.

| Naam van eigenschap | Beschrijving                                                                                            |
| ------------------- | ------------------------------------------------------------------------------------------------------- |
| kleurenschema       | kleur van het icoon omlijnd                                                                             |
| weergavenaam        | de naam van dit onderdeel (d.w.z. icon1)                                                                |
| evenementen         | gegenereerd of geactiveerd wanneer bepaalde acties of gebeurtenissen binnen het component plaatsvinden. |
| hidden              | verborgen status (waar of onwaar)                                                                       |
| iconNaam            | de labelnaam voor het icoon                                                                             |
| tooltipText         | waarde van tekst tooltip                                                                                |

Voorbeeld: {{icon1.iconName}}

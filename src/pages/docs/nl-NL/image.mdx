## <h2 hidden>Wat is afbeelding?</h2>

De afbeeldingscomponent in ILLA Cloud is een veelzijdig hulpmiddel waarmee gebruikers afbeeldingen kunnen toevoegen en weergeven binnen hun applicaties. Het biedt verschillende functies om de visuele ervaring voor gebruikers te verbeteren.

### Eigenschappen

| Eigenschappen           | Beschrijving                                                                                                                                                  |
| ----------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Bron afbeelding         | Specificeer de bron van de afbeelding die moet worden weergegeven                                                                                             |
| Alternatieve tekst      | alternatieve tekst voor de afbeelding. Het wordt weergegeven wanneer de afbeelding niet kan worden geladen of voor gebruikers met visuele beperkingen         |
| Scale Type              | hoe de afbeelding wordt geschaald binnen de afmetingen van de component, inclusief container, cover, vul, niets, schaalomlaag.                                |
| Event Handler           | staat gebruikers toe om acties te definiÃ«ren die worden geactiveerd door interacties van de gebruiker met de afbeelding                                       |
| Knopinfo                | voeg een tooltip of een beschrijvende tekst toe die wordt weergegeven wanneer de gebruiker over de afbeelding beweegt                                         |
| Hidden                  | de zichtbaarheid van het onderdeel afbeelding beheren.                                                                                                        |
| Styles                  | staat gebruikers toe om het uiterlijk aan te passen. Gebruikers kunnen stijlen toepassen zoals radius                                                         |

### Methode

U kunt andere componenten gebruiken om het onderdeel te beheren. Wij steunen de volgende methoden:

-   **setImageUrl**

De bron of URL van de afbeelding dynamisch wijzigen.

| Eigenschappen   | Beschrijving        |
| --------------- | ------------------- |
| Afbeelding URL  | bron afbeelding url |

### Event handler

| Gebeurtenis | Beschrijving                                                                         |
| ----------- | ------------------------------------------------------------------------------------ |
| Click       | specifieke acties of code uitvoeren wanneer op de afbeeldingscomponent wordt geklikt |

### Gegevens

Het component heeft enkele vaak gebruikte gegevens, die kunnen worden opgeroepen via **`{{componentName.propertyName}}`** in de app.

| Naam van eigenschap | Beschrijving                                                                                                                                       |
| ------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------- |
| altText             | een tekstgebaseerd alternatief dat de inhoud of het doel van de afbeelding beschrijft                                                              |
| weergavenaam        | Een tekenreeks die de naam of titel van de afbeelding weergeeft.                                                                                   |
| evenementen         | Een array of object die de event handlers definieert die zijn geassocieerd met de afbeelding component, zoals klik                                 |
| hidden              | Een booleaanse waarde die bepaalt of de afbeeldingscomponent verborgen is in de gebruikersinterface.                                               |
| afbeeldingSrc       | de bron of locatie van de afbeelding die in het component wordt weergegeven                                                                        |
| ObjectFit           | hoe de afbeelding wordt aangebracht of geschaald binnen de afmetingen van de component                                                             |
| radius              | Pas afgeronde hoeken toe op de afbeeldingscomponent.                                                                                               |
| tooltipText         | tekenreekswaarde die de tooltip of de zwevende tekst vertegenwoordigt om weer te geven wanneer de gebruiker over het afbeeldingscomponent beweegt. |

### Gebruik geval

Vervolgens laten we zien hoe we de afbeelding met een knop kunnen bedienen.

#### Stap 1 Componenten toevoegen

Voeg een **`Image`** component en een **`button`** component toe aan het canvas.

Je kunt de `afbeeldingsbron` van het afbeeldingscomponent instellen op elk adres dat je wilt weergeven. Hier gebruiken we de standaard afbeelding als een beginvoorbeeld en label de knop als "Wijzig" zoals hieronder weergegeven

![image_layout](https://cdn.illacloud.com/official-website/img/official-site/components/image_layout.jpg)

#### Stap 2 Configureer het component

Voor het **`button`** component kunnen we het configureren om de bron van de afbeelding van het **`image`** component op een nieuw afbeeldingsadres in te stellen.

1.  Klik op de `**button**` component om de inspectie pagina te openen. Onder **`Event handler`**, klik `**+ Nieuw**`.
2.  In de Event Handler Bewerken Selecteer **`Control component`** in actie, en selecteer het onderdeel **`image1`** dat je wilt veranderen als doel van het event.
3.  Kies de **`setImageUrl`** actie
4.  Plaats het nieuwe adres van de afbeelding in het veld **`Value`**

![image_config](https://cdn.illacloud.com/official-website/img/official-site/components/image_config.jpg)

#### Stap 3 test

Wanneer je nu op de knop "Afspelen" klikt, moet de afbeelding wisselen

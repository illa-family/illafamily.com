## <h2 hidden>Wat is Hydra?</h2>

Hydra is een krachtige integratie binnen het ILLA Cloud-platform dat diverse gegevensbronnen en toepassingen naadloos verbindt en consolideert. Hydra fungeert als een centraal hub, en maakt efficiÃ«nte gegevensstroom en communicatie tussen verschillende systemen mogelijk, wat real-time data-synchronisatie mogelijk maakt, geautomatiseerde workflows en gestroomlijnde processen. Met zijn robuuste mogelijkheden zorgt Hydra ervoor dat bedrijven hun activiteiten optimaliseren, de samenwerking vergroten, en maak met gemak beslissingen op basis van data binnen het geÃ¯ntegreerde ecosysteem van ILLA Cloud.

### Maak Hydra

Er zijn twee manieren om een bron te maken in ILLA na het inloggen op uw account.

-   Maak in Bronnen

Log in op je Illa account, selecteer `**Resources**` bovenaan de pagina, en klik op de `**Maak Nieuw**` knop.

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

Selecteer **`Hydra`** uit de database lijst.

![action_list](https://cdn.illacloud.com/official-website/img/docs/action_list.png)

Maak verbinding met de database met de vereiste parameters beschreven in `Connectie Instellingen` hieronder.

Klik op `**Test Connection**` om te zien of we met succes verbinding kunnen maken met de database. Zo ja, klik **`Resources`**, anders controleer je de hostnaam, port, gebruikersnaam en wachtwoord correct.

Na het creÃ«ren van een grondstof, wordt de geopende Hydra weergegeven zoals getoond.

![hydra_ex_ready](https://cdn.illacloud.com/official-website/img/official-site/intergration/hydra_ex_ready.jpeg)

-   Maak in Bouwer

Log in op je Illa account, maak een project aan in Illa Builder in de `**App**` pagina en navigeer naar de **`Action List`** onderaan de pagina. Klik op `**nieuw**`, en selecteer dan **`Hydra`** uit de database lijst. Maak vervolgens verbinding met de database met de vereiste parameters beschreven in `Connectie Instellingen` hieronder.

Klik op `**Test Connection**` om te zien of we met succes verbinding kunnen maken met de database. Zo ja, klik **`Resources`**, anders controleer je de hostnaam, port, gebruikersnaam en wachtwoord correct.

### Configure Hydra

| Eigenschappen  | Beschrijving                                                                                      |
| -------------- | ------------------------------------------------------------------------------------------------- |
| SQL query      | sql commando's om op te halen (select), invoegen, bijwerken, gegevens uit de database verwijderen |
| Transformator  | data omzetten in de stijl die je leuk vindt met Javascript                                        |

Voorbeeld gebruik:

-   Selecteren

```sql
SELECTEER * VAN gebruikers BY achternaam;
```

-   Insert

```sql
INSERT INTO users

	(firstname, lastname, email)

VALUES

	(

		{{ NewUser.data.firstNameInput }},

		{{ NewUser.data.lastNameInput }},

		{{ NewUser.data.emailInput }}

	);
```

-   Vernieuwen

```sql
UPDATE users
  SET firstname = '{{form1.updatedProfile.first}}'
  WHERE id = {{ form1.updatedProfile.uid}};
```

-   Verwijderen

```sql
DELETE FROM users WHERE id = {{ form1.updatedProfile.uid }};
```

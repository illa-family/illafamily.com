## <h2 hidden>Wat is Modal?</h2>

In ILLA Cloud is het modale component een gebruikersinterface element dat wordt gebruikt om inhoud weer te geven. informatie of acties in een gericht en overlapt venster. De modal component verschijnt meestal boven op het hoofdtoepassingsscherm, onderbreekt tijdelijk de workflow van de gebruiker en vestigt hun aandacht op de modale inhoud.

### Eigenschappen

| Eigenschappen                       | Beschrijving                                                                                                      |
| ----------------------------------- | ----------------------------------------------------------------------------------------------------------------- |
| Event handler                       | detecteren en reageren op specifieke gebruikersacties of gebeurtenissen, zoals klikken, toetsen, waarden wijzigen |
| Klik op masker om modaal te sluiten | het klikken op achtergrond overlay of masker van de modder sluit het modaal venster                               |
| Toon koptekst                       | of het onderdeel een kop- of titelsectie moet weergeven                                                           |
| Toon voettekst                      | of het onderdeel een voettekst sectie moet weergeven                                                              |
| Rand kleur                          | kleur van de rand die een modaal onderdeel omringt                                                                |
| Rand Straal                         | kromming van de hoeken van een modale component grens                                                             |
| Border Width                        | dikte of breedte van de grens om een modale component                                                             |
| Achtergrond                         | definieer de kleur of afbeelding die het inhoudsgebied van een modale component vult                              |

### Methode

U kunt andere componenten gebruiken om het onderdeel te beheren. Wij steunen de volgende twee methoden:

-   openModal

de modal component openen

-   closeModal

sluit de modal component

### Gebruik geval

We zullen laten zien hoe we een formulier maken met naam en leeftijd, als invoer, evenals vooruitgang op de balk om de voortgang van het formulier te laten zien.

#### Stap 1 Componenten toevoegen

Voeg een **`button`** component "Open Modal" toe

Voeg een modal component toe aan het canvas.

#### Stap 2 Configureer het component

Voor de "Open Modal" **`button`** componenten, kunnen we het configureren om het **`modal`** component te openen als er op geklikt wordt.

1.  Klik op de **`button`** component om de inspectie pagina te openen. Onder **`Event handler`**, klik `**+ Nieuw**`.
2.  In de Event Handler Bewerken Selecteer **`Control component`** in actie, en selecteer de **`modal`** component die u wilt bijwerken als doel van het event.
3.  Kies de **`openModal`** methode

![open_modal](https://cdn.illacloud.com/official-website/img/docs/open_modal.jpg)

Klik op deze knop om de modale component te openen en je kunt alles in het midden van de modale plaatsen plaatsen.

Voor het "Annuleren" **`button`** component, kunnen we het configureren om het venster te sluiten als er op geklikt wordt

1.  Klik op de **`button`** component om de inspectie pagina te openen. Onder **`Event handler`**, klik `**+ Nieuw**`.
2.  In de Event Handler Bewerken Selecteer **`Control component`** in actie, en selecteer de **`modal`** component die u wilt bijwerken als doel van het event.
3.  Kies de **`closeModal`** methode

![close_modal](https://cdn.illacloud.com/official-website/img/docs/close_modal.jpg)

Op dezelfde manier kunnen we de "Bevestigen" knop configureren om gefeliciteerd te worden

1.  Klik op de **`button`** component om de inspectie pagina te openen. Onder **`Event handler`**, klik `**+ Nieuw**`.
2.  In de Event handler bewerken, selecteer **`Toon melding`** in actie, **`title`** "Gefeliciteerd!" en **`description`** "Gefeliciteerd!". Kies `Success` voor **`type`**, zet `{{2000}}` voor de **`duration`** (2 seconden)

![modal_confirm](https://cdn.illacloud.com/official-website/img/docs/modal_confirm.jpg)

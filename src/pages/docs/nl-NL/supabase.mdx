## <h2 hidden>Supabase DB</h2>

Supabase is een open-source vuurbasis alternatief waarmee ontwikkelaars real-time web en mobiele applicaties kunnen bouwen. Het biedt veel van dezelfde functies als Firebase, zoals real-time databases, gebruikersauthenticatie, en hosting, maar met de toegevoegde voordelen van open-source zijn en het aanbieden van een SQL API. Dit stelt ontwikkelaars in staat hun bestaande SQL kennis en tools te gebruiken, waardoor het gemakkelijk is om te integreren met andere systemen. Daarnaast biedt Supabase een flexibeler en aanpasbaar platform dan Firebase, die restrictiever kan zijn met betrekking tot het modelleren en gebruik van gegevens.

Deze handleiding schetst het proces van het maken van een Admin Panel met behulp van ILLA Builder en Supabase in een paar eenvoudige stappen. ILLA is een laag programmeerplatform voor ontwikkelaars dat het mogelijk maakt interne hulpmiddelen snel te ontwikkelen en te implementeren. Het maakt het mogelijk om pagina's te maken door UI-componenten te slepen en te laten vallen, door verbinding te maken met elke database of API en door JavaScript te schrijven. Om meer te weten te komen over Supabase.com/, ga naar hun website op \[https://supabase.com/). Laten we beginnen!

  src='https://cdn.illacloud.com/official-website/img/docs/assemble/supabase6.png'

Klik in de actielijst op `+ Nieuw` en selecteer Supabase DB.

<img
  src='https://cdn.illacloud.com/official-website/img/docs/assemble/supabase7.png'
  alt=""
/>

Vul het formulier in om verbinding te maken met uw Supabase instantie. Verbinding testen en bron opslaan.

<img
  src='https://cdn.illacloud.com/official-website/img/docs/assemble/supabase8.png'
  alt=""
/>

Klik op 'Actie maken' om een actie aan te maken met de Supabase resource en configuratie van uw CRUD.

<img
  src='https://cdn.illacloud.com/official-website/img/docs/assemble/supabase9.png'
  alt=""
/>

Gebruik {{ om de front-end invoergegevens te krijgen. Het volgende is een voorbeeld van de pagina Gebruikersbeheer in het Beheer Paneel.

Zoek naar een gebruiker met de naam ingevoerd in invoer1

```
SELECTEER *
VAN gebruiker
WAAR naam = "{{input1.value}}"
;
```

Gebruikersgegevens bijwerken. Update gebruikersinformatie wanneer id overeenkomt

```
UPDATE gebruiker
SET naam = "{{input3.value}}"
, email = "{{input4.value}}"
WHERE id="{{input2.value}}"
;
```

Gebruikersgegevens toevoegen

```
INSERT INTO gebruiker VALUES("{{input5.value}}","{{input6.value}}","{{input7.value}}");
```

Verwijder een gebruiker met id

```
VERWIJDER VAN gebruiker WHERE id = "{{input2.value}}";
```

## <h2 hidden>Appwrite Database</h2>

Binnen de verschillende diensten van Appwrite's kunt u gebruikersauthenticatie en accountbeheer vinden, gebruikersvoorkeuren, database en opslag volharding, cloudfuncties, localisatie, afbeeldingsmanipulatie en meer. 

De combinatie van ILLA Cloud en Appwrite, dat wil zeggen de combinatie van server-side low-code en frontend low-code zullen echt de ontwikkeling van low-code full-stack realiseren.

[Klik hier](https://appwrite.io/) om meer te leren over Appwrite.

### Stel Appwrite resource in op ILLA Cloud

#### Twee manieren om bronnen aan te maken

De eerste manier is om het tabblad `Resources` in te voeren > klik op `Create New` om Appwrite te selecteren. 

<img src='https://cdn.illacloud.com/official-website/img/docs/assemble/appwrite1.png' alt="" />

Een andere manier is om de `app editor` pagina in te voeren > klik op `+ Nieuw` om Appwrite te selecteren > klik op `+ Nieuwe Bronnen` om de nieuwe configuratie toe te voegen. 

<img src='https://cdn.illacloud.com/official-website/img/docs/assemble/appwrite2.png' alt="" />

#### Appwrite instellen door de volgende velden in te vullen

Voordat je een Appwrite bron verbindt, zorg ervoor dat je ingezette Appwrite op je lokale machine of een cloud provider hebt gebruikt. [Klik hier](https://appwrite.io/docs/installation) voor meer informatie over de installatie van Appwrite. 

| Eigenschap  | is_verplicht | Beschrijving                                                                                                              |
| ----------- | ------------ | ------------------------------------------------------------------------------------------------------------------------- |
| naam        | verplicht    | De naam voor resource bij het maken van acties in de ILLA. Je kunt een naam aanpassen voor een eenvoudige identificatie.  |
| Hostnaam    | verplicht    | HOSTNAAM OF IP.                                                                                                           |
| Database ID | verplicht    | Database ID                                                                                                               |
| Project ID  | verplicht    | Project ID                                                                                                                |
| API sleutel | verplicht    | TOKEN                                                                                                                     |

<img src='https://cdn.illacloud.com/official-website/img/docs/assemble/appwrite3.png' alt="" />

### Maak acties in de app-editor

#### Acties maken op de volgende manier

Klik op `+ Nieuw` in de app editor pagina > selecteer Appwrite > selecteer een bestaand Appwrite document of maak een nieuw document aan > klik op `Create Action` om een nieuwe actie aan te maken. 

<img src='https://cdn.illacloud.com/official-website/img/docs/assemble/appwrite4.png' alt="" />

Na het aanmaken van een actie, selecteer een methode. Wij bieden de volgende methoden aan: Lijstdocumenten, een document aanmaken, een document bijwerken, een document opvragen en een document verwijderen. De configuratie van elke methode is als volgt. 

#### Documenten weergeven

Het volgende is een eenvoudig configuratievoorbeeld: Laat het document met $id 5 zien. 

<img src='https://cdn.illacloud.com/official-website/img/docs/assemble/appwrite5.png' alt="" />

| -------------- | ------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Verzameling ID | vereist      | Voer in of selecteer een collectie-id uit de database                                                                                                                                                                                                                           |
| Filteren       | Optioneel    | Ondersteun meerdere filters. Een filter bestaat uit een sleutel, een waarde en een voorwaarde. 1. sleutel: voer een attribuut 2 in. Selecteer een voorwaarde: selecteer een voorwaarde, met inbegrip van gelijke, notal, lesThanEqual, hoger, enz. 3. waarde: voer de waarde in |

Selecteer asc(standaard ) of desc                                                                                                                                      |

Het volgende is een voorbeeld van hoe de gegevens die in de component zijn ingevoerd in de actie moeten worden gebruikt:

1.  Voeg een invoercomponent toe aan het frame zodat gebruikers het document-id kunnen invoeren dat ze willen vinden.

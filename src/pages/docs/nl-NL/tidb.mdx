## <h2 hidden>Wat is TiDB?</h2>

TiDB is een database integratie waarmee u verbinding en interactie kunt maken met een TiDB database. Het is een verdeeld, horizontaal schaalbaar relationeel database management systeem dat is ontworpen om high-volume, high-concurrency workload te verwerken.

Met de TiDB integratie in Illa, kunt u gegevens opvragen, bijwerken en verwijderen uit een TiDB database met behulp van SQL commando's. U kunt ook aangepaste SQL-query's maken en uitvoeren binnen Illa. Dit maakt het eenvoudig om op data gedreven applicaties en dashboards te bouwen die real-time gegevens weergeven uit een TiDB database.

### TiDB aanmaken

Er zijn twee manieren om een bron in Illa te maken nadat je bent ingelogd op je Illa account.

-   Maak in Bronnen

Log in op je Illa account, selecteer `**Resources**` bovenaan de pagina, en klik op de `**Maak Nieuw**` knop.

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

Selecteer **`TiDB`** uit de database lijst.

![action_list](https://cdn.illacloud.com/official-website/img/docs/action_list.png)

Maak verbinding met de database met de vereiste parameters beschreven in `Connectie Instellingen` hieronder.

Klik op `**Test Connection**` om te zien of we met succes verbinding kunnen maken met de database. Zo ja, klik **`Resources`**, anders controleer je de hostnaam, port, gebruikersnaam en wachtwoord correct.

Na het aanmaken van een bron zal de klaar TiDB worden weergegeven zoals getoond.

![ti_ex_ready](https://cdn.illacloud.com/official-website/img/docs/ti_ex_ready.png)

-   Maak in Bouwer

Log in op je Illa account, maak een project aan in Illa Builder in de `**App**` pagina en navigeer naar de **`Action List`** onderaan de pagina. Klik `**nieuw**` en selecteer dan **`TiDB`** uit de database lijst. Maak vervolgens verbinding met de database met de vereiste parameters beschreven in `Connectie Instellingen` hieronder.

Klik op `**Test Connection**` om te zien of we met succes verbinding kunnen maken met de database. Zo ja, klik **`Resources`**, anders controleer je de hostnaam, port, gebruikersnaam en wachtwoord correct.

### Acties aanmaken

We hebben een TiDB bron gemaakt, we kunnen de actie toevoegen door TiDB uit de actie lijst te selecteren en te kiezen voor de **`Create action`** knop.

![ti_resource_list](https://cdn.illacloud.com/official-website/img/docs/ti_resource_list.png)

Nu hebben we de TiDB server toegevoegd als actie aan onze bouwpagina.

![ti](https://cdn.illacloud.com/official-website/img/docs/ti.png)

### TiDB configureren

| Eigenschappen  | Beschrijving                                                                                      |
| -------------- | ------------------------------------------------------------------------------------------------- |
| SQL query      | sql commando's om op te halen (select), invoegen, bijwerken, gegevens uit de database verwijderen |
| Transformator  | data omzetten in de stijl die je leuk vindt met Javascript                                        |

Voorbeeld gebruik:

-   Selecteren

```sql
SELECTEER * VAN gebruikers BY achternaam;
```

-   Insert

```sql
INSERT INTO users

	(firstname, lastname, email)

VALUES

	(

		{{ NewUser.data.firstNameInput }},

		{{ NewUser.data.lastNameInput }},

		{{ NewUser.data.emailInput }}

	);
```

-   Vernieuwen

```sql
UPDATE users
  SET firstname = '{{form1.updatedProfile.first}}'
  WHERE id = {{ form1.updatedProfile.uid}};
```

-   Verwijderen

```sql
DELETE FROM users WHERE id = {{ form1.updatedProfile.uid }};
```

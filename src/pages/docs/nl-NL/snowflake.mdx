## <h2 hidden>Wat is Snowflake?</h2>

In Illa kunt u de Snowflake query editor gebruiken om SQL queries uit te voeren tegen uw Snowflake data warehouse. Je kunt verbinding maken met Snowflake door je Snowflake account inloggegevens in te voeren, inclusief de accountnaam, gebruikersnaam, wachtwoord en magazijn. Zodra je verbinding hebt gemaakt met Snowflake, kun je de queryeditor gebruiken om SQL queries te schrijven en uit te voeren om je gegevens op te halen, te manipuleren en te analyseren.

### Sneeuwvlok maken

Er zijn twee manieren om een bron in Illa te maken nadat je bent ingelogd op je Illa account.

-   Maak in Bronnen

Log in op je Illa account, selecteer `**Resources**` bovenaan de pagina, en klik op de `**Maak Nieuw**` knop.

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

Selecteer **`Snowflake`** uit de database lijst.

![action_list](https://cdn.illacloud.com/official-website/img/docs/action_list.png)

Maak verbinding met de database met de vereiste parameters beschreven in `Connectie Instellingen` hieronder.

Klik op `**Test Connection**` om te zien of we met succes verbinding kunnen maken met de database. Zo ja, klik **`Resources`**, anders controleer je de hostnaam, port, gebruikersnaam en wachtwoord correct.

Na het maken van een bron, zal de klaarstaande Sneeuwvlok worden weergegeven zoals getoond.

![snow_ex_ready](https://cdn.illacloud.com/official-website/img/docs/snow_ex_Ready.png)

-   Maak in Bouwer

Log in op je Illa account, maak een project aan in Illa Builder in de `**App**` pagina en navigeer naar de **`Action List`** onderaan de pagina. Klik op `**nieuw**`, en selecteer dan **`Snowflake`** uit de database lijst. Maak vervolgens verbinding met de database met de vereiste parameters beschreven in `Connectie Instellingen` hieronder.

Klik op `**Test Connection**` om te zien of we met succes verbinding kunnen maken met de database. Zo ja, klik **`Resources`**, anders controleer je de hostnaam, port, gebruikersnaam en wachtwoord correct.

### Acties aanmaken

We hebben een Snowflake bron aangemaakt. We kunnen de actie toevoegen door Snowflake te selecteren uit de actielijst en de **`Create action`** knop te kiezen.

![snow_resource_list](https://cdn.illacloud.com/official-website/img/docs/snow_resource_list.png)

Nu hebben we de Snowflake server toegevoegd als actie aan onze bouwpagina.

![snow](https://cdn.illacloud.com/official-website/img/docs/snow.png)

### Sneeuwvlok configureren

| Eigenschappen  | Beschrijving                                                                                      |
| -------------- | ------------------------------------------------------------------------------------------------- |
| SQL query      | sql commando's om op te halen (select), invoegen, bijwerken, gegevens uit de database verwijderen |
| Transformator  | data omzetten naar de stijl die je leuk vindt met behulp van JavaScript                           |

Voorbeeld gebruik:

-   Selecteren

```sql
SELECTEER * VAN gebruikers BY achternaam;
```

-   Insert

```sql
INSERT INTO users

	(firstname, lastname, email)

VALUES

	(

		{{ NewUser.data.firstNameInput }},

		{{ NewUser.data.lastNameInput }},

		{{ NewUser.data.emailInput }}

	);
```

-   Vernieuwen

```sql
UPDATE users
  SET firstname = '{{form1.updatedProfile.first}}'
  WHERE id = {{ form1.updatedProfile.uid}};
```

-   Verwijderen

```sql
DELETE FROM users WHERE id = {{ form1.updatedProfile.uid }};
```

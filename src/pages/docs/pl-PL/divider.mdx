## <h2 hidden>Co to jest Divider?</h2>

W chmurze ILLA komponent rozdzielacza jest elementem wizualnym, który pomaga rozdzielić lub dzielić części treści w aplikacji internetowej. Jest to prosta linia pozioma lub pionowa, która może być umieszczona między różnymi sekcjami lub komponentami w celu poprawy organizacji wizualnej i jasności interfejsu użytkownika.

### Właściwości

| Właściwości     | Opis                                                                              |
| --------------- | --------------------------------------------------------------------------------- |
| Tekst           | dodaj dodatkowe informacje do komponentu rozdzielającego                          |
| Text align      | określa poziome wyrównanie tekstu w obrębie elementu rozdzielającego              |
| Hidden          | ukryj komponent separatora z interfejsu użytkownika                               |
| Rozmiar tekstu  | Dostosuj rozmiar czcionki w komponencie separatora                                |

### Dane

Komponent posiada powszechnie używane dane, które mogą być wywoływane przez **`{{componentName.propertyName}}`** w aplikacji.

| Nazwa własności      | Opis                                                                       |
| -------------------- | -------------------------------------------------------------------------- |
| schemat kolorów      | definiuje kolor tekstu w komponencie                                       |
| nazwa wyświetlacza   | reprezentuje nazwę lub etykietę przypisaną do komponentu                   |
| fs                   | system plików (fs) powiązany z separatorem                                 |
| hidden               | wartość logiczna, która określa, czy komponent jest widoczny czy ukryty    |
| tekst                | wartość tekstowa                                                           |

Przykład: {{divider1.text}}

### Przypadek użycia

Możesz umieścić go gdziekolwiek chcesz podzielić stronę na dwie sekcje. Tutaj pokażemy rozdzielacz z trzema różnymi dostosowaniami tekstu.

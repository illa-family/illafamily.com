## <h2 hidden>Uwierzytelnianie API</h2>

ILLA może obsługiwać prawie każdy rodzaj uwierzytelnienia. Współpracowaliśmy z różnymi firmami w celu wsparcia wszystkich standardowych rodzajów uwierzytelniania i opracowaliśmy metody wspierania niestandardowych metod uwierzytelniania, których nie popieramy.

Możesz użyć tokenów logowania statycznego lub utworzyć system uwierzytelniania, który pyta użytkownika końcowego o ich dane uwierzytelniające za każdym razem, gdy uzyskają dostęp do aplikacji.

Ta strona dostarczy Ci instrukcje jak uwierzytelnić Twój API. Jeśli masz pytania lub potrzebujesz pomocy, możesz skontaktować się z naszym zespołem pomocy technicznej za pomocą bańki Intercom na tej stronie lub ikony pomocy wewnątrz edytora.

## <h2 hidden>Co to jest Hydra?</h2>

Hydra jest silną integracją z platformą ILLA, która bezproblemowo łączy i konsoliduje różne źródła danych i aplikacje. Jako węzeł centralny Hydra umożliwia efektywny przepływ danych i komunikację między różnymi systemami, umożliwiając synchronizację danych w czasie rzeczywistym, automatyczne przepływy pracy i usprawnione procesy. Dzięki solidnym możliwościom Hydra umożliwia przedsiębiorstwom optymalizację ich działalności, zacieśnienie współpracy, i ułatwiać podejmowanie decyzji opartych na danych w ramach zintegrowanego ekosystemu chmury ILLA.

### Utwórz Hydrę

Istnieją dwa sposoby na utworzenie zasobu w ILLA po zalogowaniu się do konta.

-   Utwórz w zasobach

Zaloguj się do swojego konta Illa wybierz `**Zasoby**` na górze strony i kliknij przycisk `**Utwórz nowy**`.

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

Wybierz **`Hydra`** z listy baz danych.

![action_list](https://cdn.illacloud.com/official-website/img/docs/action_list_3.jpeg)

Połącz się z bazą danych z wymaganymi parametrami opisanymi poniżej w "Ustawienia połączenia".

Kliknij `**Test Connection**` aby sprawdzić, czy możemy połączyć się z bazą danych. Jeśli tak, kliknij **`Zapisz zasoby`**, sprawdź dwukrotnie nazwę hosta, port, nazwę użytkownika i hasło jest poprawne.

Po utworzeniu zasobu gotowy Hydra wyświetli się jak pokazano.

![hydra_ex_ready](https://cdn.illacloud.com/official-website/img/official-site/intergration/hydra_ex_ready.jpeg)

-   Utwórz w Konstruktorze

Zaloguj się na swoje konto Illa, utwórz projekt w Illa Builder na stronie `**App**`, i przejdź do **listy działań\`** na dole strony. Kliknij `**nowe**`, a następnie wybierz **`Hydra`** z listy baz danych. Następnie połącz się z bazą danych z wymaganymi parametrami opisanymi w `Connection Settings` poniżej.

Kliknij `**Test Connection**` aby sprawdzić, czy możemy połączyć się z bazą danych. Jeśli tak, kliknij **`Zapisz zasoby`**, sprawdź dwukrotnie nazwę hosta, port, nazwę użytkownika i hasło jest poprawne.

### Ustawienia połączenia

Tutaj musimy dostarczyć informacji umożliwiających połączenie z bazą danych Hydra.

![hydra_config](https://cdn.illacloud.com/official-website/img/official-site/intergration/hydra_config.jpeg)

| Właściwości                 | Wymagane     | Opis                                                                                                                                                                                                                                         |
| --------------------------- | ------------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Nazwisko                    | wymagane     | Nazwa zasobu podczas tworzenia akcji w ILLA.                                                                                                                                                                                                 |
| Hostname                    | wymagane     | Adres URL lub IP dla Twojej bazy danych                                                                                                                                                                                                      |
| Port                        | wymagane     | Numer portu hosta serwera, który powinieneś użyć do połączenia. Jeśli nie określisz portu, domyślnym portem jest '5432'.                                                                                                                     |
| Baza danych                 | wymagane     | Nazwa bazy danych                                                                                                                                                                                                                            |
| Nazwa użytkownika           | wymagane     | nazwę użytkownika, której chcesz używać podczas logowania do serwera Hydra.                                                                                                                                                                  |
| Hasło                       | wymagane     | Użyj tego hasła do uwierzytelniania.                                                                                                                                                                                                         |
| Opcje SSL                   | fakultatywne | decyduje, jak wysokie jest bezpieczne połączenie SSL TCP/IP podczas negocjacji z serwerem.                                                                                                                                                   |
| Certyfikat CA               | wymagane     | certyfikat cyfrowy, wykorzystywany do weryfikacji tożsamości serwera i ustanowienia zabezpieczenia, zaszyfrowane połączenie między klientem a klastrem Hydra. Jeśli go nie podasz, wypełniliśmy domyślny certyfikat dla Ciebie.              |
| Klucz klienta               | fakultatywne | parametr używany do ustanowienia bezpiecznego połączenia między klientem Hydra a serwerem Hydra.                                                                                                                                             |
| Certyfikat klienta          | fakultatywne | funkcja bezpieczeństwa, która pozwala klientowi uwierzytelnić się na serwerze Hydra                                                                                                                                                          |

### Utwórz akcje

Stworzyliśmy zasób Hydra, możemy dodać akcję, wybierając Hyd z listy akcji i wybierając przycisk **Utwórz akcję\`**.

![hydra_resource_list](https://cdn.illacloud.com/official-website/img/official-site/intergration/hydra_resource_list.jpeg)

Teraz dodaliśmy serwer Hydra jako działanie do naszej strony budynku.

![hydra](https://cdn.illacloud.com/official-website/img/official-site/intergration/hydra.jpeg)

### Configure Hydra

| Właściwości     | Opis                                                                                        |
| --------------- | ------------------------------------------------------------------------------------------- |
| Zapytanie SQL   | polecenia sql do pobrania (select), wstawiania, aktualizacji, usuwania danych z bazy danych |
| Transformator   | przekształcanie danych w styl jaki lubisz za pomocą Javascript                              |

Przykładowe użycie:

-   Wybierz

```sql
WYBIERZ * od użytkowników ORDER według nazwiska;
```

-   Insert

```sql
INSERT INTO users

	(firstname, lastname, email)

VALUES

	(

		{{ NewUser.data.firstNameInput }},

		{{ NewUser.data.lastNameInput }},

		{{ NewUser.data.emailInput }}

	);
```

-   Aktualizuj

```sql
UPDATE users
  SET firstname = '{{form1.updatedProfile.first}}'
  WHERE id = {{ form1.updatedProfile.uid}};
```

-   Usuń

```sql
DELETE FROM users WHERE id = {{ form1.updatedProfile.uid }};
```

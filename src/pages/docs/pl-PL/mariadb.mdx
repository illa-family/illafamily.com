## <h2 hidden>Co to jest MariaDB?</h2>

MariaDB w Illa to integracja z bazą danych, która pozwala na połączenie i interakcję z bazą danych MariaDB. MariaDB jest popularnym otwartym systemem zarządzania bazami danych opartymi na MySQL.

Dzięki integracji z MariaDB możesz zapytać, wstawiać, aktualizować i usuwać dane z bazy danych MariaDB za pomocą poleceń SQL. Możesz również tworzyć niestandardowe zapytania SQL i uruchamiać je w Illa. Ułatwia to tworzenie aplikacji opartych na danych i pulpitów nawigacyjnych, które wyświetlają dane w czasie rzeczywistym z bazy danych MariaDB.

### Utwórz MariaDB

Istnieją dwa sposoby na utworzenie zasobu w Illa po zalogowaniu się na swoje konto Illa. 

-   Utwórz w zasobach

Zaloguj się do swojego konta Illa wybierz `**Zasoby**` na górze strony i kliknij przycisk `**Utwórz nowy**`. 

<img src='https://cdn.illacloud.com/official-website/img/docs/external_resource.png' alt='' />

Wybierz **`MariaDB`** z listy baz danych.

<img src='https://cdn.illacloud.com/official-website/img/docs/action_list.png' alt='' />

Połącz się z bazą danych z wymaganymi parametrami opisanymi poniżej w "Ustawienia połączenia". 

Kliknij `**Test Connection**` aby sprawdzić, czy możemy połączyć się z bazą danych. Jeśli tak, kliknij **`Zapisz zasoby`**, sprawdź dwukrotnie nazwę hosta, port, nazwę użytkownika i hasło jest poprawne.

Po utworzeniu zasobu, gotowe MariaDB wyświetli się jak pokazano.

<img src='https://cdn.illacloud.com/official-website/img/docs/maria_ex_ready.png' alt='' />

-   Utwórz w Konstruktorze

Zaloguj się na swoje konto Illa, utwórz projekt w Illa Builder na stronie `**App**`, i przejdź do **listy działań\`** na dole strony. Kliknij `**nowe**`, a następnie wybierz **`MariaDB`** z listy baz danych. Następnie połącz się z bazą danych z wymaganymi parametrami opisanymi w `Connection Settings` poniżej.

Kliknij `**Test Connection**` aby sprawdzić, czy możemy połączyć się z bazą danych. Jeśli tak, kliknij **`Zapisz zasoby`**, sprawdź dwukrotnie nazwę hosta, port, nazwę użytkownika i hasło jest poprawne.

### Ustawienia połączenia

Tutaj musimy dostarczyć informacji na temat połączenia z bazą danych MariaDB.

<img src='https://cdn.illacloud.com/official-website/img/docs/maria_config.png' alt='' />

| Właściwości       | Wymagane     | Opis                                                                                                                     |
| ----------------- | ------------ | ------------------------------------------------------------------------------------------------------------------------ |
| Nazwisko          | wymagane     | Nazwa zasobu podczas tworzenia akcji w ILLA.                                                                             |
| Hostname          | wymagane     | Adres URL lub IP dla Twojej bazy danych                                                                                  |
| Port              | wymagane     | Numer portu hosta serwera, który powinieneś użyć do połączenia. Jeśli nie określisz portu, domyślnym portem jest '3306'. |
| Baza danych       | wymagane     | Nazwa bazy danych                                                                                                        |
| Nazwa użytkownika | wymagane     | nazwę użytkownika, której chcesz używać podczas logowania do serwera MariaDB.                                            |
| Hasło             | wymagane     | Użyj tego hasła do uwierzytelniania.                                                                                     |
| Opcje SSL         | fakultatywne | decyduje, jak wysokie jest bezpieczne połączenie SSL TCP/IP podczas negocjacji z serwerem.                               |

### Utwórz akcje

Stworzyliśmy zasób MariaDB, możemy dodać akcję wybierając MariaDB z listy akcji i wybierając przycisk **`Utwórz akcję`**.

<img src='https://cdn.illacloud.com/official-website/img/docs/maria_resource_list.png' alt='' />

 Teraz dodaliśmy serwer MariaDB jako działanie do naszej strony budynku.

<img src='https://cdn.illacloud.com/official-website/img/docs/maria.png' alt='' />

### Configure MariaDB

| Właściwości   | Opis                                                                                        |
| ------------- | ------------------------------------------------------------------------------------------- |
| Zapytanie SQL | polecenia sql do pobrania (select), wstawiania, aktualizacji, usuwania danych z bazy danych |
| Transformator | przekształcanie danych w styl jaki lubisz za pomocą Javascript                              |

Przykładowe użycie:

-   Wybierz

```sql
WYBIERZ * od użytkowników ORDER według nazwiska;
```

-   Insert

```sql
WEJŚCIE DO użytkowników

	(imię, nazwisko, e-mail)

WARTOŚCI

	(

		{{ NewUser.data.firstNameInput }},

		{{ NewUser.data.lastNameInput }},

		{{ NewUser.data.emailInput }}

);
```

-   Aktualizuj

```sql
AKTUALIZUJ użytkowników
  SET imię = '{{form1.updatedProfile.first}}'
  WHERE id = {{ form1.updatedProfile.uid}};
```

-   Usuń

```sql
USUŃ użytkowników gdzie id = {{ form1.updatedProfile.uid }};
```

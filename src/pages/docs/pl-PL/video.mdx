## <h2 hidden>Czym jest wideo?</h2>

Komponent wideo w chmurze ILLA jest potężnym narzędziem, które umożliwia użytkownikom włączenie treści multimedialnych do ich aplikacji lub projektów. Dzięki tej funkcji użytkownicy mogą bezproblemowo zintegrować filmy ze swoimi interfejsami użytkownika i tworzyć inspirujące i interaktywne doświadczenia dla swoich użytkowników końcowych.

Komponent wideo obsługuje różne formaty wideo, w tym popularne formaty, takie jak MP4, AVI i MOV. Użytkownicy mogą przesyłać filmy bezpośrednio ze swoich urządzeń lub udostępniać adresy URL do zewnętrznych źródeł wideo. Ta elastyczność umożliwia użytkownikom wykorzystanie istniejących treści wideo lub tworzenie nowych filmów specjalnie dla ich aplikacji.

### Właściwości

| Właściwości                   | Opis                                                                                                                                                     |
| ----------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Źródło wideo                  | określ lokalizację pliku wideo lub podaj adres URL do zewnętrznego źródła wideo                                                                          |
| Automatyczne odtwarzanie      | umożliwia automatyczne odtwarzanie wideo po załadowaniu strony                                                                                           |
| Pętla                         | pozwala na ciągłe odtwarzanie wideo w pętli                                                                                                              |
| Pokaż ustawienia              | czy ustawienia odtwarzacza wideo są widoczne czy ukryte                                                                                                  |
| Wycisz                        | pozwala na odtwarzanie wideo bez żadnego dźwięku                                                                                                         |
| Objętość                      | kontroluje poziom dźwięku wideo                                                                                                                          |
| Obsługa zdarzeń               | Wyzwalacz zapytań, sterowanie komponentami lub wywoływanie innych interfejsów API w odpowiedzi na zdarzenia komponentów.                                 |
| Wskazówka                     | Użytkownicy mogą tutaj wprowadzić podpowiedź składową. Podpowiedź zostanie wyświetlona, gdy jest ostrością. Format Markdown jest obsługiwany.            |
| Hidden                        | Dynamicznie kontroluj czy komponent jest ukryty. Możesz zmienić ukryty status za pomocą dynamicznej wartości logicznej.                                  |

### Metoda

Możesz użyć innych komponentów do kontroli komponentu. Popieramy następujące osiem metod:

-   **graj**

rozpocznij odtwarzanie wideo. Po wywołaniu inicjuje odtwarzanie wideo z bieżącej pozycji.

-   **wstrzymaj**

wstrzymać odtwarzanie wideo. Po wywołaniu urządzenie tymczasowo zatrzymuje wideo w swojej obecnej pozycji.

-   **showControls**

przełącz widoczność sterowania odtwarzaczem wideo. Powołując się na tę metodę, urządzenia sterujące mogą być wyświetlane lub ukryte, umożliwiając użytkownikom interakcję z odtwarzaniem wideo.

-   **setVideoUrl**

zmienia dynamicznie źródło lub adres URL wideo.

| Właściwości | Opis                     |
| ----------- | ------------------------ |
| URL wideo   | źródłowy adres URL wideo |

-   **setVolume**

dostosowanie głośności dźwięku wideo. Pobiera wartość liczbową jako objętość parametru

| Właściwości  | Opis                                                                                                                 |
| ------------ | -------------------------------------------------------------------------------------------------------------------- |
| Objętość     | wartość liczbowa w zakresie od 0 do 1, gdzie 0 oznacza wyciszenie lub brak dźwięku, a 1 oznacza maksymalną objętość. |

-   **setSpeed**

zmiana prędkości odtwarzania wideo. Akceptuje wartość numeryczną jako prędkość parametru

| Właściwości   | Opis                                                                                                                                                              |
| ------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Prędkość      | numeric value. 1 reprezentuje normalną prędkość odtwarzania. Wartości mniejsze niż 1 spowalniają film, podczas gdy wartości większe niż 1 przyspieszają ten film. |

-   **setLoop**

włącz lub wyłącz funkcję pętli wideo. Podając wartość logiczną jako parametr, wideo można ustawić w pętli ciągłej lub odtwarzać tylko raz.

-   **szukaj**

poszukiwanie określonego czasu lub miejsca w filmie. Przyjmuje wartości liczbowe jako czas parametru

| Właściwości | Opis                                                                                                                    |
| ----------- | ----------------------------------------------------------------------------------------------------------------------- |
| Czas(y)     | wartość liczbowa przedstawiająca pożądany czas w sekundach jako parametr i odpowiednio dostosowuje pozycję odtwarzania. |

### Obsługa zdarzeń

| Wydarzenie  | Opis                                                                                                                                                                                                                           |
| ----------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Odtwórz     | wykonaj określone akcje lub wykonaj kod, gdy rozpocznie się odtwarzanie wideo                                                                                                                                                  |
| Wstrzymaj   | obsługuje zdarzenie pauzy i wykonuje odpowiednio akcje                                                                                                                                                                         |
| Załadowano  | wykonywanie zadań, które muszą być wykonane po pełnym załadowaniu wideo, np. aktualizacja interfejsu użytkownika, aby wyświetlić czas trwania wideo lub inicjowanie dodatkowych funkcji związanych z odtwarzaniem wideo        |
| Zakończono  | obsługuje koniec wideo i wykonuje takie czynności, jak wyświetlanie przycisku powtórki, wyświetlanie powiązanej zawartości lub wykonywanie dowolnej innej pożądanej logiki po zakończeniu odtwarzania wideo                    |

### Dane

Komponent posiada powszechnie używane dane, które mogą być wywoływane przez **`{{componentName.propertyName}}`** w aplikacji.

| Nazwa własności               | Opis                                                                                                                                                    |
| ----------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Automatyczne odtwarzanie      | Wartość logiczna, która określa, czy wideo powinno automatycznie rozpocząć odtwarzanie po załadowaniu.                                                  |
| kontrola                      | Wartość logiczna, która określa, czy odtwarzacz wideo kontroluje odtwarzanie, wstrzymanie i głośność, są wyświetlane użytkownikowi.                     |
| nazwa wyświetlacza            | Wartość ciągu reprezentująca nazwę lub tytuł wideo.                                                                                                     |
| wydarzenia                    | Tablica lub obiekt, który definiuje obsługę zdarzeń związanych z komponentem wideo, takie jak odtwarzanie, wstrzymanie, załadowanie, zakończenie itp.   |
| hidden                        | Wartość logiczna określająca czy komponent wideo jest ukryty z interfejsu użytkownika.                                                                  |
| pętla                         | Wartość logiczna, która określa, czy wideo powinno automatycznie uruchomić się ponownie od początku po dotarciu do końca.                               |
| wyciszony                     | Wartość logiczna określająca czy film powinien być wyciszony, tj. odtwarzany bez dźwięku.                                                               |
| odtwarzanie                   | Wartość logiczna wskazująca, czy wideo jest aktualnie odtwarzane.                                                                                       |
| tekst                         | wartość ciągu, która reprezentuje tekst podpowiedzi lub wskazówek do wyświetlenia, gdy użytkownik uskakuje nad komponentem wideo.                       |
| url                           | Wartość ciągu określająca adres URL lub źródło pliku wideo do odtworzenia.                                                                              |
| głośność                      | Wartość liczbowa pomiędzy 0 a 1 reprezentuje poziom głośności filmu, gdzie 0 jest wyciszone, a 1 jest maksymalną objętością.                            |

Przykład: {{video1.volume}}

### Przypadek użycia

Następnie pokażemy jak sterować odtwarzanym wideo za pomocą przycisku.

#### Krok 1 Dodaj komponenty

Dodaj komponent **`video`** i komponent **`przycisk`** do płótna.

Ustawiamy źródło wideo komponentu wideo na link z dowolnego filmu, który chcesz odtworzyć. Tutaj użyjemy domyślnego filmu jako przykładu i oznaczono jako przycisk "Odtwórz", jak pokazano poniżej

![video_layout](https://cdn.illacloud.com/official-website/img/official-site/components/video_layout.jpg)

#### Krok 2 Konfiguruj komponent

Dla komponentu **`przycisk`** możemy go skonfigurować, aby ustawić komponent **`film`**, jeśli film nie jest odtwarzany.

1.  Kliknij komponent `**button**` aby otworzyć jego stronę kontrolną. W **`Obsługa zdarzeń`**, kliknij `**+ Nowa**`.
2.  W obsłudze zdarzeń edytuj wybierz **`składnik kontroli`** w akcji, i wybierz komponent **`video1`**, który chcesz zmienić jako cel wydarzenia.
3.  Wybierz akcję **`play`**
4.  W **Pole `Uruchom tylko, gdy`** umieść `{{!video1.playing}}`.

![video_config](https://cdn.illacloud.com/official-website/img/official-site/components/video_config.jpg)

Podobnie, aby zatrzymać odtwarzanie wideo, tworzymy nowy program obsługi zdarzeń i wykonujemy pierwsze dwa kroki tak samo jak powyżej.

1.  Wybierz akcję **pauza\`**
2.  W **Pole `Uruchom tylko, gdy`** wpisz `{{video1.playing}}`.

![video_config_1](https://cdn.illacloud.com/official-website/img/official-site/components/video_config_1.jpg)

#### Etap 3 Badanie

Teraz, gdy klikniesz przycisk "Odtwórz", film powinien odtwarzać jeśli nie odtwarzaj i wstrzymywać podczas odtwarzania.

![video_test](https://cdn.illacloud.com/official-website/img/official-site/components/video_test.gif)

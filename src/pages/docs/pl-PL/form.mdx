## <h2 hidden>Co to jest formularz?</h2>

Komponent formularza jest elementem wizualnym, który reprezentuje postęp lub ukończenie zadania lub procesu. Wyświetla poziomy pasek, który wypełnia lub kurczy się w oparciu o określoną wartość postępu.

### Właściwości

| Właściwości                                    | Opis                                                                                                                                                                                         |
| ---------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Obsługa zdarzeń                                | wykrywanie i reagowanie na konkretne akcje lub zdarzenia użytkownika, takie jak kliknięcia, klawisze lub postulaty formularza.                                                               |
| Wyłączone                                      | nieinteraktywne i nie mogą być modyfikowane ani uruchamiane przez użytkownika.                                                                                                               |
| Wyłącz wysyłanie                               | wyłączanie funkcji przesyłania formularza.                                                                                                                                                   |
| Zatwierdź dane wejściowe przy przedłożeniu     | formularz danych wejściowych powinien być potwierdzony pod względem poprawności lub kompletności w momencie składania formularza.                                                            |
| Resetuj po pomyślnym wysłaniu                  | automatycznie resetuje swoje wartości i ustawia je na wartości początkowe lub domyślne po pomyślnym przedłożeniu                                                                             |
| Wskazówka                                      | mały wyskakujący komunikat lub komunikat informacyjny, który pojawia się, gdy użytkownik ukrywa się nad nim lub wchodzi w interakcję z określonym elementem, takim jak komponent formularza. |
| Pokaż nagłówek                                 | czy komponent formularza powinien wyświetlać nagłówek lub sekcję tytułu                                                                                                                      |
| Pokaż stopkę                                   | czy komponent formularza powinien wyświetlać sekcję stopki                                                                                                                                   |
| Hidden                                         | ukrywa komponent formularza z widoku użytkownika                                                                                                                                             |
| Kolor obramowania                              | kolor obramowania otaczającego komponent formularza                                                                                                                                          |
| Promień obramowania                            | krzywizna narożników obramowania elementu formularza.                                                                                                                                        |
| Border Width                                   | grubość lub szerokość obramowania otaczającego element formularza                                                                                                                            |
| Kontekst                                       | definiuje kolor lub obraz, który wypełnia obszar zawartości komponentu formularza                                                                                                            |
| Cień                                           | efekt wizualny dodawania podcienia, przesunięcie "cień" do komponentu formularza, sprawiając, że pojawiają się one lekko wypukłe lub unoszące się nad tłem                                   |

### Metoda

Możesz użyć innych komponentów do kontroli komponentu. Popieramy następujące dwie metody:

-   **zgłoś**

Przesyłanie informacji wypełnionych w polu formularza

-   **reset**

zresetuj wszystkie pola wejścia w komponencie formularza

-   **setValue**

Aby ustawić wartość pola tekstowego, na przykład {{"value1"}}

| Właściwości  | Opis        |
| ------------ | ----------- |
| Wartość      | Input value |

-   validate

Sprawdzenie, czy informacje wejściowe są zgodne z prawem

### Dane

Komponent posiada powszechnie używane dane, które mogą być wywoływane przez **`{{componentName.propertyName}}`** w aplikacji.

| Nazwa własności               | Opis                                                                                                                                                             |
| ----------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Kolor tła                     | określa kolor tła komponentu formularza                                                                                                                          |
| Kolor ramki                   | definiuje kolor obramowania elementu formularza                                                                                                                  |
| borderWidth                   | określa grubość lub szerokość obramowania otaczającego element formularza.                                                                                       |
| wyłączone                     | czy komponent formularza jest wyłączony czy nie                                                                                                                  |
| nazwa wyświetlacza            | reprezentuje nazwę lub etykietę przypisaną do komponentu formularza                                                                                              |
| wysokość stopy                | określa wysokość lub rozmiar sekcji stopki elementu formularza                                                                                                   |
| FormData                      | posiada dane lub wartości wprowadzone przez użytkownika w komponencie formularza                                                                                 |
| Wysokość nagłówka             | określa wysokość lub rozmiar sekcji nagłówka komponentu formularza                                                                                               |
| niepoprawny                   | czy dane wejściowe lub wartość komponentu formularza są uważane za nieprawidłowe, czy nie.                                                                       |
| radius                        | określa promień obramowania lub krzywizny narożników elementu formularza.                                                                                        |
| resetAfterful                 | wskazuje, że komponent formularza powinien automatycznie zresetować swoje wartości i określić ich wartości początkowe lub domyślne po pomyślnym przedłożeniu     |
| cień                          | kontroluje obecność i wygląd efektu cienia dla elementu formularza;                                                                                              |
| stopka prysznic               | określa, czy sekcja stopki komponentu formularza powinna być wyświetlana, czy nie                                                                                |
| Nagłówek prysznica            | widoczność sekcji nagłówka komponentu formularza                                                                                                                 |
| tekst                         | zawiera tekst lub treść podpowiedzi skojarzonej z komponentem formularza                                                                                         |
| validateInputOnSubmit         | określa, że wartości wejściowe składnika formularza powinny być walidowane pod względem poprawności lub kompletności przy składaniu formularza.                  |

Przykład: {{form1.disabled}}

### Przypadek użycia

Pokażemy jak stworzyć formę o nazwie i wieku jako wkład, jak również postęp na pasku aby pokazać postęp ukończenia formularza.

#### Krok 1 Dodaj komponenty

Dodaj komponent **form\`** z domyślnym tekstem "Form" i przyciskiem "Wyślij" do płótna.

Dodaj komponent **`input`**, komponent **liczba\`** i **komponent `bar progress`** z `**Inser**` do formularza.

Oznacz komponent wejściowy jako "Nazwa", komponent wejściowy liczb, który ma być "Wiek" i komponent postępu paska będą "Postęp".

Ustaw symbol zastępczy komponentu wejściowego na "Proszę wpisać swoją nazwę tutaj" i domyślną wartość wejścia numeru na 0 jak pokazano poniżej.

![bar_layout](https://cdn.illacloud.com/official-website/img/official-site/components/bar_layout.jpg)

#### Krok 2 Konfiguruj komponent

Dla komponentów **`input`** i **number input\`** możemy je skonfigurować, aby ustawić wartość w komponencie **`bar progress`**, aby odzwierciedlić postęp w ukończeniu.

1.  Kliknij komponent **`input`** i **number input\`**, aby otworzyć jego stronę kontrolną. W **`Obsługa zdarzeń`**, kliknij `**+ Nowa**`.
2.  W obsłudze zdarzeń edytuj wybierz **`składnik kontroli`** w akcji, i wybierz komponent **Postęp w pasku\`**, który chcesz zaktualizować jako cel wydarzenia.
3.  Wybierz działanie **`setValue`** i ustaw wartość jako

```jsx
{
  {
    input1.value && numberInput1.value
      ? "100"
      : input1.value || numberInput1.value
      ? "50"
      : "0";
  }
}
```

![bar_input_num_config](https://cdn.illacloud.com/official-website/img/official-site/components/bar_input_num_config.jpg)

Uwaga: wymaga ustawienia obsługi zdarzeń w sposób opisany powyżej na komponenty wejściowe i wejściowe BOTH.

Dla komponentu **`przycisk`** możemy go skonfigurować do przesłania formularza, jeśli formularz jest wypełniony (tj. postęp paska 100%)

1.  Kliknij komponent **`przycisk`, aby otworzyć jego stronę kontrolną. W **`Obsługa zdarzeń`**, kliknij \`**+ Nowa\*\*\`.
2.  W obsłudze zdarzeń edytuj wybierz **`składnik kontroli`** w akcji, i wybierz komponent **formularz\`**, który chcesz zaktualizować jako cel wydarzenia.
3.  Wybierz metodę **`zgłoś`** i umieść `{{barProgress1.value == 100}}` w **`Tylko kiedy`**

![bar_button_submit_config](https://cdn.illacloud.com/official-website/img/official-site/components/bar_button_submit_config.jpg)

Podobnie możemy skonfigurować przycisk, aby pokazać ostrzeżenie, gdy użytkownik próbuje przesłać niewypełniony formularz.

1.  Kliknij komponent **`przycisk`, aby otworzyć jego stronę kontrolną. W **`Obsługa zdarzeń`**, kliknij \`**+ Nowa\*\*\`.
2.  W obsłudze zdarzeń edytuj wybierz **`Pokaż powiadomienie`** w akcji, **`title`** "Error!" i **`description`** "Wypełnij formularz przed wysłaniem". Wybierz `Error` dla **`type`**, wprowadź `{{2000}}` na **`czas trwania`** (2 sekundy) i umieść `{{barProgress1.value<100}}` dla **`Tylko kiedy`.**

![bar_button_show_config](https://cdn.illacloud.com/official-website/img/official-site/components/bar_button_show_config.jpg)

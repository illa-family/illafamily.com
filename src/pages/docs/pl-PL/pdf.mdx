## <h2 hidden>Co to jest PDF?</h2>

Komponent PDF jest funkcją lub elementem, który umożliwia użytkownikom przeglądanie i interakcja z dokumentami PDF w aplikacji. Zapewnia on funkcjonalność renderowania, nawigacji i manipulowania plikami PDF, umożliwiając użytkownikom łatwy dostęp do treści PDF i pracę z nimi.

### Właściwości

| Właściwości          | Opis                                                                                  |
| -------------------- | ------------------------------------------------------------------------------------- |
| Źródło PDF           | określ źródło pdf do wyświetlenia                                                     |
| Pokaż pasek narzędzi | kontroluje widoczność paska narzędzi w komponencie PDF                                |
| Wskazówka            | dodaj podpowiedź lub tekst opisowy, który pojawia się gdy użytkownik uskakuje nad pdf |
| Skaluj stronę do     | określa zachowanie skalowania stron PDF w komponencie                                 |
| Hidden               | określa widoczność składnika PDF                                                      |

### Metoda

Możesz użyć innych komponentów do kontroli komponentu. Popieramy następujące metody:

-   **setFileUrl**

dynamicznie zmienia źródło lub adres URL pliku pdf.

| Właściwości | Opis             |
| ----------- | ---------------- |
| Źródło PDF  | źródłowy url pdf |

### Dane

Komponent posiada powszechnie używane dane, które mogą być wywoływane przez **`{{componentName.propertyName}}`** w aplikacji.

| Nazwa własności      | Opis                                                                                                                            |
| -------------------- | ------------------------------------------------------------------------------------------------------------------------------- |
| nazwa wyświetlacza   | Wartość ciągu reprezentująca nazwę lub tytuł pliku pdf.                                                                         |
| wysokość             | wymiar pionowy lub rozmiar komponentu PDF.                                                                                      |
| hidden               | Wartość logiczna określająca czy komponent pdf jest ukryty z interfejsu użytkownika.                                            |
| tryb skalowania      | określa, w jaki sposób dokument PDF jest skalowany lub pasuje do dostępnego obszaru wyświetlania komponentu                     |
| showToolBar          | czy pasek narzędzi lub panel sterowania są wyświetlane razem z komponentem PDF                                                  |
| tekst                | wartość ciągu, która reprezentuje tekst podpowiedzi lub wskazówek do wyświetlenia, gdy użytkownik uskakuje nad komponentem pdf. |
| url                  | źródło pliku PDF, który ma być wyświetlany w komponencie                                                                        |
| width                | przekrojowy wymiar lub rozmiar komponentu PDF,                                                                                  |

Przykład: {{pdf1.displayName}}

### Przypadek użycia

Następnie pokażemy jak sterować wyświetlaczem pdf za pomocą przycisku.

#### Krok 1 Dodaj komponenty

Dodaj komponent **PDF\`** i komponent **`przycisk`** do płótna.

![pdf_layout](https://cdn.illacloud.com/official-website/img/docs/pdf_layout.jpg)

#### Krok 2 Konfiguruj komponent

Dla komponentu **`przycisk`** możemy go skonfigurować, aby ustawić źródło obrazu komponentu **`PDF`** na adres pdf.

1.  Kliknij komponent `**button**` aby otworzyć jego stronę kontrolną. W **`Obsługa zdarzeń`**, kliknij `**+ Nowa**`.
2.  W obsłudze zdarzeń edytuj wybierz **`składnik kontroli`** w akcji, i wybierz komponent **PDF\`**, który chcesz zmienić jako cel wydarzenia.
3.  Wybierz działanie **`setFileUrl`**
4.  Umieść nowy adres obrazu w polu **`Wartość`**

![pdf_config](https://cdn.illacloud.com/official-website/img/docs/pdf_config.jpg)

#### Etap 3 Badanie

Teraz, gdy klikniesz przycisk "Odtwórz", obraz powinien się przełączać

![pdf_test](https://cdn.illacloud.com/official-website/img/docs/pdf_test.jpg)

## <h2 hidden>Co to jest tabela?</h2>

Komponent tabeli w chmurze ILLA jest potężną funkcją, która umożliwia użytkownikom wyświetlanie, organizowanie i manipulowanie danymi tabularnymi w aplikacji. Zapewnia ustrukturyzowany układ z wierszami i kolumnami, umożliwiający użytkownikom oglądanie i interakcję z danymi w wygodny i uporządkowany sposób.

### Właściwości

| Właściwości                                                                                                                      | Opis                                                                                                                                    |
| -------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------- |
| Źródło danych                                                                                                                    | źródło danych, które wypełniają tabelę                                                                                                  |
| Pusty stan                                                                                                                       | zawartość lub wiadomość wyświetlana, gdy tabela nie ma danych do wyświetlenia                                                           |
| Ładowanie                                                                                                                        | czy komponent tabeli jest obecnie w stanie załadowania                                                                                  |
| Kwota, którą należy zgłosić w kolumnie 060 tego wiersza: pierwotne odliczenie, art. 36 ust. 1 lit. b) CRR.                       | struktura i konfiguracja kolumn w tabeli. (Więcej szczegółów na temat kolumn poniżej)                                                   |
| Kolumna                                                                                                                          | liczba kolumn                                                                                                                           |
| Wybór wielu wierszy                                                                                                              | Użyj {{table.selectedRow[n].columnName}} aby uzyskać dostęp do danych.                                                                  |
| Wybór komórki                                                                                                                    | Czy pozwala użytkownikom wybrać komórkę                                                                                                 |
| Kliknij na zewnątrz, aby odznaczyć                                                                                               | Obsługiwane w trybie pojedynczego wyboru wiersza. Po włączeniu tego, kliknij obszar inny niż tabela, aby odznaczyć wiersz.              |
| Przepełnienie                                                                                                                    | w jaki sposób tabela obsługuje zawartość, która przekracza dostępną przestrzeń w pojemniku tabeli (paginacja lub przewijanie)           |
| Włącz stronicowanie po stronie serwera                                                                                           | Czy włączyć paginację serwera                                                                                                           |
| Rozmiar strony                                                                                                                   | liczba wierszy wyświetlanych na stronie, gdy włączona jest stronicowanie po stronie serwera                                             |
| Odśwież                                                                                                                          | Czy pokazać ikonę odświeżania.                                                                                                          |
| Pobierz                                                                                                                          | Pokaż przycisk pobierania na pasku narzędzi.                                                                                            |
| Pobierz dane surowe                                                                                                              | Czy pokazać ikonę pobierania surowych danych.                                                                                           |
| Filtr                                                                                                                            | Pokaż przycisk filtra na pasku narzędzi                                                                                                 |
| Obsługa zdarzeń                                                                                                                  | wykrywanie i reagowanie na konkretne akcje lub zdarzenia użytkownika, takie jak kliknięcia, klawisze lub postulaty formularza.          |
| Wyłączone                                                                                                                        | nieinteraktywne i nie mogą być modyfikowane ani uruchamiane przez użytkownika.                                                          |

#### **Przegląd właściwości kolumny**

Dla każdej kolumny możemy skonfigurować jej właściwości, aby dostosować dane wyświetlane dla każdej kolumny.

| Właściwości                                          | Opis                                                                                                                                                                                                                          |
| ---------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Tytuł kolumny                                        | określa nazwę lub etykietę kolumny, która służy jako nagłówek do identyfikacji danych, które reprezentuje.                                                                                                                    |
| Typ kolumny                                          | definiuje typ danych, które zawiera kolumna, w tym automatyczny, tekst, datę, tag, godzinę, dateTime, numer, procent, link, przycisk, grupa przycisków, wartość logiczna, obraz, iconGroup, Ring, Markdown, waluta            |
| Wartość odwzorowana                                  | Użyj {{ currentRow }} aby uzyskać dostęp do danych bieżącego wiersza. Na przykład, aby obliczyć "koszt za kliknięcie" na podstawie kolumn "koszt" i "kliknij Czas":                                                           |
| {{ currentRow.cost / currentRow.clickTimes }}        |                                                                                                                                                                                                                               |
| Włącz sortowanie                                     | czy kolumna pozwala użytkownikom sortować dane tabeli w oparciu o wartości w tej konkretnej kolumnie                                                                                                                          |
| Kontekst                                             | Ustaw tło kolumny. Użyj {{ write conditional statement aby ustawić kolor wyświetlany przez każdą komórkę w różnych warunkach.                                                                                                 |
| Wyrównanie                                           | kontroluje poziome wyrównanie zawartości komórek kolumny                                                                                                                                                                      |

#### Typy kolumn:

-   Auto

typ dynamiczny, który automatycznie określa typ danych na podstawie wartości w kolumnie

-   Tekst

reprezentuje kolumnę, która zawiera dane tekstowe

-   Dane

reprezentuje wartości danych odpowiadające określonym datom

| Właściwości      | Opis                                                                   |
| ---------------- | ---------------------------------------------------------------------- |
| Formatowanie     | przyjmuje różne kody lub wzory formatowania, domyślnie jest RRRR-MM-DD |

-   Znacznik

specjalistyczny format wyświetlania tagów lub etykiet związanych z wpisami danych

| Właściwości    | Opis                                                                |
| -------------- | ------------------------------------------------------------------- |
| Etykieta tagu  | reprezentuje tekst lub etykietę przypisaną do tagu w kolumnie "tag" |
| Kolor tagu     | definiuje kolor przypisany do tagu w kolumnie "tag"                 |

-   Czas

dokładnie opisuje i manipuluje wartościami czasowymi w tabeli

| Właściwości      | Opis                                                                        |
| ---------------- | --------------------------------------------------------------------------- |
| Formatowanie     | wizualne przedstawienie danych o czasie w kolumnie, domyślnie jest HH:mm:ss |

-   Data i godzina

przechowuje i reprezentuje daty i godziny

| Właściwości      | Opis                                                                                                                |
| ---------------- | ------------------------------------------------------------------------------------------------------------------- |
| Formatowanie     | określa, w jaki sposób wartości daty i czasu powinny być wyświetlane w kolumnie, domyślnie jest RRRR-MM-DD HH:mm:ss |

-   Numer

obsługa danych numerycznych

| Właściwości                  | Opis                                                                                                                    |
| ---------------------------- | ----------------------------------------------------------------------------------------------------------------------- |
| Miejsca dziesiętne           | określ liczbę miejsc po przecinku dla wartości numerycznych w kolumnie                                                  |
| Pokaż separator tysięcy      | czy kolumna powinna wyświetlać separator tysięcy w celu poprawy czytelności i łatwości interpretacji dla dużych numerów |

-   Procent

obsługuje wartości procentowe w zwięzły i ciekawy sposób

| Właściwości                  | Opis                                                                                     |
| ---------------------------- | ---------------------------------------------------------------------------------------- |
| Miejsca dziesiętne           | określ liczbę miejsc po przecinku do wyświetlenia dla wartości procentowych w kolumnie   |
| Pokaż separator tysięcy      | czy separator tysięcy powinien być uwzględniony w wyświetlanych wartościach procentowych |

-   Link

dołącz klikalne linki w określonej kolumnie

-   Przycisk

dołącz interaktywne przyciski w określonej kolumnie

| Właściwości     | Opis                                                                                     |
| --------------- | ---------------------------------------------------------------------------------------- |
| Obsługa zdarzeń | definiuje akcję lub zdarzenie, które występuje po kliknięciu przycisku w kolumnie        |
| Wyłączone       | czy przycisk w kolumnie jest włączony czy wyłączony                                      |
| Kolor motywu    | wybierz kolor motywu dla przycisku                                                       |
| Wariant         | wizualny styl lub wariant przycisku (wypełnij lub obrysowa)                              |

-   grupa przycisków

umożliwia użytkownikom tworzenie interaktywnych grup przycisków w kolumnie

Dla każdego przycisku w kolumnach możemy skonfigurować jego właściwości.

| Właściwości          | Opis                                                                                                                                |
| -------------------- | ----------------------------------------------------------------------------------------------------------------------------------- |
| Wartość odwzorowana  | powiąż każdy przycisk w grupie przycisków z określoną wartością lub polem ze źródła danych                                          |
| Wariant              | wizualny styl lub wygląd przycisków w grupie przycisków                                                                             |
| Obsługa zdarzeń      | działanie lub funkcja wyzwalane po kliknięciu przycisku w grupie lub po interakcji:                                                 |
| Wyłączone            | czy przyciski w grupie przycisków są włączone czy wyłączone                                                                         |
| Kolor motywu         | dostosuj kolor lub styl przycisków w grupie, aby dostosować do ogólnego motywu lub marki ich aplikacji lub strony internetowej      |

-   Boolean

reprezentuje dane, które mają dwa możliwe stany: prawdziwe lub fałszywe

-   Obraz

wypełnij kolumnę w tabeli danymi obrazu

| Właściwości | Opis                                                    |
| ----------- | ------------------------------------------------------- |
| Typ skali   | jak obraz jest skalowany lub zmieniany w komórce tabeli |

-   Grupa Icona

specjalistyczna kolumna, która pozwala użytkownikom wyświetlać ikony w grupie lub kolekcji

Dla każdego przycisku w kolumnach możemy skonfigurować jego właściwości.

| Właściwości     | Opis                                                                                                                                   |
| --------------- | -------------------------------------------------------------------------------------------------------------------------------------- |
| Ikona           | pozwala użytkownikom wybrać lub określić ikonę wyświetlaną w kolumnie IconGroup                                                        |
| Kolor motywu    | określa schemat kolorów lub motyw zastosowany do ikony w kolumnie IconGroup                                                            |
| Obsługa zdarzeń | umożliwia użytkownikom zdefiniowanie zdarzenia lub akcji wywołanej gdy ikona w kolumnie IconGroup jest interakcyjna przez użytkowników |
| Wyłączone       | stan kolumny IconGroup określa, czy jest ona wyłączona czy włączona dla interakcji użytkownika                                         |

-   Ocena

reprezentuje oceny lub opinie przekazane przedmiotom lub podmiotom

-   Markdown

możliwość renderowania sformatowanego tekstu przy użyciu składni Markdown w komórkach kolumny

-   Waluta

obsługa i formatowanie wartości waluty.

| Właściwości                  | Opis                                                                               |
| ---------------------------- | ---------------------------------------------------------------------------------- |
| Miejsca dziesiętne           | liczba miejsc dziesiętnych wyświetlanych dla wartości waluty w kolumnie            |
| Kod waluty                   | określa kod lub skrót opisujący walutę użytą w kolumnie                            |
| Pokaż separator tysięcy      | czy separator tysięcy jest wyświetlany dla dużych wartości walutowych              |

### Metoda

Możesz użyć innych komponentów do kontroli komponentu. Popieramy następujące dwie metody:

-   **selectPage**

wybierz wszystkie wiersze na konkretnej stronie w tabeli paginowanej

| Właściwości      | Opis                                                 |
| ---------------- | ---------------------------------------------------- |
| Wartość końcowa  | numer strony lub indeks strony, która ma być wybrana |

-   **selectWiersz**

wybierz konkretny wiersz w tabeli

| Właściwości              | Opis                                                                                                                                                                                  |
| ------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Domyślnie wybrany wiersz | domyślnie wybrana wartość wiersza może być indeksem wierszy, unikalnym identyfikatorem lub jakąkolwiek inną właściwość, która jednoznacznie identyfikuje wiersz, który ma być wybrany |

-   **clearSelection**

wyczyść bieżący wybór w tabeli

-   **setFilters**

zastosuj filtry do danych tabeli

| Właściwości      | Opis                                   |
| ---------------- | -------------------------------------- |
| Filtry           | warunki lub kryteria filtrowania       |
| Tryb filtrowania | określa sposób stosowania filtrów      |

-   **ClearFilters**

czyści wszystkie zastosowane filtry w tabeli

-   **setSort**

ustaw sortowanie dla tabeli

| Właściwości | Opis                                                         |
| ----------- | ------------------------------------------------------------ |
| Kolumna     | kolumna, do której należy stosować sortowanie                |
| Kierunek    | kierunek sortowania, który może być "rosnący" lub "malejący" |

### Przypadek użycia

Następnie pokażemy jak mapować dane z `**Upstash**` do komponentu `**table**`.

#### Krok 1 Dodaj akcję

Utwórzmy tabelę w Upstashu o nazwie **`employee_profile`, ** zawierającą 4 kolumny `name`, `level`, `age`, i `żonaty`.

Następnie możemy utworzyć nową akcję dla Upstasha z listy akcji w Chmurze ILLA i nazwać upstash2.

Aby wyświetlić wszystkie dane w profilu pracownicye_profile wprowadź fragment kodu poniżej w sekcji zapytania Redis.

```sql
ZDOBĄDŹ profil pracowniczy
```

Ponieważ dane wyjściowe są listą par słowników, chcemy je przekształcić w tablicę. W transformatorze włącz go i umieść kod poniżej.

```jsx
const jsonString = data[0].result;

// Parse the JSON string into a JavaScript object
const jsonData = JSON.parse(jsonString);

// Access the "employees" property and map it to a new array
const mappedArray = jsonData.employees.map((employee) => ({
  name: employee.name,
  level: employee.level,
  age: employee.age,
  married: employee.married,
}));

return mappedArray;
```

Kliknij **`Zapisz`** i **Uruchom\`**, aby aktywować tę akcję.

#### Krok 2 Dodaj komponenty

Następnie możemy dodać komponent **`table`** do płótna.

W **polu `Źródło danych`** na stronie kontroli po kliknięciu na komponent, wprowadź `{{upstash2.data}}` aby pobrać dane z Upstash.

## <h2 hidden>Czym jest oś czasu?</h2>

Składnikiem osi czasu jest graficzne przedstawienie zdarzeń lub punktów danych ułożonych chronologicznie wzdłuż osi poziomej lub pionowej. Pozwala użytkownikom wizualizować i nawigować przez szereg wydarzeń lub kamieni milowych w sposób liniowy.

### Właściwości

Dostępne właściwości osi czasu. JavaScript może być napisany w celu przeczytania lub zmiany informacji o komponentach.

| Właściwości     | Opis                                                                                |
| --------------- | ----------------------------------------------------------------------------------- |
| Przedmioty      | gromadzenie indywidualnych zdarzeń lub punktów danych wyświetlanych na osi czasu;   |
| Kierunek        | orientacja lub układ elementu osi czasu (poziomy lub pionowy)                       |
| Oczekujące      | status elementu na osi czasu                                                        |
| Wysokość        | rozmiar pionowy lub wysokość elementu osi czasu                                     |
| Hidden          | czy komponent osi czasu jest widoczny czy ukryty                                    |

### Metody

Możesz użyć innych komponentów do kontroli komponentu. Popieramy następujące dwie metody:

-   **setValue**

Aby ustawić wartość pola tekstowego, na przykład {{"value1"}}

| Właściwości  | Opis        |
| ------------ | ----------- |
| Wartość      | Input value |

-   **clearValue**

Aby wyczyścić wartość wybranego komponentu

### Dane

Komponent posiada powszechnie używane dane, które mogą być wywoływane przez **`{{componentName.propertyName}}`** w aplikacji.

| Nazwa własności          | Opis                                                                                                       |
| ------------------------ | ---------------------------------------------------------------------------------------------------------- |
| kierunek                 | orientacja lub układ osi czasu                                                                             |
| wysokość dynamiczna      | zdolność dynamicznego dostosowania wysokości osi czasu na podstawie jej treści lub innych czynników.       |
| nazwa wyświetlacza       | nazwa tego komponentu (tj. oś czasowa1)                                                                    |
| hidden                   | czy komponent osi czasu jest widoczny czy ukryty                                                           |
| elementy                 | gromadzenie indywidualnych zdarzeń lub punktów danych wyświetlanych na osi czasu;                          |
| oczekujące               | status elementu na osi czasu                                                                               |
| kierunek zmiany rozmiaru | kierunek, w którym komponent osi czasu może być przeskalowany przez użytkownika                            |

Przykład: {{timeline1.items}}

### Przypadek użycia

Następnie pokażemy jak zamapować dane z źródła danych do komponentu osi czasu.

#### Krok 1 Dodaj akcję

Utwórzmy tabelę w Supabase o nazwie **`oś czasowa`, ** zawierającą tylko 1 kolumny: kamienie milowe. Istnieją 4 etapy mapowania komponentu osi czasu.

Następnie możemy utworzyć nową akcję dla Supabase z listy akcji i nazwać supabasedb1.

Aby wyświetlić wszystkie dane w kamieniu milowym, umieść fragment kodu poniżej w sekcji zapytania SQL.

```sql
wybierz * z osi czasu;
```

Ponieważ dane wyjściowe są listą par słowników, chcemy je przekształcić w tablicę. W transformatorze włącz go i umieść kod poniżej.

```jsx
const transformedData = data.map((obj) => Object.values(obj)[0]);
return transformedData;
```

Kliknij **`Zapisz`** i **Uruchom\`**, aby aktywować tę akcję.

![timeline_data](https://cdn.illacloud.com/official-website/img/official-site/components/timeline_data.jpg)

#### Krok 2 Dodaj komponenty

Następnie możemy dodać komponent **`oś czasowa`** do płótna.

W polu `**Items**` na stronie Przeglądaj po kliknięciu na komponent, wprowadź `{{supabasedb1.data}}` aby pobrać dane z suabase.

#### Etap 3 Badanie

Ostatni wygląd powinien być pokazany.

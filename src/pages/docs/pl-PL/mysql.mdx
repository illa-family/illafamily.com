## <h2 hidden>Co to jest MySQL?</h2>

MySQL w Illa to integracja z bazą danych, która pozwala na połączenie i interakcję z bazą danych MySQL. MySQL jest popularnym otwartym systemem zarządzania bazami danych, który jest znany ze swojej prędkości, niezawodności i łatwości użycia.

Z integracją MySQL w Illa, możesz zapytać, wstawiać, aktualizować i usuwać dane z bazy danych MySQL za pomocą poleceń SQL. Możesz również tworzyć niestandardowe zapytania SQL i uruchamiać je w Illa. Ułatwia to tworzenie aplikacji opartych na danych i pulpitów nawigacyjnych, które wyświetlają dane w czasie rzeczywistym z bazy danych MySQL.

### Create MySQL

Istnieją dwa sposoby na utworzenie zasobu w Illa po zalogowaniu się na swoje konto Illa.

-   Utwórz w zasobach

Zaloguj się do swojego konta Illa wybierz `**Zasoby**` na górze strony i kliknij przycisk `**Utwórz nowy**`.

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

Wybierz **`MySQL`** z listy baz danych.

![action_list](https://cdn.illacloud.com/official-website/img/docs/action_list.png)

Połącz się z bazą danych z wymaganymi parametrami opisanymi poniżej w "Ustawienia połączenia".

Kliknij `**Test Connection**` aby sprawdzić, czy możemy połączyć się z bazą danych. Jeśli tak, kliknij **`Zapisz zasoby`**, sprawdź dwukrotnie nazwę hosta, port, nazwę użytkownika i hasło jest poprawne.

Po utworzeniu zasobu, gotowe MySQL wyświetli się jak pokazano.

![my_ex_ready](https://cdn.illacloud.com/official-website/img/docs/my_ex_ready.png)

-   Utwórz w Konstruktorze

Zaloguj się na swoje konto Illa, utwórz projekt w Illa Builder na stronie `**App**`, i przejdź do **listy działań\`** na dole strony. Kliknij `**nowe**`, a następnie wybierz **`MySQL`** z listy bazy danych. Następnie połącz się z bazą danych z wymaganymi parametrami opisanymi w `Connection Settings` poniżej.

Kliknij `**Test Connection**` aby sprawdzić, czy możemy połączyć się z bazą danych. Jeśli tak, kliknij **`Zapisz zasoby`**, sprawdź dwukrotnie nazwę hosta, port, nazwę użytkownika i hasło jest poprawne.

### Ustawienia połączenia

Tutaj musimy dostarczyć informacji do połączenia z bazą danych MySQL.

![my_config](https://cdn.illacloud.com/official-website/img/docs/my_config.png)

| Właściwości          | Wymagane     | Opis                                                                                                                         |
| -------------------- | ------------ | ---------------------------------------------------------------------------------------------------------------------------- |
| Nazwisko             | wymagane     | Nazwa zasobu podczas tworzenia akcji w ILLA.                                                                                 |
| Hostname             | wymagane     | Adres URL lub IP dla Twojej bazy danych                                                                                      |
| Port                 | wymagane     | Numer portu hosta serwera, który powinieneś użyć do połączenia. Jeśli nie określisz portu, domyślnym portem jest '3306'.     |
| Baza danych          | wymagane     | Nazwa bazy danych                                                                                                            |
| Nazwa użytkownika    | wymagane     | nazwę użytkownika, której chcesz używać podczas logowania do serwera MySQL.                                                  |
| Hasło                | wymagane     | Użyj tego hasła do uwierzytelniania.                                                                                         |
| Opcje SSL            | fakultatywne | decyduje, jak wysokie jest bezpieczne połączenie SSL TCP/IP podczas negocjacji z serwerem.                                   |

### Utwórz akcje

Stworzyliśmy zasób MySQL, możemy dodać akcję wybierając MySQL z listy akcji i wybierając przycisk **Utwórz akcję\`**.

![my_resource_list](https://cdn.illacloud.com/official-website/img/docs/my_resource_list.png)

Teraz dodaliśmy serwer MySQL jako akcję do naszej strony budynku.

![my](https://cdn.illacloud.com/official-website/img/docs/my.png)

### Configure MySQL

| Właściwości     | Opis                                                                                        |
| --------------- | ------------------------------------------------------------------------------------------- |
| Zapytanie SQL   | polecenia sql do pobrania (select), wstawiania, aktualizacji, usuwania danych z bazy danych |
| Transformator   | przekształcanie danych w styl jaki lubisz za pomocą JavaScript                              |

Przykładowe użycie:

-   Wybierz

```sql
WYBIERZ * od użytkowników ORDER według nazwiska;
```

-   Insert

```sql
INSERT INTO users

	(firstname, lastname, email)

VALUES

	(

		{{ NewUser.data.firstNameInput }},

		{{ NewUser.data.lastNameInput }},

		{{ NewUser.data.emailInput }}

	);
```

-   Aktualizuj

```sql
AKTUALIZUJ użytkowników
  SET imię = '{{form1.updatedProfile.first}}'
  WHERE id = {{ form1.updatedProfile.uid}};
```

-   Usuń

```sql
USUŃ użytkowników gdzie id = {{ form1.updatedProfile.uid }};
```

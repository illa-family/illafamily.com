## <h2 hidden>Co to jest ikona?</h2>

Komponent Ikony w chmurze ILLA jest potężną funkcją, która pozwala użytkownikom na zwiększenie atrakcyjności wizualnej i funkcjonalności ich aplikacji. Z komponentem ikony użytkownicy mogą łatwo dodawać ikony z popularnych bibliotek ikon do swoich interfejsów użytkownika, zapewnienie użytkownikom bardziej intuicyjnego i angażującego doświadczenia.

### Właściwości

| Właściwości     | Opis                                                                                                                                                     |
| --------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Ikona           | pozwala użytkownikom na łatwe wybranie ikony z predefiniowanego zestawu opcji                                                                            |
| Obsługa zdarzeń | Wyzwalacz zapytań, sterowanie komponentami lub wywoływanie innych interfejsów API w odpowiedzi na zdarzenia komponentów.                                 |
| Wskazówka       | Użytkownicy mogą tutaj wprowadzić podpowiedź składową. Podpowiedź zostanie wyświetlona, gdy jest ostrością. Format Markdown jest obsługiwany.            |
| Hidden          | Dynamicznie kontroluj czy komponent jest ukryty. Możesz zmienić ukryty status za pomocą dynamicznej wartości logicznej.                                  |
| Kolor motywu    | Pozwala użytkownikom określić kolor tła i przezroczystość przycisku                                                                                      |

### Obsługa zdarzeń

| Wydarzenie | Opis                                                                                        |
| ---------- | ------------------------------------------------------------------------------------------- |
| Click      | Gdy użytkownik kliknie tę ikonę, wykonaj określoną czynność dostosowaną przez użytkowników. |

### Dane

Komponent posiada powszechnie używane dane, które mogą być wywoływane przez **`{{componentName.propertyName}}`** w aplikacji.

| Nazwa własności      | Opis                                                                                            |
| -------------------- | ----------------------------------------------------------------------------------------------- |
| schemat kolorów      | kolor obrysu ikony                                                                              |
| nazwa wyświetlacza   | nazwa tego komponentu (tj. ikona 1)                                                             |
| wydarzenia           | wygenerowane lub wywołane, gdy w obrębie komponentu wystąpią określone działania lub zdarzenia. |
| hidden               | ukryty status (prawda lub fałsz)                                                                |
| nazwa ikony          | nazwa etykiety dla ikony                                                                        |
| tekst                | wartość tekstu podpowiedzi                                                                      |

Przykład: {{icon1.iconName}}

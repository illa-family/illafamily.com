## <h2 hidden>Co to jest PostgreSQL?</h2>

PostgreSQL w Illa to integracja bazy danych, która pozwala na łączenie i interakcję z bazą danych PostgreSQL. PostgreSQL jest popularnym systemem zarządzania relacyjną bazą danych open source, który jest znany ze swojej stabilności, niezawodności i potężnych funkcji.

Z integracją PostgreSQL w Illa można zapytać, wstawiać, aktualizować i usuwać dane z bazy danych PostgreSQL za pomocą poleceń SQL. Możesz również tworzyć niestandardowe zapytania SQL i uruchamiać je w akcji Illa. Ułatwia to tworzenie aplikacji opartych na danych i pulpitów nawigacyjnych, które wyświetlają dane w czasie rzeczywistym z bazy danych PostgreSQL.

### Utwórz PostgreSQL

Istnieją dwa sposoby na utworzenie zasobu w Illa po zalogowaniu się na swoje konto Illa.

-   Utwórz w zasobach

Zaloguj się do swojego konta Illa wybierz `**Zasoby**` na górze strony i kliknij przycisk `**Utwórz nowy**`.

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

Wybierz **`PostgreSQL`** z listy baz danych.

![action_list](https://cdn.illacloud.com/official-website/img/docs/action_list.png)

Połącz się z bazą danych z wymaganymi parametrami opisanymi poniżej w "Ustawienia połączenia".

Kliknij `**Test Connection**` aby sprawdzić, czy możemy połączyć się z bazą danych. Jeśli tak, kliknij **`Zapisz zasoby`**, sprawdź dwukrotnie nazwę hosta, port, nazwę użytkownika i hasło jest poprawne.

Po utworzeniu zasobu gotowy PostgreSQL będzie wyświetlany jak pokazano.

![ex_post_ready](https://cdn.illacloud.com/official-website/img/docs/ex_post_ready.png)

-   Utwórz w Konstruktorze

Zaloguj się na swoje konto Illa, utwórz projekt w Illa Builder na stronie `**App**`, i przejdź do **listy działań\`** na dole strony. Kliknij `**nowe**`, a następnie wybierz **`PostgreSQL`** z listy bazy danych. Następnie połącz się z bazą danych z wymaganymi parametrami opisanymi w `Connection Settings` poniżej.

Kliknij `**Test Connection**` aby sprawdzić, czy możemy połączyć się z bazą danych. Jeśli tak, kliknij **`Zapisz zasoby`**, sprawdź dwukrotnie nazwę hosta, port, nazwę użytkownika i hasło jest poprawne.

### Ustawienia połączenia

Tutaj musimy dostarczyć informacji do połączenia z bazą danych PostgreSQL.

![post_config](https://cdn.illacloud.com/official-website/img/docs/post_config.png)

| Właściwości          | Wymagane     | Opis                                                                                                                         |
| -------------------- | ------------ | ---------------------------------------------------------------------------------------------------------------------------- |
| Nazwisko             | wymagane     | Nazwa zasobu podczas tworzenia akcji w ILLA.                                                                                 |
| Hostname             | wymagane     | Adres URL lub IP dla Twojej bazy danych                                                                                      |
| Port                 | wymagane     | Numer portu hosta serwera, który powinieneś użyć do połączenia. Jeśli nie określisz portu, domyślnym portem jest '5432'.     |
| Baza danych          | wymagane     | Nazwa bazy danych                                                                                                            |
| Nazwa użytkownika    | wymagane     | nazwę użytkownika, której chcesz używać podczas logowania do serwera PostgreSQL.                                             |
| Hasło                | wymagane     | Użyj tego hasła do uwierzytelniania.                                                                                         |
| Opcje SSL            | fakultatywne | decyduje, jak wysokie jest bezpieczne połączenie SSL TCP/IP podczas negocjacji z serwerem.                                   |

### Utwórz akcje

Stworzyliśmy zasób PostgreSQL, możemy dodać akcję wybierając PostgreSQL z listy akcji i wybierając przycisk **Utwórz akcję\`**.

![post_resource_list](https://cdn.illacloud.com/official-website/img/docs/post_resource_list.png)

Teraz dodaliśmy serwer PostgreSQL jako działanie do naszej strony budynku.

![post](https://cdn.illacloud.com/official-website/img/docs/post.png)

### Konfiguruj PostgreSQL

| Właściwości     | Opis                                                                                        |
| --------------- | ------------------------------------------------------------------------------------------- |
| Zapytanie SQL   | polecenia sql do pobrania (select), wstawiania, aktualizacji, usuwania danych z bazy danych |
| Transformator   | przekształcanie danych w styl jaki lubisz za pomocą JavaScript                              |

Przykładowe użycie:

-   Wybierz

```sql
SELECT * FROM users ORDER BY lastname;
```

-   Insert

```sql
INSERT INTO users

	(firstname, lastname, email)

VALUES

	(

		{{ NewUser.data.firstNameInput }},

		{{ NewUser.data.lastNameInput }},

		{{ NewUser.data.emailInput }}

	);
```

-   Aktualizuj

```sql
UPDATE users
  SET firstname = '{{form1.updatedProfile.first}}'
  WHERE id = {{ form1.updatedProfile.uid}};
```

-   Usuń

```sql
DELETE FROM users WHERE id = {{ form1.updatedProfile.uid }};
```

## <h2 hidden>Co to jest obrazek?</h2>

Komponent obrazu w chmurze ILLA jest wszechstronnym narzędziem, które pozwala użytkownikom dodawać i wyświetlać obrazy w swoich aplikacjach. Zapewnia on użytkownikom różne funkcje mające na celu poszerzenie wiedzy wizualnej.

### Właściwości

| Właściwości             | Opis                                                                                                                                      |
| ----------------------- | ----------------------------------------------------------------------------------------------------------------------------------------- |
| Źródło obrazu           | określa źródło obrazu, który ma być wyświetlany                                                                                           |
| Tekst alternatywny      | alternatywny tekst obrazu. Jest wyświetlany, gdy obraz nie może być załadowany lub dla użytkowników z upośledzeniem wzroku                |
| Typ skali               | w jaki sposób obraz jest skalowany w wymiarach komponentu, w tym pojemnik, osłonę, wypełnienie, brak, skalowanie w dół.                   |
| Obsługa zdarzeń         | zezwalanie użytkownikom na definiowanie akcji uruchamianych przez interakcje użytkownika z obrazem                                        |
| Wskazówka               | dodaj podpowiedź lub tekst opisowy, który pojawia się gdy użytkownik uskakuje nad obrazem                                                 |
| Hidden                  | kontrolowanie widoczności elementu obrazu.                                                                                                |
| Styles                  | umożliwienie użytkownikom dostosowania jego wyglądu. Użytkownicy mogą stosować style takie jak promień                                    |

### Metoda

Możesz użyć innych komponentów do kontroli komponentu. Popieramy następujące metody:

-   **setImageUrl**

zmienia dynamicznie źródło lub adres URL obrazu.

| Właściwości       | Opis                    |
| ----------------- | ----------------------- |
| Adres URL obrazu  | adres URL źródła obrazu |

### Obsługa zdarzeń

| Wydarzenie | Opis                                                                         |
| ---------- | ---------------------------------------------------------------------------- |
| Click      | wykonaj określone akcje lub wykonaj kod, gdy komponent obrazu jest kliknięty |

### Dane

Komponent posiada powszechnie używane dane, które mogą być wywoływane przez **`{{componentName.propertyName}}`** w aplikacji.

| Nazwa własności      | Opis                                                                                                                          |
| -------------------- | ----------------------------------------------------------------------------------------------------------------------------- |
| altText              | alternatywa tekstowa opisująca zawartość lub cel obrazu.                                                                      |
| nazwa wyświetlacza   | Wartość ciągu reprezentująca nazwę lub tytuł obrazu.                                                                          |
| wydarzenia           | Tablica lub obiekt, który definiuje obsługę zdarzeń związanych z komponentem obrazu, np. kliknięcie                           |
| hidden               | Wartość logiczna określająca czy komponent obrazu jest ukryty z interfejsu użytkownika.                                       |
| imageSrc             | źródło lub położenie obrazu, który ma być wyświetlany w komponencie                                                           |
| Pasek obiektu        | w jaki sposób obraz jest montowany lub skalowany w obrębie wymiarów komponentu                                                |
| radius               | zastosuj zaokrąglone narożniki do komponentu obrazu.                                                                          |
| tekst                | wartość ciągu reprezentująca tekst podpowiedzi lub wskazówek do wyświetlenia, gdy użytkownik uskakuje nad komponentem obrazu. |

Przykład: {{image1.displayName}}

### Przypadek użycia

Następnie pokażemy jak sterować wyświetlaniem obrazów za pomocą przycisku.

#### Krok 1 Dodaj komponenty

Dodaj komponent **Image\`** i komponent **`przycisk`** do płótna.

Możesz ustawić `image source` dla komponentu obrazu na dowolny adres obrazu, który chcesz wyświetlić. Tutaj użyjemy domyślnego obrazu jako przykładu startu i oznacz przycisk jako "Zmień", jak pokazano poniżej

![image_layout](https://cdn.illacloud.com/official-website/img/official-site/components/image_layout.jpg)

#### Krok 2 Konfiguruj komponent

Dla komponentu **`przycisk`** możemy go skonfigurować, aby ustawić źródło obrazu składnika **`obraz`** na nowy adres obrazu.

1.  Kliknij komponent `**button**` aby otworzyć jego stronę kontrolną. W **`Obsługa zdarzeń`**, kliknij `**+ Nowa**`.
2.  W obsłudze zdarzeń edytuj wybierz **`składnik kontroli`** w akcji, i wybierz komponent **image1\`**, który chcesz zmienić jako cel wydarzenia.
3.  Wybierz działanie **`setImageUrl`**
4.  Umieść nowy adres obrazu w polu **`Wartość`**

![image_config](https://cdn.illacloud.com/official-website/img/official-site/components/image_config.jpg)

#### Etap 3 Badanie

Teraz, gdy klikniesz przycisk "Odtwórz", obraz powinien się przełączać

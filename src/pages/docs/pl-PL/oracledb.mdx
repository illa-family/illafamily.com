## <h2 hidden>Co to jest Oracle DB?</h2>

Oracle DB w Illa to integracja z bazą danych, która pozwala na połączenie i interakcję z bazą danych Oracle. Oracle DB jest popularnym otwartym systemem zarządzania bazami danych, który jest znany ze swojej prędkości, niezawodności i łatwości użycia.

Dzięki integracji bazy danych Oracle w Illa, możesz zapytać, wstawiać, aktualizować i usuwać dane z bazy danych Oracle DB za pomocą poleceń SQL. Możesz również tworzyć niestandardowe zapytania SQL i uruchamiać je w Illa. Ułatwia to tworzenie aplikacji opartych na danych i pulpitów nawigacyjnych, które wyświetlają dane w czasie rzeczywistym z bazy danych Oracle.

### Utwórz bazę danych Oracle

Istnieją dwa sposoby na utworzenie zasobu w Illa po zalogowaniu się na swoje konto Illa. 

-   Utwórz w zasobach

Zaloguj się do swojego konta Illa wybierz `**Zasoby**` na górze strony i kliknij przycisk `**Utwórz nowy**`. 

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

Wybierz **`Oracle DB`** z listy baz danych.

![action_list](https://cdn.illacloud.com/official-website/img/docs/action_list.png)

Połącz się z bazą danych z wymaganymi parametrami opisanymi poniżej w "Ustawienia połączenia". 

Kliknij `**Test Connection**` aby sprawdzić, czy możemy połączyć się z bazą danych. Jeśli tak, kliknij **`Zapisz zasoby`**, sprawdź dwukrotnie nazwę hosta, port, nazwę użytkownika i hasło jest poprawne.

Po utworzeniu zasobu gotowy DB Oracle będzie wyświetlany jak pokazano.

![oracle_ex_ready](https://cdn.illacloud.com/official-website/img/docs/oracle_ex_ready.png)

-   Utwórz w Konstruktorze

Zaloguj się na swoje konto Illa, utwórz projekt w Illa Builder na stronie `**App**`, i przejdź do **listy działań\`** na dole strony. Kliknij `**nowe**`, a następnie wybierz **`Oracle DB`** z listy baz danych. Następnie połącz się z bazą danych z wymaganymi parametrami opisanymi w `Connection Settings` poniżej.

Kliknij `**Test Connection**` aby sprawdzić, czy możemy połączyć się z bazą danych. Jeśli tak, kliknij **`Zapisz zasoby`**, sprawdź dwukrotnie nazwę hosta, port, nazwę użytkownika i hasło jest poprawne.

### Ustawienia połączenia

Tutaj musimy dostarczyć informacji do połączenia z bazą danych Oracle.

![oracle_config](https://cdn.illacloud.com/official-website/img/docs/oracle_config.png)

| Właściwości                    | Wymagane  | Opis                                                                                                                     |
| ------------------------------ | --------- | ------------------------------------------------------------------------------------------------------------------------ |
| Nazwisko                       | wymagane  | Nazwa zasobu podczas tworzenia akcji w ILLA.                                                                             |
| Hostname                       | wymagane  | Adres URL lub IP dla Twojej bazy danych                                                                                  |
| Port                           | wymagane  | Numer portu hosta serwera, który powinieneś użyć do połączenia. Jeśli nie określisz portu, domyślnym portem jest '3306'. |
| Baza danych                    | wymagane  | Nazwa bazy danych                                                                                                        |
| Traktuj nazwę bazy danych jako | wymagane  | identyfikuje instancję bazy danych z SID lub nazwą usługi.                                                               |
| Nazwa użytkownika              | wymagane  | nazwa użytkownika, której chcesz używać podczas logowania do serwera Oracle DB.                                          |
| Hasło                          | wymagane  | Użyj tego hasła do uwierzytelniania.                                                                                     |
| SSL                            | wymagane  | decyduje, jak wysokie jest bezpieczne połączenie SSL TCP/IP podczas negocjacji z serwerem.                               |

### Utwórz akcje

Stworzyliśmy zasób bazy danych Oracle, możemy dodać akcję wybierając bazę danych Oracle z listy akcji i wybierając przycisk **Utwórz akcję\`**.

![oracle_resource_list](https://cdn.illacloud.com/official-website/img/docs/oracle_resource_list.png)

 Teraz dodaliśmy serwer Oracle DB jako działanie do naszej strony budynku.

![oracle](https://cdn.illacloud.com/official-website/img/docs/oracle.png)

### Konfiguracja bazy danych Oracle

| Właściwości   | Opis                                                                                        |
| ------------- | ------------------------------------------------------------------------------------------- |
| Zapytanie SQL | polecenia sql do pobrania (select), wstawiania, aktualizacji, usuwania danych z bazy danych |
| Transformator | przekształcanie danych w styl jaki lubisz za pomocą JavaScript                              |

Przykładowe użycie:

-   Wybierz

```sql
WYBIERZ * od użytkowników ORDER według nazwiska;
```

-   Insert

```sql
INSERT INTO users

	(firstname, lastname, email)

VALUES

	(

		{{ NewUser.data.firstNameInput }},

		{{ NewUser.data.lastNameInput }},

		{{ NewUser.data.emailInput }}

	);
```

-   Aktualizuj

```sql
UPDATE users
  SET firstname = '{{form1.updatedProfile.first}}'
  WHERE id = {{ form1.updatedProfile.uid}};
```

-   Usuń

```sql
DELETE FROM users WHERE id = {{ form1.updatedProfile.uid }};
```

## <Heading hidden>Lista Integracji</Heading>

ILLA oferuje szereg wstępnie zbudowanych integracji, które pozwalają na łatwe łączenie się ze źródłami danych z minimalną konfiguracją. Po prostu wybierz odpowiednią integrację z listy dostępnych opcji, wprowadź niezbędne dane logowania i możesz zacząć korzystać z źródła danych w ILLA. To eliminuje potrzebę konfiguracji skomplikowanych konfiguracji i umożliwia szybkie rozpoczęcie pracy z danymi.

### Integracje bazy danych

| Nazwisko                | Opis                                                                                                                                                                                                                   |
| ----------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Mysql                   | MySQL jest systemem zarządzania bazą danych relacyjnych (RDBMS), opartym na strukturalnym języku zapytania (SQL).                                                                                                      |
| PostgreSQL              | PostgreSQL jest otwartym systemem zarządzania bazą danych obiektów (ORDBMS), który jest znany ze swojej wiarygodności i bogactwa funkcji.                                                                              |
| MariaDB                 | MariaDB jest otwartym systemem zarządzania relacyjną bazą danych, który jest forkiem bazy danych MySQL.                                                                                                                |
| TiDB                    | TiDB jest otwarto-źródłową bazą danych NewSQL, która obsługuje obciążenia pracą Hybrid Transactional and Analytical Processing.                                                                                        |
| Redis                   | Redis jest otwarto-źródłowym magazynem danych w pamięci, który jest często używany jako baza danych, pamięć podręczna i broker wiadomości.                                                                             |
| MongoDB                 | MongoDB jest darmowym i otwartym oprogramowaniem wieloplatformowym zorientowanym na dokumentację.                                                                                                                      |
| Wyszukiwanie elastyczne | Elasticsearch jest wyszukiwarką open-source, rozproszoną i wyszukiwarką opartą na Apache Lucene.                                                                                                                       |
| Firebase                | Firebase jest platformą rozwoju aplikacji mobilnych i internetowych stworzoną przez Google.                                                                                                                            |
| Supabase DB             | Supabase jest platformą open-source do budowania aplikacji w czasie rzeczywistym. Opiera się on na PostgreSQL i zapewnia szereg narzędzi i usług umożliwiających szybkie tworzenie i wdrażanie aplikacji bez serwerów. |
| Clickhouse              | ClickHouse jest otwarto-źródłowym systemem zarządzania bazą danych, który jest zaprojektowany do analizy danych w czasie rzeczywistym (OLAP) i do analizy danych w czasie rzeczywistym.                                |

### Integracje API

| Nazwisko              | Opis                                                                                                                                           |
| --------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------- |
| API REST              | REST to styl architektoniczny dla usług sieciowych. RESTful API to API zgodne ze stylem architektonicznym REST.                                |
| Amazon S3(General S3) | Amazon S3 jest usługą przechowywania obiektów oferowaną przez AWS. Zapewnia bezpieczne, skalowalne i tanie przechowywanie danych w internecie. |
| SMTP                  | SMTP (Simple Mail Transfer Protocol) jest protokołem do wysyłania i odbierania wiadomości e-mail przez Internet.                               |
| Wykres QL             | GraphQL to język zapytania API i czas wykonania tych zapytań z istniejącymi danymi.                                                            |

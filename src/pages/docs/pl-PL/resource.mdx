## <Heading hidden>Zasób</Heading>

Połącz się ze swoim źródłem danych i zadzwoń do niego w swoim konstruktorze ILLA. Dostarczamy Ci elastyczne moduły połączeń dla różnych typów źródeł danych. Możesz zbudować stabilne połączenie pomiędzy danymi a ILLA w krótkim czasie, znacznie upraszczając proces dostępu do danych.

Po pomyślnym utworzeniu zasobu, możesz dzwonić do dowolnej aplikacji w ILLA. Możesz również udostępnić swoje zasoby swoim współtwórcom, zwiększając wydajność tworzenia aplikacji.

### Obsługiwany typ zasobów

ILLA obsługuje wiele głównych baz danych, takich jak MySQL i Postage, a także API takie jak Rest API i GraphQL.

### Utwórz zasób

Zasoby reprezentują połączenia między konstruktorem ILLA a twoimi źródłami danych. Utwórz zasób przez następujące kroki:

#### 1.  Przejdź do zakładki Zasoby na pasku nawigacyjnym strony głównej i kliknij przycisk "Utwórz nową"

  src='https://cdn.illacloud.com/official-website/img/docs/re_e_1.png'

/>

#### 2.  Wybierz typ danych tutaj

<img
  src='https://cdn.illacloud.com/official-website/img/docs/qs_e_2.png'
  alt=""
/>

#### 3.  Uściśl informacje o źródle danych i kliknij przycisk "Utwórz zasob", a następnie dokończ proces

<img
  src='https://cdn.illacloud.com/official-website/img/docs/re_e_2.png'

Ponadto możesz tworzyć zasoby na liście akcji w edytorze aplikacji

> ### Test połączenia
>
> Podczas tworzenia lub przeglądania zasobów, możesz kliknąć przycisk testowy w panelu zasobów, aby sprawdzić, czy źródło danych jest podłączone pomyślnie.

**Πρέπει να έχετε Docker πριν χρησιμοποιήσετε το ILLA CLI για να αναπτύξετε μια αυτο-φιλοξενημένη εγκατάσταση. Όταν χρησιμοποιείτε ILLA CLI, θα πρέπει να προσθέσετε τον τρέχοντα χρήστη στην ομάδα docker ή να χρησιμοποιήσετε στη ρίζα.**

## Λήψη ILLA CLI

Εάν έχετε εγκαταστήσει το Rust σας, μπορείτε εύκολα να χτίσετε + να εγκαταστήσετε την τελευταία έκδοση ILLA CLI με **φορτίο**:

```
> cargo install illa
```

Το εργαλείο **φορτίου** θα ολοκληρώσει τη λήψη του ILLA CLI με τις εξαρτήσεις πηγής, χτίστε και εγκαταστήστε το στο μονοπάτι bin φορτίου έτσι ώστε να μπορούμε να το τρέξουμε.
Μόλις εγκατασταθεί, μπορείτε να εκτελέσετε το ILLA CLI με την εντολή **illa** .

Εναλλακτικά, μπορείτε απλά να κατεβάσετε το ILLA CLI για το λειτουργικό σας σύστημα εδώ:

-   [Windows](https://github.com/illacloud/illa/releases/latest/download/illa-x86_64-win.zip)
-   [Linux](https://github.com/illacloud/illa/releases/latest/download/illa-x86_64-linux.tar.gz)
-   [Mac](https://github.com/illacloud/illa/releases/latest/download/illa-x86_64-macos.tar.gz)

Αν κάνετε λήψη από τους παραπάνω συνδέσμους, τα βήματα είναι ελαφρώς διαφορετικά σε σύγκριση με τη λήψη του ILLA CLI από **φορτίο**. Μετά τη λήψη του CLI, θα πρέπει να το κάνετε εκτελέσιμο. Ας πάρουμε το illa-linux για παράδειγμα:

```
## κατεβάστε το δυαδικό αρχείο illa cli
> wget https://github.com/illacloud/illa/releases/latest/download/illa-x86_64-linux.tar. z

## αποσυμπιέστε το αρχείο
> tar -zxvf illa-x86_64-linux.tar. z

> cd illa-x86_64-linux

> chmod +x illa

## εκτελέστε το illa cli
> ./illa

```

### Έλεγχος των προϋποθέσεων της αυτοφιλοξενούμενης εγκατάστασης

**Αυτές οι θύρες 5432, 9999 και 8000 πρέπει να παραμείνουν για τον Κατασκευαστή ILLA**

Η ακόλουθη εντολή θα ελέγξει τις προϋποθέσεις της αυτοφιλοξενούμενης εγκατάστασης στο λειτουργικό σας σύστημα:

```
> illa doctor
```

### Ανάπτυξη Κατασκευαστή ILLA

Μόλις εγκατασταθεί το ILLA CLI, μπορείτε να εκτελέσετε μια αυτο-φιλοξενία ή [εγκατάσταση στο σύννεφο](https://cloud.illacloud.com/).

Γρήγορη Έναρξη

```
> illa ανάπτυξη --self --port=10000
```

Μπορείτε απλά να αναπτύξετε έναν αυτοφιλοξενούμενο δημιουργό ILA εκτελώντας την ακόλουθη εντολή:

```
# η θύρα στην οποία μπορεί να προσπελαστεί η ILLA Builder μπορεί να αλλάξει
# το <MOUNT_PATH> είναι η προσαρμοσμένη διαδρομή προσάρτησης του ILLA Builder, η προεπιλεγμένη τιμή είναι μια διαδρομή κάτω από τον κατάλογο αρχικής σελίδας χρήστη αν δεν είναι γεμάτη
> illa deploy --self --port=10000 --mount=<MOUNT_PATH>
```

Τώρα, μπορείτε να αποκτήσετε πρόσβαση στο ILLA Builder: http://localhost:10000

## Ενημέρωση Δημιουργού ILLA

Για να ενημερώσετε τον δημιουργό ILLA με τις τελευταίες εικόνες docker, χρησιμοποιήστε την ακόλουθη εντολή:

```
> illa ενημέρωση --self
```

## Άλλες πράξεις

```
# λίστα με το ILLA Builder
> illa list --self

# σταμάτησε το ILLA Builder
> illa stop --self

# αφαίρεσε το ILLA Builder
> illa remove --self

# αναγκαστική αφαίρεση του εκτελούντος ILLA Builder και επίμονα δεδομένα
> sudo illa remove --force --self --data

# επανεκκινήστε το ILLA Builder
> illa restart --self

# help πληροφορίες
> illa help
```

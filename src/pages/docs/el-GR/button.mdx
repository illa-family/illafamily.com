## <h2 hidden>Τι είναι το Κουμπί?</h2>

Το στοιχείο Κουμπί επιτρέπει στους χρήστες να ενεργοποιήσουν μια ενέργεια ή να εκτελέσουν μια συνάρτηση κάνοντας κλικ στο κουμπί. 

### Ιδιότητες

| Ιδιότητες           | Περιγραφή                                                                                                                                           |
| ------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------- |
| Κείμενο             | Το κείμενο που εμφανίζεται για το κουμπί                                                                                                            |
| Χειριστής Συμβάντων | Ερωτήματα ενεργοποίησης, έλεγχος εφαρμογών, ή κλήση άλλων API σε απόκριση σε συμβάντα εφαρμογής.                                                    |
| Φόρτωση             | Αν το στοιχείο θα εμφανίζει μια ένδειξη φόρτωσης.                                                                                                   |
| Απενεργοποιημένο    | Ελέγξτε την κατάσταση του αν το στοιχείο είναι απενεργοποιημένο. Το στοιχείο δεν μπορεί να τροποποιηθεί ή να εστιαστεί όταν είναι απενεργοποιημένο. |
| Υπόδειξη            | Ο χρήστης μπορεί να εισαγάγει εδώ την υπόδειξη συστατικού. Η υπόδειξη θα εμφανίζεται όταν εστιάζεται. Υποστηρίζεται μορφή Markdown                  |
| Hidden              | Δυναμική ρύθμιση της απόκρυψης του συστατικού. Μπορείτε να αλλάξετε την κρυφή κατάσταση μέσω της δυναμικής δυαδικής τιμής.                          |
| Παραλλαγή           | Να επιτρέπεται στους χρήστες να επιλέγουν μεταξύ δύο διαφορετικών στυλ για το κουμπί: συμπλήρωση ή περίγραμμα.                                      |
| Χρώμα Θέματος       | Επιτρέπει στους χρήστες να καθορίσουν το χρώμα φόντου και την αδιαφάνεια του κουμπιού                                                               |

### Χειριστής συμβάντων

| Γεγονός | Περιγραφή                                                                                                       |
| ------- | --------------------------------------------------------------------------------------------------------------- |
| Click   | Όταν ένας χρήστης κάνει κλικ σε αυτό το κουμπί, εκτελέστε συγκεκριμένη ενέργεια προσαρμοσμένη από τους χρήστες. |

### Δεδομένα

Το στοιχείο έχει κάποια κοινώς χρησιμοποιούμενα δεδομένα, τα οποία μπορούν να κληθούν μέσω του **`{{componentName.propertyName}}`** στην εφαρμογή.

| Όνομα ιδιότητας   | Περιγραφή                                                            |
| ----------------- | -------------------------------------------------------------------- |
| colorScheme       | χρώμα φόντου του κουμπιού                                            |
| απενεργοποιημένο  | μια δυαδική τιμή δείχνει την κατάσταση απενεργοποιημένης λειτουργίας |
| Εμφάνιση Ονόματος | το όνομα αυτού του στοιχείου (δηλαδή το κουμπί1)                     |
| hidden            | μια δυαδική τιμή δείχνει την κρυφή κατάσταση του κουμπιού            |
| HiddenDynamic     | μια δυαδική τιμή δείχνει την κρυφή δυναμική κατάσταση του κουμπιού   |
| φόρτωση           | τιμή φόρτωσης                                                        |
| κείμενο           | τιμή κειμένου που εμφανίζεται στο κουμπί                             |
| tooltipText       | τιμή κειμένου υπόδειξης                                              |
| παραλλαγή         | στυλ παραλλαγής του κουμπιού (γεμίστε ή περιγράψετε)                 |

### Περίπτωση χρήσης

Στη συνέχεια, θα δείξουμε πώς να χρησιμοποιήσετε το κουμπί για τον έλεγχο του στοιχείου **`form`** δυναμικά. 

#### Βήμα 1 Προσθέστε μια ενέργεια

Έχουμε δημιουργήσει έναν πίνακα με όνομα **`countries`**, συμπεριλαμβανομένων 5 στηλών: **`num_code`**, **`alpha_2_code`**, **`alpha_3_code`**, **`en_short_name`**, **`nationality`**. Τα ονόματα των χωρών είναι αυτά που θέλουμε να χρησιμοποιήσουμε ως τιμή επιλογών, τα οποία αποθηκεύονται στη στήλη **`en_short_name`**.

Στη συνέχεια, μπορούμε να δημιουργήσουμε μια νέα ενέργεια για Supabase από τη λίστα ενεργειών, αναφέροντας όλα τα δεδομένα σε **`χώρες`** και με το όνομα supabasedb1 

```sql
επιλέξτε ξεχωριστό en_short_name ως τιμή από την παραγγελία "Χώρες" κατά τιμή;
```

Αυτό είναι που έχουμε ως έξοδο:

<img src='https://cdn.illacloud.com/official-website/img/docs/countries.png' alt='' />

Πατήστε **`Save`** και **`Run`** για να ενεργοποιήσετε αυτήν την ενέργεια.

#### Βήμα 2 Προσθέστε ένα στοιχείο

Πρώτα προσθέτουμε το **`Form`** component και το ονομάζουμε 'Survey'. Στη συνέχεια, σύρουμε ένα στοιχείο εισαγωγής, ένα στοιχείο επιλογής και ένα στοιχείο εισαγωγής αριθμού στη φόρμα, επισημαίνοντάς τα ως **`Name`, `Nationality`,** και **`Age`.**

Στη συνέχεια χαρτογραφούμε τις τιμές **`countries`** για να επιλέξετε το στοιχείο από 

1.  σε αντιστοιχισμένη λειτουργία, ορισμός πηγών δεδομένων ώστε να είναι `{{supabasedb1.data}}` 
2.  στην αντιστοιχισμένη επιλογή, ορίστε τιμή ως `{{item.value}}`

<img src='https://cdn.illacloud.com/official-website/img/docs/button1.png' alt='' />

Στη συνέχεια, μπορούμε να ρυθμίσουμε το κουμπί **`Submit`** για να ελέγξουμε τη μέθοδο **`Send`** του στοιχείου **`Form`**. 

1.  Στο χειριστή συμβάντων επεξεργασίας, επιλέξτε **`Control component`** εν ενεργεία, επιλέξτε το στοιχείο **`form`** που θέλετε να ενημερώσετε ως στόχο του συμβάντος.
2.  Επιλέξτε την ενέργεια **`Submit`**

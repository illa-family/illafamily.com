## <h2 hidden>Τι είναι η μορφή?</h2>

Το στοιχείο της φόρμας είναι ένα οπτικό στοιχείο που αντιπροσωπεύει την πρόοδο ή την ολοκλήρωση μιας εργασίας ή διεργασίας. Εμφανίζει οριζόντια γραμμή που γεμίζει ή συρρικνώνεται με βάση την καθορισμένη τιμή προόδου.

### Ιδιότητες

| Ιδιότητες                              | Περιγραφή                                                                                                                                                       |
| -------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Χειριστής συμβάντων                    | ανίχνευση και ανταπόκριση σε συγκεκριμένες ενέργειες χρήστη ή γεγονότα, όπως κλικ, πατήματα πλήκτρων ή υποβολή φόρμας.                                          |
| Απενεργοποιημένο                       | μη διαδραστικό και δεν μπορεί να τροποποιηθεί ή να ενεργοποιηθεί από το χρήστη.                                                                                 |
| Απενεργοποίηση υποβολής                | απενεργοποίηση της λειτουργίας υποβολής φόρμας.                                                                                                                 |
| Επικύρωση εισόδων κατά την υποβολή     | οι εισροές του εντύπου πρέπει να επικυρώνονται για την ορθότητα ή την πληρότητα όταν υποβάλλεται το έντυπο.                                                     |
| Επαναφορά μετά από επιτυχή υποβολή     | αυτόματα επαναφέρει τις τιμές και την κατάσταση στις αρχικές ή προκαθορισμένες τιμές τους μετά από επιτυχή υποβολή                                              |
| Υπόδειξη                               | ένα μικρό αναδυόμενο ή ενημερωτικό μήνυμα που εμφανίζεται όταν ο χρήστης αιωρείται ή αλληλεπιδρά με ένα συγκεκριμένο στοιχείο, όπως ένα στοιχείο φόρμας.        |
| Εμφάνιση κεφαλίδας                     | αν ένα στοιχείο φόρμας θα εμφανίζει μια κεφαλίδα ή μια ενότητα τίτλου                                                                                           |
| Εμφάνιση υποσέλιδου                    | αν ένα στοιχείο φόρμας πρέπει να εμφανίζει ένα τμήμα υποσέλιδου                                                                                                 |
| Hidden                                 | αποκρύπτει τη συνιστώσα της φόρμας από την προβολή του χρήστη                                                                                                   |
| Χρώμα Περιγράμματος                    | χρώμα του περιγράμματος που περιβάλλει ένα συστατικό μορφής                                                                                                     |
| Ακτίνα Περιγράμματος                   | καμπυλότητα των γωνιών των συνόρων ενός στοιχείου του εντύπου                                                                                                   |
| Border Width                           | πάχος ή πλάτος του περιγράμματος που περιβάλλει ένα στοιχείο μορφής                                                                                             |
| Φόντο                                  | καθορίζει το χρώμα ή την εικόνα που γεμίζει την περιοχή περιεχομένου ενός στοιχείου φόρμας.                                                                     |
| Σκιά                                   | οπτική επίδραση της προσθήκης ενός λεπτού, μετατόπιση "σκιά" σε ένα στοιχείο μορφής, καθιστώντας την ελαφρώς ανυψωμένη ή επιπλέοντας πάνω από το φόντο          |

### Μέθοδος

Μπορείτε να χρησιμοποιήσετε άλλα στοιχεία για τον έλεγχο της εφαρμογής. Υποστηρίζουμε τις ακόλουθες δύο μεθόδους:

-   **υποβολή**

Για την υποβολή των πληροφοριών που συμπληρώνονται στο πεδίο του εντύπου

-   **reset**

επαναφέρετε όλα τα πεδία εισαγωγής στο στοιχείο φόρμας

-   **ορισμός**

Για να ορίσετε την τιμή εισόδου της περιοχής κειμένου, για παράδειγμα, {{"value1"}}

| Ιδιότητες | Περιγραφή   |
| --------- | ----------- |
| Τιμή      | Input value |

-   validate

Για να επιβεβαιωθεί ότι οι πληροφορίες εισόδου είναι νόμιμες

### Δεδομένα

Το στοιχείο έχει κάποια κοινώς χρησιμοποιούμενα δεδομένα, τα οποία μπορούν να κληθούν μέσω του **`{{componentName.propertyName}}`** στην εφαρμογή.

| Όνομα ιδιότητας               | Περιγραφή                                                                                                                                                                |
| ----------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Χρώμα φόντου                  | καθορίζει το χρώμα φόντου ενός στοιχείου φόρμας                                                                                                                          |
| περίγραμμαΧρώμα               | καθορίζει το χρώμα του περιγράμματος που περιβάλλει ένα στοιχείο της φόρμας.                                                                                             |
| borderWidth                   | καθορίζει το πάχος ή το πλάτος του περιγράμματος που περικλείει ένα στοιχείο μορφής                                                                                      |
| απενεργοποιημένο              | αν ένα στοιχείο φόρμας είναι απενεργοποιημένο ή όχι                                                                                                                      |
| Εμφάνιση Ονόματος             | αντιπροσωπεύει το όνομα ή την ετικέτα που έχει ανατεθεί σε ένα στοιχείο φόρμας.                                                                                          |
| ΥψοςΎψος                      | καθορίζει το ύψος ή το μέγεθος του τμήματος υποσέλιδου ενός στοιχείου φόρμας.                                                                                            |
| formData                      | διατηρεί τα δεδομένα ή τις τιμές που εισήγαγε ο χρήστης σε συνιστώσα φόρμας.                                                                                             |
| ύψοςκεφαλίδας                 | καθορίζει το ύψος ή το μέγεθος της ενότητας κεφαλίδας ενός συστατικού εντύπου                                                                                            |
| μη έγκυρο                     | αν η εισαγωγή ή η τιμή ενός στοιχείου φόρμας θεωρείται άκυρη ή όχι.                                                                                                      |
| radius                        | καθορίζει την ακτίνα ή την καμπυλότητα των γωνιών ενός συστατικού εντύπου                                                                                                |
| resetAfterSuccessful          | δηλώνει ότι μια συνιστώσα της φόρμας θα πρέπει να επαναφέρει αυτόματα τις τιμές και την κατάσταση στις αρχικές ή προκαθορισμένες τιμές τους μετά από μια επιτυχή υποβολή |
| σκιά                          | ελέγχει την παρουσία και την εμφάνιση ενός εφέ σκιάς σε ένα συστατικό της φόρμας                                                                                         |
| Υποσέλιδο                     | καθορίζει αν θα πρέπει να εμφανίζεται ή όχι το τμήμα υποσέλιδου ενός στοιχείου της φόρμας.                                                                               |
| κεφαλίδα ντους                | ορατότητα του τμήματος κεφαλίδας ενός στοιχείου εντύπου                                                                                                                  |
| tooltipText                   | διατηρεί το κείμενο ή το περιεχόμενο μιας υπόδειξης που σχετίζεται με μια συνιστώσα φόρμας                                                                               |
| validateInputOnSubmit         | ορίζει ότι οι τιμές εισόδου ενός στοιχείου του εντύπου πρέπει να επικυρώνονται για ορθότητα ή πληρότητα όταν υποβάλλεται το έντυπο.                                      |

Παράδειγμα: {{form1.disabled}}

### Περίπτωση χρήσης

Θα δείξουμε πώς να φτιάξουμε μια φόρμα με το όνομα και την ηλικία ως είσοδο, καθώς και την πρόοδο της γραμμής για να δείξει την πρόοδο ολοκλήρωσης της φόρμας.

#### Βήμα 1 Προσθήκη Στοιχείων

Πρόσθεσε ένα στοιχείο **`form`** με το προεπιλεγμένο κείμενο "Form" και ένα κουμπί "Submit" σε καμβά.

Πρόσθεσε ένα **`input`** component, ένα **`number input`** component, και ένα **`bar progress`** component από `**Insert**` στη φόρμα.

Ετικέτα το στοιχείο εισόδου να είναι "Όνομα", το στοιχείο εισόδου αριθμού να είναι "Ηλικία" και το στοιχείο προόδου μπάρα να είναι "Πρόοδος".

Ορίστε το σύμβολο κράτησης θέσης του στοιχείου εισόδου σε "Παρακαλώ εισάγετε το όνομά σας εδώ" και την προεπιλεγμένη τιμή εισόδου του αριθμού στο 0, όπως φαίνεται παρακάτω.

![bar_layout](https://cdn.illacloud.com/official-website/img/official-site/components/bar_layout.jpg)

#### Βήμα 2 Διαμόρφωση του εξαρτήματος

Για τα **`input`** και **`number input`** components, μπορούμε να τα ρυθμίσουμε ώστε να ορίσετε την τιμή σε **`bar progress`** component ώστε να αντικατοπτρίζει την πρόοδο της ολοκλήρωσης.

1.  Κάνε κλικ στο στοιχείο **`input`** και **`number input`** για να ανοίξεις την σελίδα ελέγχου της. Κάτω από **`Χειριστής Συμβάντων`**, κάντε κλικ στο `**+ Νέο**`.
2.  Στο χειριστή συμβάντων επεξεργασίας, επιλέξτε **`Control component`** εν ενεργεία, και επιλέξτε το στοιχείο **`bar progress`** που θέλετε να ενημερώσετε ως στόχο της εκδήλωσης.
3.  Επιλέξτε την **`setValue`** ενέργεια και ορίστε την τιμή ως

```jsx
{
  {
    input1.value && numberInput1.value
      ? "100"
      : input1.value || numberInput1.value
      ? "50"
      : "0";
  }
}
```

![bar_input_num_config](https://cdn.illacloud.com/official-website/img/official-site/components/bar_input_num_config.jpg)

Σημείωση: πρέπει να ρυθμίσετε τον χειριστή συμβάντος, όπως περιγράφεται παραπάνω, σε στοιχεία εισόδου BOTH και εισόδου αριθμού.

Για το **`button`** component, μπορούμε να το ρυθμίσουμε ώστε να υποβάλουμε τη φόρμα αν η φόρμα ολοκληρωθεί (δηλαδή πρόοδος στη γραμμή 100%)

1.  Κάντε κλικ στο στοιχείο **`button`** για να ανοίξετε τη σελίδα ελέγχου. Κάτω από **`Χειριστής Συμβάντων`**, κάντε κλικ στο `**+ Νέο**`.
2.  Στο χειριστή συμβάντων επεξεργασίας, επιλέξτε **`Control component`** εν ενεργεία, και επιλέξτε το στοιχείο **`form`** που θέλετε να ενημερώσετε ως στόχο του γεγονότος.
3.  Επιλέξτε τη μέθοδο **`submit`** και βάλτε `{{barProgress1.value == 100}}}` στο **`Only run when`**

![bar_button_submit_config](https://cdn.illacloud.com/official-website/img/official-site/components/bar_button_submit_config.jpg)

Ομοίως, μπορούμε να ρυθμίσουμε το κουμπί για την εμφάνιση προειδοποίησης όταν ο χρήστης προσπαθεί να υποβάλει μια μη ολοκληρωμένη φόρμα.

1.  Κάντε κλικ στο στοιχείο **`button`** για να ανοίξετε τη σελίδα ελέγχου. Κάτω από **`Χειριστής Συμβάντων`**, κάντε κλικ στο `**+ Νέο**`.
2.  Στο χειριστή συμβάντων επεξεργασίας, επιλέξτε **`Show notification`** in action, **`title`** "Error!" και **`description`** "Παρακαλώ συμπληρώστε τη φόρμα πριν την υποβολή". Επιλέξτε το `Error` για το **`type`**, βάλτε το `{{2000}}` για το **`duration`** (2 δευτερόλεπτα) και βάλτε το `{{barProgress1.value<100}}` για το **`Only run when`.**

![bar_button_show_config](https://cdn.illacloud.com/official-website/img/official-site/components/bar_button_show_config.jpg)

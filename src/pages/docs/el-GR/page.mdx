## <Heading hidden>Σελίδα</Heading>

Μια πολυσέλιδη επίδειξη χτισμένη στο ILLA: 

<img src='https://cdn.illacloud.com/official-website/img/docs/assemble/page1.gif' alt="" />

Μια εφαρμογή μιας σελίδας είναι μια εφαρμογή που μπορεί να αλλάξει σελίδες χωρίς επαναφόρτωση όλων των στοιχείων στη σελίδα. Οι περισσότεροι πόροι (συμπεριλαμβανομένων των HTML, CSS και Script) φορτώνονται μόνο μία φορά σε όλο τον κύκλο ζωής της εφαρμογής, κάνοντας μια εφαρμογή μιας σελίδας γρήγορα.

<img src='https://cdn.illacloud.com/official-website/img/docs/assemble/page2.png' alt="" />

Σε μια εφαρμογή πολλαπλών σελίδων, κάθε αλλαγή, όπως η εμφάνιση των δεδομένων ή η υποβολή δεδομένων πίσω στα αιτήματα του διακομιστή που αποδίδουν μια νέα σελίδα από το διακομιστή. Η εφαρμογή πολλαπλών σελίδων έχει μερικά επίπεδα, γεγονός που καθιστά τη δομή της εφαρμογής απλή και απλή για τους χρήστες.

<img src='https://cdn.illacloud.com/official-website/img/docs/assemble/page3.png' alt="" />

Στο ILLA Builder, μπορεί να κατασκευαστεί μια εφαρμογή μιας σελίδας ρυθμίζοντας το πλαίσιο της σελίδας και προσθέτοντας προβολές σε τμήματα, και μια εφαρμογή πολλαπλών σελίδων μπορεί να κατασκευαστεί με την προσθήκη πολλαπλών σελίδων, και οι προγραμματιστές μπορούν να χρησιμοποιήσουν και τα δύο αυτά μοτίβα σχεδιασμού σε μία εφαρμογή.

<img src='https://cdn.illacloud.com/official-website/img/docs/assemble/page4.png' alt="" />

### Ορίστε τη διάταξη των σελίδων

Μεταβείτε στη ρύθμιση παραμέτρων της σελίδας ως τα ακόλουθα βήματα：

<img src='https://cdn.illacloud.com/official-website/img/docs/assemble/page5.png' alt="" />

Μπορείτε να χρησιμοποιήσετε τις προκαθορισμένες διατάξεις ή χειροκίνητα να προσθέσετε και να διαγράψετε ενότητες για να ορίσετε τη διάταξη. 

<img src='https://cdn.illacloud.com/official-website/img/docs/assemble/page6.gif' alt="" />

### Προσθήκη προβολών σε μια σελίδα

`View path`: μια συμβολοσειρά μετά το URL της σελίδας για να έχει πρόσβαση στην καθορισμένη προβολή. Μετά την ανάπτυξη της εφαρμογής, η πλήρης διεύθυνση πρόσβασης έχει ως εξής: https://builder.illacloud.com/{{team_identifier}}/deploy/app/{{app_id}}/{{page_name}}/{{view_path}}

Μπορείτε επίσης να εισάγετε την αρχική σελίδα και την προεπιλεγμένη προβολή μέσω της ακόλουθης διαδρομής:

https://builder.illacloud.com/{{team_identifier}}/deploy/app/{{app_id}}. 

<img src='https://cdn.illacloud.com/official-website/img/docs/assemble/page7.png' alt="" />

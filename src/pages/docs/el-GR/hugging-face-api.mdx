## <h2 hidden>Καλύπτει το Πρόσωπο API</h2>

Hugging Face είναι το Github της κοινότητας μηχανικής μάθησης, με εκατοντάδες χιλιάδες προ-εκπαιδευμένα μοντέλα και 10.000 σύνολα δεδομένων που είναι διαθέσιμα σήμερα. Μπορείτε να έχετε ελεύθερα πρόσβαση σε μοντέλα και σύνολα δεδομένων που μοιράζονται άλλοι ειδικοί του κλάδου ή φιλοξενούν και να αναπτύξετε τα μοντέλα σας σε Hugging Face.

Μερικά παραδείγματα μοντέλων που διατίθενται στη βιβλιοθήκη Hugging Face περιλαμβάνουν:

1.  BERT (Bidirectional Encoder Representations from Transformers): Το BERT είναι ένα μοντέλο βασισμένο στον μετασχηματιστή που αναπτύχθηκε από την Google για διάφορες εργασίες NLP. Έχει επιτύχει υπερσύγχρονα αποτελέσματα στην κατανόηση γλωσσών και μεταφραστικών εργασιών.
2.  GPT (Generative Pre-training Transformer): GPT είναι ένα άλλο μοντέλο βασισμένο στον μετασχηματιστή που αναπτύχθηκε από το OpenAI. Χρησιμοποιείται κυρίως για εργασίες δημιουργίας γλωσσών, όπως η μετάφραση και η σύνοψη κειμένου.
3.  RoBERTa (Robustly Optimized BERT): RoBERTa είναι μια επέκταση του μοντέλου BERT που αναπτύχθηκε για τη βελτίωση της απόδοσης του BERT σε διάφορες εργασίες NLP.
4.  XLNet (eXtraordinary LanguageNet): Το XLNet είναι ένα μοντέλο βασισμένο στον μετασχηματιστή που αναπτύχθηκε από την Google και έχει σχεδιαστεί για τη βελτίωση της απόδοσης των μοντέλων μετασχηματιστή σε εργασίες κατανόησης γλωσσών.
5.  ALBERT (A Lite BERT): ALBERT είναι μια έκδοση του μοντέλου BERT που αναπτύχθηκε για να είναι πιο αποτελεσματική και ταχύτερη για να εκπαιδεύσει διατηρώντας παράλληλα καλή απόδοση σε εργασίες NLP.

### Τι μπορείτε να κάνετε με το Hugging Face in ILLA Builder

Στο Hugging Face, πάνω από 130.000 μοντέλα μάθησης μηχανημάτων διατίθενται μέσω του δημόσιου API, το οποίο μπορείτε να χρησιμοποιήσετε και να δοκιμάσετε δωρεάν στο <https://huggingface.co/models>. Επιπλέον, αν χρειάζεστε μια λύση για σενάρια παραγωγής, μπορείτε να χρησιμοποιήσετε τα σημεία εισόδου Hugging Face, τα οποία μπορούν να αναπτυχθούν και να προσπελαστούν στο <https://huggingface.co/docs/inference-endpoints/index>.

Το ILLA Builder παρέχει δεκάδες κοινά χρησιμοποιούμενα εξαρτήματα front-end, επιτρέποντάς σας να κατασκευάσετε διαφορετικές διεπαφές πρόσοψης ανάλογα με τις συγκεκριμένες ανάγκες σας γρήγορα. Ταυτόχρονα, η ILLA προσφέρει μια σύνδεση στο Hugging Face, επιτρέποντάς σας να συνδεθείτε γρήγορα με το API, να στείλετε αιτήματα και να λάβετε επιστρεφόμενα δεδομένα. Συνδέοντας τα στοιχεία API και front-end, μπορείτε να εφαρμόσετε την απαίτηση ότι οι χρήστες μπορούν να εισέλθουν στο περιεχόμενο μέσα από το μπροστινό άκρο και να το υποβάλουν στο API. Το API επιστρέφει το παραγόμενο περιεχόμενο που θα εμφανίζεται στο μπροστινό μέρος.

### Ρύθμιση πόρου Hugging Face API

| Ιδιότητες  | Απαιτείται | Περιγραφή                                                                                         |
| ---------- | ---------- | ------------------------------------------------------------------------------------------------- |
| Όνομα      | απαιτείται | Ορίστε ένα όνομα πόρου που θα χρησιμοποιηθεί για την εμφάνιση στο ILLA                            |
| Διακριτικό | απαιτείται | Η πρόσβαση χρήστη ή API token. Μπορείτε να το βρείτε στο https://huggingface.co/settings/tokens.  |

### Ρύθμιση Ενέργειας

| Ιδιότητες             | Απαιτείται | Περιγραφή                                                                                                                                                                                                                                                                                               |
| --------------------- | ---------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Αναγνωριστικό Μοντέλο | απαιτείται | Αναζήτηση μοντέλων: https://huggingface.co/models                                                                                                                                                                                                                                                       |
| Τύπος παραμέτρου      | απαιτείται | Ο τύπος παραμέτρου του τελικού σας σημείου. Για παράδειγμα, αν το τελικό σας σημείο χρειάζεται μια εισαγωγή κειμένου, επιλέξτε συμπληρώστε την παράμετρο "inputs" με κείμενο. Αν το τελικό σημείο σας χρειάζεται μια είσοδο JSON, επιλέξτε συμπληρώστε την παράμετρο "εισροές" με JSON ή τιμή κλειδιού. |
| Παράμετρος            | απαιτείται | Εισάγετε την παράμετρο σας. Χρησιμοποιήστε {{ componentName.value }} για να χρησιμοποιήσετε τα δεδομένα των εφαρμογών.                                                                                                                                                                                  |

# Πώς να χρησιμοποιήσετε Hugging Πρόσωπο σε ILLA Builder

## Usecase 1

### Βήμα 1: Κατασκευάστε το UI με εξαρτήματα σε ILLA Builder

Εδώ θα δείξουμε πώς να οικοδομήσουμε ένα απλό κείμενο που εισάγει το κείμενο προτροπή και μια ερώτηση κειμένου για την έξοδο μιας απάντησης κειμένου. 

Χρειάζεστε δύο συστατικά textarea με την ονομασία "Enter the original text" και "Enter the question here", και στοιχείο κουμπιού με την ένδειξη "Εκτέλεση" και ένα στοιχείο περιοχής κειμένου με την ένδειξη "Anwer" για το κείμενο εξόδου. Η παρακάτω εικόνα είναι ένα παράδειγμα όπως περιγράφεται παραπάνω.

![hfapi](https://cdn.illacloud.com/official-website/img/docs/connect/hfapi1.png)

### Βήμα 2: Δημιουργήστε έναν μαγευτικό Πόρο Προσώπου και ρυθμίστε τις Ενέργειες

Κάντε κλικ στο κουμπί + Νέο στη λίστα ενεργειών και επιλέξτε Api πρόσδεσης Προσώπου Hugging.

![hfapi2](https://cdn.illacloud.com/official-website/img/docs/connect/hfapi2.png)

Συμπληρώστε τη φόρμα για να συνδεθείτε στο Hugging Face:

Όνομα: Το όνομα που θα εμφανίζεται στην ILLA

Token: Αποκτήστε στο μαγευτικό σας πρόσωπο [ρυθμίσεις προφίλ](https://huggingface.co/settings/tokens)

![hfapi3](https://cdn.illacloud.com/official-website/img/docs/connect/hfapi3.png)

Επιβεβαιώστε τις πληροφορίες μοντέλου στο Hugging Face πριν ρυθμίσετε τις ενέργειες:

Επιλέξτε ένα μοντέλο σε [Hugging Face Model Page](https://huggingface.co/models)

Ας χρησιμοποιήσουμε για παράδειγμα το [deepset/roberta-base-squad2](https://huggingface.co/deepset/roberta-base-squad2). Εισάγετε τη σελίδα λεπτομερειών > κάντε κλικ στο κουμπί Ανάπτυξη > Κάντε κλικ στο API εισαγωγής

![hfapi4](https://cdn.illacloud.com/official-website/img/docs/connect/hfapi4.png)

Οι παράμετροι μετά το `"εισροές"` είναι αυτό που πρέπει να συμπληρώσετε το ILLA. 

![hfapi5](https://cdn.illacloud.com/official-website/img/docs/connect/hfapi5.png)

Στο ILLA Builder, θα πρέπει να συμπληρώσετε το πρότυπο ID και την παράμετρο. Λαμβάνοντας ως παράδειγμα το παραπάνω μοντέλο, το `“inputs”` είναι ένα ζευγάρι αξίας κλειδιού, ώστε να μπορούμε να το συμπληρώσουμε με τιμή κλειδιού ή JSON. 

![hfapi6](https://cdn.illacloud.com/official-website/img/docs/connect/hfapi6.png)

![hfapi7](https://cdn.illacloud.com/official-website/img/docs/connect/hfapi7.png)

Και υποστηρίζουμε επίσης την εισαγωγή κειμένου και δυαδική είσοδο που μπορεί να καλύψει όλες τις συνδέσεις Hugging Face Inference API.

### Βήμα 3: Συνδέστε τις ενέργειες με τα εξαρτήματα

Για να περάσει το front-end εισόδου του χρήστη στο API, μπορείτε να χρησιμοποιήσετε {„για να ανακτήσετε δεδομένα που εισήχθησαν στο στοιχείο. Για παράδειγμα, η είσοδος2 είναι το συστατικό στοιχείο για την εισαγωγή της ερώτησης και εισόδου1 είναι το στοιχείο στο πλαίσιο εισόδου, μπορούμε να συμπληρώσουμε το `question` και το `context` στο κλειδί και να συμπληρώσουμε το `{{ input.value }}` σε αξία:

```jsx
◆
"ερώτηση": {{input2.value}},
"περιβάλλον": {{input1.value}}
}
```

Πριν από την εμφάνιση των δεδομένων εξόδου της Δράσης στο στοιχείο front-end, θα πρέπει να επιβεβαιώσουμε σε ποιο πεδίο τοποθετείται η έξοδος των διαφορετικών μοντέλων. Ακόμα να παίρνετε ως παράδειγμα το `deepset/roberta-base-squad2`, τα αποτελέσματα είναι τα εξής:

![hfapi8](https://cdn.illacloud.com/official-website/img/docs/connect/hfapi8.png)

Έτσι, μπορούμε να πάρουμε την απάντηση με το `{{ textQuestion.data[0].answer }}`(το `textQuestion` είναι το όνομα της ενέργειας). 

![hfapi9](https://cdn.illacloud.com/official-website/img/docs/connect/hfapi9.png)

## Usecase 2

Εδώ θα δείξουμε πώς να χρησιμοποιήσετε το μοντέλο σταθερής διάχυσης thorugh το API Προσώπου Hugging στο Illa Cloud.

**Βήμα 1: Χτίζοντας μια Διεπαφή Προσώπου**

Κατασκευάζουμε ένα front-end interface χρησιμοποιώντας μια προσέγγιση drag-and-drop για να τοποθετήσετε βασικά στοιχεία, όπως πεδία εισαγωγής, κουμπιά, εικόνες και πολλά άλλα. Αφού ρυθμίσουμε τα στυλ των εξαρτημάτων, θα λάβουμε την ακόλουθη πλήρη ιστοσελίδα.

![hugging_layout](https://cdn.illacloud.com/official-website/img/official-site/components/hugging_layout.png)

**Βήμα 2: Δημιουργία Πόρων και Δράσεων**

Δημιουργούμε πόρους και ενέργειες χρησιμοποιώντας το API πρόσληψης προσώπου Hugging για να συνδεθείτε στο μοντέλο Stable Diffusion. Μπορούν να χρησιμοποιηθούν δύο μοντέλα: `runwayml/stable-diffusion-v1-5` και `stabilityai/stable-diffusion-2-1`.

Επιλέξτε το "Hugging Face Inference API" για το σκοπό αυτό.

![action_list](https://cdn.illacloud.com/official-website/img/official-site/components/action_list.png)

Δώστε ένα όνομα για αυτόν τον πόρο και εισάγετε το διακριτικό σας από την πλατφόρμα Hugging Face .

![hugging_config](https://cdn.illacloud.com/official-website/img/official-site/components/hugging_config.png)

Στον πίνακα ρυθμίσεων Ενέργεια, παρακαλώ εισάγετε το Αναγνωριστικό μοντέλου και την Παράμετρο. Θα ανακτήσουμε το επιλεγμένο μοντέλο από το radioGroup1, οπότε συμπληρώστε το Model ID ως `{{radioGroup1.value}}`. Για την είσοδο, δεδομένου ότι λαμβάνεται από το πεδίο εισαγωγής, συμπληρώστε την παράμετρο ως `{{input1.value}}`. Η διαμόρφωση πρέπει να είναι όπως φαίνεται στην παρακάτω εικόνα.

![hugging_demo](https://cdn.illacloud.com/official-website/img/official-site/components/hugging_demo.png)

Προσπαθούμε να εισάγουμε "Ένα ρομπότ mecha σε ένα φαβέλα σε εξπρεσιονιστικό στυλ" στο στοιχείο `input` και να εκτελέσουμε τη Δράση. Η εκτέλεση που προκύπτει είναι η ακόλουθη. Από τον αριστερό πίνακα, μπορείτε να δείτε τα διαθέσιμα δεδομένα που μπορούν να κληθούν, συμπεριλαμβανομένων των `base64binary` και `dataURI`.

![hugging_output](https://cdn.illacloud.com/official-website/img/official-site/components/hugging_output.png)

**Βήμα 3: Εμφάνιση δεδομένων στα στοιχεία**

Για να εμφανίσετε την εικόνα που λαμβάνεται από το Βήμα 2, τροποποιούμε την πηγή εικόνας του στοιχείου εικόνα σε `{{generateInput.fileData.dataURI}}`. Αυτό θα μας επιτρέψει να εμφανίσουμε την παραγόμενη εικόνα.

![hugging_display](https://cdn.illacloud.com/official-website/img/official-site/components/hugging_display.png)

**Βήμα 4: Εκτέλεση της ενέργειας με στοιχεία**

Για να εκτελέσετε την ενέργεια που δημιουργήθηκε στο Βήμα 2 όταν πατηθεί το εξάρτημα πλήκτρου, προσθέστε ένα χειριστή γεγονότων στο στοιχείο πλήκτρο.

![hugging_event_config](https://cdn.illacloud.com/official-website/img/official-site/components/hugging_event_config.png)

**Βήμα 5: Δοκιμή**

Ακολουθώντας τα προηγούμενα τέσσερα βήματα, μπορείτε να χρησιμοποιήσετε επιπλέον στοιχεία και πηγές δεδομένων για να ολοκληρώσετε άλλες εργασίες και να δημιουργήσετε ένα πιο ολοκληρωμένο εργαλείο. Για παράδειγμα, μπορείτε να χρησιμοποιήσετε άλλα μοντέλα για να βοηθήσετε στη δημιουργία προτροπών ή την αποθήκευση προτροπών στην τοπική αποθήκευση ή σε μια βάση δεδομένων. Ας ρίξουμε μια ματιά στο πλήρες αποτέλεσμα όταν όλα τα βήματα υλοποιούνται.

![hugging_test](https://www.youtube.com/watch?v=SwAry_jIXns)

Τώρα μπορείτε να παίξετε με αυτό! Δοκιμάστε άλλα μοντέλα αιχμής που παρέχονται από Hugging πρόσωπο μέσω αυτού του API. Υπάρχουν περισσότερα για να εξερευνήσετε, όπως σταθερή μετάδοση μοντέλα anime download, την κατάρτιση ενός στυλ καλλιτέχνη, τη μέθοδο δειγματοληψίας για ρεαλιστικές εικόνες.

## <Heading hidden>Δοχείο</Heading>

Μπορείτε να ομαδοποιήσετε τα στοιχεία που σχετίζονται μαζί σε μια κάρτα χρησιμοποιώντας το στοιχείο Container. Τα εμπορευματοκιβώτια υποστηρίζουν επίσης πολλαπλές απόψεις που σας επιτρέπουν να διαχωρίσετε περιεχόμενο σε ξεχωριστές απόψεις που οι χρήστες μπορούν να αλλάξουν μεταξύ τους.

Τα εμπορευματοκιβώτια λειτουργούν όπως κάθε άλλο στοιχείο. Μόλις προστεθεί, μπορείτε να ομαδοποιήσετε συστατικά μέσα σε αυτό πατώντας D ή K και σύροντάς τα πάνω στο δοχείο.

### Πολλαπλές προβολές

Τα εμπορευματοκιβώτια υποστηρίζουν πολλαπλές απόψεις για ομαδοποίηση περιεχομένου. Μπορείτε να ορίσετε τις προβολές στην ενότητα **Προβολές** της καρτέλας **Επιθεώρηση**. Μόνο μια προβολή είναι ορατή σε μια στιγμή και οι χρήστες εναλλάσσονται μεταξύ τους αλλάζοντας την τρέχουσα προβολή του στοιχείου Container.

### Χρησιμοποιήστε άλλα στοιχεία για να αλλάξετε προβολές

Μπορείτε να χρησιμοποιήσετε άλλα στοιχεία για τον έλεγχο των εμπορευματοκιβωτίων με την προσθήκη ενός χειριστή συμβάντων σε άλλα εξαρτήματα. Υποστηρίζονται οι ακόλουθες μέθοδοι:

1.  Ορισμός τρέχουσας προβολής με βάση το κλειδί
2.  Ορισμός τρέχουσας προβολής βάσει ευρετηρίου
3.  Εμφάνιση της επόμενης (ορατής) προβολής
4.  Εμφάνιση της προηγούμενης (ορατής) προβολής

### Συνδέστε μια καρτέλα σε ένα κοντέινερ

Μπορείτε να χρησιμοποιήσετε ένα στοιχείο καρτέλας για τον έλεγχο ενός εμπορευματοκιβωτίου ενεργοποιώντας το **Link σε ένα εμπορευματοκιβώτιο** της καρτέλας **Inspect** .

Μετά τη σύνδεση με ένα δοχείο, οι ετικέτες των καρτελών θα αλλάξουν στις ετικέτες των όψεων. Μπορείτε να αλλάξετε προβολές κάνοντας κλικ στις καρτέλες χωρίς χειριστές γεγονότων.

## <h2 hidden>Supabase DB</h2>

Το Supabase είναι μια εναλλακτική λύση Firebase ανοιχτού κώδικα που επιτρέπει στους προγραμματιστές να κατασκευάζουν σε πραγματικό χρόνο web και mobile εφαρμογές. Παρέχει πολλά από τα ίδια χαρακτηριστικά με το Firebase, όπως βάσεις δεδομένων σε πραγματικό χρόνο, έλεγχο ταυτότητας χρήστη, και φιλοξενία, αλλά με τα πρόσθετα οφέλη του να είναι ανοιχτού κώδικα και την παροχή ενός SQL API. Αυτό επιτρέπει στους προγραμματιστές να χρησιμοποιούν τις υπάρχουσες γνώσεις και τα εργαλεία SQL τους, καθιστώντας εύκολη την ενσωμάτωση με άλλα συστήματα. Επιπλέον, Supabase παρέχει μια πιο ευέλικτη και προσαρμόσιμη πλατφόρμα από Firebase, η οποία μπορεί να είναι πιο περιοριστική όσον αφορά τη μοντελοποίηση και τη χρήση δεδομένων.

Αυτό το σεμινάριο περιγράφει τη διαδικασία της δημιουργίας ενός Πίνακα Διαχειριστή χρησιμοποιώντας ILLA Builder και Supabase σε μερικά απλά βήματα. Το ILLA είναι μια πλατφόρμα χαμηλού κώδικα για προγραμματιστές που επιτρέπει την ταχεία ανάπτυξη και ανάπτυξη εσωτερικών εργαλείων. Επιτρέπει τη δημιουργία σελίδων σύροντας και απορρίπτοντας στοιχεία UI, που συνδέονται με οποιαδήποτε βάση δεδομένων ή API, και γράφοντας JavaScript. Για να μάθετε περισσότερα σχετικά με το Supabase, επισκεφθείτε την ιστοσελίδα τους στο <https://supabase.com/>. Ας ξεκινήσουμε!

Συμπληρώστε τη φόρμα για να συνδεθείτε στην παρουσία Supabase. Δοκιμή σύνδεσης και αποθήκευση πόρου.

<img

Κάντε κλικ στο κουμπί `Create Action` για να δημιουργήσετε μια ενέργεια με τον πόρο Supabase και να ρυθμίσετε το CRUD σας.

<img
  src='https://cdn.illacloud.com/official-website/img/docs/assemble/supabase9.png'
  alt=""
/>

Χρησιμοποίησε {Şos. Για να πάρεις τα δεδομένα εισόδου στο μπροστινό μέρος. Το παρακάτω είναι ένα παράδειγμα της σελίδας Διαχείρισης Χρηστών στο Admin Panel.

Αναζήτηση για ένα χρήστη με το όνομα που εισάγεται στο input1

```
SELECT *
FROM user
WHERE name = "{{input1.value}}"
;
```

Ενημέρωση δεδομένων χρήστη. Ενημέρωση πληροφοριών χρήστη όταν το id ταιριάζει

```
UPDATE user
SET name = "{{input3.value}}"
, email = "{{input4.value}}"
WHERE id="{{input2.value}}"
;
```

Εισαγωγή δεδομένων χρήστη

```
ΕΙΣΑΓΩΓΗ ΣΤΙΣ ΤΙΜΕΣ Χρηστών ("{{input5.value}}","{{input6.value}}","{{input7.value}}");
```

Διαγραφή ενός χρήστη με id

```
ΔΙΑΓΡΑΦΗ ΑΠΟ το χρήστη WHERE id = "{{input2.value}}";
```

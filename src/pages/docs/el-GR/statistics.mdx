## <h2 hidden>Τι είναι Στατιστικά?</h2>

Το στοιχείο Στατιστικά στο ILLA Cloud είναι ένα ισχυρό εργαλείο για την εμφάνιση και ανάλυση αριθμητικών δεδομένων. Παρέχει έναν οπτικά ελκυστικό και προσαρμόσιμο τρόπο για να παρουσιάσει βασικές μετρήσεις και στατιστικές.

### Μέθοδος

Μπορείτε να χρησιμοποιήσετε άλλα στοιχεία για τον έλεγχο της εφαρμογής. Υποστηρίζουμε τις ακόλουθες δύο μεθόδους:

-   **setPrimaryvalue**

Για να ορίσετε την κύρια τιμή, για παράδειγμα, {{"value1"}}

| Ιδιότητες       | Περιγραφή               |
| --------------- | ----------------------- |
| Πρωτεύουσα Τιμή | Πρωταρχική τιμή εισόδου |

-   **resetPrimaryValue**

Για να επαναφέρετε την τιμή στην προεπιλεγμένη τιμή της επιλεγμένης εφαρμογής. Εάν η προεπιλεγμένη τιμή δεν έχει καθοριστεί, καθαρίστε την τιμή.

### Περίπτωση χρήσης

Στη συνέχεια, θα δείξουμε πώς να αντιστοιχίσουμε τα δεδομένα από την πηγή δεδομένων στα στατιστικά με ένα κουμπί.

#### Βήμα 1 Προσθέστε μια ενέργεια

Ας δημιουργήσουμε ένα τραπέζι στο Amazon S3. Ένα από τα έγγραφα ονομάζεται

'1.txt' με το δεκαδικό '1.1111' ως δεδομένα του. Αυτή είναι η τιμή που θέλουμε να εμφανίσουμε στην είσοδο αριθμού.

![s3data](https://cdn.illacloud.com/official-website/img/official-site/components/s3data.jpg)

Στη συνέχεια, μπορούμε να δημιουργήσουμε μια νέα δράση για το Amazon S3 από τη λίστα δράσης, λίστα όλων των δεδομένων στον κάδο και ονομάζεται s31.

Επιλέξτε `Ανάγνωση αντικειμένου` για `**Είδος Ενέργειας**`. Βάλτε το `1.txt` για το `**Object Key`.\*\*

Πατήστε **`Save`** και **`Run`** για να ενεργοποιήσετε αυτήν την ενέργεια.

![stat_data](https://cdn.illacloud.com/official-website/img/official-site/components/stat_data.jpg)

#### Βήμα 2 Προσθήκη Συστατικών

Στη συνέχεια, μπορούμε να προσθέσουμε ένα **`statistics`** component και ένα **`button`** component στον καμβά.

Ορίζουμε την προεπιλεγμένη κύρια τιμή των στατιστικών στο 0 και επισημαίνουμε το κουμπί ως "Ορισμός τιμής" όπως φαίνεται παρακάτω

![stat_layout](https://cdn.illacloud.com/official-website/img/official-site/components/stat_layout.jpg)

#### Βήμα 3 Διαμόρφωση του εξαρτήματος

Για το στοιχείο **`button`** μπορούμε να το ρυθμίσουμε ώστε να ορίζει την τιμή στο στοιχείο **`statistics`** στα δεδομένα του εγγράφου που διαβάζουμε από το api του s31.

1.  Στο χειριστή συμβάντων επεξεργασίας, επιλέξτε **`Control component`** εν ενεργεία, και επιλέξτε το **`statistics`** component που θέλετε να ενημερώσετε ως στόχο του γεγονότος.
2.  Επιλέξτε την **`setValue`** ενέργεια και ορίστε την τιμή ως `{{s31.data}}`

![stat_button_config](https://cdn.illacloud.com/official-website/img/official-site/components/stat_config.jpg)

#### 4Ο Στάδιο Δοκιμή

Τώρα που όταν κάνετε κλικ στο κουμπί "Ορισμός τιμής", η τιμή θα πρέπει να αλλάξει σε 1.1111.

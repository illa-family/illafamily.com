## <h2 hidden>Βάση δεδομένων Appwrite</h2>

Μέσα στις διαφορετικές υπηρεσίες του Appwrite, μπορείτε να βρείτε έλεγχο ταυτότητας χρήστη και διαχείριση λογαριασμού, προτιμήσεις χρηστών, βάση δεδομένων και αποθήκευση επιμονή, λειτουργίες σύννεφο, τοπικοποίηση, χειρισμό εικόνας, και πολλά άλλα. 

Ο συνδυασμός της ILLA Cloud και Appwrite, δηλαδή, ο συνδυασμός του server-side χαμηλού κώδικα και του front-end χαμηλού κώδικα, θα πραγματοποιήσει πραγματικά την ανάπτυξη πλήρους stack χαμηλού κώδικα.

[Κάντε κλικ εδώ](https://appwrite.io/) για να μάθετε περισσότερα σχετικά με το Appwrite.

### Ρύθμιση πόρου εγγραφής εφαρμογής στο ILLA Cloud

#### Δύο τρόποι για να δημιουργήσετε πόρους

Ο πρώτος τρόπος είναι να εισαγάγετε την καρτέλα `Resources` > κάντε κλικ στο κουμπί `Create New` για να επιλέξετε Appwrite. 

<img src='https://cdn.illacloud.com/official-website/img/docs/assemble/appwrite1.png' alt="" />

Ο άλλος τρόπος είναι να εισαγάγετε τη σελίδα `app editor` > κάντε κλικ στο κουμπί `+ New` για να επιλέξετε Appwrite > κάντε κλικ στο κουμπί `+ Νέοι πόροι` για να προσθέσετε τις νέες ρυθμίσεις. 

<img src='https://cdn.illacloud.com/official-website/img/docs/assemble/appwrite2.png' alt="" />

#### Ρύθμιση Appwrite συμπληρώνοντας τα ακόλουθα πεδία

Πριν συνδέσετε έναν πόρο Appwriting, βεβαιωθείτε ότι έχετε αναπτύξει Appwrite στον τοπικό σας υπολογιστή ή έναν πάροχο cloud. [Κάντε κλικ εδώ](https://appwrite.io/docs/installation) για να μάθετε περισσότερα σχετικά με την εγκατάσταση του Appwrite.

### Δημιουργία ενεργειών στον επεξεργαστή εφαρμογών

#### Δημιουργία ενεργειών με τον ακόλουθο τρόπο

Κάντε κλικ στο κουμπί `+ New` στη σελίδα επεξεργαστή εφαρμογών> για να επιλέξετε Appwrite > επιλέξτε έναν υπάρχοντα πόρο Appwrite ή δημιουργήστε έναν νέο πόρο > κάντε κλικ στο κουμπί `Create Action` για να δημιουργήσετε μια νέα ενέργεια. 

<img src='https://cdn.illacloud.com/official-website/img/docs/assemble/appwrite4.png' alt="" />

Μετά τη δημιουργία μιας ενέργειας, παρακαλώ επιλέξτε μια μέθοδο. Παρέχουμε τις ακόλουθες μεθόδους: Λίστα εγγράφων, Δημιουργία εγγράφου, Ενημέρωση εγγράφου, Λήψη εγγράφου και Διαγραφή εγγράφου. Η διαμόρφωση κάθε μεθόδου είναι η ακόλουθη. 

#### Λίστα εγγράφων

Το παρακάτω είναι ένα απλό παράδειγμα διαμόρφωσης: Καταχωρίστε το έγγραφο με $id 5. 

<img src='https://cdn.illacloud.com/official-website/img/docs/assemble/appwrite5.png' alt="" />

| Ιδιότητα        | απαιτείται\_απαιτείται | Περιγραφή                                                                                                                                                                                                                                                                                 |

| Id Συλλογής     | Απαιτείται             | Εισαγωγή ή επιλογή ενός id συλλογής από τη βάση δεδομένων                                                                                                                                                                                                                                 |
| Φίλτρο          | Προαιρετικό            | Υποστήριξη πολλαπλών φίλτρων. Ένα φίλτρο αποτελείται από ένα κλειδί, μια τιμή και μια κατάσταση. 1. κλειδί: εισάγετε ένα χαρακτηριστικό 2. συνθήκη: επιλέξτε μια προϋπόθεση, συμπεριλαμβανομένων των ίσος, όχιίσος, lessThan, lessThanEqual, greaterThan, κλπ. 3. τιμή: εισάγετε την τιμή |
| Ταξινόμηση κατά | Προαιρετικό            | Υποστήριξη για την παραγγελία με πολλαπλά χαρακτηριστικά. 1. εισάγετε ένα χαρακτηριστικό για να παραγγείλετε κατά 2. Ακολουθεί ένα παράδειγμα του τρόπου χρήσης των δεδομένων που εισάγονται στο στοιχείο στην ενέργεια:

1.  Προσθέστε ένα στοιχείο εισόδου στο πλαίσιο, έτσι ώστε οι χρήστες να μπορούν να εισαγάγουν το αναγνωριστικό εγγράφου που θέλουν να βρουν.  
2.  Προσθέστε ένα φίλτρο: ορίστε το πεδίο χαρακτηριστικού σε $id και εισάγετε {{ input1.value }} στο πεδίο τιμής. 

<img src='https://cdn.illacloud.com/official-website/img/docs/assemble/appwrite6.png' alt="" />

Έξοδος

Ελέγξτε τη δομή δεδομένων στον αριστερό πίνακα. Για παράδειγμα, χρησιμοποιήστε το `{{list.data[0].documents[0].id}}`(`list` είναι το όνομα της ενέργειας) για να λάβετε το αναγνωριστικό του πρώτου εγγράφου. 

<img src='https://cdn.illacloud.com/official-website/img/docs/assemble/appwrite7.png' alt="" />

Μπορείτε επίσης να χρησιμοποιήσετε το `{{list.data[0].documents.map(documents => documents)}}` για να εμφανίσετε όλα τα δεδομένα στο στοιχείο `table`.

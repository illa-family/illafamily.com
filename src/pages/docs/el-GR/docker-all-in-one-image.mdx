## <Heading hidden>Εικόνα Docker όλα-σε-ένα</Heading>

### Περιγραφή

Φτιάξτε illa ωφελεί μια εικόνα όλα-σε-ένα και να τρέξει στο μηχάνημά σας. Μπορείτε να δείτε το αρχείο σεναρίων στο φάκελο [scripts](https://github.com/illacloud/deploy-illa-manually/blob/main/docker/scripts) για περισσότερες λεπτομέρειες.

Σημείωση:

Προτείνουμε την ανάπτυξη με τα εργαλεία αυτόματης ανάπτυξης[illa-cli](https://github.com/illacloud/illa).

Και προς το παρόν δεν υποστηρίζουμε το Apple Silicon M1 (darwin-arm64 arch).

### Εκτέλεση επίσημης εικόνας

Εγκατάσταση του GNU make και Τύπος:

```jsx
δημιουργία run-official-image
```

ή απλά εκτελέστε:

```jsx
/bin/bash scripts/run-official-image.sh
```

αυτή η εντολή θα τραβήξει illasoft επίσημη εικόνα και θα την τρέξει στο περιβάλλον docker σας.

Και συνδεθείτε με το προεπιλεγμένο όνομα χρήστη **`root`** και τον κωδικό πρόσβασης **`password`**.

### Κατασκευή εικόνας χειροκίνητα και εκτέλεση

Εγκατάσταση GNU make (ή εκτέλεση σεναρίων κελύφους στο φάκελο σεναρίων χειροκίνητα).

Για την κατασκευή της εικόνας και τρέξιμο της illa όλα-σε-ένα:

```jsx
φτιάξε όλα
```

για το podman, τύπος:

```jsx
make run-by-podman
```

ή να εκτελέσει σενάρια κελύφους `run-by-podman.sh` σε φάκελο σεναρίων.

### Διακοπή και αφαίρεση περιέκτη

```jsx
κάνε stop-and-remove-container
```

### Καθαρισμός Αρχείου Βάσης Δεδομένων Postgres

** Η ΠΑΡΟΥΣΑ ΕΠΙΤΡΟΠΗ ΘΑ ΔΙΑΡΚΕΙΤΑΙ ΤΟ ΑΡΘΡΟ ΔΕΔΟΜΕΝΩΝ ΣΑΣ!!! \*\*\*** ΠΙΣΤΟΠΟΙΗΤΙΚΟ ΚΑΙ ΔΙΑΦΕΡΟΜΕΝΟ \*\*\*

Στην προεπιλογή, η βάση δεδομένων Postgres θα αποθηκευτεί στο φάκελο `/var/lib/illa/database/postgresql/`.

Εκτελέστε την ακόλουθη εντολή για να την διαγράψετε.

```jsx
δημιουργία αρχείου καθαρής-postgres-βάσης δεδομένων
```

### Μεταβλητές Περιβάλλοντος Εκτέλεσης Ρυθμίσεων

Επεξεργαστείτε το `scripts/run.sh` όταν χρειάζεται:

```jsx
# postgres password βάσης δεδομένων
PG_PASS=mysecretpassword

# repo home dir
ILLA_HOME_DIR=/var/lib/illa

# postgres φάκελος αποθήκευσης αρχείων βάσης δεδομένων
PG_VOLUMN=${ILLA_HOME_DIR}/database/postgresql
DRIVE_VOLUMN=${ILLA_HOME_DIR}/drive
```

### Ιδιότητες

| Ιδιότητα            | Περιγραφή                                                                                                                                                                                                                                   |
| ------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Περιεχόμενο         | Ορίστε το κείμενο που εμφανίζεται στην εφαρμογή Κείμενο. Υποστηρίζονται Markdown και απλό κείμενο.                                                                                                                                          |
| Υπόδειξη            | Μια υπόδειξη που εμφανίζεται στο στοιχείο όταν οι χρήστες αιωρούνται πάνω από το στοιχείο.                                                                                                                                                  |
| Οριζόντια στοίχιση  | Για να ορίσετε την οριζόντια θέση του περιεχομένου κειμένου στο στοιχείο                                                                                                                                                                    |
| Κατακόρυφη στοίχιση | Για να ορίσετε την κάθετη θέση του περιεχομένου κειμένου στο στοιχείο                                                                                                                                                                       |
| Hidden              | Για να ορίσετε αν θα εμφανίζεται αυτό το εξάρτημα και υπό ποιες συνθήκες θα εμφανίζεται. Για παράδειγμα, εάν ο τρέχων χρήστης δεν είναι χρήστης Α, αυτό το στοιχείο θα πρέπει να είναι κρυμμένο: {//currentUserInfo.nickname != 'User A' }} |
| Χρώμα               | Για να ορίσετε το χρώμα κειμένου σε μορφή χρώματος κειμένου                                                                                                                                                                                 |
| Μέγεθος κειμένου    | Για να ορίσετε το μέγεθος κειμένου σε μορφή κειμένου χρώματος                                                                                                                                                                               |

<img src='https://cdn.illacloud.com/official-website/img/docs/assemble/text2.png' alt="" />

```jsx

```


### Εμφάνιση δυναμικών δεδομένων

Χρήση `{{` για λήψη δεδομένων από άλλα στοιχεία ή ενέργειες ή καθολικές μεταβλητές. Για παράδειγμα, για να εμφανίσετε το ψευδώνυμο του τρέχοντος χρήστη:

```jsx
## {{👋 Γεια σας! '+ currentUserInfo.nickname }}
```

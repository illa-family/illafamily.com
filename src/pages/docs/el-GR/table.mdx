## <h2 hidden>Τι είναι ο πίνακας?</h2>

Το στοιχείο του πίνακα στο σύννεφο ILLA είναι ένα ισχυρό χαρακτηριστικό που επιτρέπει στους χρήστες να εμφανίζουν, να οργανώνουν και να χειρίζονται δεδομένα πίνακα μέσα στην εφαρμογή. Παρέχει μια δομημένη διάταξη με γραμμές και στήλες, επιτρέποντας στους χρήστες να βλέπουν και να αλληλεπιδρούν με τα δεδομένα με βολικό και δομημένο τρόπο.

### Ιδιότητες

| Ιδιότητες                               | Περιγραφή                                                                                                                                                       |
| --------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Πηγή δεδομένων                          | πηγή δεδομένων που συμπληρώνει τον πίνακα                                                                                                                       |
| Κενή κατάσταση                          | το περιεχόμενο ή το μήνυμα που εμφανίζεται όταν ο πίνακας δεν έχει δεδομένα για εμφάνιση                                                                        |
| Φόρτωση                                 | αν το στοιχείο του πίνακα βρίσκεται επί του παρόντος σε κατάσταση φόρτωσης                                                                                      |
| Στήλες                                  | δομή και διαμόρφωση των στηλών στον πίνακα. (Δείτε την ενότητα σχετικά με τις στήλες παρακάτω για περισσότερες λεπτομέρειες)                                    |
| Στήλη                                   | ο αριθμός των στηλών                                                                                                                                            |
| Επιλογή Πολλαπλών Γραμμών               | Χρήση {{table.selectedRow[n].columnName}} για πρόσβαση στα δεδομένα.                                                                                            |
| Επιλογή Κελιού                          | Αν θα επιτρέπεται στους χρήστες να επιλέγουν ένα κελί                                                                                                           |
| Κάντε κλικ έξω για να αποεπιλέξετε      | Υποστηρίζεται στη σειρά μονής λειτουργίας επιλογής. Αφού το ενεργοποιήσετε, κάντε κλικ σε μια περιοχή διαφορετική από τον πίνακα για να αποεπιλέξετε τη γραμμή. |
| Υπερχείλιση                             | πώς ο πίνακας χειρίζεται το περιεχόμενο που υπερβαίνει τον διαθέσιμο χώρο μέσα στο δοχείο του πίνακα (σελιδοποίηση ή κύλιση)                                    |
| Ενεργοποίηση σελιδοποίησης διακομιστή   | Αν θα ενεργοποιηθεί η σελιδοποίηση του διακομιστή                                                                                                               |
| Μέγεθος Σελίδας                         | ο αριθμός των γραμμών που θα εμφανίζονται ανά σελίδα όταν είναι ενεργοποιημένη η σελιδοποίηση στην πλευρά του διακομιστή                                        |
| Ανανέωση                                | Αν θα εμφανίζεται το εικονίδιο ανανέωσης.                                                                                                                       |
| Λήψη                                    | Εμφάνιση κουμπιού λήψης στη γραμμή εργαλείων.                                                                                                                   |
| Λήψη ακατέργαστων δεδομένων             | Αν θα εμφανίζεται το εικονίδιο λήψης ακατέργαστων δεδομένων.                                                                                                    |
| Φίλτρο                                  | Εμφάνιση κουμπιού φίλτρου στη γραμμή εργαλείων                                                                                                                  |
| Χειριστής συμβάντων                     | ανίχνευση και ανταπόκριση σε συγκεκριμένες ενέργειες χρήστη ή γεγονότα, όπως κλικ, πατήματα πλήκτρων ή υποβολή φόρμας.                                          |
| Απενεργοποιημένο                        | μη διαδραστικό και δεν μπορεί να τροποποιηθεί ή να ενεργοποιηθεί από το χρήστη.                                                                                 |

Για κάθε στήλη, μπορούμε να ρυθμίσουμε τις ιδιότητές του για να προσαρμόσουμε τα δεδομένα που εμφανίζονται για κάθε στήλη.

| Ιδιότητες                                               | Περιγραφή                                                                                                                                                                                                                                                          |
| ------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Τίτλος στήλης                                           | καθορίζει το όνομα ή την ετικέτα της στήλης, η οποία χρησιμεύει ως κεφαλίδα για την αναγνώριση των δεδομένων που αντιπροσωπεύει                                                                                                                                    |
| Τύπος στήλης                                            | καθορίζει τον τύπο των δεδομένων που περιέχει η στήλη, συμπεριλαμβανομένου του αυτόματου, κειμένου, της ημερομηνίας, της ημερομηνίας, της ημερομηνίας, αριθμός, τοις εκατό, σύνδεση, κουμπί, ομάδα, δυαδική ομάδα, εικόνα, εικονίδιο, αξιολόγηση, σήμανση, νόμισμα |
| Αντιστοιχούμενη Τιμή                                    | Χρήση {{ currentRow }} για πρόσβαση στα δεδομένα της τρέχουσας γραμμής. Για παράδειγμα, για να υπολογίσετε το "κόστος ανά κλικ" με βάση στήλες "κόστος" και "κλικTimes":                                                                                           |
| {mptomcurrentRow.cost / currentRow.clickTimes }}        |                                                                                                                                                                                                                                                                    |
| Ενεργοποίηση Ταξινόμησης                                | αν η στήλη επιτρέπει στους χρήστες να ταξινομήσουν τα δεδομένα του πίνακα με βάση τις τιμές της συγκεκριμένης στήλης                                                                                                                                               |
| Φόντο                                                   | Ορίστε το φόντο της στήλης. Χρησιμοποίησε {ņwrite μια υπό όρους δήλωση για να ορίσεις το χρώμα που θα εμφανίζεται από κάθε κελί υπό διαφορετικές συνθήκες.                                                                                                         |
| Ευθυγράμμιση                                            | ελέγχει την οριζόντια ευθυγράμμιση του περιεχομένου εντός των κελιών της στήλης                                                                                                                                                                                    |

#### Τύποι στηλών:

-   Αυτόματο

δυναμικό τύπο που καθορίζει αυτόματα τον τύπο δεδομένων με βάση τις τιμές εντός της στήλης

-   Κείμενο

αντιπροσωπεύει μια στήλη που περιέχει δεδομένα κειμένου

-   Ημερομηνία

αντιπροσωπεύει τιμές δεδομένων που αντιστοιχούν σε συγκεκριμένες ημερομηνίες

| Ιδιότητες | Περιγραφή                                                                         |
| --------- | --------------------------------------------------------------------------------- |
| Μορφή     | δέχεται διάφορους κωδικούς ή πρότυπα μορφοποίησης, η προεπιλογή είναι η ΕΕΕ-MM-ΗΗ |

-   Ετικέτα

εξειδικευμένη μορφή για την εμφάνιση ετικετών ή ετικετών που σχετίζονται με καταχωρήσεις δεδομένων

| Ιδιότητες         | Περιγραφή                                                                                     |
| ----------------- | --------------------------------------------------------------------------------------------- |
| Ετικέτα ετικέτας  | αντιπροσωπεύει το κείμενο ή την ετικέτα που έχει εκχωρηθεί σε μια ετικέτα στη στήλη "ετικέτα" |
| Χρώμα ετικέτας    | καθορίζει το χρώμα που έχει ανατεθεί στην ετικέτα στη στήλη "ετικέτα"                         |

-   Ώρα

απεικονίζει με ακρίβεια και χειρίζεται τις τιμές χρόνου εντός του πίνακα

| Ιδιότητες | Περιγραφή                                                                              |
| --------- | -------------------------------------------------------------------------------------- |
| Μορφή     | η οπτική αναπαράσταση των δεδομένων χρόνου εντός της στήλης, προεπιλογή είναι HH:mm:ss |

-   ΗμερομηνίαΏρα

αποθηκεύει και αντιπροσωπεύει τιμές ημερομηνίας και ώρας

| Ιδιότητες | Περιγραφή                                                                                                        |
| --------- | ---------------------------------------------------------------------------------------------------------------- |
| Μορφή     | καθορίζει τον τρόπο εμφάνισης των τιμών ημερομηνίας και ώρας στη στήλη, η προεπιλογή είναι ΕΕΕΕΕΕ-ΜΜ-ΗΗ HH:mm:ss |

-   Αριθμός

χειρισμός αριθμητικών δεδομένων

| Ιδιότητες                       | Περιγραφή                                                                                                                                    |
| ------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------- |
| Δεκαδικά ψηφία                  | καθορίστε τον αριθμό δεκαδικών ψηφίων που θα εμφανίζονται για τις αριθμητικές τιμές στη στήλη                                                |
| Εμφάνιση διαχωριστικού χιλιάδων | αν η στήλη πρέπει να εμφανίζει ένα διαχωριστικό χιλιάδων για να βελτιώσει την αναγνωσιμότητα και την ευκολία ερμηνείας για μεγάλους αριθμούς |

-   Ποσοστό

χειρίζεται τις ποσοστιαίες τιμές με συνοπτικό και οπτικά ελκυστικό τρόπο

| Ιδιότητες                       | Περιγραφή                                                                                        |
| ------------------------------- | ------------------------------------------------------------------------------------------------ |
| Δεκαδικά ψηφία                  | προσδιορίστε τον αριθμό δεκαδικών ψηφίων που θα εμφανίζονται για τις ποσοστιαίες τιμές στη στήλη |
| Εμφάνιση διαχωριστικού χιλιάδων | αν ένα διαχωριστικό χιλιάδων πρέπει να συμπεριληφθεί στις εμφανιζόμενες ποσοστιαίες τιμές        |

-   Σύνδεσμος

περιλαμβάνει συνδέσμους με δυνατότητα επιλογής σε μια συγκεκριμένη στήλη

-   Κουμπί

περιλαμβάνει διαδραστικά κουμπιά σε μια συγκεκριμένη στήλη

| Ιδιότητες             | Περιγραφή                                                                                    |
| --------------------- | -------------------------------------------------------------------------------------------- |
| Χειριστής συμβάντων   | ορίζει την ενέργεια ή το γεγονός που εμφανίζεται όταν πατηθεί το κουμπί μέσα στη στήλη       |
| Απενεργοποιημένο      | αν το κουμπί μέσα στη στήλη είναι ενεργοποιημένο ή απενεργοποιημένο                          |
| Χρώμα θέματος         | επιλέξτε ένα χρώμα θέματος για το κουμπί                                                     |
| Παραλλαγή             | οπτικό στυλ ή παραλλαγή του κουμπιού (συμπλήρωση ή περίγραμμα)                               |

-   buttonGroup

επιτρέπει στους χρήστες να δημιουργήσουν διαδραστικές ομάδες κουμπιών μέσα σε μια στήλη

Για κάθε κουμπί στις στήλες, μπορούμε να ρυθμίσουμε τις ιδιότητές του.

| Ιδιότητες             | Περιγραφή                                                                                                                                            |
| --------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------- |
| Αντιστοιχούμενη Τιμή  | συσχέτιση κάθε κουμπιού στην ομάδα κουμπιών με μια συγκεκριμένη τιμή ή πεδίο από την πηγή δεδομένων                                                  |
| Παραλλαγή             | οπτικό στυλ ή εμφάνιση των κουμπιών στην ομάδα κουμπιών                                                                                              |
| Χειριστής συμβάντων   | την ενέργεια ή τη λειτουργία που ενεργοποιείται όταν ένα κουμπί της ομάδας πατηθεί ή αλληλεπιδρά με                                                  |
| Απενεργοποιημένο      | αν τα πλήκτρα της ομάδας πλήκτρων είναι ενεργοποιημένα ή απενεργοποιημένα                                                                            |
| Χρώμα θέματος         | προσαρμόστε το χρώμα ή το styling των κουμπιών της ομάδας για να ευθυγραμμίσετε με το γενικό θέμα ή το branding της εφαρμογής ή της ιστοσελίδας τους |

-   Boolean

αντιπροσωπεύει δεδομένα που έχουν δύο πιθανές καταστάσεις: αληθή ή ψευδή

-   Εικόνα

συμπλήρωση μιας στήλης στον πίνακα με δεδομένα εικόνας

| Ιδιότητες      | Περιγραφή                                                            |
| -------------- | -------------------------------------------------------------------- |
| Τύπος Κλίμακας | πώς η εικόνα κλιμακώνεται ή αλλάζει μέγεθος μέσα στο κελί του πίνακα |

-   Ομάδα Εικονιδίων

μια εξειδικευμένη στήλη που επιτρέπει στους χρήστες να εμφανίζουν εικονίδια σε μια ομάδα ή συλλογή

Για κάθε κουμπί στις στήλες, μπορούμε να ρυθμίσουμε τις ιδιότητές του.

| Ιδιότητες             | Περιγραφή                                                                                                                                      |
| --------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------- |
| Εικονίδιο             | επιτρέπει στους χρήστες να επιλέξουν ή να καθορίσουν το εικονίδιο που θα εμφανίζεται στη στήλη IconGroup                                       |
| Χρώμα θέματος         | καθορίζει το θέμα χρωμάτων ή το θέμα που εφαρμόζεται στο εικονίδιο μέσα στη στήλη IconGroup                                                    |
| Χειριστής συμβάντων   | επιτρέπει στους χρήστες να ορίσουν ένα γεγονός ή ενέργεια που ενεργοποιείται όταν το εικονίδιο στη στήλη IconGroup αλληλεπιδρά με τους χρήστες |
| Απενεργοποιημένο      | κατάσταση της στήλης IconGroup, που καθορίζει αν είναι απενεργοποιημένη ή ενεργοποιημένη για αλληλεπίδραση χρήστη                              |

-   Βαθμολογία

αφορά αξιολογήσεις ή σχόλια που δίνονται σε στοιχεία ή οντότητες

-   Markdown

δυνατότητα απόδοσης μορφοποιημένου κειμένου χρησιμοποιώντας σύνταξη Markdown μέσα στα κελιά στήλης

-   Νόμισμα

χειρίζονται και διαμορφώνουν τις τιμές συναλλάγματος.

| Ιδιότητες                       | Περιγραφή                                                                                                |
| ------------------------------- | -------------------------------------------------------------------------------------------------------- |
| Δεκαδικά ψηφία                  | τον αριθμό δεκαδικών ψηφίων που θα εμφανίζονται για τις τιμές συναλλάγματος εντός της στήλης             |
| Κωδικός νομίσματος              | προσδιορίζει τον κωδικό ή τη συντομογραφία που αντιπροσωπεύει το νόμισμα που χρησιμοποιείται στη στήλη   |
| Εμφάνιση διαχωριστικού χιλιάδων | αν ένα διαχωριστικό χιλιάδων εμφανίζεται για μεγάλες αξίες συναλλάγματος                                 |

### Μέθοδος

Μπορείτε να χρησιμοποιήσετε άλλα στοιχεία για τον έλεγχο της εφαρμογής. Υποστηρίζουμε τις ακόλουθες δύο μεθόδους:

-   **selectPage**

επιλογή όλων των γραμμών σε μια συγκεκριμένη σελίδα σε έναν σελιδοποιημένο πίνακα

| Ιδιότητες    | Περιγραφή                                                     |
| ------------ | ------------------------------------------------------------- |
| Τελική Τιμή  | τον αριθμό σελίδας ή το ευρετήριο της σελίδας που θα επιλεγεί |

-   **επιλογή**

επιλέξτε μια συγκεκριμένη γραμμή στον πίνακα

| Ιδιότητες                       | Περιγραφή                                                                                                                                                                           |
| ------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Προεπιλεγμένη επιλεγμένη γραμμή | προεπιλεγμένη επιλεγμένη τιμή σειράς, μπορεί να είναι ο δείκτης σειράς, ένα μοναδικό αναγνωριστικό, ή οποιαδήποτε άλλη ιδιότητα που προσδιορίζει μοναδικά τη γραμμή που θα επιλεγεί |

-   **εκκαθάριση**

καθαρισμός της τρέχουσας επιλογής στον πίνακα

-   **setFilters**

εφαρμογή φίλτρων στα δεδομένα του πίνακα

| Ιδιότητες          | Περιγραφή                                |
| ------------------ | ---------------------------------------- |
| Φίλτρα             | συνθήκες ή κριτήρια φιλτραρίσματος       |
| Λειτουργία φίλτρου | καθορίζει πώς εφαρμόζονται τα φίλτρα     |

-   **ClearFilters**

καθαρίζει τυχόν εφαρμοσμένα φίλτρα στον πίνακα

-   **setSort**

όρισε τη διαλογή του πίνακα

| Ιδιότητες   | Περιγραφή                                                             |
| ----------- | --------------------------------------------------------------------- |
| Στήλη       | τη στήλη στην οποία πρέπει να εφαρμοστεί η διαλογή                    |
| Κατεύθυνση  | κατεύθυνση του είδους, η οποία μπορεί να είναι "αύξουσα" ή "φθίνουσα" |

## Περίπτωση χρήσης

Στη συνέχεια, θα δείξουμε πώς να αντιστοιχίσουμε τα δεδομένα από το `**Upstash**` στο στοιχείο `**table**`.

#### Βήμα 1 Προσθέστε μια ενέργεια

Ας δημιουργήσουμε έναν πίνακα στην Upstash που ονομάζεται **`employee_profile`,** συμπεριλαμβανομένων 4 στηλών `name`, `level`, `age`, and `married`.

Στη συνέχεια, μπορούμε να δημιουργήσουμε μια νέα ενέργεια για Upstash από τη λίστα ενεργειών στο ILLA Cloud και ονομάζεται upstash2.

Για να απαριθμήσετε όλα τα δεδομένα στο employee_profile βάλτε τον κώδικα αποσπάσματος παρακάτω στην ενότητα ερώτημα Redis.

```sql
GET employee_profile
```

Δεδομένου ότι τα δεδομένα εξόδου είναι μια λίστα με ζεύγος λεξικών, θέλουμε να τα μετατρέψουμε σε ένα πίνακα. Στο Transformer, ενεργοποιήστε το και βάλτε τον παρακάτω κώδικα.

```jsx
const jsonString = data[0].result;

// Parse the JSON string into a JavaScript object
const jsonData = JSON.parse(jsonString);

// Access the "employees" property and map it to a new array
const mappedArray = jsonData.employees.map((employee) => ({
  name: employee.name,
  level: employee.level,
  age: employee.age,
  married: employee.married,
}));

return mappedArray;
```

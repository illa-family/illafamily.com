## <h2 hidden>Πίνακας</h2>

Οι πίνακες είναι ένας κοινός τρόπος προβολής και αλληλεπίδρασης με τα δεδομένα σας. Μπορείτε να ταξινομήσετε, να φιλτράρετε, να σελιδοποιήσετε, και να κατεβάσετε σειρές πληροφοριών. Πίνακες υποστηρίζουν επίσης την επιλογή γραμμών και προσαρμοσμένες στήλες.

<img
  src='https://cdn.illacloud.com/official-website/img/docs/assemble/table.png'
  alt=""
/>

### Φόρτωση δεδομένων σε έναν πίνακα

Όταν σύρετε ένα στοιχείο πίνακα στο πλαίσιο, εμφανίζει αυτόματα τα δεδομένα δοκιμών σε μορφή JSON. Μπορείτε να αλλάξετε τα δεδομένα που εμφανίζονται σε έναν πίνακα με την επεξεργασία της ιδιότητας προέλευσης δεδομένων. Χρησιμοποίησε τη μορφή `{{ actionName.data }}` για να επιλέξεις την ενέργεια.

<img
  src='https://cdn.illacloud.com/official-website/img/docs/assemble/table1.png'
  alt=""
/>

Μπορείτε επίσης να ορίσετε την πηγή δεδομένων εισάγοντας έναν πίνακα.

### Ρυθμίσεις στήλης

**Ορισμός ψευδωνύμων για στήλες δεδομένων**

Όταν ορίσετε την πηγή δεδομένων των πινάκων, το όνομα της στήλης θα οριστεί αυτόματα με βάση την πηγή δεδομένων. Μπορείτε να μετονομάσετε όλες τις στήλες κάνοντας κλικ στη στήλη στην καρτέλα **Επιθεώρηση** και ορίστε τον **Τίτλο στήλης**.

**Αλλαγή τύπων στηλών**

Όταν ορίζετε την πηγή δεδομένων των πινάκων, οι στήλες έχουν οριστεί σε **Κείμενο**. Μπορείτε να ορίσετε τύπους στηλών χειροκίνητα κάνοντας κλικ στη στήλη στην καρτέλα **Επιθεώρηση** και επιλέγοντας τον τύπο από το αναπτυσσόμενο μενού **Τύπος στήλης** .

Οι τύποι στηλών περιλαμβάνουν:

1.  Κείμενο
2.  Ημερομηνία, μπορείτε να ορίσετε τη μορφή εμφάνισης της ημερομηνίας στον τύπο ημερομηνίας.
3.  Αριθμό, μπορείτε να ορίσετε τις δεκαδικές θέσεις στον τύπο αριθμού.
4.  Τοις τοις εκατό, μπορείς να ορίσεις τα δεκαδικά ψηφία με τον τύπο τοις εκατό.
5.  Σύνδεσμος
6.  Κουμπί, μπορείτε να προσθέσετε χειριστές γεγονότων στον τύπο του κουμπιού.

/>

**Αντιστοιχισμένη τιμή προσαρμοσμένων στηλών**

Μπορείτε να προσθέσετε προσαρμοσμένες στήλες σε πίνακες χρησιμοποιώντας το κουμπί **+ Προσθήκη** στην καρτέλα **Επιθεώρηση** . Οι προσαρμοσμένες στήλες χρησιμοποιούνται συχνά για τον υπολογισμό δεδομένων με βάση άλλες τιμές στην εφαρμογή σας. Για παράδειγμα, αν έχετε στήλες `revenue` και `user_num` και θέλετε να υπολογίσετε μια στήλη `ARPU`, θα μπορούσατε να χρησιμοποιήσετε `{{currentRow.revenue/currentRow.user_num }}`

Επιπλέον, μετά τον υπολογισμό, μπορείτε να ορίσετε τον τύπο στήλης σε αριθμό ή ποσοστό και να ορίσετε τα δεκαδικά για τη βελτίωση της αναγνωσιμότητας.

### Δεδομένα πρόσβασης του πίνακα

Χρησιμοποιήστε το `{{ table1.selectedRow[i].columnName }}` για να αποκτήσετε πρόσβαση στα δεδομένα των επιλεγμένων γραμμών σε άλλα στοιχεία. Αν δεν έχετε ενεργοποιήσει την **Επιλογή Πολλαπλών Γραμμών**, το `i` θα είναι πάντα 0.

<img
  src='https://cdn.illacloud.com/official-website/img/docs/assemble/table4.png'
  alt=""
/>

**Ενημέρωση των δεδομένων του πίνακα**

1.  Χρησιμοποιήστε το εξάρτημα εισαγωγής ή άλλα δεδομένα εισόδου cpmponents για να εμφανίσετε το `{{ table1.selectedRow[i].columnName }}`.
2.  Χρησιμοποίησε το `{{ input1.value }}` στις ενέργειες ενημέρωσης για να πάρεις επιλεγμένες τιμές και ενημερωμένες τιμές.
3.  Προσθέστε έναν χειριστή συμβάντων για να ενημερώσετε το κουμπί για να εκτελέσετε την ενέργεια ενημέρωσης όταν πατηθεί.
4.  Ενημερώστε τα δεδομένα της επιλεγμένης γραμμής αλλάζοντας την τιμή εισόδου και κάνοντας κλικ στο κουμπί Ενημέρωση.

<img
  src='https://cdn.illacloud.com/official-website/img/docs/assemble/table5.png'
  alt=""
/>

**Εισαγωγή νέας γραμμής στον πίνακα**

1.  Χρησιμοποιήστε το `{{ input1.value }}` στις ενέργειες εισαγωγής λάβετε τιμές.
2.  Πρόσθεσε έναν χειριστή συμβάντος στο κουμπί για να εκτελέσεις την ενέργεια εισαγωγής όταν πατηθεί.
3.  Εισάγετε νέα δεδομένα εισάγοντας τιμές στα στοιχεία εισόδου και κάνοντας κλικ στο κουμπί Εισαγωγή.

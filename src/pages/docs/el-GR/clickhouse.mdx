## <h2 hidden>Τι είναι το ClickHouse?</h2>

Το ClickHouse είναι ένα γρήγορο, προσανατολισμένο σε στήλες σύστημα διαχείρισης βάσεων δεδομένων που έχει σχεδιαστεί για online αναλυτική επεξεργασία (OLAP) φορτία εργασίας. Πρόκειται για ένα έργο ανοιχτού κώδικα που δημιουργήθηκε από την Yandex, τη ρωσική εταιρεία μηχανών αναζήτησης. ClickHouse είναι γνωστό για την ικανότητά του να επεξεργάζεται μεγάλες ποσότητες δεδομένων γρήγορα και αποτελεσματικά, καθιστώντας το μια ιδανική λύση για μεγάλες αναλύσεις δεδομένων.

Με το Illa Cloud, μπορείτε να αναπτύξετε και να διαχειριστείτε τα clusters ClickHouse στο σύννεφο, χωρίς να χρειάζεται να ανησυχείτε για την υποκείμενη υποδομή και συντήρηση.

Illa Cloud παρέχει μια διαδικτυακή διεπαφή για τη διαχείριση clusters ClickHouse, που σας επιτρέπει να παρακολουθείτε την απόδοση του συμπλέγματος, τους πόρους κλίμακας προς τα πάνω ή προς τα κάτω, και να ρυθμίσετε τα αντίγραφα ασφαλείας και τις ρυθμίσεις ασφαλείας. Μπορείτε επίσης να χρησιμοποιήσετε το Illa Cloud για να εκτελέσετε ερωτήματα στα δεδομένα του ClickHouse χρησιμοποιώντας έναν επεξεργαστή ερωτημάτων στο διαδίκτυο.

### Δημιουργία ClickHouse

Υπάρχουν δύο τρόποι για να δημιουργήσετε έναν πόρο στην Illa μετά την είσοδό σας στο λογαριασμό Illa.

-   Δημιουργία σε πόρους

Συνδεθείτε στον Illa λογαριασμό σας, επιλέξτε `**Πόροι**` στην κορυφή της σελίδας και κάντε κλικ στο κουμπί `**Δημιουργία Νέου**`.

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

Επιλέξτε **`ClickHouse`** από τη λίστα της βάσης δεδομένων.

![action_list](https://cdn.illacloud.com/official-website/img/docs/action_list.png)

Συνδεθείτε στη βάση δεδομένων με τις απαιτούμενες παραμέτρους που περιγράφονται στις παρακάτω ρυθμίσεις σύνδεσης.

Κάντε κλικ στο κουμπί `**Δοκιμή Σύνδεσης**` για να δείτε αν μπορούμε να συνδεθείτε με επιτυχία στη βάση δεδομένων. Εάν ναι, κάντε κλικ στο **`Αποθήκευση πόρων`**, αλλιώς, ελέγξτε ξανά το όνομα κεντρικού υπολογιστή, τη θύρα, το όνομα χρήστη και τον κωδικό πρόσβασης είναι σωστά.

Μετά τη δημιουργία ενός πόρου, το έτοιμο ClickHouse θα εμφανίζεται όπως εμφανίζεται.

![click_config](https://cdn.illacloud.com/official-website/img/docs/click_config.png)

-   Δημιουργία στο Builder

Συνδεθείτε στον Illa λογαριασμό σας, δημιουργήστε ένα έργο στο Illa Builder στη σελίδα `**App**`, και πλοηγηθείτε στο **`Λίστα Ενεργειών`** στο κάτω μέρος της σελίδας. Κάντε κλικ στο κουμπί `**new**`, και μετά επιλέξτε **`ClickHouse`** από τη λίστα της βάσης δεδομένων. Στη συνέχεια, συνδεθείτε στη βάση δεδομένων με τις απαιτούμενες παραμέτρους που περιγράφονται στις "Ρυθμίσεις Σύνδεσης" παρακάτω.

Κάντε κλικ στο κουμπί `**Δοκιμή Σύνδεσης**` για να δείτε αν μπορούμε να συνδεθείτε με επιτυχία στη βάση δεδομένων. Εάν ναι, κάντε κλικ στο **`Αποθήκευση πόρων`**, αλλιώς, ελέγξτε ξανά το όνομα κεντρικού υπολογιστή, τη θύρα, το όνομα χρήστη και τον κωδικό πρόσβασης είναι σωστά.

### Ρυθμίσεις Σύνδεσης

Εδώ πρέπει να παρέχουμε πληροφορίες για τη σύνδεση με τη βάση δεδομένων ClickHouse.

![click_config](https://cdn.illacloud.com/official-website/img/docs/click_config.png)

| Ιδιότητες         | Απαιτείται  | Περιγραφή                                                                                                                                            |
| ----------------- | ----------- | ---------------------------------------------------------------------------------------------------------------------------------------------------- |
| Όνομα             | απαιτείται  | Το όνομα του πόρου κατά τη δημιουργία ενεργειών στο ILLA.                                                                                            |
| Hostname          | απαιτείται  | Η διεύθυνση URL ή IP για τη βάση δεδομένων σας                                                                                                       |
| Θύρα              | απαιτείται  | Ο αριθμός θύρας του διακομιστή που θα πρέπει να χρησιμοποιήσετε για να συνδεθείτε. Αν δεν καθορίσετε μια θύρα, η προεπιλεγμένη θύρα είναι το '9440'. |
| Βάση Δεδομένων    | απαιτείται  | Το όνομα της βάσης δεδομένων                                                                                                                         |
| Όνομα Χρήστη      | απαιτείται  | το όνομα χρήστη που θέλετε να χρησιμοποιήσετε κατά τη σύνδεση στο διακομιστή ClickHouse.                                                             |
| Κωδικός           | απαιτείται  | Χρησιμοποιήστε αυτόν τον κωδικό πρόσβασης για ταυτοποίηση.                                                                                           |
| Επιλογές SSL      | προαιρετικό | αποφασίζει πόσο υψηλή προτεραιότητα έχει μια ασφαλής σύνδεση SSL TCP/IP κατά τη διαπραγμάτευση με το διακομιστή.                                     |

### Δημιουργία Ενεργειών

Δημιουργήσαμε έναν πόρο ClickHouse, μπορούμε να προσθέσουμε την ενέργεια επιλέγοντας το ClickHouse από τη λίστα ενεργειών και επιλέγοντας το κουμπί **`Δημιουργία δράσης`**.

![click_resource_list](https://cdn.illacloud.com/official-website/img/docs/clik_resource_list.png)

Τώρα έχουμε προσθέσει τον εξυπηρετητή ClickHouse ως ενέργεια στη σελίδα μας κτηρίου.

![click](https://cdn.illacloud.com/official-website/img/docs/click.png)

### Configure ClickHouse

| Ιδιότητες       | Περιγραφή                                                                                     |
| --------------- | --------------------------------------------------------------------------------------------- |
| Ερώτημα SQL     | εντολές sql για λήψη (επιλογή), εισαγωγή, ενημέρωση, διαγραφή δεδομένων από τη βάση δεδομένων |
| Μετασχηματιστής | μετατρέποντας τα δεδομένα σε στυλ που σας αρέσει χρησιμοποιώντας JavaScript                   |

Χρήση παραδείγματος:

-   Επιλογή

```sql
SELECT * FROM users ORDER BY lastname;
```

-   Insert

```sql
INSERT INTO users

	(firstname, lastname, email)

VALUES

	(

		{{ NewUser.data.firstNameInput }},

		{{ NewUser.data.lastNameInput }},

		{{ NewUser.data.emailInput }}

	);
```

-   Ενημέρωση

```sql
UPDATE users
  SET firstname = '{{form1.updatedProfile.first}}'
  WHERE id = {{ form1.updatedProfile.uid}};
```

-   Διαγραφή

```sql
DELETE FROM users WHERE id = {{ form1.updatedProfile.uid }};
```

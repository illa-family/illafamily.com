# <Heading hidden>Ανάπτυξη στο χώρο</Heading>

Αυτο-φιλοξενία ILLA σημαίνει φιλοξενία και διαχείριση της πλατφόρμας ILLA στο δικό σας διακομιστή ή μηχανήματος, αντί να χρησιμοποιήσετε έναν τρίτο πάροχο υπηρεσιών. Αυτή η προσέγγιση θεωρείται συχνά πιο ασφαλής και ιδιωτική, καθώς έχετε πλήρη έλεγχο των δεδομένων σας και της χρήσης του.

ILLA θα προσφέρει μια έκδοση cloud-hosted και την επιλογή να ρυθμίσετε την περίπτωσή σας σε διάφορες πλατφόρμες σύντομα.

## Αυτο-φιλοξενείται

Η αυτο-φιλοξενία ILLA στη δική σας υποδομή σας επιτρέπει να την ενσωματώσετε απρόσκοπτα με την υπάρχουσα εγκατάσταση σας χωρίς την ανάγκη να διατηρήσετε μια ξεχωριστή διαμόρφωση. Αυτό σας επιτρέπει να χρησιμοποιήσετε τις δυνατότητες και τις δυνατότητες της υπάρχουσας υποδομής σας και να τις επεκτείνετε στο ILLA. ### Ασφάλεια

Με τη φιλοξενία της ILLA στη δική σας υποδομή, μπορείτε να κρατήσετε τα δεδομένα σας on-premises και να βεβαιωθείτε ότι τα κλειδιά API ή τα διαπιστευτήρια ασφαλείας σας δεν εγκαταλείπουν ποτέ το διακομιστή σας. Αυτό σας επιτρέπει να εφαρμόσετε τους ίδιους κανόνες ασφαλείας στην ILLA που εφαρμόζετε σε οποιαδήποτε άλλη εφαρμογή στον διακομιστή σας. Οι εφαρμογές που κατασκευάζετε με χρήση ILLA θα φιλοξενηθούν στην υποδομή σας και μέσα στα εταιρικά σας δίκτυα. Μπορείτε επίσης να επιλέξετε να φιλοξενήσετε το ILLA σε εσωτερικά δίκτυα πίσω από ένα τείχος προστασίας και να το ρυθμίσετε ώστε να συνεργάζεται με τις προσαρμοσμένες πολιτικές τομέα και ασφάλειας σας.

### Εσωτερική βάση δεδομένων

Με την αυτο-φιλοξενία ILLA, μπορείτε να συνδεθείτε σε μια βάση δεδομένων που τρέχει στο ίδιο μηχάνημα ή intranet και να την χρησιμοποιήσετε για την κατασκευή εφαρμογών. Το ILLA χρησιμοποιεί την MongoDB για να αποθηκεύει δεδομένα εφαρμογών και το Redis για να αποθηκεύει πληροφορίες συνεδρίας και μη κρίσιμες κρύπτες. Αν θέλετε να ενσωματώσετε το ILLA με προσαρμοσμένους διακομιστές MongoDB ή Redis που φιλοξενούνται στην υποδομή σας, θα χρειαστεί να τροποποιήσετε τη ρύθμιση παραμέτρων των μεταβλητών περιβάλλοντος και να τις υποδείξετε στις προσαρμοσμένες παρουσίες MongoDB/Redis.

### Ενημερώσεις

Με την αυτο-φιλοξενία ILLA, μπορείτε να ρυθμίσετε πιο ευέλικτη ενημέρωση χρονοδιαγράμματα και άλλες δραστηριότητες συντήρησης. Μπορείτε να ακολουθήσετε τα βήματα που παρέχονται για να ρυθμίσετε ένα παράθυρο συντήρησης και τον προγραμματισμό αυτόματων ενημερώσεων. Αυτό σας επιτρέπει να διατηρήσετε και να ενημερώσετε την παρουσία ILLA σας με έναν τρόπο που είναι βολικό για εσάς και τον οργανισμό σας.

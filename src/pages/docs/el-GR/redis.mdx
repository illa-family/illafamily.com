## <h2 hidden>Τι είναι Redis?</h2>

Το Redis in Illa είναι μια ενσωμάτωση βάσης δεδομένων που σας επιτρέπει να συνδεθείτε και να αλληλεπιδράσετε με μια βάση δεδομένων Redis. Το Redis είναι ένα δημοφιλές σχετικό σύστημα διαχείρισης βάσεων δεδομένων ανοικτού κώδικα που είναι γνωστό για την ταχύτητα, την αξιοπιστία και την ευκολία χρήσης.

Με την ενσωμάτωση του Redis στο Illa, μπορείτε να ρωτήσετε, να εισαγάγετε και να διαγράψετε δεδομένα από μια βάση δεδομένων Redis χρησιμοποιώντας εντολές SQL. Μπορείτε επίσης να δημιουργήσετε προσαρμοσμένα ερωτήματα SQL και να τα εκτελέσετε μέσα Illa. Αυτό καθιστά εύκολη την κατασκευή εφαρμογών καθοδηγούμενων από δεδομένα και ταμπλό που εμφανίζουν δεδομένα σε πραγματικό χρόνο από μια βάση δεδομένων Redis.

### Δημιουργία Redis

Υπάρχουν δύο τρόποι για να δημιουργήσετε έναν πόρο στην Illa μετά την είσοδό σας στο λογαριασμό Illa.

-   Δημιουργία σε πόρους

Συνδεθείτε στον Illa λογαριασμό σας, επιλέξτε `**Πόροι**` στην κορυφή της σελίδας και κάντε κλικ στο κουμπί `**Δημιουργία Νέου**`.

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

Επιλέξτε **`Redis`** από τη λίστα της βάσης δεδομένων.

![action_list](https://cdn.illacloud.com/official-website/img/docs/action_list.png)

Συνδεθείτε στη βάση δεδομένων με τις απαιτούμενες παραμέτρους που περιγράφονται στις παρακάτω ρυθμίσεις σύνδεσης.

Κάντε κλικ στο κουμπί `**Δοκιμή Σύνδεσης**` για να δείτε αν μπορούμε να συνδεθείτε με επιτυχία στη βάση δεδομένων. Εάν ναι, κάντε κλικ στο **`Αποθήκευση πόρων`**, αλλιώς, ελέγξτε ξανά το όνομα κεντρικού υπολογιστή, τη θύρα, το όνομα χρήστη και τον κωδικό πρόσβασης είναι σωστά.

Μετά τη δημιουργία ενός πόρου, το έτοιμο Redis θα εμφανίζεται όπως εμφανίζεται.

![redis_ex_ready](https://cdn.illacloud.com/official-website/img/docs/redis_ex_ready.png)

-   Δημιουργία στο Builder

Συνδεθείτε στον Illa λογαριασμό σας, δημιουργήστε ένα έργο στο Illa Builder στη σελίδα `**App**`, και πλοηγηθείτε στο **`Λίστα Ενεργειών`** στο κάτω μέρος της σελίδας. Κάντε κλικ στο κουμπί `**new**`, και μετά επιλέξτε **`Redis`** από τη λίστα της βάσης δεδομένων. Στη συνέχεια, συνδεθείτε στη βάση δεδομένων με τις απαιτούμενες παραμέτρους που περιγράφονται στις "Ρυθμίσεις Σύνδεσης" παρακάτω.

Κάντε κλικ στο κουμπί `**Δοκιμή Σύνδεσης**` για να δείτε αν μπορούμε να συνδεθείτε με επιτυχία στη βάση δεδομένων. Εάν ναι, κάντε κλικ στο **`Αποθήκευση πόρων`**, αλλιώς, ελέγξτε ξανά το όνομα κεντρικού υπολογιστή, τη θύρα, το όνομα χρήστη και τον κωδικό πρόσβασης είναι σωστά.

### Ρυθμίσεις Σύνδεσης

Εδώ πρέπει να παρέχουμε πληροφορίες για τη σύνδεση με τη βάση δεδομένων Redis.

![redis_config](https://cdn.illacloud.com/official-website/img/docs/redis_config.png)

| Ιδιότητες                 | Απαιτείται  | Περιγραφή                                                                                                                                            |
| ------------------------- | ----------- | ---------------------------------------------------------------------------------------------------------------------------------------------------- |
| Όνομα                     | απαιτείται  | Το όνομα του πόρου κατά τη δημιουργία ενεργειών στο ILLA.                                                                                            |
| Hostname                  | απαιτείται  | Η διεύθυνση URL ή IP για τη βάση δεδομένων σας                                                                                                       |
| Θύρα                      | απαιτείται  | Ο αριθμός θύρας του διακομιστή που θα πρέπει να χρησιμοποιήσετε για να συνδεθείτε. Αν δεν καθορίσετε μια θύρα, η προεπιλεγμένη θύρα είναι '6379'.    |
| Ευρετήριο Βάσης Δεδομένων | προαιρετικό | μια δομή δεδομένων που χρησιμοποιείται για τη βελτιστοποίηση της απόδοσης των ερωτημάτων που ανακτούν δεδομένα από μια βάση δεδομένων Redis          |
| Βάση Δεδομένων            | απαιτείται  | Το όνομα της βάσης δεδομένων                                                                                                                         |
| Όνομα Χρήστη              | απαιτείται  | το όνομα χρήστη που θέλετε να χρησιμοποιήσετε κατά τη σύνδεση στον διακομιστή Redis.                                                                 |
| Κωδικός                   | απαιτείται  | Χρησιμοποιήστε αυτόν τον κωδικό πρόσβασης για ταυτοποίηση.                                                                                           |
| Επιλογές SSL              | προαιρετικό | αποφασίζει πόσο υψηλή προτεραιότητα έχει μια ασφαλής σύνδεση SSL TCP/IP κατά τη διαπραγμάτευση με το διακομιστή.                                     |

### Δημιουργία Ενεργειών

Δημιουργήσαμε έναν πόρο Redis, μπορούμε να προσθέσουμε την ενέργεια επιλέγοντας Redis από τη λίστα ενεργειών και επιλέγοντας το κουμπί **`Δημιουργία ενέργειας`**.

![redis_resource_list](https://cdn.illacloud.com/official-website/img/docs/redis_resource_list.png)

Τώρα έχουμε προσθέσει τον διακομιστή Redis ως ενέργεια στη σελίδα δημιουργίας μας.

![redis](https://cdn.illacloud.com/official-website/img/docs/redis.png)

### Ρύθμιση Redis

| Ιδιότητες       | Περιγραφή                                                                   |
| --------------- | --------------------------------------------------------------------------- |
| ερώτημα         | εντολές για ανάκτηση δεδομένων από βάση δεδομένων                           |
| Μετασχηματιστής | μετατρέποντας τα δεδομένα σε στυλ που σας αρέσει χρησιμοποιώντας JavaScript |

Χρήση παραδείγματος:

-   ΚΛΕΙΔΑ

Αυτή η εντολή ανακτά μια λίστα με όλα τα κλειδιά στη βάση δεδομένων Redis. Μπορείτε να χρησιμοποιήσετε μπαλαντέρ για να φιλτράρετε τα πλήκτρα με μοτίβο.

```sql
KEYS *
KEYS user:*
```

-   GET

Αυτή η εντολή ανακτά την τιμή που σχετίζεται με ένα συγκεκριμένο κλειδί.

```sql
GET user:12345
```

-   HGETALL

Αυτή η εντολή ανακτά όλα τα πεδία και τις τιμές τους από μια δομή δεδομένων κατακερματισμού Redis.

```sql
HGETALL user:12345
```

-   ZRANGE

Αυτή η εντολή ανακτά ένα εύρος τιμών από μια δομή δεδομένων ταξινόμησης Redis, με βάση τις βαθμολογίες τους.

```sql
ZRANGE high_scores 0 9 WITHSCORES
```

-   ΡΥΘΜΙΣΗ

Αυτή η εντολή ανακτά ένα εύρος τιμών από μια δομή δεδομένων λίστας Redis.

```sql
LRANGE my_list 0 4
```

-   ΥΦΙΣΤΑΜΕΝΕΣ

Αυτή η εντολή ελέγχει αν υπάρχει ένα δεδομένο κλειδί στη βάση δεδομένων Redis.

```sql
EXISTS user:12345
```

-   TTL

Αυτή η εντολή ανακτά την τιμή χρόνου σε ζωντανή (TTL) που σχετίζεται με ένα κλειδί, το οποίο καθορίζει πόσο καιρό θα παραμείνει το κλειδί στη βάση δεδομένων πριν λήξει.

```sql
TTL user:12345
```

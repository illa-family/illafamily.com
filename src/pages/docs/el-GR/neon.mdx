## <h2 hidden>Τι είναι Neon?</h2>

Neon στην Illa είναι μια ενσωμάτωση της βάσης δεδομένων που σας επιτρέπει να συνδεθείτε και να αλληλεπιδράσετε με μια βάση δεδομένων Neon. Το Neon είναι ένα δημοφιλές σχετικό σύστημα διαχείρισης βάσεων δεδομένων ανοικτού κώδικα που είναι γνωστό για την ταχύτητα, την αξιοπιστία και την ευκολία χρήσης.

Με την ενσωμάτωση Neon στην Illa, μπορείτε να ρωτήσετε, να εισαγάγετε και να διαγράψετε δεδομένα από μια βάση δεδομένων Neon χρησιμοποιώντας εντολές SQL. Μπορείτε επίσης να δημιουργήσετε προσαρμοσμένα ερωτήματα SQL και να τα εκτελέσετε μέσα Illa. Αυτό καθιστά εύκολο να οικοδομήσουμε τις εφαρμογές δεδομένων και τους πίνακες ελέγχου που εμφανίζουν δεδομένα σε πραγματικό χρόνο από μια βάση δεδομένων Neon.

Αυτό το σεμινάριο περιγράφει τη διαδικασία δημιουργίας ενός Admin Panel χρησιμοποιώντας ILLA Builder και Neon σε μερικά απλά βήματα. Το ILLA είναι μια πλατφόρμα χαμηλού κώδικα για προγραμματιστές που επιτρέπει την ταχεία ανάπτυξη και ανάπτυξη εσωτερικών εργαλείων. Επιτρέπει τη δημιουργία σελίδων σύροντας και απορρίπτοντας στοιχεία UI, που συνδέονται με οποιαδήποτε βάση δεδομένων ή API, και γράφοντας JavaScript. Για να μάθετε περισσότερα για το Neon, επισκεφθείτε την ιστοσελίδα τους στο https://neon.tech/ Ας ξεκινήσουμε!

### Ρυθμίστε το Πίσω Τέλος στο Νέον

Στον πίνακα ελέγχου Νέον, κάντε κλικ στο κουμπί `Νέο έργο` και ορίστε το όνομα σε `adminPanel`.

![neon_create_project](https://cdn.illacloud.com/official-website/img/docs/neon_create_project.png)

Στη συνέχεια, μπορούμε να πλοηγηθούμε στη σελίδα του έργου.

![neon_create_project_1](https://cdn.illacloud.com/official-website/img/docs/neon_create_project_1.png)

Εισαγωγή δεδομένων μέσω δείγματος ερωτημάτων SQL.

![neon_create_sql](https://cdn.illacloud.com/official-website/img/docs/neon_create_sql.png)

Τώρα έχουμε δημιουργήσει τη βάση δεδομένων δειγμάτων.

![neon_sample_data](https://cdn.illacloud.com/official-website/img/docs/neon_sample_data.png)

### Δημιουργία Νέον

Υπάρχουν δύο τρόποι για να δημιουργήσετε έναν πόρο στην Illa μετά την είσοδό σας στο λογαριασμό Illa. 

-   Δημιουργία σε πόρους

Συνδεθείτε στον Illa λογαριασμό σας, επιλέξτε `**Πόροι**` στην κορυφή της σελίδας και κάντε κλικ στο κουμπί `**Δημιουργία Νέου**`. 

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

Επιλέξτε **`Neon`** από τη λίστα της βάσης δεδομένων.

![action_list](https://cdn.illacloud.com/official-website/img/docs/action_list.png)

Συνδεθείτε στη βάση δεδομένων με τις απαιτούμενες παραμέτρους που περιγράφονται στις παρακάτω ρυθμίσεις σύνδεσης. 

Κάντε κλικ στο κουμπί `**Δοκιμή Σύνδεσης**` για να δείτε αν μπορούμε να συνδεθείτε με επιτυχία στη βάση δεδομένων. Εάν ναι, κάντε κλικ στο **`Αποθήκευση πόρων`**, αλλιώς, ελέγξτε ξανά το όνομα κεντρικού υπολογιστή, τη θύρα, το όνομα χρήστη και τον κωδικό πρόσβασης είναι σωστά.

Μετά τη δημιουργία ενός πόρου, το έτοιμο Νέον θα εμφανίζεται όπως εμφανίζεται.

![neon_ex_ready](https://cdn.illacloud.com/official-website/img/docs/neon_ex_ready.png)

-   Δημιουργία στο Builder

Συνδεθείτε στον Illa λογαριασμό σας, δημιουργήστε ένα έργο στο Illa Builder στη σελίδα `**App**`, και πλοηγηθείτε στο **`Λίστα Ενεργειών`** στο κάτω μέρος της σελίδας. Κάντε κλικ στο κουμπί `**new**`, και μετά επιλέξτε **`Neon`** από τη λίστα της βάσης δεδομένων. Στη συνέχεια, συνδεθείτε στη βάση δεδομένων με τις απαιτούμενες παραμέτρους που περιγράφονται στις "Ρυθμίσεις Σύνδεσης" παρακάτω.

Κάντε κλικ στο κουμπί `**Δοκιμή Σύνδεσης**` για να δείτε αν μπορούμε να συνδεθείτε με επιτυχία στη βάση δεδομένων. Εάν ναι, κάντε κλικ στο **`Αποθήκευση πόρων`**, αλλιώς, ελέγξτε ξανά το όνομα κεντρικού υπολογιστή, τη θύρα, το όνομα χρήστη και τον κωδικό πρόσβασης είναι σωστά.

### Ρυθμίσεις Σύνδεσης

Εδώ πρέπει να παρέχουμε πληροφορίες για τη σύνδεση με τη βάση δεδομένων Neon.

![neon_config](https://cdn.illacloud.com/official-website/img/docs/neon_config.png)

| Ιδιότητες             | Απαιτείται  | Περιγραφή                                                                                                                                                           |
| --------------------- | ----------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Όνομα                 | απαιτείται  | Το όνομα του πόρου κατά τη δημιουργία ενεργειών στο ILLA.                                                                                                           |
| Συμβολοσειρά σύνδεσης | προαιρετικό | μια ρύθμιση ρύθμισης παραμέτρων που χρησιμοποιείται για τον καθορισμό των λεπτομερειών σύνδεσης σε μια εξωτερική πηγή δεδομένων, όπως μια βάση δεδομένων ή ένα API. |
| Hostname              | απαιτείται  | Η διεύθυνση URL ή IP για τη βάση δεδομένων σας                                                                                                                      |
| Θύρα                  | απαιτείται  | Ο αριθμός θύρας του διακομιστή που θα πρέπει να χρησιμοποιήσετε για να συνδεθείτε. Αν δεν καθορίσετε μια θύρα, η προεπιλεγμένη θύρα είναι το '5432'.                |
| Βάση Δεδομένων        | απαιτείται  | Το όνομα της βάσης δεδομένων                                                                                                                                        |
| Όνομα Χρήστη          | απαιτείται  | το όνομα χρήστη που θέλετε να χρησιμοποιήσετε κατά τη σύνδεση στον εξυπηρετητή Neon.                                                                                |
| Κωδικός               | απαιτείται  | Χρησιμοποιήστε αυτόν τον κωδικό πρόσβασης για ταυτοποίηση.                                                                                                          |

Υπάρχουν δύο τρόποι για να συμπληρώσετε τις πληροφορίες για τη ρύθμιση παραμέτρων Neon.

-   Ανάλυση δοσμένης συμβολοσειράς σύνδεσης

Στη σελίδα του έργου Neon, η κύλιση έγινε, έχουμε μια δοσμένη συμβολοσειρά σύνδεσης σε **`Άμεση σύνδεση`**.

![neon_connection](https://cdn.illacloud.com/official-website/img/docs/neon_connection.png)

Σημείωση: Πατώντας το θόλωμα εμφανίζει τους κρυμμένους αριθμούς, τότε μπορούμε να τους αντιγράψουμε.

Μετά την επικόλληση της συμβολοσειράς στην περιοχή κειμένου της συμβολοσειράς σύνδεσης στις ρυθμίσεις, κάντε κλικ στο **`Ανάλυση`**.

![neon_parsing](https://cdn.illacloud.com/official-website/img/docs/neon_parsing.png)

Έχουμε όλες τις πληροφορίες που συμπληρώνονται αυτόματα για εμάς.

![neon_parse_done](https://cdn.illacloud.com/official-website/img/docs/neon_parse_done.png)

-   Συμπληρώστε χειροκίνητα τις πληροφορίες.

Μπορούμε επίσης να βρούμε τις απαιτούμενες πληροφορίες από τη σελίδα του έργου και να τις συμπληρώσουμε με μη αυτόματο τρόπο στις ρυθμίσεις στο Illa. 

### Δημιουργία Ενεργειών

Έχουμε δημιουργήσει έναν πόρο Neon, μπορούμε να προσθέσουμε την ενέργεια επιλέγοντας Neon από τη λίστα ενεργειών και επιλέγοντας το κουμπί **`Δημιουργία ενέργειας`**.

![neon_resource_list](https://cdn.illacloud.com/official-website/img/docs/neon_resource_list.png)

 Τώρα έχουμε προσθέσει τον εξυπηρετητή Neon ως ενέργεια στη σελίδα μας κτηρίου.

![neon](https://cdn.illacloud.com/official-website/img/docs/neon.png)

### Ρύθμιση Νέον

| Ιδιότητες       | Περιγραφή                                                                                     |
| --------------- | --------------------------------------------------------------------------------------------- |
| Ερώτημα SQL     | εντολές sql για λήψη (επιλογή), εισαγωγή, ενημέρωση, διαγραφή δεδομένων από τη βάση δεδομένων |
| Μετασχηματιστής | μετατρέποντας τα δεδομένα σε στυλ που σας αρέσει χρησιμοποιώντας JavaScript                   |

Χρήση παραδείγματος:

-   Επιλογή

```sql
SELECT * ΑΠΟ χρήστες ORDER BY επώνυμο;
```

-   Insert

```sql
INSERT INTO users

	(firstname, lastname, email)

VALUES

	(

		{{ NewUser.data.firstNameInput }},

		{{ NewUser.data.lastNameInput }},

		{{ NewUser.data.emailInput }}

	);
```

-   Ενημέρωση

```sql
UPDATE users
  SET firstname = '{{form1.updatedProfile.first }}'
  WHERE id = {{ form1.updatedProfile.uid }};
```

-   Διαγραφή

```sql
DELETE FROM users WHERE id = {{ form1.updatedProfile.uid }};
```

## <h2 hidden>Τι είναι το PostgreSQL?</h2>

PostgreSQL στο Illa είναι μια ενσωμάτωση βάσης δεδομένων που σας επιτρέπει να συνδεθείτε και να αλληλεπιδράσετε με μια βάση δεδομένων PostgreSQL. Το PostgreSQL είναι ένα δημοφιλές σχεσιακό σύστημα διαχείρισης βάσεων δεδομένων ανοιχτού κώδικα που είναι γνωστό για τη σταθερότητα, την αξιοπιστία και τα ισχυρά χαρακτηριστικά του.

Με την ενσωμάτωση PostgreSQL στο Illa, μπορείτε να θέσετε, να εισαγάγετε και να διαγράψετε δεδομένα από μια βάση δεδομένων PostgreSQL χρησιμοποιώντας εντολές SQL. Μπορείτε επίσης να δημιουργήσετε προσαρμοσμένα ερωτήματα SQL και να τα εκτελέσετε μέσα από την ενέργεια Illa. Αυτό καθιστά εύκολη την κατασκευή εφαρμογών που βασίζονται σε δεδομένα και πίνακες ελέγχου που εμφανίζουν δεδομένα σε πραγματικό χρόνο από μια βάση δεδομένων PostgreSQL.

### Δημιουργία PostgreSQL

Υπάρχουν δύο τρόποι για να δημιουργήσετε έναν πόρο στην Illa μετά την είσοδό σας στο λογαριασμό Illa.

-   Δημιουργία σε πόρους

Συνδεθείτε στον Illa λογαριασμό σας, επιλέξτε `**Πόροι**` στην κορυφή της σελίδας και κάντε κλικ στο κουμπί `**Δημιουργία Νέου**`.

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

Επιλέξτε **`PostgreSQL`** από τη λίστα της βάσης δεδομένων.

![action_list](https://cdn.illacloud.com/official-website/img/docs/action_list.png)

Συνδεθείτε στη βάση δεδομένων με τις απαιτούμενες παραμέτρους που περιγράφονται στις παρακάτω ρυθμίσεις σύνδεσης.

Κάντε κλικ στο κουμπί `**Δοκιμή Σύνδεσης**` για να δείτε αν μπορούμε να συνδεθείτε με επιτυχία στη βάση δεδομένων. Εάν ναι, κάντε κλικ στο **`Αποθήκευση πόρων`**, αλλιώς, ελέγξτε ξανά το όνομα κεντρικού υπολογιστή, τη θύρα, το όνομα χρήστη και τον κωδικό πρόσβασης είναι σωστά.

Μετά τη δημιουργία ενός πόρου, το έτοιμο PostgreSQL θα εμφανίζεται όπως εμφανίζεται.

![ex_post_ready](https://cdn.illacloud.com/official-website/img/docs/ex_post_ready.png)

-   Δημιουργία στο Builder

Συνδεθείτε στον Illa λογαριασμό σας, δημιουργήστε ένα έργο στο Illa Builder στη σελίδα `**App**`, και πλοηγηθείτε στο **`Λίστα Ενεργειών`** στο κάτω μέρος της σελίδας. Κάντε κλικ στο κουμπί `**new**`, και μετά επιλέξτε **`PostgreSQL`** από τη λίστα της βάσης δεδομένων. Στη συνέχεια, συνδεθείτε στη βάση δεδομένων με τις απαιτούμενες παραμέτρους που περιγράφονται στις "Ρυθμίσεις Σύνδεσης" παρακάτω.

Κάντε κλικ στο κουμπί `**Δοκιμή Σύνδεσης**` για να δείτε αν μπορούμε να συνδεθείτε με επιτυχία στη βάση δεδομένων. Εάν ναι, κάντε κλικ στο **`Αποθήκευση πόρων`**, αλλιώς, ελέγξτε ξανά το όνομα κεντρικού υπολογιστή, τη θύρα, το όνομα χρήστη και τον κωδικό πρόσβασης είναι σωστά.

### Ρυθμίσεις Σύνδεσης

Εδώ πρέπει να παρέχουμε πληροφορίες για τη σύνδεση με τη βάση δεδομένων PostgreSQL.

![post_config](https://cdn.illacloud.com/official-website/img/docs/post_config.png)

| Ιδιότητες         | Απαιτείται  | Περιγραφή                                                                                                                                            |
| ----------------- | ----------- | ---------------------------------------------------------------------------------------------------------------------------------------------------- |
| Όνομα             | απαιτείται  | Το όνομα του πόρου κατά τη δημιουργία ενεργειών στο ILLA.                                                                                            |
| Hostname          | απαιτείται  | Η διεύθυνση URL ή IP για τη βάση δεδομένων σας                                                                                                       |
| Θύρα              | απαιτείται  | Ο αριθμός θύρας του διακομιστή που θα πρέπει να χρησιμοποιήσετε για να συνδεθείτε. Αν δεν καθορίσετε μια θύρα, η προεπιλεγμένη θύρα είναι το '5432'. |
| Βάση Δεδομένων    | απαιτείται  | Το όνομα της βάσης δεδομένων                                                                                                                         |
| Όνομα Χρήστη      | απαιτείται  | το όνομα χρήστη που θέλετε να χρησιμοποιήσετε κατά τη σύνδεση στον διακομιστή PostgreSQL.                                                            |
| Κωδικός           | απαιτείται  | Χρησιμοποιήστε αυτόν τον κωδικό πρόσβασης για ταυτοποίηση.                                                                                           |
| Επιλογές SSL      | προαιρετικό | αποφασίζει πόσο υψηλή προτεραιότητα έχει μια ασφαλής σύνδεση SSL TCP/IP κατά τη διαπραγμάτευση με το διακομιστή.                                     |

### Δημιουργία Ενεργειών

Έχουμε δημιουργήσει έναν πόρο PostgreSQL, μπορούμε να προσθέσουμε την ενέργεια επιλέγοντας PostgreSQL από τη λίστα ενεργειών και επιλέγοντας το κουμπί **`Δημιουργία δράσης`**.

![post_resource_list](https://cdn.illacloud.com/official-website/img/docs/post_resource_list.png)

Τώρα έχουμε προσθέσει τον διακομιστή PostgreSQL ως ενέργεια στη σελίδα μας κτηρίου.

![post](https://cdn.illacloud.com/official-website/img/docs/post.png)

### Ρύθμιση PostgreSQL

| Ιδιότητες       | Περιγραφή                                                                                     |
| --------------- | --------------------------------------------------------------------------------------------- |
| Ερώτημα SQL     | εντολές sql για λήψη (επιλογή), εισαγωγή, ενημέρωση, διαγραφή δεδομένων από τη βάση δεδομένων |
| Μετασχηματιστής | μετατρέποντας τα δεδομένα σε στυλ που σας αρέσει χρησιμοποιώντας JavaScript                   |

Χρήση παραδείγματος:

-   Επιλογή

```sql
SELECT * FROM users ORDER BY lastname;
```

-   Insert

```sql
INSERT INTO users

	(firstname, lastname, email)

VALUES

	(

		{{ NewUser.data.firstNameInput }},

		{{ NewUser.data.lastNameInput }},

		{{ NewUser.data.emailInput }}

	);
```

-   Ενημέρωση

```sql
UPDATE users
  SET firstname = '{{form1.updatedProfile.first}}'
  WHERE id = {{ form1.updatedProfile.uid}};
```

-   Διαγραφή

```sql
DELETE FROM users WHERE id = {{ form1.updatedProfile.uid }};
```

## <h2 hidden>Τι είναι το Μενού?</h2>

Το στοιχείο μενού είναι ένα στοιχείο διεπαφής χρήστη που παρέχει μια δομή πλοήγησης για την πρόσβαση σε διαφορετικές λειτουργίες, επιλογές ή τμήματα μέσα στην εφαρμογή ILLA Cloud. Συνήθως αποτελείται από μια λίστα στοιχείων μενού ή συνδέσμους με τους οποίους οι χρήστες μπορούν να αλληλεπιδράσουν για να περιηγηθούν σε διάφορες σελίδες ή να εκτελέσουν συγκεκριμένες ενέργειες.

### Ιδιότητες

| Ιδιότητες   | Περιγραφή                                                                                                 |
| ----------- | --------------------------------------------------------------------------------------------------------- |
| Μενού       | πραγματικό περιεχόμενο και επιλογές που εμφανίζονται στην εφαρμογή μενού                                  |
| Hidden      | αποκρύπτει τη συνιστώσα της φόρμας από την προβολή του χρήστη                                             |
| Κατεύθυνση  | τον προσανατολισμό ή τη διάταξη της εφαρμογής μενού                                                       |
| Align       | την τοποθέτηση ή την ευθυγράμμιση της συνιστώσας μενού μέσα στο δοχείο ή το γονικό στοιχείο               |

### Δεδομένα

Το στοιχείο έχει κάποια κοινώς χρησιμοποιούμενα δεδομένα, τα οποία μπορούν να κληθούν μέσω του **`{{componentName.propertyName}}`** στην εφαρμογή.

| Όνομα ιδιότητας       | Περιγραφή                                                                                                                   |
| --------------------- | --------------------------------------------------------------------------------------------------------------------------- |
| Εμφάνιση Ονόματος     | αντιπροσωπεύει το όνομα ή την ετικέτα που έχει ανατεθεί στο στοιχείο                                                        |
| hidden                | αν η εφαρμογή μενού είναι ορατή ή απόκρυψη                                                                                  |
| οριζόντιαΣτοίχιση     | ελέγχει την οριζόντια τοποθέτηση ή ευθυγράμμιση της συνιστώσας μενού μέσα στο δοχείο ή το γονικό στοιχείο                   |
| αντικείμενα           | συλλογή στοιχείων μενού ή επιλογών που είναι διαθέσιμα στο στοιχείο μενού                                                   |
| λειτουργία            | πώς λειτουργεί το μενού ή αλληλεπιδρά με το χρήστη                                                                          |
| selectedValues        | διατηρεί τις τιμές των στοιχείων μενού που έχουν επιλεγεί ή επιλεγεί από τον χρήστη.                                        |

### Περίπτωση χρήσης

Θα δείξουμε πώς να φτιάξουμε μια φόρμα με το όνομα και την ηλικία ως είσοδο, καθώς και την πρόοδο της γραμμής για να δείξει την πρόοδο ολοκλήρωσης της φόρμας.

#### Βήμα 1 Ορίστε τη διάταξη των σελίδων

Μεταβείτε στη ρύθμιση παραμέτρων της σελίδας ως τα ακόλουθα βήματα：

![pages](https://cdn.illacloud.com/official-website/img/docs/pages.png)

Μπορείτε να χρησιμοποιήσετε τις προκαθορισμένες διατάξεις ή χειροκίνητα να προσθέσετε και να διαγράψετε ενότητες για να ορίσετε τη διάταξη.

#### Βήμα 2 Προσθήκη προβολών σε μια σελίδα

**`View path`**: μια συμβολοσειρά μετά το URL της σελίδας για να έχει πρόσβαση στην καθορισμένη προβολή. Μετά την ανάπτυξη της εφαρμογής, η πλήρης διεύθυνση πρόσβασης έχει ως εξής: https://builder.illacloud.com/{{team_identifier}}/deploy/app/{{app_id}}/{{page_name}}/{{view_path}}

Μπορείτε επίσης να εισάγετε την αρχική σελίδα και την προεπιλεγμένη προβολή μέσω της ακόλουθης διαδρομής:

https://builder.illacloud.com/{{team_identifier}}/deploy/app/{{app_id}}

![add_page](https://cdn.illacloud.com/official-website/img/docs/add_page.png)

#### Βήμα 3 Σύνδεση μενού και σελίδων

Πρόσθεσε ένα στοιχείο **`Menu`** στην κορυφή της σελίδας.

Προσθήκη χειριστών συμβάντων σε στοιχεία μενού → ορίστε την Ενέργεια για να ορίσετε δρομολογητή → επιλέξτε τη σελίδα και να προβάλετε

Για τα **`menu`** components, μπορούμε να τα ρυθμίσουμε ώστε να αλλάζουν μεταξύ σελίδων και προβολών

1.  Κάντε κλικ στο στοιχείο **`menu`** για να ανοίξετε τη σελίδα ελέγχου.
2.  Κάντε κλικ σε ένα αντικείμενο για να ανοίξετε τις ρυθμίσεις του.
3.  Κάτω από **`Χειριστής Συμβάντων`**, κάντε κλικ στο `**+ Νέο**`.
4.  Στο χειριστή συμβάντων επεξεργασίας, επιλέξτε **`Set Router`** εν ενεργεία, και επιλέξτε τη σελίδα και την προβολή που θέλετε να δρομολογήσετε ως στόχο της εκδήλωσης.

![menu_items](https://cdn.illacloud.com/official-website/img/docs/menu_items.jpg)

Μπορούμε να κάνουμε το ίδιο και για άλλες σελίδες και να δείτε ότι θέλετε να ταξιδέψετε.

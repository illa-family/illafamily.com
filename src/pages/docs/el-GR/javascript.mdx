## <h2 hidden>Οδηγός JavaScript</h2>

Η JavaScript είναι το πιο σημαντικό συστατικό της ILLA Builder, το οποίο είναι το εργαλείο που χρησιμοποιείτε για να δημιουργήσετε την πρόσβαση στα δεδομένα και την κατασκευή λογική του APP σας. Mastering πώς να καλέσετε JavaScript, έχετε κατακτήσει τη χρήση του ILLA Builder.

Όταν χτίζεις το APP σου στο ILLA Builder, μπορείς να χρησιμοποιήσεις το `{{template syntax}}` για να γράψεις προσαρμοσμένες δηλώσεις JavaScript για κλήσεις και χειρισμό δεδομένων, ή γράψτε κώδικα JS απευθείας στο μετασχηματιστή.

### Δεδομένα λειτουργίας

#### Πρόσβαση στα δεδομένα σε αντικείμενα

Στο ILLA Builder μπορείτε να εισάγετε εκφράσεις JavaScript σε οποιοδήποτε πλαίσιο εισαγωγής κειμένου χρησιμοποιώντας τη σύνταξη προτύπων. Αντικείμενα δεδομένων είναι δυνατότητα παραπομπής και μπορείτε να χειριστείτε τα δεδομένα όλων των παρεμβάσεων, ενεργειών και μετασχηματιστών μέσω JavaScript. Για παράδειγμα, μπορείτε να χρησιμοποιήσετε το `{{textInput.data}}` για να αποκτήσετε πρόσβαση στο περιεχόμενο ενός ένθετου εισαγωγής κειμένου, `{{action.data}}` για πρόσβαση στα δεδομένα από μια βάση δεδομένων ενεργειών και `{{transformer.value}}` για πρόσβαση στα δεδομένα από έναν μετασχηματιστή.

Έχουμε προσθέσει και τροποποιήσει το ένθετο καθρέφτη κώδικα στο προϊόν, το προσάρμοσε στο περιβάλλον προγραμματισμού κατασκευαστών ILLA, υποστηριζόμενο τονισμό σύνταξης και αυτόματη εσοχή, και πρόσθεσε λεπτομερείς εξηγήσεις μεταβλητών και λειτουργιών που συμβαίνουν κατά τη διάρκεια του προγραμματισμού, έτσι ώστε να μπορείτε να μάθετε πώς να χρησιμοποιήσετε ILLA Builder σε χαμηλό κόστος. Αφού εισαγάγετε το όνομα του αντικειμένου, πληκτρολογήστε μια τελεία. Η προτροπή θα περιέχει όλες τις ιδιότητες του αντικειμένου, τον τύπο δεδομένων και μια μερική περιγραφή κειμένου, ώστε να μπορείτε να αναγνωρίσετε την ιδιότητα που θα έχει πρόσβαση σε χρόνο μηδέν.

#### Πρόσβαση σε αριθμούς σε πίνακες

Όταν πρέπει να έχετε πρόσβαση σε μια τιμή σε ένα συγκεκριμένο πίνακα, μπορείτε να πάρετε την τιμή από τη θέση ή τη συνδρομή των δεδομένων που χρειάζεστε για την πρόσβαση.

```javascript
const list = ['a', 'b', 'c']
return list[0] // Return the first value in list 'a'
```

### Προβολή δεδομένων

Υπάρχουν τέσσερις τοποθεσίες για να δείτε τα δεδομένα στο ILLA Builder: ο χώρος εργασίας δεδομένων στο αριστερό πάνελ, η προτροπή δεδομένων κάτω από το πλαίσιο κειμένου μετά την εισαγωγή μιας έκφρασης JavaScript και την εμφάνιση δεδομένων μέσω του ένθετου δεδομένων για την εμφάνιση των δεδομένων.

#### Παρουσίαση χώρου εργασίας δεδομένων

Μπορείτε να βρείτε το χώρο εργασίας δεδομένων ανοίγοντας τον αριστερό πίνακα στον επεξεργαστή εφαρμογών, έτσι τα δεδομένα παρουσιάζονται ως δομημένη JSON στην οποία κάθε ιδιότητα και λίστα μπορεί να επεκταθεί και όλα τα δεδομένα εμφανίζονται οπτικά στο χώρο εργασίας.

### Μετασχηματιστής

Μπορείτε να γράψετε πολλαπλές γραμμές κώδικα JS στον μετασχηματιστή, και σας συνιστούμε να επιλέξετε να γράψετε στο μετασχηματιστή και όχι σε διπλά σγουρά στηρίγματα. Μπορείτε να προγραμματίσετε JavaScript απευθείας στον μετασχηματιστή καθώς και να καλέσετε τα δεδομένα στη λειτουργία με τη βοήθεια του `{}}`.

Κάντε κλικ εδώ για λεπτομερή χρήση του μετασχηματιστή.

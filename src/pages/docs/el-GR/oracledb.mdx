## <h2 hidden>Τι είναι το Oracle DB?</h2>

Oracle DB στην Illa είναι μια ενσωμάτωση της βάσης δεδομένων που σας επιτρέπει να συνδεθείτε και να αλληλεπιδράσετε με μια βάση δεδομένων Oracle DB. Oracle DB είναι ένα δημοφιλές συσχετιακό σύστημα διαχείρισης βάσεων δεδομένων ανοικτού κώδικα που είναι γνωστό για την ταχύτητα, την αξιοπιστία και την ευκολία χρήσης.

Με την ενσωμάτωση της Oracle DB στην Illa, μπορείτε να ρωτήσετε, να εισαγάγετε και να διαγράψετε δεδομένα από μια βάση δεδομένων Oracle DB χρησιμοποιώντας SQL εντολές. Μπορείτε επίσης να δημιουργήσετε προσαρμοσμένα ερωτήματα SQL και να τα εκτελέσετε μέσα Illa. Αυτό καθιστά εύκολο να οικοδομήσουμε τις εφαρμογές δεδομένων και τους πίνακες ελέγχου που εμφανίζουν δεδομένα σε πραγματικό χρόνο από μια βάση δεδομένων Oracle DB.

### Δημιουργία Oracle DB

Υπάρχουν δύο τρόποι για να δημιουργήσετε έναν πόρο στην Illa μετά την είσοδό σας στο λογαριασμό Illa. 

-   Δημιουργία σε πόρους

Συνδεθείτε στον Illa λογαριασμό σας, επιλέξτε `**Πόροι**` στην κορυφή της σελίδας και κάντε κλικ στο κουμπί `**Δημιουργία Νέου**`. 

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

Επιλέξτε **`Oracle DB`** από τη λίστα της βάσης δεδομένων.

![action_list](https://cdn.illacloud.com/official-website/img/docs/action_list.png)

Συνδεθείτε στη βάση δεδομένων με τις απαιτούμενες παραμέτρους που περιγράφονται στις παρακάτω ρυθμίσεις σύνδεσης. 

Κάντε κλικ στο κουμπί `**Δοκιμή Σύνδεσης**` για να δείτε αν μπορούμε να συνδεθείτε με επιτυχία στη βάση δεδομένων. Εάν ναι, κάντε κλικ στο **`Αποθήκευση πόρων`**, αλλιώς, ελέγξτε ξανά το όνομα κεντρικού υπολογιστή, τη θύρα, το όνομα χρήστη και τον κωδικό πρόσβασης είναι σωστά.

Μετά τη δημιουργία ενός πόρου, το έτοιμο Oracle DB θα εμφανίζεται όπως εμφανίζεται.

![oracle_ex_ready](https://cdn.illacloud.com/official-website/img/docs/oracle_ex_ready.png)

-   Δημιουργία στο Builder

Συνδεθείτε στον Illa λογαριασμό σας, δημιουργήστε ένα έργο στο Illa Builder στη σελίδα `**App**`, και πλοηγηθείτε στο **`Λίστα Ενεργειών`** στο κάτω μέρος της σελίδας. Κάντε κλικ στο κουμπί `**new**`, στη συνέχεια επιλέξτε **`Oracle DB`** από τη λίστα της βάσης δεδομένων. Στη συνέχεια, συνδεθείτε στη βάση δεδομένων με τις απαιτούμενες παραμέτρους που περιγράφονται στις "Ρυθμίσεις Σύνδεσης" παρακάτω.

Κάντε κλικ στο κουμπί `**Δοκιμή Σύνδεσης**` για να δείτε αν μπορούμε να συνδεθείτε με επιτυχία στη βάση δεδομένων. Εάν ναι, κάντε κλικ στο **`Αποθήκευση πόρων`**, αλλιώς, ελέγξτε ξανά το όνομα κεντρικού υπολογιστή, τη θύρα, το όνομα χρήστη και τον κωδικό πρόσβασης είναι σωστά.

### Ρυθμίσεις Σύνδεσης

Εδώ πρέπει να παρέχουμε πληροφορίες για τη σύνδεση με τη βάση δεδομένων Oracle DB.

![oracle_config](https://cdn.illacloud.com/official-website/img/docs/oracle_config.png)

| Ιδιότητες                                | Απαιτείται  | Περιγραφή                                                                                                                                         |
| ---------------------------------------- | ----------- | ------------------------------------------------------------------------------------------------------------------------------------------------- |
| Όνομα                                    | απαιτείται  | Το όνομα του πόρου κατά τη δημιουργία ενεργειών στο ILLA.                                                                                         |
| Hostname                                 | απαιτείται  | Η διεύθυνση URL ή IP για τη βάση δεδομένων σας                                                                                                    |
| Θύρα                                     | απαιτείται  | Ο αριθμός θύρας του διακομιστή που θα πρέπει να χρησιμοποιήσετε για να συνδεθείτε. Αν δεν καθορίσετε μια θύρα, η προεπιλεγμένη θύρα είναι '3306'. |
| Βάση Δεδομένων                           | απαιτείται  | Το όνομα της βάσης δεδομένων                                                                                                                      |
| Αντιμετώπιση ονόματος βάσης δεδομένων ως | απαιτείται  | προσδιορίζει μια παρουσία βάσης δεδομένων με SID ή όνομα υπηρεσίας.                                                                               |
| Όνομα Χρήστη                             | απαιτείται  | το όνομα χρήστη που θέλετε να χρησιμοποιήσετε κατά τη σύνδεση στον εξυπηρετητή Oracle DB.                                                         |
| Κωδικός                                  | απαιτείται  | Χρησιμοποιήστε αυτόν τον κωδικό πρόσβασης για ταυτοποίηση.                                                                                        |
| SSL                                      | απαιτείται  | αποφασίζει πόσο υψηλή προτεραιότητα έχει μια ασφαλής σύνδεση SSL TCP/IP κατά τη διαπραγμάτευση με το διακομιστή.                                  |

### Δημιουργία Ενεργειών

Έχουμε δημιουργήσει έναν πόρο Oracle DB, μπορούμε να προσθέσουμε την ενέργεια επιλέγοντας Oracle DB από τη λίστα ενεργειών και επιλέγοντας το κουμπί **`Δημιουργία ενέργειας`**.

![oracle_resource_list](https://cdn.illacloud.com/official-website/img/docs/oracle_resource_list.png)

 Τώρα έχουμε προσθέσει τον εξυπηρετητή Oracle DB ως ενέργεια στη σελίδα μας κτηρίου.

![oracle](https://cdn.illacloud.com/official-website/img/docs/oracle.png)

### Ρύθμιση Oracle DB

| Ιδιότητες       | Περιγραφή                                                                                     |
| --------------- | --------------------------------------------------------------------------------------------- |
| Ερώτημα SQL     | εντολές sql για λήψη (επιλογή), εισαγωγή, ενημέρωση, διαγραφή δεδομένων από τη βάση δεδομένων |
| Μετασχηματιστής | μετατρέποντας τα δεδομένα σε στυλ που σας αρέσει χρησιμοποιώντας JavaScript                   |

Χρήση παραδείγματος:

-   Επιλογή

```sql
SELECT * ΑΠΟ χρήστες ORDER BY επώνυμο;
```

-   Insert

```sql
INSERT INTO users

	(firstname, lastname, email)

VALUES

	(

		{{ NewUser.data.firstNameInput }},

		{{ NewUser.data.lastNameInput }},

		{{ NewUser.data.emailInput }}

	);
```

-   Ενημέρωση

```sql
UPDATE users
  SET firstname = '{{form1.updatedProfile.first}}'
  WHERE id = {{ form1.updatedProfile.uid}};
```

-   Διαγραφή

```sql
DELETE FROM users WHERE id = {{ form1.updatedProfile.uid }};
```

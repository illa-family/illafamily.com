## <h2 hidden>Τι είναι η εικόνα?</h2>

Το στοιχείο της εικόνας στο Cloud ILLA είναι ένα ευέλικτο εργαλείο που επιτρέπει στους χρήστες να προσθέτουν και να εμφανίζουν εικόνες μέσα στις εφαρμογές τους. Παρέχει διάφορες λειτουργίες για την ενίσχυση της οπτικής εμπειρίας για τους χρήστες.

### Ιδιότητες

| Ιδιότητες                | Περιγραφή                                                                                                                                              |
| ------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Πηγή εικόνας             | καθορίστε την πηγή της εικόνας που θα εμφανιστεί                                                                                                       |
| Εναλλακτικό κείμενο      | εναλλακτικό κείμενο για την εικόνα. Εμφανίζεται όταν η εικόνα δεν μπορεί να φορτωθεί ή για χρήστες με προβλήματα όρασης                                |
| Τύπος Κλίμακας           | πώς η εικόνα κλιμακώνεται εντός των διαστάσεων του στοιχείου, συμπεριλαμβανομένου του δοχείου, του καλύμματος, γεμίστε, κανένα, κάτω.                  |
| Χειριστής Συμβάντων      | επιτρέποντας στους χρήστες να καθορίζουν ενέργειες που προκαλούνται από αλληλεπιδράσεις του χρήστη με την εικόνα                                       |
| Υπόδειξη                 | προσθέστε μια υπόδειξη ή ένα περιγραφικό κείμενο που εμφανίζεται όταν ο χρήστης αιωρείται πάνω από την εικόνα                                          |
| Hidden                   | έλεγχο της ορατότητας του στοιχείου της εικόνας.                                                                                                       |
| Styles                   | επιτρέποντας στους χρήστες να προσαρμόσουν την εμφάνισή του. Οι χρήστες μπορούν να εφαρμόσουν στυλ όπως ακτίνα                                         |

### Μέθοδος

Μπορείτε να χρησιμοποιήσετε άλλα στοιχεία για τον έλεγχο της εφαρμογής. Υποστηρίζουμε τις ακόλουθες μεθόδους:

-   **setImageUrl**

αλλάζει δυναμικά την πηγή ή το URL της εικόνας.

| Ιδιότητες    | Περιγραφή                  |
| ------------ | -------------------------- |
| Url Εικόνας  | URL πηγαίου κώδικα εικόνας |

### Χειριστής συμβάντων

| Γεγονός | Περιγραφή                                                                               |
| ------- | --------------------------------------------------------------------------------------- |
| Click   | εκτέλεση συγκεκριμένων ενεργειών ή εκτέλεση κώδικα όταν πατηθεί το στοιχείο της εικόνας |

### Δεδομένα

Το στοιχείο έχει κάποια κοινώς χρησιμοποιούμενα δεδομένα, τα οποία μπορούν να κληθούν μέσω του **`{{componentName.propertyName}}`** στην εφαρμογή.

| Όνομα ιδιότητας     | Περιγραφή                                                                                                                                     |
| ------------------- | --------------------------------------------------------------------------------------------------------------------------------------------- |
| altText             | εναλλακτική λύση βασισμένη σε κείμενο που περιγράφει το περιεχόμενο ή το σκοπό της εικόνας                                                    |
| Εμφάνιση Ονόματος   | Μια συμβολοσειρά που αναπαριστά το όνομα ή τον τίτλο της εικόνας.                                                                             |
| συμβάντα            | Ένας πίνακας ή ένα αντικείμενο που καθορίζει τους χειριστές γεγονότων που σχετίζονται με το στοιχείο εικόνας, όπως κάντε κλικ                 |
| hidden              | Μια δυαδική τιμή που καθορίζει αν το στοιχείο της εικόνας είναι κρυμμένο από τη διεπαφή χρήστη.                                               |
| imageSrc            | την πηγή ή τη θέση της εικόνας που θα εμφανίζεται στο στοιχείο                                                                                |
| objectFit           | πώς η εικόνα τοποθετείται ή κλιμακώνεται εντός των διαστάσεων του κατασκευαστικού στοιχείου                                                   |
| radius              | εφαρμόστε στρογγυλεμένες γωνίες στο στοιχείο εικόνας.                                                                                         |
| tooltipText         | τιμή συμβολοσειράς που αναπαριστά την υπόδειξη ή περνάει το κείμενο που θα εμφανιστεί όταν ο χρήστης αιωρείται πάνω από την εφαρμογή εικόνας. |

### Περίπτωση χρήσης

Στη συνέχεια, θα δείξουμε πώς να ελέγξετε την οθόνη εικόνας με ένα κουμπί.

#### Βήμα 1 Προσθήκη Στοιχείων

Πρόσθεσε ένα στοιχείο **`Image`** και ένα **κουμπί \`\`** στον καμβά.

Μπορείτε να ορίσετε την "πηγή εικόνας" του στοιχείου της εικόνας σε οποιαδήποτε διεύθυνση εικόνας θέλετε να εμφανίσετε. Εδώ θα χρησιμοποιήσουμε την προεπιλεγμένη εικόνα ως παράδειγμα έναρξης και θα επισημάνουμε το κουμπί ως "Αλλαγή" όπως φαίνεται παρακάτω

![image_layout](https://cdn.illacloud.com/official-website/img/official-site/components/image_layout.jpg)

#### Βήμα 2 Διαμόρφωση του εξαρτήματος

Για την εφαρμογή **`button`** μπορούμε να την ρυθμίσουμε ώστε να ορίσει την πηγή εικόνας του στοιχείου **`image`** σε μια νέα διεύθυνση εικόνας.

1.  Κάντε κλικ στο στοιχείο `****` για να ανοίξετε τη σελίδα ελέγχου. Κάτω από **`Χειριστής Συμβάντων`**, κάντε κλικ στο `**+ Νέο**`.
2.  Στο χειριστή συμβάντων επεξεργασίας, επιλέξτε **`Control component`** εν ενεργεία, και επιλέξτε το **`image1`** component που θέλετε να αλλάξετε ως στόχο του γεγονότος.
3.  Επιλέξτε την **`setImageUrl`** ενέργεια
4.  Τοποθετήστε τη νέα διεύθυνση εικόνας στο πεδίο **`Value`**

![image_config](https://cdn.illacloud.com/official-website/img/official-site/components/image_config.jpg)

#### Στάδιο 3 Δοκιμή

Τώρα όταν κάνετε κλικ στο κουμπί "Αναπαραγωγή", η εικόνα θα πρέπει να αλλάξει

![image_test](https://cdn.illacloud.com/official-website/img/official-site/components/image_test.gif)

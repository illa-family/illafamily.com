## <h2 hidden>Φτιάξτε τις εφαρμογές σας</h2>

Με την εκτεταμένη βιβλιοθήκη των εξαρτημάτων UI, μπορείτε γρήγορα και εύκολα να δημιουργήσετε πολύ γυαλισμένες διεπαφές. Μπορείτε να γράψετε ερωτήματα για να αλληλεπιδράσετε με τα δεδομένα και να χρησιμοποιήσετε JavaScript σε πολλά διαφορετικά μέρη της εφαρμογής σας.

### Επεξεργαστής

Κατασκευάστε, αναπτύξτε και εκτελέστε εφαρμογές με το περιβάλλον ανάπτυξης βασισμένο στο διαδίκτυο της ILLA

### Σχεδιασμός και χρηστικότητα (σελίδες, μετασχηματιστές)

Εξατομίκευση εφαρμογών με προσαρμοσμένα στυλ και μορφοποίηση

### Χειριστές Συμβάντων

Οι χειριστές συμβάντων σας επιτρέπουν να ενεργοποιείτε ερωτήματα, να ελέγχετε στοιχεία και να εκτελείτε άλλες ενέργειες σε απάντηση στις αλληλεπιδράσεις του χρήστη

## Συγκέντρωση Συστατικών

Χρησιμοποιήστε τη βιβλιοθήκη των συστατικών UI της ILLA για να δημιουργήσετε το περιβάλλον χρήστη της εφαρμογής σας.

Τα στοιχεία είναι ένα ουσιαστικό μέρος της ILLA, και λειτουργούν ομοίως με τα React components στο ότι μπορούν να δεχτούν ιδιότητες και να έχουν εσωτερική κατάσταση. Συνδυάζοντας τη βιβλιοθήκη συστατικών της ILLA με τα δεδομένα σας, μπορείτε να δημιουργήσετε διαδραστικές εφαρμογές.

Μερικά κοινά συστατικά περιλαμβάνουν:

-   Το στοιχείο του πίνακα χρησιμοποιείται για την εμφάνιση δεδομένων σε μορφή πίνακα, και προσφέρει χαρακτηριστικά όπως η ταξινόμηση, το φιλτράρισμα, η σελιδοποίηση και οι εξαγωγές CSV.
-   Η συνιστώσα Φόρμας χρησιμοποιείται για την ομαδοποίηση των πεδίων εισόδου και την υποβολή τους ως μία μόνο μονάδα.
-   Το στοιχείο Επιλογή εισόδου είναι ένας συνδυασμός ενός αναπτυσσόμενου μενού και ένα πεδίο εισόδου που επιτρέπει στους χρήστες να επιλέγουν είτε μια τιμή από μια λίστα ή να εισάγετε μια τιμή απευθείας ως κείμενο.
-   Το στοιχείο του χάρτη χρησιμοποιείται για να δημιουργήσει οπτικοποίηση των δεδομένων χρησιμοποιώντας διαγράμματα και γραφήματα.

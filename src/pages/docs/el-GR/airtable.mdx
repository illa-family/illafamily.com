## <h2 hidden>Τι είναι το Airtable?</h2>

Το Airtable είναι μια δημοφιλής βάση δεδομένων σχεσιακής βάσης δεδομένων και υπολογιστικών φύλλων που συνδυάζει την απλότητα ενός υπολογιστικού φύλλου με την πολυπλοκότητα μιας παραδοσιακής βάσης δεδομένων. Επιτρέπει στους χρήστες να δημιουργήσουν και να οργανώσουν δομημένα δεδομένα σε ένα φιλικό προς το χρήστη περιβάλλον, καθιστώντας το χρήσιμο για διάφορες εργασίες όπως η διαχείριση έργου, ο σχεδιασμός περιεχομένου και η συλλογή δεδομένων.

### Ρυθμίστε το Πίσω Τέλος στο Airtable

Δημιουργήστε ένα λογαριασμό στο `Airtable` και δημιουργήστε τη βάση σας ανάλογα με την ανάγκη σας.

Για να συνδέσετε τη βάση σας στο ILLA Cloud, μεταβείτε στο `developer hub`.

![air_developer_hub](https://cdn.illacloud.com/official-website/img/official-site/intergration/air_developer_hub.png)

Στη σελίδα `Personal Access Token`, κάντε κλικ στο κουμπί `Δημιουργία νέου Token`.

![air_create_new](https://cdn.illacloud.com/official-website/img/official-site/intergration/air_create_new.jpeg)

Αυτό θα ανοίξει τις επιλογές ρύθμισης παραμέτρων για το PAT που δημιουργείτε.

Όνομα το PAT, στη συνέχεια, επιλέξτε τα πεδία που θα θέλατε να έχετε πρόσβαση σε αυτό το PAT . Τέλος, επιλέξτε ποιο επίπεδο πρόσβασης θα έχει αυτή η ΠΑΤ.

![air_new_token](https://cdn.illacloud.com/official-website/img/official-site/intergration/air_new_token.jpeg)

Κάντε κλικ στο **`create token`** κουμπί, τώρα έχουμε ένα PAT έτοιμο για σύνδεση.

### Δημιουργία Airtable στο σύννεφο ILLA

Υπάρχουν δύο τρόποι για να δημιουργήσετε έναν πόρο στην Illa μετά την είσοδό σας στο λογαριασμό Illa.

-   Δημιουργία σε πόρους

Συνδεθείτε στον Illa λογαριασμό σας, επιλέξτε `**Πόρες**` στο πάνω μέρος της σελίδας και κάντε κλικ στο κουμπί `**Δημιουργία Νέου**`.

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

Επιλέξτε **`Airtable`** από τη λίστα της βάσης δεδομένων.

![action_list_3](https://cdn.illacloud.com/official-website/img/official-site/intergration/action_list_3.jpeg)

Συνδεθείτε στη βάση δεδομένων με τις απαιτούμενες παραμέτρους που περιγράφονται στις παρακάτω ρυθμίσεις σύνδεσης. Σε αυτή την περίπτωση, συμπληρώστε το **`Personal Access Token`** που αντιγράψαμε από το `Airtable` και επικολλήστε το σε **`Access Token`** στις ρυθμίσεις.

Κάντε κλικ στο κουμπί **`Αποθήκευση πόρων`** στο \*\*\*\*\*Δημιουργία πόρου.

Το έτοιμο `Airtable` θα εμφανιστεί ως εμφανιζόμενο.

![air_ex_ready](https://cdn.illacloud.com/official-website/img/official-site/intergration/air_ex_ready.jpeg)

-   Δημιουργία στο Builder

Συνδεθείτε στον Illa λογαριασμό σας, δημιουργήστε ένα έργο στο Illa Builder στη σελίδα `**App**`, και πλοηγηθείτε στο **`Λίστα Ενεργειών`** στο κάτω μέρος της σελίδας. Κάντε κλικ στο κουμπί `**new**`, στη συνέχεια, επιλέξτε **`Airtable`** από τη λίστα της βάσης δεδομένων. Στη συνέχεια, συνδεθείτε στη βάση δεδομένων με τις απαιτούμενες παραμέτρους που περιγράφονται στις "Ρυθμίσεις Σύνδεσης" παρακάτω.

Κάντε κλικ στο κουμπί `**Δοκιμή Σύνδεσης**` για να δείτε αν μπορούμε να συνδεθείτε με επιτυχία στη βάση δεδομένων. Εάν ναι, κάντε κλικ στο **`Αποθήκευση πόρων`**, αλλιώς, ελέγξτε ξανά το όνομα κεντρικού υπολογιστή, τη θύρα, το όνομα χρήστη και τον κωδικό πρόσβασης είναι σωστά.

### Ρυθμίσεις Σύνδεσης

Εδώ πρέπει να παρέχουμε πληροφορίες για τη σύνδεση με το Airtable API.

![air_config](https://cdn.illacloud.com/official-website/img/official-site/intergration/air_config.jpeg)

| Ιδιότητες            | Απαιτείται | Περιγραφή                                                                                                                                                                 |
| -------------------- | ---------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Όνομα                | απαιτείται | Το όνομα του πόρου κατά τη δημιουργία ενεργειών στο ILLA.                                                                                                                 |
| Διακριτικό πρόσβασης | απαιτείται | μια πληροφορία που παρέχει άδεια σε μια εφαρμογή ή υπηρεσία τρίτου μέρους για πρόσβαση στα δεδομένα του λογαριασμού σας του Airtable και εκτέλεση ενεργειών εκ μέρους σας |

### Δημιουργία Ενεργειών

Δημιουργήσαμε έναν Airtable πόρο, μπορούμε να προσθέσουμε την ενέργεια επιλέγοντας Airtable από τη λίστα ενεργειών και επιλέγοντας το κουμπί **`Δημιουργία δράσης`**.

![air_resource_list](https://cdn.illacloud.com/official-website/img/official-site/intergration/air_resource_list.jpeg)

Τώρα έχουμε προσθέσει το Airtable API ως ενέργεια στη σελίδα μας κτηρίου.

![air](https://cdn.illacloud.com/official-website/img/official-site/intergration/air.jpeg)

#### Εγγραφές Λίστας

ανακτά μια λίστα εγγραφών από έναν καθορισμένο πίνακα στο Airtable

** Input **

| Ιδιότητες               | Απαιτείται  | Περιγραφή                                                                                                      |
| ----------------------- | ----------- | -------------------------------------------------------------------------------------------------------------- |
| Προβολή                 | προαιρετικό | οργάνωση και εμφάνιση εγγραφών με βάση συγκεκριμένα κριτήρια                                                   |
| Πεδία                   | προαιρετικό | Τα πεδία αναφέρονται στις στήλες του πίνακα Airtable                                                           |
| Φιλτράρισμα ανά τύπο    | προαιρετικό | ορίστε έναν τύπο που φιλτράρει τις εγγραφές που θέλετε να ανακτήσετε ή να αλληλεπιδράσετε με                   |
| Ταξινόμηση              | προαιρετικό | τη σειρά με την οποία παρουσιάζονται ή ανακτώνται οι εγγραφές                                                  |
| Μέγιστες εγγραφές       | προαιρετικό | περιορίζει τον αριθμό των εγγραφών που σας φέρνει ή αλληλεπιδρά σε ένα μόνο αίτημα                             |
| Μέγεθος σελίδας         | προαιρετικό | συχνά χρησιμοποιούνται σε σενάρια όπου τα δεδομένα πρέπει να ανακτηθούν σε πολλαπλές παρτίδες (σελίδες)        |
| Μετατόπιση              | προαιρετικό | καθορίζει το σημείο εκκίνησης για την ανάκτηση εγγραφών σε μια συγκεκριμένη σελίδα                             |
| Μορφή κυψέλης           | προαιρετικό | πώς εμφανίζονται τα δεδομένα σε ένα συγκεκριμένο πεδίο                                                         |
| Ζώνη ώρας               | προαιρετικό | διασφαλίζει ότι οι υπολογισμοί ημερομηνίας και ώρας και παρουσίασης είναι ακριβείς και συνεπείς                |
| Τοποθεσία χρήστη        | προαιρετικό | να προσαρμόζει την έξοδο ενσωμάτωσης με βάση τη γεωγραφική θέση ή τις προτιμήσεις του χρήστη                   |

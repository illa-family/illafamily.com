## <h2 hidden>Τι είναι το PDF?</h2>

Το στοιχείο PDF είναι ένα χαρακτηριστικό ή στοιχείο που επιτρέπει στους χρήστες να βλέπουν και να αλληλεπιδρούν με έγγραφα PDF εντός της εφαρμογής. Παρέχει λειτουργικότητα για την απόδοση, την πλοήγηση και το χειρισμό αρχείων PDF, επιτρέποντας στους χρήστες να έχουν πρόσβαση και να εργάζονται με περιεχόμενο PDF απρόσκοπτα.

### Ιδιότητες

| Ιδιότητες                  | Περιγραφή                                                                                                 |
| -------------------------- | --------------------------------------------------------------------------------------------------------- |
| Πηγή PDF                   | προσδιορίστε την πηγή του pdf που θα εμφανίζεται                                                          |
| Εμφάνιση γραμμής εργαλείων | ελέγχει την ορατότητα της γραμμής εργαλείων μέσα στο στοιχείο PDF                                         |
| Υπόδειξη                   | προσθέστε μια υπόδειξη ή ένα περιγραφικό κείμενο που εμφανίζεται όταν ο χρήστης αιωρείται πάνω από το pdf |
| Κλιμάκωση σελίδας σε       | καθορίζει τη συμπεριφορά κλιμάκωσης των σελίδων PDF μέσα στο στοιχείο                                     |
| Hidden                     | καθορίζει την ορατότητα του στοιχείου PDF                                                                 |

### Μέθοδος

Μπορείτε να χρησιμοποιήσετε άλλα στοιχεία για τον έλεγχο της εφαρμογής. Υποστηρίζουμε τις ακόλουθες μεθόδους:

-   **setFileUrl**

αλλαγή δυναμικά της πηγής ή URL του αρχείου pdf.

| Ιδιότητες | Περιγραφή          |
| --------- | ------------------ |
| Πηγή PDF  | URL προέλευσης pdf |

### Δεδομένα

Το στοιχείο έχει κάποια κοινώς χρησιμοποιούμενα δεδομένα, τα οποία μπορούν να κληθούν μέσω του **`{{componentName.propertyName}}`** στην εφαρμογή.

| Όνομα ιδιότητας         | Περιγραφή                                                                                                                                   |
| ----------------------- | ------------------------------------------------------------------------------------------------------------------------------------------- |
| Εμφάνιση Ονόματος       | Μια συμβολοσειρά που αναπαριστά το όνομα ή τον τίτλο του pdf.                                                                               |
| ύψος                    | την κατακόρυφη διάσταση ή το μέγεθος της συνιστώσας PDF                                                                                     |
| hidden                  | Μια δυαδική τιμή που καθορίζει αν το στοιχείο pdf είναι κρυμμένο από τη διεπαφή χρήστη.                                                     |
| Λειτουργία κλίμακας     | καθορίζει τον τρόπο με τον οποίο το έγγραφο PDF κλιμακώνεται ή χωράει εντός της διαθέσιμης περιοχής εμφάνισης του στοιχείου                 |
| showToolBar             | αν εμφανίζεται μια γραμμή εργαλείων ή ένας πίνακας ελέγχου μαζί με το στοιχείο PDF                                                          |
| tooltipText             | τιμή συμβολοσειράς που αναπαριστά την υπόδειξη ή υπερκαλύπτει το κείμενο που εμφανίζεται όταν ο χρήστης αιωρείται πάνω από το στοιχείο pdf. |
| url                     | πηγή του αρχείου PDF που θα εμφανίζεται στο στοιχείο                                                                                        |
| width                   | την οριζόντια διάσταση ή το μέγεθος της συνιστώσας PDF                                                                                      |

Παράδειγμα: {{pdf1.displayName}}

### Περίπτωση χρήσης

Στη συνέχεια, θα δείξουμε πώς να ελέγξετε την οθόνη pdf με ένα κουμπί.

#### Βήμα 1 Προσθήκη Στοιχείων

Πρόσθεσε ένα στοιχείο **`PDF`** και ένα **κουμπί \`\`** στον καμβά.

![pdf_layout](https://cdn.illacloud.com/official-website/img/docs/pdf_layout.jpg)

#### Βήμα 2 Διαμόρφωση του εξαρτήματος

Για το στοιχείο **`button`** μπορούμε να το ρυθμίσουμε ώστε η πηγή εικόνας του στοιχείου **`PDF`** να είναι μια διεύθυνση pdf.

1.  Κάντε κλικ στο στοιχείο `****` για να ανοίξετε τη σελίδα ελέγχου. Κάτω από **`Χειριστής Συμβάντων`**, κάντε κλικ στο `**+ Νέο**`.
2.  Στο χειριστή συμβάντων επεξεργασίας, επιλέξτε **`Control component`** εν ενεργεία, και επιλέξτε το στοιχείο **`PDF`** που θέλετε να αλλάξετε ως στόχο του γεγονότος.
3.  Επιλέξτε την **`setFileUrl`** ενέργεια
4.  Τοποθετήστε τη νέα διεύθυνση εικόνας στο πεδίο **`Value`**

![pdf_config](https://cdn.illacloud.com/official-website/img/docs/pdf_config.jpg)

#### Στάδιο 3 Δοκιμή

Τώρα όταν κάνετε κλικ στο κουμπί "Αναπαραγωγή", η εικόνα θα πρέπει να αλλάξει

![pdf_test](https://cdn.illacloud.com/official-website/img/docs/pdf_test.jpg)

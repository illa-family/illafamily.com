## <h2 hidden>Συνεργασία με μέλη</h2>

Οι προγραμματιστές μπορούν να εργαστούν ταυτόχρονα στο ILLA Builder μοιράζοντας συνδέσμους. Οι προγραμματιστές μπορούν να συν-επεξεργάζονται και να επικοινωνούν με άλλους σε πραγματικό χρόνο για να δώσουν έγκαιρη ανατροφοδότηση, η οποία θα βελτιώσει σημαντικά την αποδοτικότητα της ανάπτυξης.

### Πρόσκληση μελών

Ο Ιδιοκτήτης ή Διαχειριστής της ομάδας μπορεί να προσκαλέσει χρήστες μέσω του κουμπιού Πρόσκληση στην αρχική σελίδα. 

<img src='https://cdn.illacloud.com/official-website/img/docs/assemble/collaboration.png' alt="" />

Στο αναδυόμενο παράθυρο, ο ιδιοκτήτης ή ο διαχειριστής μπορεί να επιλέξει τον τύπο πρόσβασης της πρόσκλησης και να καλέσει τα memebers να συνεργαστούν μέσω email ή μέσω συνδέσμου.

<img src='https://cdn.illacloud.com/official-website/img/docs/assemble/collaboration(1).png' alt="" />

### Διαχείριση μελών

<img src='https://cdn.illacloud.com/official-website/img/docs/assemble/collaboration(2).png' alt="" />

Οι συντάκτες μπορούν να προσκαλέσουν ή να διαχειριστούν συντάκτες και θεατές μπορούν να προσκαλέσουν ή να διαχειριστούν θεατές. Εάν δεν θέλετε οι συντάκτες και οι θεατές να προσκαλέσουν ή να διαχειριστούν μέλη ή να ελέγξουν τη λίστα μελών, παρακαλούμε να απενεργοποιήσετε το 'Επιτρέψτε στους συντάκτες και τους θεατές να διαχειρίζονται τα μέλη'.

### Συνεργαστείτε στην ίδια εφαρμογή

Επιτρέπουμε στα μέλη να επεξεργάζονται μια εφαρμογή ταυτόχρονα. Μπορείτε να δείτε ποια στοιχεία επεξεργάζονται. Όπως φαίνεται παρακάτω.

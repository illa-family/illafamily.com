## <h2 hidden>Τι είναι Modal?</h2>

Στο ILLA Cloud, το modal component είναι ένα στοιχείο διεπαφής χρήστη που χρησιμοποιείται για την εμφάνιση περιεχομένου, πληροφορίες, ή ενέργειες σε ένα εστιασμένο και επικαλυπτόμενο παράθυρο. Το στοιχείο modal εμφανίζεται συνήθως στην κορυφή της κύριας οθόνης εφαρμογής, διακόπτοντας προσωρινά τη ροή εργασίας του χρήστη και εφιστώντας την προσοχή τους στο περιεχόμενο modal.

### Ιδιότητες

| Ιδιότητες                                     | Περιγραφή                                                                                                                         |
| --------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------- |
| Χειριστής συμβάντων                           | ανίχνευση και ανταπόκριση σε συγκεκριμένες ενέργειες χρήστη ή γεγονότα, όπως κλικ, πατήματα πλήκτρων, αλλαγές τιμών               |
| Κάντε κλικ στη μάσκα για να κλείσετε το modal | αν κάνοντας κλικ στην επικάλυψη φόντου ή μάσκα του τρόπου θα κλείσει το παράθυρο modal                                            |
| Εμφάνιση κεφαλίδας                            | αν το στοιχείο πρέπει να εμφανίζει μια κεφαλίδα ή μια ενότητα τίτλου                                                              |
| Εμφάνιση υποσέλιδου                           | αν το κατασκευαστικό στοιχείο πρέπει να εμφανίζει μια ενότητα υποσέλιδου                                                          |
| Χρώμα Περιγράμματος                           | χρώμα του περιγράμματος που περιβάλλει ένα στοιχείο μοντάλ                                                                        |
| Ακτίνα Περιγράμματος                          | καμπυλότητα των γωνιών του περιγράμματος ενός στοιχείου μοντάλ                                                                    |
| Border Width                                  | πάχος ή πλάτος του περιγράμματος που περιβάλλει ένα στοιχείο του μοντάλ                                                           |
| Φόντο                                         | καθορίζει το χρώμα ή την εικόνα που γεμίζει την περιοχή περιεχομένου ενός στοιχείου τρόπου λειτουργίας                            |

### Μέθοδος

Μπορείτε να χρησιμοποιήσετε άλλα στοιχεία για τον έλεγχο της εφαρμογής. Υποστηρίζουμε τις ακόλουθες δύο μεθόδους:

-   openModal

άνοιγμα του στοιχείου modal

-   closeModal

κλείσιμο του στοιχείου modal

### Περίπτωση χρήσης

Θα δείξουμε πώς να φτιάξουμε μια φόρμα με το όνομα και την ηλικία ως είσοδο, καθώς και την πρόοδο της γραμμής για να δείξει την πρόοδο ολοκλήρωσης της φόρμας.

#### Βήμα 1 Προσθήκη Στοιχείων

Προσθήκη ενός **κουμπιού\`** στοιχείου με την ένδειξη "Άνοιγμα μοντέλου"

Προσθέστε ένα στοιχείο modal στον καμβά.

#### Βήμα 2 Διαμόρφωση του εξαρτήματος

Για τις εφαρμογές "Open Modal" **`button`** μπορούμε να τις ρυθμίσουμε ώστε να ανοίξουν το **`modal`** component αν πατηθούν.

1.  Κάντε κλικ στο στοιχείο **`button`** για να ανοίξετε τη σελίδα ελέγχου. Κάτω από **`Χειριστής Συμβάντων`**, κάντε κλικ στο `**+ Νέο**`.
2.  Στο χειριστή συμβάντων επεξεργασίας, επιλέξτε **`Control component`** εν ενεργεία, και επιλέξτε το στοιχείο **`modal`** που θέλετε να ενημερώσετε ως στόχο του γεγονότος.
3.  Επιλέξτε τη μέθοδο **`openModal`**

![open_modal](https://cdn.illacloud.com/official-website/img/docs/open_modal.jpg)

Κάντε κλικ σε αυτό το κουμπί για να ανοίξετε το modal component και μπορείτε να βάλετε ό, τι θέλετε στη μέση περιοχή του modal.

Για την εφαρμογή "Άκυρο" **`κουμπί`** μπορούμε να την ρυθμίσουμε ώστε να κλείσει το modal αν πατηθεί

1.  Κάντε κλικ στο στοιχείο **`button`** για να ανοίξετε τη σελίδα ελέγχου. Κάτω από **`Χειριστής Συμβάντων`**, κάντε κλικ στο `**+ Νέο**`.
2.  Στο χειριστή συμβάντων επεξεργασίας, επιλέξτε **`Control component`** εν ενεργεία, και επιλέξτε το στοιχείο **`modal`** που θέλετε να ενημερώσετε ως στόχο του γεγονότος.
3.  Επιλέξτε τη μέθοδο **`closeModal`**

![close_modal](https://cdn.illacloud.com/official-website/img/docs/close_modal.jpg)

Ομοίως, μπορούμε να διαμορφώσουμε το κουμπί "Επιβεβαίωση" για να χαιρετίσουμε

1.  Κάντε κλικ στο στοιχείο **`button`** για να ανοίξετε τη σελίδα ελέγχου. Κάτω από **`Χειριστής Συμβάντων`**, κάντε κλικ στο `**+ Νέο**`.
2.  Στο χειριστή συμβάντων επεξεργασίας, επιλέξτε **`Show notification`** in action, **`title`** "Congrat!" και **`description`** "Συγχαρητήρια!". Επιλέξτε το `Success` για **`type`**, βάλτε `{{2000}}` για **`duration`** (2 δευτερόλεπτα)

![modal_confirm](https://cdn.illacloud.com/official-website/img/docs/modal_confirm.jpg)

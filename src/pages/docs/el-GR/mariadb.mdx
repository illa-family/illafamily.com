## <h2 hidden>Τι είναι η MariaDB?</h2>

MariaDB στην Illa είναι μια ενσωμάτωση της βάσης δεδομένων που σας επιτρέπει να συνδεθείτε και να αλληλεπιδράσετε με μια βάση δεδομένων MariaDB. Η MariaDB είναι ένα δημοφιλές σχεσιακό σύστημα διαχείρισης βάσεων δεδομένων ανοιχτού κώδικα που προέρχεται από τη MySQL.

Με την ενσωμάτωση της MariaDB, μπορείτε να ρωτήσετε, να εισαγάγετε και να διαγράψετε δεδομένα από μια βάση δεδομένων MariaDB χρησιμοποιώντας εντολές SQL. Μπορείτε επίσης να δημιουργήσετε προσαρμοσμένα ερωτήματα SQL και να τα εκτελέσετε μέσα Illa. Αυτό καθιστά εύκολο να οικοδομήσουμε τις εφαρμογές δεδομένων και τους πίνακες ελέγχου που εμφανίζουν δεδομένα σε πραγματικό χρόνο από μια βάση δεδομένων MariaDB.

### Δημιουργία MariaDB

Υπάρχουν δύο τρόποι για να δημιουργήσετε έναν πόρο στην Illa μετά την είσοδό σας στο λογαριασμό Illa. 

-   Δημιουργία σε πόρους

Συνδεθείτε στον Illa λογαριασμό σας, επιλέξτε `**Πόροι**` στην κορυφή της σελίδας και κάντε κλικ στο κουμπί `**Δημιουργία Νέου**`. 

<img src='https://cdn.illacloud.com/official-website/img/docs/external_resource.png' alt='' />

Επιλέξτε **`MariaDB`** από τη λίστα της βάσης δεδομένων.

<img src='https://cdn.illacloud.com/official-website/img/docs/action_list.png' alt='' />

Συνδεθείτε στη βάση δεδομένων με τις απαιτούμενες παραμέτρους που περιγράφονται στις παρακάτω ρυθμίσεις σύνδεσης. 

Κάντε κλικ στο κουμπί `**Δοκιμή Σύνδεσης**` για να δείτε αν μπορούμε να συνδεθείτε με επιτυχία στη βάση δεδομένων. Εάν ναι, κάντε κλικ στο **`Αποθήκευση πόρων`**, αλλιώς, ελέγξτε ξανά το όνομα κεντρικού υπολογιστή, τη θύρα, το όνομα χρήστη και τον κωδικό πρόσβασης είναι σωστά.

Μετά τη δημιουργία ενός πόρου, το έτοιμο MariaDB θα εμφανιστεί όπως εμφανίζεται.

<img src='https://cdn.illacloud.com/official-website/img/docs/maria_ex_ready.png' alt='' />

-   Δημιουργία στο Builder

Συνδεθείτε στον Illa λογαριασμό σας, δημιουργήστε ένα έργο στο Illa Builder στη σελίδα `**App**`, και πλοηγηθείτε στο **`Λίστα Ενεργειών`** στο κάτω μέρος της σελίδας. Κάντε κλικ στο κουμπί `**new**`, στη συνέχεια επιλέξτε **`MariaDB`** από τη λίστα της βάσης δεδομένων. Στη συνέχεια, συνδεθείτε στη βάση δεδομένων με τις απαιτούμενες παραμέτρους που περιγράφονται στις "Ρυθμίσεις Σύνδεσης" παρακάτω.

Κάντε κλικ στο κουμπί `**Δοκιμή Σύνδεσης**` για να δείτε αν μπορούμε να συνδεθείτε με επιτυχία στη βάση δεδομένων. Εάν ναι, κάντε κλικ στο **`Αποθήκευση πόρων`**, αλλιώς, ελέγξτε ξανά το όνομα κεντρικού υπολογιστή, τη θύρα, το όνομα χρήστη και τον κωδικό πρόσβασης είναι σωστά.

### Ρυθμίσεις Σύνδεσης

Εδώ πρέπει να παρέχουμε πληροφορίες για τη σύνδεση με τη βάση δεδομένων MariaDB.

<img src='https://cdn.illacloud.com/official-website/img/docs/maria_config.png' alt='' />

| Ιδιότητες      | Απαιτείται  | Περιγραφή                                                                                                                                         |
| -------------- | ----------- | ------------------------------------------------------------------------------------------------------------------------------------------------- |
| Όνομα          | απαιτείται  | Το όνομα του πόρου κατά τη δημιουργία ενεργειών στο ILLA.                                                                                         |
| Hostname       | απαιτείται  | Η διεύθυνση URL ή IP για τη βάση δεδομένων σας                                                                                                    |
| Θύρα           | απαιτείται  | Ο αριθμός θύρας του διακομιστή που θα πρέπει να χρησιμοποιήσετε για να συνδεθείτε. Αν δεν καθορίσετε μια θύρα, η προεπιλεγμένη θύρα είναι '3306'. |
| Βάση Δεδομένων | απαιτείται  | Το όνομα της βάσης δεδομένων                                                                                                                      |
| Όνομα Χρήστη   | απαιτείται  | το όνομα χρήστη που θέλετε να χρησιμοποιήσετε κατά τη σύνδεση στον εξυπηρετητή MariaDB.                                                           |
| Κωδικός        | απαιτείται  | Χρησιμοποιήστε αυτόν τον κωδικό πρόσβασης για ταυτοποίηση.                                                                                        |
| Επιλογές SSL   | προαιρετικό | αποφασίζει πόσο υψηλή προτεραιότητα έχει μια ασφαλής σύνδεση SSL TCP/IP κατά τη διαπραγμάτευση με το διακομιστή.                                  |

### Δημιουργία Ενεργειών

Έχουμε δημιουργήσει έναν πόρο MariaDB, μπορούμε να προσθέσουμε την ενέργεια επιλέγοντας MariaDB από τη λίστα ενεργειών και επιλέγοντας το κουμπί **`Δημιουργία δράσης`**.

<img src='https://cdn.illacloud.com/official-website/img/docs/maria_resource_list.png' alt='' />

 Τώρα έχουμε προσθέσει τον εξυπηρετητή MariaDB ως ενέργεια στη σελίδα μας κτηρίου.

<img src='https://cdn.illacloud.com/official-website/img/docs/maria.png' alt='' />

### Configure MariaDB

| Ιδιότητες       | Περιγραφή                                                                                     |
| --------------- | --------------------------------------------------------------------------------------------- |
| Ερώτημα SQL     | εντολές sql για λήψη (επιλογή), εισαγωγή, ενημέρωση, διαγραφή δεδομένων από τη βάση δεδομένων |
| Μετασχηματιστής | μετατροπή δεδομένων σε στυλ που σας αρέσει χρησιμοποιώντας Javascript                         |

Χρήση παραδείγματος:

-   Επιλογή

```sql
SELECT * FROM users ORDER BY lastname;
```

-   Insert

```sql
INSERT INTO users

	(firstname, lastname, email)

VALUES

	(

		{{ NewUser.data.firstNameInput }},

		{{ NewUser.data.lastNameInput }},

		{{ NewUser.data.emailInput }}

	);
```

-   Ενημέρωση

```sql
UPDATE users
  SET firstname = '{{form1.updatedProfile.first}}'
  WHERE id = {{ form1.updatedProfile.uid}};
```

-   Διαγραφή

```sql
DELETE FROM users WHERE id = {{ form1.updatedProfile.uid }};
```

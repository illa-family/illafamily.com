## <h2 hidden>Τι είναι TiDB?</h2>

TiDB είναι μια ενσωμάτωση της βάσης δεδομένων που σας επιτρέπει να συνδεθείτε και να αλληλεπιδράσετε με μια βάση δεδομένων TiDB. Είναι ένα κατανεμημένο, οριζόντια επεκτάσιμη σχεσιακή σύστημα διαχείρισης βάσεων δεδομένων που έχει σχεδιαστεί για να χειριστεί υψηλής έντασης, υψηλής concurrency φόρτους εργασίας.

Με την ενσωμάτωση TiDB στην Illa, μπορείτε να ρωτήσετε, να ενημερώσετε και να διαγράψετε τα δεδομένα από μια βάση δεδομένων TiDB χρησιμοποιώντας εντολές SQL. Μπορείτε επίσης να δημιουργήσετε προσαρμοσμένα ερωτήματα SQL και να τα εκτελέσετε μέσα Illa. Αυτό καθιστά εύκολο να οικοδομήσουμε τις εφαρμογές δεδομένων και τους πίνακες ελέγχου που εμφανίζουν δεδομένα σε πραγματικό χρόνο από μια βάση δεδομένων TiDB.

### Δημιουργία TiDB

Υπάρχουν δύο τρόποι για να δημιουργήσετε έναν πόρο στην Illa μετά την είσοδό σας στο λογαριασμό Illa.

-   Δημιουργία σε πόρους

Συνδεθείτε στον Illa λογαριασμό σας, επιλέξτε `**Πόροι**` στην κορυφή της σελίδας και κάντε κλικ στο κουμπί `**Δημιουργία Νέου**`.

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

Επιλέξτε **`TiDB`** από τη λίστα της βάσης δεδομένων.

![action_list](https://cdn.illacloud.com/official-website/img/docs/action_list.png)

Συνδεθείτε στη βάση δεδομένων με τις απαιτούμενες παραμέτρους που περιγράφονται στις παρακάτω ρυθμίσεις σύνδεσης.

Κάντε κλικ στο κουμπί `**Δοκιμή Σύνδεσης**` για να δείτε αν μπορούμε να συνδεθείτε με επιτυχία στη βάση δεδομένων. Εάν ναι, κάντε κλικ στο **`Αποθήκευση πόρων`**, αλλιώς, ελέγξτε ξανά το όνομα κεντρικού υπολογιστή, τη θύρα, το όνομα χρήστη και τον κωδικό πρόσβασης είναι σωστά.

Μετά τη δημιουργία ενός πόρου, το έτοιμο TiDB θα εμφανίζεται όπως εμφανίζεται.

![ti_ex_ready](https://cdn.illacloud.com/official-website/img/docs/ti_ex_ready.png)

-   Δημιουργία στο Builder

Συνδεθείτε στον Illa λογαριασμό σας, δημιουργήστε ένα έργο στο Illa Builder στη σελίδα `**App**`, και πλοηγηθείτε στο **`Λίστα Ενεργειών`** στο κάτω μέρος της σελίδας. Κάντε κλικ στο κουμπί `**new**`, και μετά επιλέξτε **`TiDB`** από τη λίστα της βάσης δεδομένων. Στη συνέχεια, συνδεθείτε στη βάση δεδομένων με τις απαιτούμενες παραμέτρους που περιγράφονται στις "Ρυθμίσεις Σύνδεσης" παρακάτω.

Κάντε κλικ στο κουμπί `**Δοκιμή Σύνδεσης**` για να δείτε αν μπορούμε να συνδεθείτε με επιτυχία στη βάση δεδομένων. Εάν ναι, κάντε κλικ στο **`Αποθήκευση πόρων`**, αλλιώς, ελέγξτε ξανά το όνομα κεντρικού υπολογιστή, τη θύρα, το όνομα χρήστη και τον κωδικό πρόσβασης είναι σωστά.

### Ρυθμίσεις Σύνδεσης

Εδώ πρέπει να παρέχουμε πληροφορίες για τη σύνδεση με τη βάση δεδομένων TiDB.

![ti_config](https://cdn.illacloud.com/official-website/img/docs/ti_config.png)

![ti_config_1](https://cdn.illacloud.com/official-website/img/docs/ti_config_1.png)

| Ιδιότητες                | Απαιτείται  | Περιγραφή                                                                                                                                                                                                                                                                          |
| ------------------------ | ----------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Όνομα                    | απαιτείται  | Το όνομα του πόρου κατά τη δημιουργία ενεργειών στο ILLA.                                                                                                                                                                                                                          |
| Hostname                 | απαιτείται  | Η διεύθυνση URL ή IP για τη βάση δεδομένων σας                                                                                                                                                                                                                                     |
| Θύρα                     | απαιτείται  | Ο αριθμός θύρας του διακομιστή που θα πρέπει να χρησιμοποιήσετε για να συνδεθείτε. Αν δεν καθορίσετε μια θύρα, η προεπιλεγμένη θύρα είναι '4000'.                                                                                                                                  |
| Βάση Δεδομένων           | απαιτείται  | Το όνομα της βάσης δεδομένων                                                                                                                                                                                                                                                       |
| Όνομα Χρήστη             | απαιτείται  | το όνομα χρήστη που θέλετε να χρησιμοποιήσετε κατά τη σύνδεση στον εξυπηρετητή TiDB.                                                                                                                                                                                               |
| Κωδικός                  | απαιτείται  | Χρησιμοποιήστε αυτόν τον κωδικό πρόσβασης για ταυτοποίηση.                                                                                                                                                                                                                         |
| Επιλογές SSL             | προαιρετικό | αποφασίζει πόσο υψηλή προτεραιότητα έχει μια ασφαλής σύνδεση SSL TCP/IP κατά τη διαπραγμάτευση με το διακομιστή.                                                                                                                                                                   |
| Πιστοποιητικό CA         | απαιτείται  | ψηφιακό πιστοποιητικό που χρησιμοποιείται για την επαλήθευση της ταυτότητας ενός εξυπηρετητή και τη δημιουργία ασφαλούς, κρυπτογραφημένη σύνδεση μεταξύ ενός πελάτη και ενός συμπλέγματος TiDB. Αν δεν το καθορίσετε, έχουμε συμπληρώσει ένα προεπιλεγμένο πιστοποιητικό για εσάς. |
| Κλειδί Πελάτη            | προαιρετικό | παράμετρος που χρησιμοποιείται για τη δημιουργία ασφαλούς σύνδεσης μεταξύ του πελάτη TiDB και του διακομιστή TiDB.                                                                                                                                                                 |
| Πιστοποιητικό Πελάτη     | προαιρετικό | ένα χαρακτηριστικό ασφαλείας που επιτρέπει σε έναν πελάτη να πιστοποιήσει τον εαυτό του σε ένα διακομιστή TiDB                                                                                                                                                                     |

### Δημιουργία Ενεργειών

Έχουμε δημιουργήσει έναν πόρο TiDB, μπορούμε να προσθέσουμε την ενέργεια επιλέγοντας TiDB από τη λίστα ενεργειών και επιλέγοντας το κουμπί **`Δημιουργία δράσης`**.

![ti_resource_list](https://cdn.illacloud.com/official-website/img/docs/ti_resource_list.png)

Τώρα έχουμε προσθέσει τον εξυπηρετητή TiDB ως ενέργεια στη σελίδα μας κτηρίου.

![ti](https://cdn.illacloud.com/official-website/img/docs/ti.png)

### Ρύθμιση TiDB

| Ιδιότητες       | Περιγραφή                                                                                     |
| --------------- | --------------------------------------------------------------------------------------------- |
| Ερώτημα SQL     | εντολές sql για λήψη (επιλογή), εισαγωγή, ενημέρωση, διαγραφή δεδομένων από τη βάση δεδομένων |
| Μετασχηματιστής | μετατροπή δεδομένων σε στυλ που σας αρέσει χρησιμοποιώντας Javascript                         |

Χρήση παραδείγματος:

-   Επιλογή

```sql
SELECT * ΑΠΟ χρήστες ORDER BY επώνυμο;
```

-   Insert

```sql
INSERT INTO users

	(firstname, lastname, email)

VALUES

	(

		{{ NewUser.data.firstNameInput }},

		{{ NewUser.data.lastNameInput }},

		{{ NewUser.data.emailInput }}

	);
```

-   Ενημέρωση

```sql
UPDATE users
  SET firstname = '{{form1.updatedProfile.first}}'
  WHERE id = {{ form1.updatedProfile.uid}};
```

-   Διαγραφή

```sql
DELETE FROM users WHERE id = {{ form1.updatedProfile.uid }};
```

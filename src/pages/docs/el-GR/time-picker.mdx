## <h2 hidden>Τι είναι ο Επιλογέας του Χρόνου?</h2>

Το στοιχείο επιλογέα χρόνου είναι ένα στοιχείο UI που επιτρέπει στους χρήστες να επιλέξουν ένα συγκεκριμένο χρόνο από μια 24ωρη διεπαφή, χρησιμοποιείται συνήθως σε μορφές ή διεπαφές εισαγωγής δεδομένων όπου ο χρήστης πρέπει να προσδιορίσει μια συγκεκριμένη ώρα.

Όταν ένα στοιχείο επιλογέα χρόνου προστίθεται σε καμβά, εμφανίζει μια 24-ωρη διεπαφή που δείχνει ώρες, λεπτά και δευτερόλεπτα. Ο χρήστης μπορεί να ρυθμίσει το χρόνο μετακινώντας τον κέρσορα του ποντικιού ή χρησιμοποιώντας τα πάνω και κάτω βέλη.

### Περίπτωση χρήσης

Στη συνέχεια, θα δείξουμε πώς να χαρτογραφήσουμε τα δεδομένα από την πηγή δεδομένων έως τον επιλογέα χρόνου και να ρυθμίσουμε την τρέχουσα ώρα με ένα κουμπί. 

#### Βήμα 1 Προσθέστε μια ενέργεια

Επιτρέψτε μας να δημιουργήσουμε έναν πίνακα στο REST API που ονομάζεται **`promoteCodes`,** συμπεριλαμβανομένων 6 στηλών: **`code`**, **`id`**, **`createdAt`**, **`expiredAt`**, **`startedAt`, `updatedAt`**. Θα χρησιμοποιήσουμε το ενημερωμένο Κατά το χρόνο για τη χρήση δειγμάτων. 

Στη συνέχεια, μπορούμε να δημιουργήσουμε μια νέα δράση για REST API από τη λίστα δράσης, λίστα όλων των δεδομένων σε promoteCodes και ονομάζεται restapi1. 

Πατήστε **`Save`** και **`Run`** για να ενεργοποιήσετε αυτήν την ενέργεια.

#### Βήμα 2 Προσθήκη Συστατικών

Στη συνέχεια, μπορούμε να προσθέσουμε στο στοιχείο **`time picker`** και ένα **`button`** στοιχείο στον καμβά. Για την προεπιλεγμένη τιμή του στοιχείου του χρόνου, θα το ρυθμίσουμε ως την τελευταία ενημέρωση του πρώτου κώδικα προώθησης από τα δεδομένα υπόλοιπο api.

```sql
{{restapi1.data.promoteCodes[0].updatedAt}}
```

Επισημάνουμε επίσης το κουμπί ως "Τώρα" όπως φαίνεται παρακάτω

![time_picker_0](https://cdn.illacloud.com/official-website/img/docs/time_picker_0.png)

#### Βήμα 3 Διαμόρφωση του εξαρτήματος

Για το **`button`** component, μπορούμε να το ρυθμίσουμε ώστε να ορίζει την ημερομηνία σε **`time picker`** component στην τρέχουσα ημερομηνία και ώρα.

1.  Στο χειριστή συμβάντων επεξεργασίας, επιλέξτε **`Control component`** εν ενεργεία, επιλέξτε το στοιχείο **`time picker`** που θέλετε να ενημερώσετε ως στόχο του γεγονότος.
2.  Επιλέξτε την **`setValue`** ενέργεια και ορίστε την τιμή ως `{{currentUserInfo.updatedAt}}` 

![time_picker_config](https://cdn.illacloud.com/official-website/img/docs/time_picker_config.png)

#### 4Ο Στάδιο Δοκιμή

Τώρα που όταν κάνετε κλικ στο κουμπί "Τώρα", ο χρόνος θα πρέπει να αλλάξει στην τρέχουσα ώρα. Όσο για εδώ, έχουμε 20:03:05 αλλά μπορεί να είναι διαφορετικό όταν το δοκιμάσετε.

## <h2 hidden>Τι είναι Tabs?</h2>

Το στοιχείο Καρτέλες στο σύννεφο ILLA είναι ένα ευέλικτο χαρακτηριστικό που επιτρέπει στους χρήστες να οργανώσουν και να παρουσιάσουν περιεχόμενο σε διάταξη καρτελών. Επιτρέπει τη δημιουργία πολλαπλών καρτελών μέσα σε ένα ενιαίο δοχείο, επιτρέποντας στους χρήστες να αλλάζουν μεταξύ διαφορετικών τμημάτων του περιεχομένου με ευκολία. Το στοιχείο καρτέλες παρέχει ένα φιλικό προς το χρήστη περιβάλλον για την πλοήγηση μέσω σχετικών πληροφοριών, βελτιώνοντας τη συνολική εμπειρία χρήστη. Με προσαρμόσιμες επιλογές styling και τη δυνατότητα να προσθέσετε διάφορους τύπους περιεχομένου, όπως κείμενο, εικόνες, ή ενσωματωμένα συστατικά, το στοιχείο καρτελών στο ILLA Cloud προσφέρει ευελιξία και ενισχύει την οργάνωση και την παρουσίαση των δεδομένων.

### Ιδιότητες

| Ιδιότητες                           | Περιγραφή                                                                                                                                                                                             |
| ----------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Σύνδεσμος σε ένα εμπορευματοκιβώτιο | συνδέει το στοιχείο Tabs με ένα συγκεκριμένο δοχείο, επιτρέποντας την απρόσκοπτη πλοήγηση μεταξύ καρτελών και του σχετικού περιεχομένου.                                                              |
| Χειριστής συμβάντων                 | ανίχνευση και ανταπόκριση σε συγκεκριμένες ενέργειες ή συμβάντα χρήστη, όπως κλικ, πατήματα πλήκτρων ή αλλαγή τιμής.                                                                                  |
| Απενεργοποιημένο                    | απενεργοποιήστε συγκεκριμένες καρτέλες, εμποδίζοντας τους χρήστες να αλληλεπιδρούν με αυτούς                                                                                                          |
| Υπόδειξη                            | πρόσθετες πληροφορίες ή περιεχόμενο σχετικά με κάθε καρτέλα                                                                                                                                           |
| Διάταξη                             | προσφέρει διαφορετικές επιλογές διάταξης, επιτρέποντάς σας να κανονίσετε τις καρτέλες οριζόντια ή κάθετα με βάση τις προτιμήσεις σχεδιασμού σας ή τον διαθέσιμο χώρο.                                 |
| Align                               | στοίχιση της συνιστώσας καρτελών σε διαφορετικές θέσεις εντός της μητρικής δεξαμενής, όπως αριστερά, κέντρο ή δεξιά ευθυγράμμιση                                                                      |
| Hidden                              | απόκρυψη συγκεκριμένων καρτελών προγραμματισμένα.                                                                                                                                                     |
| Χρώματα Κειμένου                    | προσαρμόσιμα χρώματα κειμένου, επιτρέποντάς σας να διαμορφώσετε τις ετικέτες καρτέλας με διαφορετικά χρώματα για να ταιριάζουν με το σχεδιασμό ή το εμπορικό σήμα της εφαρμογής σας.                  |

### Δεδομένα

Το στοιχείο έχει κάποια κοινώς χρησιμοποιούμενα δεδομένα, τα οποία μπορούν να κληθούν μέσω του **`{{componentName.propertyName}}`** στην εφαρμογή.

| Όνομα ιδιότητας              | Περιγραφή                                                                                                                                   |
| ---------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------- |
| align                        | ευθυγράμμιση των καρτελών μέσα στο στοιχείο, όπως αριστερά, κέντρο ή δεξιά ευθυγράμμιση                                                     |
| colorScheme                  | καθορίζει το χρώμα του κειμένου στο στοιχείο                                                                                                |
| currentIndex                 | αντιπροσωπεύει το ευρετήριο της τρέχουσας επιλεγμένης καρτέλας μέσα στη λίστα καρτελών                                                      |
| currentKey                   | μοναδικό αναγνωριστικό ή κλειδί της τρέχουσας επιλεγμένης καρτέλας                                                                          |
| Εμφάνιση Ονόματος            | αντιπροσωπεύει το όνομα ή την ετικέτα που έχει ανατεθεί στο στοιχείο                                                                        |
| hidden                       | τιμή boolean που καθορίζει αν η συνιστώσα των καρτελών είναι ορατή ή κρυμμένη                                                               |
| linkWidgetDisplayName        | εμφάνιση ονόματος ή ετικέτας του widget συνδέσμου που σχετίζεται με μια συγκεκριμένη καρτέλα                                                |
| navigateContainer            | εμπορευματοκιβώτιο ή προορισμός όπου το περιεχόμενο καρτέλας θα πρέπει να εμφανίζεται ή να πλοηγείται σε                                    |
| Λίστα καρτελών               | κατάλογος πινάκων, συμπεριλαμβανομένων των ονομάτων απεικόνισης, των κλειδιών και άλλων σχετικών πληροφοριών                                |
| tabPosition                  | θέση της ταινίας καρτέλας, όπως πάνω, κάτω, αριστερά ή δεξιά θέση                                                                           |
| tooltipText                  | κείμενο που εμφανίζεται ως υπόδειξη όταν ο χρήστης αιωρείται πάνω από μια συγκεκριμένη καρτέλα                                              |
| viewList                     | προβολές ή περιεχόμενο που σχετίζονται με κάθε καρτέλα, ενεργοποιώντας την απόδοση διαφορετικού περιεχομένου με βάση την επιλεγμένη καρτέλα |

### Περίπτωση χρήσης

Θα δείξουμε πώς να χρησιμοποιήσετε το στοιχείο καρτελών σε συνδυασμό με ένα στοιχείο εμπορευματοκιβωτίων για να δείξει τις τρεις απόψεις.

#### Βήμα 1 Προσθήκη και διαμόρφωση στοιχείων

Πρόσθεσε ένα στοιχείο **`container`** και ένα στοιχείο **`tabs`**. Στη σελίδα `inspect` της εφαρμογής tabs, ενεργοποιήστε το **`Link προς ένα Δοχείο`** και επιλέξτε το Δοχείο με το οποίο θέλετε να συνδέσετε για `**Container**`.

-   Προβολή 1:

Αυτή είναι η αρχική σελίδα, επομένως τοποθετούμε ένα **`text`** component με το "Home" στην επάνω αριστερή γωνία του εμπορευματοκιβωτίου.

Εμφάνιση ενός **`text`** component "Γεια! Καλώς ήρθατε στην προσωπική μου ιστοσελίδα" στη μέση και ένα **`button`** συστατικό με την ένδειξη "Μάθετε περισσότερα για μένα" κάτω από αυτό.

Για το **`button`** component, μπορούμε να το ρυθμίσουμε ώστε να μεταβεί στην επόμενη προβολή (view2) αν πατηθεί.

1.  Κάντε κλικ στο στοιχείο **`button`** για να ανοίξετε τη σελίδα ελέγχου. Κάτω από **`Χειριστής Συμβάντων`**, κάντε κλικ στο `**+ Νέο**`.
2.  Στο χειριστή συμβάντων επεξεργασίας, επιλέξτε **`Control component`** εν ενεργεία, και επιλέξτε το στοιχείο **`container`** που θέλετε να ενημερώσετε ως στόχο του γεγονότος.
3.  Επιλέξτε τη μέθοδο **`showNextView`**.

![tab_view1](https://cdn.illacloud.com/official-website/img/docs/tab_view1.jpg)

-   Προβολή 2:

Αυτή είναι η σελίδα περίπου, επομένως τοποθετούμε ένα **`text`** component με το "About me" στην επάνω αριστερή γωνία του δοχείου.

Εμφάνιση του "όνομα", "ηλικία", και "φύλο" πληροφορίες με **`text`** στοιχεία και μια εικόνα προφίλ σας ή οποιονδήποτε θέλετε να χρησιμοποιήσετε με το στοιχείο `**εικόνα**`.

Πρόσθεσε ένα **κουμπί \`\`** συστατικό με την ένδειξη "Κάνε φίλους μαζί μου".

Για το **`button`** component, μπορούμε να το ρυθμίσουμε ώστε να μεταβεί στην επόμενη προβολή (view3) αν πατηθεί.

1.  Κάντε κλικ στο στοιχείο **`button`** για να ανοίξετε τη σελίδα ελέγχου. Κάτω από **`Χειριστής Συμβάντων`**, κάντε κλικ στο `**+ Νέο**`.
2.  Στο χειριστή συμβάντων επεξεργασίας, επιλέξτε **`Control component`** εν ενεργεία, και επιλέξτε το στοιχείο **`container`** που θέλετε να ενημερώσετε ως στόχο του γεγονότος.
3.  Επιλέξτε τη μέθοδο **`showNextView`**.

![tab_view2](https://cdn.illacloud.com/official-website/img/docs/tab_view2.jpg)

-   Προβολή 3

Αυτή είναι η σελίδα της φόρμας, ο επισκέπτης μπορεί να συμπληρώσει τη φόρμα με τις πληροφορίες του για να στείλει ένα αίτημα φιλίας στον ιδιοκτήτη.

Υπάρχει ένα στοιχείο φόρμας στη μέση του περιέκτη με στοιχεία εισόδου που επισημαίνονται ως "Το όνομά σας", "Η ηλικία σας", και "Φύλο".

Αλλάξτε το κείμενο του κουμπιού "Υποβολή" στην κάτω δεξιά γωνία σε "Προσκεκλημένο".

![tab_view3](https://cdn.illacloud.com/official-website/img/docs/tab_view3.jpg)

Για το στοιχείο "Πρόσκληση" **`button`** της εφαρμογής, μπορούμε να το ρυθμίσουμε ώστε να υποβάλλουμε τις πληροφορίες με τη μορφή αν γίνει κλικ.

1.  Κάντε κλικ στο στοιχείο **`button`** για να ανοίξετε τη σελίδα ελέγχου. Κάτω από **`Χειριστής Συμβάντων`**, κάντε κλικ στο `**+ Νέο**`.
2.  Στο χειριστή συμβάντων επεξεργασίας, επιλέξτε **`Control component`** εν ενεργεία, και επιλέξτε το στοιχείο **`form`** που θέλετε να ενημερώσετε ως στόχο του γεγονότος.
3.  Επιλέξτε τη μέθοδο **`submit`**.

Για το **`form`** component, μπορούμε να το ρυθμίσουμε ώστε να εμφανίζει ειδοποίηση εάν υποβληθεί.

1.  Κάνε κλικ στο στοιχείο **`form`** για να ανοίξεις τη σελίδα ελέγχου. Κάτω από **`Χειριστής Συμβάντων`**, κάντε κλικ στο `**+ Νέο**`.
2.  Στο χειριστή συμβάντων επεξεργασίας, επιλέξτε **`submit`** στο Event, επιλέξτε **`Show notification`** in **`action`**, Βάλτε το "Submit με επιτυχία" για τίτλο, το "Αίτημα Φιλίας εστάλη" για Περιγραφή, και επιλέξτε το `Success` για το **`Type`**. Θέλουμε η ειδοποίηση να εμφανίζεται για 2s έτσι βάλτε το `{{2000}}` για τη διάρκεια.

![tab_form_event](https://cdn.illacloud.com/official-website/img/docs/tab_form_event.jpg)

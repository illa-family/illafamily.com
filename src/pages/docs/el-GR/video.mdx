## <h2 hidden>Τι είναι το βίντεο?</h2>

Το στοιχείο βίντεο στο ILLA Cloud είναι ένα ισχυρό εργαλείο που επιτρέπει στους χρήστες να ενσωματώνουν περιεχόμενο πολυμέσων στις εφαρμογές ή τα έργα τους. Με αυτή τη λειτουργία, οι χρήστες μπορούν να ενσωματώσουν απρόσκοπτα βίντεο στις διεπαφές χρηστών τους και να δημιουργήσουν ελκυστικές και διαδραστικές εμπειρίες για τους τελικούς χρήστες τους.

Το στοιχείο βίντεο υποστηρίζει διάφορες μορφές βίντεο, συμπεριλαμβανομένων δημοφιλών μορφών όπως MP4, AVI και MOV. Οι χρήστες μπορούν να ανεβάζουν βίντεο απευθείας από τις συσκευές τους ή να παρέχουν διευθύνσεις URL σε εξωτερικές πηγές βίντεο. Αυτή η ευελιξία επιτρέπει στους χρήστες να αξιοποιήσουν υπάρχον περιεχόμενο βίντεο ή να δημιουργήσουν νέα βίντεο ειδικά για τις εφαρμογές τους.

### Ιδιότητες

| Ιδιότητες                  | Περιγραφή                                                                                                                                           |
| -------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------- |
| Πηγή βίντεο                | καθορίζει την τοποθεσία του αρχείου βίντεο ή παρέχει μια διεύθυνση URL σε μια εξωτερική πηγή βίντεο                                                 |
| Αυτόματο Μηχάνημα          | επιτρέπει στο βίντεο να ξεκινά την αναπαραγωγή αυτόματα όταν φορτώσει η ιστοσελίδα                                                                  |
| Επανάληψη                  | επιτρέπει στο βίντεο να αναπαράγεται συνεχώς σε έναν βρόχο                                                                                          |
| Εμφάνιση στοιχείων ελέγχου | αν τα στοιχεία ελέγχου του προγράμματος αναπαραγωγής βίντεο είναι ορατά ή κρυμμένα                                                                  |
| Σίγαση                     | επιτρέπει στο βίντεο να αναπαράγεται χωρίς κανένα ήχο                                                                                               |
| Όγκος                      | ελέγχει το επίπεδο ήχου του βίντεο                                                                                                                  |
| Χειριστής Συμβάντων        | Ερωτήματα ενεργοποίησης, έλεγχος εφαρμογών, ή κλήση άλλων API σε απόκριση σε συμβάντα εφαρμογής.                                                    |
| Υπόδειξη                   | Οι χρήστες μπορούν να εισάγουν εδώ την υπόδειξη συστατικού. Η υπόδειξη θα εμφανίζεται όταν εστιάζεται. Υποστηρίζεται μορφή Markdown                 |
| Hidden                     | Δυναμική ρύθμιση της απόκρυψης του συστατικού. Μπορείτε να αλλάξετε την κρυφή κατάσταση μέσω μιας δυναμικής δυαδικής τιμής.                         |

### Μέθοδος

Μπορείτε να χρησιμοποιήσετε άλλα στοιχεία για τον έλεγχο της εφαρμογής. Υποστηρίζουμε τις ακόλουθες οκτώ μεθόδους:

-   **παίξτε**

αρχίστε να παίζετε το βίντεο. Όταν επικαλεστεί, ξεκινά η αναπαραγωγή του βίντεο από την τρέχουσα θέση.

-   **παύση **

παύση της αναπαραγωγής βίντεο. Όταν καλείται, σταματά προσωρινά το βίντεο στην τρέχουσα θέση του.

-   **εμφάνιση**

εναλλαγή της ορατότητας του ελέγχου του προγράμματος αναπαραγωγής βίντεο. Με την επίκληση αυτής της μεθόδου, τα χειριστήρια μπορούν να εμφανιστούν ή να κρυφτούν, επιτρέποντας στους χρήστες να αλληλεπιδρούν με την αναπαραγωγή βίντεο.

-   **setVideoUrl**

αλλάζει δυναμικά την πηγή ή το URL του βίντεο.

| Ιδιότητες   | Περιγραφή             |
| ----------- | --------------------- |
| Url Βίντεο  | url προέλευσης βίντεο |

-   **setVolume**

ρύθμιση της έντασης ήχου του βίντεο. Παίρνει μια αριθμητική τιμή ως ένταση παραμέτρου

| Ιδιότητες | Περιγραφή                                                                                                                       |
| --------- | ------------------------------------------------------------------------------------------------------------------------------- |
| Όγκος     | αριθμητική τιμή, που κυμαίνεται από 0 έως 1, όπου το 0 αντιπροσωπεύει σίγαση ή όχι ήχο, και το 1 αντιπροσωπεύει μέγιστη ένταση. |

-   **setSpeed**

αλλαγή της ταχύτητας αναπαραγωγής του βίντεο. Δέχεται μια αριθμητική τιμή ως παράμετρος Ταχύτητα

| Ιδιότητες     | Περιγραφή                                                                                                                                                           |
| ------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Ταχύτητα      | numeric value. 1 αντιπροσωπεύει την κανονική ταχύτητα αναπαραγωγής. Τιμές λιγότερες από 1 επιβραδύνουν το βίντεο, ενώ οι τιμές είναι μεγαλύτερες από 1 επιταχύνουν. |

-   **setLoop**

ενεργοποίηση ή απενεργοποίηση της λειτουργικότητας κυκλώματος του βίντεο. Περνώντας μια τιμή boolean ως παράμετρος, το βίντεο μπορεί να ρυθμιστεί σε κύκλωμα συνεχώς ή να παίξει μόνο μία φορά.

-   **αναζήτηση**

αναζήτηση συγκεκριμένου χρόνου ή θέσης στο βίντεο. Παίρνει αριθμητικές τιμές ως χρόνος(οι) παράμετρος(οι)

| Ιδιότητες     | Περιγραφή                                                                                                                          |
| ------------- | ---------------------------------------------------------------------------------------------------------------------------------- |
| Χρόνος(οι)    | μια αριθμητική τιμή που αντιπροσωπεύει τον επιθυμητό χρόνο σε δευτερόλεπτα ως παράμετρο και ρυθμίζει ανάλογα τη θέση αναπαραγωγής. |

### Χειριστής συμβάντων

| Γεγονός       | Περιγραφή                                                                                                                                                                                                                                          |
| ------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Αναπαραγωγή   | εκτέλεση συγκεκριμένων ενεργειών ή εκτέλεση κώδικα όταν ξεκινά η αναπαραγωγή βίντεο                                                                                                                                                                |
| Παύση         | χειρίζεται το γεγονός παύσης και εκτελεί ενέργειες αναλόγως                                                                                                                                                                                        |
| Φορτώθηκε     | εκτέλεση εργασιών που πρέπει να εκτελεστούν μετά την πλήρη φόρτωση του βίντεο, όπως η ενημέρωση του UI για την εμφάνιση της διάρκειας του βίντεο ή η προετοιμασία πρόσθετων λειτουργιών που σχετίζονται με την αναπαραγωγή βίντεο                  |
| Τελείωσε      | χειρίζεται το τέλος του βίντεο και εκτελεί ενέργειες όπως η εμφάνιση ενός κουμπιού επανάληψης, δείχνει σχετικό περιεχόμενο, ή την εκτέλεση οποιασδήποτε άλλης επιθυμητής λογικής, όταν το βίντεο έχει τελειώσει την αναπαραγωγή                    |

### Δεδομένα

Το στοιχείο έχει κάποια κοινώς χρησιμοποιούμενα δεδομένα, τα οποία μπορούν να κληθούν μέσω του **`{{componentName.propertyName}}`** στην εφαρμογή.

| Όνομα ιδιότητας      | Περιγραφή                                                                                                                                                        |
| -------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| αυτόματο toplay      | Μια δυαδική τιμή που καθορίζει αν το βίντεο θα ξεκινήσει αυτόματα την αναπαραγωγή όταν φορτωθεί.                                                                 |
| έλεγχοι              | Μια δυαδική τιμή που καθορίζει αν το πρόγραμμα αναπαραγωγής βίντεο ελέγχει, όπως το παιχνίδι, η παύση και η ένταση, εμφανίζονται στον χρήστη.                    |
| Εμφάνιση Ονόματος    | Μια συμβολοσειρά που αναπαριστά το όνομα ή τον τίτλο του βίντεο.                                                                                                 |
| συμβάντα             | Ένας πίνακας ή ένα αντικείμενο που καθορίζει τους χειριστές συμβάντων που σχετίζονται με το στοιχείο βίντεο, όπως το παιχνίδι, η παύση, η φόρτωση, η λήξη κλπ.   |
| hidden               | Μια δυαδική τιμή που καθορίζει αν το στοιχείο βίντεο είναι κρυμμένο από τη διεπαφή χρήστη.                                                                       |
| κύκλωμα              | Μια δυαδική τιμή που καθορίζει αν το βίντεο θα επανεκκινήσει αυτόματα την αναπαραγωγή από την αρχή μόλις φτάσει στο τέλος.                                       |
| σίγαση               | Μια δυαδική τιμή που καθορίζει αν το βίντεο θα πρέπει να είναι σε σίγαση, δηλαδή να αναπαράγεται χωρίς ήχο.                                                      |
| αναπαραγωγή          | Μια δυαδική τιμή που δείχνει αν το βίντεο αναπαράγεται αυτή τη στιγμή.                                                                                           |
| tooltipText          | τιμή συμβολοσειράς που αναπαριστά την υπόδειξη ή περνάει το κείμενο που θα εμφανιστεί όταν ο χρήστης περνάει από την εφαρμογή βίντεο.                            |
| url                  | Μια συμβολοσειρά που καθορίζει τη διεύθυνση URL ή την πηγή του αρχείου βίντεο που θα αναπαραχθεί.                                                                |
| ένταση               | Μια τιμή αριθμού μεταξύ 0 και 1 αντιπροσωπεύει το επίπεδο έντασης του βίντεο, όπου 0 είναι σε σίγαση και 1 είναι η μέγιστη ένταση.                               |

### Περίπτωση χρήσης

Στη συνέχεια, θα δείξουμε πώς να ελέγχουμε την αναπαραγωγή βίντεο με ένα κουμπί.

#### Βήμα 1 Προσθήκη Στοιχείων

Πρόσθεσε ένα στοιχείο **`Video`** και ένα **κουμπί \`\`** στον καμβά.

Ορίζουμε την πηγή βίντεο του στοιχείου βίντεο στο σύνδεσμο youtube οποιουδήποτε βίντεο που θέλετε να παίξετε. Εδώ θα χρησιμοποιήσουμε το προεπιλεγμένο βίντεο ως παράδειγμα και θα επισημάνουμε το κουμπί ως “Play” όπως φαίνεται παρακάτω

![video_layout](https://cdn.illacloud.com/official-website/img/official-site/components/video_layout.jpg)

#### Βήμα 2 Διαμόρφωση του εξαρτήματος

Για το **`button`** component, μπορούμε να το ρυθμίσουμε ώστε το **`video`** component να αναπαράγεται αν το βίντεο δεν αναπαράγεται.

1.  Κάντε κλικ στο στοιχείο `****` για να ανοίξετε τη σελίδα ελέγχου. Κάτω από **`Χειριστής Συμβάντων`**, κάντε κλικ στο `**+ Νέο**`.
2.  Στο χειριστή συμβάντων επεξεργασίας, επιλέξτε **`Control component`** εν ενεργεία, και επιλέξτε το **`video1`** component που θέλετε να αλλάξετε ως στόχο της εκδήλωσης.
3.  Επιλέξτε την ενέργεια **`play`**
4.  Στο **`Only run when`** field, βάλτε `{{!video1.playing}}`.

![video_config](https://cdn.illacloud.com/official-website/img/official-site/components/video_config.jpg)

Ομοίως, για να σταματήσουμε το video playing, δημιουργούμε ένα νέο Event handler και ακολουθούμε τα δύο πρώτα βήματα όπως παραπάνω.

1.  Επιλέξτε την **`pause`** ενέργεια
2.  Στο **`Only run when`** field, βάλτε `{{video1.playing}}`.

![video_config_1](https://cdn.illacloud.com/official-website/img/official-site/components/video_config_1.jpg)

#### Στάδιο 3 Δοκιμή

Τώρα που όταν κάνετε κλικ στο κουμπί «Αναπαραγωγή», το βίντεο θα πρέπει να παίξει, αν δεν παίζει και να σταματήσει, αν παίζει.

![video_test](https://cdn.illacloud.com/official-website/img/official-site/components/video_test.gif)

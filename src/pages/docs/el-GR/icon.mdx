## <h2 hidden>Τι είναι το εικονίδιο?</h2>

Το στοιχείο Εικονίδιο στο σύννεφο ILLA είναι ένα ισχυρό χαρακτηριστικό που επιτρέπει στους χρήστες να ενισχύσουν την οπτική έκκληση και λειτουργικότητα των εφαρμογών τους. Με το στοιχείο εικονιδίων, οι χρήστες μπορούν εύκολα να προσθέσουν εικονίδια από τις δημοφιλείς βιβλιοθήκες εικονιδίων στις διεπαφές χρήστη τους, παρέχοντας μια πιο διαισθητική και ελκυστική εμπειρία για τους χρήστες τους.

### Ιδιότητες

| Ιδιότητες           | Περιγραφή                                                                                                                                          |
| ------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------- |
| Εικονίδιο           | επιτρέπει στους χρήστες να επιλέγουν εύκολα ένα εικονίδιο από ένα προκαθορισμένο σύνολο επιλογών                                                   |
| Χειριστής Συμβάντων | Ερωτήματα ενεργοποίησης, έλεγχος εφαρμογών, ή κλήση άλλων API σε απόκριση σε συμβάντα εφαρμογής.                                                   |
| Υπόδειξη            | Οι χρήστες μπορούν να εισάγουν εδώ την υπόδειξη συστατικού. Η υπόδειξη θα εμφανίζεται όταν εστιάζεται. Υποστηρίζεται μορφή Markdown                |
| Hidden              | Δυναμική ρύθμιση της απόκρυψης του συστατικού. Μπορείτε να αλλάξετε την κρυφή κατάσταση μέσω μιας δυναμικής δυαδικής τιμής.                        |
| Χρώμα Θέματος       | Επιτρέπει στους χρήστες να καθορίσουν το χρώμα φόντου και την αδιαφάνεια του κουμπιού                                                              |

### Χειριστής συμβάντων

| Γεγονός | Περιγραφή                                                                                                          |
| ------- | ------------------------------------------------------------------------------------------------------------------ |
| Click   | Όταν ένας χρήστης κάνει κλικ σε αυτό το εικονίδιο, εκτελέστε συγκεκριμένη ενέργεια προσαρμοσμένη από τους χρήστες. |

### Δεδομένα

Το στοιχείο έχει κάποια κοινώς χρησιμοποιούμενα δεδομένα, τα οποία μπορούν να κληθούν μέσω του **`{{componentName.propertyName}}`** στην εφαρμογή.

| Όνομα ιδιότητας     | Περιγραφή                                                                                          |
| ------------------- | -------------------------------------------------------------------------------------------------- |
| colorScheme         | χρώμα πινελιάς του εικονιδίου                                                                      |
| Εμφάνιση Ονόματος   | το όνομα αυτού του στοιχείου (δηλαδή εικονίδιο)                                                    |
| συμβάντα            | δημιουργούνται ή ενεργοποιούνται όταν ορισμένες ενέργειες ή συμβάντα συμβαίνουν μέσα στο στοιχείο. |
| hidden              | κρυφή κατάσταση (αληθής ή ψευδές)                                                                  |
| εικονίδιοΌνομα      | το όνομα της ετικέτας για το εικονίδιο                                                             |
| tooltipText         | τιμή κειμένου υπόδειξης                                                                            |

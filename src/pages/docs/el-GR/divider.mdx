## <h2 hidden>Τι είναι ο Διαχωριστικός?</h2>

Στο ILLA Cloud, η συνιστώσα διαχωριστικού είναι ένα οπτικό στοιχείο που βοηθά στο διαχωρισμό ή τη διαίρεση τμημάτων περιεχομένου μέσα σε μια εφαρμογή ιστού. Είναι μια απλή οριζόντια γραμμή ή κάθετη γραμμή που μπορεί να εισαχθεί μεταξύ διαφορετικών τμημάτων ή εξαρτημάτων για τη βελτίωση της οπτικής οργάνωσης και της σαφήνειας της διεπαφής χρήστη.

### Ιδιότητες

| Ιδιότητες         | Περιγραφή                                                                                          |
| ----------------- | -------------------------------------------------------------------------------------------------- |
| Κείμενο           | προσθέστε πρόσθετες πληροφορίες στο διαχωριστικό στοιχείο                                          |
| Text align        | καθορίζει την οριζόντια ευθυγράμμιση του κειμένου εντός της συνιστώσας διαχωριστή                  |
| Hidden            | απόκρυψη του στοιχείου διαιρέτη από το περιβάλλον εργασίας χρήστη                                  |
| Μέγεθος κειμένου  | ρύθμιση του μεγέθους της γραμματοσειράς του κειμένου μέσα στο στοιχείο διαχωριστικό                |

### Δεδομένα

Το στοιχείο έχει κάποια κοινώς χρησιμοποιούμενα δεδομένα, τα οποία μπορούν να κληθούν μέσω του **`{{componentName.propertyName}}`** στην εφαρμογή.

| Όνομα ιδιότητας     | Περιγραφή                                                                              |
| ------------------- | -------------------------------------------------------------------------------------- |
| colorScheme         | καθορίζει το χρώμα του κειμένου στο στοιχείο                                           |
| Εμφάνιση Ονόματος   | αντιπροσωπεύει το όνομα ή την ετικέτα που έχει ανατεθεί στο στοιχείο                   |
| fs                  | σύστημα αρχείων (fs) που συνδέεται με το διαχωριστικό                                  |
| hidden              | τιμή boolean που καθορίζει αν το συστατικό είναι ορατό ή κρυμμένο                      |
| κείμενο             | η τιμή κειμένου                                                                        |

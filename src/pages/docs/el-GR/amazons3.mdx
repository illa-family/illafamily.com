## <h2 hidden>Τι είναι το Amazon S3?</h2>

Το Amazon S3 είναι μια εξαιρετικά επεκτάσιμη και ασφαλής υπηρεσία αποθήκευσης σύννεφων που προσφέρεται από τις Υπηρεσίες Ιστού Amazon (AWS). Όταν συνδυάζονται με το ILLA Cloud, οι προγραμματιστές μπορούν απρόσκοπτα να ενσωματώσουν το Amazon S3 στις εφαρμογές τους, αξιοποιώντας τις ισχυρές δυνατότητες αποθήκευσης του για την αποτελεσματική αποθήκευση και ανάκτηση δεδομένων. Το ILLA Cloud απλοποιεί τη διαδικασία ολοκλήρωσης, επιτρέποντας στους προγραμματιστές να συνδεθούν απρόσκοπτα με το Amazon S3 και να χρησιμοποιήσουν τα ισχυρά χαρακτηριστικά του, όπως κρυπτογράφηση δεδομένων, έκδοση εκδόσεων, έλεγχος πρόσβασης και απρόσκοπτη επεκτασιμότητα. Με την ενσωμάτωση του Amazon S3 στο ILLA Cloud, οι προγραμματιστές μπορούν να ενισχύσουν τις εφαρμογές τους με αξιόπιστες και επεκτάσιμες λύσεις αποθήκευσης, επιτρέποντάς τους να επικεντρωθούν στην οικοδόμηση καινοτόμων εμπειριών και εμπειριών με γνώμονα τα δεδομένα.

### Δημιουργία Amazon S3 API

Υπάρχουν δύο τρόποι για να δημιουργήσετε έναν πόρο στην Illa μετά την είσοδό σας στο λογαριασμό Illa.

-   Δημιουργία σε πόρους

Συνδεθείτε στον Illa λογαριασμό σας, επιλέξτε `**Πόρων**` στην κορυφή της σελίδας και κάντε κλικ στο κουμπί `**Δημιουργία Νέου**`.

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

Επιλέξτε **`Amazon S3`** από τη λίστα API.

![action_list](https://cdn.illacloud.com/official-website/img/docs/action_list.png)

Συνδεθείτε στη βάση δεδομένων με τις απαιτούμενες παραμέτρους που περιγράφονται στις παρακάτω ρυθμίσεις σύνδεσης.

Κάντε κλικ στο κουμπί `**Δοκιμή Σύνδεσης**` για να δείτε αν μπορούμε να συνδεθείτε με επιτυχία στη βάση δεδομένων. Εάν ναι, κάντε κλικ στο **`Αποθήκευση πόρων`**, αλλιώς, ελέγξτε ξανά το όνομα κεντρικού υπολογιστή, τη θύρα, το όνομα χρήστη και τον κωδικό πρόσβασης είναι σωστά.

Μετά τη δημιουργία ενός πόρου, το έτοιμο Amazon S3 θα εμφανίζεται όπως εμφανίζεται.

![amazon_ex_ready](https://cdn.illacloud.com/official-website/img/docs/assemble/amazon_ex_ready.jpg)

-   Δημιουργία στο Builder

Συνδεθείτε στον Illa λογαριασμό σας, δημιουργήστε ένα έργο στο Illa Builder στη σελίδα `**App**`, και πλοηγηθείτε στο **`Λίστα Ενεργειών`** στο κάτω μέρος της σελίδας. Κάντε κλικ στο κουμπί `**new**`, στη συνέχεια επιλέξτε **`Amazon S3`** από τη λίστα της βάσης δεδομένων. Στη συνέχεια, συνδεθείτε στη βάση δεδομένων με τις απαιτούμενες παραμέτρους που περιγράφονται στις "Ρυθμίσεις Σύνδεσης" παρακάτω.

Κάντε κλικ στο κουμπί `**Δοκιμή Σύνδεσης**` για να δείτε αν μπορούμε να συνδεθείτε με επιτυχία στη βάση δεδομένων. Εάν ναι, κάντε κλικ στο **`Αποθήκευση πόρων`**, αλλιώς, ελέγξτε ξανά το όνομα κεντρικού υπολογιστή, τη θύρα, το όνομα χρήστη και τον κωδικό πρόσβασης είναι σωστά.

### Ρυθμίσεις Σύνδεσης

Εδώ πρέπει να παρέχουμε πληροφορίες για τη σύνδεση με το Amazon S3 API.

![amazon_config](https://cdn.illacloud.com/official-website/img/docs/assemble/amazon_config.jpg)

| Ιδιότητες                             | Απαιτείται  | Περιγραφή                                                                                                                                                                                                                                                              |
| ------------------------------------- | ----------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Όνομα                                 | απαιτείται  | Το όνομα του πόρου κατά τη δημιουργία ενεργειών στο ILLA.                                                                                                                                                                                                              |
| Όνομα κουπονιού                       | προαιρετικό | ένα μοναδικό αναγνωριστικό που αντιπροσωπεύει ένα δοχείο αποθήκευσης στο Amazon S3. Παρέχει έναν λογικό διαχωρισμό για την οργάνωση και την πρόσβαση στα αρχεία σας                                                                                                    |
| S3 ACL για ανεβασμένα αρχεία          | προαιρετικό | σας επιτρέπει να ορίσετε λεπτόκοκκα δικαιώματα πρόσβασης για αρχεία μέσα στον κάδο σας. Καθορίζει ποιος μπορεί να εκτελέσει συγκεκριμένες λειτουργίες, όπως ανάγνωση, εγγραφή ή διαγραφή αρχείων.                                                                      |
| Περιοχή                               | απαιτείται  | γεωγραφική τοποθεσία όπου αποθηκεύεται ο κάδος Amazon S3. Καθορίζει τη φυσική θέση των δεδομένων σας και μπορεί να επηρεάσει παράγοντες όπως το κόστος λανθάνουσας κατάστασης και μεταφοράς δεδομένων.                                                                 |
| Προσαρμοσμένο τελικό σημείο S3        | προαιρετικό | σας επιτρέπει να ρυθμίσετε ένα συγκεκριμένο URL τελικού σημείου για την πρόσβαση στους πόρους σας Amazon S3. Μπορεί να είναι χρήσιμο για σενάρια όπου θέλετε να χρησιμοποιήσετε έναν προσαρμοσμένο τομέα ή να ενσωματώσετε με ένα CDN (Δίκτυο Παράδοσης Περιεχομένου). |
| Κλειδί πρόσβασης                      | απαιτείται  | διαπιστευτήρια που πιστοποιούν τα αιτήματά σας για πρόσβαση στο Amazon S3                                                                                                                                                                                              |
| Μυστικό κλειδί πρόσβασης              | απαιτείται  | δημιουργούνται όταν δημιουργείτε ένα ζεύγος κλειδιών IAM (Ταυτότητα και Διαχείριση Πρόσβασης) ή ένα ζεύγος κλειδιών AWS (Amazon Web Services).                                                                                                                         |

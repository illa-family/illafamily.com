## <h2 hidden>Τι είναι το Switch?</h2>

Το στοιχείο Switch είναι ένα στοιχείο διεπαφής χρήστη που επιτρέπει στους χρήστες να εναλλάσσουν μεταξύ δύο καταστάσεων: "on" και "off".

### Ιδιότητες

| Ιδιότητες           | Περιγραφή                                                                                                                                           |
| ------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------- |
| Προεπιλεγμένη τιμή  | Η αρχική τιμή του εξαρτήματος. Μπορείτε να αλλάξετε δυναμικά την αρχική τιμή πληκτρολογώντας JavaScript στην {{}}.                                  |
| Ετικέτα             | Το όνομα του πεδίου που εμφανίζεται στο χρήστη                                                                                                      |
| Λεζάντα             | Μια λεζάντα που χρησιμοποιείται για να περιγράψει λεπτομερώς το πεδίο                                                                               |
| Κρυφή ετικέτα       | Ορίστε αν θα εμφανίζεται η ετικέτα                                                                                                                  |
| Θέση                | Ορίστε τη θέση της ετικέτας σε σχέση με το στοιχείο                                                                                                 |
| Ευθυγράμμιση        | Ορίστε την ευθυγράμμιση της ετικέτας                                                                                                                |
| Χειριστής Συμβάντων | Ερωτήματα ενεργοποίησης, έλεγχος εφαρμογών, ή κλήση άλλων API σε απόκριση σε συμβάντα εφαρμογής.                                                    |
| Φόρτωση             | Αν το στοιχείο θα εμφανίζει μια ένδειξη φόρτωσης.                                                                                                   |
| Απενεργοποιημένο    | Ελέγξτε την κατάσταση του αν το στοιχείο είναι απενεργοποιημένο. Το στοιχείο δεν μπορεί να τροποποιηθεί ή να εστιαστεί όταν είναι απενεργοποιημένο. |
| Υπόδειξη            | Ο χρήστης μπορεί να εισαγάγει εδώ την υπόδειξη συστατικού. Η υπόδειξη θα εμφανίζεται όταν εστιάζεται. Υποστηρίζεται μορφή Markdown                  |
| Hidden              | Δυναμική ρύθμιση της απόκρυψης του συστατικού. Μπορείτε να αλλάξετε την κρυφή κατάσταση μέσω της δυναμικής δυαδικής τιμής.                          |
| Χρώμα Θέματος       | Επιτρέπει στους χρήστες να καθορίσουν το χρώμα φόντου και την αδιαφάνεια του κουμπιού                                                               |

### Μέθοδος

Μπορείτε να χρησιμοποιήσετε άλλα στοιχεία για τον έλεγχο της εφαρμογής. Υποστηρίζουμε τις ακόλουθες τρεις μεθόδους:

-   **ορισμός**

Για να ορίσετε την επιλεγμένη τιμή, για παράδειγμα, {{'value1'}}

| Ιδιότητες | Περιγραφή       |
| --------- | --------------- |
| Τιμή      | Επιλεγμένη τιμή |

-   **καθαρή αξία**

Για να καθαρίσετε την επιλεγμένη τιμή

-   **εναλλαγή**

αλλαγή του διακόπτη από την κατάσταση "off" στην κατάσταση "on" ή αντίστροφα

Παράδειγμα Χρήσης:

Εναλλάσσετε την υποστήριξη στοιχείων ακούγοντας την εκδήλωση `onClick` άλλων στοιχείων χρησιμοποιώντας το ενσωματωμένο σύστημα γεγονότων. Ρυθμίστε το ακολουθώντας τα παρακάτω βήματα:

1.  Προσθέστε μια σκανδάλη συμβάντος στο στοιχείο που θέλετε να ακούσετε. Για παράδειγμα, αν θέλετε να ακούσετε το **`onClick`** γεγονός ενός στοιχείου Κουμπιού, θα προσθέτετε μια σκανδάλη συμβάντος σε αυτό το στοιχείο Κουμπί.
2.  Στο χειριστή συμβάντων επεξεργασίας, επιλέξτε **`Control component`** εν ενεργεία, επιλέξτε το στοιχείο **`Switch`** που θέλετε να ενημερώσετε ως στόχο του συμβάντος.
3.  Επιλέξτε την ενέργεια **`toggle`**. Όταν ενεργοποιείται το συμβάν (δηλαδή όταν πατηθεί το κουμπί), το στοιχείο διακόπτη με εναλλαγή και η κατάστασή του θα ενημερώνεται.
4.  Αποθηκεύστε τις ρυθμίσεις ενεργοποίησης του συμβάντος και επαναλάβετε τη διαδικασία για οποιαδήποτε άλλα στοιχεία θέλετε να ακούσετε.

Μόλις ρυθμίσετε την ενεργοποίηση του γεγονότος, το στοιχείο εναλλαγής θα ενημερώνεται αυτόματα κάθε φορά που το συμβάν **`onClick`** ενεργοποιείται στα άλλα στοιχεία. Αυτό σας επιτρέπει να δημιουργήσετε δυναμικές διεπαφές που ανταποκρίνονται στην είσοδο του χρήστη σε πραγματικό χρόνο, καθιστώντας ευκολότερη την πλοήγηση και την αλληλεπίδραση των χρηστών με την εφαρμογή σας.

### Χειριστής συμβάντων

| Γεγονός | Περιγραφή                                                                                                               |
| ------- | ----------------------------------------------------------------------------------------------------------------------- |
| Αλλαγή  | Όταν ένας χρήστης αλλάζει την επιλεγμένη τιμή επιλογής, εκτελέστε συγκεκριμένη ενέργεια προσαρμοσμένη από τους χρήστες. |

### Δεδομένα

Το στοιχείο του κουμπιού έχει κάποια κοινώς χρησιμοποιούμενα δεδομένα, τα οποία μπορούν να κληθούν μέσω του **`{{componentName.propertyName}}`** στην εφαρμογή.

| Όνομα ιδιότητας   | Περιγραφή                                                     |
| ----------------- | ------------------------------------------------------------- |
| colorScheme       | χρώμα φόντου του κουμπιού                                     |
| Εμφάνιση Ονόματος | το όνομα αυτού του στοιχείου (δηλαδή το κουμπί1)              |
| hidden            | μια τιμή boolean δείχνει την κρυφή κατάσταση του κουμπιού     |
| ετικέτα           | τιμή ετικέτας                                                 |
| labelΣτοίχιση     | στοίχιση της ετικέτας διακόπτη (αριστερά ή δεξιά)             |
| ετικέταΘέση       | θέση του διακόπτη ετικέτας (αριστερά ή δεξιά)                 |
| ετικέταΠλάτος     | ο ακέραιος αριθμός που αντιπροσωπεύει το πλάτος της ετικέτας. |
| ετικέταΠλήρης     | μια δυαδική τιμή δείχνει αν η ετικέτα είναι πλήρης.           |
| τιμή              | Ένας χρήστης ορίζει την τιμή να είναι αληθής ή ψευδής         |

### Περίπτωση χρήσης

Στη συνέχεια, θα δείξουμε πώς να αλλάξετε την κατάσταση των άλλων εξαρτημάτων δυναμικά. 

#### Βήμα 1 Προσθέστε ένα στοιχείο

Πρώτα προσθέτουμε το **`Switch`** component και το ονομάζουμε 'Hide'. Στη συνέχεια, χρησιμοποιούμε το στοιχείο **`text`** ως επηρεασμένο στοιχείο που αλλάζει από την κατάσταση 'displayed' σε 'hidden'. Αλλάζουμε το περιεχόμενό του σε 'Hello World' 

#### Βήμα 2 Διαμόρφωση ενός στοιχείου

Στην ιδιοκτησία **`Hidden`** για Κείμενο. Κάντε κλικ στο εικονίδιο **`fx`** ώστε να μπορούμε να ελέγχουμε την κρυφή κατάσταση του στοιχείου ρυθμίζοντας την ίδια με την τιμή του διακόπτη μας χρησιμοποιώντας JavaScript:

```jsx
{{switch1.value}}
```

Τώρα, κάθε φορά που ο διακόπτης είναι ενεργοποιημένος, η τιμή για τον διακόπτη είναι **`true`,** το κείμενο θα είναι κρυμμένο και αντίστροφα.

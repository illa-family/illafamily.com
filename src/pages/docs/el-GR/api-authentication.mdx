## <h2 hidden>Επαλήθευση API</h2>

Το ILLA μπορεί να υποστηρίξει σχεδόν οποιοδήποτε είδος ελέγχου ταυτότητας. Έχουμε συνεργαστεί με διάφορες εταιρείες για να υποστηρίξουμε όλους τους τυποποιημένους τύπους ελέγχου ταυτότητας και έχουμε επίσης αναπτύξει μεθόδους για να υποστηρίξουμε οποιεσδήποτε προσαρμοσμένες μεθόδους ελέγχου ταυτότητας που δεν υποστηρίζουμε εγγενώς.

Μπορείτε να χρησιμοποιήσετε στατικά διακριτικά ελέγχου ταυτότητας ή να δημιουργήσετε ένα σύστημα ελέγχου ταυτότητας που θα προτρέπει τον τελικό χρήστη για τα διαπιστευτήριά του κάθε φορά που θα έχουν πρόσβαση στην εφαρμογή.

Αυτή η σελίδα θα σας δώσει οδηγίες για τον τρόπο ελέγχου ταυτότητας API σας. Αν έχετε ερωτήσεις ή χρειάζεστε βοήθεια, μπορείτε να επικοινωνήσετε με την ομάδα υποστήριξής μας χρησιμοποιώντας τη φούσκα Intercom σε αυτή τη σελίδα ή το εικονίδιο βοήθειας μέσα στον επεξεργαστή.

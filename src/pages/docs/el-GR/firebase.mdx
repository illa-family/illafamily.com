## <h2 hidden>Τι είναι το Firebase?</h2>

Το Firebase είναι μια ολοκληρωμένη πλατφόρμα που παρέχεται από την Google και προσφέρει ένα ευρύ φάσμα εργαλείων και υπηρεσιών για την ανάπτυξη και διαχείριση εφαρμογών web και για φορητές συσκευές. Παρέχει στους προγραμματιστές μια επεκτάσιμη και ασφαλή υποδομή backend, μαζί με μια σουίτα προκατασκευασμένων χαρακτηριστικών και λειτουργιών για την επιτάχυνση της διαδικασίας ανάπτυξης. Με την ενσωμάτωση του Firebase στο ILLA Cloud, οι προγραμματιστές μπορούν να ενισχύσουν τις εφαρμογές τους με αξιόπιστες και επεκτάσιμες λύσεις αποθήκευσης, επιτρέποντάς τους να επικεντρωθούν στην οικοδόμηση καινοτόμων εμπειριών και εμπειριών με γνώμονα τα δεδομένα.

### Δημιουργία API Firebase

Υπάρχουν δύο τρόποι για να δημιουργήσετε έναν πόρο στην Illa μετά την είσοδό σας στο λογαριασμό Illa.

-   Δημιουργία σε πόρους

Συνδεθείτε στον Illa λογαριασμό σας, επιλέξτε `**Πόρων**` στην κορυφή της σελίδας και κάντε κλικ στο κουμπί `**Δημιουργία Νέου**`.

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

Επιλέξτε **`Firebase`** από τη λίστα API.

![action_list](https://cdn.illacloud.com/official-website/img/docs/action_list.png)

Συνδεθείτε στη βάση δεδομένων με τις απαιτούμενες παραμέτρους που περιγράφονται στις παρακάτω ρυθμίσεις σύνδεσης.

Κάντε κλικ στο κουμπί `**Δοκιμή Σύνδεσης**` για να δείτε αν μπορούμε να συνδεθείτε με επιτυχία στη βάση δεδομένων. Εάν ναι, κάντε κλικ στο **`Αποθήκευση πόρων`**, αλλιώς, ελέγξτε ξανά το όνομα κεντρικού υπολογιστή, τη θύρα, το όνομα χρήστη και τον κωδικό πρόσβασης είναι σωστά.

Μετά τη δημιουργία ενός πόρου, το έτοιμο Firebase θα εμφανίζεται όπως εμφανίζεται.

![firebase_ex_ready](https://cdn.illacloud.com/official-website/img/official-site/intergration/firebase_ex_ready.jpg)

-   Δημιουργία στο Builder

Συνδεθείτε στον Illa λογαριασμό σας, δημιουργήστε ένα έργο στο Illa Builder στη σελίδα `**App**`, και πλοηγηθείτε στο **`Λίστα Ενεργειών`** στο κάτω μέρος της σελίδας. Κάντε κλικ στο κουμπί `**new**`, και μετά επιλέξτε **`Firebase`** από τη λίστα της βάσης δεδομένων. Στη συνέχεια, συνδεθείτε στη βάση δεδομένων με τις απαιτούμενες παραμέτρους που περιγράφονται στις "Ρυθμίσεις Σύνδεσης" παρακάτω.

Κάντε κλικ στο κουμπί `**Δοκιμή Σύνδεσης**` για να δείτε αν μπορούμε να συνδεθείτε με επιτυχία στη βάση δεδομένων. Εάν ναι, κάντε κλικ στο **`Αποθήκευση πόρων`**, αλλιώς, ελέγξτε ξανά το όνομα κεντρικού υπολογιστή, τη θύρα, το όνομα χρήστη και τον κωδικό πρόσβασης είναι σωστά.

#### Λήψη χρήστη από UID

ανάκτηση πληροφοριών χρήστη, προσδιορίζοντας το μοναδικό αναγνωριστικό του χρήστη (UID). Σας επιτρέπει να λάβετε τα στοιχεία του χρήστη, όπως το όνομα, το ηλεκτρονικό ταχυδρομείο, τον αριθμό τηλεφώνου και προσαρμοσμένα χαρακτηριστικά που σχετίζονται με το χρήστη.

** Input **

| Ιδιότητες  | Απαιτείται | Περιγραφή                                                                                                                               |
| ---------- | ---------- | --------------------------------------------------------------------------------------------------------------------------------------- |
| UID        | απαιτείται | "Αναγνωριστικό Χρήστη." Πρόκειται για ένα μοναδικό αναγνωριστικό που εκχωρείται σε κάθε χρήστη στο σύστημα ελέγχου ταυτότητας Firebase. |

## <h2 hidden>Τι είναι η νιφάδα Snowflake?</h2>

Στην Illa, μπορείτε να χρησιμοποιήσετε τον επεξεργαστή ερωτημάτων Snowflake για να εκτελέσετε ερωτήματα SQL ενάντια στην αποθήκη δεδομένων Snowflake σας. Μπορείτε να συνδεθείτε στο Snowflake εισάγοντας τα διαπιστευτήρια του λογαριασμού σας Snowflake, συμπεριλαμβανομένου του ονόματος λογαριασμού, του ονόματος χρήστη, του κωδικού πρόσβασης και του ονόματος αποθήκης. Μόλις συνδεθείτε στο Snowflake, μπορείτε να χρησιμοποιήσετε τον επεξεργαστή ερωτήματος για να γράψετε και να εκτελέσετε ερωτήματα SQL για να ανακτήσετε, να χειριστείτε και να αναλύσετε τα δεδομένα σας.

### Δημιουργία Νιφάδας Χιονιού

Υπάρχουν δύο τρόποι για να δημιουργήσετε έναν πόρο στην Illa μετά την είσοδό σας στο λογαριασμό Illa.

-   Δημιουργία σε πόρους

Συνδεθείτε στον Illa λογαριασμό σας, επιλέξτε `**Πόροι**` στην κορυφή της σελίδας και κάντε κλικ στο κουμπί `**Δημιουργία Νέου**`.

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

Επιλέξτε **`Snowflake`** από τη λίστα της βάσης δεδομένων.

![action_list](https://cdn.illacloud.com/official-website/img/docs/action_list.png)

Συνδεθείτε στη βάση δεδομένων με τις απαιτούμενες παραμέτρους που περιγράφονται στις παρακάτω ρυθμίσεις σύνδεσης.

Κάντε κλικ στο κουμπί `**Δοκιμή Σύνδεσης**` για να δείτε αν μπορούμε να συνδεθείτε με επιτυχία στη βάση δεδομένων. Εάν ναι, κάντε κλικ στο **`Αποθήκευση πόρων`**, αλλιώς, ελέγξτε ξανά το όνομα κεντρικού υπολογιστή, τη θύρα, το όνομα χρήστη και τον κωδικό πρόσβασης είναι σωστά.

Μετά τη δημιουργία ενός πόρου, το έτοιμο νιφάδα χιονιού θα εμφανίζεται όπως εμφανίζεται.

![snow_ex_ready](https://cdn.illacloud.com/official-website/img/docs/snow_ex_Ready.png)

-   Δημιουργία στο Builder

Συνδεθείτε στον Illa λογαριασμό σας, δημιουργήστε ένα έργο στο Illa Builder στη σελίδα `**App**`, και πλοηγηθείτε στο **`Λίστα Ενεργειών`** στο κάτω μέρος της σελίδας. Κάντε κλικ στο κουμπί `**new**`, στη συνέχεια επιλέξτε **`Snowflake`** από τη λίστα της βάσης δεδομένων. Στη συνέχεια, συνδεθείτε στη βάση δεδομένων με τις απαιτούμενες παραμέτρους που περιγράφονται στις "Ρυθμίσεις Σύνδεσης" παρακάτω.

Κάντε κλικ στο κουμπί `**Δοκιμή Σύνδεσης**` για να δείτε αν μπορούμε να συνδεθείτε με επιτυχία στη βάση δεδομένων. Εάν ναι, κάντε κλικ στο **`Αποθήκευση πόρων`**, αλλιώς, ελέγξτε ξανά το όνομα κεντρικού υπολογιστή, τη θύρα, το όνομα χρήστη και τον κωδικό πρόσβασης είναι σωστά.

### Ρυθμίσεις Σύνδεσης

Εδώ πρέπει να παρέχουμε πληροφορίες για τη σύνδεση με τη βάση δεδομένων Snowflake.

![snow_config](https://cdn.illacloud.com/official-website/img/docs/snow_config.png)

| Ιδιότητες            | Απαιτείται  | Περιγραφή                                                                                                                                                                   |
| -------------------- | ----------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Όνομα                | απαιτείται  | Το όνομα του πόρου κατά τη δημιουργία ενεργειών στο ILLA.                                                                                                                   |
| Όνομα λογαριασμού    | απαιτείται  | μοναδικό αναγνωριστικό που έχει ανατεθεί σε κάθε λογαριασμό Snowflake                                                                                                       |
| Αποθήκη              | απαιτείται  | δέσμη υπολογιστικών πόρων που χρησιμοποιούνται για την επεξεργασία ερωτημάτων και την εκτέλεση αναλυτικών φορτίων εργασίας                                                  |
| Βάση Δεδομένων       | απαιτείται  | Το όνομα της βάσης δεδομένων                                                                                                                                                |
| Σχήμα                | προαιρετικό | δοχείο που περιέχει αντικείμενα βάσης δεδομένων, όπως πίνακες, προβολές και διαδικασίες                                                                                     |
| Ρόλος                | προαιρετικό | ένα ονοματεπώνυμο σύνολο δικαιωμάτων που μπορούν να εκχωρηθούν σε έναν ή περισσότερους χρήστες                                                                              |
| Ταυτοποίηση          | απαιτείται  | ασφαλή πρόσβαση στα δεδομένα και τους πόρους σας στην αποθήκη δεδομένων Snowflake (Υποστηρίζονται μόνο τα κρυπτογραφημένα ιδιωτικά κλειδιά))                                |
| Όνομα Χρήστη         | απαιτείται  | το όνομα χρήστη που θέλετε να χρησιμοποιήσετε κατά τη σύνδεση στον εξυπηρετητή Snowflake.                                                                                   |
| Κωδικός              | απαιτείται  | Χρησιμοποιήστε αυτόν τον κωδικό πρόσβασης για ταυτοποίηση.                                                                                                                  |

### Δημιουργία Ενεργειών

Έχουμε δημιουργήσει έναν πόρο νιφάδας Snowflake, μπορούμε να προσθέσουμε την ενέργεια επιλέγοντας Snowflake από τη λίστα ενεργειών και επιλέγοντας το κουμπί **`Δημιουργία δράσης`**.

![snow_resource_list](https://cdn.illacloud.com/official-website/img/docs/snow_resource_list.png)

Τώρα έχουμε προσθέσει τον εξυπηρετητή Snowflake ως ενέργεια στη σελίδα μας για το κτίριο.

![snow](https://cdn.illacloud.com/official-website/img/docs/snow.png)

### Ρύθμιση Νιφάδων Χιονιού

| Ιδιότητες       | Περιγραφή                                                                                     |
| --------------- | --------------------------------------------------------------------------------------------- |
| Ερώτημα SQL     | εντολές sql για λήψη (επιλογή), εισαγωγή, ενημέρωση, διαγραφή δεδομένων από τη βάση δεδομένων |
| Μετασχηματιστής | μετατρέποντας τα δεδομένα σε στυλ που σας αρέσει χρησιμοποιώντας JavaScript                   |

Χρήση παραδείγματος:

-   Επιλογή

```sql
SELECT * ΑΠΟ χρήστες ORDER BY επώνυμο;
```

-   Insert

```sql
INSERT INTO users

	(firstname, lastname, email)

VALUES

	(

		{{ NewUser.data.firstNameInput }},

		{{ NewUser.data.lastNameInput }},

		{{ NewUser.data.emailInput }}

	);
```

-   Ενημέρωση

```sql
UPDATE users
  SET firstname = '{{form1.updatedProfile.first}}'
  WHERE id = {{ form1.updatedProfile.uid}};
```

-   Διαγραφή

```sql
DELETE FROM users WHERE id = {{ form1.updatedProfile.uid }};
```

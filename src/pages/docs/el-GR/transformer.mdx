## <h2 hidden>Μετασχηματιστής</h2>

Έχουμε βάλει ένα μετασχηματιστή στο ILLA για να καλύψει τις απαιτήσεις σας για τη σύνταξη JavaScript για να βελτιώσετε τη λογική της εφαρμογής σας. Στο Transformer, δεν χρειάζεται να κωδικοποιήσετε σε JavaScript χρησιμοποιώντας {{template syntax}} όπως κάνετε στο πλαίσιο εισαγωγής χαρακτήρων, έτσι ώστε να μπορείτε να γράψετε μακρύ και πολύπλοκο μπλοκ κώδικα JS πιο άπτατα. Επιπλέον, ο Μετασχηματιστής βοηθά τους προγραμματιστές που είναι υπεύθυνοι για το σχεδιασμό σελίδων και την κατασκευή λογικής για να ολοκληρώσουν τις εργασίες ανάπτυξής τους πιο αποτελεσματικά, χωρίς να περιορίζονται από την πρόοδο του άλλου.

Μπορείτε να χρησιμοποιήσετε τον Μετασχηματιστή όχι μόνο για καθιερωμένες λειτουργίες δεδομένων αλλά και για τη δημιουργία λειτουργιών μετασχηματιστή κατά τη μετατροπή δεδομένων. Θα ήταν καλύτερο να επιλέξετε έναν ανεξάρτητο μετασχηματιστή όταν έχετε πολλαπλές λειτουργίες που εμπλέκονται στη μετατροπή δεδομένων, ή πρέπει να χρησιμοποιήσετε πολύπλοκη λογική JavaScript για να μετατρέψει τα δεδομένα.

### Δημιουργικοί Μετασχηματιστές

Κάντε κλικ στην καρτέλα Μετασχηματιστές στη λίστα ενεργειών, και κάντε κλικ στο "+ Νέο" για να δημιουργήσετε ένα μετασχηματιστή. Θα εμφανιστεί στη λίστα ενεργειών μετά τη δημιουργία. Μπορείτε να αλλάξετε το όνομα, να γράψετε τον κώδικα JS, να τον αποθηκεύσετε και να τον εκτελέσετε στον πίνακα ρυθμίσεων μετά την επιλογή.

> Μπορείτε να χρησιμοποιήσετε το κουμπί 'προεπισκόπηση' για να δείτε τα αποτελέσματα ενός μετασχηματιστή. Σημειώστε ότι πρέπει να αποθηκεύσετε τον μετασχηματιστή για το υπόλοιπο της εφαρμογής για να πάρετε τη νέα τιμή.

Το ένθετο καλεί τον μετασχηματιστή με μια τιμή ίση με τα δεδομένα που επιστρέφονται εντός του μετασχηματιστή, τα δεδομένα αυτά θα αλλάξουν καθώς τα δεδομένα που εισάγονται στον μετασχηματιστή αλλάζουν με τον κωδικό του μετασχηματιστή. Όπως και με το πλαίσιο εισαγωγής χαρακτήρων, μπορείτε ακόμα να χρησιμοποιήσετε τη σύνταξη του προτύπου για να καλέσετε την παγκόσμια μεταβλητή αριθ. Όταν γράφετε κώδικα JS για να καλέσετε δεδομένα, μπορείτε να βρείτε το συγκεκριμένο όνομα μεταβλητής δεδομένων στο χώρο εργασίας δεδομένων στον αριστερό πίνακα για να σας βοηθήσει να γράψετε κώδικα πιο γρήγορα.

Μπορείτε να καλέσετε τη μεταβλητή του μετασχηματιστή στο πλαίσιο εισόδου δεδομένων χρησιμοποιώντας `{transformerName. tagName}}` για να καλέσετε τη μεταβλητή μετασχηματιστή στο πλαίσιο καταχώρησης δεδομένων.

### Μετασχηματιστής Ερωτήματος

Έχουμε βάλει ένα Μετασχηματιστή στη λειτουργία ερωτήματος για να σας επιτρέψουμε να μετατρέψετε τα δεδομένα που έχετε λάβει στην επιθυμητή μορφή απευθείας μετά την ολοκλήρωση της ενέργειας Μπορείτε στη συνέχεια να περάσετε το {{actionName.data}} για να ανακτήσετε τα δεδομένα μόλις ολοκληρώσετε τη μετατροπή στον μετασχηματιστή εντός της ενέργειας.

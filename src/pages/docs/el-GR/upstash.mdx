## <h2 hidden>Τι είναι Upstash?</h2>

Το Upstash είναι μια υπηρεσία χωρίς διακομιστές cloud-native με συμβατό το Redis που παρέχει έναν εύκολο τρόπο για την ενσωμάτωση της λειτουργίας Redis σε εφαρμογές. Στο πλαίσιο του ILLA Cloud, η ενσωμάτωση Upstash επιτρέπει στους χρήστες να αξιοποιήσουν τα χαρακτηριστικά και τις δυνατότητες του Upstash μέσα στην εφαρμογή ILLA Cloud.

Αυτό το σεμινάριο περιγράφει τη διαδικασία δημιουργίας ενός Πίνακα Διαχειριστή χρησιμοποιώντας ILLA Builder και Upstash σε μερικά απλά βήματα. Το ILLA Cloud είναι μια πλατφόρμα χαμηλού κώδικα για προγραμματιστές που επιτρέπει την ταχεία ανάπτυξη και ανάπτυξη εσωτερικών εργαλείων. Επιτρέπει τη δημιουργία σελίδων σύροντας και απορρίπτοντας στοιχεία UI, που συνδέονται με οποιαδήποτε βάση δεδομένων ή API, και γράφοντας JavaScript. Για να μάθετε περισσότερα σχετικά με το Upstash, επισκεφθείτε την ιστοσελίδα τους στο https://upstash.com/(https://upstash.com/) Ας ξεκινήσουμε!

### Ρυθμίστε το Πίσω Τέλος στο Upstash

Δημιουργία λογαριασμού στο Upstash. Μπορείτε να συνδεθείτε χρησιμοποιώντας τους λογαριασμούς σας Amazon, Github ή Google εναλλακτικά.

Μετά την είσοδο στο λογαριασμό σας στο `Upstash`, κάντε κλικ στο κουμπί `Δημιουργία βάσης δεδομένων` στο Redis.

![up_home](https://cdn.illacloud.com/official-website/img/docs/up_home.jpg)

Στον πίνακα `Upstash`, κάντε κλικ στο κουμπί `Δημιουργία βάσης δεδομένων`.

![up_create_db](https://cdn.illacloud.com/official-website/img/docs/up_create_db.jpg)

Σημείωση: μπορείτε να έχετε μόνο μια βάση δεδομένων στην ελεύθερη βαθμίδα.

Για τις ρυθμίσεις της βάσης δεδομένων, Μπορείτε να ονομάσετε τη βάση δεδομένων σας όπως θέλετε. Εδώ το ονομάσαμε `db1`. Επιλέξτε την "κύρια περιοχή" και την "αναγνωσμένη περιοχή" βάσει των τοποθεσιών εγγραφής και ανάγνωσης χρηστών. Ενεργοποίηση `TSL/SSL`. Είναι προαιρετικό να ενεργοποιήσετε το Eviction.

![up_db_config](https://cdn.illacloud.com/official-website/img/docs/up_db_config.jpg)

Κάντε κλικ στο κουμπί **`create`**, τώρα έχουμε μια βάση δεδομένων έτοιμη για χρήση.

### Δημιουργία Upstash στο σύννεφο ILLA

Υπάρχουν δύο τρόποι για να δημιουργήσετε έναν πόρο στην Illa μετά την είσοδό σας στο λογαριασμό Illa.

-   Δημιουργία σε πόρους

Συνδεθείτε στον Illa λογαριασμό σας, επιλέξτε `**Πόρων**` στην κορυφή της σελίδας και κάντε κλικ στο κουμπί `**Δημιουργία Νέου**`.

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

Επιλέξτε **`Upstash`** από τη λίστα της βάσης δεδομένων.

![action_list_0](https://cdn.illacloud.com/official-website/img/docs/action_list_0.jpg)

Συνδεθείτε στη βάση δεδομένων με τις απαιτούμενες παραμέτρους που περιγράφονται στις παρακάτω ρυθμίσεις σύνδεσης.

Κάντε κλικ στο κουμπί `**Δοκιμή Σύνδεσης**` για να δείτε αν μπορούμε να συνδεθείτε με επιτυχία στη βάση δεδομένων. Εάν ναι, κάντε κλικ στο **`Αποθήκευση πόρων`**, αλλιώς, ελέγξτε ξανά το όνομα κεντρικού υπολογιστή, τη θύρα, το όνομα χρήστη και τον κωδικό πρόσβασης είναι σωστά.

Μετά τη δημιουργία ενός πόρου, το έτοιμο `Upstash` θα εμφανιστεί όπως εμφανίζεται.

![up_ex_ready](https://cdn.illacloud.com/official-website/img/docs/up_ex_ready.jpg)

-   Δημιουργία στο Builder

Συνδεθείτε στον Illa λογαριασμό σας, δημιουργήστε ένα έργο στο Illa Builder στη σελίδα `**App**`, και πλοηγηθείτε στο **`Λίστα Ενεργειών`** στο κάτω μέρος της σελίδας. Κάντε κλικ στο κουμπί `**new**`, και μετά επιλέξτε **`Upstash`** από τη λίστα της βάσης δεδομένων. Στη συνέχεια, συνδεθείτε στη βάση δεδομένων με τις απαιτούμενες παραμέτρους που περιγράφονται στις "Ρυθμίσεις Σύνδεσης" παρακάτω.

Κάντε κλικ στο κουμπί `**Δοκιμή Σύνδεσης**` για να δείτε αν μπορούμε να συνδεθείτε με επιτυχία στη βάση δεδομένων. Εάν ναι, κάντε κλικ στο **`Αποθήκευση πόρων`**, αλλιώς, ελέγξτε ξανά το όνομα κεντρικού υπολογιστή, τη θύρα, το όνομα χρήστη και τον κωδικό πρόσβασης είναι σωστά.

### Ρύθμιση Upstash

| Ιδιότητες       | Περιγραφή                                                                                               |
| --------------- | ------------------------------------------------------------------------------------------------------- |
| Ερώτημα Redis   | χρήση ερωτήματος για ανάκτηση και επεξεργασία δεδομένων που είναι αποθηκευμένα στη βάση δεδομένων Redis |
| Μετασχηματιστής | μετατρέποντας τα δεδομένα σε στυλ που σας αρέσει χρησιμοποιώντας JavaScript                             |

Χρήση παραδείγματος:

1.  Λήψη Τιμών:
    -   Λάβετε μια τιμή: Χρησιμοποιήστε την εντολή **`GET`** για να ανακτήσετε την τιμή που σχετίζεται με ένα συγκεκριμένο κλειδί. Για παράδειγμα: **`GET key`**.
2.  Ορισμός Τιμές:
    -   Όρισε μια τιμή: Χρησιμοποίησε την εντολή **`SET`** για να ορίσεις την τιμή ενός κλειδιού. Για παράδειγμα **`set key value`**.
3.  Λειτουργίες Hash:
    -   Ορίστε ζεύγη τιμών πεδίου σε ένα hash: Χρησιμοποιήστε την εντολή **`HSET`** για να ορίσετε ένα ζεύγος τιμών πεδίου σε ένα hash. Για παράδειγμα: **`HSET hashKey field value`**.
    -   Λάβετε μια τιμή από ένα hash: Χρησιμοποιήστε την εντολή **`HGET`** για να ανακτήσετε την τιμή ενός συγκεκριμένου πεδίου σε ένα hash. Για παράδειγμα: **`HGET hashKey field`**.
4.  Λειτουργίες Λίστας:
    -   Πρόσθεσε ένα στοιχείο σε μια λίστα: Χρησιμοποίησε την εντολή **`LPUSH`** ή **`RPUSH`** για να προσθέσεις ένα στοιχείο στην αρχή ή στο τέλος μιας λίστας, αντίστοιχα. Για παράδειγμα: **`LPUSH listKey value`**.
    -   Ανάκτηση στοιχείων από μια λίστα: Χρησιμοποίησε την εντολή **`LRANGE`** για να φέρεις ένα εύρος στοιχείων από μια λίστα. Για παράδειγμα: **`LRANGE listKey start end`**.
5.  Ορισμός Λειτουργιών:
    -   Πρόσθεσε στοιχεία σε ένα σύνολο: Χρησιμοποίησε την εντολή **`SADD`** για να προσθέσεις ένα ή περισσότερα στοιχεία σε ένα σύνολο. Για παράδειγμα: **`SADD setKey value1 value2`**.
    -   Ανάκτηση στοιχείων από ένα σετ: Χρησιμοποιήστε την εντολή **`SMEMBERS`** για να πάρετε όλα τα μέλη ενός σετ. Για παράδειγμα: **`SMEMBERS setKey`**.
6.  Ταξινομημένες Καθορισμένες Λειτουργίες:
    -   Πρόσθεσε στοιχεία σε ένα ταξινομημένο σύνολο: Χρησιμοποίησε την εντολή **`ZADD`** για να προσθέσεις στοιχεία σε ένα ταξινομημένο σύνολο μαζί με τις βαθμολογίες τους. Για παράδειγμα: **`ZADD ταξινόμησηSetKey score1 member1 score2 member2`**.
    -   Ανάκτηση στοιχείων από ένα ταξινομημένο σύνολο: Χρησιμοποιήστε την εντολή **`ZRANGE`** για να φέρετε ένα εύρος στοιχείων από ένα ταξινομημένο σύνολο με βάση τις βαθμολογίες τους. Για παράδειγμα: **`ZRANGE sortedSetKey start end`**.

Σημείωση: Μπορείτε να ανατρέξετε στην τεκμηρίωση Redis για μια πλήρως ολοκληρωμένη λίστα εντολών και τη χρήση τους.

### Περίπτωση χρήσης

Στη συνέχεια, θα δείξουμε πώς να αντιστοιχίσουμε τα δεδομένα από το `**Upstash**` στο στοιχείο `table`.

#### Βήμα 1 Προσθέστε μια ενέργεια

Ας δημιουργήσουμε έναν πίνακα στην Upstash που ονομάζεται **`employee_profile`,** συμπεριλαμβανομένων 4 στηλών `name`, `level`, `age`, and `married`.

Στη συνέχεια, μπορούμε να δημιουργήσουμε μια νέα ενέργεια για Upstash από τη λίστα ενεργειών στο ILLA Cloud και το όνομα upstash2.

Για να απαριθμήσετε όλα τα δεδομένα στο employee_profile βάλτε τον κώδικα αποσπάσματος παρακάτω στην ενότητα ερώτημα Redis.

```sql
GET employee_profile
```

Δεδομένου ότι τα δεδομένα εξόδου είναι μια λίστα με ζεύγος λεξικών, θέλουμε να τα μετατρέψουμε σε ένα πίνακα. Στο Transformer, ενεργοποιήστε το και βάλτε τον παρακάτω κώδικα.

```jsx
const jsonString = data[0].result;

// Parse the JSON string into a JavaScript object
const jsonData = JSON.parse(jsonString);

// Access the "employees" property and map it to a new array
const mappedArray = jsonData.employees.map((employee) => ({
  name: employee.name,
  level: employee.level,
  age: employee.age,
  married: employee.married,
}));

return mappedArray;
```

![up_code](https://cdn.illacloud.com/official-website/img/docs/up_code.jpg)

Πατήστε **`Save`** και **`Run`** για να ενεργοποιήσετε αυτήν την ενέργεια.

![up_data](https://cdn.illacloud.com/official-website/img/docs/up_data.jpg)

#### Βήμα 2 Προσθήκη Συστατικών

Στη συνέχεια, μπορούμε να προσθέσουμε ένα στοιχείο **`table`** στον καμβά.

Στο **πεδίο `Data Source`** στη σελίδα Inspect αφού κάνετε κλικ στο στοιχείο, βάλτε `{{upstash2.data}}` για να ανακτήσετε τα δεδομένα από τον upstash.

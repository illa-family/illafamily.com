## <h2 hidden>Τι είναι το MongoDB?</h2>

Η MongoDB είναι μια δημοφιλής βάση δεδομένων με προσανατολισμό το έγγραφο NoSQL που χρησιμοποιείται ευρέως για διαδικτυακές εφαρμογές και άλλα λογισμικό με γνώμονα τα δεδομένα. Η MongoDB έχει σχεδιαστεί για να είναι ευέλικτη, επεκτάσιμη και γρήγορη, και είναι ιδιαίτερα κατάλληλη για τη διαχείριση μεγάλων ποσοτήτων μη δομημένων ή ημι-δομημένων δεδομένων.

Στην Illa, μπορείτε να συνδεθείτε σε μια βάση δεδομένων MongoDB χρησιμοποιώντας το MongoDB Query Resource. Αυτός ο πόρος σας επιτρέπει να εκτελέσετε ερωτήματα MongoDB απευθείας από την Illa, χωρίς να χρειάζεται να γράψετε οποιοδήποτε κώδικα. Μπορείτε να χρησιμοποιήσετε τον επεξεργαστή ερωτήματος για να γράψετε ερωτήματα στη μητρική γλώσσα ερωτημάτων του MongoDB, το οποίο βασίζεται σε JavaScript και χρησιμοποιεί σύνταξη τύπου JSON-όπως για εργασία με έγγραφα.

### Create MongoDB

Υπάρχουν δύο τρόποι για να δημιουργήσετε έναν πόρο στην Illa μετά την είσοδό σας στο λογαριασμό Illa.

-   Δημιουργία σε πόρους

Συνδεθείτε στον Illa λογαριασμό σας, επιλέξτε `**Πόροι**` στην κορυφή της σελίδας και κάντε κλικ στο κουμπί `**Δημιουργία Νέου**`.

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

Επιλέξτε **`MongoDB`** από τη λίστα της βάσης δεδομένων.

![action_list](https://cdn.illacloud.com/official-website/img/docs/action_list.png)

Συνδεθείτε στη βάση δεδομένων με τις απαιτούμενες παραμέτρους που περιγράφονται στις παρακάτω ρυθμίσεις σύνδεσης.

Κάντε κλικ στο κουμπί `**Δοκιμή Σύνδεσης**` για να δείτε αν μπορούμε να συνδεθείτε με επιτυχία στη βάση δεδομένων. Εάν ναι, κάντε κλικ στο **`Αποθήκευση πόρων`**, αλλιώς, ελέγξτε ξανά το όνομα κεντρικού υπολογιστή, τη θύρα, το όνομα χρήστη και τον κωδικό πρόσβασης είναι σωστά.

Μετά τη δημιουργία ενός πόρου, το έτοιμο MongoDB θα εμφανίζεται όπως εμφανίζεται.

![mongo_ex_ready](https://cdn.illacloud.com/official-website/img/docs/mongo_ex_ready.png)

-   Δημιουργία στο Builder

Συνδεθείτε στον Illa λογαριασμό σας, δημιουργήστε ένα έργο στο Illa Builder στη σελίδα `**App**`, και πλοηγηθείτε στο **`Λίστα Ενεργειών`** στο κάτω μέρος της σελίδας. Κάντε κλικ στο κουμπί `**new**` και μετά επιλέξτε **`MongoDB`** από τη λίστα της βάσης δεδομένων. Στη συνέχεια, συνδεθείτε στη βάση δεδομένων με τις απαιτούμενες παραμέτρους που περιγράφονται στις "Ρυθμίσεις Σύνδεσης" παρακάτω.

Κάντε κλικ στο κουμπί `**Δοκιμή Σύνδεσης**` για να δείτε αν μπορούμε να συνδεθείτε με επιτυχία στη βάση δεδομένων. Εάν ναι, κάντε κλικ στο **`Αποθήκευση πόρων`**, αλλιώς, ελέγξτε ξανά το όνομα κεντρικού υπολογιστή, τη θύρα, το όνομα χρήστη και τον κωδικό πρόσβασης είναι σωστά.

### Configure MongoDB

#### Επισκόπηση

| Όνομα Μεθόδου   | Περιγραφή                                                                                    |
| --------------- | -------------------------------------------------------------------------------------------- |
| Τύπος Ενέργειας | aggregate, bulkwrite, count, deleteMany. deleteOne, ευδιάκριτο, βρείτε, βρείτεΈνα            |
| Συλλογή         | ομάδα συναφών εγγράφων που αποθηκεύονται μαζί σε βάση δεδομένων                              |
| Μετασχηματιστής | μετατρέποντας τα δεδομένα σε στυλ που σας αρέσει χρησιμοποιώντας JavaScript                  |

#### Συγκεντρωτικό

επεξεργάζεται πολλαπλά έγγραφα και τα υπολογισμένα αποτελέσματα επιστροφής.

** Input **

| Ιδιότητες  | Απαιτείται  | Περιγραφή                                                     |
| ---------- | ----------- | ------------------------------------------------------------- |
| Άθροιση    | προαιρετικό | καθορίζει τις εργασίες που πρέπει να εκτελούνται στα δεδομένα |

Για παράδειγμα, για να πάρετε όλα τα αποτελέσματα των οποίων το μέγεθος είναι medium=>ομαδοποιήστε τα αποτελέσματα κατά type=>Υπολογίστε το άθροισμα των τιμών σε κάθε τύπο για **`Aggregation`**:

```sql
[
	{
		"$match": { "size": "medium" }
	},
	{
		"$group": { "_id": "$type", "totalQuantity": { "$sum": "$price" } }
	}
]
```

** Έξοδος **

Μια σειρά από υπολογισμένα αποτελέσματα.

![mongo_agg_data](https://cdn.illacloud.com/official-website/img/docs/mongo_agg_data.png)

Χρησιμοποίησε το `{{mongodb1.data[0].result}}` για να πάρεις όλα.

![mongo_agg_code](https://cdn.illacloud.com/official-website/img/docs/mongo_agg_code.png)

#### Bulkwrite

εκτέλεση πολλαπλών εργασιών εγγραφής (εισαγωγή, ενημέρωση και διαγραφή) σε ένα αίτημα στο διακομιστή

** Input **

| Ιδιότητες   | Απαιτείται  | Περιγραφή                                                                                                        |
| ----------- | ----------- | ---------------------------------------------------------------------------------------------------------------- |
| Λειτουργίες | προαιρετικό | αντικείμενο που προσδιορίζει τον τύπο της πράξης που πρέπει να εκτελεστεί και τα δεδομένα που πρέπει να γραφτούν |

Ένα παράδειγμα για **`Operations`**:

```sql
[
	{ "insertOne": { "document": { "_id": 3, "type": "beef", "size": "medium", "price": 6 } } },
	{ "insertOne": { "document": { "_id": 4, "type": "sausage", "size": "large", "price": 10 } } },
	{ "updateOne": {
			"filter": { "type": "cheese" },
			"update": { "$set": { "price": 8 } }
		}
	}
]
```

** Έξοδος **

Το ενημερωμένο μήνυμα εμφανίζεται παρακάτω.

![mongo_bulk_data](https://cdn.illacloud.com/official-website/img/docs/mongo_bulk_data.png)

Δεδομένου ότι αυτή η μέθοδος κάνει αλλαγές, αλλά δεν επιστρέφει τίποτα. Δεν υπάρχουν δεδομένα εξόδου για πρόσβαση.

#### Μέτρηση

μετράει τον αριθμό των εγγράφων που ταιριάζουν με ένα δοσμένο ερώτημα σε μια συλλογή

** Input **

| Ιδιότητες    | Απαιτείται  | Περιγραφή                                                 |
| ------------ | ----------- | --------------------------------------------------------- |
| Ερώτημα      | προαιρετικό | καθορίζει τα κριτήρια επιλογής για την πράξη καταμέτρησης |

Για παράδειγμα, για να μετρήσουμε τον αριθμό του αντικειμένου με τύπο “βοείου κρέατος”, μπορεί να βάλουμε τον παρακάτω κώδικα σε **`Query`**:

```sql
{"τύπος":"βόειο κρέας"}
```

#### βρες

ανάκτηση εγγράφων από μια συλλογή που ταιριάζει με ένα καθορισμένο σύνολο κριτηρίων

** Input **

| Ιδιότητες          | Απαιτείται  | Περιγραφή                                                                                                                            |
| ------------------ | ----------- | ------------------------------------------------------------------------------------------------------------------------------------ |
| Ερώτημα            | προαιρετικό | καθορίζει τα κριτήρια επιλογής για τη λειτουργία εύρεσης                                                                             |
| Προβολή            | προαιρετικό | προσδιορίζει ποια πεδία θα συμπεριληφθούν ή θα αποκλειστούν στα αποτελέσματα του ερωτήματος                                          |
| Ταξινόμηση Κατά    | προαιρετικό | προσδιορίζει τη σειρά ταξινόμησης των επιστρεφόμενων εγγράφων                                                                        |
| Όριο               | προαιρετικό | περιορίζει τον αριθμό των επιστρεφόμενων εγγράφων, χωρίς προεπιλογή κανένα όριο. Ένα όριο 0 ισοδυναμεί με κανένα όριο.               |
| Παράλειψη          | προαιρετικό | προσδιορίζει τον αριθμό των εγγράφων προς παράλειψη, από προεπιλογή στο 0.                                                           |

Για παράδειγμα, για να βρείτε το αναγνωριστικό, την τιμή και το είδος όλων των πορτοκαλιών που ταξινομούνται από την ταυτότητά τους.

Για **`Query`**, `{"type": "πορτοκαλί"}`

Για **`Projection`**, `{"_id": 1, "τύπος": 1, "τιμή": 1}`
Για **`Sort By`**, `{"_id":1}`

Σημείωση: αντικαταστήστε το “1” με το “true” εξακολουθεί να το κάνει!

## <h2 hidden>Ανάπτυξη εισαγωγής</h2>

### Σύννεφο

Δοκιμάστε την ανάπτυξη με ένα κλικ στο ILLA Cloud για την ευκολότερη εμπειρία.

### Αυτοεξυπηρετητής

Μπορείτε να αυτο-φιλοξενήσει τοπικά με ILLA CLI, Docker compose, k8s πηδάλιο κλπ. Έχουμε βελτιστοποιήσει επανειλημμένα τη διαδικασία αυτο-φιλοξενίας για να σας δώσει τη δυνατότητα να κάνετε αυτο-φιλοξενία και ανάπτυξη λογισμικού μέσα σε λίγα λεπτά. Σε σύγκριση με το σύννεφο, μπορείτε να χρησιμοποιήσετε τα προϊόντα ILLA πιο ασφαλή και αξιόπιστα, and you can use ILLA Builder by simply visiting the deployment site after hosting and deploying Builder locally.

Για περισσότερες πληροφορίες σχετικά με την αυτο-φιλοξενία, ανατρέξτε στο ILA CLI, Docker compose, k8s helm boot pages.
Αν έχετε προβλήματα με την αυτο-ανάπτυξη σας, παρακαλούμε επικοινωνήστε μαζί μας στο GitHub ή business@illasoft.com.

#### Ανάπτυξη με ILLA CLI

Παρακαλούμε ολοκληρώστε: [ILLA CLI](https://www.illacloud.com/docs/illa-cli)

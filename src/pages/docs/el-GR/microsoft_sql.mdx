## <h2 hidden>Τι είναι το Microsoft SQL?</h2>

Η Microsoft SQL στο Illa είναι μια ενσωμάτωση βάσης δεδομένων που επιτρέπει στους χρήστες να συνδεθούν και να αλληλεπιδράσουν με μια βάση δεδομένων Microsoft SQL (MS SQL). Το MS SQL είναι ένα δημοφιλές συσχετιακό σύστημα διαχείρισης βάσεων δεδομένων ανοικτού κώδικα που είναι γνωστό για την ταχύτητα, την αξιοπιστία και την ευκολία χρήσης.

Με την ενσωμάτωση MS SQL στο Illa, μπορείτε να ρωτήσετε, να εισαγάγετε, να ενημερώσετε και να διαγράψετε δεδομένα από μια βάση δεδομένων MS SQL χρησιμοποιώντας εντολές SQL. Μπορείτε επίσης να δημιουργήσετε προσαρμοσμένα ερωτήματα SQL και να τα εκτελέσετε μέσα Illa. Αυτό καθιστά εύκολο να οικοδομήσουμε δεδομένα-οδηγείται εφαρμογές και πίνακες ελέγχου που εμφανίζουν δεδομένα σε πραγματικό χρόνο από μια βάση δεδομένων MS SQL.

### Δημιουργία MS SQL

Υπάρχουν δύο τρόποι για να δημιουργήσετε έναν πόρο στην Illa μετά την είσοδό σας στο λογαριασμό Illa.

-   Δημιουργία σε πόρους

Συνδεθείτε στον Illa λογαριασμό σας, επιλέξτε `**Πόροι**` στην κορυφή της σελίδας και κάντε κλικ στο κουμπί `**Δημιουργία Νέου**`.

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

Επιλέξτε **`Microsoft SQL`** από τη λίστα της βάσης δεδομένων.

![action_list](https://cdn.illacloud.com/official-website/img/docs/action_list.png)

Συνδεθείτε στη βάση δεδομένων με τις απαιτούμενες παραμέτρους που περιγράφονται στις παρακάτω ρυθμίσεις σύνδεσης.

Κάντε κλικ στο κουμπί `**Δοκιμή Σύνδεσης**` για να δείτε αν μπορούμε να συνδεθείτε με επιτυχία στη βάση δεδομένων. Εάν ναι, κάντε κλικ στο **`Αποθήκευση πόρων`**, αλλιώς, ελέγξτε ξανά το όνομα κεντρικού υπολογιστή, τη θύρα, το όνομα χρήστη και τον κωδικό πρόσβασης είναι σωστά.

Μετά τη δημιουργία ενός πόρου, το έτοιμο MS SQL θα εμφανιστεί όπως εμφανίζεται.

![ms_ex_ready](https://cdn.illacloud.com/official-website/img/docs/ms_ex_ready.png)

-   Δημιουργία στο Builder

Συνδεθείτε στον Illa λογαριασμό σας, δημιουργήστε ένα έργο στο Illa Builder στη σελίδα `**App**`, και πλοηγηθείτε στο **`Λίστα Ενεργειών`** στο κάτω μέρος της σελίδας. Κάντε κλικ στο κουμπί `**new**`, στη συνέχεια, επιλέξτε **`Microsoft SQL`** από τη λίστα της βάσης δεδομένων. Στη συνέχεια, συνδεθείτε στη βάση δεδομένων με τις απαιτούμενες παραμέτρους που περιγράφονται στις "Ρυθμίσεις Σύνδεσης" παρακάτω.

Κάντε κλικ στο κουμπί `**Δοκιμή Σύνδεσης**` για να δείτε αν μπορούμε να συνδεθείτε με επιτυχία στη βάση δεδομένων. Εάν ναι, κάντε κλικ στο **`Αποθήκευση πόρων`**, αλλιώς, ελέγξτε ξανά το όνομα κεντρικού υπολογιστή, τη θύρα, το όνομα χρήστη και τον κωδικό πρόσβασης είναι σωστά.

### Ρυθμίσεις Σύνδεσης

Εδώ πρέπει να παρέχουμε πληροφορίες για τη σύνδεση με MS SQL βάση δεδομένων.

![ms_config](https://cdn.illacloud.com/official-website/img/docs/ms_config.png)

| Ιδιότητες               | Απαιτείται  | Περιγραφή                                                                                                                                            |
| ----------------------- | ----------- | ---------------------------------------------------------------------------------------------------------------------------------------------------- |
| Όνομα                   | απαιτείται  | Το όνομα του πόρου κατά τη δημιουργία ενεργειών στο ILLA.                                                                                            |
| Hostname                | απαιτείται  | Η διεύθυνση URL ή IP για τη βάση δεδομένων σας                                                                                                       |
| Θύρα                    | απαιτείται  | Ο αριθμός θύρας του διακομιστή που θα πρέπει να χρησιμοποιήσετε για να συνδεθείτε. Αν δεν καθορίσετε μια θύρα, η προεπιλεγμένη θύρα είναι το '1433'. |
| Βάση Δεδομένων          | απαιτείται  | Το όνομα της βάσης δεδομένων                                                                                                                         |
| Όνομα Χρήστη            | απαιτείται  | το όνομα χρήστη που θέλετε να χρησιμοποιήσετε κατά τη σύνδεση στον MS SQL server.                                                                    |
| Κωδικός                 | απαιτείται  | Χρησιμοποιήστε αυτόν τον κωδικό πρόσβασης για ταυτοποίηση.                                                                                           |
| Επιλογή σύνδεσης        | προαιρετικό | ζεύγος τιμής κλειδιού για να καθορίσετε την επιλογή σύνδεσης                                                                                         |
| Επιλογές SSL            | προαιρετικό | αποφασίζει πόσο υψηλή προτεραιότητα έχει μια ασφαλής σύνδεση SSL TCP/IP κατά τη διαπραγμάτευση με το διακομιστή.                                     |

### Δημιουργία Ενεργειών

Έχουμε δημιουργήσει ένα MS SQL πόρο. Μπορούμε να προσθέσουμε την ενέργεια επιλέγοντας MS SQL από τη λίστα ενεργειών και επιλέγοντας το κουμπί **`Δημιουργία ενέργειας`**.

![ms_resource_list](https://cdn.illacloud.com/official-website/img/docs/ms_resource_list.png)

Τώρα έχουμε προσθέσει το MS SQL server ως ενέργεια στη σελίδα δημιουργίας μας.

![ms](https://cdn.illacloud.com/official-website/img/docs/ms.png)

### Ρύθμιση Του MS SQL

| Ιδιότητες       | Περιγραφή                                                                                                                                                                                                                                                                                                                        |
| --------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Τύπος Ρυθμίσεων | τον τύπο ρύθμισης παραμέτρων που χρησιμοποιείτε για να συνδεθείτε στη βάση δεδομένων του MS SQL Server: SQL και μαζικό ένθετο. Για τη λειτουργία SQL, μπορείτε να χρησιμοποιήσετε SQL ερώτημα. Για τη λειτουργία Μαζικής εισαγωγής, μπορείτε να εισαγάγετε δεδομένα σε έναν επιλεγμένο πίνακα με τη μορφή {{{{{x:a, y:b}, … ] }} |
| Ερώτημα SQL     | εντολές sql για λήψη (επιλογή), εισαγωγή, ενημέρωση, διαγραφή δεδομένων από τη βάση δεδομένων.                                                                                                                                                                                                                                   |
| Μετασχηματιστής | μετατρέποντας τα δεδομένα σε στυλ που σας αρέσει χρησιμοποιώντας JavaScript                                                                                                                                                                                                                                                      |

Χρήση παραδείγματος:

-   Επιλογή

```sql
SELECT * FROM users ORDER BY lastname;
```

-   Insert

```sql
INSERT INTO users

	(firstname, lastname, email)

VALUES

	(

		{{ NewUser.data.firstNameInput }},

		{{ NewUser.data.lastNameInput }},

		{{ NewUser.data.emailInput }}

	);
```

-   Ενημέρωση

```sql
UPDATE users
  SET firstname = '{{form1.updatedProfile.first}}'
  WHERE id = {{ form1.updatedProfile.uid}};
```

-   Διαγραφή

```sql
DELETE FROM users WHERE id = {{ form1.updatedProfile.uid }};
```

## <h2 hidden>Συνδεθείτε με τις πηγές δεδομένων σας (Δράση)</h2>

Μάθετε για τη σύνδεση με την πηγή των δεδομένων σας και καλέστε την στον Δημιουργό ILLA.

Το ILLA σας επιτρέπει να συνδέσετε και να ενσωματώσετε ένα ευρύ φάσμα πηγών δεδομένων, συμπεριλαμβανομένων βάσεων δεδομένων και API, στον οργανισμό σας για χρήση ως πόρους.

### Πηγές δεδομένων

Οι πηγές δεδομένων είναι τοποθεσίες όπου αποθηκεύονται και υφίστανται επεξεργασία τα δεδομένα. Παραδείγματα πηγών δεδομένων περιλαμβάνουν βάσεις δεδομένων PostgreSQL, Twilio REST API, και κάδους Amazon S3. Στο ILLA, μπορείτε να προσθέσετε πολλαπλές πηγές δεδομένων και να τις καταστήσετε προσβάσιμες στους χρήστες για ερωτήματα.

#### Ενσωματώσεις

ILLA παρέχει ενσωματωμένες ενσωματώσεις για πολλές δημοφιλείς πηγές δεδομένων που απαιτούν ελάχιστη ρύθμιση, όπως βάσεις δεδομένων SQL και Stripe. Αυτές οι ενσωματώσεις, γνωστές και ως πόροι, σας επιτρέπουν να συνδεθείτε και να ρωτήσετε τις πηγές δεδομένων σας χρησιμοποιώντας ILLA. Εάν ένας συγκεκριμένος τύπος πηγής δεδομένων δεν υποστηρίζεται από ενσωματωμένη ενσωμάτωση, μπορείτε να χρησιμοποιήσετε τις ενσωματώσεις REST, GraphQL, ή SOAP για να συνδεθείτε σχεδόν σε οποιοδήποτε API. Αυτό σας επιτρέπει να αποκτήσετε πρόσβαση σε ένα ευρύ φάσμα πηγών δεδομένων από μέσα ILLA-builder.

#### Σύνδεση σε βάσεις δεδομένων

Σύνδεση στις περισσότερες βάσεις δεδομένων και στα καταστήματα δεδομένων

#### Σύνδεση σε APIs

Συνδεθείτε στο REST και GraphQL.

### Πόροι

Ένας πόρος στο ILLA είναι μια αποθηκευμένη διαμόρφωση που καθορίζει τον τρόπο σύνδεσης της ILLA σε μια πηγή δεδομένων. Για να χρησιμοποιήσετε μια πηγή δεδομένων στο ILLA, πρέπει να δημιουργήσετε έναν πόρο για αυτό. Μπορείτε να επιλέξετε τον κατάλληλο πόρο για να αποκτήσετε πρόσβαση στα δεδομένα κατά την εγγραφή ενός ερωτήματος.

Κάθε πόρος στο ILLA έχει τις δικές του επιλογές ρύθμισης παραμέτρων που καθορίζουν πώς η ILLA συνδέεται με την αντίστοιχη πηγή δεδομένων. Αυτές οι επιλογές μπορούν να περιλαμβάνουν διαπιστευτήρια ελέγχου ταυτότητας και ροή, ρυθμίσεις σύνδεσης βάσης δεδομένων και ρυθμίσεις API, όπως βασική διεύθυνση URL και προεπιλεγμένες κεφαλίδες.

#### Πώς λειτουργούν οι πόροι

Οι πόροι ILLA αποθηκεύουν τις απαραίτητες ρυθμίσεις για την ασφαλή σύνδεση με την πηγή δεδομένων σας. Αυτό σας επιτρέπει να αναζητάτε δεδομένα εύκολα χωρίς να εκθέτετε την πηγή δεδομένων απευθείας στους χρήστες σας. Όταν εκτελείται ένα ερώτημα για έναν πόρο, Το ILLA χρησιμοποιεί τις ρυθμίσεις διαμόρφωσης του πόρου για να διαμεσολαβήσει το αίτημα στην πηγή δεδομένων στην πλευρά του διακομιστή. Αυτό διασφαλίζει ότι μόνο η ILLA έχει άμεση πρόσβαση στην πηγή δεδομένων.

Το ILLA απλοποιεί τη διαδικασία σύνδεσης με μια πηγή δεδομένων για τους χρήστες σας. Πρέπει μόνο να επιλέξετε τον σχετικό πόρο όταν γράφετε ένα ερώτημα χωρίς να ανησυχείτε για τις λεπτομέρειες της πηγής δεδομένων. Μπορείτε επίσης να προσθέσετε πολλαπλούς πόρους του ίδιου τύπου, όπως πολλαπλούς κάδους του Amazon S3, για να διακρίνετε μεταξύ τους.

Αυτή η προσέγγιση κρατά τις πηγές δεδομένων σας κρυμμένες από τους χρήστες σας και εξασφαλίζει ότι όλες οι αλληλεπιδράσεις με την πηγή δεδομένων αντιμετωπίζονται με ασφάλεια. Η ILLA αποφεύγει να κάνει συνδέσεις από την πλευρά του πελάτη στην πηγή δεδομένων που θα μπορούσαν να εκθέσουν ευαίσθητες πληροφορίες και να χρησιμοποιούν τις ίδιες συνθήκες για όλες τις αλληλεπιδράσεις. Επιπλέον, οι πόροι επιτρέπουν την προσωρινή αποθήκευση ερωτημάτων, η οποία μπορεί να μειώσει τον αριθμό των αιτήσεων που υποβάλλονται σε μια πηγή δεδομένων και δυνητικά χαμηλότερο κόστος.

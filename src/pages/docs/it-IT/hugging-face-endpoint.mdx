## <h2 hidden>Hugging Face Endpoint</h2>

Con Hugging Face Inference Endpoint, puoi facilmente distribuire Transformers, Diffusers o qualsiasi modello su un'infrastruttura dedicata e completamente gestita. [Clicca qui](https://ui.endpoints.huggingface.co/new) per creare un endpoint. 

### Crea una risorsa endpoint Hugging Face

Ci sono due modi per aggiungere una risorsa Hugging Face Endpoint. 

1.  Inserisci il `ILLA Builder` >> Clicca la scheda `Resources` >> Clicca `Crea Nuovo` >> Scegli `Hugging Face Endpoint` >> Configura le informazioni di connessione e clicca `Save Resource`
2.  Inserisci la pagina di modifica >> clicca `+ New` nella lista di azioni >> Scegli `Hugging Face Endpoint` >> Configura le informazioni di connessione o clicca `+ New Resource` per aggiungere nuove informazioni di connessione

### Caso di utilizzo

Abbiamo distribuito `openai/whisper-base` che ottiene un ingresso di file audio e converte in testo. √à adatto per i minuti di riunione, podcast a testo, ecc. Successivamente introdurremo come utilizzare questo modello per costruire un'applicazione in ILLA Cloud.

#### Passo 1: Costruisci l'interfaccia front-end con i componenti

Abbiamo costruito un'interfaccia utilizzando i componenti come caricamento di file e pulsante, come segue.

<img src='https://cdn.illacloud.com/official-website/img/docs/connect/hf1.png' alt="" />

#### Passo 2: Aggiungi una risorsa Faccia Abbronzante

Compila i campi mostrati qui sotto per completare la configurazione delle risorse. [Crea un Endpoint](https://ui.endpoints.huggingface.co/new) e ottieni l'URL di Endpoint. E ottieni il token dell'organizzazione API in [impostazioni profilo](https://huggingface.co/settings/tokens). 

<img src='https://cdn.illacloud.com/official-website/img/docs/connect/hf2.png' alt="" />

#### Passo 3: Configura un'azione

Selezionare prima un tipo di parametro. Prendi come esempio `openai/whisper-base`, questo modello richiede l'ingresso di file binari in modo da cambiare il parametro in `Binary`.

<img src='https://cdn.illacloud.com/official-website/img/docs/connect/hf3.png' alt="" />

Imposta il binario da passare al modello come i dati del file caricati dal componente di caricamento del file. Per esempio, `{{upload1.value[0]}}`

<img src='https://cdn.illacloud.com/official-website/img/docs/connect/hf4.png' alt="" />

#### Passo 4: Collegare i componenti e le azioni

Aggiungi un gestore di eventi al pulsante per attivare l'azione quando viene fatto clic sul pulsante. E imposta il valore del componente di testo su {{whisper.data[0].text}} per visualizzare il risultato di conversione sul componente di testo. 

<img src='https://cdn.illacloud.com/official-website/img/docs/connect/hf5.png' alt="" />

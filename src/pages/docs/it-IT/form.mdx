#### Passo 1 Aggiungi Componenti

Aggiungi un componente **`form`** con il testo predefinito "Form" e un pulsante "Invia" per la superficie.

#### Passo 2 Configura il componente

Per i componenti **`input`** e **`number input`** possiamo configurarli per impostare il valore nel componente **`bar progress`** per riflettere l'avanzamento del completamento.

1.  Clicca sul componente **`input`** e **`number input`** per aprire la pagina di ispezione. Sotto **`Gestore evento`**, clicca `**+ Nuovo**`.
2.  Nel gestore degli eventi di Modifica, seleziona **`Componente di controllo`** in azione, e seleziona il componente **`bar progress`** che vuoi aggiornare come obiettivo dell'evento.
3.  Scegli l'azione **`setValue`** e imposta il valore come

```jsx
{
  {
    input1.value && numberInput1.value
      ? "100"
      : input1.value || numberInput1.value
      ? "50"
      : "0";
  }
}
```

Nota: √® necessario impostare il gestore degli eventi come descritto sopra ai componenti di input BOTH e di input numerico.

2.  Nel gestore dell'evento Modifica, seleziona **`Mostra notifica`** in azione, **`title`** "Error!" e **`description`** "Compila il modulo prima di inviare". Scegli `Error` per **`type`**, inserisci `{{2000}}` per **`duration`** (2 secondi), e inserisci `{{barProgress1.value<100}}` per **`Solo quando avvia`.**

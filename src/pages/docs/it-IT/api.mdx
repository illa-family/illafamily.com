Connettiti a un'API
Connetti a qualsiasi API REST, GraphQL.
È possibile collegare ILLA alla maggior parte delle API utilizzando le nostre integrazioni native. Se hai bisogno di connetterti a un'API interna, o se non c'è un'integrazione supportata, è possibile connettersi utilizzando la nostra integrazione API REST o integrazione di GraphQL.

1.  Crea una risorsa API 
    Le risorse rappresentano connessioni alle fonti di dati e sono ciò che chiedi all'interno delle tue app ILLA. Accedi all'ILLA e vai alla scheda Risorse > fai clic su Crea nuovo e seleziona l'API a cui vuoi connetterti, come le API REST. Questo apre il Configura un modal di risorsa dove aggiungi le informazioni di connessione.

Come minimo, è necessario l'URL di base dell'API a cui si desidera connettersi. Puoi anche passare parametri URL, intestazioni, ecc. Se la tua API richiede l'autenticazione, puoi configurarla anche su questa pagina. Per esempio, OAuth2.0 richiede un URL di autorizzazione, URL di accesso, ecc. Dopo aver inserito i dettagli di connessione, fare clic su Salva risorsa. 
2\. Interroga la tua API
Nell'editor di App, fare clic su + Nuovo in Elenco azioni > selezionare REST API e scegliere un REST API esistente o crearne uno nuovo > fare clic creare azione e aggiornare le azioni con ulteriori parametri URL, intestazioni, ecc.

### Configura Upstash

| ProprietÃ             | Descrizione                                                                           |
| -------------------- | ------------------------------------------------------------------------------------- |
| Interrogazione Redis | utilizzare la query per recuperare e manipolare i dati memorizzati nel database Redis |
| Trasformatore        | trasformare i dati nello stile che ti piace usando JavaScript                         |

Esempio di utilizzo:

1.  Ottieni Valori:
    -   Ottieni un valore: usa il comando **`GET`** per recuperare il valore associato a una chiave specifica. Per esempio: **`GET key`**.
2.  Imposta Valori:
    -   Imposta un valore: usa il comando **`SET`** per impostare il valore di una chiave. Per esempio **`SET key value`**.
3.  Operazioni Di Hash:
    -   Imposta coppie di valori di campo in un hash: Usa il comando **`HSET`** per impostare una coppia di valori di campo in un hash. Per esempio: **`HSET hashKey valore del campo`**.
    -   Ottieni un valore da un hash: usa il comando **`HGET`** per recuperare il valore di un campo specifico in un hash. Per esempio: **`HGET hashKey field`**.
4.  Elenco Operazioni:
    -   Aggiungi un elemento a una lista: usa il comando **`LPUSH`** o **`RPUSH`** per aggiungere un elemento all'inizio o alla fine di una lista, rispettivamente. Per esempio: **`LPUSH listKey value`**.
    -   Recupera elementi da una lista: usa il comando **`LRANGE`** per recuperare un intervallo di elementi da una lista. Per esempio: **`LRANGE listKey start end`**.
5.  Imposta Operazioni:
    -   Aggiungi elementi a un set: usa il comando **`SADD`** per aggiungere uno o piÃ¹ elementi a un set. Per esempio: **`SADD setKey value1 value2`**.
    -   Recupera elementi da un set: usa il comando **`SMEMBERS`** per ottenere tutti i membri di un set. Per esempio: **`SMEMBERS setKey`**.
6.  Operazioni Set Ordinate:
    -   Aggiungi elementi a un set ordinato: usa il comando **`ZADD`** per aggiungere elementi a un set ordinato insieme ai loro punteggi. Per esempio: **`ZADD sortedSetKey score1 member1 score2 member2`**.
    -   Recupera gli elementi da un set ordinato: Usa il comando **`ZRANGE`** per recuperare un intervallo di elementi da un set ordinato in base ai loro punteggi. Per esempio: **`ZRANGE sortedSetKey start end`**.

Nota: Ãˆ possibile fare riferimento alla documentazione Redis per un elenco completo di comandi e il loro utilizzo.

### 大文字小文字を区別する

次に、データソースからのデータをボタンでテキストエリアにマッピングする方法を紹介します。 

#### ステップ 1 アクションを追加

`Project` と呼ばれるテーブルを作成します。  Projectには、`id` と `name` の2つの属性があります。 `name` はテキストエリアの入力にマッピングしたい値です。

そして、アクションリストからSupabaseの新しいアクションを作成し、プロジェクト内のすべてのデータと名前を`supabasedb2.`としてリストします。

**`Save`** と **`Run`**をクリックしてアクションを有効にします。

![text_area_data](https://cdn.illacloud.com/official-website/img/official-site/components/text_area_data.jpg)

#### ステップ2 コンポーネントを追加

次に、**`text area input`** コンポーネントと **`button`** コンポーネントをキャンバスに追加します。 

テキスト領域のデフォルト値を「なし」に設定し、下記のようにボタンに「値を設定」とラベル付けしました。

![text_area_look](https://cdn.illacloud.com/official-website/img/official-site/components/text_area_look.jpg)

#### ステップ 3 コンポーネントの設定

**`button`** コンポーネントの場合 **`text area input`** コンポーネントの値を supabasedb2 から読み取ったドキュメントのデータに設定することができます。

1.  Edit event handlerで**`Control component`** を実行中に選択します。 をクリックし、更新する **`textarea1`** コンポーネントをイベントのターゲットとして選択します。
2.  **`setValue`**アクションを選択し、値を`{{s31.data}}`に設定します。 

![text_area_config](https://cdn.illacloud.com/official-website/img/official-site/components/text_area_config.jpg)

#### ステップ4 テスト

「値を設定」ボタンをクリックすると、値が名前に変更されます。

![text_area_test](https://cdn.illacloud.com/official-website/img/official-site/components/text_area_test.jpg)

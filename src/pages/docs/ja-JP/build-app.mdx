UIコンポーネントの豊富なライブラリを使用することで、高度に洗練されたインターフェースを迅速かつ簡単に作成できます。 データとやり取りするためのクエリを記述し、アプリケーションのさまざまな部分でJavaScriptを使用することができます。

コンポーネントは ILLA の必須要素であり、プロパティを受け入れて内部の state を持つことができるという点で React コンポーネントと同様に機能します。 ILLAのコンポーネントライブラリとデータを組み合わせることで、インタラクティブなアプリを構築できます。
いくつかの一般的なコンポーネントが含まれます:
テーブルコンポーネントは、表形式でデータを表示するために使用されます。 並べ替え、フィルタリング、ページネーション、CSV エクスポートなどの機能を提供します。 
フォームコンポーネントは、入力フィールドをグループ化し、単一の単位として送信するために使用されます。 
Select 入力コンポーネントはドロップダウンと入力フィールドの組み合わせで、ユーザーはリストから値を選択するか、テキストとして直接値を入力することができます。 
チャートコンポーネントは、グラフとグラフを使用してデータの可視化を作成するために使用されます。

### プロパティー

| プロパティー                    | 説明                                                                                            |
| ------------------------- | --------------------------------------------------------------------------------------------- |
| イベントハンドラ：                 | クリック、キープレス、値の変更など、特定のユーザーアクションやイベントを検出し、応答します                                                 |
| マスクをクリックしてモーダルを閉じる        | 背景オーバーレイをクリックするかモーダルのマスクをクリックするとモーダルウィンドウが閉じます                                                |
| ヘッダーを表示                   | コンポーネントがヘッダーまたはタイトルセクションを表示するかどうか                                                             |
| フッターを表示                   | コンポーネントがフッターセクションを表示するかどうか                                                                    |
| 枠線の色                      | モーダル成分を囲む境界線の色                                                                                |
| 境界半径                      | モーダルコンポーネントの境界線の角の曲率は                                                                         |
| Border Width              | モーダル成分を囲む境界線の太さまたは幅                                                                           |
| 背景                        | は、モーダルコンポーネントのコンテンツエリアを埋める色または画像を定義します。                                                       |

<Heading hidden>SQL生成</Heading>

PostgreSQL、MySQL、Oracle DB、MariaDB、TIDB、SupabaseDB、ClickHouseなどのリソースでSQL生成をサポートしています。 これらのリソースを使用すると、自然言語を入力し、実行可能なSQL文を取得できます。 

**アプリ編集ページを入力し、上記のリソースのいずれかを使用してアクションを作成します。** 

<img src='https://cdn.illacloud.com/official-website/img/docs/assemble/sql_generate.png' alt="" />

**生成したい文の種類を選択してください**

<img src='https://cdn.illacloud.com/official-website/img/docs/assemble/sql_generate(1).gif' alt="" />

**詳細を入力し、SQL生成をクリック**

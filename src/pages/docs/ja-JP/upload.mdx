#### S3リソースを介して単一のファイルをアップロード

アクションの構成は以下のとおりです。 

オブジェクト名をアップロード

```jsx
{{upload1.files.map(file =>file.name)[0]}}
```

データをアップロード

```jsx
{{upload1.value[0]}}
```


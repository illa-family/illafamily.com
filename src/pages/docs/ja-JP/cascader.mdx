## <h2 hidden>カスケードとは?</h2>

カスケードは、ドロップダウンメニューから階層データ(カテゴリやサブカテゴリなど)を選択できる入力コンポーネントの一種です。 ユーザーがドロップダウンメニューからオプションを選択した場合。 キャスカダーは自動的に更新され、ネストされたドロップダウンメニューの関連するサブカテゴリやオプションが表示されます。 ユーザーがドロップダウンメニューからオプションを選択した場合。 キャスカダーは自動的に更新され、ネストされたドロップダウンメニューの関連するサブカテゴリやオプションが表示されます。

例えば、商品カテゴリを選択するためのカスケード入力コンポーネントがある場合。 最初のドロップダウンメニューには、「Electronics」、「衣料品」、「本」など、利用可能なすべてのカテゴリが表示されます。 ユーザが「Electronics」を選択すると、カスケードは「電話」、「ラップトップ」、「タブレット」などの電子のサブカテゴリを表示するように更新されます。 これにより、ユーザーは階層データをすばやく簡単にナビゲートし、必要なオプションを選択できます。

#### ステップ 1 アクションを追加

**`project` と呼ばれるサブテーブルと **`therapist`** と呼ばれるサブテーブルを **`project_id` で整列させて作成しましょう。

```sql
SELECT [project.id](http://project.id/) as project_id, [therapist.name](http://therapist.name/) as therapist_name from therapist join project on thaperist.project_id = [project.id](http://project.id/)
```

#### ステップ 2 データを変換する

次に、Select Section の下にある **`transformer`** セクションを **`enable`** することができます。 次に、Select Section の下にある **`transformer`** セクションを **`enable`** することができます。 テキスト領域内では、必要な値とラベルだけを取得し、JavaScript を使用してカスケード対応のオプションに変換することができます。 次に、Select Section の下にある **`transformer`** セクションを **`enable`** することができます。 テキスト領域内では、必要な値とラベルだけを取得し、JavaScript を使用してカスケード対応のオプションに変換することができます。

```jsx
const transformedArray = data.reduce((acc, current) => {
  const existing = acc.find((item) => item.value === current.project_id);
  if (existing) {
    const therapist = { value: current.therapist_id, label: current.therapist_name };
    existing.children.push(therapist);
  } else {
    const project = { value: current.project_id, label: current.project_name, children: [] };
    const therapist = { value: current.therapist_id, label: current.therapist_name };
    project.children.push(therapist);
    acc.push(project);
  }
  return acc;
}, []);

return transformedArray;
```

## <Heading hidden>API に接続</Heading>

任意のREST、GraphQL APIに接続します。

ILLAをほとんどのAPIに接続するには、ネイティブインテグレーションを使用します。 内部 API に接続する必要がある場合、またはサポートされていない場合。 REST API インテグレーションまたは GraphQL インテグレーションを使用して接続できます。

### 1.  API リソースを作成

リソースはデータソースへの接続を表し、ILLAアプリケーション内で問い合わせるものです。ILLAにログインし、**リソース**タブ>**新規作成**をクリックし、接続したいAPIを選択してください。 例えば**REST API** **Configure a modal to add your connection information**.

<img
  src='https://cdn.illacloud.com/official-website/img/docs/connect/connect-api/config.png'
  alt=""
/>

最低でも接続したいAPIのベースURLが必要です。 URLパラメータ、ヘッダなどを渡すこともできます。 API が認証を必要とする場合は、このページでも設定できます。 例えば、OAuth2.0では認証URL、アクセストークンURLなどが必要です。 接続の詳細を入力したら、**保存**をクリックしてください。

### 2.  API のクエリ

アプリエディタで **Action List** > select **REST API** and choose an existing REST API or create a new one > click **create action** and update the actions with additional URL parameters. ヘッダなどです

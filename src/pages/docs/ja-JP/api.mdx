API に接続
任意の REST, GraphQL API に接続します。
ILLAをほとんどのAPIに接続するには、ネイティブインテグレーションを使用します。 内部 API に接続する必要がある場合、またはサポートされていない場合。 REST API インテグレーションまたは GraphQL インテグレーションを使用して接続できます。

1.  APIリソースの作成 
    リソースはデータソースへの接続を表し、ILLAアプリケーション内で問い合わせるものです。 ILLAにログインし、format@@0タブに移動し、format@@1をクリックし、接続したいAPIを選択します。 例えばREST APIなど。 これにより、format@@0 が開き、接続情報を追加します。

最低でも接続したいAPIのベースURLが必要です。 URLパラメータ、ヘッダなどを渡すこともできます。 API が認証を必要とする場合は、このページでも設定できます。 例えば、OAuth2.0では認証URL、アクセストークンURLなどが必要です。 接続の詳細を入力したら、format@@0をクリックします。

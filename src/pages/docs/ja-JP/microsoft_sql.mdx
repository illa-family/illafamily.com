## <h2 hidden>Microsoft SQLとは?</h2>

IllaのMicrosoft SQLは、ユーザーがMicrosoft SQL(MS SQL)データベースに接続して対話することができるデータベース統合です。 MS SQLは、速度、信頼性、使いやすさで知られている一般的なオープンソースのリレーショナルデータベース管理システムです。

SQL コマンドを使用して、MS SQL データベースからデータをクエリ、挿入、更新、および削除することができます。 また、カスタムSQLクエリを作成し、Ila内で実行することもできます。 これにより、MS SQLデータベースからリアルタイムデータを表示するデータ駆動型アプリケーションやダッシュボードを簡単に構築できます。

### MS SQL を作成

Illaアカウントにログインした後、Illaにリソースを作成するには2つの方法があります。

-   リソースに作成

Illaアカウントにサインインし、ページ上部の「**リソース**」を選択し、「**新規作成**」ボタンをクリックします。

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

データベース リストから **`Microsoft SQL`** を選択します。

![action_list](https://cdn.illacloud.com/official-website/img/docs/action_list.png)

下の「接続設定」で説明されている必要なパラメータを使用してデータベースに接続します。

データベースに正常に接続できるかどうかを確認するには、`**接続をテスト**` をクリックします。 はいの場合は**`リソースを保存`**をクリックします。それ以外の場合は、ホスト名、ポート、ユーザー名、パスワードが正しいことを確認してください。 はいの場合は**`リソースを保存`**をクリックします。それ以外の場合は、ホスト名、ポート、ユーザー名、パスワードが正しいことを確認してください。

リソースを作成すると、準備完了の MS SQL が表示されます。

![ms_ex_ready](https://cdn.illacloud.com/official-website/img/docs/ms_ex_ready.png)

使用例:

-   選択

```sql
SELECT * FROM users ORDER BY lastname;
```

-   Insert

```sql
INSERT INTO users

	(firstname, lastname, email)

VALUES

	(

		{{ NewUser.data.firstNameInput }},

		{{ NewUser.data.lastNameInput }},

		{{ NewUser.data.emailInput }}

	);
```

-   更新

```sql
UPDATE users
  SET firstname = '{{form1.updatedProfile.first}}'
  WHERE id = {{ form1.updatedProfile.uid}};
```

-   削除

```sql
DELETE FROM users WHERE id = {{ form1.updatedProfile.uid }};
```

## <h2 hidden>分割線とは?</h2>

ILLA Cloud では、分割コンポーネントはウェブアプリケーション内のコンテンツのセクションを分割または分割するのに役立つ視覚的な要素です。 これは、単純な水平線または垂直線であり、異なるセクションまたはコンポーネントの間に挿入して、視覚的な組織とユーザーインターフェイスの明快さを向上させることができます。

### プロパティー

| プロパティー     | 説明                                                                  |
| ---------- | ------------------------------------------------------------------- |
| テキスト       | 分割装置コンポーネントに追加情報を追加する                                               |
| Text align | は、分割コンポーネント内のテキストの水平方向の配置を指定します。                                    |
| Hidden     | 分割線のコンポーネントをユーザーインターフェースから非表示にする                                    |
| 文字サイズ      | 区切り線内のテキストのフォントサイズを調整する                                             |

### データ

コンポーネントには一般的に使用されているデータがいくつか含まれており、アプリで **`{{componentName.propertyName}}`** を介して呼び出すことができます。

| プロパティ名        | 説明                                                               |
| ------------- | ---------------------------------------------------------------- |
| colorScheme   | コンポーネント内のテキストの色を定義します。                                           |
| displayName   | は、コンポーネントに割り当てられた名前またはラベルを表します。                                  |
| fs            | 区切り線に関連付けられているファイル システム (fs)                                     |
| hidden        | コンポーネントが表示されているか非表示かを決定する boolean 値                              |
| テキスト          | テキストの値                                                           |

例: {{divider1.text}}

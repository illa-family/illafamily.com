## <h2 hidden>フォームとは?</h2>

フォームコンポーネントは、タスクまたはプロセスの進捗または完了を表す視覚的な要素です。 指定した進捗値に基づいて、水平バーがいっぱいになったり縮小したりすることができます。

### プロパティー

| プロパティー                        | 説明                                                                                                                                    |
| ----------------------------- | ------------------------------------------------------------------------------------------------------------------------------------- |
| イベントハンドラ：                     | クリック、キープレス、フォームの送信など、特定のユーザーアクションやイベントを検出し、応答します。                                                                                     |
| 無効                            | 非対話的で、ユーザーによって変更またはトリガーすることはできません。                                                                                                    |
| 送信を無効化                        | フォーム送信機能を無効にします。                                                                                                                      |
| 送信時に入力を検証する                   | フォームが送信されるときフォーム入力は正確性または完全性を検証する必要があります。                                                                                             |
| 送信成功後にリセット                    | 提出が成功した後、自動的に値と状態を初期値またはデフォルト値にリセットします。                                                                                               |
| ツールチップ                        | ユーザーが特定の要素(フォームコンポーネントなど)にホバリングまたは相互作用したときに表示される小さなポップアップまたは情報メッセージ。                                                                  |
| ヘッダーを表示                       | フォームコンポーネントがヘッダーまたはタイトルセクションを表示するかどうか                                                                                                 |
| シャワーフッター                      | フォームコンポーネントにフッターセクションを表示するかどうか                                                                                                        |
| Hidden                        | フォームコンポーネントをユーザのビューから非表示にします。                                                                                                         |
| 枠線の色                          | フォームコンポーネントを囲む枠線の色                                                                                                                    |
| 境界半径                          | フォームコンポーネントの境界線の角の曲線                                                                                                                  |
| Border Width                  | フォームコンポーネントを囲む境界線の太さまたは幅                                                                                                              |
| 背景                            | は、フォームコンポーネントのコンテンツエリアを埋める色または画像を定義します。                                                                                               |
| 影                             | 微妙な追加、"shadow" をフォームコンポーネントにオフセットすることで視覚的な効果が得られ、少し上げられたり、背景の上に浮かんで表示されます。                                                            |

### データ

コンポーネントには一般的に使用されているデータがいくつか含まれており、アプリで **`{{componentName.propertyName}}`** を介して呼び出すことができます。

| プロパティ名                  | 説明                                                                                                                                      |
| ----------------------- | --------------------------------------------------------------------------------------------------------------------------------------- |
| backgroundColor         | はフォームコンポーネントの背景色を指定します。                                                                                                                 |
| borderColor             | フォームコンポーネントを囲む境界線の色を定義します。                                                                                                              |
| borderWidth             | フォームコンポーネントを囲む境界の太さや幅を決定します。                                                                                                            |
| 無効                      | フォームコンポーネントが無効になっているかどうか                                                                                                                |
| displayName             | は、フォームコンポーネントに割り当てられた名前またはラベルを表します。                                                                                                     |
| footerHeight            | は、フォームコンポーネントのフッターセクションの高さまたはサイズを指定します。                                                                                                 |
| formData                | ユーザーが入力したデータまたは値をフォームコンポーネントに保持します。                                                                                                     |
| headerHeight            | フォームコンポーネントのヘッダーセクションの高さまたはサイズを指定します。                                                                                                   |
| 無効                      | フォームコンポーネントの入力または値が無効であるかどうか。                                                                                                           |
| radius                  | は、フォームコンポーネントの角の外枠半径または曲率を定義します。                                                                                                        |
| resetAfterSuccessfully  | は、フォームコンポーネントが送信に成功した後に自動的に値と状態を初期値またはデフォルト値にリセットすることを示します。                                                                             |
| 影                       | フォームコンポーネントに対するシャドウ効果の存在と外観を制御します                                                                                                       |
| シャワーフッター                | フォームコンポーネントのフッターセクションを表示するかどうかを決定します                                                                                                    |
| showHeader              | フォームコンポーネントのヘッダーセクションの表示                                                                                                                |
| tooltipText             | フォームコンポーネントに関連付けられたツールチップのテキストまたは内容を保持します                                                                                               |
| validateInputOnSubmit   | は、フォームが送信されたときにフォームコンポーネントの入力値が正しいか完全性を検証する必要があることを指定します。                                                                               |

例: {{form1.disabled}}

### 大文字小文字を区別する

名前と年齢を入力とするフォームの作成方法を紹介します。 フォームの完了進行状況を表示するには、バーの進行状況を表示します。

#### ステップ1 コンポーネントを追加

デフォルトのテキスト"Form"とキャンバスに"Submit"ボタンで**`form`**コンポーネントを追加します。

**`input`** コンポーネント、**`number input`** コンポーネント、そして `**Insert**` から **`bar progress`** コンポーネントをフォームに追加します。

入力コンポーネントに「名前」、数値入力コンポーネントに「年齢」、および「進行状況」コンポーネントにラベル付けします。

入力コンポーネントのプレースホルダを「ここにあなたの名前を入力してください」に設定し、入力された数値のデフォルト値を以下のように0に設定します。

#### ステップ 2 コンポーネントの設定

**`input`** と **`number input`** コンポーネントについては、 **`bar progress`** コンポーネント内の値を設定して、完了の進行状況を反映させるように設定します。

1.  **`input`** と **`number input`** コンポーネントをクリックして、検査ページを開きます。 **`Event handler`**の下で `*** 新しい**` をクリックします。 **`Event handler`**の下で `*** 新しい**` をクリックします。 **`Event handler`**の下で `*** 新しい**` をクリックします。 **`Event handler`**の下で `*** 新しい**` をクリックします。 **`Event handler`**の下で `*** 新しい**` をクリックします。 **`Event handler`**の下で `*** 新しい**` をクリックします。 **`Event handler`**の下で `*** 新しい**` をクリックします。 **`Event handler`**の下で `*** 新しい**` をクリックします。
2.  Edit event handlerで**`Control component`** を実行中に選択します。 をクリックし、イベントのターゲットとして更新する **`bar progress`** コンポーネントを選択します。
3.  **`setValue`**アクションを選択し、値を次のように設定します

```jsx
{
  {
    input1.value && numberInput1.value
      ? "100"
      : input1.value || numberInput1.value
      ? "50"
      : "0";
  }
}
```

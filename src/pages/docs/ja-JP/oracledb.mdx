## <h2 hidden>Oracle DBとは?</h2>

IllaのOracle DBは、Oracle DBデータベースに接続して対話できるデータベース統合です。 Oracle DBは、速度、信頼性、使いやすさで知られている一般的なオープンソースのリレーショナルデータベース管理システムです。

IraでOracle DB統合を使用すると、SQL コマンドを使用してOracle DBデータベースからデータをクエリ、挿入、更新、および削除することができます。 また、カスタムSQLクエリを作成し、Ila内で実行することもできます。 これにより、Oracle DBデータベースからリアルタイムデータを表示するデータドリブンなアプリケーションやダッシュボードを簡単に構築できます。

### Oracle DBの作成

Illaアカウントにログインした後、Illaにリソースを作成するには2つの方法があります。 

-   リソースに作成

Illaアカウントにサインインし、ページ上部の「**リソース**」を選択し、「**新規作成**」ボタンをクリックします。 

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

データベース リストから **`Oracle DB`** を選択します。

![action_list](https://cdn.illacloud.com/official-website/img/docs/action_list.png)

下の「接続設定」で説明されている必要なパラメータを使用してデータベースに接続します。 

データベースに正常に接続できるかどうかを確認するには、`**接続をテスト**` をクリックします。 はいの場合は**`リソースを保存`**をクリックします。それ以外の場合は、ホスト名、ポート、ユーザー名、パスワードが正しいことを確認してください。

リソースを作成すると、Ready Oracle DBが次のように表示されます。

![oracle_ex_ready](https://cdn.illacloud.com/official-website/img/docs/oracle_ex_ready.png)

-   ビルダーで作成

Illa アカウントにサインインし、`**App**` ページで Illa Builder でプロジェクトを作成します。 をクリックし、ページの下部にある **`Action List`** に移動します。 `**new**`をクリックし、データベースリストから**`Oracle DB`**を選択します。 次に、下の「接続設定」で説明されている必要なパラメータを使用してデータベースに接続します。

データベースに正常に接続できるかどうかを確認するには、`**接続をテスト**` をクリックします。 はいの場合は**`リソースを保存`**をクリックします。それ以外の場合は、ホスト名、ポート、ユーザー名、パスワードが正しいことを確認してください。

### 接続設定

ここでは、Oracle DBデータベースに接続するための情報を提供する必要があります。

![oracle_config](https://cdn.illacloud.com/official-website/img/docs/oracle_config.png)

| プロパティー             | 必須  | 説明                                                      |
| ------------------ | --- | ------------------------------------------------------- |
| 名前                 | 必須  | ILLAでアクションを作成する際のリソース名です。                               |
| Hostname           | 必須  | データベースの URL または IP アドレス                                 |
| ポート                | 必須  | 接続に使用するサーバーホストのポート番号。 ポートを指定しない場合、デフォルトのポートは '3306' です。 |
| データベース             | 必須  | データベースの名前                                               |
| データベース名を次のように扱います: | 必須  | SIDまたはサービス名でデータベースインスタンスを識別します。                         |
| ユーザー名              | 必須  | Oracle DBサーバーにログインするときに使用するユーザー名。                       |
| パスワード              | 必須  | 認証にこのパスワードを使用してください。                                    |
| SSL                | 必須  | サーバとのネゴシエーション中にセキュアな SSL TCP/IP 接続が優先される方法を決定します。       |

### アクションを作成

Oracle DBリソースを作成しました。アクションリストからOracle DBを選択し、**`Create action`**ボタンを選択することでアクションを追加できます。

![oracle_resource_list](https://cdn.illacloud.com/official-website/img/docs/oracle_resource_list.png)

 これで、ビルドページにOracle DBサーバーがアクションとして追加されました。

![oracle](https://cdn.illacloud.com/official-website/img/docs/oracle.png)

### Oracle DBの構成

| プロパティー  | 説明                                        |
| ------- | ----------------------------------------- |
| SQL クエリ | データベースからデータを取得(選択)、挿入、更新、削除するための sql コマンド |
| トランス    | JavaScriptを使用して好きなスタイルにデータを変換します          |

使用例:

-   選択

```sql
SELECT * FROM users ORDER BY last name;
```

-   Insert

```sql
INSERT INTO users

	(firstname, lastname, email)

VALUES

	(

		{{ NewUser.data.firstNameInput }},

		{{ NewUser.data.lastNameInput }},

		{{ NewUser.data.emailInput }}

	);
```

-   更新

```sql
UPDATE users
  SET firstname = '{{form1.updatedProfile.first}}'
  WHERE id = {{ form1.updatedProfile.uid}};
```

-   削除

```sql
DELETE FROM users WHERE id = {{ form1.updatedProfile.uid }};
```

### プロパティー

| プロパティー     | 説明                                                                               |
| ---------- | -------------------------------------------------------------------------------- |
| メニュー       | メニューコンポーネント内に表示される実際のコンテンツとオプション                                                 |
| Hidden     | フォームコンポーネントをユーザのビューから非表示にします。                                                    |
| 方向         | メニューコンポーネントの方向やレイアウトを指定します。                                                      |
| Align      | コンテナまたは親要素内のメニューコンポーネントの配置または配置                                                  |

### データ

コンポーネントには一般的に使用されているデータがいくつか含まれており、アプリで **`{{componentName.propertyName}}`** を介して呼び出すことができます。

| プロパティ名          | 説明                                                                                                   |
| --------------- | ---------------------------------------------------------------------------------------------------- |
| displayName     | は、コンポーネントに割り当てられた名前またはラベルを表します。                                                                      |
| hidden          | メニューコンポーネントが表示されているか非表示か                                                                             |
| 水平揃え            | コンテナまたは親要素内のメニューコンポーネントの水平方向の配置や配置を制御します                                                             |
| 項目              | メニューコンポーネント内で利用可能なメニューアイテムまたはオプションのコレクション                                                            |
| モード             | メニューがどのように動作し、ユーザーと相互作用するのか                                                                          |
| selectedValues  | は、ユーザーが選択または選択したメニューアイテムの値を保持します。                                                                    |

例: {{menu1.displayName}}

#### ステップ3 接続メニューとページ

ページの先頭に **`Menu`** コンポーネントを追加します。

メニューアイテムにイベントハンドラを追加する → ルーターを設定するアクションを設定する → ページと表示を選択する

**`menu`** コンポーネントについては、ページとビューを切り替えるように設定します。

1.  **`menu`** コンポーネントをクリックして検査ページを開きます。
2.  項目をクリックして構成を開きます。
3.  **`Event handler`**の下で `*** 新しい**` をクリックします。
4.  Edit event handlerで**`Set Router`** を実行中に選択します。 を選択して、イベントのターゲットとしてルーティングするページとビューを選択します。

![menu_items](https://cdn.illacloud.com/official-website/img/docs/menu_items.jpg)

他のページでも同様に行うことができ、ルーティングしたいページを表示することができます。

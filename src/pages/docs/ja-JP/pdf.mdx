### プロパティー

| プロパティー        | 説明                                                                        |
| ------------- | ------------------------------------------------------------------------- |
| PDF ソース       | 表示するPDFのソースを指定します                                                         |
| ツールバーを表示      | PDF コンポーネント内のツールバーの表示を制御します                                               |
| ツールチップ        | ツールチップまたはユーザーがPDFにカーソルを合わせたときに表示される説明テキストを追加します                           |

| Hidden        | PDFコンポーネントの可視性を決定します                                                      |

### データ

コンポーネントには一般的に使用されているデータがいくつか含まれており、アプリで **`{{componentName.propertyName}}`** を介して呼び出すことができます。

| プロパティ名        | 説明                                                                                                     |
| ------------- | ------------------------------------------------------------------------------------------------------ |
| displayName   | pdfの名前またはタイトルを表す文字列値。                                                                                  |
| 高さ            | PDFコンポーネントの垂直方向の寸法またはサイズ                                                                               |
| hidden        | pdfコンポーネントがユーザーインターフェイスから非表示になるかどうかを決定するブール値。                                                          |
| scaleMode     | は、コンポーネントの利用可能な表示領域内で PDF ドキュメントがどのようにスケーリングされるか、またはフィットするかを指定します。                                     |
| showToolBar   | ツールバーまたはコントロールパネルが PDF コンポーネントとともに表示されているかどうか                                                          |
| tooltipText   | ユーザーがPDFコンポーネントにカーソルを合わせたときに表示するツールチップまたはホバリングテキストを表す文字列の値。                                            |
| URL           | コンポーネントに表示する PDF ファイルのソース                                                                              |
| width         | PDFコンポーネントの水平方向の寸法またはサイズ                                                                               |

例: {{pdf1.displayName}}

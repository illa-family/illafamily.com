### データ

コンポーネントには一般的に使用されているデータがいくつか含まれており、アプリで **`{{componentName.propertyName}}`** を介して呼び出すことができます。

| プロパティ名         | 説明                                                                                                       |
| -------------- | -------------------------------------------------------------------------------------------------------- |
| altText        | 画像の内容や目的を説明するテキストベースの代替案は                                                                                |
| displayName    | 画像の名前またはタイトルを表す文字列の値。                                                                                    |
| イベント           | イメージコンポーネントに関連付けられたイベントハンドラを定義する配列またはオブジェクト (クリックなど)                                                     |
| hidden         | イメージコンポーネントがユーザーインターフェイスから非表示になるかどうかを決定する真偽値。                                                            |
| imageSrc       | コンポーネントに表示される画像のソースまたは場所。                                                                                |
| オブジェクトフィット     | コンポーネントの寸法内で画像がどのように適合または拡大縮小するか。                                                                        |
| radius         | 画像コンポーネントに角丸を適用します。                                                                                      |
| tooltipText    | ユーザーが画像コンポーネントにカーソルを合わせたときに表示するツールチップまたはホバリングテキストを表す文字列の値。                                               |

例: {{image1.displayName}}

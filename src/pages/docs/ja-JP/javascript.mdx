## <Heading hidden>JavaScript Guider</Heading>

JavaScript は ILLA ビルダーの最も重要なコンポーネントです。 これは、APPのデータアクセスとロジック構築に使用するツールです。 JavaScriptの呼び出し方をマスターして、ILLAビルダーの使用をマスターしました。

ILLA BuilderでAPPを構築する場合、`{{template syntax}}`を使用してデータの呼び出しや操作のためのカスタムJavaScript文を作成できます。 または、変換器に直接 JS コードを記述します。

### 操作データ

#### オブジェクト上のデータへのアクセス

ILLA Builder では、テンプレート構文を使用して任意のテキスト入力ボックスに JavaScript 式を入力できます。 データオブジェクトは参照可能で、JavaScript を介してすべての挿入、アクション、変換のデータを操作できます。 For example, you can use `{{textInput.data}}` to access the content of a text input insert, `{{action.data}}` to access the data from an action database and `{{transformer.value}}` to access the data from a transformer.

コードミラーインサートを製品に追加および変更しました。 ILLA Builderプログラミング環境に適応し、構文の強調表示と自動インデントをサポートしました。 プログラミング中に発生する変数や関数の詳細な説明を追加し、低コストでILLA Builderを使用する方法を学ぶことができます。 オブジェクト名を入力したら、ドットを入力します。 プロンプトには、オブジェクトのすべてのプロパティ、データ型、および部分的なテキスト説明が含まれます。 すぐにアクセスできるようになります

#### 配列内の数字へのアクセス

特定の配列の値にアクセスする必要がある場合 アクセスする必要があるデータの位置またはサブスクリプションで値を得ることができます。

```javascript
const list = ['a', 'b', 'c']
return list[0] // Return the first value in list 'a'
```

### データを表示

ILLA Builderには4つの場所があります。左側のパネルのデータ ワークスペースです。 JavaScript式を入力すると、テキストボックスの下にあるデータプロンプトが表示され、データ挿入を介してデータが表示されます。

#### データ ワークスペースのプレゼンテーション

アプリエディターで左側のパネルを開くと、データ ワークスペースが表示されます。 したがって、データは構造化されたJSONとして表示され、すべてのプロパティとリストを拡張でき、すべてのデータがワークスペースに視覚的に表示されます。

#### データ入力のライブプレビュー

入力ボックス内でアクションまたはトランス操作を呼び出すと、その操作のデータが入力ボックスの下に自動的に表示されます。 ここの情報に基づいて正しいデータがあることが確認できます

[//]: # "[图片]"

#### データプレゼンテーション

データ表示インサートは、データの最も直感的なビューを提供します。 コンポーネントをフレームにドラッグ&ドロップするだけで、データフィールドに該当するデータソースの名前を入力できます。 フィルタリングして整列すると、データ表示情報にアクセスできます。

### トランス

変換器に複数行の JS コードを記述することができます。 2重括弧ではなくトランスで書くことをお勧めします。 JavaScriptはトランスで直接プログラムしたり、 `{{}}` の助けを借りて操作中のデータを呼び出したりすることができます。

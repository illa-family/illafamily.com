## <h2 hidden>導入概要</h2>

### クラウド

最も簡単な体験を得るために、ILLA Cloud でワンクリックで展開してみてください。

### 自己ホスト

ILLA CLI、Docker compose、k8s ヘルムなどでローカルでセルフホストできます。 セルフホストプロセスを繰り返し最適化し、わずか数分でセルフホストとソフトウェアのデプロイを行えるようにしました。 クラウドと比較して、ILLA製品をより安全かつ確実に使用できます。 また、ILLA Builder をローカルにホスティングおよびデプロイした後に、展開サイトにアクセスするだけで使用できます。

セルフホストの詳細については、ILA CLI、Docker compose、k8s helm boot ページを参照してください。
自己展開に問題がある場合は、GitHub または business@illasoft.com までご連絡ください。

#### ILLA CLI でデプロイ

チェックアウトしてください: [ILLA CLI](https://www.illacloud.com/docs/illa-cli)

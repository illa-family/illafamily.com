## <h2 hidden>タイムラインとは？</h2>

タイムラインコンポーネントは、水平または垂直軸に沿って時系列に配置されたイベントまたはデータ ポイントのグラフィカル表示です。 ユーザーは、一連のイベントやマイルストーンを線形に視覚化し、ナビゲートすることができます。

### プロパティー

タイムラインの利用可能なプロパティ。 JavaScript はコンポーネントに関する情報を読み取りまたは変更するために書かれることがあります。

| プロパティー     | 説明                                                                           |
| ---------- | ---------------------------------------------------------------------------- |
| アイテム       | タイムラインに表示される個々のイベントまたはデータポイントの収集                                             |
| 方向         | タイムラインコンポーネントの方向またはレイアウト (水平または垂直)                                           |
| 保留中        | タイムライン内のアイテムの状態                                                              |
| 高さ         | タイムラインコンポーネントの垂直方向のサイズまたは高さ                                                  |
| Hidden     | タイムラインコンポーネントが表示されているか非表示か                                                   |

### メソッド

他のコンポーネントを使用してコンポーネントを制御できます。 以下の 2 つの方法をサポートします。

-   **setValue**

テキストエリアの入力値を設定するには、例えば {{"value1"}}

| プロパティー | 説明          |
| ------ | ----------- |
| 値      | Input value |

-   **clearValue**

選択したコンポーネントの値をクリアするには

### データ

コンポーネントには一般的に使用されているデータがいくつか含まれており、アプリで **`{{componentName.propertyName}}`** を介して呼び出すことができます。

| プロパティ名          | 説明                                                                                         |
| --------------- | ------------------------------------------------------------------------------------------ |
| 方向              | タイムラインの向きまたはレイアウト                                                                          |
| dynamicHeight   | タイムラインの高さを動的に調整する能力はその内容などに基づいています                                                         |
| displayName     | このコンポーネントの名前 (タイムライン1)                                                                     |
| hidden          | タイムラインコンポーネントが表示されているか非表示か                                                                 |
| 項目              | タイムラインに表示される個々のイベントまたはデータポイントの収集                                                           |
| 保留中             | タイムライン内のアイテムの状態                                                                            |
| サイズ変更           | タイムラインコンポーネントのサイズをユーザーによって変更できる方向                                                          |

例: {{timeline1.items}}

### 大文字小文字を区別する

次に、データソースからタイムラインコンポーネントにデータをマッピングする方法を説明します。

#### ステップ 1 アクションを追加

**`timeline`,** という名前の Supabase にテーブルを作成しましょう。 タイムラインコンポーネントにマップしたい4つのマイルストーンがあります。

次に、アクションリストからSupabaseの新しいアクションを作成し、supabasedb1という名前を付けます。

マイルストーン内のすべてのデータを一覧表示するには、以下のコードスニペットをSQLクエリセクションに置きます。

```sql
タイムラインから * を選択;
```

出力データは辞書ペアのリストなので、配列に変換したいと考えています。 変換器で有効にして、コードを下に置いてください。

```jsx
const transformedData = data.map((obj) => Object.values(obj)[0]);
return transformedData;
```

**`Save`** と **`Run`**をクリックしてアクションを有効にします。

![timeline_data](https://cdn.illacloud.com/official-website/img/official-site/components/timeline_data.jpg)

#### ステップ2 コンポーネントを追加

次に、**`timeline`** コンポーネントをキャンバスに追加します。

コンポーネントをクリックした後に「検査」ページの「**アイテム**」フィールドに、「{{supabasedb1.data}}」を置いて、スパベースからデータを取得します。

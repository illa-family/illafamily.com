## <h2 hidden>App editor</h2>

APPエディタは、アプリケーションを開発するためのインターフェイスであり、以下の主要部分で構成されています。

-   ツールバー
-   Frame
-   アクションエディタ
-   データ ワークスペース
-   Insert
-   調査

<img alt="editor" src={'https://cdn.illacloud.com/official-website/img/official-site/content.png'} />

### ツールバー

ツールバーは、主にAPPを設定するために使用され、次のタスクに使用することができますAPPエディタの上部にあります：

-   左の「ILLA」アイコンをクリックすると、ホームページに戻ります。
-   APP情報を表示して状態を保存します。
-   左の項目を表示/非表示にします。右の項目、下のパネルを表示します。
-   アイテムの拡大縮小を調整します。
-   アプリの現在の開発者をすべて表示します。
-   アプリに現在のバグの状態を知らせています。
-   アプリを他の開発者と共有します。
-   アプリを実行しています

#### さらに多くの機能が停止しています

-   json からアプリをインポートします。
-   Reproduction APPs.
-   apps を json 形式でエクスポートします。
-   ご希望のAPPバージョンを選択します。
-   編集履歴を表示します。
-   すべての操作を再読み込みします。

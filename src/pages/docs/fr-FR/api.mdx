Connectez-vous à une API
Connectez-vous à n'importe quelle API REST, GraphQL API.
Vous pouvez connecter ILLA à la plupart des APIs en utilisant nos intégrations natives. Si vous avez besoin de vous connecter à une API interne, ou s'il n'y a pas d'intégration supportée, vous pouvez vous connecter en utilisant notre intégration REST API ou l'intégration GraphQL.

1.  Créez une ressource API 
    Les ressources représentent les connexions aux sources de données et sont ce que vous demandez dans vos applications ILLA. Connectez-vous à ILLA et accédez à l'onglet Ressources > cliquez sur Créer une nouvelle et sélectionnez l'API à laquelle vous voulez vous connecter, comme REST API. Cela ouvre la configuration d'une ressource modale où vous ajoutez vos informations de connexion.

Au minimum, vous avez besoin de l'URL de base de l'API à laquelle vous voulez vous connecter. Vous pouvez également passer des paramètres d'URL, des en-têtes, etc. Si votre API nécessite une authentification, vous pouvez également le configurer sur cette page. Par exemple, OAuth2.0 requiert une URL d'autorisation, une URL de jeton d'accès, etc. Après avoir entré vos informations de connexion, cliquez sur Enregistrer la ressource. 
2\. Demander votre API
dans l'éditeur d'application, cliquez sur + Nouveau dans la Liste d'Action > sélectionnez REST API et choisissez une API REST existante ou créez une nouvelle API > cliquez sur Créer une action et mettre à jour les actions avec des paramètres d'URL supplémentaires, en-têtes, etc.

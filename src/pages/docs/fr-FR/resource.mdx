## <Heading hidden>Ressource</Heading>

Connectez-vous à votre source de données et appelez-la dans votre constructeur ILLA. Nous vous fournissons des modules de connexion adaptables pour différents types de sources de données. Vous pouvez construire une connexion stable entre vos données et ILLA en un court laps de temps, en simplifiant grandement votre processus d'accès aux données.

Après avoir créé avec succès une ressource, vous pouvez l'appeler à n'importe quelle application en ILLA. Vous pouvez également partager vos ressources avec vos co-développeurs, améliorant l'efficacité des applications de construction.

### Type de ressources supportées

ILLA supporte un certain nombre de bases de données importantes, telles que MySQL et Postage, ainsi que les APIs comme l'API Rest et GraphQL.

### Créer une ressource

Les ressources représentent les connexions entre ILLA Builder et vos sources de données. Créer une ressource en suivant les étapes suivantes :

#### 1.  Accédez à l'onglet Ressources sur la barre de navigation de la page d'accueil et cliquez sur le bouton "Créer nouveau"

<img
  src='https://cdn.illacloud.com/official-website/img/docs/re_e_1.png'
  alt=""
/>

#### 2.  Choisissez votre type de données ici

<img
  src='https://cdn.illacloud.com/official-website/img/docs/qs_e_2.png'
  alt=""
/>

#### 3.  Affinez vos informations de source de données et cliquez sur le bouton "Créer une ressource" puis terminez le processus

<img
  src='https://cdn.illacloud.com/official-website/img/docs/re_e_2.png'
  alt=""
/>

De plus, vous pouvez créer des ressources dans la liste d'actions sous l'éditeur d'application

> ### Tester la connexion
>
> Lorsque vous créez ou visualisez des ressources, vous pouvez cliquer sur le bouton de test sous le panneau de ressources pour vérifier si la source de données est bien connectée.

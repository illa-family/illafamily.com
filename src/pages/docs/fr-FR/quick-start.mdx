## <Heading hidden>Aperçu</Heading>

### 1.  Déployer le constructeur ILLA

Avant de commencer à utiliser le constructeur ILLA, vous devriez choisir d'utiliser ILLA Cloud ou l'hébergement ILLA pour héberger vos applications. Nous vous fournissons ILLA CLI, Docker Compose, et K8s pour l'auto-hôte ILLA en moins de 5 minutes. Nous vous fournirons ILLA Cloud plus tard, pour vous aider simplement à commencer avec ILLA Builder.

<img
  src='https://cdn.illacloud.com/official-website/img/docs/qs_e_1.png'
  alt=""
/>

### 2.  Obtenir les données des utilisateurs

Enregistrez-vous et connectez-vous à ILLA Builder, puis entrez l'onglet Ressources. Cliquez sur le bouton "Créer un nouveau" puis vous pouvez sélectionner le type de ressource ici. Nous vous fournissons l'API REST et MySQL pour connecter vos données. Postgres, Redis et MongoDB seront disponibles ce mois-ci.

<img
  src='https://cdn.illacloud.com/official-website/img/docs/qs_e_2.png'
  alt=""
/>

### 3.  Glisser-déposer des inserts

Lorsque vos données sont préparées, vous pouvez retourner à l'onglet Applications et créer une application. La liste d'insertion se trouve sur le côté droit de la page. Vous pouvez choisir l'un d'eux et le faire glisser dans le cadre. Cliquez ensuite sur l'insertion et modifiez ses paramètres dans le panneau d'inspection. Nous vous fournissons le mode markdown pour éditer le libellé de chaque insert. Si vous êtes familier avec la syntaxe Markdown, vous pouvez rapidement ajuster la taille et l'épaisseur du texte. En attendant, vous pouvez utiliser {{double brackets}} pour invoquer des données d'action dans la zone de saisie de texte pour obtenir plus de résultats dans ILLA Builder.

### 4.  Ajouter une action dans les applications

Une action est nécessaire pour transformer les données des ressources en insertion. Nous l'appelons action. Vous pouvez écrire une requête en action pour capturer des données de votre ressource. Sur cette base, vous pouvez écrire JavaScript dans des actions pour exécuter des instructions de traitement de données, telles que le filtre et la fusion. Un exemple MySQL est donné ci-dessous:

<img
  src='https://cdn.illacloud.com/official-website/img/docs/qs_e_3.png'
  alt=""
/>

1.  Créez une action MySQL dans l'éditeur d'applications. Nommé action1.

2.  Entrez le code de la requête dans le panneau, par exemple SELECT \* FROM Users.

3.  Exécuter cette action, le résultat de la requête sera affiché sur le panneau inférieur.

### 5.  Afficher vos données

Lorsque l'action est créée avec succès, vous devez lier le résultat de l'action au composant. Vous devriez remplacer la variable de valeur sur le panneau d'inspection par le résultat des actions {{action1.data}}. Ensuite, les données du composant se lieront avec le résultat de l'action.

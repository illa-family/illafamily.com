### Rechercher un composant

Vous pouvez cliquer sur l'onglet INSERT sur le panneau de droite pour entrer dans la liste des composants. Entrez le nom du composant et recherchez le composant dans la zone de saisie ci-dessus.

<img
  src='https://cdn.illacloud.com/official-website/img/docs/assemble/illa_components(34).png'
  alt=""
/>

### Utiliser le gestionnaire d'événements pour déclencher des événements

Prenant le composant de bouton comme exemple, vous pouvez définir un gestionnaire d'événements pour le composant à l'aide de l'image affichée pour déclencher une action ou contrôler d'autres composants, etc. Lorsque vous cliquez sur un composant, ou lorsque les données du composant changent, les événements correspondants peuvent être déclenchés.

<img
  src='https://cdn.illacloud.com/official-website/img/docs/assemble/illa_components(35).png'
  alt=""
/>

### Utiliser les données du composant dans les actions

Vous pouvez utiliser des données de composants dans des actions ou d'autres composants. En prenant le composant d'entrée comme exemple, vous pouvez utiliser {{input1.value}} pour obtenir la valeur entrée dans le composant. Les différents composants peuvent avoir des formats de données différents. Veuillez consulter les formats de données à travers le panneau de gauche.

<img
  src='https://cdn.illacloud.com/official-website/img/docs/assemble/illa_components(36).png'
  alt=""
/>

## <h2 hidden>Redis是什么？</h2>

Illa 中的 Redis 是一个数据库集成，允许您连接并与 Redis 数据库交互。 Redis 是一个受欢迎的开放源码关系数据库管理系统，它以其速度、可靠性和易用性而著称。

Illa中的 Redis 集成后，您可以通过 SQL 命令查询、插入、更新和删除Redis 数据库中的数据。 您也可以创建自定义 SQL 查询并在Illa中运行它们。 这便于建立数据驱动应用程序和仪表板，从Redis数据库中显示实时数据。

### 创建 Redis

在登录到您的 Illa 帐户后，有两种方式在Illa 中创建资源。

-   在资源中创建

登录到您的 Illa 帐户，在页面顶部选择 `**资源**` 并点击 `**创建新的**` 按钮。

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

从数据库列表中选择 **`Redis`** 。

![action_list](https://cdn.illacloud.com/official-website/img/docs/action_list.png)

使用下面`链接设置`描述的必填参数连接到数据库。

点击 "**测试连接**" 查看我们是否可以成功地连接到数据库。 如果是，请点击 **\`保存资源** \*，请重新检查主机名、端口、用户名和密码是否正确。

创建资源后，准备好的 Redis 将显示。

![redis_ex_ready](https://cdn.illacloud.com/official-website/img/docs/redis_ex_ready.png)

-   在构建器中创建

登录您的 Illa 帐户，在`**App**` 页面中创建一个项目 并导航到页面底部的 **`Action List` **。 点击`**新**`，然后从数据库列表中选择 **`Redis`** 。 然后，用下面`链接设置`描述的参数连接到数据库。

点击 "**测试连接**" 查看我们是否可以成功地连接到数据库。 如果是，请点击 **\`保存资源** \*，请重新检查主机名、端口、用户名和密码是否正确。

### 连接设置

这里我们需要提供连接到 Redis 数据库的信息。

![redis_config](https://cdn.illacloud.com/official-website/img/docs/redis_config.png)

| 属性             | 必填  | 描述                                                                                                            |
| -------------- | --- | ------------------------------------------------------------------------------------------------------------- |
| 名称             | 必填  | 在 ILLA 中创建动作时的资源名称。                                                                                           |
| Hostname       | 必填  | 您数据库的 URL 或 IP 地址                                                                                             |
| 端口             | 必填  | 您应该使用服务器主机的端口号进行连接。 如果您没有指定端口，默认端口为“6379”。                                                                    |
| 数据库索引          | 可选的 | 用于优化从Redis数据库检索数据的查询性能的数据结构                                                                                   |
| 数据库            | 必填  | 数据库名称                                                                                                         |
| 用户名            | 必填  | 登录到 Redis 服务器时您想使用的用户名。                                                                                       |
| 密码             | 必填  | 使用此密码进行身份验证。                                                                                                  |
| SSL 选项         | 可选的 | 在与服务器谈判时，决定安全的 SSL TCP/IP 连接的优先级程度。                                                                           |

### 创建动作

我们创建了 Redis 资源，我们可以从动作列表中选择 Redis 并选择 **`创建动作` ** 按钮来添加动作。

![redis_resource_list](https://cdn.illacloud.com/official-website/img/docs/redis_resource_list.png)

现在我们已经将Redis服务器添加到我们的建筑页面作为一个动作。

![redis](https://cdn.illacloud.com/official-website/img/docs/redis.png)

### 配置 Redis

| 属性       | 描述                                                |
| -------- | ------------------------------------------------- |
| 查询       | 从数据库检索数据的命令                                       |
| 转换器      | 将数据转换为你喜欢使用 JavaScript 的样式                        |

示例用法：

-   密钥

此命令检索Redis数据库中所有键的列表。 您可以使用通配符来筛选密钥。

```sql
KEYS *
KEYS user:*
```

-   获取

此命令检索与给定键相关联的值。

```sql
GET user:12345
```

-   HGETALL

此命令从Redis 散列数据结构中检索所有字段及其值。

```sql
HGETALL user:12345
```

-   ZRANGE

这个命令根据他们的得分从Redis分类的数据结构中检索一系列值。

```sql
ZRANGE high_scores 0 9 WITHSCORES
```

-   上线

此命令从Redis列表数据结构中获取一系列值。

```sql
LRANGE my_list 0 4
```

-   到期时间

此命令检查Redis数据库中是否存在给定的密钥。

```sql
EXISTS user: 12345
```

-   TTL

此命令检索与键值相关联的时间对应值 (TTL)，指定键值到期之前在数据库中保留多长时间。

```sql
TTL user:12345
```

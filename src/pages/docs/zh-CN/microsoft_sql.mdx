## <h2 hidden>Microsoft SQL是什么？</h2>

Illa 中的 Microsoft SQL 是一个数据库集成，允许用户连接并与 Microsoft SQL (MS SQL) 数据库交互。 MS SQL 是一个受欢迎的开放源码关系数据库管理系统，它以其速度、可靠性和易用性而著称。

在Illa中使用 MS SQL 集成后，您可以查询、插入、更新和删除使用 SQL 命令的 MS SQL 数据库中的数据。 您也可以创建自定义 SQL 查询并在Illa中运行它们。 这便于构建数据驱动应用程序和仪表板，显示来自MS SQL数据库的实时数据。

### 创建 MS SQL

在登录到您的 Illa 帐户后，有两种方式在Illa 中创建资源。

-   在资源中创建

登录到您的 Illa 帐户，在页面顶部选择 `**资源**` 并点击 `**创建新的**` 按钮。

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

从数据库列表中选择 **`Microsoft SQL`** 。

![action_list](https://cdn.illacloud.com/official-website/img/docs/action_list.png)

使用下面`链接设置`描述的必填参数连接到数据库。

点击 "**测试连接**" 查看我们是否可以成功地连接到数据库。 如果是，请点击 **\`保存资源** \*，请重新检查主机名、端口、用户名和密码是否正确。

创建资源后，已准备好的 MS SQL 将显示。

![ms_ex_ready](https://cdn.illacloud.com/official-website/img/docs/ms_ex_ready.png)

-   在构建器中创建

登录您的 Illa 帐户，在`**App**` 页面中创建一个项目 并导航到页面底部的 **`Action List` **。 点击`**新**`，然后从数据库列表中选择 **`Microsoft SQL`** 。 然后，用下面`链接设置`描述的参数连接到数据库。

点击 "**测试连接**" 查看我们是否可以成功地连接到数据库。 如果是，请点击 **\`保存资源** \*，请重新检查主机名、端口、用户名和密码是否正确。

### 连接设置

这里我们需要提供连接MS SQL数据库的信息。

![ms_config](https://cdn.illacloud.com/official-website/img/docs/ms_config.png)

| 属性                | 必填  | 描述                                                                                                        |
| ----------------- | --- | --------------------------------------------------------------------------------------------------------- |
| 名称                | 必填  | 在 ILLA 中创建动作时的资源名称。                                                                                       |
| Hostname          | 必填  | 您数据库的 URL 或 IP 地址                                                                                         |
| 端口                | 必填  | 您应该使用服务器主机的端口号进行连接。 如果您没有指定端口，默认端口是 '1433'。 如果您没有指定端口，默认端口是 '1433'。                                       |
| 数据库               | 必填  | 数据库名称                                                                                                     |
| 用户名               | 必填  | 登录到 MS SQL 服务器时您想使用的用户名。                                                                                  |
| 密码                | 必填  | 使用此密码进行身份验证。                                                                                              |
| 连接选项              | 可选的 | 指定连接选项的密钥对值                                                                                               |
| SSL 选项            | 可选的 | 在与服务器谈判时，决定安全的 SSL TCP/IP 连接的优先级程度。                                                                       |

### 创建动作

我们已经创建了一个 MS SQL 资源；我们可以从动作列表中选择MS SQL 并选择 **`Create action` ** 按钮来添加动作。

![ms_resource_list](https://cdn.illacloud.com/official-website/img/docs/ms_resource_list.png)

现在我们已经将MS SQL服务器添加到我们的建筑页面作为一个动作。

![ms](https://cdn.illacloud.com/official-website/img/docs/ms.png)

### 配置 MS SQL

| 属性       | 描述                                                                                                                                                                                                                                        |
| -------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 配置类型     | 您用来连接到MS SQL Server 数据库的配置类型：SQL 和 Bulk 插入 对于SQL模式，您可以使用 SQL 查询。 您用来连接到MS SQL Server 数据库的配置类型：SQL 和 Bulk 插入 对于SQL模式，您可以使用 SQL 查询。 对于批量插入模式，您可以将数据插入一个选定的表格，格式为 {}。[{x:a, y:b}， … ] }}                                                     |
| SQL 查询   | 要获取的 sql 命令 (选择), 插入, 更新, 删除数据库中的数据。                                                                                                                                                                                                      |
| 转化器      | 将数据转换为你喜欢使用 JavaScript 的样式                                                                                                                                                                                                                |

示例用法：

-   选择

```sql
SELECT * FROM users ORDER BY lastname;
```

-   Insert

```sql
INSERT INTO users

	(firstname, lastname, email)

VALUES

	(

		{{ NewUser.data.firstNameInput }},

		{{ NewUser.data.lastNameInput }},

		{{ NewUser.data.emailInput }}

	);
```

-   更新

```sql
UPDATE users
  SET firstname = '{{form1.updatedProfile.first}}'
  WHERE id = {{ form1.updatedProfile.uid}};
```

-   删除

```sql
DELETE FROM users WHERE id = {{ form1.updatedProfile.uid }};
```

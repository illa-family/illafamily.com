## <h2 hidden>条是什么样的进展？</h2>

### 使用大小写

我们将演示如何在底部显示表格完成进度的栏进度表。

#### 步骤 1 添加组件

添加 **`form`** 组件, 默认文本"Form " 和 "提交"按钮到画面。

添加 **`input`** 组件、 **`number input`** 组件、 **`bar progress`** 组件从`**插入**` 添加到表单中。

将输入组件标注为“名称”，将输入组件标注为“年龄”，将输入组件标注为“进度”。

将输入组件的占位符设置为“请将您的名字放在这里”，以及下面显示的数字输入的默认值。

#### 步骤 2 配置组件

对于**`input`** 和 **`number input`** 组件, 我们可以将它们配置为**`bar progress`** 组件中的值, 以反映完成的进度。

1.  点击 **`input`** 和 **`number input`** 组件来打开它的检查页面。 在 **`Event handler`**下单击`**+ 新**`。 在 **`Event handler`**下单击`**+ 新**`。 在 **`Event handler`**下单击`**+ 新**`。 在 **`Event handler`**下单击`**+ 新**`。 在 **`Event handler`**下单击`**+ 新**`。 在 **`Event handler`**下单击`**+ 新**`。 在 **`Event handler`**下单击`**+ 新**`。 在 **`Event handler`**下单击`**+ 新**`。 在 **`Event handler`**下单击`**+ 新**`。 在 **`Event handler`**下单击`**+ 新**`。 在 **`Event handler`**下单击`**+ 新**`。
2.  在编辑事件处理器中，选择 **`Control component` ** 正在执行中。 并选择你想要更新的 **\`bar 进度** 组件作为事件的目标。
3.  选择 **`setValue` ** 动作并将其设置为

```jsx
{
  {
    input1.value && numberInput1.value
      ? "100"
      : input1.value || numberInput1.value
      ? "50"
      : "0";
  }
}
```

![bar_input_num_config](https://cdn.illacloud.com/official-website/img/official-site/components/bar_input_num_config.jpg)

注意：需要设置上述事件处理程序同时到输入和数字输入组件。

2.  在编辑事件处理器中，选择 **`Control component` ** 正在执行中。 并选择 **`form`** 你想要更新的组件作为事件的目标。

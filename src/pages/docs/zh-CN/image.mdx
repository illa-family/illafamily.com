### 数据

组件有一些常用的数据，可以通过应用程序中的 **`{{componentName.propertyName}}`** 调用。

| 属性名称          | 描述                                                                                                       |
| ------------- | -------------------------------------------------------------------------------------------------------- |
| altText       | 描述图像内容或目的的基于文本的备选办法                                                                                      |
| 显示名称          | 代表图像名称或标题的字符串值。                                                                                          |
| 事件            | 定义与图像组件相关联的事件处理数组或对象，如点击                                                                                 |
| hidden        | 判断图像组件是否隐藏在用户界面的布尔值。                                                                                     |
| 图像Src         | 要在组件中显示的图像的来源或位置                                                                                         |
| 适合对象          | 图像如何在组件的尺寸中配装或缩放                                                                                         |
| radius        | 将圆角应用于图像组件。                                                                                              |
| 工具提示文本        | 当用户悬停在图像组件上时显示工具提示或悬停文字的字符串值。                                                                            |

#### 步骤 2 配置组件

在 **`按钮`** 组件中，我们可以配置它来将 **`图像`** 组件的图像源设置为一个新的图像地址。

2.  在编辑事件处理器中，选择 **`控制组件` ** 正在执行中。 并选择你想要更改的 **`image1` ** 组件作为事件的目标。
3.  选择 **\`setImageUrl** 动作
4.  将新的图像地址放入**`Value`** 字段

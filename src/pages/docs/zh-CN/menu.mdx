## <h2 hidden>菜单是什么？</h2>

菜单组件是一个用户界面元素，它为访问ILLA Cloud应用程序中的不同功能、选项或部分提供一个导航结构。 它通常包含一个菜单项或链接，用户可以与之交互以浏览不同页面或执行特定操作。

### 属性

| 属性         | 描述                                                                               |
| ---------- | -------------------------------------------------------------------------------- |
| 菜单         | 菜单组件中显示的实际内容和选项                                                                  |
| Hidden     | 从用户视图中隐藏表单组件                                                                     |
| 方向         | 菜单组件的方向或布局                                                                       |
| Align      | 菜单组件在容器或父元素中的位置或对齐                                                               |

### 数据

组件有一些常用的数据，可以通过应用程序中的 **`{{componentName.propertyName}}`** 调用。

| 属性名称            | 描述                                                                                                   |
| --------------- | ---------------------------------------------------------------------------------------------------- |
| 显示名称            | 代表分配给组件的名称或标签                                                                                        |
| hidden          | 菜单组件是可见还是隐藏                                                                                          |
| 水平对齐            | 控制菜单组件在容器或父元素中的水平定位或对齐方式                                                                             |
| 项目              | 菜单项或选项集合在菜单组件中可用                                                                                     |
| 模式              | 菜单如何操作或与用户互动                                                                                         |
| selectedValues  | 持有用户选择或选择的菜单项的值。                                                                                     |

示例： {{menu1.displayName}}

### 使用大小写

我们将演示如何以输入的名称和年龄制作表单， 以及显示表单完成进度的进度栏。

#### 第 1 步设置页面布局

转到页面配置作为以下步骤：

![pages](https://cdn.illacloud.com/official-website/img/docs/pages.png)

您可以使用预设布局，或者手动添加和删除版块来设置布局。

#### 步骤 2 将视图添加到页面

**`查看路径`**：一个字符串在页面URL之后才能访问指定的视图。 应用部署后，完整的访问地址如下：https://builder.illacloud.com/{{team_identifier}}/dep/app/{{app_id}}/{{page_name}}/{{view_path}}

您也可以通过以下路径输入主页和默认视图：

https://builder.illacloud.com/{{team_identifier}}/dep/app/{{app_id}}

![add_page](https://cdn.illacloud.com/official-website/img/docs/add_page.png)

#### 步骤 3 连接菜单和页面

在页面顶部添加 **`Menu`** 组件.

将事件处理器添加到菜单项 → 设置设置路由器的动作 -> 选择页面和视图

在 **`menu`** 组件中，我们可以配置它在页面和视图之间切换

1.  点击 **`menu`** 组件来打开它的检查页面。
2.  单击一个项目以打开它的配置。
3.  在 **`Event handler`**下单击`**+ 新**`。
4.  在编辑事件处理器中，选择 **`Set Router`** 正在执行中。 并选择您想要前往的页面和视图作为事件的目标。

![menu_items](https://cdn.illacloud.com/official-website/img/docs/menu_items.jpg)

我们可以对其他页面做同样的操作，并查看您想要路线到的页面。

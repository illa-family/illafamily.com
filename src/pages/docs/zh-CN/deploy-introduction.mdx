## <Heading hidden>部署方式简介</Heading>

### ILLA Cloud

在ILLA Cloud中尝试一键化部署，享受最轻松的使用体验：使用ILLA Cloud快速开始

### 私有化部署

您可以使用ILLA CLI, Docker compose, k8s helm 等进行自动操作。 我们已经多次优化了私有化部署流程，使您能够在几分钟内完成部署。 与Cloud版本相比，您可以更加安全地使用ILLA 产品。

想要了解更多私有化部署的信息，请查看ILLA CLI、Docker compose、k8s helm 的引导页面
如果您在自部署中遇到了困难，请在GitHub中联系我们，或通过business@illasoft.com 与我们取得联系
如果您在私有化部署时遇到困难，请在 GitHub 或 business@illasoft.com 上联系我们。

#### 使用 ILLA CLI 自动部署

详见: [ILLA CLI 搭建](https://www.illacloud.com/zh-CN/docs/illa-cli)

#### 手动部署

您也可以选择以下方式进行手动部署ILLA Builder：

-   [Docker All-in-One Image](https://github.com/illacloud/deply-illa-manualy/blob/main/docker/README.md)
-   [Docker Compose](https://github.com/illacloud/deply-illa-manualy/blob/main/docker-compose/README.md)
-   [Kubernetes](https://github.com/illacloud/deply-illa-manualy/blob/main/kubernetes/README.md)

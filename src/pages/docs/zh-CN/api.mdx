连接到 API
连接到任何REST ，GraphQL API。
您可以使用我们的本地集成连接ILLA 到大多数API。 如果您需要连接到内部API，或者如果不支持集成， 您可以使用我们的 REST API 集成或GraphQL 集成连接。

1.  创建 API 资源 
    表示数据源的连接，是您在 ILLA 应用程序中查询的内容。 登录到 ILLA 并点击资源 > 点击创建新的并选择您想要连接的 API 例如REST API。 打开配置一个资源框以添加您的连接信息。

您至少需要您想要连接的 API 的基本URL。 您也可以通过 URL 参数、头部等。 如果您的 API 需要身份验证，您也可以在这个页面上配置它。 例如，OAuth2.0 需要授权URL、访问令牌URL等。 当您输入您的连接详细信息后，点击保存资源。 
2\. 在应用程序编辑器中查询您的 API
点击 + 创建新 Action > 选择 REST API 并选择已存在的 REST API 或创建一个新的 > 点击创建Action并使用额外的 URL 参数，headers更新Action

## <Heading hidden>JavaScript 指南</Heading>

JavaScript是ILLA Builder中最重要的组成部分，这是您搭建项目中数据接入与逻辑建设的工具。可以说， 掌握调用 JavaScript 的方法，你已经掌握了ILLA Builder的使用。

当您在ILLA Builder中搭建项目时，可以使用{{模版语法}}书写自定义的JavaScript语句，用于调用、操作数据，或是在Transformer中直接书写JS代码

### 操作数据

#### 访问对象中的数据

在 ILLA 构建器中，您可以使用模板语法在任何文本输入框中输入 JavaScript 表达式。 数据对象是可引用的，你可以通过 JavaScript 操纵所有Action和Transformer的数据。 例如，您可以使用 `{{textInput.data}}` 来访问文本输入插入的内容，{{action.data}}`从一个动作数据库访问数据，以及`{{transformer.value}}\`从变压器访问数据。

我们已经添加和修改了插入到产品的代码镜像。 调整它以适应ILLA Builder 编程环境，支持语法高亮和自动缩进， 并添加详细说明编程期间出现的变量和函数，以便您可以学习如何低成本使用ILLA Builder。 在您输入对象名称后，请输入一个点（.）。 然后ILLA提示您包含的所有属性、数据类型和部分文字描述， 这样您可以随时识别要访问的属性。

#### 访问数组中的数据

当您需要访问具体数组中的值时，您可以通过需要访问的数据位置或下标获取数值。例如：

```javascript
const list = ['a', 'b', 'c']
return list[0] // Return the first value in alist 'a'
```

### 查看数据

在ILLA Builder中，一共有四处位置可以查看数据，分别是：左侧面板的数据工作区，文本框中输入JavaScript表达式后下方数据提示，通过数据展示组件显示数据。

#### 数据工作区展示

您可以在编辑器中打开左侧面板找到数据工作区，所以数据在其中以结构化JSON的形式展示，每一个属性与列表都可以展开，所有数据都在工作区中直观展示。

[//]: # "[图片]"

#### 数据输入实时预览

当您在输入框中调用了一个查询或Transformer时，输入框下方会自动弹出该Action的数据展示，您可以根据此处的信息确认调用的数据是否正确。

[//]: # "[图片]"

#### 数据展示组件

数据显示插入使您能够最直观地查看数据。 数据展示组件可以让您最直观的查看数据，您只需要将相应的组件拖拽至画布中，在其数据栏中输入对应的数据源名称，加以筛选和排列，就可以获得数据展示的信息了

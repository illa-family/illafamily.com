## <h2 hidden>时间线是什么？</h2>

时间线组件是按横向或纵向轴线按时间顺序排列的事件或数据点的图形说明。 它允许用户以线性方式直观和浏览一系列事件或里程碑。 它允许用户以线性方式直观和浏览一系列事件或里程碑。 它允许用户以线性方式直观和浏览一系列事件或里程碑。

### 属性

Timeline可用的属性。 Timeline可用的属性。 JavaScript 可以被写入阅读或更改组件信息。

| 属性         | 描述                                                                           |
| ---------- | ---------------------------------------------------------------------------- |
| 项目         | 时间线上显示的个别事件或数据点的收集                                                           |
| 方向         | 时间线组件的方向或布局 (水平或垂直)                                                          |
| 待定         | 时间轴中某个项目的状态                                                                  |
| 高度         | 时间线组件的垂直大小或高度                                                                |
| Hidden     | 时间线组件是可见还是隐藏                                                                 |

组件有一些常用的数据，可以通过应用程序中的 **`{{componentName.propertyName}}`** 调用。

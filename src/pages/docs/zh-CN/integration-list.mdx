## <h2 hidden>Integration List</h2>

ILLA提供了一系列集成，使您能够以最小的配置轻松连接到您的数据源。只需从可用选项列表中选择适当的集成，输入必要的凭证，您就可以在ILLA中开始使用您的数据源。使用ILLA的资源避免了繁杂的配置过程，使您能够迅速开始使用您的数据。 只需从可用选项列表中选择适当的集成，输入必要的凭据，您可以在ILLA 中开始使用您的数据源。 这将消除设置复杂配置的需要，并使您能够快速开始处理您的数据。

### 数据库集成

| 名称          | 描述                                                                                                                         |
| ----------- | -------------------------------------------------------------------------------------------------------------------------- |
| Mysql       | MySQL是一个开源的关系型数据库管理系统（RDBMS），它基于结构化查询语言（SQL）。                                                                              |
| PostgreSQL  | PostgreSQL是一个开源的对象关系型数据库管理系统（ORDBMS），以其可靠性和丰富的功能而闻名。                                                                       |
| MariaDB     | MariaDB是一个开源的关系型数据库管理系统，是MySQL数据库的分支。                                                                                      |
| TiDB        | TiDB是一个开源的NewSQL数据库，支持混合交易和分析处理工作负载。                                                                                       |
| 雷迪斯         | Redis是一个开源的内存数据结构存储，通常被用作数据库、缓存和消息代理。                                                                                      |
| MongoDB     | MongoDB是一个免费和开源的跨平台的面向文档的数据库程序。                                                                                            |
| 弹性搜索        | Elastic search是一个建立在Apache Lucene之上的开源、分布式搜索和分析引擎。                                                                         |
| Firebase    | Firebase是一个由谷歌开发的基于云的移动和网络应用开发平台。                                                                                          |
| Supabase DB | Supabase是一个用于构建实时无服务器应用程序的开源平台。它建立在PostgreSQL之上，为快速构建和部署无服务器应用程序提供了许多工具和服务。 它建立在 PostgreSQL 顶端，为快速构建和部署无服务器应用程序提供了一些工具和服务。 |
| Clickhouse  | ClickHouse是一个开源的面向列的数据库管理系统，是为在线分析处理（OLAP）和实时数据分析而设计的。                                                                     |

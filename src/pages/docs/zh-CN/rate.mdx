## <h2 hidden>什么是评分？</h2>

ILLA 云中的速率组件是一个强大的工具，用户可以将评分功能添加到他们的应用中。 带有率组件。 用户可以显示一组恒星或任何其他自定义符号，并允许用户给某个项目评级或提供反馈。

### 属性

| 属性                      | 描述                                                                                                                                                  |
| ----------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------- |
| 默认值                     | 设置评分的初始值                                                                                                                                            |
| 图标                      | 自定义用于评分的图标或符号                                                                                                                                       |
| 最大数量                    | 可以在比率组件中显示的最大图标或恒星数                                                                                                                                 |
| 允许减半数                   | 当启用半属性时，它允许用户选择和显示半星增量的评分                                                                                                                           |
| 允许清除                    | 允许用户清除或重置选定的评分                                                                                                                                      |
| 标签                      | 显示给用户的字段名称                                                                                                                                          |
| 标题                      | 详细描述字段的标题                                                                                                                                           |
| 隐藏标签                    | 设置是否显示标签                                                                                                                                            |
| 位置                      | 设置标签相对于组件的位置                                                                                                                                        |
| 对齐                      | 设置标签的对齐(对齐到左或右)                                                                                                                                     |
| Width                   | 当标签位于组件左侧时，设置标签的宽度。                                                                                                                                 |
| 事件处理程序                  | 响应组件事件触发查询、控制组件或调用其他API。                                                                                                                            |
| 已禁用                     | 控制组件是否被禁用的状态。 当组件被禁用时，它不能被修改或聚焦。                                                                                                                    |
| 只读                      | 控制组件是否为只读状态. 一个只读组件可以被选择和集中，但不能被修改。                                                                                                                 |
| 工具提示                    | 用户可以在此输入组件工具提示。 工具提示将在其焦点时显示。 支持Markdown 格式。                                                                                                        |
| 必填字段                    | 只有当开启开关时才有效。                                                                                                                                        |
| 自定义规则                   | 在此创建您的验证逻辑。 规则应该在 JavaScript 中制定，并且由 {{}} 涵盖。                                                                                                       |
| 隐藏验证消息                  | 当输入值不正确时，您可以通过切换隐藏状态来隐藏错误消息。 您可以用JavaScript动态地更改隐藏状态。                                                                                               |
| Hidden                  | 动态控制组件是否隐藏。 您可以通过动态布尔值更改隐藏状态。                                                                                                                       |

### 方法

您可以使用其他组件来控制组件。 我们支持以下两种方法：

-   **setValue** 值

设置比率值，例如：{{"value1"}}

| 属性     | 描述          |
| ------ | ----------- |
| 值      | Input value |

-   **clearValue**

清除选中组件的值

-   **validate**

验证输入信息是合法的

-   **清除验证**

清除验证消息

### 事件处理程序

| 事件  | 描述                                                                |
| --- | ----------------------------------------------------------------- |
| 更改  | 当用户更改值时，执行用户定制的特定操作。                                              |

### 数据

组件有一些常用的数据，可以通过应用程序中的 **`{{componentName.propertyName}}`** 调用。

| 属性名称          | 描述                                                                        |
| ------------- | ------------------------------------------------------------------------- |
| 半边距           | a 是否允许一半比率被起诉的布尔值                                                         |
| 自定义规则         | 可以应用于组件的用户定义规则或验证逻辑。                                                      |
| 已禁用           | 一个布尔值表示禁用状态                                                               |
| 显示名称          | 此组件的名称 (e 按钮1)                                                            |
| 事件            | 当组件中发生某些动作或事件时生成或触发。                                                      |
| hidden        | 隐藏状态 (true 或 false)                                                       |
| 图标            | 用于评分的图标或符号                                                                |
| 标签            | 标签值                                                                       |
| 标签对齐          | 输入标签对齐(左或右)                                                               |
| 标签位置          | 输入标签的位置(左或右)                                                              |
| 标签宽度          | 代表标签宽度的整数。                                                                |
| maxCount      | 最大计数值                                                                     |
| 占位符           | 占位符值                                                                      |
| 只读            | 一个布尔值表示输入的准备状态。                                                           |
| 必填            | 一个布尔值表示输入的必填状态。                                                           |
| 工具提示文本        | 工具提示文本的值                                                                  |
| 值             | 用户输入值                                                                     |

示例： {{rate1.value}}

### 使用大小写

接下来，我们将演示如何从数据源映射数据并用按钮给组件评级。

#### 步骤 1 添加动作

让我们在Amazon S3中创建一个表。 其中一个文件称为:

数字为“1.txt”，数字为“3.5”。 这是我们想要以比率值显示的值。

！[s3data](https://cdn.illacloud.com/official-website/img/official-site/components/s3data.jpg)

然后我们可以从动作列表中为Amazon S3创建一个新的动作，列出桶中的所有数据并命名为s31。

选择 `**Action Type**` 的`读取对象'。 放置`1.txt`为`\*\*对象键\`。\*\*

点击 **\`保存** 和 **`Run`** 激活此操作。

！[rate_data](https://cdn.illacloud.com/official-website/img/official-site/components/rate_data.jpg)

#### 步骤 2 添加组件

接下来，我们可以在画布中添加一个 **`rate` ** 组件和 **`buton` **。

我们将比率组件的默认值设置为 0。 将其标为“评分”，并将按钮标为“设置值”，如下所示：

！[rate_layout](https://cdn.illacloud.com/official-website/img/official-site/components/rate_layout.jpg)

#### 步骤 3 配置组件

**`buton`** 组件. 我们可以配置它来设置我们从 s31 api 读取的文档数据的 \*\*`rate` 组件中的值。

1.  在编辑事件处理器中，选择 **`Control component` ** 正在执行中。 并选择你想要更新的 **`rate` **组件作为事件的目标。
2.  选择 **`setValue`** 动作并设为 `{{s31.data}}`

！[rate_config](https://cdn.illacloud.com/official-website/img/official-site/components/rate_config.jpg)

#### 步骤4 测试

现在当您点击“设置值”按钮时，值应该更改为111。

！[rate_test](https://cdn.illacloud.com/official-website/img/official-site/components/rate_test.jpg)

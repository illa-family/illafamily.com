## <h2 hidden>已知问题</h2>

### 挂载类型问题

运行`illa deprivation --self -port=1000 -s={your-ip-address}` 或 `run-official-image.sh`, 返回错误后：

`Docker responded with status code 400: invalid mount config for type "bind": bind source path does not exist: /tmp/illa-data`

现在，挂载类型只支持 `mount`，请将 `docker -v` 设置为 `mount` 模式。 或者通过以下脚本重新安装停靠。

[reinstall-docker-with-ubuntu.sh](https://github.com/illacloud/deply-illa-manualy/blob/main/utils/reinstall-docker-with-ubuntu.sh)

并再次运行部署命令。

更多问题：https://github.com/illacloud/illa-builder/issues

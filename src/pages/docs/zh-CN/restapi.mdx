### 创建 REST API

在登录到您的 Illa 帐户后，有两种方式在Illa 中创建资源。

-   在资源中创建

登录到您的 Illa 帐户，在页面顶部选择 `**资源**` 并点击 `**创建新的**` 按钮。

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

从数据库列表中选择 **\`REST API** 。

![action_list](https://cdn.illacloud.com/official-website/img/docs/action_list.png)

使用下面`链接设置`描述的必填参数连接到数据库。

点击 "**测试连接**" 查看我们是否可以成功地连接到数据库。 如果是，请点击 **\`保存资源** \*，请重新检查主机名、端口、用户名和密码是否正确。 如果是，请点击 **\`保存资源** \*，请重新检查主机名、端口、用户名和密码是否正确。 如果是，请点击 **\`保存资源** \*，请重新检查主机名、端口、用户名和密码是否正确。

在创建资源后，准备好的 REST API 将会显示。

![rest_ex_ready](https://cdn.illacloud.com/official-website/img/official-site/components/rest_ex_ready.jpg)

-   在构建器中创建

登录您的 Illa 帐户，在`**App**` 页面中创建一个项目 并导航到页面底部的 **`Action List` **。 点击`**新**`，然后从数据库列表中选择 **`REST API`** 。 然后，用下面`链接设置`描述的参数连接到数据库。

### 连接设置

这里我们需要提供连接到 REST API 数据库的信息。

![rest_config](https://cdn.illacloud.com/official-website/img/official-site/components/rest_config.jpg)

| 属性            | 必填  | 描述                                                                            |
| ------------- | --- | ----------------------------------------------------------------------------- |
| 名称            | 必填  | 在 ILLA 中创建动作时的资源名称。                                                           |
| 基本网址          | 必填  | 您数据库的 URL 或 IP 地址                                                             |
| URL 参数        | 可选的 | 提供额外信息或修改 API 请求的行为                                                           |
| 信头            | 可选的 | 关于 API 请求或响应的额外元数据                                                            |
| Cookie        | 可选的 | 小块数据存储在客户端并与每个API请求一起发送                                                       |
| 认证            | 可选的 | 验证用户身份或提出API请求的应用程序                                                           |

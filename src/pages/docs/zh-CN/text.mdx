### 属性

| 属性                   | 描述                                                                                       |
| -------------------- | ---------------------------------------------------------------------------------------- |
| Content              | 设置文本组件中显示的文本。 支持Markdown和纯文本。                                                            |
| Tooltip              | 当用户悬停在组件上时在组件上显示工具提示。                                                                    |
| Horizontal alignment | 设置组件中文本内容的水平位置                                                                           |
| Vertical alignment   | 在组件中设置文本内容的顶点位置                                                                          |
| Hidden               | 设置是否显示此组件以及在什么条件下显示它。 例如，如果当前用户不是用户A，这个组件应该隐藏：{{ currentUserInfo.nickname != 'User A' }} |
| Color                | 以绘制文本格式设置文本颜色                                                                            |
| Text size            | 设置绘制文本格式的文本大小                                                                            |

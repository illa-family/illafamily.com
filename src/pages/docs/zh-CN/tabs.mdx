## <h2 hidden>什么是Tabs？</h2>

ILLA Cloud中的标签组件是一个多功能功能，用户可以在标签页中组织和介绍内容。 它允许在单个容器内创建多个标签，使用户能够轻松地在不同的内容部分间切换。 标签组件提供一个方便用户的接口，用于浏览相关信息，改进用户的总体体验。 具有可定制的样式选项和添加各种类型内容的能力，例如文本、图像或嵌入组件， ILLA 云中的标签组件提供了灵活性并加强了数据的编排和列出。

### 属性

| 属性                  | 描述                                                                                                                                           |
| ------------------- | -------------------------------------------------------------------------------------------------------------------------------------------- |
| 链接到一个容器             | 将标签组件连接到一个特定的容器，从而使标签和相关内容之间的无缝导航成为可能。                                                                                                       |
| 事件处理程序              | 检测和响应特定用户动作或事件，如点击、按键或值变更。                                                                                                                   |
| 已禁用                 | 禁用特定标签，阻止用户与他们进行互动                                                                                                                           |
| 工具提示                | 每个标签页的附加信息或上下文信息                                                                                                                             |
| 布局                  | 提供不同的布局选项，允许您根据您的设计首选项或可用空间水平或垂直排列标签页。                                                                                                       |
| Align               | 标签组件与父容器内的不同位置对齐，如左、中心或右对齐                                                                                                                   |
| Hidden              | 程序性隐藏特定标签页。                                                                                                                                  |
| 文本颜色                | 可自定义的文本颜色，允许您用不同颜色样式标签标签以匹配您应用程序的设计或品牌。                                                                                                      |

### 数据

组件有一些常用的数据，可以通过应用程序中的 **`{{componentName.propertyName}}`** 调用。

| 属性名称                  | 描述                                                                                                      |
| --------------------- | ------------------------------------------------------------------------------------------------------- |
| align                 | 选项卡在组件中的对齐，如左、中心或右对齐                                                                                    |
| 颜色方案                  | 定义组件中文本的颜色                                                                                              |
| currentIndex          | 代表标签列表中当前选中标签的索引                                                                                        |
| 当前密钥                  | 当前选中标签的唯一标识符或密钥                                                                                         |
| 显示名称                  | 代表分配给组件的名称或标签                                                                                           |
| hidden                | 判断标签组件是否可见或隐藏的布尔值                                                                                       |
| 链接小部件名称               | 显示与特定标签相关联的链接小部件的名称或标签                                                                                  |
| navigateContainer     | 标签内容应该显示或导航到的容器或目标                                                                                      |
| 标签列表                  | 标签列表，包括它们的显示名称、密钥和其他相关信息                                                                                |
| tabPosition           | 标签条条纹的位置，如顶部、底部、左部或右部位置                                                                                 |
| 工具提示文本                | 当用户在特定标签上悬停时显示为工具提示的文本                                                                                  |
| 查看列表                  | 视图或与每个标签相关联的内容，允许根据所选标签呈现不同的内容                                                                          |

1.  点击 **`form`** 组件来打开它的检查页面。 在 **`Event handler`**下单击`**+ 新**`。
2.  在编辑事件处理器中，选择 **`submit`** 在事件中选择 **`Show notification`** 在 **`action`**， 将"提交成功"的标题"、"发送"朋友请求"用于描述，并选择 "Success" 用于**`Type`**。 我们想要通知显示2秒，将{{2000}}放到持续时间内。

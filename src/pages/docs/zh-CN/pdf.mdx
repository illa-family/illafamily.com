## <h2 hidden>PDF是什么？</h2>

PDF 组件是一个功能或元素，用户可以在应用程序内查看 PDF 文档并进行交互。 它提供了渲染、导航和操纵PDF文件的功能，使用户能够无缝访问 PDF内容。 它提供了渲染、导航和操纵PDF文件的功能，使用户能够无缝访问 PDF内容。

### 属性

| 属性            | 描述                                                                        |
| ------------- | ------------------------------------------------------------------------- |
| PDF 源         | 指定要显示的 pdf 来源                                                             |
| 显示工具栏         | 控制工具栏在 PDF 组件中的可见性                                                        |
| 工具提示          | 添加显示在用户悬停在pdf 上的工具提示或描述性文本                                                |
| 缩放页面到         | 决定组件内PDF页面的缩放行为                                                           |
| Hidden        | 决定 PDF 组件的可见性                                                             |

### 方法

您可以使用其他组件来控制组件。 我们支持以下方法： 我们支持以下方法：

-   **setFileUrl**

动态地更改 pdf 文件的源或 URL。

| 属性    | 描述     |
| ----- | ------ |
| PDF 源 | pdf源网址 |

### 数据

组件有一些常用的数据，可以通过应用程序中的 **`{{componentName.propertyName}}`** 调用。

| 属性名称          | 描述                                                                                                     |
| ------------- | ------------------------------------------------------------------------------------------------------ |
| 显示名称          | 表示pdf名称或标题的字符串值。                                                                                       |
| 高度            | PDF 组件的垂直尺寸或大小                                                                                         |
| hidden        | 判断pdf 组件是否隐藏在用户界面的布尔值。                                                                                 |
| 缩放模式          | 指定 PDF 文档如何缩放或适合组件可用显示区域                                                                               |
| showToolBar   | 工具栏或控制面板是否与 PDF 组件一起显示                                                                                 |
| 工具提示文本        | 当用户在 pdf 组件上悬停时显示的工具提示或悬停文字的字符串值。                                                                      |
| 网址            | 要在组件中显示的 PDF 文件源                                                                                       |
| width         | PDF 组件的水平尺寸或大小                                                                                         |

示例： {{pdf1.displayName}}

### 使用大小写

接下来，我们将演示如何使用按钮控制pdf显示。

#### 步骤 1 添加组件

在画布中添加 **`PDF` 组件和 **`按钮`\*\* 组件.

![pdf_layout](https://cdn.illacloud.com/official-website/img/docs/pdf_layout.jpg)

#### 步骤 2 配置组件

在 **`按钮`** 组件中，我们可以配置它来将 **`PDF`** 组件的图像源设置为 pdf 地址。

1.  点击 "**按钮**" 组件来打开它的检查页面。 在 **`Event handler`**下单击`**+ 新**`。 在 **`Event handler`**下单击`**+ 新**`。
2.  在编辑事件处理器中，选择 **`Control component` ** 正在执行中。 并选择你想要更改的 \*\*`PDF` 组件作为事件的目标。
3.  选择 **\`setFileUrl** 动作
4.  将新的图像地址放入**`Value`** 字段

![pdf_config](https://cdn.illacloud.com/official-website/img/docs/pdf_config.jpg)

#### 第 3 步测试

现在当您点击“播放”按钮时，图像应该切换

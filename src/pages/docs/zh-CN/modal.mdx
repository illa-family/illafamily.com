## <h2 hidden>什么是模态框？</h2>

在ILLA Cloud中，模式组件是一个用于显示内容的用户界面元素。 信息或在一个焦点和叠加层窗口中的动作。 模式组件通常出现在主应用程序屏幕上，暂时中断用户的工作流程，并提请他们注意模式内容。 模式组件通常出现在主应用程序屏幕上，暂时中断用户的工作流程，并提请他们注意模式内容。

### 属性

| 属性                        | 描述                                                                                            |
| ------------------------- | --------------------------------------------------------------------------------------------- |
| 事件处理程序                    | 检测和响应特定用户动作或事件，如点击、按键、值变化                                                                     |
| 点击掩码关闭模式                  | 是否单击模式的背景叠加层或掩码将关闭模式窗口                                                                        |
| 显示标题                      | 组件应该显示页眉还是标题部分                                                                                |
| 显示页脚                      | 组件是否应该显示页脚部分                                                                                  |
| 边框颜色                      | 围绕模式组件的边框颜色                                                                                   |
| 边框半径                      | 模式组件边框角落的曲线                                                                                   |
| Border Width              | 模型组件周围边框厚度或宽度                                                                                 |
| 背景                        | 定义填充模式组件内容区域的颜色或图像                                                                            |

#### 步骤 2 配置组件

对于"打开模块" **`按钮`** 组件，如果点击它，我们可以配置它来打开 **`modal`** 组件。

1.  点击 **`按钮`** 组件来打开它的检查页面。 在 **`Event handler`**下单击`**+ 新**`。 在 **`Event handler`**下单击`**+ 新**`。
2.  在编辑事件处理器中，选择 **`Control component` ** 正在执行中。 并选择你想要更新的 **`modal`** 组件作为事件的目标。
3.  选择 **`openModal` ** 方法

![open_modal](https://cdn.illacloud.com/official-website/img/docs/open_modal.jpg)

单击此按钮打开模式组件，您可以将任何想要的内容放置在模式的中间区域。

对于"取消" **`按钮`** 组件，如果点击的话，我们可以配置它来关闭模式

1.  点击 **`按钮`** 组件来打开它的检查页面。 在 **`Event handler`**下单击`**+ 新**`。 在 **`Event handler`**下单击`**+ 新**`。
2.  在编辑事件处理器中，选择 **`Control component` ** 正在执行中。 并选择你想要更新的 **`modal`** 组件作为事件的目标。
3.  选择 **`closeModal` ** 方法

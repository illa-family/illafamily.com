## <h2 hidden>TiDB是什么？</h2>

TiDB 是一个数据库集成，允许您连接并与 TiDB 数据库交互。 这是一个可横向扩大的关系数据库管理系统，旨在处理大量、同币值高的工作量。

在Illa中使用 TiDB 集成，您可以通过 SQL 命令查询、更新和删除TiDB 数据库中的数据。 您也可以创建自定义 SQL 查询并在Illa中运行它们。 这便于建立数据驱动应用程序和仪表板，显示来自TiDB数据库的实时数据。

### 创建 TiDB

在登录到您的 Illa 帐户后，有两种方式在Illa 中创建资源。

-   在资源中创建

登录到您的 Illa 帐户，在页面顶部选择 `**资源**` 并点击 `**创建新的**` 按钮。

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

从数据库列表中选择 **`TiDB`** 。

![action_list](https://cdn.illacloud.com/official-website/img/docs/action_list.png)

使用下面\`连接设置'描述的必填参数连接到数据库。

点击 "**测试连接**" 查看我们是否可以成功地连接到数据库。 如果是，请点击 **\`保存资源** \*，请重新检查主机名、端口、用户名和密码是否正确。

创建资源后，TiDB将显示。

![ti_ex_ready](https://cdn.illacloud.com/official-website/img/docs/ti_ex_ready.png)

-   在构建器中创建

登录您的 Illa 帐户，在`**App**` 页面中创建一个项目 并导航到页面底部的 **`Action List` **。 点击`**新**`，然后从数据库列表中选择 **`TiDB`** 。 然后，用下面`连接设置`描述的参数连接到数据库。

点击 "**测试连接**" 查看我们是否可以成功地连接到数据库。 如果是，请点击 **\`保存资源** \*，请重新检查主机名、端口、用户名和密码是否正确。

### 连接设置

这里我们需要提供连接TiDB数据库的信息。

![ti_config](https://cdn.illacloud.com/official-website/img/docs/ti_config.png)

![ti_config_1](https://cdn.illacloud.com/official-website/img/docs/ti_config_1.png)

| 属性                 | 必填  | 描述                                                                                                                                                                                                                       |
| ------------------ | --- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| 名称                 | 必填  | 在 ILLA 中创建动作时的资源名称。                                                                                                                                                                                                      |
| Hostname           | 必填  | 您数据库的 URL 或 IP 地址                                                                                                                                                                                                        |
| 端口                 | 必填  | 您应该使用服务器主机的端口号进行连接。 如果您没有指定端口，默认端口是 '4000'。                                                                                                                                                                              |
| 数据库                | 必填  | 数据库名称                                                                                                                                                                                                                    |
| 用户名                | 必填  | 登录到 TiDB 服务器时您想要使用的用户名。                                                                                                                                                                                                  |
| 密码                 | 必填  | 使用此密码进行身份验证。                                                                                                                                                                                                             |
| SSL 选项             | 可选的 | 在与服务器谈判时，决定安全的 SSL TCP/IP 连接的优先级程度。                                                                                                                                                                                      |
| CA 证书              | 必填  | 用来核实服务器身份和确定安全的数字证书， 已加密客户端和 TiDB 集群之间的连接。 如果您没有指定它，我们已经为您填写了默认证书。                                                                                                                                                       |
| 客户密钥               | 可选的 | 用于在 TiDB 客户端和 TiDB 服务器之间建立安全连接的参数。                                                                                                                                                                                       |
| 客户端证书              | 可选的 | 允许客户端验证自己到 TiDB 服务器的安全功能                                                                                                                                                                                                 |

### 创建动作

我们创建了一个 TiDB 资源，我们可以通过从动作列表中选择 TiDB 并选择 **`Create action` ** 按钮来添加动作。

![ti_resource_list](https://cdn.illacloud.com/official-website/img/docs/ti_resource_list.png)

现在我们已经将 TiDB 服务器添加到我们的建筑页面作为一个动作。

![ti](https://cdn.illacloud.com/official-website/img/docs/ti.png)

### 配置 TiDB

| 属性       | 描述                                                               |
| -------- | ---------------------------------------------------------------- |
| SQL 查询   | 要获取的 sql 命令 (选择), 插入, 更新, 删除数据库中的数据                              |
| 转换器      | 转换数据到你喜欢使用 Javascript 的样式                                        |

示例用法：

-   选择

```sql
选择* 从用户排序按姓排列；
```

-   Insert

```sql
INSERT INTO users

	(firstname, lastname, email)

VALUES

	(

		{{ NewUser.data.firstNameInput }},

		{{ NewUser.data.lastNameInput }},

		{{ NewUser.data.emailInput }}

	);
```

-   更新

```sql
UPDATE users
  SET firstname = '{{form1.updatedProfile.first}}'
  WHERE id = {{ form1.updatedProfile.uid}};
```

-   删除

```sql
DELETE FROM users WHERE id = {{ form1.updatedProfile.uid }};
```

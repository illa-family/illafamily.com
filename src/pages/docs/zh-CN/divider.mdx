## <h2 hidden>分隔符是什么？</h2>

在ILLA Cloud中，分隔符组件是一个视觉元素，有助于在网页应用程序中分离或分割内容部分。 这是一个简单的横向线或纵向线，可以在不同的部分或组件之间插入，以改进用户界面的视觉结构和清晰度。 这是一个简单的横向线或纵向线，可以在不同的部分或组件之间插入，以改进用户界面的视觉结构和清晰度。

### 属性

| 属性         | 描述                                                                  |
| ---------- | ------------------------------------------------------------------- |
| 文本         | 向分隔符组件添加额外信息                                                        |
| Text align | 决定文本在分隔符组件中的水平对齐方式                                                  |
| Hidden     | 隐藏用户界面中的分隔符组件                                                       |
| 文本大小       | 调整分隔符组件中文本的字体大小                                                     |

### 数据

组件有一些常用的数据，可以通过应用程序中的 **`{{componentName.propertyName}}`** 调用。

| 属性名称          | 描述                                                              |
| ------------- | --------------------------------------------------------------- |
| 颜色方案          | 定义组件中文本的颜色                                                      |
| 显示名称          | 代表分配给组件的名称或标签                                                   |
| fs            | 与分隔符相关联的文件系统 (fs)                                               |
| hidden        | 判断组件是可见还是隐藏的布尔值                                                 |
| 文本            | 文本值                                                             |

示例： {{divider1.text}}

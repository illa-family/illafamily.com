## <h2 hidden>API authentication</h2>

ILLA kan støtte nesten alle slags autentisering. Vi har samarbeidet med ulike selskaper for å støtte alle standardtyper autentisering, og har også utviklet metoder for å støtte eventuelle tilpassede autentiseringsmetoder vi ikke støtter opp om.

Du kan bruke statiske autentiseringstokener eller opprette et autentiseringssystem som spør sluttbrukeren om sin legitimasjon hver gang de bruker applikasjonen.

Denne siden gir deg instruksjoner om hvordan du logger inn din API. Hvis du har spørsmål eller trenger hjelp, du kan kontakte vårt supportteam ved hjelp av Intercom boblen på denne siden eller hjelpeikonet i editoren.

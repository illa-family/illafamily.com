## <h2 hidden>Hva er tabell?</h2>

Tabellkomponenten i ILLA Cloud er en kraftig funksjon som lar brukere vise, organisere og manipulere tabellariske data i programmet. Det har et strukturert oppsett med rader og kolonner, slik at brukerne kan se og samhandle med data på en hensiktsmessig og strukturert måte.

### Egenskaper

| Egenskaper                            | Beskrivelse                                                                                                                        |
| ------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------- |
| Data kilde                            | kilde til data som fyller tabellen                                                                                                 |
| Tom tilstand                          | innholdet eller meldingen vises når tabellen ikke har noen data å vise                                                             |
| Laster                                | om tabellkomponenten for øyeblikket er i en lasttilstand                                                                           |
| Kolonner                              | konstruksjon og utforming av søylene i tabellen. (Se avsnittet om kolonner nedenfor for mer informasjon)                           |
| Kolonne                               | antall kolonner                                                                                                                    |
| Valg for flere rader                  | Bruk {{table.selectedRow[n].columnName}} for å få tilgang til data.                                                                |
| Celle utvalg                          | Om brukere kan velge en celle                                                                                                      |
| Klikk utenfor for å fjerne            | Støttet i rad som enkeltvalgsmodus. Etter aktivering kan du klikke på et annet område enn tabellen for å fjerne raden.             |
| Overflyt                              | hvordan tabellen håndterer innhold som overskrider det tilgjengelige området i tabellens beholder (strekning eller rull)           |
| Aktiver paginering på server          | Om tjenersidenavigering                                                                                                            |
| Sidestørrelse                         | antall rader som skal vises per side når serverside er aktivert                                                                    |
| Oppdater                              | Om du vil vise oppdateringsikonet.                                                                                                 |
| Nedlasting                            | Vis nedlastingsknappen i verktøylinjen.                                                                                            |
| Last ned rådata                       | Om du vil vise rådata ikonet.                                                                                                      |
| Filtrer                               | Vis filterknapp i verktøylinjen                                                                                                    |
| Arrangement behandler                 | oppdage og svare på spesifikke brukerhandlinger eller hendelser, som klikk, tastaturer eller skjemavisninger.                      |
| Deaktivert                            | ikke-interaktiv og kan ikke endres eller utløses av brukeren.                                                                      |

#### **kolonneegenskaper Oversikt**

For hver kolonne kan vi konfigurere sine egenskaper for å tilpasse dataene som vises for hver kolonne.

| Egenskaper                                         | Beskrivelse                                                                                                                                                                                                                     |
| -------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Tittel på kolonne                                  | angir navnet på eller etiketten til kolonnen som tjener som topptekst for å identifisere dataene den representerer                                                                                                              |
| Kolonne type                                       | definerer typen data som kolonnen inneholder, inkludert auto, tekst, dato, dato, dato, klokkeslett, dato/klokkeslett, antall, prosent, link, knapp, knappegruppe, boolsk, bilde, ikongruppe, Rating, Marked, valuta             |
| Mappet verdi                                       | Bruk {{ currentRow }} for å få tilgang til dataene i gjeldende rad. For eksempel for å beregne "kostnad per klikk" basert på kolonner "cost" og "clickTimes":                                                                   |
| {{{currentRow.cost / currentRow.clickTimes }}      |                                                                                                                                                                                                                                 |
| Aktiver sortering                                  | om kolonnen lar brukerne sortere tabelldataene basert på verdiene i den bestemte kolonnen                                                                                                                                       |
| Bakgrunn                                           | Sett bakgrunn til kolonnen. (Automatic Translation) Bruk {{ skrive en betinget angivelse for å angi fargen som vises for hver celle under forskjellige forhold.                                                                 |
| Justering                                          | betjener horisontal justering av innholdet i kolonnecellene                                                                                                                                                                     |

#### Kolonnetyper:

-   Automatisk

en dynamisk type som automatisk bestemmer datatype basert på verdier i kolonnen

-   Tekst

representerer en kolonne som inneholder tekstdata

-   Dato

representerer dataverdier som samsvarer med spesifikke datoer

| Egenskaper   | Beskrivelse                                                               |
| ------------ | ------------------------------------------------------------------------- |
| Formatér     | aksepterer ulike formateringskoder eller -mønster, standard er YYYY-MM-DD |

-   Tagg

et spesialisert format for å vise emneord eller etiketter tilknyttet dataoppføringer

| Egenskaper       | Beskrivelse                                                                    |
| ---------------- | ------------------------------------------------------------------------------ |
| Etikett etikett  | representerer teksten eller etiketten som er tilordnet en tag i kolonnen "tag" |
| Tagg farge       | definerer fargen som er knyttet til taggen i kolonnen "tag"                    |

-   Tid

representerer og manipulerer tidsverdiene i tabellen

| Egenskaper   | Beskrivelse                                                                       |
| ------------ | --------------------------------------------------------------------------------- |
| Formatér     | den visuelle representasjonen av tidsdata innenfor kolonnen, standard er HH:mm:ss |

-   DatoTid

butikker og representerer dato og klokkeslett verdier

| Egenskaper   | Beskrivelse                                                                                              |
| ------------ | -------------------------------------------------------------------------------------------------------- |
| Formatér     | angir hvordan dato og klokkeslettverdiene skal vises i kolonnen, standard er ÅÅÅÅ-MM-DD HH:mm:ss         |

-   Nummer

håndtere numeriske data

| Egenskaper                 | Beskrivelse                                                                                                                 |
| -------------------------- | --------------------------------------------------------------------------------------------------------------------------- |
| Antall desimaler           | angi antall desimalplasser som skal vises for numeriske verdier i kolonnen                                                  |
| Vis tusenskilletegn        | om kolonnen bør vise en tusenvis separator for å forbedre lesbarheten og fortolkningen av store tall                        |

-   Prosent

Håndter prosentverdier på en konsis og visuelt ankende måte

| Egenskaper                 | Beskrivelse                                                                         |
| -------------------------- | ----------------------------------------------------------------------------------- |
| Antall desimaler           | angi antall desimalplasser som skal vises for prosentverdier i kolonnen             |
| Vis tusenskilletegn        | om en tusentalskilletegn skal inkluderes i de viste prosentverdiene                 |

-   Lenke

inkluder klikkbare lenker i en bestemt kolonne

-   Knapp

inkluder interaktive knapper innenfor en spesifikk kolonne

| Egenskaper            | Beskrivelse                                                                            |
| --------------------- | -------------------------------------------------------------------------------------- |
| Arrangement behandler | definerer handlingen eller hendelsen som oppstår når knappen i kolonnen blir klikket   |
| Deaktivert            | om knappen i kolonnen er aktivert eller deaktivert                                     |
| Tema farge            | velg en temafarge for knappen                                                          |
| Variant               | visuell stil eller variant av knappen (fyll eller omriss)                              |

-   knappegruppe

Tillat brukere å opprette interaktive knappegrupper i en kolonne

For hver knapp i kolonnene kan vi konfigurere sine egenskaper.

| Egenskaper            | Beskrivelse                                                                                                                              |
| --------------------- | ---------------------------------------------------------------------------------------------------------------------------------------- |
| Mappet verdi          | knytte hver knapp i knappgruppen til en bestemt verdi eller et felt fra datakilden                                                       |
| Variant               | visuell stil eller utseende på knappene i knappegruppen                                                                                  |
| Arrangement behandler | handlingen eller funksjonen utløst når en knapp i gruppen blir klikket eller interagert med                                              |
| Deaktivert            | om knappene i knappgruppen er aktivert eller deaktivert                                                                                  |
| Tema farge            | tilpasse farge eller styling av knappene i gruppen for å justere med overordnet tema eller profilering av programmet eller nettstedet    |

-   Boolean

vise data som har to mulige tilstander: sanne eller falske

-   Bilde

fyll en kolonne i tabellen med bildedata

| Egenskaper | Beskrivelse                                              |
| ---------- | -------------------------------------------------------- |
| Skala type | hvordan bildet er skalert eller endret i tabellcellen    |

-   Ikongruppe

en spesialisert kolonne som tillater brukere å vise ikoner i en gruppe eller samling

For hver knapp i kolonnene kan vi konfigurere sine egenskaper.

| Egenskaper            | Beskrivelse                                                                                                                               |
| --------------------- | ----------------------------------------------------------------------------------------------------------------------------------------- |
| Ikon                  | lar brukere velge eller angi ikonet som skal vises i kolonnen for Ikongruppe                                                              |
| Tema farge            | bestemmer farge-skjemaet eller -temaet som er anvendt på ikonet i Ikongruppe-kolonnen                                                     |
| Arrangement behandler | gjør det mulig for brukere å definere en hendelse eller handling som utløses når ikonet i Ikongruppe-kolonnen blir interagert med brukere |
| Deaktivert            | tilstand i ikongruppe-kolonnen, som bestemmer om den er deaktivert eller aktivert for brukerinteraksjon                                   |

-   Vurdering

vise vurderinger eller tilbakemeldinger til enheter eller enheter

-   Markdown

mulighet til å gjengi formatert tekst ved Markdown-syntaks i kolonneceller

-   Valuta

Håndtere og formatere valutaverdier.

| Egenskaper                 | Beskrivelse                                                                       |
| -------------------------- | --------------------------------------------------------------------------------- |
| Antall desimaler           | antall desimalplasser som skal vises for valutaverdier innenfor kolonnen          |
| Valuta kode                | angir kode eller forkortelse som representerer valuta som brukes i kolonnen       |
| Vis tusenskilletegn        | om en tusenvis separator skal vises for store valutaverdier                       |

### Metode

Du kan bruke andre komponenter til å kontrollere komponenten. Vi støtter følgende to metoder:

-   **selectPage**

velg alle radene på en bestemt side i en sidetabell

| Egenskaper     | Beskrivelse                                         |
| -------------- | --------------------------------------------------- |
| Endelig Verdi  | side nummer eller indeks for siden som skal velges  |

-   **selectRow**

velg en bestemt rad i tabellen

| Egenskaper           | Beskrivelse                                                                                                                                  |
| -------------------- | -------------------------------------------------------------------------------------------------------------------------------------------- |
| Standard markert rad | standard radverdi kan være radindeksen, en unik identifikator eller annen egenskap som unikt identifiserer raden                             |

-   \*\*eliminering

nullstill gjeldende utvalg i tabellen

-   **setFilters**

bruk filtre til tabelldataene

| Egenskaper     | Beskrivelse                              |
| -------------- | ---------------------------------------- |
| Filtre         | filtreringsforhold eller kriterier       |
| Filtrere modus | bestemmer hvordan filtrene brukes        |

-   **ClearFilters**

tømmer alle angitte filtre i tabellen

-   **setSort**

angi sortering for tabellen

| Egenskaper  | Beskrivelse                                                     |
| ----------- | --------------------------------------------------------------- |
| Kolonne     | kolonnen som sorteringen bør brukes på                          |
| Retning     | retning for sorten som kan "gå opp" eller "synkende"            |

## Bruk kasse

Deretter vil vi vise hvordan du kartlegger dataene fra `**Upstash**` til `**table**` komponenten.

#### Trinn 1 Legg til en handling

La oss lage en tabell i Upstash kalt **`ansatte-profile`, ** inkludert 4 kolonner `navn`, `level`, `age`, og `married`.

Da kan vi opprette en ny handling for Upstash fra handlingslisten i ILLA Cloud og kalt upstash2.

For å liste opp alle data i ansatte-profilen har du satt koden under i delen Redis spørringer.

```sql
FÅ ansatte_profil
```

Siden utdataen er en liste over ordlister, ønsker vi å omdanne den til en liste. I Transformator, aktiver det og sett ned koden nedenfor.

```jsx
const jsonString = data[0].result;

// Parse the JSON string into a JavaScript object
const jsonData = JSON.parse(jsonString);

// Access the "employees" property and map it to a new array
const mappedArray = jsonData.employees.map((employee) => ({
  name: employee.name,
  level: employee.level,
  age: employee.age,
  married: employee.married,
}));

return mappedArray;
```

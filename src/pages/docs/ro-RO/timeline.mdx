## <h2 hidden>Ce este Timeline?</h2>

O componentă de cronologie este o reprezentare grafică a evenimentelor sau punctelor de date dispuse cronologic de-a lungul unei axe orizontale sau verticale. Acesta permite utilizatorilor să vizualizeze și să navigheze printr-o serie de evenimente sau repere într-o manieră liniară.

### Proprietăți

Proprietățile disponibile ale cronometrului. JavaScript poate fi scris pentru a citi sau a modifica informații despre componente.

| Proprietăți     | Descriere                                                                                            |
| --------------- | ---------------------------------------------------------------------------------------------------- |
| Articole        | colectarea evenimentelor individuale sau a punctelor de date care sunt afișate pe cronologia de timp |
| Direcție        | orientarea sau dispunerea componentei temporale (orizontală sau verticală)                           |
| In asteptare    | starea unui element în cronologia                                                                    |
| Înălțime        | dimensiunea verticală sau înălțimea componentei de cronologie                                        |
| Hidden          | dacă componenta de cronologie este vizibilă sau ascunsă                                              |

### Metode

Puteţi utiliza alte componente pentru a controla componenta. Sprijinim următoarele două metode:

-   **setValue**

Pentru a seta valoarea de intrare a zonei de text, de exemplu, {{"value1"}}

| Proprietăți  | Descriere   |
| ------------ | ----------- |
| Valoare      | Input value |

-   **clearValue**

Pentru a șterge valoarea componentei selectate

### Date

Componenta are niște date utilizate în mod obișnuit, care pot fi apelate prin **`{{componentName.propertyName}}`** în aplicație.

| Nume proprietate       | Descriere                                                                                                      |
| ---------------------- | -------------------------------------------------------------------------------------------------------------- |
| direcție               | orientare sau dispunere a cronologiei                                                                          |
| înălțime dinamică      | capacitatea de a regla înălțimea cronologiei în funcție de conținutul acesteia sau de alți factori.            |
| AfișeazăNume           | numele acestei componente (ex timeline1)                                                                       |
| hidden                 | dacă componenta de cronologie este vizibilă sau ascunsă                                                        |
| articole               | colectarea de evenimente individuale sau puncte de date afișate pe cronologia de timp                          |
| în așteptare           | starea unui element în cronologia                                                                              |
| redimensionareDirecție | direcția în care componenta cronologică poate fi redimensionată de către utilizator                            |

Exemplu: {{timeline1.items}}

### Utilizare majusculă

Apoi, vom demonstra cum să cartografiem datele de la sursa de date la componenta de cronologie.

#### Pasul 1 Adăugare acţiune

Haideţi să creăm un tabel în Supabase numit **`timelin`,** care include doar 1 coloane: repere Există 4 repere pe care dorim să le cartografiem la componenta cronologică.

Apoi putem crea o nouă acțiune pentru Supabase din lista de acțiuni și numită supabasedb1.

Pentru a afișa toate datele din piatra de hotar, puneți codul snippet de mai jos în secțiunea de interogare SQL.

```sql
selectează * din cronologie;
```

Deoarece datele de ieşire sunt o listă de perechi de dicţionar, vrem să le transformăm într-un array. În Transformer, activați-l și puneți codul de mai jos.

```jsx
const transformedData = data.map(obj) => Object.values(obj)[0]);
return transformedData;
```

Faceţi clic pe **`Salvează`** şi **`Run`** pentru a activa această acţiune.

![timeline_data](https://cdn.illacloud.com/official-website/img/official-site/components/timeline_data.jpg)

#### Pasul 2 Adaugă Componente

Apoi, putem adăuga o componentă **`cronelină`** la pânză.

În câmpul `**Elemente**` din pagina Inspect după ce ați făcut clic pe componentă, introduceți `{{supabasedb1.data}}` pentru a prelua datele de la supabase.

## <h2 hidden>Ce este Upstash?</h2>

Upstash este un serviciu nativ fără servere cu Redis compatibil care oferă o modalitate ușoară de a integra funcționalitatea Redis în aplicații. În contextul ILLA Cloud, integrarea Upstash permite utilizatorilor să mobilizeze funcțiile și capacitățile Upstash în aplicația ILLA Cloud.

Acest tutorial prezintă procesul de creare a unui Panou Administrator folosind ILLA Builder și Upstash în câțiva pași simpli. ILLA Cloud este o platformă cu coduri joase pentru dezvoltatori care permite dezvoltarea și implementarea rapidă a instrumentelor interne. Permite crearea de pagini prin glisarea și fixarea componentelor UI, conectarea la orice bază de date sau API și scrierea JavaScript. Pentru a afla mai multe despre Upstash, vizitați site-ul lor la https://upstash.com/(https://upstash.com/) Să începem!

### Configurează-ți spatele pe Upstash

Creează un cont pe Upstash. Vă puteți conecta folosind conturile Amazon, Github sau Google alternativ.

După ce te conectezi în contul tău pe `Upstash`, dă click pe butonul `Creează baza de date` din Redis.

![up_home](https://cdn.illacloud.com/official-website/img/docs/up_home.jpg)

Pe tabloul de bord `Upstash`, apăsați `Creare bază de date`.

![up_create_db](https://cdn.illacloud.com/official-website/img/docs/up_create_db.jpg)

Notă: puteţi avea o singură bază de date la nivelul gratuit.

Pentru configurarea bazei de date, puteți denumi baza de date așa cum doriți. Aici numim `db1`. Alegeți `regiunea primară` și `citește regiunea` în funcție de locațiile tale de scriere și citire a utilizatorului. Activați `TSL/SSL`. Este opţional să activezi Evictia.

![up_db_config](https://cdn.illacloud.com/official-website/img/docs/up_db_config.jpg)

Faceți clic pe butonul **`create`**, acum avem o bază de date gata de utilizare.

### Crează Upstash în Norul ILLA

Există două modalități de a crea o resursă în Illa după ce te autentifici în contul tău Illa.

-   Creează în resurse

Conectează-te la contul tău Illa, selectează `**Resurse**` din partea de sus a paginii și apasă butonul `**Creează nou**`.

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

Selectaţi **`Upstash`** din lista de baze de date.

![action_list_0](https://cdn.illacloud.com/official-website/img/docs/action_list_0.jpg)

Conectați-vă la baza de date cu parametrii necesari descriși în `Setări conexiuni` de mai jos.

Faceți clic pe `**Test Connection**` pentru a vedea dacă ne putem conecta cu succes la baza de date. Dacă da, apasă pe **`Salvează Resurse`**, în caz contrar, verifică de două ori numele gazdei, portul, numele de utilizator și parola sunt corecte.

După crearea unei resurse, `Upstash` gata se va afișa ca afișat.

![up_ex_ready](https://cdn.illacloud.com/official-website/img/docs/up_ex_ready.jpg)

-   Creează în Constructor

Conectează-te la contul tău Illa, creează un proiect în Illa Builder pe pagina `**App**`, şi navighează la **`Lista de Acţiuni`** în partea de jos a paginii. Faceţi clic pe `**nou**`, apoi selectaţi **`Upstash`** din lista de baze de date. Apoi, conectați-vă la baza de date cu parametrii necesari descriși în `Setări conexiuni` de mai jos.

Faceți clic pe `**Test Connection**` pentru a vedea dacă ne putem conecta cu succes la baza de date. Dacă da, apasă pe **`Salvează Resurse`**, în caz contrar, verifică de două ori numele gazdei, portul, numele de utilizator și parola sunt corecte.

### Configurare Upstash

| Proprietăți      | Descriere                                                                              |
| ---------------- | -------------------------------------------------------------------------------------- |
| Interogare Redis | folosește interogarea pentru a prelua și manipula datele stocate în baza de date Redis |
| Transformator    | Transformarea datelor în stilul care îți place să folosești JavaScript                 |

Exemplu de utilizare:

1.  Obține valori:
    -   Obține o valoare: Folosește comanda **`GET`** pentru a prelua valoarea asociată cu o anumită cheie. De exemplu: **`cheie GET`**.
2.  Setează valorile:
    -   Set a value: Use the **`SET`** command to set the value of a key. (Automatic Copy) De exemplu **`SET key value`**.
3.  Operațiuni Hash:
    -   Setați perechile de câmpuri valoare-câmp într-un hash: Utilizați comanda **`HSET`** pentru a seta o pereche de câmp valoare-câmp într-un hash. De exemplu: **`HSET hashKey field value`**.
    -   Obțineți o valoare dintr-un hash: Utilizați comanda **`HGET`** pentru a prelua valoarea unui câmp specific într-un hash. De exemplu: **`HGET hashKey field`**.
4.  Operațiuni listă:
    -   Adaugă un element la o listă: Foloseşte comanda **`LPUSH`** sau **`RPUSH`** pentru a adăuga un element la începutul sau sfârşitul unei liste, respectiv. De exemplu: **`LPUSH listKey value`**.
    -   Recuperează elemente dintr-o listă: Folosește comanda **`LRANGE`** pentru a prelua o gamă de elemente dintr-o listă. De exemplu: **`LRANGE listKey start end`**.
5.  Setare operațiuni:
    -   Adaugă elemente la un set: Folosește comanda **`SADD`** pentru a adăuga unul sau mai multe elemente la un set. De exemplu: **`SADD setKey value1 value2`**.
    -   Recuperează elemente dintr-un set: Folosește comanda **`SMEMBERS`** pentru a obține toți membrii unui set. De exemplu: **`SMEMBERS setKey`**.
6.  Seturi sortate:
    -   Adaugă elemente la un set sortat: Folosește comanda **`ZADD`** pentru a adăuga elemente la un set sortat împreună cu scorurile lor. De exemplu: **`ZADD sortedSetKey score1 member1 score2 member2`**.
    -   Recuperează elemente dintr-un set sortat: Folosește comanda **`ZRANGE`** pentru a prelua o gamă de elemente dintr-un set sortat bazat pe scorurile lor. De exemplu: **`ZRANGE sortedSetKey începe să se termine`**.

Notă: Puteți face trimitere la documentația Redis pentru o listă completă de comenzi și utilizarea lor.

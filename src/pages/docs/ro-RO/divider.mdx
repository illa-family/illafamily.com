## <h2 hidden>Ce este Divider?</h2>

În ILLA Cloud, componenta divider este un element vizual care ajută la separarea sau împărțirea secțiunilor de conținut într-o aplicație web. Este o linie orizontală simplă sau o linie verticală care poate fi inserată între diferite secțiuni sau componente pentru a îmbunătăți organizarea vizuală și claritatea interfeței utilizatorului.

### Proprietăți

| Proprietăți      | Descriere                                                                                    |
| ---------------- | -------------------------------------------------------------------------------------------- |
| Text             | adăugaţi informaţii suplimentare la componenta de divider                                    |
| Text align       | determină alinierea orizontală a textului în cadrul componentei de separare                  |
| Hidden           | ascunde componenta de compartimentare din interfaţa utilizatorului                           |
| Dimensiune text  | ajustează dimensiunea fontului textului în cadrul componentei divider                        |

### Date

Componenta are niște date utilizate în mod obișnuit, care pot fi apelate prin **`{{componentName.propertyName}}`** în aplicație.

| Nume proprietate | Descriere                                                                        |
| ---------------- | -------------------------------------------------------------------------------- |
| culori           | definește culoarea textului în componentă                                        |
| AfișeazăNume     | reprezintă numele sau eticheta atribuită componentei                             |
| fs               | sistemul (sistemele) de fișiere asociat cu divizorul                             |
| hidden           | valoare boolean care determină dacă componenta este vizibilă sau ascunsă         |
| text             | valoarea textului                                                                |

Exemplu: {{divider1.text}}

### Utilizare majusculă

Îl puteți plasa oriunde doriți să împărțiți pagina în două secțiuni. Aici vom arăta despărțirea cu trei alinieri diferite de text.

## <h2 hidden>Ce este calendarul evenimentului?</h2>

Componenta Calendar Eveniment din ILLA Cloud este un instrument puternic pentru organizarea și afișarea evenimentelor într-un mod atractiv și ușor de utilizat. Acesta permite utilizatorilor să creeze, să gestioneze și să prezinte evenimente cu ușurință, oferind caracteristici cum ar fi data de navigare, clasificarea evenimentelor și informații detaliate despre evenimente. Componenta Calendar de Evenimente oferă aspecte personalizabile, interfețe de creare a evenimentelor intuitive și opțiuni pentru integrarea cu alte sisteme, transformarea acesteia într-un instrument esenţial pentru întreprinderi şi organizaţii în gestionarea şi promovarea eficientă a evenimentelor.

### Proprietăți

#### Opțiuni manuale

Adăugați opțiuni noi sau ștergeți manual opțiunile și configurați una câte una. Proprietățile opțiunilor sunt următoarele:

| Proprietăți        | Descriere                                                                                                                                                                                                         |
| ------------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Id eveniment       | identifică în mod unic fiecare eveniment din calendar                                                                                                                                                             |
| Titlu              | Titlul sau numele evenimentului                                                                                                                                                                                   |
| Descriere          | Descrierea evenimentului                                                                                                                                                                                          |
| Pornire            | data și ora începerii evenimentului                                                                                                                                                                               |
| Sfârșit            | data și ora de sfârșit ale evenimentului                                                                                                                                                                          |
| Id resursă         | identifică resursa sau entitatea asociată evenimentului                                                                                                                                                           |
| Titlul resurselor  | titlul sau numele resursei asociate cu evenimentul                                                                                                                                                                |
| Toată ziua         | dacă evenimentul este un eveniment de o zi sau nu                                                                                                                                                                 |
| Draggable          | dacă evenimentul poate fi tras sau mutat în interfața calendarului                                                                                                                                                |
| Resizable          | Indiferent dacă permiteți utilizatorilor să actualizeze intervalul de timp al evenimentului și schimbați intervalul de timp prin glisarea marginii unui eveniment pentru a prelungi sau scurta intervalul de timp |

#### Optiuni afisate

De asemenea, puteți adăuga opțiuni dintr-o sursă de date prin maparea etichetei și a proprietăților valorii în câmpurile de date corespunzătoare. Poți obține opțiuni dinamice din baza de date în acest mod. La finalul acestui articol, vom demonstra cum să îl folosim printr-un exemplu.

| Proprietăți        | Descriere                                                                                                                                                                                                         |
| ------------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Sursă de date      | Setați sursa de date pentru opțiuni                                                                                                                                                                               |
| Id eveniment       | Un ID unic utilizat pentru identificarea evenimentului                                                                                                                                                            |
| Titlu              | Titlul evenimentului                                                                                                                                                                                              |
| Descriere          | Descrierea evenimentului                                                                                                                                                                                          |
| Pornire            | Ora începerii evenimentului                                                                                                                                                                                       |
| Sfârșit            | Ora de încheiere a evenimentului                                                                                                                                                                                  |
| Id resursă         | Resursele ar putea fi o persoană, o sală de ședințe, etc. Poți seta un id de resurse și să îl conectezi la eveniment.                                                                                             |
| Titlul resurselor  | Numele resursei afișat în vizualizarea resurselor                                                                                                                                                                 |
| Toată ziua         | Fie că este un eveniment de o zi                                                                                                                                                                                  |
| Draggable          | Dacă permiteţi utilizatorilor să actualizeze intervalul de timp al evenimentului prin glisarea pentru a muta evenimentul                                                                                          |
| Resizable          | Indiferent dacă permiteți utilizatorilor să actualizeze intervalul de timp al evenimentului și schimbați intervalul de timp prin glisarea marginii unui eveniment pentru a prelungi sau scurta intervalul de timp |

| eventConfigureMod            | determină modul de configurare pentru evenimentele din calendar                                                                               |

### Utilizare majusculă

Apoi, vom demonstra cum să adăugăm evenimente noi afișate în calendarul evenimentului cu un buton.

#### Pasul 1 Adăugare acţiune

Haideţi să creăm un tabel în Supabase numit `evenimente`. Are 9 coloane: `title`, `description`, `resource_id`, `resource_title`, `all_day`, `draggable`, `resizable`, `start_time`, şi `end_time` .

Apoi putem crea o nouă acţiune pentru Supabase din lista de acţiuni, enumerând toate datele cu codul snippet de mai jos şi numit `supabasedb1.`

```sql
selectează * dintre evenimente;
```

Schimbă din `Execută acţiunea doar când a fost declanşată manual" în `Rulează acţiunea automat când schimbaţi intrările" în colţul din dreapta sus de sub butonul **Salvează\`**.

Faceţi clic pe **`Salvează`** şi **`Run`** pentru a activa această acţiune.

![event_data](https://cdn.illacloud.com/official-website/img/official-site/components/event_data.jpg)

Pentru a adăuga un eveniment nou, avem nevoie și de o altă acțiune. Alegeți aceeași resursă Supabase din lista de acțiuni numită `supabasedb2`, și puneți codul snippet mai jos.

```sql
INSERT INTO events (id, title, description, start_time, end_time, resource_id, resource_title, all_day, draggable, resizable)
VALUES ((SELECT MAX(id) + 1 FROM events), '{{input1.value}}', '{{input2.value}}', '{{dateTime1.value}}', '{{dateTime2.value}}', {{numberInput1.value}}, '{{input4.value}}', {{switch1.value}}, {{switch2.value}}, {{switch3.value}});
```

Componentele de intrare, număr, dateTime și comutator sunt din componentele modale. Există descrieri mai detaliate ale acestor componente.

#### Pasul 2 Configurare acțiuni

Pentru supabasedb2, vrem să afișăm o notificare atunci când noile informații despre eveniment sunt adăugate cu succes în baza de date.

În Acțiuni, faceți clic pe `supabasedb2`, în pagina `General`, derulați în jos la `Gestionarul evenimentului`.

1.  Faceți clic pe **`+ Nou`** pentru a adăuga un nou gestionar de evenimente.
2.  În gestionarul de editare a evenimentului, selectaţi **`Arată notificarea`** în acţiune, **`title`** "Succes" şi **`description`** "Adaugă cu succes un nou eveniment. Alege `Succes` pentru **`type`**, pune `{{2000}}` pentru **`duraţie`** (2 secunde)

![event_action_success](https://cdn.illacloud.com/official-website/img/official-site/components/event_action_success.jpg)

#### Pasul 2 Adaugă Componente

Apoi, putem adăuga o componentă **`calendar de eveniment`**, o componentă **`button`**, și o componentă `**modal**` la pânză, Etichetați componenta butonului în `Adaugă eveniment nou`.

![event_layout](https://cdn.illacloud.com/official-website/img/official-site/components/event_layout.jpg)

În componenta modală, schimbăm textul din colțul din stânga sus în „Eveniment nou”. Apoi adăugăm componentele de intrare `Title` şi `Description`, `Start time` şi `Ora de sfârşit`, numărul de introducere al \`ID-ului de resurse, și componentele de introducere a "Titlului de resurse", "Toată ziua", "Draggable" și "Resizable" și etichetați-le după cum este necesar.

![event_modal_layout](https://cdn.illacloud.com/official-website/img/official-site/components/event_modal_layout.jpg)

#### Pasul 3 Configurarea componentei

Pentru componenta `**eveniment calendar**`, putem cartografia datele de la `supabasedb1` pentru a fi afișate. Mai întâi alegem `mapped` din opţiuni, apoi punem `{{supabasedb1.data}}` în sursa de date şi completăm câmpurile `Mapped Option` aşa cum se arată mai jos.

![event_data_source](https://cdn.illacloud.com/official-website/img/official-site/components/event_data_source.jpg)

Pentru componenta **`button`**, o putem configura pentru a deschide modala, când faceți clic pe ea.

1.  Faceți clic pe componenta **`button`** pentru a deschide pagina sa de inspecție. Sub **`Manipulatorul de evenimente`**, apasă `**+ New**`.
2.  În gestionarul de editare a evenimentului, selectați **`Componenta Control`** în acțiune, și selectați componenta **`modal`** pe care doriți să o actualizați ca țintă a evenimentului.
3.  Alege metoda **`openModal`**

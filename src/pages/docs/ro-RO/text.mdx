## <h2 hidden>Text</h2>

<img src='https://cdn.illacloud.com/official-website/img/docs/assemble/text1.png' alt="" />

### Proprietăți

| Proprietate         | Descriere                                                                                                                                                                                                                           |
| ------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Conținut            | Setați textul afișat pe componenta de text. Markdown și text simplu sunt acceptate.                                                                                                                                                 |
| Tooltip             | Un tip de instrument afișat pe componentă atunci când utilizatorii planează deasupra componentei.                                                                                                                                   |
| Aliniere orizontală | Pentru a seta poziția orizontală a conținutului textului în componentă                                                                                                                                                              |
| Aliniere verticală  | Pentru a seta poziția verțială a conținutului textului în componentă                                                                                                                                                                |
| Hidden              | Pentru a seta dacă se afișează această componentă și în ce condiții se afișează. De exemplu, dacă utilizatorul curent nu este utilizator A, această componentă ar trebui să fie ascunsă: {{ currentUserInfo.nickname != 'User A' }} |
| Culoare             | Pentru a seta culoarea textului în formatul de pictură                                                                                                                                                                              |
| Dimensiune text     | Pentru a seta dimensiunea textului în formatul de vopsea text                                                                                                                                                                       |

### Afișare date dinamice

Folosiți `{{` pentru a obține date de la alte componente sau acțiuni sau de la variabile globale. De exemplu, pentru a afișa porecla utilizatorului curent:

```jsx
## {{ '👋 Salut! '+ currentUserInfo.nickname }}
```

## <h2 hidden>Autentificare API personalizată</h2>

ILLA suportă implicit multe standarde de autentificare (a se vedea aici pentru mai multe informații). 
În plus, ILLA are un mod personalizat de autentificare Auth care vă permite să combinați mai mulți pași suportând practic orice autentificare API. 
Aceste metode de autentificare pot fi utilizate atât pentru resurse REST, cât și GraphQL.

## <h2 hidden>Ce este pictograma?</h2>

Componenta pictogramă din ILLA Cloud este o caracteristică puternică care permite utilizatorilor să mărească atractivitatea vizuală și funcționalitatea aplicațiilor lor. Cu componenta pictogramă, utilizatorii pot adăuga cu ușurință iconițe din bibliotecile populare de pictograme la interfețele lor de utilizatori, o experiență mai intuitivă și mai antrenantă pentru utilizatorii lor.

### Proprietăți

| Proprietăți             | Descriere                                                                                                                                         |
| ----------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------- |
| Iconiță                 | permite utilizatorilor să selecteze cu ușurință o pictogramă dintr-un set predefinit de opțiuni                                                   |
| Gestionar de evenimente | Declanșează interogări, controlează componente sau apelează alte API-uri ca răspuns la evenimentele componentei.                                  |
| Tooltip                 | Utilizatorii pot introduce aici pontul componentei. Sfatul de instrumente va fi afișat când este concentrat. Formatul markdown este acceptat.     |
| Hidden                  | Controlează dinamic dacă componenta este ascunsă. Puteți schimba starea ascunsă printr-o valoare booleană dinamică.                               |
| Culoare temă            | Permite utilizatorilor să specifice culoarea de fundal şi opacitatea butonului                                                                    |

### Gestionar de evenimente

| Eveniment | Descriere                                                                                                          |
| --------- | ------------------------------------------------------------------------------------------------------------------ |
| Click     | Când un utilizator dă click pe această iconiță, efectuează o acțiune specifică personalizată de către utilizatori. |

### Date

Componenta are niște date utilizate în mod obișnuit, care pot fi apelate prin **`{{componentName.propertyName}}`** în aplicație.

| Nume proprietate  | Descriere                                                                                                 |
| ----------------- | --------------------------------------------------------------------------------------------------------- |
| culori            | culoarea conturului pictogramei                                                                           |
| AfișeazăNume      | numele acestei componente (ie icon1)                                                                      |
| evenimente        | generate sau declanșate atunci când apar anumite acțiuni sau evenimente în cadrul componentei respective. |
| hidden            | stare ascunsă (adevărată sau falsă)                                                                       |
| iconnume          | numele etichetei pentru pictogramă                                                                        |
| tooltipText       | valoarea textului cu înclinare unelte                                                                     |

Exemplu: {{icon1.iconName}}

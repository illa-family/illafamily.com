## <h2 hidden>Ce este hidra?</h2>

Hydra este o integrare puternică în cadrul platformei ILLA Cloud care conectează și consolidează fără întreruperi diferite surse și aplicații de date. Acționând ca nod central, Hydra permite un flux eficient de date și comunicarea între diferite sisteme, permițând sincronizarea datelor în timp real, fluxuri de lucru automatizate și procese raționalizate. Cu capacităţile sale robuste, Hydra împuterniceşte întreprinderile să îşi optimizeze operaţiunile, să îmbunătăţească colaborarea, și să ia decizii bazate pe date cu ușurință, toate în cadrul ecosistemului integrat al ILLA Cloud.

### Creează Hydra

Există două modalități de a crea o resursă în ILLA după ce te autentifici în contul tău.

-   Creează în resurse

Conectează-te la contul tău Illa, selectează `**Resurse**` din partea de sus a paginii și apasă butonul `**Creează nou**`.

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

Selectaţi **`hidra`** din lista de baze de date.

![action_list](https://cdn.illacloud.com/official-website/img/docs/action_list_3.jpeg)

Conectați-vă la baza de date cu parametrii necesari descriși în `Setări conexiuni` de mai jos.

Faceți clic pe `**Test Connection**` pentru a vedea dacă ne putem conecta cu succes la baza de date. Dacă da, apasă pe **`Salvează Resurse`**, în caz contrar, verifică de două ori numele gazdei, portul, numele de utilizator și parola sunt corecte.

După crearea unei resurse, Hydra gata se va afișa ca afișat.

![hydra_ex_ready](https://cdn.illacloud.com/official-website/img/official-site/intergration/hydra_ex_ready.jpeg)

-   Creează în Constructor

Conectează-te la contul tău Illa, creează un proiect în Illa Builder în pagina `**App**`, şi navighează la **`Lista de Acţiuni`** în partea de jos a paginii. Faceţi clic pe `**nou**`, apoi selectaţi **`Hydra`** din lista de baze de date. Apoi, conectați-vă la baza de date cu parametrii necesari descriși în `Setări conexiuni` de mai jos.

Faceți clic pe `**Test Connection**` pentru a vedea dacă ne putem conecta cu succes la baza de date. Dacă da, apasă pe **`Salvează Resurse`**, în caz contrar, verifică de două ori numele gazdei, portul, numele de utilizator și parola sunt corecte.

### Setări conexiune

Aici trebuie să oferim informaţii pentru conectarea la baza de date Hydra.

![hydra_config](https://cdn.illacloud.com/official-website/img/official-site/intergration/hydra_config.jpeg)

![hydra_config_1](https://cdn.illacloud.com/official-website/img/official-site/intergration/hydra_config_1.jpeg)

| Proprietăți            | Necesar  | Descriere                                                                                                                                                                                                                                                         |
| ---------------------- | -------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Nume                   | necesar  | Numele pentru resurse la crearea de acțiuni în ILLA.                                                                                                                                                                                                              |
| Hostname               | necesar  | Adresa URL sau adresa IP a bazei de date                                                                                                                                                                                                                          |
| Portul                 | necesar  | Numărul de port al serverului gazdă pe care ar trebui să-l utilizați pentru conectare. Dacă nu specificați un port, portul implicit este '5432'.                                                                                                                  |
| Baza de date           | necesar  | Numele bazei de date                                                                                                                                                                                                                                              |
| Nume                   | necesar  | numele de utilizator pe care dorești să îl folosești atunci când te conectezi la serverul Hydra.                                                                                                                                                                  |
| Parolă                 | necesar  | Utilizați această parolă pentru autentificare.                                                                                                                                                                                                                    |
| Opțiuni SSL            | opţional | decide cât de mare este prioritizată o conexiune SSL TCP/IP în timpul negocierilor cu serverul.                                                                                                                                                                   |
| Certificat CA          | necesar  | un certificat digital care este utilizat pentru a verifica identitatea unui server și pentru a stabili o securitate sigură, conexiune criptată între un client și un cluster Hydra. Dacă nu o specificați, am completat un certificat implicit pentru dvs.        |
| Cheie client           | opţional | un parametru care este folosit pentru a stabili o conexiune securizată între clientul Hydra și serverul Hydra.                                                                                                                                                    |
| Certificat client      | opţional | o funcție de securitate care permite unui client să se autentifice la un server Hydra                                                                                                                                                                             |

### Configure Hydra

| Proprietăți      | Descriere                                                                                   |
| ---------------- | ------------------------------------------------------------------------------------------- |
| Interogare SQL   | comenzi sql pentru a prelua (select), inserare, actualizare, ștergere date din baza de date |
| Transformator    | Transformarea datelor în stilul care îți place folosind Javascript                          |

Exemplu de utilizare:

-   Selectare

```sql
SELECTARE * DE LA utilizatorii ORDUL DUPĂ Nume;
```

-   Insert

```sql
INSERT INTO users

	(firstname, lastname, email)

VALUES

	(

		{{ NewUser.data.firstNameInput }},

		{{ NewUser.data.lastNameInput }},

		{{ NewUser.data.emailInput }}

	);
```

-   Actualizare

```sql
UPDATE users
  SET firstname = '{{form1.updatedProfile.first}}'
  WHERE id = {{ form1.updatedProfile.uid}};
```

-   Ștergere

```sql
DELETE FROM users WHERE id = {{ form1.updatedProfile.uid }};
```

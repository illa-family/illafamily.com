## <h2 hidden>Ce este tabelul?</h2>

Componenta de tabel din ILLA Cloud este o caracteristică puternică care permite utilizatorilor să afișeze, să organizeze și să manipuleze datele tabulare din aplicație. Acesta oferă un aspect structurat cu rânduri și coloane, permițând utilizatorilor să vizualizeze și să interacționeze cu datele într-un mod convenabil și structurat.

### Proprietăți

| Proprietăți                             | Descriere                                                                                                                                 |
| --------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------- |
| Sursă de date                           | sursa de date care populează tabelul                                                                                                      |
| Stare goală                             | conținutul sau mesajul afișat atunci când tabelul nu are date de afișat                                                                   |
| Încărcare                               | dacă componenta Tabel este în prezent în stare de încărcare                                                                               |
| Coloane                                 | structura și configurarea coloanelor din tabel. (Vezi secţiunea despre coloanele de mai jos pentru mai multe detalii)                     |
| Coloana                                 | numărul de coloane                                                                                                                        |
| Selectare multi-rând                    | Utilizați {{table.selectedRow[n].columnName}} pentru a accesa datele.                                                                     |
| Selectare celulă                        | Dacă permiteți utilizatorilor să selecteze o celulă                                                                                       |
| Click afară pentru a deselecta          | Acceptat în modul de selecție unică a rândului. După activare, apasă pe o altă zonă decât tabelul pentru a deselecta rândul.              |
| Recuperare                              | cum se manipulează tabelul conținutul care depășește spațiul disponibil în recipientul tabelului (paginare sau derulare)                  |
| Activează paginarea pe server           | Activarea paginării serverului                                                                                                            |
| Dimensiune pagină                       | numărul de rânduri afişate per pagină atunci când paginarea de pe partea serverului este activată                                         |
| Împrospătează                           | Dacă se afișează iconița de reîmprospătare.                                                                                               |
| Descărcare                              | Arată butonul de descărcare în bara de instrumente.                                                                                       |
| Descarcă datele brute                   | Dacă se afișează iconița de date brute.                                                                                                   |
| Filtrare                                | Arată butonul de filtrare în bara de instrumente                                                                                          |
| Gestionar de evenimente                 | detectare și răspuns la acțiuni sau evenimente specifice ale utilizatorilor, cum ar fi clicuri, taste sau depuneri de forme.              |
| Dezactivat                              | non-interactiv și nu poate fi modificat sau declanșat de către utilizator.                                                                |

#### **Prezentare generală proprietăți coloane**

Pentru fiecare coloană, îi putem configura proprietățile pentru a personaliza datele afișate pentru fiecare coloană.

| Proprietăți                                         | Descriere                                                                                                                                                                                                             |
| --------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Titlul coloanei                                     | specifică numele sau eticheta coloanei, care servește ca antet pentru a identifica datele pe care le reprezintă                                                                                                       |
| Tip coloană                                         | definește tipul de date pe care coloana le conține, inclusiv auto, text, data, etichetă, oră, dateTime, număr, procente, link, buton, grup buton, boolean, imagine, grup, rating, marcare, monedă                     |
| Valoare minuțioasă                                  | Utilizați {{ currentRow }} pentru a accesa datele rândului curent. De exemplu, pentru a calcula "cost per click" pe baza coloanelor "cost" şi "clickTimes":                                                           |
| {{ currentRow.cost / currentRow.clickTimes }}       |                                                                                                                                                                                                                       |
| Activează sortarea                                  | dacă coloana permite utilizatorilor să sorteze datele tabelului pe baza valorilor din coloana respectivă                                                                                                              |
| Context                                             | Setează fundalul coloanei. Utilizaţi {{ scrieți o instrucțiune condiționată pentru a seta culoarea afișată de fiecare celulă în condiții diferite.                                                                    |
| Aliniere                                            | controlează alinierea orizontală a conținutului în interiorul celulelor coloanei                                                                                                                                      |

#### Tipuri de coloane:

-   Automat

un tip dinamic care determină automat tipul de date pe baza valorilor din coloană

-   Text

reprezintă o coloană care conține date textuale

-   Data

reprezintă valori ale datelor care corespund unor date specifice

| Proprietăți | Descriere                                                                |
| ----------- | ------------------------------------------------------------------------ |
| Format      | acceptă diverse coduri sau modele de formatare, implicit este AAAA-LL-ZZ |

-   Etichetă

un format specializat pentru afișarea etichetelor sau a etichetelor asociate cu intrările de date

| Proprietăți        | Descriere                                                                    |
| ------------------ | ---------------------------------------------------------------------------- |
| Etichetă etichetă  | reprezintă textul sau eticheta atribuită unei etichete în coloana "etichetă" |
| Culoare etichetă   | definește culoarea atribuită etichetei în coloana "etichetă"                 |

-   Timp

reprezintă și manipulează corect valorile timpului din tabel

| Proprietăți | Descriere                                                                       |
| ----------- | ------------------------------------------------------------------------------- |
| Format      | reprezentarea vizuală a timpului în interiorul coloanei, implicit este HH:mm:ss |

-   DataOra

stochează și reprezintă data și valorile timpului

| Proprietăți | Descriere                                                                                              |
| ----------- | ------------------------------------------------------------------------------------------------------ |
| Format      | specifică cum ar trebui afișate data și ora în coloană, valoarea implicită este AAAA-LL-ZZ HH:mm:ss    |

-   Număr

gestionează datele numerice

| Proprietăți               | Descriere                                                                                                                                          |
| ------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------- |
| Zecimale                  | specificați numărul de zecimale de afișat pentru valorile numerice din coloană                                                                     |
| Arată separator mii       | dacă coloana ar trebui să afișeze un separator de mii de persoane pentru a îmbunătăți lizibilitatea și ușurința interpretării pentru numerele mari |

-   Procent

gestionează valorile procentuale într-un mod concis și atractiv vizual

| Proprietăți               | Descriere                                                                             |
| ------------------------- | ------------------------------------------------------------------------------------- |
| Zecimale                  | specificați numărul de zecimale de afișat pentru valorile procentuale din coloană     |
| Arată separator mii       | dacă un separator de mii de persoane ar trebui inclus în valorile procentuale afișate |

-   Link-ul

include link-uri care pot fi accesate într-o anumită coloană

-   Buton

include butoane interactive într-o anumită coloană

| Proprietăți             | Descriere                                                                                 |
| ----------------------- | ----------------------------------------------------------------------------------------- |
| Gestionar de evenimente | definește acțiunea sau evenimentul care apare atunci când butonul din coloană este apăsat |
| Dezactivat              | dacă butonul din coloană este activat sau dezactivat                                      |
| Culoarea temei          | alege o culoare pentru temă pentru buton                                                  |
| Variantă                | stilul vizual sau varianta butonului (încarcă sau contur)                                 |

-   buton Grup

permite utilizatorilor să creeze grupuri de butoane interactive într-o coloană

Pentru fiecare buton din coloane, putem configura proprietățile acestuia.

| Proprietăți             | Descriere                                                                                                                                   |
| ----------------------- | ------------------------------------------------------------------------------------------------------------------------------------------- |
| Valoare minuțioasă      | asociază fiecare buton din grupul de butoane cu o anumită valoare sau un câmp din sursa datelor                                             |
| Variantă                | stilul vizual sau aspectul butoanelor în grupul de butoane                                                                                  |
| Gestionar de evenimente | acțiunea sau funcția declanșată atunci când un buton din grup este apăsat sau interacționat cu                                              |
| Dezactivat              | dacă butoanele din grupul de butoane sunt activate sau dezactivate                                                                          |
| Culoarea temei          | personalizează culoarea sau stilul butoanelor din grup pentru a se alinia cu tema sau marca generală a aplicației sau site-ului             |

-   Boolean

reprezintă date care au două posibile state: adevărat sau fals

-   Imagine

populează o coloană în tabel cu datele imaginii

| Proprietăți | Descriere                                                                   |
| ----------- | --------------------------------------------------------------------------- |
| Tip scară   | cum este scalată sau redimensionată imaginea în interiorul celulei de tabel |

-   IconGrup

o coloană specializată care permite utilizatorilor să afișeze pictograme în cadrul unui grup sau al unei colecții

Pentru fiecare buton din coloane, putem configura proprietățile acestuia.

| Proprietăți             | Descriere                                                                                                                                                      |
| ----------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Iconiță                 | permite utilizatorilor să selecteze sau să specifice pictograma care va fi afișată în coloana IconGroup                                                        |
| Culoarea temei          | determină schema de culori sau tema aplicată pictogramei din coloana IconGroup                                                                                 |
| Gestionar de evenimente | permite utilizatorilor să definească un eveniment sau o acţiune declanşată atunci când pictograma din coloana de IconGroup este interacţionată cu utilizatorii |
| Dezactivat              | starea coloanei de IconGroup, determinând dacă este dezactivată sau activată pentru interacțiunea cu utilizatorul                                              |

-   Clasificare

acordă ratinguri sau feedback elementelor sau entităților

-   Markdown

abilitatea de a reda textul formatat folosind sintaxa Markdown în interiorul celulelor de coloană

-   Monedă

valută de cont și format valută.

| Proprietăți               | Descriere                                                                                  |
| ------------------------- | ------------------------------------------------------------------------------------------ |
| Zecimale                  | numărul de zecimale de afișat pentru valorile valutare din coloană                         |
| Codul valutei             | specifică codul sau abrevierea care reprezintă moneda folosită în coloană                  |
| Arată separator mii       | dacă un separator de mii de persoane este afișat pentru valori în monedă mare              |

### Metodă

Puteţi utiliza alte componente pentru a controla componenta. Sprijinim următoarele două metode:

-   **selectPage**

selectează toate rândurile pe o anumită pagină într-un tabel paginat

| Proprietăți         | Descriere                                                |
| ------------------- | -------------------------------------------------------- |
| Valoare de sfârșit  | numărul paginii sau indexul paginii care va fi selectată |

-   **selectRow**

selectează un anumit rând în tabel

| Proprietăți              | Descriere                                                                                                                                                                     |
| ------------------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Rândul selectat implicit | valoarea implicită a rândului selectat poate fi indexul rândurilor, un identificator unic, sau orice altă proprietate care identifică în mod unic rândul pentru a fi selectat |

-   **clearSelection**

curățați selecția curentă din tabel

-   **setFilters**

aplică filtre la datele din tabel

| Proprietăți  | Descriere                               |
| ------------ | --------------------------------------- |
| Filtre       | condiții sau criterii de filtrare       |
| Mod filtru   | determină cum sunt aplicate filtrele    |

-   **ClearFilters**

șterge orice filtre aplicate în tabel

-   **setSort**

setează sortarea pentru tabel

| Proprietăți | Descriere                                                     |
| ----------- | ------------------------------------------------------------- |
| Coloana     | coloana pe care trebuie aplicată sortarea                     |
| Direcție    | direcția tipului, care poate fi ascendentă sau descendentă    |

### Utilizare majusculă

Apoi, vom demonstra cum să cartografiem datele din `**Upstash**` la componenta `**table**`.

#### Pasul 1 Adăugare acţiune

Să creăm un tabel în Upstash numit **`employee_profile`,** incluzând 4 coloane `name`, `level`, `age`, şi `married`.

Apoi putem crea o nouă acțiune pentru Upstash din lista de acțiuni în ILLA Cloud și numit upstash2.

Pentru a lista toate datele din angajat_profil, puneți codul snippet de mai jos în secțiunea de interogare Redis.

```sql
OBT angajat_profil
```

Deoarece datele de ieşire sunt o listă de perechi de dicţionar, vrem să le transformăm într-un array. În Transformer, activați-l și puneți codul de mai jos.

```jsx
const jsonString = data[0].result;

// Parse the JSON string into a JavaScript object
const jsonData = JSON.parse(jsonString);

// Access the "employees" property and map it to a new array
const mappedArray = jsonData.employees.map((employee) => ({
  name: employee.name,
  level: employee.level,
  age: employee.age,
  married: employee.married,
}));

return mappedArray;
```

Faceţi clic pe **`Salvează`** şi **`Run`** pentru a activa această acţiune.

#### Pasul 2 Adaugă Componente

Apoi, putem adăuga o componentă **`table`** la pânză.

În câmpul **`Sursa datelor`** din pagina Inspect după ce faceți clic pe componentă, introduceți `{{upstash2.data}}` pentru a prelua datele de la Upstash.

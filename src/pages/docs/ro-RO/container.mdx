## <h2 hidden>Container</h2>

Puteţi grupa componentele înrudite într-un card folosind componenta de container. Containerele suportă, de asemenea, vizualizări multiple care vă permit să separați conținutul în vizualizări distincte pe care utilizatorii le pot comuta între.

Funcţia containerelor ca orice altă componentă. Odată adăugat, puteți grupa componentele din interiorul acestuia apăsând D sau K și tragându-le pe Container.

### Mai multe vizualizări

Containerele suportă vizualizări multiple pentru gruparea conținutului. Puteți seta vizualizări în secțiunea **Vizualizare** din tab-ul **Inspectare**. Doar o singură vizualizare este vizibilă la un moment dat şi utilizatorii comută între ele schimbând vizualizarea curentă a componentei Container.

### Utilizați alte componente pentru a schimba vizualizările

Puteţi utiliza alte componente pentru a controla recipientele prin adăugarea unui manipulator de evenimente la alte componente. Sunt sprijinite următoarele metode:

1.  Setaţi vizualizarea curentă după cheie
2.  Setaţi vizualizarea curentă după index
3.  Arată următoarea vizualizare (vizibilă)
4.  Arată vizualizarea anterioară (vizibilă)

### Conectați o filă la un container

Puteți folosi o componentă de tab pentru a controla un container prin activarea **Link către un container** din fila **Inspectare**.

După conectarea la un recipient, etichetele din file se vor schimba în etichetele de vizualizări. Poți schimba vizualizările făcând clic pe file fără gestionari de evenimente.

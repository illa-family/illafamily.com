## <h2 hidden>Ce este ClickHouse?</h2>

ClickHouse este un sistem rapid de gestionare a bazei de date orientat către coloane, conceput pentru procesarea analitică online (OLAP). Este un proiect open-source care a fost creat de Yandex, compania rusească de motoare de căutare. ClickHouse este cunoscut pentru capacitatea sa de a prelucra cantități mari de date rapid și eficient, transformând-o într-o soluție ideală pentru analiza volumelor mari de date.

Cu Illa Cloud, puteţi desfăşura şi gestiona grupuri ClickHouse în cloud, fără să fiţi îngrijoraţi de infrastructura de bază şi de întreţinere.

Illa Cloud oferă o interfață web pentru gestionarea clusterelor ClickHouse, vă permite să monitorizați performanța clusterului, scara resurselor în sus sau în jos și să configurați copiile de rezervă și setările de securitate. De asemenea, puteți utiliza Illa Cloud pentru a efectua interogări pe datele dumneavoastră ClickHouse folosind un editor de interogări web.

### Creați ClickHouse

Există două modalități de a crea o resursă în Illa după ce te autentifici în contul tău Illa.

-   Creează în resurse

Conectează-te la contul tău Illa, selectează `**Resurse**` din partea de sus a paginii și apasă butonul `**Creează nou**`.

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

Selectați **`ClickHouse`** din lista de baze de date.

![action_list](https://cdn.illacloud.com/official-website/img/docs/action_list.png)

Conectați-vă la baza de date cu parametrii necesari descriși în `Setări conexiuni` de mai jos.

Faceți clic pe `**Test Connection**` pentru a vedea dacă ne putem conecta cu succes la baza de date. Dacă da, apasă pe **`Salvează Resurse`**, în caz contrar, verifică de două ori numele gazdei, portul, numele de utilizator și parola sunt corecte.

După crearea unei resurse, ClickHouse gata se va afișa după cum se arată.

![click_config](https://cdn.illacloud.com/official-website/img/docs/click_config.png)

-   Creează în Constructor

Conectează-te la contul tău Illa, creează un proiect în Illa Builder în pagina `**App**`, şi navighează la **`Lista de Acţiuni`** în partea de jos a paginii. Faceți clic pe `**nou**`, apoi selectați **`ClickHouse`** din lista de baze de date. Apoi, conectați-vă la baza de date cu parametrii necesari descriși în `Setări conexiuni` de mai jos.

Faceți clic pe `**Test Connection**` pentru a vedea dacă ne putem conecta cu succes la baza de date. Dacă da, apasă pe **`Salvează Resurse`**, în caz contrar, verifică de două ori numele gazdei, portul, numele de utilizator și parola sunt corecte.

### Setări conexiune

Aici trebuie să oferim informaţii pentru conectarea la baza de date ClickHouse.

![click_config](https://cdn.illacloud.com/official-website/img/docs/click_config.png)

| Proprietăți     | Necesar  | Descriere                                                                                                                                        |
| --------------- | -------- | ------------------------------------------------------------------------------------------------------------------------------------------------ |
| Nume            | necesar  | Numele pentru resurse la crearea de acțiuni în ILLA.                                                                                             |
| Hostname        | necesar  | Adresa URL sau adresa IP a bazei de date                                                                                                         |
| Portul          | necesar  | Numărul de port al serverului gazdă pe care ar trebui să-l utilizați pentru conectare. Dacă nu specificați un port, portul implicit este '9440'. |
| Baza de date    | necesar  | Numele bazei de date                                                                                                                             |
| Nume            | necesar  | numele de utilizator pe care doriți să îl utilizați atunci când vă conectați la serverul ClickHouse.                                             |
| Parolă          | necesar  | Utilizați această parolă pentru autentificare.                                                                                                   |
| Opțiuni SSL     | opţional | decide cât de mare este prioritizată o conexiune SSL TCP/IP în timpul negocierilor cu serverul.                                                  |

### Configure ClickHouse

| Proprietăți      | Descriere                                                                                   |
| ---------------- | ------------------------------------------------------------------------------------------- |
| Interogare SQL   | comenzi sql pentru a prelua (select), inserare, actualizare, ștergere date din baza de date |
| Transformator    | Transformarea datelor în stilul care îți place să folosești JavaScript                      |

Exemplu de utilizare:

-   Selectare

```sql
SELECTARE * DE LA utilizatorii ORDUL DUPĂ Nume;
```

-   Insert

```sql
INSERT INTO users

	(firstname, lastname, email)

VALUES

	(

		{{ NewUser.data.firstNameInput }},

		{{ NewUser.data.lastNameInput }},

		{{ NewUser.data.emailInput }}

	);
```

-   Actualizare

```sql
UPDATE users
  SET firstname = '{{form1.updatedProfile.first}}'
  WHERE id = {{ form1.updatedProfile.uid}};
```

-   Ștergere

```sql
DELETE FROM users WHERE id = {{ form1.updatedProfile.uid }};
```

## <h2 hidden>Ce este Firebase?</h2>

Firebase este o platformă cuprinzătoare furnizată de Google care oferă o gamă largă de instrumente și servicii pentru dezvoltarea și gestionarea aplicațiilor web și a celor mobile. Acesta oferă dezvoltatorilor o infrastructură backend scalabilă și sigură, împreună cu o serie de caracteristici și funcționalități preconstruite pentru a accelera procesul de dezvoltare. Odată cu integrarea Firebase pe ILLA Cloud, dezvoltatorii își pot îmbunătăți aplicațiile cu soluții fiabile și scalabile de stocare; să le permită să se concentreze asupra construirii de experiențe inovatoare și bazate pe date.

### Creare API Firebase

Există două modalități de a crea o resursă în Illa după ce te autentifici în contul tău Illa.

-   Creează în resurse

Conectează-te la contul tău Illa, selectează `**Resurse**` din partea de sus a paginii și apasă butonul `**Creează nou**`.

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

Selectați **`Firebase`** din lista API.

![action_list](https://cdn.illacloud.com/official-website/img/docs/action_list.png)

Conectați-vă la baza de date cu parametrii necesari descriși în `Setări conexiuni` de mai jos.

Faceți clic pe `**Test Connection**` pentru a vedea dacă ne putem conecta cu succes la baza de date. Dacă da, apasă pe **`Salvează Resurse`**, în caz contrar, verifică de două ori numele gazdei, portul, numele de utilizator și parola sunt corecte.

Dupa crearea unei resurse, Firebase gata se va afisa ca afisat.

![firebase_ex_ready](https://cdn.illacloud.com/official-website/img/official-site/intergration/firebase_ex_ready.jpg)

-   Creează în Constructor

Conectează-te la contul tău Illa, creează un proiect în Illa Builder pe pagina `**App**`, şi navighează la **`Lista de Acţiuni`** în partea de jos a paginii. Faceți clic pe `**nou**`, apoi selectați **`Firebase`** din lista de baze de date. Apoi, conectați-vă la baza de date cu parametrii necesari descriși în `Setări conexiuni` de mai jos.

Faceți clic pe `**Test Connection**` pentru a vedea dacă ne putem conecta cu succes la baza de date. Dacă da, apasă pe **`Salvează Resurse`**, în caz contrar, verifică de două ori numele gazdei, portul, numele de utilizator și parola sunt corecte.

### Setări conexiune

În acest caz, trebuie să furnizăm informaţii pentru conectarea la Firebase API.

![firebase_config](https://cdn.illacloud.com/official-website/img/official-site/intergration/firebase_config.jpg)

| Proprietăți                               | Necesar  | Descriere                                                                                                     |
| ----------------------------------------- | -------- | ------------------------------------------------------------------------------------------------------------- |
| Nume                                      | necesar  | Numele pentru resurse la crearea de acțiuni în ILLA.                                                          |
| Firebase Baza URL (Automatic Translation) | opţional | punctul final sau adresa URL unde este găzduită baza de date de Firebase Realtime                             |
| Firestare proiect ID                      | opţional | un identificator unic atribuit fiecărui proiect Firebase care utilizează serviciul Firebase                   |
| Cheie privată                             | necesar  | o acreditare de securitate utilizată pentru autentificarea și accesarea programatică a serviciilor Firebase   |

### Configurare Firebase

#### Prezentare generală

| Tip de serviciu | Firebase Auth                                                                 | Restabilire                   | Baza de date                   |
| --------------- | ----------------------------------------------------------------------------- | ----------------------------- | ------------------------------ |
| Tip de acțiune  | Obține utilizatorul prin UID                                                  | Interogare Firebase,          | Interogare Bază de date        |
|                 | Obțineți utilizatorul prin e-mail                                             | Inserare document             | Setare date                    |
|                 | Obține utilizatorul după numărul de telefon                                   | Actualizare document          | Actualizare date               |
|                 | Creează un utilizator                                                         | Obține documentul după ID     | Adăugare date la o listă       |
|                 | Actualizează un utilizator                                                    | Șterge un document            |                                |
|                 | Ștergeți un utilizator                                                        | Obține Colecții               |                                |
|                 | Listează utilizatorul                                                         | Interogare Grup Colectare     |                                |
| Transformator   | Transformarea datelor în stilul care îți place să folosești JavaScript        |                               |                                |

#### Obține utilizatorul prin UID

regăsiți informațiile utilizatorului prin specificarea identificatorului unic al utilizatorului (UID). Vă permite să preluați detalii de utilizator cum ar fi numele afișat, e-mailul, numărul de telefon și atributele personalizate asociate cu utilizatorul.

** Input **

| Proprietăți | Necesar | Descriere                                                                                                                    |
| ----------- | ------- | ---------------------------------------------------------------------------------------------------------------------------- |
| UID         | necesar | Identificator de utilizator. Este un identificator unic atribuit fiecărui utilizator din sistemul de autentificare Firebase. |

#### Obțineți utilizatorul prin e-mail

recuperează informaţiile utilizatorului prin specificarea adresei de e-mail a utilizatorului. Vă permite să preluaţi detalii de utilizator similare metodei bazate pe UID, dar folosind e-mailul ca identificator.

** Input **

| Proprietăți | Necesar | Descriere                                                                    |
| ----------- | ------- | ---------------------------------------------------------------------------- |
| email       | necesar | e-mailul înregistrat asociat cu utilizatorul pe care doriți să îl recuperați |

#### Obține utilizatorul după numărul de telefon

Preia informaţiile utilizatorului specificând numărul de telefon al utilizatorului. Vă permite să preluaţi detalii de utilizator similare metodei bazate pe UID, dar folosind numărul de telefon ca identificator.

** Input **

| Proprietăți      | Necesar | Descriere                                                                             |
| ---------------- | ------- | ------------------------------------------------------------------------------------- |
| număr de telefon | necesar | numărul de telefon înregistrat asociat cu utilizatorul pe care doriţi să-l recuperaţi |

#### Creează un utilizator

creează un utilizator nou în sistemul de autentificare Firebas. Puteţi furniza e-mailul utilizatorului, parola, afişarea numelui şi alte informaţii relevante în timpul procesului de creare a utilizatorului.

** Input **

| Proprietăți       | Necesar | Descriere                                          |
| ----------------- | ------- | -------------------------------------------------- |
| Obiect utilizator | necesar | un obiect care conține informațiile utilizatorului |

#### Actualizează un utilizator

actualizează atributele și proprietățile unui utilizator existent. Puteți modifica numele afișat al utilizatorului, e-mailul, numărul de telefon, parola și alte atribute personalizate asociate cu utilizatorul.

** Input **

| Proprietăți       | Necesar | Descriere                                                                                                                    |
| ----------------- | ------- | ---------------------------------------------------------------------------------------------------------------------------- |
| UID               | necesar | Identificator de utilizator. Este un identificator unic atribuit fiecărui utilizator din sistemul de autentificare Firebase. |
| Obiect utilizator | necesar | un obiect care conține informațiile utilizatorului                                                                           |

#### Ștergeți un utilizator

şterge un utilizator din sistemul de autentificare Firebas. (Automatic Translation) Această acţiune elimină definitiv utilizatorul şi datele asociate acestuia din sistem.

** Input **

| Proprietăți | Necesar | Descriere                                                                                                                    |
| ----------- | ------- | ---------------------------------------------------------------------------------------------------------------------------- |
| UID         | necesar | Identificator de utilizator. Este un identificator unic atribuit fiecărui utilizator din sistemul de autentificare Firebase. |

#### Listează utilizatorii

regăsiți o listă cu toți utilizatorii înregistrați în sistemul de autentificare Firebase. Oferă rezultate paginate, care îți permit să iterezi prin lista de utilizatori și să recuperezi detalii specifice utilizatorului.

** Input **

| Proprietăți                | Necesar  | Descriere                                                                                              |
| -------------------------- | -------- | ------------------------------------------------------------------------------------------------------ |
| Lungimea listei            | opţional | numărul de utilizatori pe care doriți să îi recuperați într-o singură solicitare                       |
| Token-ul paginii următoare | opţional | paginare atunci când sunt mai mulţi utilizatori decât se poate returna într-un singur răspuns          |

#### Interogare Firebase

efectuează interogări în baza de date folosind diferite condiţii şi filtre

** Input **

| Proprietăți             | Necesar  | Descriere                                                                                |
| ----------------------- | -------- | ---------------------------------------------------------------------------------------- |
| Colectare               | necesar  | poate folosi meniul derulant pentru a selecta sau utiliza un id brut                     |
| unde                    | necesar  | adaugă condiții la interogare                                                            |
| limită                  | opţional | limitează numărul de documente returnate de interogare.                                  |
| Ordonare după           | opţional | specificați câmpul prin care doriți să ordonați rezultatele interogării                  |
| Direcție ordine         | opţional | direcția comenzii în rezultatele interogării                                             |

#### Inserare document

creează un nou document într-o colecție sau o subcolecție în Firestore. Furnizați datele care urmează să fie stocate în document, iar Firebase atribuie un identificator unic (documentul ID).

** Input **

| Proprietăți   | Necesar  | Descriere                                                                                                                                     |
| ------------- | -------- | --------------------------------------------------------------------------------------------------------------------------------------------- |
| Colectare     | necesar  | specifică numele colecției în care doriți să introduceți documentul. poate folosi meniul derulant pentru a selecta sau utiliza un id brut     |
| ID document   | opţional | specifică un identificator personalizat pentru documentul care este inserat                                                                   |
| valoare       | necesar  | datele reale pe care doriţi să le inseraţi în document                                                                                        |

#### Actualizare document

actualizează un document existent în Firestore. Specificați ID-ul documentului și furnizați datele actualizate, care vor suprascrie câmpurile existente ale documentului.

** Input **

| Proprietăți   | Necesar  | Descriere                                                                                                                                           |
| ------------- | -------- | --------------------------------------------------------------------------------------------------------------------------------------------------- |
| Colectare     | necesar  | specifică numele colecției în care doriți să actualizați documentul. poate folosi meniul derulant pentru a selecta sau utiliza un id brut           |
| ID document   | opţional | specificați un identificator personalizat pentru documentul care este actualizat                                                                    |
| valoare       | necesar  | datele reale pe care doriți să le actualizați în document                                                                                           |

#### Interogare Grup Colectare

interogare documente pe mai multe colecții cu același nume. Poți efectua interogări complexe și să recuperezi documente care se potrivesc din colecții diferite care împart același nume.

** Input **

| Proprietăți             | Necesar  | Descriere                                                                                |
| ----------------------- | -------- | ---------------------------------------------------------------------------------------- |
| Colectare               | necesar  | poate folosi meniul derulant pentru a selecta sau utiliza un id brut                     |
| unde                    | necesar  | adaugă condiții la interogare                                                            |
| limită                  | opţional | limitează numărul de documente returnate de interogare.                                  |
| Ordonare după           | opţional | specificați câmpul prin care doriți să ordonați rezultatele interogării                  |
| Direcție ordine         | opţional | direcția comenzii în rezultatele interogării                                             |

#### Interogare bază de date

interoga baza de date Realtime pentru a prelua date care îndeplinesc criterii specifice. Puteţi defini filtre, sortare şi alte condiţii pentru a prelua un subset de date dintr-o anumită locaţie din baza de date.

** Input **

| Proprietăți   | Necesar | Descriere                                          |
| ------------- | ------- | -------------------------------------------------- |
| Date ref      | necesar | o trimitere la o anumită locație din baza de date. |

#### Setare date

** Input **

| Proprietăți     | Necesar | Descriere                                                                                                           |
| --------------- | ------- | ------------------------------------------------------------------------------------------------------------------- |
| Date ref        | necesar | o trimitere la o anumită locație din baza de date.                                                                  |
| Obiect de setat | necesar | obiectul conține datele pe care dorim să le scriem în baza de date, inclusiv numele, vârsta și câmpurile de e-mail. |

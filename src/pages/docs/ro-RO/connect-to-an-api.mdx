## <Heading hidden>Conectați-vă la un API</Heading>

Conectează-te la orice REST, GraphQL API.

Puteți conecta ILLA la cele mai multe API-uri folosind integrările noastre native. Dacă trebuie să te conectezi la un API intern sau dacă nu există o integrare acceptată, vă puteți conecta folosind integrarea noastră REST API sau GraphQL Integrare.

### 1.  Creează o resursă API

Resursele reprezintă conexiuni la surse de date și sunt ceea ce interogați în cadrul aplicațiilor ILLA.Loghează-te la ILLA și navighează la fila **Resurse** > apasă **Creează nou** și selectează API la care vrei să te conectezi; precum **REST API**. Aceasta deschide modul **Configurați o resursă** unde adăugați informațiile dvs. de conexiune.

<img
  src='https://cdn.illacloud.com/official-website/img/docs/connect/connect-api/config.png'
  alt=""
/>

Cel puțin, aveți nevoie de URL-ul de bază al API la care doriți să vă conectați. De asemenea, puteți trece de parametrii URL-ului, anteturi, etc. Dacă API-ul dvs. necesită autentificare, puteți configura acest lucru și pe această pagină. De exemplu, OAuth2.0 necesită un URL de autorizare, acces token URL, etc. După ce introduci detaliile conexiunii, apasă **Salvează resursa.**

### 2.  Interogare API

În editorul de aplicații, faceți clic pe **+ Nou** în **Lista de acțiuni** > selectați **REST API** și alegeți un REST API existent sau creați unul nou > faceți clic pe **creare acțiune** și actualizați acțiunile cu parametri URL suplimentari, anteturi, etc.

## <h2 hidden>Ce este imagine?</h2>

Componenta de imagine din ILLA Cloud este un instrument versatile care permite utilizatorilor să adauge și să afișeze imagini în aplicațiile lor. Oferă diverse funcționalități pentru a îmbunătăți experiența vizuală pentru utilizatori.

### Proprietăți

| Proprietăți             | Descriere                                                                                                                                        |
| ----------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------ |
| Sursa imaginii          | specifica sursa imaginii care va fi afişată                                                                                                      |
| Text alternativ         | text alternativ pentru imagine. Este afișat atunci când imaginea nu poate fi încărcată sau pentru utilizatorii cu deficiențe de vedere           |
| Tip scară               | cum este scalată imaginea în interiorul dimensiunilor componentei, inclusiv recipientul, acoperișul, umplerea, niciunul, scala-în jos.           |
| Gestionar de evenimente | permite utilizatorilor să definească acţiunile declanşate de interacţiunile utilizatorului cu imaginea                                           |
| Tooltip                 | adăugați un indiciu de unealtă sau un text descriptiv care apare atunci când utilizatorul planează peste imagine                                 |
| Hidden                  | controlează vizibilitatea componentei de imagini.                                                                                                |
| Styles                  | permite utilizatorilor să îşi personalizeze înfăţişarea. Utilizatorii pot aplica stiluri precum raza                                             |

### Metodă

Puteţi utiliza alte componente pentru a controla componenta. Sprijinim următoarele metode:

-   **setImageUrl**

schimbă dinamic sursa sau URL-ul imaginii.

| Proprietăți  | Descriere            |
| ------------ | -------------------- |
| URL imagine  | URL-ul sursă imagine |

### Gestionar de evenimente

| Eveniment | Descriere                                                                               |
| --------- | --------------------------------------------------------------------------------------- |
| Click     | execută acțiuni specifice sau execută cod atunci când componenta imagine este selectată |

### Date

Componenta are niște date utilizate în mod obișnuit, care pot fi apelate prin **`{{componentName.propertyName}}`** în aplicație.

| Nume proprietate  | Descriere                                                                                                                                                     |
| ----------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| altText           | o alternativă bazată pe text care descrie conținutul sau scopul imaginii                                                                                      |
| AfișeazăNume      | O valoare a șirului care reprezintă numele sau titlul imaginii.                                                                                               |
| evenimente        | Un array sau obiect care definește manipulatorii de evenimente asociate componentei imagine, cum ar fi clic                                                   |
| hidden            | O valoare booleană care determină dacă componenta imaginii este ascunsă de interfața utilizatorului.                                                          |
| imageSrc          | sursa sau locația imaginii care va fi afișată în componentă                                                                                                   |
| obiect            | modul în care imaginea este montată sau scalată în dimensiunile componentei                                                                                   |
| radius            | aplică colțuri rotunjite componentei de imagine.                                                                                                              |
| tooltipText       | valoare-șir, care reprezintă vârful de unealtă sau textul care planează pentru a fi afișat atunci când utilizatorul planează deasupra componentei de imagine. |

Exemplu: {{image1.displayName}}

### Utilizare majusculă

Apoi, vom demonstra cum să controlăm afişarea imaginii cu un buton.

#### Pasul 1 Adaugă Componente

Adăugați o componentă **`Image`** și o componentă **`button`** pe pânză de pânză.

Puteți seta \`sursa imaginii" componentei imaginii la orice adresă de imagine pe care doriți să o afișați. Aici vom folosi imaginea implicită ca exemplu de început și eticheta butonul ca "Modifică" cum este afișat mai jos

![image_layout](https://cdn.illacloud.com/official-website/img/official-site/components/image_layout.jpg)

#### Pasul 2 Configurați componenta

Pentru componenta **`button`**, o putem configura pentru a seta sursa de imagine a componentei **`imagine`** într-o nouă adresă a imaginii.

1.  Faceți clic pe componenta `**button**` pentru a deschide pagina sa de inspectare. Sub **`Manipulatorul de evenimente`**, apasă `**+ New**`.
2.  În gestionarul de editare a evenimentului, selectați **`Componenta Control`** în acțiune, și selectați componenta **`image1`** pe care doriți să o schimbați ca țintă a evenimentului.
3.  Alege acțiunea **`setImageUrl`**
4.  Pune noua adresă a imaginii în câmpul **`Value`**

![image_config](https://cdn.illacloud.com/official-website/img/official-site/components/image_config.jpg)

Acum că atunci când faceți clic pe butonul "Redare", imaginea ar trebui să schimbe

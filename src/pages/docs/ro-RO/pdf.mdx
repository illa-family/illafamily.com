## <h2 hidden>Ce este PDF?</h2>

Componenta PDF este o caracteristică sau element care permite utilizatorilor sa vizualizeze si sa interactioneze cu documentele PDF din aplicatie. Oferă funcționalitate pentru redarea, navigarea și manipularea fișierelor PDF, permițând utilizatorilor să acceseze și să lucreze perfect cu conținut PDF.

### Proprietăți

| Proprietăți          | Descriere                                                                                              |
| -------------------- | ------------------------------------------------------------------------------------------------------ |
| Sursă PDF            | specificați sursa pentru fișierul pdf care va fi afișat                                                |
| Arată bara de unelte | controlează vizibilitatea barei de instrumente în cadrul componentei PDF                               |
| Tooltip              | adaugă un pont de unelte sau un text descriptiv care apare atunci când utilizatorul planează peste pdf |
| Scalează pagina la   | determină comportamentul de scalare al paginilor PDF în cadrul componentei                             |
| Hidden               | determină vizibilitatea componentei PDF                                                                |

### Metodă

Puteţi utiliza alte componente pentru a controla componenta. Sprijinim următoarele metode:

-   **setFileUrl**

schimbă dinamic sursa sau URL-ul fișierului pdf.

| Proprietăți | Descriere        |
| ----------- | ---------------- |
| Sursă PDF   | URL-ul sursă pdf |

### Date

Componenta are niște date utilizate în mod obișnuit, care pot fi apelate prin **`{{componentName.propertyName}}`** în aplicație.

| Nume proprietate | Descriere                                                                                                                                      |
| ---------------- | ---------------------------------------------------------------------------------------------------------------------------------------------- |
| AfișeazăNume     | O valoare a șirului care reprezintă numele sau titlul pdf.                                                                                     |
| înălțime         | dimensiunea verticală sau dimensiunea componentei PDF                                                                                          |
| hidden           | O valoare booleană care determină dacă componenta pdf este ascunsă de interfața utilizatorului.                                                |
| scaleMod         | specifică modul în care documentul PDF este scalat sau se potrivește în zona de afișare disponibilă a componentei                              |
| showToolBar      | dacă este afișată o bară de instrumente sau un panou de control împreună cu componenta PDF                                                     |
| tooltipText      | valoare șir care reprezintă vârful de unealtă sau textul de planare pentru a fi afișat atunci când utilizatorul planează peste componenta pdf. |
| URL              | sursa fișierului PDF care va fi afișată în componentă                                                                                          |
| width            | dimensiunea sau dimensiunea orizontală a componentei PDF                                                                                       |

Exemplu: {{pdf1.displayName}}

### Utilizare majusculă

Apoi, vom demonstra cum să controlăm afișajul pdf cu un buton.

#### Pasul 1 Adaugă Componente

Adăugați o componentă **`PDF`** și o componentă **`button`** pe pânză de pânză.

![pdf_layout](https://cdn.illacloud.com/official-website/img/docs/pdf_layout.jpg)

#### Pasul 2 Configurați componenta

Pentru componenta **`button`**, o putem configura pentru a seta sursa de imagine a componentei **`PDF`** la o adresă pdf.

1.  Faceți clic pe componenta `**button**` pentru a deschide pagina sa de inspectare. Sub **`Manipulatorul de evenimente`**, apasă `**+ New**`.
2.  În gestionarul de editare a evenimentului, selectați **`Componenta Control`** în acțiune, şi selectaţi componenta **`PDF`** pe care doriţi să o schimbaţi ca ţintă a evenimentului.
3.  Alege acțiunea **`setFileUrl`**
4.  Pune noua adresă a imaginii în câmpul **`Value`**

![pdf_config](https://cdn.illacloud.com/official-website/img/docs/pdf_config.jpg)

#### Etapa 3 Test

Acum că atunci când faceți clic pe butonul "Redare", imaginea ar trebui să schimbe

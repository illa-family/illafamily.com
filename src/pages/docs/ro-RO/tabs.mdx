## <h2 hidden>Ce este Tabs?</h2>

Componenta de file din ILLA Cloud este o caracteristică versatilă care permite utilizatorilor să se organizeze și să prezinte conținut într-un layout de tabel. Permite crearea mai multor file în interiorul unui singur container, permițând utilizatorilor să schimbe între diferite secțiuni de conținut cu ușurință. Componenta de file oferă o interfață ușor de utilizat pentru navigarea prin informații conexe, îmbunătățind experiența generală a utilizatorului. Cu opțiuni personalizabile de stilizare și posibilitatea de a adăuga diferite tipuri de conținut, cum ar fi text, imagini sau componente încorporate, componenta de file din ILLA Cloud oferă flexibilitate și îmbunătățește organizarea și prezentarea datelor.

### Proprietăți

| Proprietăți                   | Descriere                                                                                                                                                                    |
| ----------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Link către un container       | leagă componenta de file de un anumit container, activând navigarea fără întreruperi între file și conținutul asociat.                                                       |
| Gestionar de evenimente       | detectare și răspuns la acțiuni sau evenimente specifice ale utilizatorilor, cum ar fi clicuri, taste sau modificări de valoare.                                             |
| Dezactivat                    | dezactivează anumite file, prevenind interacțiunea utilizatorilor cu ei                                                                                                      |
| Tooltip                       | informații suplimentare sau context despre fiecare filă                                                                                                                      |
| Aspect                        | oferă opțiuni de aspect diferite, permițându-vă să aranjați filele orizontal sau vertical în funcție de preferințele de design sau de spațiul disponibil.                    |
| Align                         | aliniază componenta File la diferite poziții din containerul părinte, cum ar fi stânga, centrul sau alinierea dreaptă                                                        |
| Hidden                        | ascunde anumite file programatic.                                                                                                                                            |
| Culori text                   | culori text personalizabile, care vă permit să stilizați etichetele filelor cu culori diferite pentru a se potrivi cu design-ul sau brandingul aplicației.                   |

### Date

Componenta are niște date utilizate în mod obișnuit, care pot fi apelate prin **`{{componentName.propertyName}}`** în aplicație.

| Nume proprietate             | Descriere                                                                                                                           |
| ---------------------------- | ----------------------------------------------------------------------------------------------------------------------------------- |
| align                        | alinierea taburilor din cadrul componentei, cum ar fi stânga, centrul sau alinierea dreaptă                                         |
| culori                       | definește culoarea textului în componentă                                                                                           |
| currentIndex                 | reprezintă indexul filei selectate în prezent din lista de taburi                                                                   |
| tastatură curentă            | identificator unic sau cheie a filei selectate                                                                                      |
| AfișeazăNume                 | reprezintă numele sau eticheta atribuită componentei                                                                                |
| hidden                       | valoare boolean care determină dacă componenta filelor este vizibilă sau ascunsă                                                    |
| linkWidgetDisplayName        | afișează numele sau eticheta widget-ului link-ului asociat cu o anumită filă                                                        |
| navigateContainer            | container sau destinație unde conținutul filei ar trebui să fie afișat sau navigat la                                               |
| Lista                        | lista de taburi, inclusiv numele de afișare, cheile și alte informații relevante                                                    |
| tabPosition                  | poziția benzii filei, cum ar fi partea de sus, jos, stânga sau dreapta                                                              |
| tooltipText                  | text afișat ca un pont de unelte atunci când utilizatorul planează peste o anumită filă                                             |
| Vizualizare                  | vizualizări sau conținut asociat cu fiecare filă, activând redarea de conținut diferit pe baza filei selectate                      |

Exemplu: {{tabs1.displayName}}

### Utilizare majusculă

Vom demonstra cum să folosim filele componente combinate cu o componentă container pentru a afișa cele trei vizualizări.

#### Pasul 1 Adaugă și configurează Componentele

Adăugați o componentă **`container`** și o componentă **`tabs`**. În pagina `inspectează` a componentei de taburi, întoarceți **`Link către un container`** pe și alegeți containerul cu care doriți să faceți legătura pentru `**Container**`.

-   Vezi 1:

Aceasta este pagina principală, astfel punem o componentă **`text`** cu "Acasă" în colțul din stânga sus al containerului.

Afișează o componentă **`text`** "Salut! Bine ați venit la website-ul meu personal" în mijloc și o componentă **`button`** etichetată "Aflați mai multe despre mine" sub el.

Pentru componenta **`button`**, o putem configura pentru a merge la următoarea vizualizare (view2) dacă se dă click.

1.  Faceți clic pe componenta **`button`** pentru a deschide pagina sa de inspecție. Sub **`Manipulatorul de evenimente`**, apasă `**+ New**`.
2.  În gestionarul de editare a evenimentului, selectați **`Componenta Control`** în acțiune, şi selectaţi componenta **`container`** pe care doriţi să o actualizaţi ca ţintă a evenimentului.
3.  Alege metoda **`showNextView`**.

![tab_view1](https://cdn.illacloud.com/official-website/img/docs/tab_view1.jpg)

-   Vedere 2:

Aceasta este pagina aproximativă, astfel punem o componentă **`text`** cu "About me" în colțul din stânga sus al containerului.

Afiseaza "name", "age", şi informaţii despre "gen" cu componentele **`text`** şi o poză de profil cu dumneavoastră sau cu oricine doriţi să utilizaţi cu componenta `**image**`.

Adăugaţi o componentă **`button`** etichetată "Faceţi prieteni cu mine".

Pentru componenta **`button`**, o putem configura pentru a merge la următoarea vizualizare (view3) dacă se dă click.

1.  Faceți clic pe componenta **`button`** pentru a deschide pagina sa de inspecție. Sub **`Manipulatorul de evenimente`**, apasă `**+ New**`.
2.  În gestionarul de editare a evenimentului, selectați **`Componenta Control`** în acțiune, şi selectaţi componenta **`container`** pe care doriţi să o actualizaţi ca ţintă a evenimentului.
3.  Alege metoda **`showNextView`**.

![tab_view2](https://cdn.illacloud.com/official-website/img/docs/tab_view2.jpg)

-   Vezi 3

Aceasta este pagina formularului, vizitatorul poate completa formularul cu informaţiile lor pentru a trimite o cerere de prietenie proprietarului.

Există o componentă de formă în mijlocul containerului cu componente de intrare etichetate ca "Numele tău", "Vârsta ta" şi "Gen".

Schimbă textul butonului "Trimite " în colțul din dreapta jos la "Invitație".

![tab_view3](https://cdn.illacloud.com/official-website/img/docs/tab_view3.jpg)

Pentru componenta "Invitație" **`button`**, o putem configura pentru a trimite informația în formular, dacă faceți clic pe aceasta.

1.  Faceți clic pe componenta **`button`** pentru a deschide pagina sa de inspecție. Sub **`Manipulatorul de evenimente`**, apasă `**+ New**`.
2.  În gestionarul de editare a evenimentului, selectați **`Componenta Control`** în acțiune, şi selectaţi componenta **`form`** pe care doriţi să o actualizaţi ca ţintă a evenimentului.
3.  Alege metoda **`submit`**.

Pentru componenta **`form`**, o putem configura pentru a afișa notificarea dacă trimite.

1.  Faceţi clic pe componenta **`form`** pentru a deschide pagina sa de inspectare. Sub **`Manipulatorul de evenimente`**, apasă `**+ New**`.
2.  În gestionarul de editare a evenimentului, selectați **`submit`** în eveniment, selectați **`Show notification`** în **`action`**, Pune "Trimite cu succes" pentru titlu, "Cerere de prietenie trimisă" pentru Descriere şi selectează `Success` pentru **`Type`**. Vrem ca notificarea să fie afișată pentru 2s așa că pune `{{2000}}` pentru o perioadă de timp.

![tab_form_event](https://cdn.illacloud.com/official-website/img/docs/tab_form_event.jpg)

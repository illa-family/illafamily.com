## <h2 hidden>Ce este Amazon S3?</h2>

Amazon S3 este un serviciu de stocare cloud extrem de scalabil și sigur oferit de Amazon Web Services (AWS). Atunci când se combină cu ILLA Cloud, dezvoltatorii pot integra perfect Amazon S3 în aplicațiile lor, utilizând capacitățile sale robuste de stocare pentru a stoca și a prelua datele în mod eficient. ILLA Cloud simplifică procesul de integrare, permițând dezvoltatorilor să se conecteze perfect la Amazon S3 și să utilizeze funcțiile sale puternice, cum ar fi criptarea datelor, versiunea, controalele de acces şi scalabilitatea fără întreruperi. Odată cu integrarea Amazon S3 pe ILLA Cloud, dezvoltatorii își pot îmbunătăți aplicațiile cu soluții de stocare fiabile și scalabile, să le permită să se concentreze asupra construirii de experiențe inovatoare și bazate pe date.

### Creare API Amazon S3

Există două modalități de a crea o resursă în Illa după ce te autentifici în contul tău Illa.

-   Creează în resurse

Conectează-te la contul tău Illa, selectează `**Resurse**` din partea de sus a paginii și apasă butonul `**Creează nou**`.

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

Selectați **`Amazon S3`** din lista API.

![action_list](https://cdn.illacloud.com/official-website/img/docs/action_list.png)

Conectați-vă la baza de date cu parametrii necesari descriși în `Setări conexiuni` de mai jos.

Faceți clic pe `**Test Connection**` pentru a vedea dacă ne putem conecta cu succes la baza de date. Dacă da, apasă pe **`Salvează Resurse`**, în caz contrar, verifică de două ori numele gazdei, portul, numele de utilizator și parola sunt corecte.

După crearea unei resurse, Amazon S3 gata se va afișa ca afișat.

![amazon_ex_ready](https://cdn.illacloud.com/official-website/img/docs/assemble/amazon_ex_ready.jpg)

-   Creează în Constructor

Conectează-te la contul tău Illa, creează un proiect în Illa Builder pe pagina `**App**`, şi navighează la **`Lista de Acţiuni`** în partea de jos a paginii. Faceţi clic pe `**nou**`, apoi selectaţi **`Amazon S3`** din lista de baze de date. Apoi, conectați-vă la baza de date cu parametrii necesari descriși în `Setări conexiuni` de mai jos.

Faceți clic pe `**Test Connection**` pentru a vedea dacă ne putem conecta cu succes la baza de date. Dacă da, apasă pe **`Salvează Resurse`**, în caz contrar, verifică de două ori numele gazdei, portul, numele de utilizator și parola sunt corecte.

### Setări conexiune

Aici trebuie să furnizăm informaţii pentru conectarea la Amazon S3 API.

![amazon_config](https://cdn.illacloud.com/official-website/img/docs/assemble/amazon_config.jpg)

| Proprietăți                       | Necesar  | Descriere                                                                                                                                                                                                                                    |
| --------------------------------- | -------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Nume                              | necesar  | Numele pentru resurse la crearea de acțiuni în ILLA.                                                                                                                                                                                         |
| Nume găleată                      | opţional | un identificator unic, care reprezintă un container de stocare în Amazon S3. Oferă o separare logică pentru organizarea și accesarea fișierelor tale                                                                                         |
| S3 ACL pentru fişierele încărcate | opţional | îți permite să definești permisiunile de acces la grătare pentru fișierele din găleată. Specifică cine poate efectua anumite operațiuni, cum ar fi citirea, scrierea sau ștergerea fișierelor.                                               |
| Regiune                           | necesar  | locația geografică unde este stocată găleata Amazon S3. Acesta determină locația fizică a datelor tale și poate avea un impact asupra unor factori precum latența și costurile transferului de date.                                         |
| Obiectiv personalizat S3          | opţional | vă permite să configurați un anumit obiectiv URL pentru accesarea resurselor Amazon S3. Poate fi util pentru scenariile în care doriţi să utilizaţi un domeniu personalizat sau să integraţi cu un CDN (Reţea de Livrare a conţinutului).    |
| Cheie de acces                    | necesar  | acreditări care autentifică cererile dvs. de a accesa Amazon S3                                                                                                                                                                              |
| Cheie de acces secret             | necesar  | generat atunci când creezi un utilizator IAM (Administrația Internațională și de Acces) sau un AWS (Amazon Web Services) accesează perechea cheie.                                                                                           |

### Configurare Amazon S3

#### Prezentare generală

| Numele metodei | Descriere                                                                                                                                                  |
| -------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Tip de acțiune | listează toate obiectele într-o găleată, citește un obiect, descarcă un obiect, șterge un obiect, șterge mai multe obiecte, încarcă date multiple          |
| Nume găleată   | un identificator unic, care reprezintă un container de stocare în Amazon S3.                                                                               |
| Transformator  | Transformarea datelor în stilul care îți place să folosești JavaScript                                                                                     |

#### Listează toate obiectele într-o găleată

recuperează o listă cu toate obiectele stocate într-o anumită găleată.

** Input **

| Proprietăți                          | Necesar  | Descriere                                                                                              |
| ------------------------------------ | -------- | ------------------------------------------------------------------------------------------------------ |
| Prefix pentru filtrarea rezultatelor | opţional | filtrează rezultatele pentru a include doar obiecte a căror cheie (nume) începe cu un prefix specific  |
| Delimitator                          | opţional | specifică un caracter care separă ierarhiile cheii de obiect                                           |
| Generează URL Semnat                 | opţional | dacă se generează un URL cu durată limitată care oferă acces temporar la un anumit obiect              |
| Taste maxime                         | opţional | limitează numărul maxim de obiecte returnate în răspuns                                                |

De exemplu, pentru a obține toate obiectele în baza de date.

\*\* Rezultat

returnează o serie de obiecte, fiecare incluzând cel puţin o proprietate `objectKey`.

![amazon_list_all_data](https://cdn.illacloud.com/official-website/img/docs/assemble/amazon_list_all_data1.jpg)

Utilizaţi `{{s31.data}}` pentru a primi toate datele returnate.

![amazon_list_all_code](https://cdn.illacloud.com/official-website/img/docs/assemble/amazon_list_all_code1.jpg)

#### Citește un obiect

recuperează conținutul unui anumit obiect stocat într-o găleată

** Input **

| Proprietăți           | Necesar  | Descriere                                                                             |
| --------------------- | -------- | ------------------------------------------------------------------------------------- |
| Cheie obiect          | necesar  | numele sau calea obiectului în interiorul găleții                                     |
| Generează URL Semnat  | opţional | generează o adresă URL limitată în timp care oferă acces temporar la un anumit obiect |

De exemplu, pentru a obţine conţinutul documentului "1111.txt", puneţi `1111.txt` pentru **`Cheia Object`**.

#### Încarcă date

încărcarea unui obiect într-o găleată

** Input **

| Proprietăți                                    | Necesar  | Descriere                                                                                                 |
| ---------------------------------------------- | -------- | --------------------------------------------------------------------------------------------------------- |
| Tip Conținut                                   | necesar  | Tip MIME de obiect pe care îl încărcați.                                                                  |
| Încărcați numele obiectului                    | necesar  | numele obiectului pe care îl încărcați.                                                                   |
| Încarcă date                                   | necesar  | datele pe care doriţi să le încărcaţi pe Amazon S3.                                                       |
| Timeout pentru încărcarea unui fișier (Minute) | opţional | timpul de încărcare în câteva minute înainte ca procesul de încărcare să fie oprit de o eroare de timeout |

De exemplu, pentru a șterge mai multe documente, pune `text` pentru **`tip de conținut`**, `text.txt` pentru **`upload object name`**, și `text` pentru **`upload data`**.

\*\* Rezultat

returnează mesajul dacă documentul a fost încărcat cu succes

![amazon_upload_data1](https://cdn.illacloud.com/official-website/img/docs/assemble/amazon_upload_data1.jpg)

![amazon_upload_data2](https://cdn.illacloud.com/official-website/img/docs/assemble/amazon_upload_data2.jpg)

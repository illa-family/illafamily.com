## <h2 hidden>Ce este formularul?</h2>

Componenta formular este un element vizual care reprezintă progresul sau finalizarea unei sarcini sau a unui proces. Afişează o bară orizontală care umple sau se micşorează în funcţie de valoarea de progres specificată.

### Proprietăți

| Proprietăți                             | Descriere                                                                                                                                                                    |
| --------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Gestionar de evenimente                 | detectare și răspuns la acțiuni sau evenimente specifice ale utilizatorilor, cum ar fi clicuri, taste sau depuneri de forme.                                                 |
| Dezactivat                              | non-interactiv și nu poate fi modificat sau declanșat de către utilizator.                                                                                                   |
| Dezactivează remiterea                  | dezactivarea funcționalității de transmitere a formularului.                                                                                                                 |
| Validare intrări la remitere            | intrările de formular ar trebui să fie validate pentru corectitudine sau completitudine atunci când formularul este trimis.                                                  |
| Resetare după trimiterea cu succes      | resetează automat valorile și starea la valorile inițiale sau implicite după o trimitere reușită                                                                             |
| Tooltip                                 | un mic pop-up sau un mesaj informațional care apare atunci când utilizatorul planează deasupra sau interacționează cu un anumit element, cum ar fi o componentă de formular. |
| Arată antet                             | dacă o componentă formular ar afișa o secțiune de antet sau titlu                                                                                                            |
| Arată subsol                            | dacă o componentă formular ar trebui să afișeze o secțiune de subsol                                                                                                         |
| Hidden                                  | ascunde componenta formular din vizualizarea utilizatorului                                                                                                                  |
| Culoare chenar                          | culoarea marginii care înconjoară o componentă de formular                                                                                                                   |
| Raza frontierei                         | curbarea colțurilor unei componente sub formă                                                                                                                                |
| Border Width                            | grosimea sau lăţimea marginii care înconjoară o componentă a formularului                                                                                                    |
| Context                                 | defineşte culoarea sau imaginea care umple zona de conţinut a unei componente formular                                                                                       |
| Umbră                                   | efectul vizual al adăugării unei subtile, offset „umbră” la o componentă formă, ceea ce face ca aceasta să fie ușor ridicată sau plutitoare deasupra fundalului              |

### Metodă

Puteţi utiliza alte componente pentru a controla componenta. Sprijinim următoarele două metode:

-   **trimis**

Pentru a trimite informaţiile completate în câmpul formular

-   **reset**

resetaţi toate câmpurile de intrare în componenta formular

-   **setValue**

Pentru a seta valoarea de intrare a zonei de text, de exemplu, {{"value1"}}

| Proprietăți  | Descriere   |
| ------------ | ----------- |
| Valoare      | Input value |

-   validate

Pentru a verifica dacă informațiile de intrare sunt legale

### Date

Componenta are niște date utilizate în mod obișnuit, care pot fi apelate prin **`{{componentName.propertyName}}`** în aplicație.

| Nume proprietate              | Descriere                                                                                                                                                                     |
| ----------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Culoare fundal                | specifică culoarea de fundal a unei componente formular                                                                                                                       |
| culoare margini               | definește culoarea marginii care înconjoară o componentă formular                                                                                                             |
| borderWidth                   | determină grosimea sau lăţimea marginii care înconjoară o componentă formular                                                                                                 |
| dezactivat                    | dacă o componentă formular este dezactivată sau nu                                                                                                                            |
| AfișeazăNume                  | reprezintă numele sau eticheta atribuită unei componente formular                                                                                                             |
| Înălțime subsol               | specifică înălţimea sau dimensiunea secţiunii subsol a unei componente formular                                                                                               |
| Formular                      | deține datele sau valorile introduse de utilizator într-o componentă de formular                                                                                              |
| Înălțime header:              | determină înălţimea sau dimensiunea secţiunii antetului a unei componente formular                                                                                            |
| nevalid                       | dacă intrarea sau valoarea unei componente a unui formular este considerată nevalidă sau nu.                                                                                  |
| radius                        | definește raza de frontieră sau curbura colțurilor unei componente de formă                                                                                                   |
| resetare Reușită              | indică faptul că o componentă formular ar trebui să reseteze automat valorile și starea la valorile inițiale sau implicite după o trimitere reușită                           |
| umbră                         | controlează prezenţa şi apariţia unui efect de umbră asupra unei componente a formei;                                                                                         |
| showfooter                    | determină dacă secțiunea subsol a unei componente formular ar trebui să fie afișată sau nu                                                                                    |
| antet de spectacol            | vizibilitatea secțiunii antet a unei componente formular                                                                                                                      |
| tooltipText                   | deține textul sau conținutul unui indiciu de unealtă asociat cu o componentă formular                                                                                         |
| validateInputOnSubmit         | specifică faptul că valorile de intrare ale unei componente formular trebuie validate pentru corectitudine sau completitudine atunci când formularul este depus               |

Exemplu: {{form1.disabled}}

### Utilizare majusculă

Vom demonstra cum să facem un formular cu numele și vârsta ca intrare, precum şi progresul barei pentru a arăta progresul de finalizare al formularului.

#### Pasul 1 Adaugă Componente

Adăugați o componentă **`form`** cu textul implicit "Form" și un buton "Trimite" pe planșă.

Adăugați o componentă **`input`**, o componentă **`number input`**, și o componentă **`bar progress`** din `**Insert**` în formular.

Etichetați componenta de intrare pentru a fi "Nume", componenta de intrare a numărului pentru a fi "Age" și componenta de progres a barei pentru a fi "Progress".

Setați substituentul componentei de intrare la "Vă rugăm să puneți numele dvs. aici" și valoarea implicită a numărului introdus la 0 după cum se arată mai jos.

![bar_layout](https://cdn.illacloud.com/official-website/img/official-site/components/bar_layout.jpg)

#### Pasul 2 Configurați componenta

Pentru componentele **`input`** și **`number input`**, le putem configura pentru a seta valoarea în componenta **`bar progress`** pentru a reflecta progresul de finalizare.

1.  Faceți clic pe componenta **`input`** și **`number input`** pentru a deschide pagina sa de inspecție. Sub **`Manipulatorul de evenimente`**, apasă `**+ New**`.
2.  În gestionarul de editare a evenimentului, selectați **`Componenta Control`** în acțiune, și selectați componenta **`bar progress`** pe care doriți să o actualizați ca țintă a evenimentului.
3.  Alegeți acțiunea **`setValue`** și setați valoarea ca

```jsx
{
  {
    input1.value && numberInput1.value
      ? "100"
      : input1.value || numberInput1.value
      ? "50"
      : "0";
  }
}
```

![bar_input_num_config](https://cdn.illacloud.com/official-website/img/official-site/components/bar_input_num_config.jpg)

Notă: este necesar să se seteze gestionarul de evenimente conform descrierii de mai sus cu componentele de intrare și de numerotare a componentelor.

Pentru componenta **`button`**, o putem configura pentru a trimite formularul dacă formularul este completat (ex bar progress 100%)

1.  Faceți clic pe componenta **`button`** pentru a deschide pagina sa de inspecție. Sub **`Manipulatorul de evenimente`**, apasă `**+ New**`.
2.  În gestionarul de editare a evenimentului, selectați **`Componenta Control`** în acțiune, şi selectaţi componenta **`form`** pe care doriţi să o actualizaţi ca ţintă a evenimentului.
3.  Alegeți metoda **`submit`** și puneți `{{barProgress1.valoare == 100}}` în **`Rulați numai când`**

![bar_button_submit_config](https://cdn.illacloud.com/official-website/img/official-site/components/bar_button_submit_config.jpg)

În mod similar, putem configura butonul pentru a afișa avertismentul atunci când utilizatorul încearcă să trimită un formular incomplet.

1.  Faceți clic pe componenta **`button`** pentru a deschide pagina sa de inspecție. Sub **`Manipulatorul de evenimente`**, apasă `**+ New**`.
2.  În managerul de editare al evenimentului, selectaţi **`Arată notificarea`** în acţiune, **`title`** "Eroare!" şi **`description`** "Vă rugăm să completaţi formularul înainte de trimitere". Alege `Error` pentru **`type`**, pune `{{2000}}` pentru **`duration`** (2 secunde), şi pune `{{barProgress1.value<100}}` pentru **`Rulează doar când`.**

![bar_button_show_config](https://cdn.illacloud.com/official-website/img/official-site/components/bar_button_show_config.jpg)

## <h2 hidden>Ce sunt statisticile?</h2>

Componenta de statistici din ILLA Cloud este un instrument puternic pentru afișarea și analiza datelor numerice. Oferă o modalitate vizuală atrăgătoare și personalizabilă de a prezenta măsurători și statistici cheie.

### Proprietăți

| Proprietăți                       | Descriere                                                                                                                                                              |
| --------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Etichetă                          | Numele câmpului afişat utilizatorului                                                                                                                                  |
| Valoare primară                   | valoarea primară sau ratingul afișat de componentă.                                                                                                                    |
| Valoare secundară                 | un rating suplimentar sau o valoare asociată cu valoarea principală, furnizând informații suplimentare sau context.                                                    |
| Primară                           |                                                                                                                                                                        |
| Loc zecimal                       | specificați numărul de zecimale care pot fi introduse în componentă                                                                                                    |
| Arată semnul tendinței            | Activarea acestei proprietăți afișează o săgeată sau un semn care indică tendința sau modificarea valorii de rating.                                                   |
| Arată separatorul de mii          | Atunci când este activată, această proprietate adaugă o mie de separatoare (virgulă sau perioadă) la valorile primare și secundare pentru o mai bună lizibilitate.     |
| Activează culoarea tendinței      | schimbă culoarea pe baza tendinței sau a modificării valorii de rating.                                                                                                |
| Prefix text                       | un text scurt care apare în partea stângă a câmpului de intrare, adesea folosit pentru a oferi un context suplimentar sau instrucțiuni utilizatorului                  |
| Text în completare                | un text scurt care pare în dreapta câmpului de intrare, utilizat adesea pentru a furniza informații suplimentare sau feedback utilizatorului                           |
| Secundar                          |                                                                                                                                                                        |
| Loc zecimal                       | specificați numărul de zecimale care pot fi introduse în componentă                                                                                                    |
| Arată semnul tendinței            | Activarea acestei proprietăți afișează o săgeată sau un semn care indică tendința sau modificarea valorii de rating.                                                   |
| Arată separatorul de mii          | Atunci când este activată, această proprietate adaugă o mie de separatoare (virgulă sau perioadă) la valorile primare și secundare pentru o mai bună lizibilitate.     |
| Activează culoarea tendinței      | schimbă culoarea pe baza tendinței sau a modificării valorii de rating.                                                                                                |
| Prefix text                       | un text scurt care apare în partea stângă a câmpului de intrare, adesea folosit pentru a oferi un context suplimentar sau instrucțiuni utilizatorului                  |
| Text în completare                | un text scurt care pare în dreapta câmpului de intrare, utilizat adesea pentru a furniza informații suplimentare sau feedback utilizatorului                           |
| Gestionar de evenimente           | definiți acțiuni personalizate sau logici care declanșează atunci când componenta este interacționată sau ratingul este schimbat.                                      |
| Tooltip                           | Utilizatorii pot introduce aici pontul componentei. Sfatul de instrumente va fi afișat când este concentrat. Formatul markdown este acceptat.                          |
| Hidden                            | Controlează dinamic dacă componenta este ascunsă. Puteți schimba starea ascunsă prin valoarea booleană dinamică.                                                       |
| Aliniere                          | alinierea componentei, cum ar fi stânga, centrul sau dreapta.                                                                                                          |
| Culoare temă                      | Permite utilizatorilor să specifice culoarea de fundal şi opacitatea butonului                                                                                         |
| Culoare pozitivă                  | determină culoarea utilizată pentru evaluări pozitive sau mari                                                                                                         |
| Culoare negativă                  | definește culoarea folosită pentru evaluări negative sau scăzute.                                                                                                      |

### Metodă

Puteţi utiliza alte componente pentru a controla componenta. Sprijinim următoarele două metode:

-   **setPrimaryValoare**

Pentru a seta valoarea principală, de exemplu, {{"value1"}}

| Proprietăți        | Descriere                   |
| ------------------ | --------------------------- |
| Valoare principală | Introducere valoare primară |

-   **resetPrimaryValoare**

Pentru a reseta valoarea la valoarea implicită a componentei selectate. În cazul în care valoarea implicită nu este specificată, se șterge valoarea.

| Eveniment | Descriere                                                                                             |
| --------- | ----------------------------------------------------------------------------------------------------- |

### Date

Componenta are niște date utilizate în mod obișnuit, care pot fi apelate prin **`{{componentName.propertyName}}`** în aplicație.

| Nume proprietate                   | Descriere                                                                                                                               |
| ---------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------- |
| culori                             | Culoarea de fundal a butonului                                                                                                          |
| AfișeazăNume                       | numele acestei componente (de ex. buton1)                                                                                               |
| evenimente                         | generat sau declanșat atunci când anumite acțiuni sau evenimente apar în interiorul componentei de intrare a numărului.                 |
| hidden                             | stare ascunsă (adevărată sau falsă)                                                                                                     |
| etichetă                           | valoare etichetă                                                                                                                        |
| negativeColorScheme                | determină schema de culori utilizată pentru afișarea valorilor negative                                                                 |
| positiveColorScheme                | setează schema de culori pentru valorile pozitive                                                                                       |
| primaryValoare                     | valoarea numerică primară care urmează să fie afișată în componenta statistică.                                                         |
| zecimalPlace                       | definește numărul de zecimale care trebuie indicat atât pentru valorile primare, cât și pentru cele secundare.                          |
| showTrendSign                      | afișează un semn pozitiv sau negativ de tendință (+ sau -) lângă valori, indicând direcția tendinței.                                   |
| textAlign                          | alinierea componentei statistici                                                                                                        |
| semn pozitiv                       | setează simbolul care va fi afișat pentru valorile pozitive                                                                             |
| Semn negativ                       | definește simbolul care trebuie arătat pentru valorile negative.                                                                        |
| Separator de spectacol             | dacă afișează un separator de o mie (cum ar fi o virgulă) pentru a îmbunătăți lizibilitatea numerelor mari.                             |
| prefixText                         | valoarea prefixului                                                                                                                     |
| Sufix                              | valoarea sufixului                                                                                                                      |
| setPrimaryValoare                  | vă permite să setați programatic sau să actualizați valoarea primară a componentei statistice.                                          |
| secundaryValoare                   | o valoare numerică secundară care poate fi afișată alături de valoarea primară.                                                         |
| secundaryDecimalPlace              | specifică numărul de zecimale pentru a fi afişate pentru valoarea secundară.                                                            |
| secundaryEnableTrendColor          | dacă aplică o schemă de culori la valoarea secundară pe baza tendinței sale                                                             |
| secundaryPositiveSign              | Setează simbolul care va fi afișat pentru valori secundare pozitive                                                                     |
| panou secundaryNegativ             | Definește simbolul care trebuie afișat pentru valorile secundare negative                                                               |
| secundaryPrefixText                | adăugați un text personalizat sau simboluri pentru a apărea înaintea valorilor secundare.                                               |
| Separator                          | afișează un separator de o mie pentru valoarea secundară.                                                                               |
| secundaryShowTrendSign             | afișează un semn de tendință (+ sau -) pentru valoarea secundară.                                                                       |
| secundarySuffixText                | adăugați un text personalizat sau simboluri pentru a apărea după valorile secundare.                                                    |
| tooltipText                        | valoarea textului cu înclinare unelte                                                                                                   |

Exemplu: {{statistics1.secondaryValue}}

### Utilizare majusculă

Apoi, vom demonstra cum să cartografiem datele de la sursa de date la statistici cu un buton.

#### Pasul 1 Adăugare acţiune

Haideți să creăm un tabel în Amazon S3. Unul dintre documente este numit

"1.txt" cu ca date zecimale "1.1111". Aceasta este valoarea pe care dorim să o afișăm în numărul introdus.

![s3data](https://cdn.illacloud.com/official-website/img/official-site/components/s3data.jpg)

Apoi putem crea o nouă acțiune pentru Amazon S3 din lista de acțiuni, enumerând toate datele din găleată și numind31.

Selectați `Citește un obiect` pentru `**Tipul de Acțiune**`. Puneți `1.txt` pentru `**Cheie Object`.\*\*

Faceţi clic pe **`Salvează`** şi **`Run`** pentru a activa această acţiune.

![stat_data](https://cdn.illacloud.com/official-website/img/official-site/components/stat_data.jpg)

#### Pasul 2 Adaugă Componente

Apoi, putem adăuga o componentă **`statistics`** şi o componentă **`button`** la pânză.

Am setat valoarea primară implicită a statisticilor la 0 și am etichetat butonul ca "Setează valoarea" ca fiind afișat mai jos

![stat_layout](https://cdn.illacloud.com/official-website/img/official-site/components/stat_layout.jpg)

#### Pasul 3 Configurarea componentei

Pentru componenta **`button`**, îl putem configura pentru a seta valoarea în componenta **`statistics`** la datele documentului pe care îl citim de la s31 api.

1.  În gestionarul de editare a evenimentului, selectați **`Componenta Control`** în acțiune, şi selectaţi componenta **`statistici`** pe care doriţi să o actualizaţi ca ţintă a evenimentului.
2.  Alegeți acțiunea **`setValue`** și setați valoarea ca `{{s31.data}}`

![stat_button_config](https://cdn.illacloud.com/official-website/img/official-site/components/stat_config.jpg)

#### Etapa 4 Test

Acum că atunci când faceți clic pe butonul "Setare valoare", valoarea ar trebui să se schimbe la 1.1111.

![stat_test](https://cdn.illacloud.com/official-website/img/official-site/components/stat_test.gif)

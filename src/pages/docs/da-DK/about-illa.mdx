## <Heading hidden>Om ILLA</Heading>

ILLA giver dig mulighed for hurtigt at opbygge og implementere interne apps. Opret forbindelse til dine databaser og API'er, saml brugergrænseflader med træk-og-slip byggesten som tabeller og formularer, og skrive forespørgsler til at interagere med data ved hjælp af SQL og JavaScript.

### Hvad er ILLA?

ILLA er en open source lav-kode platform for udviklere til at opbygge og implementere interne værktøjer bemærkelsesværdigt hurtigt.

<img
  src='https://cdn.illacloud.com/official-website/img/docs/introduction/about-illa-cloud.png'
  alt=""
/>

Med ILLA kan du nemt oprette forbindelse til forskellige datakilder som databaser, API/GraphQL endpoints, SaaS værktøjer og cloud storage tjenester. Disse omfatter populære muligheder som PostgreSQL, MongoDB, MySQL, Stripe, Google Sheets, og AWS S3. Når den er tilsluttet, kan ILLA køre forespørgsler for at hente og opdatere data fra disse kilder. Brugergrænsefladen (UI) giver praktiske komponenter til visualisering og ændring af data, såsom tabeller, diagrammer og formularer.

Med ILLA, kan du hurtigt og nemt bygge en brugergrænseflade (UI) ved hjælp af præ-bygget widgets, som du kan trække og slippe på et gitter stil lærred. Dette forenkler integrationen af front-end og back-end af din ansøgning og optimerer byggeprocessen. Derudover ILLA understøtter JavaScript inde widgets, forespørgsler og andre komponenter, så du kan tilføje logik, omdanne data, og definere komplekse arbejdsgange. Dette gør det nemt at tilpasse din ansøgning til dine specifikke behov.

### Hent Start

For at prøve ILLA er den mest bekvemme måde at tilmelde dig og logge ind på [ILLA Cloud](https://cloud.illacloud.com/). 
For at fremskynde revisionsprocessen inviterer vi dig til at deltage i vores [Discord Community](https://discord.gg/illacloud), hvor du hurtigere kan få en invitationskode.

### Fællesskab

For at modtage mere rettidig støtte, mens du bruger ILLA, overveje at deltage i vores [Discord Community](https://discord.gg/illacloud). Dette vil give dig mulighed for at få støtte direkte fra fællesskabet, som kan hjælpe dig med at løse eventuelle problemer hurtigere.

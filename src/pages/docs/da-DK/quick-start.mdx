## <Heading hidden>Oversigt</Heading>

### 1.  Deploy ILLA Builder

Før du begynder at bruge ILLA Builder, bør du vælge at bruge ILLA Cloud eller Self-host ILLA til at være vært for dine apps. Vi giver dig ILLA CLI, Docker Compose, og K8s til selv-vært ILLA på under 5 minutter. Vi vil give dig ILLA Cloud senere, for at hjælpe dig med blot at komme i gang med ILLA Builder.

<img
  src='https://cdn.illacloud.com/official-website/img/docs/qs_e_1.png'
  alt=""
/>

### 2.  Hent brugerdata

Registrer og log ind på ILLA Builder, og indtast derefter ressource fanen. Klik på knappen "Opret ny", så du kan vælge ressourcetypen her. Vi giver dig REST API og MySQL til at forbinde dine data. Postgres, Redis og MongoDB vil være tilgængelige i denne måned.

<img
  src='https://cdn.illacloud.com/official-website/img/docs/qs_e_2.png'
  alt=""
/>

### 3.  Træk og slip indsatser

Når dine data er forberedt, kan du gå tilbage til fanen Apps og oprette en app. Indsæt listen er på højre side af siden. Du kan vælge en af dem og trække den ind i rammen. Klik derefter på Indsæt og ændret dens parametre i Kontrolpanel. Vi giver dig markdown tilstand til at redigere etiketten på hver indsats. Hvis du er bekendt med Markdown syntaks, kan du hurtigt justere tekststørrelse og tykkelse. I mellemtiden kan du bruge {{double brackets}} til at påberåbe sig handlingsdata i tekstindtastningsfeltet for at opnå flere resultater i ILLA Builder.

### 4.  Tilføj handling i apps

En handling er nødvendig for at omdanne data fra ressourcer til at indføre. Vi kalder det for handling. Du kan skrive en forespørgsel i aktion for at fange data fra din ressource. På dette grundlag kan du skrive JavaScript i handlinger for at udføre databehandlingsinstruktioner, såsom filter og fletning. Et MySQL eksempel er givet nedenfor:

<img
  src='https://cdn.illacloud.com/official-website/img/docs/qs_e_3.png'
  alt=""
/>

1.  Opret en MySQL handling i App editor. Navngivet den handling1.

2.  Indtast forespørgselskoden i panelet, f.eks. SELECT \* FRA brugere.

3.  Kør denne handling, vil forespørgselsresultatet blive vist i det nederste panel.

### 5.  Vis dine data

Når handlingen er oprettet, skal du binde resultatet af handlingen til komponenten. Du bør erstatte værdivariablen på inspektionspanelet med resultatet af handlinger {{action1.data}}. Så komponenten data vil binde med handlingen resultat.

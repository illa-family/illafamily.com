## <Heading hidden>Introdução de implantação</Heading>

### Nuvem

Tente implantação em um clique na Nuvem ILLA para uma experiência mais fácil.

### Auto-hospedeiro

Você pode se autohospedar localmente com ILLA CLI, Compose Docker, k8s helm etc. Nós otimizamos repetidamente o processo de auto-anfitrião para permitir que você faça implantações de software e hospedagem em alguns minutos. Comparado com a nuvem, você pode usar produtos ILLA de forma mais segura e confiável, e você pode usar o Construtor de ILLA simplesmente visitando o site de implantação após hospedar e implantar o Construtor localmente.

Para obter mais informações sobre o host, consulte as páginas de inicialização de elmo ILA CLI, o Docker compose, k8s.
Se tiver dificuldades com sua auto-implantação, entre em contato conosco no GitHub ou business@illasoft.com.

#### Implantar com CLI ILLA

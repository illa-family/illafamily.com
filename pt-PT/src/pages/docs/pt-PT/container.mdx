## <Heading hidden>Contêiner</Heading>

Você pode agrupar componentes relacionados juntos em um cartão usando o componente Contêiner. Os contêineres também suportam várias visualizações que permitem separar o conteúdo em visualizações distintas que os usuários podem alternar entre elas.

Contêineres funcionam como qualquer outro componente. Uma vez adicionado, você pode agrupar componentes dentro dele pressionando D ou K e arrastando-os para o Contêiner.

### Várias visualizações

Os contêineres suportam várias views para conteúdo em agrupamento. Você pode definir visualizações na seção **Visualizações** da aba **Inspecionar**. Apenas uma exibição é visível de um momento e os usuários alteram entre eles alterando a visão atual do componente de Contêiner.

### Use outros componentes para alterar as visualizações

Você pode usar outros componentes para controlar contêineres, adicionando um manipulador de eventos a outros componentes. Os seguintes métodos são suportados:

1.  Definir visualização atual pela chave
2.  Configura visão atual por índice
3.  Mostrar a próxima visualização (visível)
4.  Mostrar a visualização anterior (visível)

### Conecte uma aba a um contêiner

Você pode usar um componente tab para controlar um contêiner ao ligar **Link para um contêiner** da guia **Inspecionar**.

Depois de vincular a um contêiner, os rótulos das guias mudarão para os rótulos das visualizações. Você pode alterar as exibições clicando nas guias sem os manipuladores de eventos.

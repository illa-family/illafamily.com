## <Heading hidden>Lista de integração</Heading>

ILLA oferece uma série de integrações pré-criadas que permitem que você se conecte facilmente às suas fontes de dados com configuração mínima. Simplesmente selecione a integração apropriada a partir da lista de opções disponíveis, digite as credenciais necessárias e você pode começar a usar sua fonte de dados dentro do ILLA. Isso elimina a necessidade de configurar configurações complexas e permite que você comece a trabalhar rapidamente com seus dados.

### Integrações do Banco

| Nome:             | Descrição:                                                                                                                                                                                                                                                                |
| ----------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Mysql             | MySQL é um sistema de gerenciamento de banco de dados de código aberto (RDBMS) baseado na Estrutura de Língua de Consulta (SQL).                                                                                                                                          |
| PostgreSQL        | PostgreSQL é um sistema de gerenciamento de banco de dados relativo a objetos de código aberto (ORDBMS) conhecido por sua confiabilidade e riqueza em recursos.                                                                                                           |
| MariaDB           | O MariaDB é um sistema de gestão de banco de dados de código aberto que é um fork do banco de dados MySQL.                                                                                                                                                                |
| TiDB              | TiDB é um banco de dados NewSQL de código aberto que suporta cargas de trabalho de Transação Híbrida e Análise de Processamento.                                                                                                                                          |
| Redis             | O Redis é um armazenamento de estrutura de dados da memória de código aberto que é frequentemente usado como um banco de dados, cache e corretor de mensagens.                                                                                                            |
| MongoDB           | MongoDB é um programa de banco de dados multiplataforma gratuito e de código aberto.                                                                                                                                                                                      |
| Pesquisa Elástica | Elasticsearch é um mecanismo de pesquisa e análise distribuída construído sobre Apache Lucene.                                                                                                                                                                            |
| Firebase          | O Firebase é uma plataforma de desenvolvimento de aplicativos web e móveis com base em nuvem desenvolvida pelo Google.                                                                                                                                                    |
| Supabase DB       | Supabase é uma plataforma de código aberto para a construção de aplicativos de servidor em tempo real. Ele é construído em cima do PostgreSQL e fornece uma série de ferramentas e serviços para rápida construção e implantação de aplicativos de servidor sem servidor. |
| Clickhouse        | ClickHouse é um sistema de gerenciamento de banco de dados orientado a colunas aberto que é projetado para processamento analítico on-line (OLAP) e análise de dados em tempo real.                                                                                       |

### Integrações com API

| Nome:               | Descrição:                                                                                                                                                    |
| ------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| API REST            | REST é um estilo arquitetônico para construir serviços da web. APIs RESTful são APIs que seguem o estilo arquitetônico REST.                                  |
| Amazon S3(Geral S3) | O Amazon S3 é um serviço de armazenamento de objetos oferecido pela AWS. Proporciona armazenamento seguro, escalável e de baixo custo para dados na Internet. |
| SMTP                | SMTP (Simple Mail Transfer Protocol) é um protocolo para enviar e receber mensagens de e-mail pela internet.                                                  |
| GrafQL              | GraphQL é um idioma de consulta para APIs e um tempo de execução para satisfazer essas consultas com seus dados existentes.                                   |

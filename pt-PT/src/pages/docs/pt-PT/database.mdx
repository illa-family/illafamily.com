ILLA conecta-se à maioria dos bancos de dados, incluindo PostgreSQL, MySQL, TiDB e MongoDB. Por favor, verifique aqui esta lista de fontes de dados ILLA suportadas atualmente. As próximas fontes de dados que planejamos apoiar estão listadas aqui. Caso esteja faltando certos recursos de dados que você está usando, por favor preencha o formulário aqui.
Os detalhes de conexão requeridos pela ILLA dependem do tipo de banco de dados que você deseja conectar. Estas informações geralmente incluem um host, porto, credenciais do usuário e string de conexão. Por favor, consulte a configuração do seu banco de dados ou documentação para obter isto.

1.  Criar um recurso de banco de dados
    Recursos representam conexões a fontes de dados e são o que você consulta dentro de seus aplicativos ILLA. Faça o login no ILLA e navegue até a aba Recursos > clique em Criar novo e selecione a base de dados à qual deseja se conectar. Isto abre a janela de configurar uma fonte de recursos onde você adiciona suas informações de conexão.

Após inserir os detalhes da sua conexão, clique em Testar conexão para verificar a ILLA pode se conectar com sucesso. Uma vez concluído, clique em Salvar recurso para salvar o recurso. 
Além disso, você pode criar recursos na lista de ações sob o editor de aplicativos. 
2\. Consulte seu banco de dados
no editor do aplicativo, clique em + Nova na Lista de ações > selecione o tipo de recurso e escolha um recurso existente ou crie um novo recurso > clique em criar ação e atualizar as ações para recuperar algumas informações de uma tabela em sua base de dados.

Clique em Salvar e depois em Executar. Os resultados da consulta são exibidos automaticamente após a conclusão da consulta. 
Existem algumas configurações avançadas, incluindo renomear, executar configurações, transformar dados e configurar a manipulação de sucesso ou falha, como mostrado abaixo.

## <Heading hidden>Autenticação de API personalizada</Heading>

ILLA suporta muitos padrões de autenticação por padrão (veja aqui para mais informações). 
Além disso, ILLA tem um modo de autenticação de autenticação personalizada que permite combinar várias etapas que suportam praticamente qualquer autenticação de API. 
Esses métodos de autenticação podem ser usados tanto para recursos REST quanto GraphQL.

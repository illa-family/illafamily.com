## <Heading hidden>Επισκόπηση</Heading>

### 1.  Ανάπτυξη Κατασκευαστή ILLA

Πριν αρχίσετε να χρησιμοποιείτε το ILLA Builder, θα πρέπει να επιλέξετε να χρησιμοποιήσετε ILLA Cloud ή Self-host ILLA για να φιλοξενήσετε τις εφαρμογές σας. Σας παρέχουμε ILLA CLI, Docker Compose, και K8s για την αυτο-φιλοξενία ILLA σε λιγότερο από 5 λεπτά. Θα σας παρέχουμε το ILLA Cloud αργότερα, για να σας βοηθήσουμε απλά να ξεκινήσετε με τον ILLA Builder.

<img
  src='https://cdn.illacloud.com/official-website/img/docs/qs_e_1.png'
  alt=""
/>

### 2.  Λήψη δεδομένων χρηστών

Εγγραφείτε και συνδεθείτε στο ILLA Builder, στη συνέχεια, εισάγετε την καρτέλα των πόρων. Κάντε κλικ στο κουμπί "Δημιουργία νέου" και στη συνέχεια μπορείτε να επιλέξετε τον τύπο πόρων εδώ. Σας παρέχουμε API REST και MySQL για να συνδέσετε τα δεδομένα σας. Οι Postgres, Redis και MongoDB θα είναι διαθέσιμες αυτό το μήνα.

<img
  src='https://cdn.illacloud.com/official-website/img/docs/qs_e_2.png'
  alt=""
/>

### 3.  Σύρετε και αποθέστε ένθετα

Όταν τα δεδομένα σας είναι έτοιμα, μπορείτε να επιστρέψετε στην καρτέλα Εφαρμογές και να δημιουργήσετε μια εφαρμογή. Η λίστα εισαγωγής βρίσκεται στη δεξιά πλευρά της σελίδας. Μπορείτε να επιλέξετε ένα από αυτά και να το σύρετε στο πλαίσιο. Στη συνέχεια, κάντε κλικ στο ένθετο και να τροποποιήσει τις παραμέτρους του πίνακα Επιθεώρηση. Σας παρέχουμε τη λειτουργία σήμανσης για να επεξεργαστείτε την ετικέτα κάθε ένθετου. Αν είστε εξοικειωμένοι με τη σύνταξη Markdown, μπορείτε να προσαρμόσετε γρήγορα το μέγεθος του κειμένου και το πάχος. Εν τω μεταξύ, μπορείτε να χρησιμοποιήσετε το {{double brackets}} για να επικαλεστείτε δεδομένα ενέργειας στο πλαίσιο εισαγωγής κειμένου για να επιτύχετε περισσότερα αποτελέσματα στο ILLA Builder.

### 4.  Προσθήκη ενέργειας στις εφαρμογές

Μια ενέργεια απαιτείται για να μετατρέψει τα δεδομένα από τους πόρους να εισαχθούν. Το αποκαλούμε πράξη. Μπορείτε να γράψετε ένα ερώτημα σε δράση για να πάρετε τα δεδομένα από τον πόρο σας. Σε αυτή τη βάση, μπορείτε να γράψετε JavaScript σε ενέργειες για να εκτελέσετε τις οδηγίες επεξεργασίας δεδομένων, όπως φίλτρο και συγχώνευση. Ένα παράδειγμα MySQL δίνεται παρακάτω:

<img
  src='https://cdn.illacloud.com/official-website/img/docs/qs_e_3.png'
  alt=""
/>

1.  Δημιουργήστε μια ενέργεια MySQL στον επεξεργαστή εφαρμογών. Ονομάστηκε ενέργεια1.

2.  Εισάγετε τον κωδικό του ερωτήματος στον πίνακα, π.χ. ΕΠΙΛΟΓΗ \* ΑΠΟ τους χρήστες.

3.  Εκτέλεση αυτής της ενέργειας, το αποτέλεσμα ερωτήματος θα εμφανιστεί στον κάτω πίνακα.

### 5.  Εμφάνιση των δεδομένων σας

Όταν η ενέργεια δημιουργηθεί με επιτυχία, πρέπει να δεσμεύσετε το αποτέλεσμα της ενέργειας στο στοιχείο. Θα πρέπει να αντικαταστήσετε τη μεταβλητή τιμής στον πίνακα επιθεώρησης με το αποτέλεσμα των ενεργειών {{action1.data}}. Στη συνέχεια, τα δεδομένα στοιχείων θα δεσμεύονται με το αποτέλεσμα ενέργειας.

## <Heading hidden>Επεξεργαστής εφαρμογών</Heading>

Ο επεξεργαστής APP είναι η διεπαφή για την ανάπτυξη της εφαρμογής και αποτελείται από τα ακόλουθα κύρια μέρη:

-   Γραμμή Εργαλείων
-   Frame
-   Επεξεργαστής ενεργειών
-   Χώρος εργασίας δεδομένων
-   Insert
-   Επιθεώρηση

<img alt="editor" src={'https://cdn.illacloud.com/official-website/img/official-site/content.png'} />

### Γραμμή Εργαλείων

Η γραμμή εργαλείων βρίσκεται στην κορυφή του επεξεργαστή APP, η οποία χρησιμοποιείται κυρίως για να ρυθμίσετε το APP και μπορεί να χρησιμοποιηθεί για τις ακόλουθες εργασίες：

-   Κάντε κλικ στο εικονίδιο ILLA στα αριστερά για να επιστρέψετε στην αρχική σελίδα.
-   Προβολή πληροφοριών APP και αποθήκευση κατάστασης.
-   Εμφάνιση/απόκρυψη αντικειμένων αριστερά, δεξιά, κάτω πίνακα.
-   Ρύθμιση της κλίμακας των αντικειμένων.
-   Εμφάνιση όλων των προγραμματιστών της εφαρμογής.
-   Ειδοποίηση της εφαρμογής στην τρέχουσα κατάσταση σφάλματος.
-   Μοιραστείτε το APP με άλλους προγραμματιστές.
-   Εκτέλεση του APP.

#### Περισσότερα χαρακτηριστικά έρχονται soom

-   Εισαγωγή APPs από json.
-   Εφαρμογές Αναπαραγωγής.
-   Εξαγωγή APP σε μορφή json.
-   Επιλέξτε την επιθυμητή έκδοση APP.
-   Δείτε το επεξεργασμένο ιστορικό.
-   Επαναφόρτωση όλων των ενεργειών.

### Frame

Το πλαίσιο είναι η περιοχή του APP όπου μπορείτε να επεξεργαστείτε τη γραφική διεπαφή του APP. Μπορείτε να σύρετε και να αποθέσετε ένθετα στο πλαίσιο από τον πίνακα ένθετα στο δεξί χέρι. μπορείτε επίσης να επιλέξετε ένθετα που έχουν τοποθετηθεί στο πλαίσιο και να χρησιμοποιήσετε το ποντίκι για να σύρετε και να ρίξετε για να τροποποιήσετε τη θέση και το μέγεθος των ενθέτων.

Όταν σύρετε και ρίχνετε τα ένθετα ή αλλάζετε το μέγεθός τους, ένα διακεκομμένο τετράγωνο θα εμφανιστεί στο παρασκήνιο του πλαισίου για να σας βοηθήσει να οργανώσετε τα ένθετα τακτοποιημένα στο πλαίσιο. Όταν συναντήσετε επικαλυπτόμενα ένθετα ενώ σύρετε και πέφτετε, Ο Κατασκευαστής ILLA θα δώσει προτεραιότητα στο ένθετο που κινείστε και θα μετακινήσει αυτόματα το ένθετο μπλοκάρισμα έξω από το δρόμο για εσάς.

#### Προεπισκόπηση

Το κουμπί εναλλαγής προεπισκόπησης βρίσκεται στην επάνω δεξιά γωνία του πλαισίου . Όταν πατηθεί, ο επεξεργαστής θα εισάγει τη λειτουργία προεπισκόπησης, τα κάτω αριστερά και δεξιά πάνελ θα κρυφτούν και τα ένθετα στο πλαίσιο δεν θα τροποποιηθούν, όλα τα ένθετα θα παρουσιαστούν μπροστά σας σε 1:1 μέγεθος.

> ### Υποστηριζόμενα προγράμματα περιήγησης
>
> Σας συνιστούμε να ανοίξετε το ILLA Builder στο Google Chrome.

### Επεξεργαστής Ενεργειών

Ο Εκδότης Δράσης βρίσκεται κάτω από το πλαίσιο και είναι η περιοχή όπου γράφετε κώδικα για να αλληλεπιδράσετε με τα δεδομένα από τους πόρους. Σε αυτή την περιοχή μπορείτε να βρείτε δημιουργημένες λειτουργίες, επιλέξτε τον τρέχοντα επιχειρησιακό πόρο, γράψτε δηλώσεις ερωτημάτων και ρυθμίστε μετασχηματιστές. ou can edit the query statement in the action editor and when you update the query statement, ο επεξεργαστής APP θα ενημερώσει αυτόματα τα αποτελέσματα της ενέργειας σας.

Κατά την εισαγωγή εντολών ενέργειας SQL, μπορείτε να επιλέξετε να χρησιμοποιήσετε τη λειτουργία SQL για να τις επεξεργαστείτε μέσω απλού κώδικα, ή μπορείτε να επιλέξετε τη λειτουργία GUI για να τις τροποποιήσετε στο πλαίσιο δράσης σας δίνουμε να ολοκληρώσετε γρήγορα την επεξεργασία ενέργειας. ή αναλυτικά σεμινάρια, ανατρέξτε στην τεκμηρίωση λειτουργίας.

### Λίστα Ενεργειών

Η λίστα ενεργειών βρίσκεται στην αριστερή πλευρά του πίνακα ρυθμίσεων, όπου μπορείτε να βρείτε την ενέργεια που χρειάζεστε για να την χρησιμοποιήσετε και να την τροποποιήσετε στον επεξεργαστή έκδοσης. ou can also add a new action here, see details of the creation process.

### Χώρος Εργασίας Δεδομένων

Ο χώρος εργασίας δεδομένων βρίσκεται στον πίνακα στα αριστερά του πλαισίου, όπου μπορείτε να δείτε τα ένθετα, ενέργειες και την παγκόσμια κατάσταση του συντάκτη, όλα εμφανίζονται σε μορφή Json. Μπορείτε επίσης να επεξεργαστείτε τα δεδομένα επιλέγοντάς τα στο χώρο εργασίας ή να τα διαγράψετε κάνοντας κλικ στο κουμπί Περισσότερες ρυθμίσεις.

### Insert

Η εισαγωγή βρίσκεται στον πίνακα δεξιά του πλαισίου, και μπορείτε να μετακινήσετε το ένθετο στο πλαίσιο σύροντας και ρίχνοντας το με το ποντίκι. Μπορείτε να βρείτε το ένθετο που χρειάζεστε με τον τύπο ένθεσης, ή μπορείτε να το αναζητήσετε και να το βρείτε γρήγορα.

### Επιθεώρηση

Η επιθεώρηση βρίσκεται στον πίνακα στα δεξιά του πλαισίου. Όταν επιλέγετε ένα ένθετο στο πλαίσιο ή στο χώρο εργασίας δεδομένων, ο πίνακας στα δεξιά θα μεταβεί αυτόματα στη σελίδα εισαγωγής. Από αυτή τη σελίδα μπορείτε να ορίσετε μεταβλητές, να ενημερώσετε το όνομα εισαγωγής, να δημιουργήσετε χειριστές συμβάντων ή να εκτελέσετε μια λειτουργία διαγραφής. Η γραμμή εργαλείων του ένθετου θα ρυθμιστεί επίσης διαφορετικά για διαφορετικά ένθετα, έτσι ώστε να μπορείτε να αλλάξετε το περιεχόμενο των διαφορετικών ενθέτων.

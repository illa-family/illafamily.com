## <Heading hidden>Λίστα ενσωμάτωσης</Heading>

Η ILLA προσφέρει μια σειρά προκατασκευασμένων ενσωματώσεων που σας επιτρέπουν να συνδεθείτε εύκολα με τις πηγές δεδομένων σας με ελάχιστη διαμόρφωση. Απλά επιλέξτε την κατάλληλη ενσωμάτωση από τη λίστα των διαθέσιμων επιλογών, εισάγετε τα απαραίτητα διαπιστευτήρια και μπορείτε να αρχίσετε να χρησιμοποιείτε την πηγή δεδομένων σας στο ILLA. Αυτό εξαλείφει την ανάγκη δημιουργίας σύνθετων διαμορφώσεων και σας επιτρέπει να αρχίσετε γρήγορα να εργάζεστε με τα δεδομένα σας.

### Ενσωμάτωση βάσης δεδομένων

| Όνομα              | Περιγραφή                                                                                                                                                                                                                                                                 |
| ------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Mysql              | Η MySQL είναι ένα σχεσιακό σύστημα διαχείρισης βάσεων δεδομένων ανοικτού κώδικα (RDBMS) που βασίζεται στη Δομημένη Γλώσσα Ερωτήματος (SQL).                                                                                                                               |
| PostgreSQL         | Το PostgreSQL είναι ένα ανοικτού κώδικα αντικειμενο-σχεσιακό σύστημα διαχείρισης βάσεων δεδομένων (ORDBMS) που είναι γνωστό για την αξιοπιστία και τον πλούτο των χαρακτηριστικών.                                                                                        |
| MariaDB            | Η MariaDB είναι ένα σχεσιακό σύστημα διαχείρισης βάσεων δεδομένων ανοιχτού κώδικα που είναι ένα πιρούνι της βάσης δεδομένων MySQL.                                                                                                                                        |
| TiDB               | TiDB είναι μια ανοιχτού κώδικα βάση δεδομένων NewSQL που υποστηρίζει Hybrid Transactional και Analytical Processing φόρτους εργασίας.                                                                                                                                     |
| Redis              | Το Redis είναι ένα κατάστημα δομών δεδομένων ανοιχτού κώδικα στη μνήμη που χρησιμοποιείται συχνά ως βάση δεδομένων, cache, και ως μεσίτης μηνυμάτων.                                                                                                                      |
| MongoDB            | MongoDB είναι ένα δωρεάν και ανοικτού κώδικα cross-platform ντοκουμέντο-προσανατολισμένο πρόγραμμα βάσης δεδομένων.                                                                                                                                                       |
| Ελαστική Αναζήτηση | Το Elasticsearch είναι μια μηχανή ανοιχτού κώδικα, κατανεμημένης αναζήτησης και ανάλυσης που χτίστηκε πάνω από το Apache Lucene.                                                                                                                                          |
| Firebase           | Το Firebase είναι μια πλατφόρμα ανάπτυξης εφαρμογών για κινητά και web με βάση το cloud που αναπτύχθηκε από την Google.                                                                                                                                                   |
| Supabase DB        | Το Supabase είναι μια πλατφόρμα ανοιχτού κώδικα για την κατασκευή εφαρμογών χωρίς διακομιστές σε πραγματικό χρόνο. Είναι χτισμένο πάνω από την PostgreSQL και παρέχει μια σειρά από εργαλεία και υπηρεσίες για γρήγορη κατασκευή και ανάπτυξη εφαρμογών χωρίς διακομιστή. |
| Clickhouse         | Το ClickHouse είναι ένα σύστημα διαχείρισης βάσεων δεδομένων ανοιχτού κώδικα που έχει σχεδιαστεί για online αναλυτική επεξεργασία (OLAP) και ανάλυση δεδομένων σε πραγματικό χρόνο.                                                                                       |

### Ενσωμάτωση API

| Όνομα                 | Περιγραφή                                                                                                                                                        |
| --------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| REST API              | Το REST είναι ένα αρχιτεκτονικό ύφος για την κατασκευή διαδικτυακών υπηρεσιών. RESTful APIs είναι APIs που ακολουθούν το REST αρχιτεκτονικό στυλ.                |
| Amazon S3(General S3) | Το Amazon S3 είναι μια υπηρεσία αποθήκευσης αντικειμένων που προσφέρει η AWS. Παρέχει ασφαλή, επεκτάσιμη και χαμηλού κόστους αποθήκευση δεδομένων στο διαδίκτυο. |
| SMTP                  | SMTP (Simple Mail Transfer Protocol) είναι ένα πρωτόκολλο για την αποστολή και λήψη μηνυμάτων ηλεκτρονικού ταχυδρομείου μέσω του διαδικτύου.                     |
| GraphQL               | Το GraphQL είναι μια γλώσσα ερωτημάτων για τα APIs και ένα runtime για την εκπλήρωση αυτών των ερωτημάτων με τα υπάρχοντα δεδομένα σας.                          |

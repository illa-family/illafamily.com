## <Heading hidden>Πόρος</Heading>

Συνδεθείτε στην πηγή των δεδομένων σας και καλέστε την στην ILLA Builder. Σας παρέχουμε προσαρμόσιμες ενότητες σύνδεσης για διαφορετικούς τύπους πηγών δεδομένων. Μπορείτε να δημιουργήσετε μια σταθερή σύνδεση μεταξύ των δεδομένων σας και ILLA σε σύντομο χρονικό διάστημα, απλοποιώντας σημαντικά τη διαδικασία πρόσβασης στα δεδομένα.

Μετά την επιτυχή δημιουργία ενός πόρου, μπορείτε να τον καλέσετε σε οποιαδήποτε εφαρμογή στο ILLA. Μπορείτε επίσης να μοιραστείτε τους πόρους σας με τους συν-προγραμματιστές σας, βελτιώνοντας την αποτελεσματικότητα της κατασκευής εφαρμογών.

### Τύπος υποστηριζόμενων πόρων

Η ILLA υποστηρίζει μια σειρά από σημαντικές βάσεις δεδομένων, όπως η MySQL και η Postage, καθώς και APIs όπως το Rest API και το GraphQL.

### Δημιουργήστε έναν πόρο

Οι πόροι αντιπροσωπεύουν συνδέσεις μεταξύ του δημιουργού ILLA και των πηγών δεδομένων σας. Δημιουργήστε έναν πόρο με τα παρακάτω βήματα:

#### 1.  Πλοηγηθείτε στην καρτέλα Πόροι στη γραμμή πλοήγησης της αρχικής σελίδας και κάντε κλικ στο κουμπί "Δημιουργία νέου"

  src='https://cdn.illacloud.com/official-website/img/docs/re_e_1.png'
  alt=""
/>

#### 2.  Επιλέξτε εδώ τον τύπο δεδομένων σας

<img
  src='https://cdn.illacloud.com/official-website/img/docs/qs_e_2.png'
  alt=""
/>

#### 3.  Επεξεργαστείτε τις πληροφορίες πηγής δεδομένων σας και κάντε κλικ στο κουμπί "Δημιουργία πόρων" και στη συνέχεια ολοκληρώστε τη διαδικασία

<img
  src='https://cdn.illacloud.com/official-website/img/docs/re_e_2.png'
  alt=""
/>

Επιπλέον, μπορείτε να δημιουργήσετε πόρους στη λίστα ενεργειών κάτω από τον επεξεργαστή εφαρμογών

> ### Δοκιμή σύνδεσης
>
> Όταν δημιουργείτε ή βλέπετε πόρους, μπορείτε να κάνετε κλικ στο κουμπί ελέγχου κάτω από τον πίνακα πόρων για να ελέγξετε αν η πηγή δεδομένων είναι συνδεδεμένη με επιτυχία.

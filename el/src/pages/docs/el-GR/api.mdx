Συνδεθείτε σε ένα API
Συνδεθείτε σε οποιαδήποτε REST, GraphQL API.
Μπορείτε να συνδέσετε το ILLA με τα περισσότερα APIs χρησιμοποιώντας τις εγγενείς ενοποιήσεις μας. Αν χρειαστεί να συνδεθείτε με ένα εσωτερικό API, ή αν δεν υπάρχει υποστηριζόμενη ενσωμάτωση, μπορείτε να συνδεθείτε χρησιμοποιώντας την ολοκλήρωση REST API ή την ολοκλήρωση GraphQL.

1.  Δημιουργήστε έναν πόρο API 
    Οι πόροι αντιπροσωπεύουν συνδέσεις με πηγές δεδομένων και είναι αυτό που αμφισβητείτε στις εφαρμογές ILLA. Συνδεθείτε στο ILLA και μεταβείτε στην καρτέλα Πόροι > κάντε κλικ στην επιλογή Δημιουργία νέου και επιλέξτε το API στο οποίο θέλετε να συνδεθείτε, όπως REST API. Ανοίγει τη ρύθμιση παραμέτρων ενός τρόπου πόρων όπου προσθέτετε πληροφορίες σύνδεσης.

Στο ελάχιστο, χρειάζεστε τη βασική διεύθυνση URL του API που θέλετε να συνδεθείτε. Μπορείτε επίσης να περάσετε παραμέτρους URL, κεφαλίδες, κλπ. Αν το API σας απαιτεί έλεγχο ταυτότητας, μπορείτε να το ρυθμίσετε και σε αυτή τη σελίδα. Για παράδειγμα, το OAuth2.0 απαιτεί URL εξουσιοδότησης, URL διακριτικού πρόσβασης, κλπ. Αφού εισάγετε τα στοιχεία σύνδεσής σας, κάντε κλικ στην επιλογή Αποθήκευση πόρου. 
2\. Ερώτηση API σας
Στον επεξεργαστή εφαρμογών, κάντε κλικ στο κουμπί + Νέα στη λίστα ενεργειών> επιλέξτε REST API και επιλέξτε ένα υπάρχον API REST ή δημιουργήστε ένα νέο> κάντε κλικ στην επιλογή Δημιουργία ενέργειας και ενημερώστε τις ενέργειες με πρόσθετες παραμέτρους URL, κεφαλίδες κ.λπ.

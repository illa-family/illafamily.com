## <Heading hidden>カスタムAPI認証</Heading>

ILLAは、デフォルトで多くの認証規格をサポートしています(詳細はこちらをご覧ください)。 
さらに、ILLAにはカスタム認証モードがあり、事実上あらゆるAPI認証をサポートする複数のステップを組み合わせることができます。 
これらの認証方法は、RESTリソースとGraphQLリソースの両方に使用できます。

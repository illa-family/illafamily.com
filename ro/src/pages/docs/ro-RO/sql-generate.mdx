<Heading hidden>SQL Generation</Heading>

Sprijinim generarea SQL în aceste resurse: PostgreSQL, MySQL, Oracle DB, MariaDB, TIDB, SupabaseDB și ClickHouse. Poți introduce limba naturală și obține o declarație SQL gata atunci când folosești aceste resurse. 

**Introduceți pagina de editare a aplicației și creați o acțiune folosind una dintre resursele de mai sus.**

**Selectați tipul de declarație pe care doriți să o generați**

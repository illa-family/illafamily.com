## <Heading hidden>Обзор</Heading>

### 1.  Развернуть конструктор ILLA

Перед началом использования строителя ILLA вы должны использовать ILLA Cloud или само-хостинг ILLA для размещения ваших приложений. Мы предоставляем Вам ILLA CLI, Docker Compose, и K8s самому принимающему ILLA менее чем за 5 минут. Позже мы предоставим Вам ILLA Cloud для того, чтобы помочь Вам просто начать работу с конструктором ILLA.

<img
  src='https://cdn.illacloud.com/official-website/img/docs/qs_e_1.png'
  alt=""
/>

### 2.  Получить данные пользователей

Зарегистрируйтесь и войдите в ILLA Builder, затем войдите на вкладку ресурсов. Нажмите на кнопку "Создать новый", затем вы можете выбрать тип ресурса здесь. Мы предоставляем вам REST API и MySQL для подключения к вашим данным. Postgres, Redis и MongoDB будут доступны в этом месяце.

<img
  src='https://cdn.illacloud.com/official-website/img/docs/qs_e_2.png'
  alt=""
/>

### 3.  Перетащите вставки

После подготовки данных вы можете вернуться на вкладку Приложения и создать приложение. Список вкладок находится в правой части страницы. Вы можете выбрать один из них и перетащить его в рамку. После этого в панели Инспектирования нажмите на кнопку вставить и отредактировать ее параметры. Мы предоставляем вам режим markdown для редактирования метки каждой вставки. Если вы знакомы с синтаксисом Markdown, вы можете быстро настроить размер и толщину текста. Тем временем вы можете использовать {{double brackets}} для вызова действий в поле ввода текста для достижения большего результата в ILLA Builder.

### 4.  Добавить действие в приложениях

Требуется действие для преобразования данных из ресурсов на вставку. Мы называем это действием. Вы можете записать запрос в действие, чтобы поймать данные из вашего ресурса. На этой основе вы можете писать JavaScript в действиях для выполнения инструкций по обработке данных, таких как фильтр и слияние. Ниже приводится пример MySQL:

<img
  src='https://cdn.illacloud.com/official-website/img/docs/qs_e_3.png'
  alt=""
/>

1.  Создайте действие MySQL в редакторе приложений. Это имя действия 1.

2.  Введите код запроса в панели, например SELECT \* FROM Users.

3.  Выполнить это действие, результат запроса будет отображаться на нижней панели.

### 5.  Показать ваши данные

Когда действие успешно создано, необходимо привязать результат действия к компоненту. Вы должны заменить переменную значения на панели проверки с результатами действий {{action1.data}}. Затем данные компонента будут привязываться к результату действия.

## <Heading hidden>Пользовательская аутентификация API</Heading>

ILLA поддерживает многие стандарты аутентификации по умолчанию (подробнее см. здесь ). 
Кроме того, ILLA имеет режим пользовательской аутентификации , который позволяет комбинировать несколько шагов, поддерживающих практически любую аутентификацию API. 
Эти методы аутентификации могут использоваться как для REST так и для ресурсов GraphQL.

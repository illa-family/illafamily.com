## <Heading hidden>Подключиться к API</Heading>

Подключиться к любому REST, GraphQL API.

Вы можете подключить ILLA к большинству API, используя наши собственные интеграции. Если вам нужно подключиться к внутреннему API, или если нет поддерживаемой интеграции, вы можете подключиться, используя интеграцию REST API или GraphQL интеграцию.

### 1.  Создать ресурс API

Ресурсы представляют собой соединения с источниками данных и это то, что вы запрашиваете в ваших ILLA приложениях.Войдите в ILLA и перейдите на вкладку **Ресурсы** > нажмите **Создать новый** и выберите API, к которому вы хотите подключиться, например **REST API**. Это открывает модаль **Настройте ресурс** где вы добавляете информацию о подключении.

<img
  src='https://cdn.illacloud.com/official-website/img/docs/connect/connect-api/config.png'
  alt=""
/>

Как минимум, вам необходим базовый URL-адрес API, к которому вы хотите подключиться. Вы также можете передавать параметры URL, заголовки и т.д. Если ваш API требует аутентификации, вы можете также настроить это на этой странице. Например, для OAuth2.0 требуется URL авторизации, URL маркера доступа и т.д. После ввода данных о подключении нажмите **Сохранить ресурс.**

### 2.  Запрос API

В редакторе приложения, нажмите **+ Новый** в **Action List** > выберите **REST API** и выберите существующий REST API или создайте новый > нажмите **Создать действие** и обновите действия с дополнительными параметрами URL, и т. д.

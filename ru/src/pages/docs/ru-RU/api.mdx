Подключиться к API
Подключиться к любому REST, GraphQL API.
Вы можете подключить ILLA к большинству API, используя наши собственные интеграции. Если вам нужно подключиться к внутреннему API, или если нет поддерживаемой интеграции, вы можете подключиться, используя интеграцию REST API или GraphQL интеграцию.

1.  Создайте ресурс API 
    Ресурсы представляют собой соединения с источниками данных и то, что вы запрашиваете в ваших ILLA приложениях. Войдите в ILLA и перейдите на вкладку Ресурсы > нажмите кнопку Создать новый и выберите API, который вы хотите подключиться, такие как REST API. Это откроет модаль ресурса, где вы добавляете информацию о подключении.

Как минимум, вам необходим базовый URL-адрес API, к которому вы хотите подключиться. Вы также можете передавать параметры URL, заголовки и т.д. Если ваш API требует аутентификации, вы можете также настроить это на этой странице. Например, для OAuth2.0 требуется URL авторизации, URL маркера доступа и т.д. После ввода данных о подключении нажмите кнопку Сохранить ресурс. 
2\. Запросить API
в редакторе приложений, нажмите + Новый в Список действий > выберите REST API и выберите существующий REST API или создайте новый > нажмите на создание действия и обновите действия с дополнительными параметрами URL, и т. д.

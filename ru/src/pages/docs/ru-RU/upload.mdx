## <Heading hidden>Загрузить</Heading>

Компонент загрузки используется для того, чтобы пользователи могли загружать файлы со своих локальных компьютеров на любое облачное хранилище через API. Далее, мы расскажем, как настроить компонент загрузки, как связать компонент загрузки с облачным сервисом хранения, как использовать компонент загрузки для выбора изображения, и отображать его с изображением и т.д.

### Метод

Вы можете использовать другие компоненты для управления компонентом загрузки. Мы поддерживаем следующие три метода:

| проверка       | Используется для очистки сообщения о проверке                           |
| -------------- | ----------------------------------------------------------------------- |
| clearValue     | Используется для очистки выбранных                                      |
| набор отключен | Используется для установки компонента загрузки в отключено или включено |

### Данные загрузчика

Компонент загрузки содержит некоторые широко используемые данные, которые можно вызвать в приложении.

`value` массив, состоящий из данных base64 файлов. 

`files` массив объектов. Ключи объектов включают `lastModified`, `name`, `size`, `type`. Вы можете использовать `{{upload1.files.map(file =>file. ame)}}`, чтобы получить массив имен файлов, а также lastModified array, массив размеров, вводите массив тем же способом.

#### Загрузить один файл через ресурс S3

Конфигурация действий является следующим. 

Загрузить имя объекта

```jsx
{{upload1.files.map(file =>file.name)[0]}}
```

Загрузить данные

```jsx
{{upload1.value[0]}}
```

<img src='https://cdn.illacloud.com/official-website/img/docs/upload3.png' alt="" />

#### Загрузить несколько файлов через ресурс S3

Загрузить список имени объекта

```jsx
{{upload1.files.map(file =>file.name)}}
```

Загрузить список данных

```jsx
{{upload1.value}}
```

<img src='https://cdn.illacloud.com/official-website/img/docs/upload4.png' alt="" />

Загрузите изображение и отобразите его на компоненте изображения
Изменить источник изображений на файл base64 данных для загрузки компонента. Данные base64 хранятся в атрибуте значения в массиве. И мы должны скопировать файл префиксом с файловыми данными. Этот код является следующим. 

```jsx
{{'data:image/jpeg;base64,'+upload1.value[0]}}
```

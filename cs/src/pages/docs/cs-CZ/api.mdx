Připojte se k API
Připojte se k jakémukoliv REST, GraphQL API.
Můžete připojit ILLA k většině API pomocí našich původních integrací. Pokud se musíte připojit k internímu API, nebo pokud neexistuje podporovaná integrace, se můžete připojit pomocí naší integrace REST API nebo integrace s GraphQL.

1.  Vytvořte zdroj API 
    Zdroje představují připojení ke zdrojům dat a jsou tím, co máte v aplikacích ILLA dotaz. Přihlaste se do ILLA a přejděte na záložku Zdroje > klikněte na Vytvořit nové a vyberte API, ke kterému se chcete připojit. např. REST API. Tímto se otevře Konfigurace modálu zdrojů, kde přidáte informace o připojení.

Minimálně potřebujete základní URL API, ke které se chcete připojit. Můžete také zadat URL parametry, záhlaví, atd. Pokud vaše API vyžaduje autentizaci, můžete to také konfigurovat na této stránce. Například OAuth2.0 vyžaduje autorizační URL, URL přístupového tokenu atd. Po zadání údajů o připojení klikněte na Uložit dokument. 
2\. dotaz na API
v editoru aplikací, klikněte + Nový v Akčním seznamu > vyberte REST API a vyberte existující REST API nebo vytvořte nové> klikněte na vytvoření akce a aktualizujte akce s dalšími parametry URL, záhlaví atd.

## <Heading hidden>Risorsa</Heading>

Connettiti alla tua sorgente dati e chiamala nel tuo ILLA Builder. Vi forniamo moduli di connessione adattabili per diversi tipi di fonti di dati. È possibile costruire una connessione stabile tra i dati e ILLA in breve tempo, semplificando notevolmente il processo di accesso ai dati.

Dopo aver creato con successo una risorsa, è possibile chiamarla in qualsiasi app in ILLA. Puoi anche condividere le tue risorse con i tuoi co-sviluppatori, migliorando l'efficienza delle app di costruzione.

### Tipo di risorse supportate

ILLA supporta un certo numero di database principali, come MySQL e Postage, così come API come Rest API e GraphQL.

### Crea una risorsa

Le risorse rappresentano connessioni tra ILLA Builder e le fonti di dati. Crea una risorsa con i seguenti passaggi:

#### 1.  Vai alla scheda Risorse nella barra di navigazione della homepage e clicca sul pulsante "Crea nuovo"

<img
  src='https://cdn.illacloud.com/official-website/img/docs/re_e_1.png'
  alt=""
/>

#### 2.  Scegli il tipo di dati qui

<img
  src='https://cdn.illacloud.com/official-website/img/docs/qs_e_2.png'
  alt=""
/>

#### 3.  Affinare le informazioni di origine dei dati e fare clic sul pulsante "Crea risorsa" quindi terminare il processo

<img
  src='https://cdn.illacloud.com/official-website/img/docs/re_e_2.png'
  alt=""
/>

Inoltre, è possibile creare risorse nell'elenco delle azioni sotto l'editor di app

> ### Verifica connessione
>
> Quando si stanno creando o visualizzando le risorse, è possibile fare clic sul pulsante di prova sotto il pannello delle risorse per verificare se la sorgente dati è collegata con successo.

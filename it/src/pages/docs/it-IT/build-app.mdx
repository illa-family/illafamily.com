Assemble Components
Utilizza la libreria ILLA di componenti UI per costruire l'interfaccia utente della tua applicazione.
I componenti sono una parte essenziale di ILLA e funzionano allo stesso modo dei componenti React in quanto possono accettare proprietà e avere stato interno. Combinando la libreria di componenti ILLA con i tuoi dati, puoi creare app interattive.
Alcuni componenti comuni includono:
Il componente Tabella viene utilizzato per visualizzare i dati in un formato tabulare, e offre caratteristiche come smistamento, filtraggio, paginazione ed esportazione CSV. 
Il componente Form viene utilizzato per raggruppare i campi di input e inviarli come una singola unità. 
Il componente Seleziona input è una combinazione di un menu a tendina e un campo di input che consente agli utenti di selezionare un valore da un elenco o inserire un valore direttamente come testo. 
Il componente Grafico viene utilizzato per creare visualizzazioni di dati utilizzando grafici e grafici.

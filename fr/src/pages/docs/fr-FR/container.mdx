## <Heading hidden>Conteneur</Heading>

Vous pouvez regrouper les composants liés ensemble dans une carte en utilisant le composant Container. Les conteneurs supportent également plusieurs vues qui vous permettent de séparer le contenu en vues distinctes que les utilisateurs peuvent changer entre eux.

Les conteneurs fonctionnent comme n'importe quel autre composant. Une fois ajouté, vous pouvez regrouper les composants à l'intérieur en appuyant sur D ou K et en les faisant glisser sur le conteneur.

### Vues multiples

Les conteneurs supportent plusieurs vues pour grouper le contenu. Vous pouvez définir des vues dans la section **Vues** de l'onglet **Inspect**. Une seule vue est visible à la fois et les utilisateurs basculent entre eux en modifiant la vue actuelle du composant Container.

### Utiliser d'autres composants pour modifier les vues

Vous pouvez utiliser d'autres composants pour contrôler les conteneurs en ajoutant un gestionnaire d'événements à d'autres composants. Les méthodes suivantes sont supportées :

1.  Définir la vue actuelle par clé
2.  Définir la vue actuelle par index
3.  Afficher la vue suivante (visible)
4.  Afficher la vue précédente (visible)

### Connecter un onglet à un conteneur

Vous pouvez utiliser un composant d'onglet pour contrôler un conteneur en activant **Lien vers un conteneur** de l'onglet **Inspect**.

Après un lien vers un conteneur, les libellés des onglets changeront sur les libellés des vues. Vous pouvez changer de vue en cliquant sur les onglets sans gestionnaires d'événements.

## <Heading hidden>Liste d'intégration</Heading>

ILLA propose une gamme d’intégrations pré-construites qui vous permettent de vous connecter facilement à vos sources de données avec une configuration minimale. Il vous suffit de sélectionner l’intégration appropriée dans la liste des options disponibles, de saisir les identifiants nécessaires, et vous pouvez commencer à utiliser votre source de données au sein de ILLA. Cela élimine la nécessité de configurer des configurations complexes et vous permet de commencer à travailler rapidement avec vos données.

### Intégration de la base de données

| Nom                 | Libellé                                                                                                                                                                                                                                                         |
| ------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Mysql               | MySQL est un système de gestion de bases de données relationnelles open-source (RDBMS) basé sur le langage de requête structuré (SQL).                                                                                                                          |
| PostgreSQL          | PostgreSQL est un système de gestion de bases de données relationnelles open-source (ORDBMS) reconnu pour sa fiabilité et sa richesse en fonctionnalités.                                                                                                       |
| MariaDB             | MariaDB est un système de gestion de base de données relationnelle open-source qui est un fork de la base de données MySQL.                                                                                                                                     |
| TiDB                | TiDB est une base de données NewSQL open-source qui supporte les charges de travail Hybrid Transactional and Analytical Processing.                                                                                                                             |
| Redis               | Redis est un stockage de structure de données en mémoire ouvert qui est souvent utilisé comme base de données, cache et courtier de messages.                                                                                                                   |
| MongoDB             | MongoDB est un programme de base de données multi-plateforme libre et open-source.                                                                                                                                                                              |
| Recherche élastique | Elasticsearch est un moteur open source, distribué de recherche et d'analyse basé sur Apache Lucene.                                                                                                                                                            |
| Firebase            | Firebase est une plate-forme de développement d'applications web et mobile basée sur le cloud, développée par Google.                                                                                                                                           |
| Supabase DB         | Supabase est une plate-forme open-source pour la création d'applications sans serveur en temps réel. Il est construit sur PostgreSQL et fournit un certain nombre d'outils et de services pour construire et déployer rapidement des applications sans serveur. |
| Clickhouse          | ClickHouse est un système de gestion de base de données open-source orienté vers les colonnes qui est conçu pour le traitement analytique en ligne (OLAP) et l'analyse de données en temps réel.                                                                |

### Intégrations API

| Nom                   | Libellé                                                                                                                                               |
| --------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------- |
| API REST              | REST est un style architectural pour la construction de services web. Les APIs RESTful sont des APIs qui suivent le style architectural REST.         |
| Amazon S3(Général S3) | Amazon S3 est un service de stockage d'objets offert par AWS. Il fournit un stockage sécurisé, évolutif et peu coûteux pour les données sur Internet. |
| SMTP                  | SMTP (Simple Mail Transfer Protocol) est un protocole pour l'envoi et la réception de messages électroniques sur Internet.                            |
| GraphQL               | GraphQL est un langage de requête pour les APIs et un runtime pour remplir ces requêtes avec vos données existantes.                                  |

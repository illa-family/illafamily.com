## <Heading hidden>Collaboration avec les membres</Heading>

Les développeurs peuvent travailler simultanément dans ILLA Builder en partageant des liens. Les développeurs peuvent coéditer et communiquer avec d'autres en temps réel pour donner des commentaires en temps opportun, ce qui améliorera considérablement l'efficacité du développement.

### Inviter des membres

Le Propriétaire ou Administrateur de l'équipe peut inviter des utilisateurs via le bouton Inviter sur la page d'accueil. 

<img src='https://cdn.illacloud.com/official-website/img/docs/assemble/collaboration.png' alt="" />

<img src='https://cdn.illacloud.com/official-website/img/docs/assemble/collaboration(1).png' alt="" />

### Gérer les membres

<img src='https://cdn.illacloud.com/official-website/img/docs/assemble/collaboration(2).png' alt="" />

Les éditeurs peuvent inviter ou gérer les éditeurs, les spectateurs et les spectateurs peuvent inviter ou gérer les spectateurs. Si vous ne voulez pas que les éditeurs et les téléspectateurs invitent ou gèrent des membres ou vérifient la liste des membres, veuillez définir le paramètre « Autoriser les éditeurs et les utilisateurs à gérer les membres» pour désactiver.

### Collaborez dans la même application

Nous autorisons les membres à modifier une application en même temps. Vous pouvez voir quels composants sont en cours d'édition. Comme indiqué ci-dessous.

## <Heading hidden>Översikt</Heading>

### 1.  Distribuera ILLA Builder

Innan du börjar använda ILLA Builder bör du välja att använda ILLA Cloud eller Self-host ILLA för att vara värd för dina appar. Vi förser dig med ILLA CLI, Docker Compose, och K8s för att själv värd ILLA på under 5 minuter. Vi kommer att förse dig med ILLA Cloud senare för att hjälpa dig att komma igång med ILLA Builder.

<img
  src='https://cdn.illacloud.com/official-website/img/docs/qs_e_1.png'
  alt=""
/>

### 2.  Hämta användardata

Registrera dig och logga in på ILLA Builder, ange sedan resursfliken. Klicka på knappen "Skapa nytt" så kan du välja resurstypen här. Vi förser dig med REST API och MySQL för att ansluta dina data. Postgres, Redis och MongoDB kommer att finnas tillgängliga denna månad.

<img
  src='https://cdn.illacloud.com/official-website/img/docs/qs_e_2.png'
  alt=""
/>

### 3.  Dra och släpp inlägg

När dina data är förberedda kan du gå tillbaka till fliken Appar och skapa en app. Inläggslistan finns på höger sida av sidan. Du kan välja en av dem och dra in den i ramen. Klicka sedan på infoga och modifiera dess parametrar i kontrollpanelen. Vi ger dig markdown-läge för att redigera etiketten för varje insats. Om du är bekant med Markdown syntax, kan du snabbt justera textstorlek och tjocklek. Under tiden kan du använda {{double brackets}} för att åberopa åtgärdsdata i textrutan för att uppnå fler resultat i ILLA Builder.

### 4.  Lägg till åtgärd i appar

En åtgärd krävs för att omvandla data från resurser att infoga. Vi kallar det handling. Du kan skriva en fråga i handling för att fånga data från din resurs. På denna grund kan du skriva JavaScript i åtgärder för att utföra instruktioner för databehandling, såsom filter och sammanfogning. Ett MySQL-exempel ges nedan:

<img
  src='https://cdn.illacloud.com/official-website/img/docs/qs_e_3.png'
  alt=""
/>

1.  Skapa en MySQL-åtgärd i appredigeraren. Namngav det åtgärd1.

2.  Ange frågekoden i panelen, t.ex. SELECT \* FRÅN användare.

3.  Kör den här åtgärden, frågeresultatet kommer att visas på den nedre panelen.

### 5.  Visa dina data

När åtgärden har skapats måste du binda resultatet av åtgärden till komponenten. Du bör ersätta värdevariabeln på kontrollpanelen med resultatet av åtgärder {{action1.data}}. Därefter kommer komponentdatan att binda med åtgärdsresultatet.

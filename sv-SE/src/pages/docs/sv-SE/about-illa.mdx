## <Heading hidden>Om ILLA</Heading>

Med ILLA kan du snabbt bygga och distribuera interna appar. Anslut till dina databaser och API:er, sätt ihop användargränssnitt med dra-och-släpp-byggstenar som tabeller och formulär, och skriv frågor för att interagera med data med SQL och JavaScript.

### Vad är ILLA?

ILLA är en lågkodsplattform med öppen källkod för utvecklare att bygga och distribuera interna verktyg anmärkningsvärt snabbt.

<img
  src='https://cdn.illacloud.com/official-website/img/docs/introduction/about-illa-cloud.png'
  alt=""
/>

Med ILLA kan du enkelt ansluta till olika datakällor såsom databaser, API/GraphQL-slutpunkter, SaaS-verktyg och molnlagringstjänster. Dessa inkluderar populära alternativ som PostgreSQL, MongoDB, MySQL, Stripe, Google Kalkylark, och AWS S3. När den är ansluten kan ILLA köra frågor för att hämta och uppdatera data från dessa källor. Användargränssnittet (UI) tillhandahåller praktiska komponenter för att visualisera och modifiera data, såsom tabeller, diagram och formulär.

Med ILLA kan du snabbt och enkelt bygga ett användargränssnitt (UI) med färdiga widgetar som du kan dra och släppa på en grid-stil duk. Detta förenklar integrationen av fram- och baksidan av din applikation och optimerar byggprocessen. Dessutom stöder ILLA JavaScript inuti widgetar, frågor och andra komponenter, så att du kan lägga till logik, omvandla data och definiera komplexa arbetsflöden. Detta gör det enkelt att anpassa din applikation efter dina specifika behov.

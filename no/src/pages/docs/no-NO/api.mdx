Koble til en API
Koble til REST, GraphQL API.
Du kan koble ILLA til de fleste APIer som bruker våre innebygde integrasjoner. Dersom du trenger å koble til en intern API, eller hvis det ikke er støttet integrasjon, du kan koble til ved hjelp av REST-API-integrasjon eller GraphQL Integrasjon.

1.  Opprett en API-ressurs 
    Ressurser representerer tilkoblinger til datakilder og er det du spør etter i ILLA-appene. Logg inn i ILLA og naviger til Ressurs-fanen > Klikk Opprett ny og velg API du vil koble til, så som REST API. Dette åpner Konfigurer en ressursmodell der du legger til din tilkoblingsinformasjon.

Som et minimum trenger du grunn-nettadressen til API-et du vil koble til. Du kan også sende URL parametere, overskrifter osv. Dersom ditt API krever autentisering, kan du konfigurere dette også på denne siden. For eksempel OAuth2.0 krever en autorisasjons URL, Access Token URL, osv. Etter at du har skrevet inn tilkoblingsdetaljer, klikk Lagre ressurs. 
2\. Spør API
i App-editoren, klikk + Ny i Handlingslisten > velg REST API og velge en eksisterende REST API eller opprette en ny > klikk opprett handling og oppdatere handlinger med flere URL-parametere, overskrifter, osv.

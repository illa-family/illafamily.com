## <Heading hidden>Integrasjonsliste</Heading>

ILLA tilbyr en rekke forbygde integrasjoner som lar deg enkelt koble til datakildene dine med minimalt konfigurasjon. Velg helt enkelt riktig integrasjon fra listen over tilgjengelige alternativer, skriv inn de nødvendige opplysningene og du kan begynne å bruke datakilden din i ILLA. Dette eliminerer behovet for å sette opp komplekse konfigurasjoner og lar deg jobbe raskt med dine data.

### Database integrasjon

| Navn         | Beskrivelse                                                                                                                                                                                                    |
| ------------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Mysql        | MySQL er et operativsystem for relasjonsdatabase (RDBMS) som er basert på strukturert spørringsspråk (SQL).                                                                                                    |
| PostgreSQL   | PostgreSQL er et objekt- relasjonsbasert databaseadministrasjonssystem (ORDBMS) som er kjent for pålitelighet og funksjonsrik.                                                                                 |
| MariaDB      | MariaDB er et operativt system for relasjon i åpen kildekode som er et forgrenet av MySQL database.                                                                                                            |
| TiDB         | TiDB er en open-source NewSQL database som støtter Hybrid Transactional and Analytical Processing workloads.                                                                                                   |
| Redis        | Redis er en åpen kildekode datastrukturlager, som ofte brukes som en database, en cache og en melding megler.                                                                                                  |
| MongoDB      | MongoDB er et dokumentorientert databaseprogram med åpen kildekode som viser en åpen og samme kilde.                                                                                                           |
| Elastisk søk | Elasticsearch er en åpen kildekode distribuert søkemotor basert på Apache Lucene.                                                                                                                              |
| Firebase     | Firebase er en sky-basert mobil-og nett-applikasjonsplattform utviklet av Google.                                                                                                                              |
| Supabase DB  | Supabase er en åpen kilde plattform for å bygge sanntid serverløse programmer. Den er bygget oppå PostgreSQL og tilbyr en rekke verktøy og tjenester for rask å bygge og distribuere serverløse applikasjoner. |
| Clickhouse   | ClickHouse er et åpen kildekode kolonneorientert databasestyringssystem som er utviklet for elektronisk behandling (OLAP) og analyse av sanntidsdata.                                                          |

### API integrasjon

| Navn                   | Beskrivelse                                                                                                                  |
| ---------------------- | ---------------------------------------------------------------------------------------------------------------------------- |
| REST API               | REST er en arkitektonisk stil for å bygge nettjenester. Nyttige APIer er APIer som følger REST-byggets stil.                 |
| Amazon S3(Generelt S3) | Amazon S3 er en objektlagringstjeneste som tilbys av AWS. Det gir sikker, skalerbar og lavkostlagring for data på Internett. |
| SMTP                   | SMTP (Simple Mail Transfer Protocol) er en protokoll for sending og mottak av e-postmeldinger over Internett.                |
| GrafQL                 | GraphQL er et spørringsspråk for APIer og en kjøretid for å oppfylle disse spørringene med dine eksisterende data.           |

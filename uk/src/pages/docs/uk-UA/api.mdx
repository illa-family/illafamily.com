Підключення до API
підключитися до будь-якого REST, GraphQL API.
Ви можете під'єднати ILLA до більшості API використовуючи наші нативні інтеграції. Якщо вам потрібно підключитися до внутрішнього API, або якщо не підтримується інтеграція, ви можете підключитися за допомогою нашої інтеграції REST API або GraphQL Integration.

1.  Створіть ресурс API 
    Ресурси представляють підключення до джерел даних і це те, що ви шукаєте в своїх програмах ILLA. Увійдіть в систему ILLA і перейдіть до вкладки Ресурси > Натисніть Створити новий і виберіть API, до якого ви хочете приєднатися, такі як REST API. Це відкриває програму налаштування ресурсу в якій ви додаєте інформацію про з’єднання.

Як мінімум, потрібен базовий URL API, до якого ви хочете підключитися. Ви також можете передати параметри URL, заголовки та ін. Якщо ваш API потребує аутентифікації, ви можете також налаштувати це на цій сторінці. Наприклад, OAuth2.0 вимагає URL-адресу авторизації, Access Token URL тощо. Після введення даних про підключення, натисніть кнопку Зберегти ресурс. 
2\. Запит API
в редакторі додатків, натисніть + Новий Список дій > оберіть REST API і оберіть вже наявний REST API або створіть новий > Натисніть кнопку "Створити" і оновіть дії з додатковими параметрами URL. заголовки, і т. д.

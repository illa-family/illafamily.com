## <Heading hidden>Огляд</Heading>

### 1.  Розгортати ILLA Будівельник

Перед початком використання ILLA Builder, вам слід вибрати метод використання ILLA Cloud або ILLA для розміщення своїх програм. Ми надаємо вам ILLA CLI, Docker Compos та K8 для самоохолодження ILLA менш ніж за 5 хвилин. Пізніше ми надамо вам ILLA Cloud для того, щоб допомогти вам просто почати роботу з ILLA Builder.

  src='https://cdn.illacloud.com/official-website/img/docs/qs_e_1.png'
  alt=""
/>

### 2.  Отримати дані користувача

Зареєструйтеся і увійдіть до конструктора ILLA, а потім введіть вкладку ресурсів. Натисніть кнопку "Створити новий", потім ви можете вибрати тип ресурсу тут. Ми надаємо вам REST API і MySQL для підключення ваших даних. Postgres, Redis, і MongoDB будуть доступні цього місяця.

<img
  src='https://cdn.illacloud.com/official-website/img/docs/qs_e_2.png'
  alt=""
/>

### 3.  Перетягування маніпуляторів

Коли дані підготовлені, можна повернутися на вкладку "Додатки" та створити додаток. Вставка списку знаходиться в правій частині сторінки. Ви можете вибрати одну з них і перетягнути її у кадрі. Потім натисніть вставити і змінити її параметри в панелі Inspect . Ми надаємо вам режим markdown для редагування мітки кожного вставки. Якщо ви знайомі з синтаксисом Markdown , можете швидко налаштувати розмір і товщину тексту. Тим часом ви можете використовувати {{double brackets}} , щоб викликати дії дані в текстовому полі, щоб досягти більше результатів в конструкторі ILLA.

### 4.  Додати дію в Додатках

Дія необхідна для перетворення даних з ресурсів для вставки. Ми називаємо її дією. Ви можете написати запит в дії, щоб отримати дані з вашого ресурсу. Для цього ви можете написати JavaScript в дії для виконання інструкцій обробки даних, таких як фільтр і злиття. Приклад MySQL наведений нижче:

<img
  src='https://cdn.illacloud.com/official-website/img/docs/qs_e_3.png'
  alt=""
/>

1.  Створіть дію MySQL в редакторі програм. Назва дії 1.

2.  Введіть код запиту в панель, наприклад, SELECT \* FROM користувачі.

3.  Запустіть цю дію, результат запиту буде відображатися на нижній панелі.

### 5.  Показати ваші дані

Коли дія успішно створена, вам потрібно пов'язати результат дії з компонентом. Слід замінити змінну значення на панелі інспекції для дії {{action1.data}}. Після цього дані компоненту зв'яжуться з результатом.

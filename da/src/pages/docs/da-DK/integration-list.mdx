## <Heading hidden>Integrationsliste</Heading>

ILLA tilbyder en række pre-bygget integrationer, der giver dig mulighed for nemt at oprette forbindelse til dine datakilder med minimal konfiguration. Du skal blot vælge den relevante integration fra listen over tilgængelige muligheder, indtaste de nødvendige legitimationsoplysninger, og du kan begynde at bruge din datakilde i ILLA. Dette eliminerer behovet for at oprette komplekse konfigurationer og giver dig mulighed for hurtigt at begynde at arbejde med dine data.

### Database integrationer

| Navn             | Varebeskrivelse                                                                                                                                                                                                                    |
| ---------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Mysql            | MySQL er et open source relationelle database management system (RDBMS), der er baseret på den strukturerede Query Language (SQL).                                                                                                 |
| PostgreSQL       | PostgreSQL er et open source objekt-relationelle database management system (ORDBMS), der er kendt for sin pålidelighed og feature-rigdom.                                                                                         |
| MariaDB          | MariaDB er et open source relationelle database management system, der er en gaffel af MySQL database.                                                                                                                             |
| TiDB             | TiDB er en open source NewSQL-database, der understøtter Hybrid Transactional og Analytical Processing arbejdsbyrder.                                                                                                              |
| Redis            | Redis er en open source in-memory data struktur butik, der ofte bruges som en database, cache, og besked mægler.                                                                                                                   |
| MongoDB          | MongoDB er et gratis og open source cross-platform dokument-orienteret database program.                                                                                                                                           |
| Elastisk Søgning | Elasticsearch er en open source, distribueret søgning og analyse motor bygget på toppen af Apache Lucene.                                                                                                                          |
| Firebase         | Firebase er en cloud-baseret mobil og webapplikationsudviklingsplatform udviklet af Google.                                                                                                                                        |
| Supabase DB      | Supabase er en open source-platform til opbygning af realtids serverløse applikationer. Den er bygget oven på PostgreSQL og giver en række værktøjer og tjenester til hurtigt at opbygge og implementere serverløse applikationer. |
| Clickhouse       | ClickHouse er et open source kolonne-orienteret database management system, der er designet til online analytisk behandling (OLAP) og real-time data analyse.                                                                      |

### API integrationer

| Navn                  | Varebeskrivelse                                                                                                                     |
| --------------------- | ----------------------------------------------------------------------------------------------------------------------------------- |
| REST API              | REST er en arkitektonisk stil til opbygning af webtjenester. RESTful API'er er API'er, der følger den REST arkitektoniske stil.     |
| Amazon S3(Generel S3) | Amazon S3 er en objekt lagringstjeneste, der tilbydes af AWS. Det giver sikker, skalerbar og billig lagring af data på internettet. |
| Smtp                  | SMTP (Simple Mail Transfer Protocol) er en protokol til afsendelse og modtagelse af e-mails over internettet.                       |
| GraphQL               | GraphQL er en forespørgsel sprog for API'er og en runtime for at opfylde disse forespørgsler med dine eksisterende data.            |

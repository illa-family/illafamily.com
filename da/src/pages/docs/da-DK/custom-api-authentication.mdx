## <Heading hidden>Brugerdefineret API-godkendelse</Heading>

ILLA understøtter mange godkendelsesstandarder som standard (se her for mere information). 
Derudover har ILLA en brugerdefineret Auth godkendelse, der giver dig mulighed for at kombinere flere trin, der understøtter stort set enhver API godkendelse. 
Disse autentificeringsmetoder kan bruges til både REST og GraphQL ressourcer.

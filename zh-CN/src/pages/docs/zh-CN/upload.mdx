## <Heading hidden>上传</Heading>

上传组件用于允许用户通过API从本地机器上传文件到任何云存储。 接下来，我们将介绍如何配置上传组件，如何将上传组件与云存储服务联系起来。 如何使用上传组件来选择一个图像，并用图像等显示它。

<img src='https://cdn.illacloud.com/official-website/img/docs/upload1.png' alt="" />

### 属性

| 财产       | 描述                                                                                                                                     |
| -------- | -------------------------------------------------------------------------------------------------------------------------------------- |
| 类型       | 设置上传组件的类型，包括按钮和 Drop区。                                                                                                                 |
| 文本       | 上传组件上显示的文本。                                                                                                                            |
| 选择类型     | 将选择类型设置为多个文件、单个文件或目录。                                                                                                                  |
| 文件类型     | 允许上传的文件扩展名列表。 没有值允许所有文件类型。                                                                                                             |
| 追加新选定的文件 | 将选中的新文件附加到选定的文件或替换多个文件或目录的选定类型中的文件。                                                                                                    |
| 文件列表     | 设置是否在上传组件下方显示选中的文件列表。 上传器的高度将根据文件列表的长度动态变化。                                                                                            |
| 解析值      | 尝试解析选中的文件，并支持 JSON、CSV、TSV、Excel和 TXT 文件。 解析后的数据可以通过 {{upload1.parsedValue}} 访问。 无法解析的文件将在数组中为空 所有文件都可以在数组中以 Base64 编码的字符串使用，无论此选项是什么。 |
| 正在加载     | 组件是否应显示加载指示器。                                                                                                                          |
| 已禁用      | 设置是否禁用此组件以及在什么条件下禁用它。 例如，如果输入1组件为空，则该组件应禁用：{请注意，输入值== altil }}                                                                         |
| 工具提示     | 当用户悬停在组件上时在组件上显示工具提示。                                                                                                                  |
| 必填字段     | 确定是否需要。                                                                                                                                |
| Min size | 要设置允许添加的最小文件大小。                                                                                                                        |
| Max size | 要设置允许添加的最大文件大小。                                                                                                                        |
| 自定义规则    | 设置验证失败时显示的文本。                                                                                                                          |
| 隐藏验证消息   | 设置验证失败时是否显示验证消息。                                                                                                                       |
| 表单数据键    | 为组件设置一个密钥。 在将此组件添加到表单后，此密钥将用于在提交表单时识别此组件的数据。                                                                                           |
| Hidden   | 设置是否显示此组件以及在什么条件下显示它。 例如，如果当前用户不是用户A，这个组件应该隐藏：{□ currentUserInfo.nickname != 'User A' }}                                               |
| 变体       | 设置按钮上传组件到大纲或填充。                                                                                                                        |
| 主题颜色     | 设置按钮上传组件的颜色。                                                                                                                           |

### 上传器数据

上传组件有一些常用的数据，可以在应用程序中调用。

`value`，一个由数据库64数据组成的数组。 

`files` 对象数组。 对象的键值包括“lastModified”、“name`、`size`、`type\`”。 您可以使用 "{{upload1.files.map(ile =>文件"。 你可以获得一个文件名数组并且你也可以获得最后修改数组、大小数组、类型数组。 

### 上传文件到云存储服务

<img src='https://cdn.illacloud.com/official-website/img/docs/upload2.gif' alt="" />

#### 通过S3资源上传单个文件

行动配置如下。 

上传对象名称

```jsx
{{upload1.files.map(ile =>file.name)[0]}}
```

上传数据

```jsx
{{upload1.value[0]}}
```

<img src='https://cdn.illacloud.com/official-website/img/docs/upload3.png' alt="" />

#### 通过S3资源上传多个文件

上传对象名称列表

```jsx
{{upload1.files.map(ile =>file.name)}}
```

上传数据列表

```jsx
{{upload1.value}}
```

<img src='https://cdn.illacloud.com/official-website/img/docs/upload4.png' alt="" />

上传图像并在图像组件上显示
将图像源更改为文件基准64 上传组件数据。 Base64 数据存储在数组中的值属性中。 我们应该将文件前缀与文件数据分割开来。 该守则如下。 

```jsx
{{'data:image/jpeg;base64'+upload1.value[0]}}
```

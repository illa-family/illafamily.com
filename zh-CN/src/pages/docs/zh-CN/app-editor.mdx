## <Heading hidden>项目编辑器</Heading>

APP编辑器是用于开发应用的界面，主要由一下几个部分组成：

-   工具栏
-   Frame
-   配置面板（Action editor）
-   数据工作区（Data workspace）
-   Insert
-   检查

<img alt="editor"  src={require('@/img/official-site/content.png').default} />

### 工具栏

项目导航栏位于APP编辑器顶部，主要用于配置项目，可以用于：

-   点击左侧ILLA图标回到主页
-   查看项目信息与项目保存状态
-   显示/隐藏项目左侧、右侧、下方面板
-   调整项目缩放比例
-   显示项目当前所有开发者
-   警告APP当前错误状态。
-   将项目分享给其他开发者
-   运行APP。

#### 更多功能敬请期待

-   从json中导入项目
-   复制APP。
-   以json格式导出项目
-   选择需要的项目版本
-   查看编辑历史
-   重新加载所有操作。

### Frame

帧是您可以编辑 APP 图形接口的 APP 区域。 您可以从右手的插入面板中拖放插入框； 您也可以选择放入帧中的机械臂，并使用鼠标拖放来修改插入的位置和大小。

当您拖放插入或修改其大小时， 一个虚线正方形将出现在框架的后台，帮助您在框架内整理好的插件。 当您在拖放时遇到重叠插入， ILLA 生成器将优先考虑您正在移动的插入，并自动将屏蔽插入移出您的位置。

#### 预览模式

预览切换按钮位于帧右上角。 预览模式的切换按钮位于画布右上角，点击后编辑器会进入预览模式，左右下侧的面板会被隐藏，画布中的组件会无法修改，所有组件将会按照1:1的大小呈现在您面前。

> ### 支持的浏览器
>
> 我们推荐您在Google Chrome中打开ILLA Builder

### 操作编辑器

动作编辑器位于框架下，是您写代码与资源数据交互的区域。 在这个领域，您可以找到创建的操作，选择当前的操作资源，写查询语句并设置变压器。 ou 可以编辑动作编辑器中的查询语句和更新查询语句。 APP 编辑器将自动更新您的操作结果。

当输入 SQL 操作语句时，您可以选择使用 SQL 模式来编辑它们。 或者您可以选择图形界面模式，在我们给您的操作框架下修改它们，快速完成操作编辑操作。 或详细的教程，请查看操作文档。

### Action列表

Action列表位于配置面板的左侧，您可以在此处查找需要使用的Action，在配置面板中进行修改。您还可以在此处新建Action，详细过程请参考创建Action

### 数据工作区

数据工作区位于画布左侧的面板中，您可以在此处查看编辑器中的组件、Action以及全局状态，所有的状态信息会以Json的格式展示在工作区中。您也可以通过在工作区中选中数据进行编辑，或是点击更多设置按钮对选中数据进行删除处理。 您也可以通过在工作区选择数据或点击更多设置按钮来删除数据。

### Insert

插入处位于帧右侧面板内， 并且您可以通过拖动和丢弃鼠标将插入框移动到帧中。 您可以通过输入类型找到您需要的插入，或者您可以搜索并快速找到它。

### 检查

检查位于框架右侧的面板。 当您选择一个插入帧或数据工作区时，右侧面板将自动切换到插入页面。 您可以在此页面中设置变量，更新插入名称，创建事件处理程序或执行插入删除操作。 插入的工具栏也将对不同的插入设置不同，以便您可以更改不同插入的内容。

## <Heading hidden>Recurso</Heading>

Conéctate a tu fuente de datos y llámalo en tu Constructor de ILLA. Le proporcionamos módulos de conexión adaptables para diferentes tipos de fuentes de datos. Puede crear una conexión estable entre sus datos e ILLA en poco tiempo, simplificando enormemente el proceso de acceso a los datos.

Después de crear con éxito un recurso, puedes llamarlo en cualquier aplicación en ILLA. También puedes compartir tus recursos con tus co-desarrolladores, mejorando la eficiencia de las aplicaciones de construcción.

### Tipo de recursos soportados

ILLA soporta una serie de bases de datos importantes, como MySQL y Postage, así como APIs como Rest API y GraphQL.

### Crear un recurso

Los recursos representan conexiones entre el Constructor de ILLA y sus fuentes de datos. Crear un recurso siguiendo los siguientes pasos:

#### 1.  Navega a la pestaña Recursos en la barra de navegación de la página de inicio y haz clic en el botón "Crear nuevo"

<img
  src='https://cdn.illacloud.com/official-website/img/docs/re_e_1.png'
  alt=""
/>

#### 2.  Elige aquí tu tipo de datos

<img
  src='https://cdn.illacloud.com/official-website/img/docs/qs_e_2.png'
  alt=""
/>

#### 3.  Refina la información de tu fuente de datos y haz clic en el botón "Crear recurso" y luego termina el proceso

<img
  src='https://cdn.illacloud.com/official-website/img/docs/re_e_2.png'
  alt=""
/>

Además, puedes crear recursos en la lista de acciones bajo el editor de aplicaciones

> ### Probar conexión
>
> Cuando está creando o viendo recursos, puede hacer clic en el botón de prueba debajo del panel de recursos para comprobar si la fuente de datos está conectada correctamente.

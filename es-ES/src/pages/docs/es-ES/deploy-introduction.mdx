## <Heading hidden>Desplegar introducción</Heading>

### Nube

Pruebe la implementación de ILLA Cloud con un solo clic para obtener la experiencia más fácil.

### Autohost

Usted puede alojarse localmente con la ILLA CLI, la composición de Docker, el timón de k8s, etc. Hemos optimizado repetidamente el proceso auto-anfitrión para permitirle realizar implementaciones de software y auto anfitrión en tan solo unos minutos. En comparación con la nube, puede utilizar los productos ILLA de forma más segura y fiable, y usted puede usar ILLA Builder simplemente visitando el sitio de implementación después de alojar y desplegar el Constructor localmente.

Para más información sobre autohost, por favor vea las páginas de arranque de casco de la ILA CLI, Docker compose, k8s.
Si tiene dificultades con su auto-implementación, póngase en contacto con nosotros en GitHub o business@illasoft.com.

#### Desplegar con ILLA CLI

Por favor, compruébalo: [CLI de ILLA](https://www.illacloud.com/docs/illa-cli)

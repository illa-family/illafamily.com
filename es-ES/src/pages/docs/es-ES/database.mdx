ILLA se conecta a la mayoría de bases de datos, incluyendo PostgreSQL, MySQL, TiDB y MongoDB. Por favor revise esta lista aquí para ver las fuentes de datos que actualmente soporta ILLA. Las próximas fuentes de datos que estamos planeando apoyar están listadas aquí. Si la ILLA carece de ciertos recursos de datos que está usando, por favor rellene el formulario aquí.
Los detalles de conexión requeridos por ILLA dependen del tipo de base de datos que desee conectar. Esta información generalmente incluye un host, puerto, credenciales de usuario y cadena de conexión. Consulte la configuración o documentación de su base de datos para obtener esto.

1.  Crear un recurso de base de datos
    Los recursos representan conexiones a fuentes de datos y son lo que consulta dentro de sus aplicaciones ILLA. Inicie sesión en ILLA y vaya a la pestaña Recursos > haga clic en Crear nueva y seleccione la base de datos a la que desea conectarse. Esto abre la modal Configurar un recurso donde se agrega la información de conexión.

Después de introducir los detalles de su conexión, haga clic en Prueba de Conexión para comprobar que ILLA puede conectarse correctamente. Una vez completado, haga clic en Guardar recurso para guardar el recurso. 
Además, puedes crear recursos en la lista de acciones bajo el editor de aplicaciones. 
2. Consulta tu base de datos
en el editor de aplicaciones, haga clic en + Nuevo en Lista de Acciones > seleccione el tipo de recurso y elija un recurso existente o cree un nuevo recurso > haga clic en crear acción y actualice las acciones para recuperar alguna información de una tabla en su base de datos.

Haga clic en Guardar y luego en Ejecutar. Los resultados de la consulta se muestran automáticamente después de completar la consulta. 
Hay algunos ajustes avanzados, incluyendo renombrar, ejecutar ajustes, transformar datos, y configurar el manejo de éxito o fallo, como se muestra a continuación.

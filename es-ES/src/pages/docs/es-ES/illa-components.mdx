### Buscar un componente

Puede hacer clic en la pestaña INSERTAR en el panel de la derecha para entrar en la lista de componentes. Introduzca el nombre del componente y busque el componente a través del cuadro de entrada de arriba.

<img
  src='https://cdn.illacloud.com/official-website/img/docs/assemble/illa_components(34).png'
  alt=""
/>

### Usa el controlador de eventos para activar eventos

Tomando el componente de botón como ejemplo, puede establecer un controlador de eventos para el componente a través de la imagen mostrada para activar una acción o controlar otros componentes, etc. Cuando hace clic en un componente, o cuando los datos del componente cambian, se pueden activar los eventos correspondientes.

<img
  src='https://cdn.illacloud.com/official-website/img/docs/assemble/illa_components(35).png'
  alt=""
/>

### Usar los datos del componente en acciones

Puede utilizar datos de componentes en acciones u otros componentes. Tomando el componente de entrada como ejemplo, puede utilizar {{input1.value}} para obtener el valor introducido en el componente. Los diferentes componentes pueden tener diferentes formatos de datos. Por favor, vea los formatos de datos a través del panel de la izquierda.

<img
  src='https://cdn.illacloud.com/official-website/img/docs/assemble/illa_components(36).png'
  alt=""
/>

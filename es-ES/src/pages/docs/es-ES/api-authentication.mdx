## <Heading hidden>Autenticación API</Heading>

La ILLA puede apoyar casi cualquier tipo de autenticación. Nos hemos asociado con varias compañías para apoyar todos los tipos estándar de autenticación y también hemos desarrollado métodos para soportar cualquier método de autenticación personalizado que no soportamos nativamente.

Puede utilizar tokens estáticos de autenticación o crear un sistema de autenticación que le pida al usuario final para sus credenciales cada vez que acceda a la aplicación.

Esta página le dará instrucciones sobre cómo autenticar su API. Si tienes preguntas o necesitas ayuda, puede ponerse en contacto con nuestro equipo de soporte usando la burbuja de Intercom en esta página o el icono de ayuda dentro del editor.
